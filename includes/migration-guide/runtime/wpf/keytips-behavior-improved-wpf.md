### <a name="keytips-behavior-improved-in-wpf"></a>Popisy tlačítek chování vylepšené v subsystému WPF

|   |   |
|---|---|
|Podrobnosti|Popisy tlačítek chování byla změněna zpřístupnit se chování v aplikaci Microsoft Word a Průzkumníka Windows. Tak, že zkontrolujete, jestli má stav klávesovou zkratku, nebo není v případě třídy <xref:System.Windows.Input.KeyEventArgs.SystemKey> (zejména <xref:System.Windows.Input.Key> nebo <xref:System.Windows.Input.Key.F11>) se stiskne, WPF zpracovává klávesovou zkratku klíče odpovídajícím způsobem. Popisy tlačítek teď zavřete nabídku i v případě, že se otevře pomocí myši.|
|Doporučení|Není k dispozici|
|Rozsah|Edge|
|Version|4.7.2|
|Type|Modul runtime|
