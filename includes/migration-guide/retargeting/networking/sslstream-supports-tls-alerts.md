|   |   |
|---|---|
|Podrobnosti|Po selhání handshake TLS <xref:System.IO.IOException?displayProperty=name> s vnitřní <xref:System.ComponentModel.Win32Exception?displayProperty=name> první operace vstupně-výstupních operací čtení a zápis, bude vyvolána výjimka. <xref:System.ComponentModel.Win32Exception.NativeErrorCode?displayProperty=name> Kód <xref:System.ComponentModel.Win32Exception?displayProperty=name> lze mapovat na protokol TLS výstrah z Vzdálená strana pomocí [kódy chyb zprostředkovatele Schannel pro TLS a SSL výstrahy](https://docs.microsoft.com/windows/desktop/SecAuthN/schannel-error-codes-for-tls-and-ssl-alerts).
