### <a name="workflow-30-types-are-obsolete"></a>Typy WorkFlow 3.0 jsou zastaralé

|   |   |
|---|---|
|Podrobnosti|Rozhraní API modelu Windows Workflow Foundation (WWF) 3.0 (tj. rozhraní z oboru názvů System.Workflow) jsou už zastaralá.|
|Doporučení|Je vhodné místo nich používat nová rozhraní API WWF 4.0 (z oboru názvů System.Activities). Příklad použití nových rozhraní API najdete [tady](~/docs/framework/windows-workflow-foundation/how-to-update-the-definition-of-a-running-workflow-instance.md) a další pokyny jsou k dispozici [tady](https://blogs.msdn.com/b/workflowteam/archive/2012/02/08/deprecatingwf3.aspx). Rozhraní API WWF 3.0 se stále podporují, můžete je tedy případně použít. Upozornění na čas sestavení se vyhnete tím, že ho potlačíte nebo použijete starší kompilátor.|
