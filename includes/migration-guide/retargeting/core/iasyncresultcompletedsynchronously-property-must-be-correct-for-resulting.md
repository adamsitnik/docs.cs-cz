### <a name="iasyncresultcompletedsynchronously-property-must-be-correct-for-the-resulting-task-to-complete"></a>Vlastnost IAsyncResult.CompletedSynchronously musí být správná pro výsledné dokončení úlohy

|   |   |
|---|---|
|Podrobnosti|Při volání metody TaskFactory.FromAsync, provádění <xref:System.IAsyncResult.CompletedSynchronously> musí být správná pro výsledné dokončení úlohy. To znamená vlastnost musí vrátit hodnotu true, pokud a pouze tehdy, pokud byla implementace dokončena synchronně. Dříve nebyla tato vlastnost zaškrtnuta.|
|Doporučení|Pokud <xref:System.IAsyncResult?displayProperty=name> implementace správně vrátí hodnotu true pro <xref:System.IAsyncResult.CompletedSynchronously?displayProperty=name> vlastnost jenom v případě, že úloha dokončena synchronně, pak bude dodržovat žádné přerušení. Uživatelé by měli zkontrolovat <xref:System.IAsyncResult?displayProperty=name> implementace vlastní (pokud existuje) k zajištění, že jsou správně vyhodnotit, jestli úloha dokončena synchronně nebo ne.|
