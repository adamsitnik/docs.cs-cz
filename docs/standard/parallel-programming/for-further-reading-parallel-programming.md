# <a name="for-further-reading-parallel-programming"></a>Pro další čtení (paralelní programování)

Následující zdroje obsahují další informace o paralelním programování v rozhraní .NET:

- [Vzory paralelního programování: Porozumění a použití paralelních vzorů s dokumentem .NET Framework 4](https://www.microsoft.com/download/details.aspx?id=19222) popisuje běžné paralelní vzory a osvědčené postupy pro vývoj paralelních komponent s využitím těchto vzorů.

- [Paralelní programování pomocí blogu .NET](https://devblogs.microsoft.com/pfxteam/) obsahuje mnoho podrobných článků o paralelním programování v .NET.

- [Ukázky pro paralelní programování s .NET Framework](https://code.msdn.microsoft.com/ParExtSamples) stránkou obsahují mnoho ukázek, které ukazují mezilehlé a pokročilé techniky paralelního programování.
