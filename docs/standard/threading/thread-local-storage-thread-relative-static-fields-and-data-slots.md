 <span data-ttu-id="d4fee-125">Statická pole v závislosti na vláknech poskytují lepší výkon než datové sloty a mají výhodu kontroly typu při kompilaci.</span><span class="sxs-lookup"><span data-stu-id="d4fee-125">Thread-relative static fields provide better performance than data slots and have the benefit of compile-time type checking.</span></span>
