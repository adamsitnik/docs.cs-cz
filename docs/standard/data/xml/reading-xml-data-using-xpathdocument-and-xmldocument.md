 <span data-ttu-id="aae58-106"><xref:System.Xml.XPath.XPathDocument> Třída poskytuje rychlé, jen pro čtení, v paměti reprezentace dokumentu XML pomocí modelu dat XPath.</span><span class="sxs-lookup"><span data-stu-id="aae58-106">The <xref:System.Xml.XPath.XPathDocument> class provides a fast, read-only, in-memory representation of an XML document using the XPath data model.</span></span> <span data-ttu-id="aae58-107">Instance <xref:System.Xml.XPath.XPathDocument> třídy jsou vytvořeny pomocí jedné z jejích šesti konstruktorů.</span><span class="sxs-lookup"><span data-stu-id="aae58-107">Instances of the <xref:System.Xml.XPath.XPathDocument> class are created using one of its six constructors.</span></span> <span data-ttu-id="aae58-108">Tyto konstruktory umožňují čtení dokumentu XML pomocí <xref:System.IO.Stream>, <xref:System.IO.TextReader>, nebo <xref:System.Xml.XmlReader> objektu, jakož i `string` cestu k souboru XML.</span><span class="sxs-lookup"><span data-stu-id="aae58-108">These constructors allow you to read an XML document using a <xref:System.IO.Stream>, <xref:System.IO.TextReader>, or <xref:System.Xml.XmlReader> object, as well as the `string` path to an XML file.</span></span>  
  
 <span data-ttu-id="aae58-109">Následující příklad ukazuje použití <xref:System.Xml.XPath.XPathDocument> třídy `string` konstruktor ke čtení dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="aae58-109">The following example illustrates using the <xref:System.Xml.XPath.XPathDocument> class's `string` constructor to read an XML document.</span></span>

  
```csharp  
XPathDocument document = new XPathDocument("books.xml");  
```

 <span data-ttu-id="aae58-111"><xref:System.Xml.XmlDocument> Třídy je upravitelné reprezentací implementace základní úrovně 1 W3C Document Object Model (DOM) a základní modelu DOM úrovně 2 dokumentu XML v paměti.</span><span class="sxs-lookup"><span data-stu-id="aae58-111">The <xref:System.Xml.XmlDocument> class is an editable in-memory representation of an XML document implementing W3C Document Object Model (DOM) Level 1 Core and Core DOM Level 2.</span></span> <span data-ttu-id="aae58-112">Instance <xref:System.Xml.XmlDocument> třídy jsou vytvořeny pomocí jedné z jeho tři konstruktory.</span><span class="sxs-lookup"><span data-stu-id="aae58-112">Instances of the <xref:System.Xml.XmlDocument> class are created using one of its three constructors.</span></span> <span data-ttu-id="aae58-113">Můžete vytvořit nový, prázdný <xref:System.Xml.XmlDocument> objektu voláním <xref:System.Xml.XmlDocument> třída konstruktor bez parametrů.</span><span class="sxs-lookup"><span data-stu-id="aae58-113">You can create a new, empty <xref:System.Xml.XmlDocument> object by calling the <xref:System.Xml.XmlDocument> class constructor with no parameters.</span></span> <span data-ttu-id="aae58-114">Po volání konstruktoru, použijte <xref:System.Xml.XmlDocument.Load%2A> metodu pro načtení dat XML do nového <xref:System.Xml.XmlDocument> objektu z <xref:System.IO.Stream>, <xref:System.IO.TextReader>, nebo <xref:System.Xml.XmlReader> objektu, stejně jako `string` cestu k souboru XML.</span><span class="sxs-lookup"><span data-stu-id="aae58-114">After calling the constructor, use the <xref:System.Xml.XmlDocument.Load%2A> method to load XML data into the new <xref:System.Xml.XmlDocument> object from a <xref:System.IO.Stream>, <xref:System.IO.TextReader>, or <xref:System.Xml.XmlReader> object, as well as the `string` path to an XML file.</span></span>  
  
 <span data-ttu-id="aae58-115">Následující příklad ukazuje použití <xref:System.Xml.XmlDocument> třída konstruktor bez parametrů a <xref:System.Xml.XmlDocument.Load%2A> metodu za účelem čtení dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="aae58-115">The following example illustrates using the <xref:System.Xml.XmlDocument> class constructor with no parameters and the <xref:System.Xml.XmlDocument.Load%2A> method to read an XML document.</span></span>
