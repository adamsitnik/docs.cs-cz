# <a name="xml-schema-object-model-som"></a>Model objektu schématu (SOM) XML
Schéma XML je výkonná a komplexní nástroj pro vytváření a ověřování struktury v kompatibilní dokumentů XML. Podobné k modelování dat v relační databázi, schéma poskytuje způsob, jak definovat strukturu dokumentů XML tak, že zadáte prvky, které lze použít v dokumentech, stejně jako struktury a typy, které tyto prvky musí dodržovat, aby byla platná pro tha t určité schéma.  
  
 Model objektu schématu (SOM) poskytuje sadu tříd v <xref:System.Xml.Schema?displayProperty=nameWithType> obor názvů, který umožní číst schéma ze souboru nebo prostřednictvím kódu programu vytvořit schéma v paměti. Schéma lze potom procházet, úpravy, zkompilován, ověřit nebo zapisují do souboru.
