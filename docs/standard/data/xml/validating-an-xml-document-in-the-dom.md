 <span data-ttu-id="e3b74-107">Po úspěšném ověření se použijí výchozí hodnoty schématu, textové hodnoty se převedou na Atomický hodnoty podle potřeby a informace o typu je přidružený k ověřených informačních položek.</span><span class="sxs-lookup"><span data-stu-id="e3b74-107">After successful validation, schema defaults are applied, text values are converted to atomic values as necessary, and type information is associated with validated information items.</span></span> <span data-ttu-id="e3b74-108">Typy dat XML v důsledku toho nahrazuje dříve netypový kód XML data.</span><span class="sxs-lookup"><span data-stu-id="e3b74-108">As a result, typed XML data replaces previously untyped XML data.</span></span>
