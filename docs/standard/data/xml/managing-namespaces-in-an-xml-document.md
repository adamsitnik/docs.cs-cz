# <a name="managing-namespaces-in-an-xml-document"></a><span data-ttu-id="99695-102">Správa oborů názvů v dokumentu XML</span><span class="sxs-lookup"><span data-stu-id="99695-102">Managing Namespaces in an XML Document</span></span>

## <a name="declaring-namespaces"></a><span data-ttu-id="99695-106">Deklarace oborů názvů</span><span class="sxs-lookup"><span data-stu-id="99695-106">Declaring namespaces</span></span>  
 <span data-ttu-id="99695-107">K deklarování oboru názvů pro element, použijete `xmlns:` atribut:</span><span class="sxs-lookup"><span data-stu-id="99695-107">To declare a namespace on an element, you use the `xmlns:` attribute:</span></span>  
  
 `xmlns:<name>=<"uri">`  
  
 <span data-ttu-id="99695-108">kde `<name>` je Předpona oboru názvů a `<"uri">` je identifikátor URI pro určení oboru názvů.</span><span class="sxs-lookup"><span data-stu-id="99695-108">where `<name>` is the namespace prefix and `<"uri">` is the URI that identifies the namespace.</span></span> <span data-ttu-id="99695-109">Po deklaraci předpona, která vám pomůže ho kvalifikovat prvkům a atributům v dokumentu XML a přidružit je k oboru názvů identifikátoru URI.</span><span class="sxs-lookup"><span data-stu-id="99695-109">After you declare the prefix, you can use it to qualify elements and attributes in an XML document and associate them with the namespace URI.</span></span> <span data-ttu-id="99695-110">Předpona oboru názvů, protože se používají v celém dokumentu by mělo být krátký délku.</span><span class="sxs-lookup"><span data-stu-id="99695-110">Because the namespace prefix is used throughout a document, it should be short in length.</span></span>  
  
 <span data-ttu-id="99695-111">Tento příklad definuje dvě `BOOK` elementy.</span><span class="sxs-lookup"><span data-stu-id="99695-111">This example defines two `BOOK` elements.</span></span> <span data-ttu-id="99695-112">První prvek je kvalifikována předponu, `mybook`, a druhý prvek je kvalifikována předponu, `bb`.</span><span class="sxs-lookup"><span data-stu-id="99695-112">The first element is qualified by the prefix, `mybook`, and the second element is qualified by the prefix, `bb`.</span></span> <span data-ttu-id="99695-113">Každou předponu je přidružen jiný obor názvů identifikátoru URI:</span><span class="sxs-lookup"><span data-stu-id="99695-113">Each prefix is associated with a different namespace URI:</span></span>  
  
```xml  
<mybook:BOOK xmlns:mybook="http://www.contoso.com/books.dtd">  
<bb:BOOK xmlns:bb="urn:blueyonderairlines">  
```  
  
 <span data-ttu-id="99695-114">Chcete-li označit, že element je součástí konkrétní obor názvů, přidejte do ní Předpona oboru názvů.</span><span class="sxs-lookup"><span data-stu-id="99695-114">To signify that an element is a part of a particular namespace, add the namespace prefix to it.</span></span> <span data-ttu-id="99695-115">Například pokud `Author` element patří do `mybook` obor názvů, je deklarována jako `<mybook:Author>`.</span><span class="sxs-lookup"><span data-stu-id="99695-115">For example, if a `Author` element belongs to the `mybook` namespace, it is declared as `<mybook:Author>`.</span></span>

## <a name="declaration-scope"></a><span data-ttu-id="99695-116">Deklarace oboru</span><span class="sxs-lookup"><span data-stu-id="99695-116">Declaration scope</span></span>  
 <span data-ttu-id="99695-117">Obor názvů je účinné z místa deklarace až do konce element byl deklarován v.</span><span class="sxs-lookup"><span data-stu-id="99695-117">A namespace is effective from its point of declaration until the end of the element it was declared in.</span></span> <span data-ttu-id="99695-118">V tomto příkladu obor názvů definovaný v `BOOK` element neplatí pro prvky mimo `BOOK` elementu, například `Publisher` element:</span><span class="sxs-lookup"><span data-stu-id="99695-118">In this example, the namespace defined in the `BOOK` element doesn't apply to elements outside the `BOOK` element, such as the `Publisher` element:</span></span>  
  
```xml  
<Author>Joe Smith</Author>  
<BOOK xmlns:book="http://www.contoso.com">  
    <title>My Wonderful Day</title>  
      <price>$3.95</price>  
</BOOK>  
<Publisher>  
    <Name>MSPress</Name>  
</Publisher>  
```  
  
 <span data-ttu-id="99695-119">Obor názvů musí být deklarována, než je možné, ale nemusí se zobrazí v horní části dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="99695-119">A namespace must be declared before it can be used, but it doesn't have to appear at the top of the XML document.</span></span>  
  
 <span data-ttu-id="99695-120">Při použití více oborů názvů v dokumentu XML, můžete definovat jeden obor názvů jako výchozí obor názvů a vytvoříte čisticí vypadající textový dokument.</span><span class="sxs-lookup"><span data-stu-id="99695-120">When you use multiple namespaces in an XML document, you can define one namespace as the default namespace to create a cleaner looking document.</span></span> <span data-ttu-id="99695-121">Výchozí obor názvů je deklarován v kořenovém prvku a platí pro všechny nekvalifikované elementy v dokumentu.</span><span class="sxs-lookup"><span data-stu-id="99695-121">The default namespace is declared in the root element and applies to all unqualified elements in the document.</span></span> <span data-ttu-id="99695-122">Výchozí obory názvů budou použity pouze prvky, nikoli atributů.</span><span class="sxs-lookup"><span data-stu-id="99695-122">Default namespaces apply to elements only, not to attributes.</span></span>  
  
 <span data-ttu-id="99695-123">Pokud chcete použít výchozí obor názvů, vynechte předponu a identit od deklarace na element:</span><span class="sxs-lookup"><span data-stu-id="99695-123">To use the default namespace, omit the prefix and the colon from the declaration on the element:</span></span>  
  
```xml  
<BOOK xmlns="http://www.contoso.com/books.dtd">  
```
