# <a name="regular-expression-examples"></a>Příklady regulárních výrazů
Tato část obsahuje příklady kódu, které ilustrují použití regulárních výrazů v běžných aplikacích.  
  
> [!NOTE]
> Obor názvů <xref:System.Web.RegularExpressions> obsahuje počet objektů regulárních výrazů, které implementují předdefinované vzory regulárních výrazů pro analýzu řetězců z HTML, XML a dokumentů ASP.NET. Například třída <xref:System.Web.RegularExpressions.TagRegex> identifikuje počáteční značky v řetězci a třída <xref:System.Web.RegularExpressions.CommentRegex> identifikuje ASP.NET komentáře v řetězci.
