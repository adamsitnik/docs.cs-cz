# <a name="manipulating-strings-in-net"></a>Manipulace s řetězci v .NET
Rozhraní .NET poskytuje rozsáhlou sadu rutin, které umožňují efektivně vytvářet, porovnávat a upravovat řetězce a rychle analyzovat velké objemy textu a dat, aby bylo možné vyhledávat, odebírat a nahrazovat textové vzory.

 Obsahuje podrobné informace o regulárních výrazech .NET, včetně prvků jazyka, chování regulárních výrazů a příkladů.
