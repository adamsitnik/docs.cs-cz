# <a name="security-in-net"></a>Zabezpečení v .NET
Modul common language runtime a .NET poskytují mnoho užitečných třídách a službách, které umožňují vývojářům jednoduše psát bezpečný kód a systém správcům umožní přizpůsobit oprávnění udělená kódu tak, aby měl přístup k chráněným prostředkům. Kromě toho modul runtime a .NET poskytují užitečné třídách a službách, které usnadňují použití šifrování a zabezpečení na základě rolí.
