# <a name="compiler-error-cs0152"></a>Compiler Error CS0152
Popisek popisek se již nachází v tomto příkazu switch  
  
 Popisek byla opakována v [přepnout](../../csharp/language-reference/keywords/switch.md) příkazu. Další informace najdete v tématu [přepnout](../../csharp/language-reference/keywords/switch.md).  
  
 Následující ukázka generuje CS0152:  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
