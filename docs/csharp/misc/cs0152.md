<span data-ttu-id="401a1-103">Popisek jmenovky se už v tomto příkazu switch vyskytuje.</span><span class="sxs-lookup"><span data-stu-id="401a1-103">The label 'label' already occurs in this switch statement</span></span>  
  
 <span data-ttu-id="401a1-104">Popisek se opakoval v příkazu [Switch](../language-reference/keywords/switch.md) .</span><span class="sxs-lookup"><span data-stu-id="401a1-104">A label was repeated in a [switch](../language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="401a1-105">Další informace naleznete v tématu [Switch](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="401a1-105">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="401a1-106">Následující ukázka generuje CS0152:</span><span class="sxs-lookup"><span data-stu-id="401a1-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
