 <span data-ttu-id="65f49-104">Deklarace třídy se pokusila o přepsání jiné než metody v základní třídě.</span><span class="sxs-lookup"><span data-stu-id="65f49-104">A class declaration attempted to override a non-method in a base class.</span></span> <span data-ttu-id="65f49-105">Přepsání se musí shodovat s typem člena.</span><span class="sxs-lookup"><span data-stu-id="65f49-105">Overrides must match the member type.</span></span> <span data-ttu-id="65f49-106">Pokud je požadována metoda se stejným názvem jako metoda v základní třídě, použijte [novou](../language-reference/keywords/new-modifier.md) (a nikoli [přepis](../language-reference/keywords/override.md)) v deklaraci metody v základní třídě.</span><span class="sxs-lookup"><span data-stu-id="65f49-106">If a method with the same name as a method in a base class is desired, use [new](../language-reference/keywords/new-modifier.md) (and not [override](../language-reference/keywords/override.md)) on the method declaration in the base class.</span></span>  
  
 <span data-ttu-id="65f49-107">Následující ukázka generuje CS0505:</span><span class="sxs-lookup"><span data-stu-id="65f49-107">The following sample generates CS0505:</span></span>  
  
```csharp  
// CS0505.cs  
// compile with: /target:library  
public class clx  
{  
   public int i;  
}  
  
public class cly : clx  
{  
   public override int i() { return 0; }   // CS0505  
}  
```
