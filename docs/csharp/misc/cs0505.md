# <a name="compiler-error-cs0505"></a>CS0505 chyby kompilátoru
'member1': nelze přepsat, protože 'členu member2' není funkce  
  
 Deklarace třídy se pokusil přepsat bez metodu v základní třídě. Přepsání musí shodovat s typem člena. Používáte-li metodu se stejným názvem jako metodu v základní třídě je žádoucí, [nové](../../csharp/language-reference/keywords/new-modifier.md) (a ne [přepsat](../../csharp/language-reference/keywords/override.md)) v deklaraci metody základní třídy.  
  
 Následující ukázka generuje CS0505:  
  
```csharp  
// CS0505.cs  
// compile with: /target:library  
public class clx  
{  
   public int i;  
}  
  
public class cly : clx  
{  
   public override int i() { return 0; }   // CS0505  
}  
```
