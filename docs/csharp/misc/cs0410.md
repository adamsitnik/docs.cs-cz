# <a name="compiler-error-cs0410"></a>Compiler Error CS0410
Žádné přetížení pro 'metody' nemá správný parametr a návratové typy  
  
 K této chybě dochází, pokud se pokusíte vytvořit instanci delegáta se funkce, která má nesprávný parametr typy. Typy parametrů delegáta musí odpovídat funkci, která přiřazujete delegátu.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0410:  
  
```csharp  
// CS0410.cs  
// compile with: /langversion:ISO-1  
  
class Test  
{  
    delegate void D(double d );  
    static void F(int i) { }  
  
    static void Main()  
    {  
        D d = new D(F);  // CS0410  
    }  
}  
```
