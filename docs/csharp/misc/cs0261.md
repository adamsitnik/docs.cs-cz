# <a name="compiler-error-cs0261"></a>CS0261 chyby kompilátoru
Částečné deklarace 'type' musí být všechny třídy, jen struktury nebo jen rozhraní  
  
 K této chybě dochází, pokud částečný typ je deklarován jako jiný typ konstruktoru na různých místech. Další informace najdete v tématu [částečné třídy a metody](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
 Následující ukázka generuje CS0261:  
  
```csharp  
// CS0261.cs  
partial class A  // CS0261 – A declared as a class here, but as a struct below  
{  
}  
  
partial struct A  
{  
}  
```
