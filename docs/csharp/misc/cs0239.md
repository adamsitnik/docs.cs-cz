# <a name="compiler-error-cs0239"></a>Compiler Error CS0239
'member': nejde přepsat zděděný člen 'zděděného členu', protože je zapečetěný.  
  
 Člen nejde [přepsat](../../csharp/language-reference/keywords/override.md) [zapečetěné](../../csharp/language-reference/keywords/sealed.md) zděděný člen. Další informace najdete v tématu [zaškrtnuto a nezaškrtnuto](../../csharp/language-reference/keywords/checked-and-unchecked.md).  
  
 Následující ukázka generuje CS0239:  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
