# <a name="compiler-error-cs0617"></a><span data-ttu-id="6afef-102">Chyba kompilátoru CS0617</span><span class="sxs-lookup"><span data-stu-id="6afef-102">Compiler Error CS0617</span></span>
<span data-ttu-id="6afef-103">odkaz není platným argumentem pojmenovaného atributu, protože se nejedná o platný typ parametru atributu.</span><span class="sxs-lookup"><span data-stu-id="6afef-103">'reference' is not a valid named attribute argument because it is not a valid attribute parameter type</span></span>  
  
 <span data-ttu-id="6afef-104">Byl proveden pokus o přístup k [soukromému](../language-reference/keywords/private.md) členu třídy atributů.</span><span class="sxs-lookup"><span data-stu-id="6afef-104">An attempt was made to access a [private](../language-reference/keywords/private.md) member of an attribute class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6afef-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="6afef-105">Example</span></span>  
 <span data-ttu-id="6afef-106">Následující ukázka generuje CS0617.</span><span class="sxs-lookup"><span data-stu-id="6afef-106">The following sample generates CS0617.</span></span>  
  
```csharp  
// CS0617.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Struct |   
                AttributeTargets.Class |  
                AttributeTargets.Interface)]  
public class MyClass : Attribute  
{  
   public int Name;  
  
   public MyClass (int sName)  
   {  
      Name = sName;  
      Bad = -1;  
      Bad2 = -1;  
   }  
  
   public readonly int Bad;  
   public int Bad2;  
}  
  
[MyClass(5, Bad=0)] class Class1 {}   // CS0617  
[MyClass(5, Bad2=0)] class Class2 {}  
```
