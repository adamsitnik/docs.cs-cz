# <a name="compiler-error-cs0568"></a><span data-ttu-id="d737b-102">CS0568 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="d737b-102">Compiler Error CS0568</span></span>
<span data-ttu-id="d737b-103">Struktury nemůžou obsahovat explicitní konstruktory bez parametrů.</span><span class="sxs-lookup"><span data-stu-id="d737b-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="d737b-104">Každý [struktura](../../csharp/language-reference/keywords/struct.md) již obsahujícího bezparametrový konstruktor, který inicializuje objekt na nulu.</span><span class="sxs-lookup"><span data-stu-id="d737b-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="d737b-105">Proto konstruktorů, které můžete vytvořit pro strukturu musí provést jeden nebo více parametrů.</span><span class="sxs-lookup"><span data-stu-id="d737b-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="d737b-106">Následující ukázka generuje CS0568:</span><span class="sxs-lookup"><span data-stu-id="d737b-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
