# <a name="compiler-error-cs1035"></a>Compiler Error CS1035
End souboru najde, ' * /' byl očekáván  
  
 Počáteční oddělovač komentáře se neshodovala s koncového oddělovače.  
  
 Následující ukázka generuje CS1035:  
  
```csharp  
// CS1035.cs  
public class a  
{  
   public static void Main()  
   {  
   }  
}  
/*   // CS1035, needs closing comment  
```
