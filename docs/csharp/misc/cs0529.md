# <a name="compiler-error-cs0529"></a>Compiler Error CS0529
Zděděné rozhraní '– rozhraní 1' způsobuje cyklus v hierarchii rozhraní 'rozhraní interface2.  
  
 Seznam dědičnosti pro [rozhraní](../../csharp/language-reference/keywords/interface.md) obsahuje přímý nebo nepřímý odkaz na sebe sama. Rozhraní nemůže dědit samo od sebe.  
  
 Následující ukázka generuje CS0529:  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
