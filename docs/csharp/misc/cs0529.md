# <a name="compiler-error-cs0529"></a><span data-ttu-id="84546-102">Chyba kompilátoru CS0529</span><span class="sxs-lookup"><span data-stu-id="84546-102">Compiler Error CS0529</span></span>
<span data-ttu-id="84546-103">Zděděné rozhraní ' rozhraní interface1 ' způsobuje cyklus v hierarchii rozhraní ' interface2 '.</span><span class="sxs-lookup"><span data-stu-id="84546-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="84546-104">Seznam dědičnosti pro [rozhraní](../language-reference/keywords/interface.md) zahrnuje přímý nebo nepřímý odkaz na sebe sama.</span><span class="sxs-lookup"><span data-stu-id="84546-104">The inheritance list for an [interface](../language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="84546-105">Rozhraní nemůže dědit od sebe sama.</span><span class="sxs-lookup"><span data-stu-id="84546-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="84546-106">Následující ukázka generuje CS0529:</span><span class="sxs-lookup"><span data-stu-id="84546-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
