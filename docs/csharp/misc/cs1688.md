# <a name="compiler-error-cs1688"></a>Compiler Error CS1688
Nelze převést blok anonymní metody bez seznamu parametrů na delegáta typu 'delegáta', protože má jednu nebo více výstupní parametry  
  
 Kompilátor umožňuje parametry na blok anonymní metody ve většině případů vynechat. Tato chyba nastane, když blok anonymní metody nemá seznam parametrů, ale má delegáta `out` parametru. Kompilátor této situaci nepovoluje, protože by bylo potřeba ignorovat přítomnost `out` parametr, který je pravděpodobné, aby správné chování.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje chybu CS1688.  
  
```csharp  
// CS1688.cs  
using System;  
delegate void OutParam(out int i);  
class ErrorCS1676  
{  
    static void Main()   
    {  
        OutParam o;  
        o = delegate  // CS1688  
        // Try this instead:  
        // o = delegate(out int i)  
        {   
            Console.WriteLine("");  
        };  
    }  
}  
```
