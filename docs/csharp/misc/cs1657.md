# <a name="compiler-error-cs1657"></a>Compiler Error CS1657
Nelze předat "parametr" ref nebo out argument protože "důvod.  
  
 Tato chyba nastane, pokud je proměnná je předána jako [ref](../../csharp/language-reference/keywords/ref.md) nebo [si](../../csharp/language-reference/keywords/out-parameter-modifier.md) argument v kontextu, ve kterém je tato proměnná jen pro čtení. Kontexty jen pro čtení zahrnují [foreach](../../csharp/language-reference/keywords/foreach-in.md) iterační proměnné [pomocí](../../csharp/language-reference/keywords/using-statement.md) proměnné, a `fixed` proměnné. Chcete-li vyřešit tuto chybu, nevolejte funkcí, které přijímají `foreach`, `using` nebo `fixed` proměnnou `ref` nebo `out` parametr v `using` bloky, `foreach` příkazy, a `fixed` příkazy.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1657:  
  
```csharp  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```  
  
## <a name="example"></a>Příklad  
 Následující kód ukazuje stejný problém v `fixed` – příkaz:  
  
```csharp  
// CS1657b.cs  
// compile with: /unsafe  
unsafe class C  
{  
    static void F(ref int* p)  
    {  
    }  
  
    static void Main()  
    {  
        int[] a = new int[5];  
        fixed(int* p = a) F(ref p); // CS1657  
    }  
}  
```
