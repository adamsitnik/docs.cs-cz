# <a name="compiler-error-cs0526"></a>Compiler Error CS0526
Rozhraní nemůžou obsahovat konstruktory  
  
 Nejde definovat konstruktory pro [rozhraní](../../csharp/language-reference/keywords/interface.md). Metoda je považován za konstruktor, pokud má stejný název jako třída a bez návratového typu.  
  
 Následující ukázka generuje CS0526:  
  
```csharp  
// CS0526.cs  
namespace x  
{  
   public interface clx  
   {  
      public clx()   // CS0526  
      {  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
