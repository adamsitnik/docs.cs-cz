# <a name="compiler-error-cs1527"></a><span data-ttu-id="99633-102">Chyba kompilátoru CS1527</span><span class="sxs-lookup"><span data-stu-id="99633-102">Compiler Error CS1527</span></span>

<span data-ttu-id="99633-103">Elementy definované v oboru názvů nejde explicitně deklarovat jako Private, Protected, Protected Internal nebo Private Protected.</span><span class="sxs-lookup"><span data-stu-id="99633-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span>
  
 <span data-ttu-id="99633-104">Deklarace typů v oboru názvů můžou mít buď [veřejný](../../csharp/language-reference/keywords/public.md) , nebo [interní](../../csharp/language-reference/keywords/internal.md) přístup.</span><span class="sxs-lookup"><span data-stu-id="99633-104">Type declarations in a namespace can have either [public](../../csharp/language-reference/keywords/public.md) or [internal](../../csharp/language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="99633-105">Pokud není zadaná žádná přístupnost, výchozí hodnota je **interní** .</span><span class="sxs-lookup"><span data-stu-id="99633-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="99633-106">Následující ukázka generuje CS1527:</span><span class="sxs-lookup"><span data-stu-id="99633-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="99633-107">Následující příklad generuje CS1527, protože pokud není v kódu programu explicitně deklarován žádný obor názvů, jsou všechny deklarace typů umístěny implicitně v rámci globálního oboru názvů.</span><span class="sxs-lookup"><span data-stu-id="99633-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
