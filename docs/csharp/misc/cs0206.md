# <a name="compiler-error-cs0206"></a>CS0206 chyby kompilátoru
Vlastnost nebo indexer nelze předat jako vnější nebo odkazovaný parametr  
  
 A [vlastnost](../../csharp/programming-guide/classes-and-structs/properties.md) není k dispozici mají být předány jako [ref](../../csharp/language-reference/keywords/ref.md) nebo [si](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametru. Další informace najdete v tématu [předání parametrů](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0206:  
  
```csharp  
// CS0206.cs  
public class MyClass  
{  
    public static int P  
    {  
        get  
        {  
            return 0;  
        }  
        set  
        {  
        }  
    }  
  
    public static void MyMeth(ref int i)  
    // public static void MyMeth(int i)  
    {  
    }  
  
    public static void Main()  
    {  
        MyMeth(ref P);   // CS0206  
        // try the following line instead  
        // MyMeth(P);   // CS0206  
    }  
}  
```
