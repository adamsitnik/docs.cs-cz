# <a name="compiler-error-cs2012"></a>CS2012 chyby kompilátoru
Nelze otevřít soubor pro zápis.  
  
 Při použití [/bugreport:](../../csharp/language-reference/compiler-options/bugreport-compiler-option.md) `file` – možnost kompilátoru, tento soubor nelze otevřít pro zápis. Ujistěte se, že jste zadali platný název souboru a že soubor není jen pro čtení.
