# <a name="compiler-error-cs2012"></a>Chyba kompilátoru CS2012
Soubor ' file ' nelze otevřít pro zápis.  
  
 Při použití možnosti kompilátoru [/bugreport:](../language-reference/compiler-options/bugreport-compiler-option.md) `file` nelze soubor otevřít pro zápis. Ujistěte se, že jste zadali platný název souboru a že soubor není jen pro čtení.
