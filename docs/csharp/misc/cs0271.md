<span data-ttu-id="d2227-103">Vlastnost nebo indexovací člen 'vlastnost nebo indexer' nelze v tomto kontextu použít, protože přistupující objekt get není dostupný.</span><span class="sxs-lookup"><span data-stu-id="d2227-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="d2227-104">K této chybě dochází při pokusu o přístup na nepřístupný `get` přistupujícího objektu.</span><span class="sxs-lookup"><span data-stu-id="d2227-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="d2227-105">Chcete-li tuto chybu vyřešit, zvyšte přístupnost přístupového objektu nebo změňte umístění volání.</span><span class="sxs-lookup"><span data-stu-id="d2227-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="d2227-106">Další informace najdete v tématu [přístupnosti přístupového objektu](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) a [vlastnosti](../../csharp/programming-guide/classes-and-structs/properties.md).</span><span class="sxs-lookup"><span data-stu-id="d2227-106">For more information, see [Accessor Accessibility](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../../csharp/programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="d2227-107">Následující příklad generuje CS0271:</span><span class="sxs-lookup"><span data-stu-id="d2227-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
