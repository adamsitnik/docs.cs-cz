# <a name="compiler-error-cs0271"></a>Compiler Error CS0271
Vlastnost nebo indexovací člen 'vlastnost nebo indexer' nelze v tomto kontextu použít, protože přistupující objekt get není dostupný.  
  
 K této chybě dochází při pokusu o přístup na nepřístupný `get` přistupujícího objektu. Chcete-li tuto chybu vyřešit, zvyšte přístupnost přístupového objektu nebo změňte umístění volání. Další informace najdete v tématu [přístupnosti přístupového objektu](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) a [vlastnosti](../../csharp/programming-guide/classes-and-structs/properties.md).  
  
 Následující příklad generuje CS0271:  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
