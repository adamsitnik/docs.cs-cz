# <a name="compiler-error-cs0663"></a><span data-ttu-id="9cb6b-102">Chyba kompilátoru CS0663</span><span class="sxs-lookup"><span data-stu-id="9cb6b-102">Compiler Error CS0663</span></span>
<span data-ttu-id="9cb6b-103">Nelze definovat přetížené metody, které se liší pouze parametrem ref a out.</span><span class="sxs-lookup"><span data-stu-id="9cb6b-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="9cb6b-104">Metody, které se liší pouze při jejich použití [v](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) a [out](../language-reference/keywords/out-parameter-modifier.md) u parametru nejsou povoleny.</span><span class="sxs-lookup"><span data-stu-id="9cb6b-104">Methods that differ only on their use of [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) and [out](../language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="9cb6b-105">Následující ukázka generuje CS0663:</span><span class="sxs-lookup"><span data-stu-id="9cb6b-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
