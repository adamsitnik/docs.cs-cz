# <a name="compiler-error-cs0663"></a><span data-ttu-id="eaa92-102">CS0663 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="eaa92-102">Compiler Error CS0663</span></span>
<span data-ttu-id="eaa92-103">Nemůže definovat přetěžované metody, které se liší jen deklaracemi ref a out.</span><span class="sxs-lookup"><span data-stu-id="eaa92-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="eaa92-104">Metody, které se liší pouze v jejich využívání [v](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) a [si](../../csharp/language-reference/keywords/out-parameter-modifier.md) nejsou povoleny v parametru.</span><span class="sxs-lookup"><span data-stu-id="eaa92-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="eaa92-105">Následující ukázka generuje CS0663:</span><span class="sxs-lookup"><span data-stu-id="eaa92-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
