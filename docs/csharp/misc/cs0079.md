# <a name="compiler-error-cs0079"></a><span data-ttu-id="cb2ab-102">Compiler Error CS0079</span><span class="sxs-lookup"><span data-stu-id="cb2ab-102">Compiler Error CS0079</span></span>
<span data-ttu-id="cb2ab-103">Událost 'událost' se může nalézat pouze na levé straně výrazu += nebo-=</span><span class="sxs-lookup"><span data-stu-id="cb2ab-103">The event 'event' can only appear on the left hand side of += or -=</span></span>  
  
 <span data-ttu-id="cb2ab-104">[Události](../../csharp/language-reference/keywords/event.md) volala se nesprávně.</span><span class="sxs-lookup"><span data-stu-id="cb2ab-104">An [event](../../csharp/language-reference/keywords/event.md) was called incorrectly.</span></span> <span data-ttu-id="cb2ab-105">Další informace najdete v tématu [události](../../csharp/programming-guide/events/index.md) a [delegáti](../../csharp/programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="cb2ab-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="cb2ab-106">Následující ukázka generuje CS0079:</span><span class="sxs-lookup"><span data-stu-id="cb2ab-106">The following sample generates CS0079:</span></span>  
  
```csharp  
// CS0079.cs  
using System;  
  
public delegate void MyEventHandler();  
  
public class Class1  
{  
   private MyEventHandler _e;  
  
   public event MyEventHandler Pow  
   {  
      add  
      {  
         _e += value;  
         Console.WriteLine("in add accessor");  
      }  
      remove  
      {  
         _e -= value;  
         Console.WriteLine("in remove accessor");  
      }  
   }  
  
   public void Handler()  
   {  
   }  
  
   public void Fire()  
   {  
      if (_e != null)  
      {  
         Pow();   // CS0079  
         // try the following line instead  
         // _e();  
      }  
   }  
  
   public static void Main()  
   {  
      Class1 p = new Class1();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p._e -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
      p.Pow -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
   }  
}  
```
