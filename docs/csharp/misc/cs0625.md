# <a name="compiler-error-cs0625"></a><span data-ttu-id="0bd29-102">Compiler Error CS0625</span><span class="sxs-lookup"><span data-stu-id="0bd29-102">Compiler Error CS0625</span></span>
<span data-ttu-id="0bd29-103">"pole": typy polí instance označené deklarací StructLayout(LayoutKind.Explicit) musí mít atribut FieldOffset.</span><span class="sxs-lookup"><span data-stu-id="0bd29-103">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="0bd29-104">Pokud struktura je označena explicitní **StructLayout** atribut, musí mít všechna pole ve struktuře [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) atribut.</span><span class="sxs-lookup"><span data-stu-id="0bd29-104">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="0bd29-105">Další informace najdete v tématu [StructLayoutAttribute – třída](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span><span class="sxs-lookup"><span data-stu-id="0bd29-105">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span> 

<span data-ttu-id="0bd29-106">Následující ukázka generuje CS0625:</span><span class="sxs-lookup"><span data-stu-id="0bd29-106">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
