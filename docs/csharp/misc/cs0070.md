# <a name="compiler-error-cs0070"></a><span data-ttu-id="485ec-102">Compiler Error CS0070</span><span class="sxs-lookup"><span data-stu-id="485ec-102">Compiler Error CS0070</span></span>
<span data-ttu-id="485ec-103">Událost 'událost' se může nalézat pouze na levé straně výrazu += nebo-= (s výjimkou při použito v rámci typu 'type')</span><span class="sxs-lookup"><span data-stu-id="485ec-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="485ec-104">Mimo třídu je definováno v [události](../../csharp/language-reference/keywords/event.md) pouze přičíst nebo odečíst odkazy.</span><span class="sxs-lookup"><span data-stu-id="485ec-104">Outside of the class it is defined in, an [event](../../csharp/language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="485ec-105">Další informace najdete v tématu [události](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="485ec-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="485ec-106">Následující ukázka generuje CS0070:</span><span class="sxs-lookup"><span data-stu-id="485ec-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
