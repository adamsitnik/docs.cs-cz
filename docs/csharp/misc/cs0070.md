# <a name="compiler-error-cs0070"></a><span data-ttu-id="0dc0a-102">Chyba kompilátoru CS0070</span><span class="sxs-lookup"><span data-stu-id="0dc0a-102">Compiler Error CS0070</span></span>
<span data-ttu-id="0dc0a-103">Událost ' Event ' se může vyskytovat pouze na levé straně operátoru + = nebo-= (kromě případu, kdy se používá v rámci typu ' type ').</span><span class="sxs-lookup"><span data-stu-id="0dc0a-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="0dc0a-104">Mimo třídu, ve které je definována, může [událost](../language-reference/keywords/event.md) přidat nebo odečíst pouze odkazy.</span><span class="sxs-lookup"><span data-stu-id="0dc0a-104">Outside of the class it is defined in, an [event](../language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="0dc0a-105">Další informace najdete v tématu [události](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="0dc0a-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="0dc0a-106">Následující ukázka generuje CS0070:</span><span class="sxs-lookup"><span data-stu-id="0dc0a-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
