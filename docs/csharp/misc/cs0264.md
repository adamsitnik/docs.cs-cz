# <a name="compiler-error-cs0264"></a>CS0264 chyby kompilátoru
Částečné deklarace 'type' musí mít stejné názvy parametrů typů ve stejném pořadí  
  
 K této chybě dochází, pokud jsou definice obecného typu v částečné deklarace a parametry typu nejsou konzistentní vzhledem k aplikacím v názvu nebo pořadí v rámci všech částečné deklarace. K odstranění této chyby, zkontrolujte parametry typu k deklaraci částečné a ujistěte se, že se používá stejný název a pořadí parametrů. Další informace najdete v tématu [částečné třídy a metody](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).a [parametry obecného typu](../../csharp/programming-guide/generics/generic-type-parameters.md).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0264.  
  
```csharp  
// CS0264.cs  
  
partial class MyClass<T>  // CS0264  
{  
}  
  
partial class MyClass <MyType>  
{  
}  
```
