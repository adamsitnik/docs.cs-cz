# <a name="compiler-error-cs0655"></a>CS0655 chyby kompilátoru
"parametr" není platný pojmenovaný argument atributu protože se nejedná platný typ parametru atributu  
  
 Zobrazit [atributy](../../csharp/programming-guide/concepts/attributes/index.md) diskuzi o typy platná hodnota parametru atributu.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0655:  
  
```csharp  
// CS0655.cs  
using System;  
  
class MyAttribute : Attribute  
{  
    // decimal is not valid attribute parameter type  
    public decimal d = 0;  
    public int e = 0;  
}  
  
[My(d = 0)]   // CS0655  
// Try the following line instead:  
// [My(e = 0)]  
class C  
{  
    public static void Main()  
    {  
    }  
}  
```
