# <a name="compiler-warning-level-1-cs0184"></a>Kompilátor CS0184 upozornění (úroveň 1)
Tento výraz je nikdy typu zadaný ('type')  
  
 Výraz nemůže být nikdy **true** vzhledem k tomu, že je proměnná testujete ani deklarována jako ***typ*** ani odvozený od ***typ***.  
  
 Následující ukázka generuje CS0184:  
  
```csharp  
// CS0184.cs  
// compile with: /W:1  
class MyClass  
{  
   public static void Main()  
   {  
      int i = 0;  
      if (i is string)   // CS0184  
         i++;  
   }  
}  
```
