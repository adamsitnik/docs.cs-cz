 <span data-ttu-id="1a10a-104">Pokus o přepsání jedné z přístupových metod pro vlastnost se nezdařil, protože přistupující objekt nelze přepsat.</span><span class="sxs-lookup"><span data-stu-id="1a10a-104">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="1a10a-105">K této chybě může dojít, pokud:</span><span class="sxs-lookup"><span data-stu-id="1a10a-105">This error can occur if:</span></span>

## <a name="example"></a><span data-ttu-id="1a10a-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="1a10a-110">Example</span></span>  
 <span data-ttu-id="1a10a-111">Následující ukázka vygeneruje CS0546, protože základní třída nedeklaruje přistupující objekt set pro vlastnost.</span><span class="sxs-lookup"><span data-stu-id="1a10a-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
