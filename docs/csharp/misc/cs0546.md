## <a name="example"></a><span data-ttu-id="bbcb5-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="bbcb5-110">Example</span></span>  
 <span data-ttu-id="bbcb5-111">Následující ukázka generuje CS0546, protože základní třída nedeklaruje přístupový objekt set pro vlastnost.</span><span class="sxs-lookup"><span data-stu-id="bbcb5-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
