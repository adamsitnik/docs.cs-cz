# <a name="compiler-error-cs1512"></a>CS1512 chyby kompilátoru
Klíčové slovo "základní" není k dispozici v rámci aktuálního kontextu  
  
 [Základní](../../csharp/language-reference/keywords/base.md) – klíčové slovo byl použit mimo metoda, vlastnost nebo konstruktoru.  
  
 Následující příklad generuje CS1512:  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
