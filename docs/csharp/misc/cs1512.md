 <span data-ttu-id="8f05c-104">Klíčové slovo [Base](../language-reference/keywords/base.md) bylo použito mimo metodu, vlastnost nebo konstruktor.</span><span class="sxs-lookup"><span data-stu-id="8f05c-104">The [base](../language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="8f05c-105">Následující příklad generuje CS1512:</span><span class="sxs-lookup"><span data-stu-id="8f05c-105">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
