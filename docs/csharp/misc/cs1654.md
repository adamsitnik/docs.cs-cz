# <a name="compiler-error-cs1654"></a>Compiler Error CS1654
Nelze upravovat členy prvku 'variable', protože je 'jen pro čtení typu proměnné'  
  
 K této chybě dochází při pokusu upravit členy proměnná, která je jen pro čtení, protože je ve speciální konstrukce.  
  
 Je jeden běžné oblasti, která k tomu dochází v rámci [foreach](../../csharp/language-reference/keywords/foreach-in.md) smyčky. Je chyba kompilace ke změně hodnoty prvků kolekce. Proto nemůže provádět žádné změny k elementům, které jsou [typů hodnot](../../csharp/language-reference/keywords/value-types.md), včetně [struktury](../../csharp/programming-guide/classes-and-structs/structs.md). V kolekci, jehož prvky jsou [referenční typy](../../csharp/language-reference/keywords/reference-types.md), můžete upravit dostupné členy jednotlivých prvků, ale bude generovat žádné zkuste přidat nebo odebrat nebo nahradit celé prvky [Chyba kompilátoru CS1656](../../csharp/language-reference/compiler-messages/cs1656.md).  
  
## <a name="example"></a>Příklad  
 Následující příklad vygeneruje chybu CS1654, protože `Book` je `struct`. Chcete-li chybu opravit, změňte `struct` k [třídy](../../csharp/language-reference/keywords/class.md).  
  
```csharp  
using System.Collections.Generic;  
using System.Text;  
  
namespace CS1654  
{  
  
    struct Book  
    {  
        public string Title;  
        public string Author;  
        public double Price;  
        public Book(string t, string a, double p)  
        {  
            Title=t;  
            Author=a;  
            Price=p;  
  
        }  
    }  
  
    class Program  
    {  
        List<Book> list;  
        static void Main(string[] args)  
        {  
             //Use a collection initializer to initialize the list  
            Program prog = new Program();  
            prog.list = new List<Book>();  
            prog.list.Add(new Book ("The C# Programming Language",  
                                    "Hejlsberg, Wiltamuth, Golde",  
                                     29.95));  
            prog.list.Add(new Book ("The C++ Programming Language",  
                                    "Stroustrup",  
                                     29.95));  
            prog.list.Add(new Book ("The C Programming Language",  
                                    "Kernighan, Ritchie",  
                                    29.95));  
            foreach(Book b in prog.list)  
            {  
                //Compile error if Book is a struct  
                //Make Book a class to modify its members  
                b.Price +=9.95; // CS1654  
            }  
  
        }  
    }  
}  
```
