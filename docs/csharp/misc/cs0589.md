# <a name="compiler-error-cs0589"></a>Chyba kompilátoru CS0589

Vnitřní chyba kompilátoru: fáze ' PARSE '

 Zkuste zjistit, jestli se kompilátor nedaří kvůli neschopnosti analyzovat neočekávanou syntaxi. Pokud tomu tak není, zkuste [nás kontaktovat](/visualstudio/ide/feedback-options).
 
> [!NOTE]
> Tato chyba kompilátoru se už v Roslyn nepoužívá.
