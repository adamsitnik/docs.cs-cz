## <a name="example"></a><span data-ttu-id="3ed1e-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="3ed1e-110">Example</span></span>  
 <span data-ttu-id="3ed1e-111">Následující příklad generuje CS1059, protože `i` je konstanta, není proměnná a `E` je `Enum` typ, jehož prvky jsou také konstantní hodnoty.</span><span class="sxs-lookup"><span data-stu-id="3ed1e-111">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
