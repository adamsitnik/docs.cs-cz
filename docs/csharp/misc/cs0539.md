# <a name="compiler-error-cs0539"></a>Compiler Error CS0539
'member' v explicitní deklaraci rozhraní není členem rozhraní  
  
 Byl proveden pokus o explicitně deklarovat [rozhraní](../../csharp/language-reference/keywords/interface.md) člena, který neexistuje. Můžete buď odstranit deklarace nebo ho změnit tak, aby se vztahuje na člen platný rozhraní.  
  
 Následující ukázka generuje CS0539:  
  
```csharp  
// CS0539.cs  
namespace x  
{  
   interface I  
   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
