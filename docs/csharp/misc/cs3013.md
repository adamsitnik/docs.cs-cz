# <a name="compiler-warning-level-1-cs3013"></a>Kompilátor CS3013 upozornění (úroveň 1)
Přidání modulů musí být označené atributem CLSCompliant tak, aby odpovídaly sestavení  
  
 Modul, který byl kompilován s [/target: Module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md) – možnost kompilátoru byl přidán do kompilace s [/addmodule](../../csharp/language-reference/compiler-options/addmodule-compiler-option.md). Dodržování předpisů modulu s specifikace CLS (Common Language) však nesouhlasí se stavem kompatibilní se Specifikací aktuální kompilace.  
  
 Dodržování specifikace CLS je označen atributem module. Například `[module:CLSCompliant(true)]` znamená, že modul je kompatibilní se Specifikací CLS, a `[module:CLSCompliant(false)]` znamená, že modul není kompatibilní se Specifikací CLS. Výchozí hodnota je `[module:CLSCompliant(false)]`. Další informace o specifikaci CLS naleznete v tématu [jazyková nezávislost a jazykově nezávislé komponenty](../../../docs/standard/language-independence-and-language-independent-components.md).
