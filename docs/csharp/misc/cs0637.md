# <a name="compiler-error-cs0637"></a>CS0637 chyby kompilátoru

Atribut FieldOffset není povolený pro pole typu static nebo const.
  
 [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) atributu nelze použít na pole označené parametrem [statické](../../csharp/language-reference/keywords/static.md) nebo [const](../../csharp/language-reference/keywords/const.md).  
  
 Následující ukázka generuje CS0637:  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
