# <a name="compiler-error-cs1024"></a>Compiler Error CS1024
Očekávala se direktiva preprocesoru  
  
 Řádek začalo se symbol křížku (#), ale následující řetězec není platným [direktivy preprocesoru](../../csharp/language-reference/preprocessor-directives/index.md).  
  
 Následující ukázka generuje CS1024:  
  
```csharp  
// CS1024.cs  
#import System   // CS1024  
```
