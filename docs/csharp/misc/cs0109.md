# <a name="compiler-warning-level-4-cs0109"></a>Kompilátor CS0109 upozornění (úroveň 4)
Člena 'member' neskrývá zděděný člen. Klíčové slovo new se nevyžaduje  
  
 Deklarace třídy zahrnuty [nové](../../csharp/language-reference/keywords/new.md) – klíčové slovo, i když deklarace nemůže přepsat existující deklarace v základní třídě. Můžete odstranit **nové** – klíčové slovo.  
  
 Následující ukázka generuje CS0109:  
  
```csharp  
// CS0109.cs  
// compile with: /W:4  
namespace x  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
