# <a name="compiler-error-cs1555"></a>Compiler Error CS1555
Nelze najít 'class' zadaný pro metodu Main  
  
 Byl určen jako třída [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) – možnost kompilátoru, ale název třídy nebyl nalezen ve zdrojovém kódu.
