# <a name="compiler-error-cs0221"></a>Compiler Error CS0221
Konstantní hodnotu 'value' nelze převést na 'type' (použijte syntaxi 'unchecked' přepsání)  
  
 Byla zjištěna operace přiřazení, která bude mít za následek ztrátu dat [zaškrtnutí](../../csharp/language-reference/keywords/checked.md), na které je ve výchozím nastavení. Opravte přiřazení nebo použijte [Nekontrolovaná](../../csharp/language-reference/keywords/unchecked.md) pro vyřešení této chyby. Další informace najdete v tématu [zaškrtnuto a nezaškrtnuto](../../csharp/language-reference/keywords/checked-and-unchecked.md).  
  
 Následující ukázka generuje CS0221:  
  
```csharp  
// CS0221.cs  
public class MyClass  
{  
   public static void Main()  
   {  
      // unchecked  
      // {  
         int a = (int)0xFFFFFFFF;   // CS0221  
         a++;  
      // }  
   }  
}  
```
