# <a name="compiler-warning-level-2-cs0444"></a>Kompilátor CS0444 upozornění (úroveň 2)
Předdefinovaný typ ' název typu 1' nebyl nalezen v ' System – obor názvů 1, ale byl nalezen v ' System – obor názvů 2  
  
 Předdefinované objektu jako <xref:System.Int32> nebyl nalezen, ve kterém kompilátor očekává se našel, ale místo toho zjistila, že v ' System – obor názvů 2.  
  
 Chyba může znamenat, že je správně nainstalované rozhraní .NET Framework. Tento problém můžete znovu nainstalujte rozhraní .NET Framework.  
  
 Pokud vytváříte vlastní knihovny základních tříd, může dojít také k této chybě. V tomto případě chcete chybu vyřešit, sestavte znovu mscorlib.
