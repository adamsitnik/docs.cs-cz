# <a name="compiler-error-cs0719"></a>Compiler Error CS0719
'type': prvky pole nemůžou být statického typu  
  
 Pole statického typu nedává smysl, protože jsou prvky pole instance, ale není možné k vytvoření instance statické typy.  
  
 Následující ukázka generuje CS0719:  
  
```csharp  
// CS0719.cs  
public static class SC  
{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
