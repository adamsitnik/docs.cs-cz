# <a name="compiler-error-cs0243"></a><span data-ttu-id="5a3fd-102">Chyba kompilátoru CS0243</span><span class="sxs-lookup"><span data-stu-id="5a3fd-102">Compiler Error CS0243</span></span>

<span data-ttu-id="5a3fd-103">Atribut Conditional není pro metodu platný, protože je to metoda override.</span><span class="sxs-lookup"><span data-stu-id="5a3fd-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="5a3fd-104">Atribut [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) není povolen pro metodu, která je označena klíčovým slovem [override](../language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="5a3fd-104">The [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="5a3fd-105">Další informace najdete v tématu o tom, [Jak používat klíčová slova override a New](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="5a3fd-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="5a3fd-106">Kompilátor nikdy neváže na metody přepsání; váže se pouze k základní metodě a modul CLR (Common Language Runtime) volá podle potřeby přepsání.</span><span class="sxs-lookup"><span data-stu-id="5a3fd-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="5a3fd-107">Následující ukázka generuje CS0243:</span><span class="sxs-lookup"><span data-stu-id="5a3fd-107">The following sample generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
