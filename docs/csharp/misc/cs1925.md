# <a name="compiler-error-cs1925"></a>Compiler Error CS1925
Objekt typu 'type' pomocí inicializátoru kolekce nelze inicializovat.  
  
 Inicializátory kolekcí jsou povolené jenom pro třídy kolekcí, které splňují určitá kritéria. Další informace najdete v tématu [inicializátory objektu a kolekce](../../csharp/programming-guide/classes-and-structs/object-and-collection-initializers.md). Tato chyba je také vytvořen při pokusu o použití krátkou formu inicializátor pole vnořené uvnitř inicializátoru kolekce.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Inicializace objektu voláním konstruktorů a metod.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje CS1925:  
  
```csharp  
// cs1925.cs  
public class Student  
{  
    public int[] Scores;  
}  
  
class Test  
{  
    static void Main(string[] args)  
    {  
        Student student = new Student { Scores = { 1, 2, 3 } }; // CS1925  
    }  
}  
```
