# <a name="compiler-error-cs0517"></a>Compiler Error CS0517
'class' nemá žádnou základní třídu a nemůže volat konstruktor základní třídy  
  
 CS0517 může dojít pouze v případě, že modul common language runtime rozhraní .NET Framework zkompiluje zdrojový kód pro třídu objektu, který je jediná třída, která nemá žádnou základní třídu.
