# <a name="compiler-error-cs0306"></a>CS0306 chyby kompilátoru
Typ "typ" nelze použít jako argument typu.  
  
 Typ použitý jako parametr typu není povolený. To může být, protože typ není typem ukazatele.  
  
 Následující příklad generuje CS0306:  
  
```csharp  
// CS0306.cs  
class C<T>  
{  
}  
  
class M  
{  
    // CS0306 – int* not allowed as a type parameter  
     C<int*> f;  
}  
```
