<span data-ttu-id="22a08-103">Typ podmíněného výrazu nejde určit, protože 'type1' a 'type2' implicitně převést mezi sebou</span><span class="sxs-lookup"><span data-stu-id="22a08-103">Type of conditional expression cannot be determined because 'type1' and 'type2' implicitly convert to one another</span></span>  
  
 <span data-ttu-id="22a08-104">V podmíněném příkazu, musí být schopen převést typy na obou stranách `:` operátor.</span><span class="sxs-lookup"><span data-stu-id="22a08-104">In a conditional statement, you must be able to convert the types on either side of the `:` operator.</span></span> <span data-ttu-id="22a08-105">Také nemůže být ke vzájemné převodní rutiny; potřebujete jenom jeden převodu.</span><span class="sxs-lookup"><span data-stu-id="22a08-105">Also, there cannot be mutual conversion routines; you only need one conversion.</span></span> <span data-ttu-id="22a08-106">Další informace najdete v tématu [operátory převodu](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="22a08-106">For more information, see [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="22a08-107">Následující ukázka generuje CS0172:</span><span class="sxs-lookup"><span data-stu-id="22a08-107">The following sample generates CS0172:</span></span>  
  
```csharp  
// CS0172.cs  
public class Square  
{  
   public class Circle  
   {  
      public static implicit operator Circle(Square aa)  
      {  
         return null;  
      }  
  
      public static implicit operator Square(Circle aa)  
      // using explicit resolves this error  
      // public static explicit operator Square(Circle aa)  
      {  
         return null;  
      }  
   }  
  
   public static void Main()  
   {  
      Circle aa = new Circle();  
      Square ii = new Square();  
      object o = (1 == 1) ? aa : ii;   // CS0172  
      // the following cast would resolve this error  
      // (1 == 1) ? aa : (Circle)ii;  
   }  
}  
```
