<span data-ttu-id="145c1-103">' konstruktor ': statický konstruktor musí být bez parametrů</span><span class="sxs-lookup"><span data-stu-id="145c1-103">'constructor' : a static constructor must be parameterless</span></span>  
  
 <span data-ttu-id="145c1-104">[Statický](../language-reference/keywords/static.md) konstruktor nelze deklarovat s jedním nebo více parametry.</span><span class="sxs-lookup"><span data-stu-id="145c1-104">A [static](../language-reference/keywords/static.md) constructor cannot be declared with one or more parameters.</span></span> <span data-ttu-id="145c1-105">Další informace naleznete v tématu [konstruktory](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="145c1-105">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="145c1-106">Následující ukázka generuje CS0132:</span><span class="sxs-lookup"><span data-stu-id="145c1-106">The following sample generates CS0132:</span></span>  
  
```csharp  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
