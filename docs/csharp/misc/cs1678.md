# <a name="compiler-error-cs1678"></a><span data-ttu-id="b7b1a-102">CS1678 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="b7b1a-102">Compiler Error CS1678</span></span>
<span data-ttu-id="b7b1a-103">Parametr 'number' je deklarován jako typ 'type1', ale měl by být 'type2'</span><span class="sxs-lookup"><span data-stu-id="b7b1a-103">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="b7b1a-104">Tato chyba nastane, pokud typ parametru v anonymní metoda se liší od deklarace delegáta, které metody, která jsou přetypování.</span><span class="sxs-lookup"><span data-stu-id="b7b1a-104">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="b7b1a-105">Následující ukázka generuje CS1678:</span><span class="sxs-lookup"><span data-stu-id="b7b1a-105">The following sample generates CS1678:</span></span>  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
