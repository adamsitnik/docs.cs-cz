# <a name="compiler-warning-level-1-cs1911"></a><span data-ttu-id="28c17-102">Kompilátor CS1911 upozornění (úroveň 1)</span><span class="sxs-lookup"><span data-stu-id="28c17-102">Compiler Warning (level 1) CS1911</span></span>
<span data-ttu-id="28c17-103">Přístup k členu "name" až "základní" klíčové slovo z anonymní metody, lambda výrazu, výrazu dotazu nebo výsledky iterátor v neověřitelný kód.</span><span class="sxs-lookup"><span data-stu-id="28c17-103">Access to member 'name' through a 'base' keyword from an anonymous method, lambda expression, query expression, or iterator results in unverifiable code.</span></span> <span data-ttu-id="28c17-104">Zvažte přesunutí přístup do pomocnou metodu iterátoru.</span><span class="sxs-lookup"><span data-stu-id="28c17-104">Consider moving the access into a helper method on the containing type.</span></span>  
  
 <span data-ttu-id="28c17-105">Volání virtuální funkce s `base` – klíčové slovo uvnitř těla metody iterátoru nebo anonymní metody způsobí neověřitelný kód.</span><span class="sxs-lookup"><span data-stu-id="28c17-105">Calling virtual functions with the `base` keyword inside the method body of an iterator or anonymous methods will result in unverifiable code.</span></span> <span data-ttu-id="28c17-106">U neověřitelného kódu se nezdaří spustit v prostředí s částečnou důvěryhodností.</span><span class="sxs-lookup"><span data-stu-id="28c17-106">Unverifiable code will fail to run in a partial trust environment.</span></span>  
  
 <span data-ttu-id="28c17-107">Jedním řešením pro CS1911 je přesunout virtuální funkce volání funkce pomocné rutiny.</span><span class="sxs-lookup"><span data-stu-id="28c17-107">One resolution for CS1911 is to move the virtual function call to a helper function.</span></span>  
  
## <a name="example"></a><span data-ttu-id="28c17-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="28c17-108">Example</span></span>  
 <span data-ttu-id="28c17-109">Následující ukázka generuje CS1911.</span><span class="sxs-lookup"><span data-stu-id="28c17-109">The following sample generates CS1911.</span></span>  
  
```csharp  
// CS1911.cs  
// compile with: /W:1  
using System;  
  
delegate void D();  
delegate D RetD();  
  
class B {  
   protected virtual void M() {  
      Console.WriteLine("B.M");  
   }  
}  
  
class Der : B {  
   protected override void M() {  
      Console.WriteLine("D.M");  
   }  
  
   void Test() { base.M(); }  
   D Test2() { return new D(base.M); }  
  
   public D CallBaseM() {  
      return delegate () { base.M(); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { Test(); };  
   }  
  
   public RetD DelToBaseM() {  
      return delegate () { return new D(base.M); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { return Test2(); };  
   }  
}  
  
class Program {  
   public static void Main() {  
      Der der = new Der();  
      D d = der.CallBaseM();  
      d();  
      RetD rd = der.DelToBaseM();  
      rd()();  
   }  
}  
```
