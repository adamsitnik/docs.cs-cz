# <a name="compiler-error-cs0502"></a>CS0502 chyby kompilátoru
'member' nemůže být extern i sealed  
  
 Člen třídy nemůže být současně [abstraktní](../../csharp/language-reference/keywords/abstract.md) a [zapečetěné](../../csharp/language-reference/keywords/sealed.md).  
  
 Následující ukázka generuje CS0502:  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
