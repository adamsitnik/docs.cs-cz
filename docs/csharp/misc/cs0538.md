# <a name="compiler-error-cs0538"></a><span data-ttu-id="9805d-102">CS0538 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="9805d-102">Compiler Error CS0538</span></span>
<span data-ttu-id="9805d-103">'name' v explicitní deklaraci rozhraní není rozhraní.</span><span class="sxs-lookup"><span data-stu-id="9805d-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="9805d-104">Byl proveden pokus o explicitně deklarovat [rozhraní](../../csharp/language-reference/keywords/interface.md), ale žádný se nezadal rozhraní.</span><span class="sxs-lookup"><span data-stu-id="9805d-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="9805d-105">Následující ukázka generuje CS0538:</span><span class="sxs-lookup"><span data-stu-id="9805d-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
