# <a name="compiler-error-cs0538"></a><span data-ttu-id="2f265-102">Chyba kompilátoru CS0538</span><span class="sxs-lookup"><span data-stu-id="2f265-102">Compiler Error CS0538</span></span>
<span data-ttu-id="2f265-103">název v explicitní deklaraci rozhraní není rozhraní.</span><span class="sxs-lookup"><span data-stu-id="2f265-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="2f265-104">Byl proveden pokus o explicitní deklaraci [rozhraní](../language-reference/keywords/interface.md), ale rozhraní nebylo zadáno.</span><span class="sxs-lookup"><span data-stu-id="2f265-104">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="2f265-105">Následující ukázka generuje CS0538:</span><span class="sxs-lookup"><span data-stu-id="2f265-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
