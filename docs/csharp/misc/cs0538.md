# <a name="compiler-error-cs0538"></a>CS0538 chyby kompilátoru
'name' v explicitní deklaraci rozhraní není rozhraní.  
  
 Byl proveden pokus o explicitně deklarovat [rozhraní](../../csharp/language-reference/keywords/interface.md), ale žádný se nezadal rozhraní.  
  
 Následující ukázka generuje CS0538:  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
