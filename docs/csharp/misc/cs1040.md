# <a name="compiler-error-cs1040"></a>Compiler Error CS1040
Direktivy preprocesoru musí být uvedená jako první znak prázdné znaky na řádek  
  
 A [direktivy preprocesoru](../../csharp/language-reference/preprocessor-directives/index.md) nebyla nalezena na řádku a nebyl prvním tokenem v řádku. Direktivu musí být prvním tokenem v řádku.  
  
 Následující ukázka generuje CS1040:  
  
```csharp  
// CS1040.cs  
/* Define a symbol, X */ #define X   // CS1040  
  
// try the following two lines instead  
// /* Define a symbol, X */  
// #define X  
  
public class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
