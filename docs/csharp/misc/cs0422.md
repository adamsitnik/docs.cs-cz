# <a name="compiler-warning-level-4-cs0422"></a>Kompilátor CS0422 upozornění (úroveň 4)
/ Možností přírůstku je již nejsou podporovány.  
  
 Přírůstkové kompilace ( **/incr** nebo **/ incremental**) není podporována v C# 2.0.
