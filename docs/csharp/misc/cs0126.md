# <a name="compiler-error-cs0126"></a>Compiler Error CS0126
Objekt, jehož typ lze převést na "typ" je povinné  
  
 Příkaz return je k dispozici, ale příkaz nevrací hodnotu očekávaného typu. Další informace najdete v tématu [vlastnosti](../../csharp/programming-guide/classes-and-structs/properties.md).  
  
 Následující ukázka generuje CS0126:  
  
```csharp  
// CS0126.cs  
public class a  
{  
   public int i  
   {  
      set  
      {  
      }  
      get  
      {  
         return;   // CS0126, specify a value to return  
      }  
   }  
}  
```
