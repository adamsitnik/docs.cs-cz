# <a name="compiler-error-cs2020"></a>Compiler Error CS2020
Pouze první sada vstupních souborů může vytvořit jiný cíl než 'module'  
  
 Více výstupu kompilace v první výstupní soubor musí být sestaveny s [/target: exe](../../csharp/language-reference/compiler-options/target-exe-compiler-option.md), [/target: winexe](../../csharp/language-reference/compiler-options/target-winexe-compiler-option.md), nebo [/target: library](../../csharp/language-reference/compiler-options/target-library-compiler-option.md). Všechny následné výstupní soubory musí být sestaveny s [/target: Module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md).
