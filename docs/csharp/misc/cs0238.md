 <span data-ttu-id="b38b2-104">[zapečetění](../language-reference/keywords/sealed.md) bylo použito pro člena, který nebyl označen také jako [přepsání](../language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="b38b2-104">[sealed](../language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../language-reference/keywords/override.md).</span></span> <span data-ttu-id="b38b2-105">Další informace najdete v tématu [Dědičnost](../programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="b38b2-105">For more information, see [Inheritance](../programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="b38b2-106">Následující ukázka generuje CS0238:</span><span class="sxs-lookup"><span data-stu-id="b38b2-106">The following sample generates CS0238:</span></span>  
  
```csharp

   public abstract void f();  
}

   }

   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
