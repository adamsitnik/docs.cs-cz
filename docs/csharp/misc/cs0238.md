# <a name="compiler-error-cs0238"></a><span data-ttu-id="bf2eb-102">CS0238 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="bf2eb-102">Compiler Error CS0238</span></span>
<span data-ttu-id="bf2eb-103">'member' nelze zapečetit, protože to není přepis.</span><span class="sxs-lookup"><span data-stu-id="bf2eb-103">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="bf2eb-104">[zapečetěné](../../csharp/language-reference/keywords/sealed.md) byla použita na člen, který také nebyla označena atributem [přepsat](../../csharp/language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="bf2eb-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="bf2eb-105">Další informace najdete v tématu [dědičnosti](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="bf2eb-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="bf2eb-106">Následující ukázka generuje CS0238:</span><span class="sxs-lookup"><span data-stu-id="bf2eb-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
