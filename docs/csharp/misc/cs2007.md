# <a name="compiler-error-cs2007"></a>Chyba kompilátoru CS2007
Nerozpoznaná možnost příkazového řádku: možnost ' Option '  
  
 Kompilátor byl předán řetězec, který nebyl [možností kompilátoru](../language-reference/compiler-options/index.md), i když začal s lomítkem (/).  
  
 Následující ukázka generuje CS2007:  
  
```csharp  
// CS2007.cs  
// compile with: /recur  
// CS2007 expected  
class x  
{  
   public static void Main() {}  
}  
```
