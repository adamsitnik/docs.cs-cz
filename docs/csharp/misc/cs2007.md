# <a name="compiler-error-cs2007"></a>Compiler Error CS2007
Nerozpoznaný přepínač příkazového řádku: 'možnost.  
  
 Kompilátor byl předán řetězec, který nebyl [– možnost kompilátoru](../../csharp/language-reference/compiler-options/index.md), i když začal s lomítkem (/).  
  
 Následující ukázka generuje CS2007:  
  
```csharp  
// CS2007.cs  
// compile with: /recur  
// CS2007 expected  
class x  
{  
   public static void Main() {}  
}  
```
