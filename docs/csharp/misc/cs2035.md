# <a name="compiler-error-cs2035"></a>CS2035 chyby kompilátoru
Chyba syntaxe příkazového řádku:  Chybí ":\<číslo >' parametru 'compiler_option.  
  
 Některé možnosti kompilátoru vyžaduje hodnotu.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS2035.  
  
```csharp  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```
