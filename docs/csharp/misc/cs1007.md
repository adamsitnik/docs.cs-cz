# <a name="compiler-error-cs1007"></a>Compiler Error CS1007
Přistupující objekt vlastnosti, které jsou již definovány.  
  
 Pokud deklarujete [vlastnost](../../csharp/programming-guide/classes-and-structs/using-properties.md), musíte také deklarovat jeho přístupových metod. Vlastnost však nelze mít více než jeden `get` přístupové metody nebo více než jeden `set` přístupové metody.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1007:  
  
```csharp  
// CS1007.cs  
public class clx  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
