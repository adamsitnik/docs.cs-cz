# <a name="compiler-error-cs1007"></a><span data-ttu-id="62183-102">Chyba kompilátoru CS1007</span><span class="sxs-lookup"><span data-stu-id="62183-102">Compiler Error CS1007</span></span>

 <span data-ttu-id="62183-104">Při deklaraci [vlastnosti](../programming-guide/classes-and-structs/using-properties.md)je nutné deklarovat také své přístupové metody.</span><span class="sxs-lookup"><span data-stu-id="62183-104">When you declare a [property](../programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="62183-105">Nicméně vlastnost nemůže mít více než jednu `get` metodu přístupového objektu nebo více než jednu `set` metodu přístupového objektu.</span><span class="sxs-lookup"><span data-stu-id="62183-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="62183-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="62183-106">Example</span></span>  
 <span data-ttu-id="62183-107">Následující ukázka generuje CS1007:</span><span class="sxs-lookup"><span data-stu-id="62183-107">The following sample generates CS1007:</span></span>  
  
```csharp  
// CS1007.cs  
public class clx  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
