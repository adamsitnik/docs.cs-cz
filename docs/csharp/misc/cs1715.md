# <a name="compiler-error-cs1715"></a>Compiler Error CS1715
'Type1': typ musí být 'Type2' aby odpovídal přepsanému členu 'MemberName.  
  
 Tato chyba je stejný jako [CS0508 Chyba kompilátoru](../../csharp/misc/cs0508.md), s tím rozdílem, že CS0508 nyní platí jenom pro metody, které mají návratové typy, zatímco CS1715 platí pro vlastnostmi a indexery, které mají pouze "typy" místo "návratové typy".  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje CS1715.  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
