# <a name="compiler-error-cs1715"></a><span data-ttu-id="1ccad-102">Compiler Error CS1715</span><span class="sxs-lookup"><span data-stu-id="1ccad-102">Compiler Error CS1715</span></span>
<span data-ttu-id="1ccad-103">'Type1': typ musí být 'Type2' aby odpovídal přepsanému členu 'MemberName.</span><span class="sxs-lookup"><span data-stu-id="1ccad-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="1ccad-104">Tato chyba je stejný jako [CS0508 Chyba kompilátoru](../../csharp/misc/cs0508.md), s tím rozdílem, že CS0508 nyní platí jenom pro metody, které mají návratové typy, zatímco CS1715 platí pro vlastnostmi a indexery, které mají pouze "typy" místo "návratové typy".</span><span class="sxs-lookup"><span data-stu-id="1ccad-104">This error is the same as [Compiler Error CS0508](../../csharp/misc/cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1ccad-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="1ccad-105">Example</span></span>  
 <span data-ttu-id="1ccad-106">Následující kód vygeneruje CS1715.</span><span class="sxs-lookup"><span data-stu-id="1ccad-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
