# <a name="compiler-error-cs1918"></a><span data-ttu-id="83910-102">Chyba kompilátoru CS1918</span><span class="sxs-lookup"><span data-stu-id="83910-102">Compiler Error CS1918</span></span>
<span data-ttu-id="83910-103">Členy vlastnosti Name typu Type nelze přiřadit k inicializátoru objektu, protože se jedná o hodnotový typ.</span><span class="sxs-lookup"><span data-stu-id="83910-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="83910-104">K této chybě dochází, pokud se pokusíte použít inicializátor objektu pro inicializaci vlastností typu struktury, který je sám vlastností třídy, která je inicializována.</span><span class="sxs-lookup"><span data-stu-id="83910-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="83910-105">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="83910-105">To correct this error</span></span>  
  
1. <span data-ttu-id="83910-106">Pokud je nutné plně inicializovat pole vlastnosti v inicializátoru objektu, změňte strukturu na typ třídy.</span><span class="sxs-lookup"><span data-stu-id="83910-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="83910-107">V opačném případě inicializujte členy struktury v samostatném volání metody po vytvoření objektu pomocí inicializátoru objektu.</span><span class="sxs-lookup"><span data-stu-id="83910-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="83910-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="83910-108">Example</span></span>  
 <span data-ttu-id="83910-109">Následující příklad generuje CS1918:</span><span class="sxs-lookup"><span data-stu-id="83910-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
