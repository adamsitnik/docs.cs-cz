# <a name="compiler-error-cs1041"></a>Compiler Error CS1041
Byl očekáván identifikátor 'klíčové slovo' je klíčové slovo  
  
 Vyhrazené slovo pro C# jazyk byl nalezen, kde byl očekáván identifikátor. Nahradit [– klíčové slovo](../../csharp/language-reference/keywords/index.md) s identifikátorem zadané uživatelem.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1041:  
  
```csharp  
// CS1041a.cs  
class MyClass  
{  
    public void f(int long)   // CS1041  
    // Try the following instead:  
    // public void f(int i)  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```  
  
## <a name="example"></a>Příklad  
 Při importu z programovací jazyk, který nemá stejnou sadu vyhrazených slov, můžete upravit rezervovaný identifikátor s @ předpony, jak je ukázáno v následujícím příkladu.  
  
 Identifikátor s `@` předpona je volána Doslovný identifikátor.  
  
```csharp  
// CS1041b.cs  
class MyClass  
{  
    public void f(int long)   // CS1041  
    // Try the following instead:  
    // public void f(int @long)  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
