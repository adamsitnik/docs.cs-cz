# <a name="compiler-error-cs0010"></a>Compiler Error CS0010
Neočekávaná závažná chyba--"Chyba".  
  
 Tato chyba se vygeneruje, když dojde ke zcela neočekávané dojde k zastavení kompilace.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Znovu zkompilujte jiný projekt. Pokud se zobrazí stejná chyba, zkuste přeinstalovat aplikaci Visual Studio. Pokud se zobrazí tato chyba, odešlete zprávu o chybách společnosti Microsoft.
