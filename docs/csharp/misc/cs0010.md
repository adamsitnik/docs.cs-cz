# <a name="compiler-error-cs0010"></a>Chyba kompilátoru CS0010
Neočekávaná závažná chyba--' error '.  
  
 Tato chyba je vygenerována v případě, že dojde k neočekávanému selhání pro zastavení kompilace.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Znovu zkompilujte další projekt. Pokud se zobrazí stejná chyba, zkuste znovu nainstalovat Visual Studio. Pokud se zobrazí tato chyba, pošlete Microsoftu zprávu o chybě.
