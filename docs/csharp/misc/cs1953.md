# <a name="compiler-error-cs1953"></a>Compiler Error CS1953
Strom výrazu lambda nesmí obsahovat skupinu metod.  
  
 Vyžaduje volání metody `()` operátor. Název metody bez tohoto operátoru odkazuje na skupinu metod, což je sada všechny přetížené metody s tímto názvem.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Pokud chcete volat metodu, přidejte `()` operátor.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS1953:  
  
```csharp  
// cs1953.cs  
using System;  
using System.Linq.Expressions;  
class CS1953  
{  
    public static void Main()  
    {  
        double num = 10;  
        Expression<Func<bool>> testExpr =  
              () => num.GetType is int; // CS1953   
    }  
}  
```
