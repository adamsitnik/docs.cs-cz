# <a name="compiler-error-cs1620"></a>Compiler Error CS1620
Argument 'number' musí být předán s klíčovým slovem '– klíčové slovo.  
  
 K této chybě dochází, pokud jsou předáním argumentu na funkci, která přijímá [ref](../../csharp/language-reference/keywords/ref.md) nebo [si](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametr a nezahrnují `ref` nebo `out` – klíčové slovo v místě volání, nebo zahrnout Chybný – klíčové slovo. Text chyby označuje odpovídající klíčové slovo použít a které argument způsobil chybu.  
  
 Následující ukázka generuje CS1620:  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
