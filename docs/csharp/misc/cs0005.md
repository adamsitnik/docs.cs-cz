# <a name="compiler-error-cs0005"></a>CS0005 chyby kompilátoru
– Možnost kompilátoru 'compiler_option' musí následovat argument  
  
 Některé možnosti kompilátoru vyžadují parametry. Pokud nepředáte argumenty potřebné možnosti kompilátoru, vygeneruje se CS0005.  
  
 Další informace najdete v tématu [ C# – možnosti kompilátoru](../../csharp/language-reference/compiler-options/index.md).
