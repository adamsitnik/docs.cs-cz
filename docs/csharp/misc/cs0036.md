# <a name="compiler-error-cs0036"></a><span data-ttu-id="064c6-102">Compiler Error CS0036</span><span class="sxs-lookup"><span data-stu-id="064c6-102">Compiler Error CS0036</span></span>
<span data-ttu-id="064c6-103">Parametr out nemůže obsahovat atribut '[In].</span><span class="sxs-lookup"><span data-stu-id="064c6-103">An out parameter cannot have the '[In]' attribute</span></span>  
  
 <span data-ttu-id="064c6-104">V současné době **v** atribut není povolený [si](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametru.</span><span class="sxs-lookup"><span data-stu-id="064c6-104">Currently, the **In** attribute is not permitted on an [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span>  
  
 <span data-ttu-id="064c6-105">Následující ukázka generuje CS0036:</span><span class="sxs-lookup"><span data-stu-id="064c6-105">The following sample generates CS0036:</span></span>  
  
```csharp  
// CS0036.cs  
  
using System;  
using System.Runtime.InteropServices;  
  
public class MyClass  
{  
   public static void TestOut([In] out char TestChar)   // CS0036  
   // try the following line instead  
   // public static void TestOut(out char TestChar)  
   {  
      TestChar = 'b';  
      Console.WriteLine(TestChar);  
   }  
  
   public static void Main()  
   {  
      char i;           //variable to receive the value  
      TestOut(out i);   // the arg must be passed as out  
      Console.WriteLine(i);  
   }  
}  
```
