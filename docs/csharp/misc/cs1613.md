# <a name="compiler-error-cs1613"></a>Chyba kompilátoru CS1613
Spravovaná třída obálky třídy coclass pro rozhraní Interface se nenašla. (nechybí odkaz na sestavení?)  
  
 Byl proveden pokus o vytvoření instance objektu COM z rozhraní. Rozhraní má **atributem ComImport** a `CoClass` atributy, ale kompilátor nemůže najít `CoClass` typ zadaný pro atribut.  
  
 Chcete-li vyřešit tuto chybu, můžete vyzkoušet jednu z následujících možností:  
  
- Přidejte odkaz na sestavení, které má coclass (nejvíce z času, že rozhraní a třída coclass by měly být ve stejném sestavení). Informace [](../language-reference/compiler-options/reference-compiler-option.md) najdete v [dialogovém okně/reference nebo přidat odkaz](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) .  
  
- Opravte `CoClass` atribut rozhraní.  
  
 Následující příklad znázorňuje správné použití **CoClassAttribute**:  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
