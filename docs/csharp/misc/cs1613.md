# <a name="compiler-error-cs1613"></a><span data-ttu-id="8a56d-102">Chyba kompilátoru CS1613</span><span class="sxs-lookup"><span data-stu-id="8a56d-102">Compiler Error CS1613</span></span>
<span data-ttu-id="8a56d-103">Spravovaná třída obálky třídy coclass pro rozhraní Interface se nenašla. (nechybí odkaz na sestavení?)</span><span class="sxs-lookup"><span data-stu-id="8a56d-103">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="8a56d-104">Byl proveden pokus o vytvoření instance objektu COM z rozhraní.</span><span class="sxs-lookup"><span data-stu-id="8a56d-104">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="8a56d-105">Rozhraní má **atributem ComImport** a `CoClass` atributy, ale kompilátor nemůže najít `CoClass` typ zadaný pro atribut.</span><span class="sxs-lookup"><span data-stu-id="8a56d-105">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="8a56d-106">Chcete-li vyřešit tuto chybu, můžete vyzkoušet jednu z následujících možností:</span><span class="sxs-lookup"><span data-stu-id="8a56d-106">To resolve this error, you can try one of the following:</span></span>  
  
- <span data-ttu-id="8a56d-107">Přidejte odkaz na sestavení, které má coclass (nejvíce z času, že rozhraní a třída coclass by měly být ve stejném sestavení).</span><span class="sxs-lookup"><span data-stu-id="8a56d-107">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="8a56d-108">Informace [](../language-reference/compiler-options/reference-compiler-option.md) najdete v [dialogovém okně/reference nebo přidat odkaz](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) .</span><span class="sxs-lookup"><span data-stu-id="8a56d-108">See [/reference](../language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
- <span data-ttu-id="8a56d-109">Opravte `CoClass` atribut rozhraní.</span><span class="sxs-lookup"><span data-stu-id="8a56d-109">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="8a56d-110">Následující příklad znázorňuje správné použití **CoClassAttribute**:</span><span class="sxs-lookup"><span data-stu-id="8a56d-110">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
