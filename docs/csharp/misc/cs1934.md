## <a name="example"></a><span data-ttu-id="acfab-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="acfab-110">Example</span></span>  
 <span data-ttu-id="acfab-111">Následující příklad ukazuje jeden ze způsobů, jak vydávat CS1934:</span><span class="sxs-lookup"><span data-stu-id="acfab-111">The following example shows one way to produce CS1934:</span></span>

using System.Collections;  
static class Test  
{  
    public static void Main()  
    {  
        var list = new ArrayList { 0, 1, 2, 3, 4, 5 };
