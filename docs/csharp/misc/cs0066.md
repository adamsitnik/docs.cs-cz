# <a name="compiler-error-cs0066"></a><span data-ttu-id="52b80-102">Chyba kompilátoru CS0066</span><span class="sxs-lookup"><span data-stu-id="52b80-102">Compiler Error CS0066</span></span>
<span data-ttu-id="52b80-103">' Event ': událost musí být typu delegát.</span><span class="sxs-lookup"><span data-stu-id="52b80-103">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="52b80-104">Klíčové slovo event vyžaduje typ [delegáta](../language-reference/builtin-types/reference-types.md) .</span><span class="sxs-lookup"><span data-stu-id="52b80-104">The event keyword requires a [delegate](../language-reference/builtin-types/reference-types.md) type.</span></span> <span data-ttu-id="52b80-105">Další informace najdete v tématu [události](../programming-guide/events/index.md) a [Delegáti](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="52b80-105">For more information, see [Events](../programming-guide/events/index.md) and [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="52b80-106">Následující ukázka generuje CS0066:</span><span class="sxs-lookup"><span data-stu-id="52b80-106">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
