# <a name="compiler-error-cs0667"></a>CS0667 chyby kompilátoru
Funkce "Neplatná funkce" je zastaralá. Místo toho použijte "platná funkce" '.  
  
 Funkce, kterou se pokoušíte použít, je nyní zastaralá. Aktualizace kódu místo toho použít funkci platná.
