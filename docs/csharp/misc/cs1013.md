# <a name="compiler-error-cs1013"></a>Compiler Error CS1013
Číslo je neplatné  
  
 Kompilátor zjistil poškozený číslo. Další informace o celočíselných typů naleznete v tématu [integrální typy tabulky](../../csharp/language-reference/builtin-types/integral-numeric-types.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1013:  
  
```csharp  
// CS1013.cs  
class Sample  
{  
    static void Main()  
    {  
        int i = 0x;   // CS1013  
    }  
}  
```
