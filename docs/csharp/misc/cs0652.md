# <a name="compiler-warning-level-2-cs0652"></a>Kompilátor CS0652 upozornění (úroveň 2)
Porovnání s integrální konstantou je zbytečné. konstanty je mimo rozsah typu 'type'  
  
 Kompilátor zjistil porovnání konstanty a proměnné, kde konstanty je mimo rozsah proměnné.  
  
 Následující ukázka generuje CS0652:  
  
```csharp  
// CS0652.cs  
// compile with: /W:2  
public class Class1  
{  
   private static byte i = 0;  
   public static void Main()  
   {  
      short j = 256;  
      if (i == 256)   // CS0652, 256 is out of range for byte  
         i = 0;  
   }  
}  
```
