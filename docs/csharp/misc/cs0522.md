# <a name="compiler-error-cs0522"></a>Chyba kompilátoru CS0522
konstruktor: struktury nemůžou volat konstruktory základních tříd.  
  
 [Struktura](../language-reference/keywords/struct.md) nemůže volat konstruktor základní třídy; odeberte volání konstruktoru základní třídy.  
  
 Následující ukázka generuje CS0522:  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
