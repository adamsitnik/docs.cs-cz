# <a name="compiler-error-cs2005"></a>CS2005 chyby kompilátoru
Pro možnost 'možnosti' chybí specifikace souboru.  
  
 A [– možnost kompilátoru](../../csharp/language-reference/compiler-options/index.md) byl zadán pouze částečně.  
  
 Například při použití [/recurse](../../csharp/language-reference/compiler-options/recurse-compiler-option.md), je třeba zadat soubor do hledání: **/recurse:***filename***.cs**.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS2005.  
  
```csharp  
// CS2005.cs  
// compile with: /recurse:  
// CS2005 expected  
class x  
{  
   public static void Main() {}  
}  
```
