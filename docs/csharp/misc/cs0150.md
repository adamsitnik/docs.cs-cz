# <a name="compiler-error-cs0150"></a><span data-ttu-id="a8ee6-102">Compiler Error CS0150</span><span class="sxs-lookup"><span data-stu-id="a8ee6-102">Compiler Error CS0150</span></span>
<span data-ttu-id="a8ee6-103">Očekává se konstantní hodnota</span><span class="sxs-lookup"><span data-stu-id="a8ee6-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="a8ee6-104">Proměnná byla nalezena, kde byla očekávána konstanta.</span><span class="sxs-lookup"><span data-stu-id="a8ee6-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="a8ee6-105">Další informace najdete v tématu [přepnout](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="a8ee6-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="a8ee6-106">Následující ukázka generuje CS0150:</span><span class="sxs-lookup"><span data-stu-id="a8ee6-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="a8ee6-107">Tato chyba je také vytvořen, pokud je velikost pole je zadána s hodnotu proměnné a inicializovat s inicializátorem pole.</span><span class="sxs-lookup"><span data-stu-id="a8ee6-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="a8ee6-108">Chcete-li odebrat chyby, inicializujte pole v samostatný příkaz nebo příkazy.</span><span class="sxs-lookup"><span data-stu-id="a8ee6-108">To remove the error, initialize the array in a separate statement or statements.</span></span>

```csharp  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
