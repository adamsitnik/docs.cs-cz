# <a name="compiler-error-cs0150"></a>Compiler Error CS0150
Očekává se konstantní hodnota  
  
 Proměnná byla nalezena, kde byla očekávána konstanta. Další informace najdete v tématu [přepnout](../../csharp/language-reference/keywords/switch.md).  
  
 Následující ukázka generuje CS0150:  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 Tato chyba je také vytvořen, pokud je velikost pole je zadána s hodnotu proměnné a inicializovat s inicializátorem pole. Chcete-li odebrat chyby, inicializujte pole v samostatný příkaz nebo příkazy.  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
