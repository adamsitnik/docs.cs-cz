# <a name="compiler-error-cs0527"></a>Compiler Error CS0527
Typ 'type' v seznamu rozhraní není rozhraní.  
  
 Je možné, [struktura](../../csharp/language-reference/keywords/struct.md) nebo [rozhraní](../../csharp/language-reference/keywords/interface.md) dědit z jiného rozhraní, ale ne z jakéhokoli jiného typu.  
  
 Následující ukázka generuje CS0527:  
  
```csharp  
// CS0527.cs  
// compile with: /target:library  
public struct clx : int {}   // CS0527 int not an interface  
```
