 <span data-ttu-id="f26b4-104">Členy třídy nebo struktury nemůže mít stejný název jako třídy nebo struktury, pokud je člen konstruktoru.</span><span class="sxs-lookup"><span data-stu-id="f26b4-104">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="f26b4-105">Následující ukázka generuje CS0542:</span><span class="sxs-lookup"><span data-stu-id="f26b4-105">The following sample generates CS0542:</span></span>  
  
```csharp

{  
    public int C;  
}  
```  
  
 <span data-ttu-id="f26b4-106">Tato chyba může být způsobena, jestliže je neúmyslně návratový typ v konstruktoru, který ve skutečnosti dostane do běžné metody.</span><span class="sxs-lookup"><span data-stu-id="f26b4-106">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="f26b4-107">Následující příklad generuje CS0542, protože `F` je metoda, není konstruktor, protože má návratový typ:</span><span class="sxs-lookup"><span data-stu-id="f26b4-107">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp

{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
