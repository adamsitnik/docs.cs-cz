## <a name="example"></a><span data-ttu-id="12bc4-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="12bc4-108">Example</span></span>

```csharp  
namespace TypeBindConflicts   
{  
    namespace NsImpAggPubImp   
    {  
        public class X { }  
    }  
}  
```
