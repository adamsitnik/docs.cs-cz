# <a name="compiler-error-cs0214"></a>Compiler Error CS0214
Ukazatele a vyrovnávací paměti pevné velikosti můžou použít jenom v nezabezpečeném kontextu.  
  
 Ukazatele lze použít pouze s [nebezpečné](../../csharp/language-reference/keywords/unsafe.md) – klíčové slovo. Další informace najdete v tématu [nezabezpečený kód a ukazatele](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Následující ukázka generuje CS0214:  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
