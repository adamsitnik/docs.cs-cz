# <a name="compiler-error-cs0543"></a>Compiler Error CS0543
'enumeration': hodnota výčtu je pro příslušný typ příliš velká  
  
 Hodnota, která byla přiřazena na prvek v [výčet](../../csharp/language-reference/keywords/enum.md) je mimo rozsah datového typu.  
  
 Následující ukázka generuje CS0543:  
  
```csharp  
// CS0543.cs  
namespace x  
{  
   enum I : byte  
   {a = 255, b, c}   // CS0543  
   public class clx  
   {  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
