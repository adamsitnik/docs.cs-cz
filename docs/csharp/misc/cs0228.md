# <a name="compiler-error-cs0228"></a>CS0228 chyby kompilátoru
'type' neobsahuje definici pro 'member', nebo není k dispozici  
  
 Můžete nahradit systémovou verzi **System.Hashtable**, **System.String**, nebo **System.Array**, a nahrazení neobsahuje `member`.  
  
 V opačném případě může potřebovat opravte nebo přeinstalujte Visual Studio.
