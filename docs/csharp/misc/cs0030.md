# <a name="compiler-error-cs0030"></a><span data-ttu-id="26173-102">Chyba kompilátoru CS0030</span><span class="sxs-lookup"><span data-stu-id="26173-102">Compiler Error CS0030</span></span>
<span data-ttu-id="26173-103">Typ typu nejde převést na typ.</span><span class="sxs-lookup"><span data-stu-id="26173-103">Cannot convert type 'type' to 'type'</span></span>  
  
<span data-ttu-id="26173-104">Mezi typy neexistuje žádný předdefinovaný převod.</span><span class="sxs-lookup"><span data-stu-id="26173-104">There is no predefined conversion between types.</span></span> <span data-ttu-id="26173-105">Můžete definovat vlastní převod mezi těmito typy.</span><span class="sxs-lookup"><span data-stu-id="26173-105">You can define a custom conversion between those types.</span></span> <span data-ttu-id="26173-106">Další informace naleznete v tématu [uživatelsky definované operátory převodu](../language-reference/operators/user-defined-conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="26173-106">For more information, see [User-defined conversion operators](../language-reference/operators/user-defined-conversion-operators.md).</span></span>  
  
 <span data-ttu-id="26173-107">Následující ukázka generuje CS0030:</span><span class="sxs-lookup"><span data-stu-id="26173-107">The following sample generates CS0030:</span></span>  
  
```csharp  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
