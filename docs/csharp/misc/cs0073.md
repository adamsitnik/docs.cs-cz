<span data-ttu-id="69d68-103">Přistupující objekt Add nebo Remove musí mít tělo.</span><span class="sxs-lookup"><span data-stu-id="69d68-103">An add or remove accessor must have a body</span></span>  
  
 <span data-ttu-id="69d68-104">Klíčové slovo **Add** nebo **Remove** v definici [události](../language-reference/keywords/event.md) musí mít tělo.</span><span class="sxs-lookup"><span data-stu-id="69d68-104">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="69d68-105">Další informace najdete v tématu [události](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="69d68-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="69d68-106">Následující ukázka generuje CS0073:</span><span class="sxs-lookup"><span data-stu-id="69d68-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
