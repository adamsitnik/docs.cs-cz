# <a name="compiler-error-cs0073"></a>Compiler Error CS0073
Přístupový objekt add nebo remove musí mít tělo.  
  
 **Přidat** nebo **odebrat** – klíčové slovo v [události](../../csharp/language-reference/keywords/event.md) definice musí mít tělo. Další informace najdete v tématu [události](../../csharp/programming-guide/events/index.md).  
  
 Následující ukázka generuje CS0073:  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
