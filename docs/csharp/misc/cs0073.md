 <span data-ttu-id="7a975-104">**Přidat** nebo **odebrat** – klíčové slovo v [události](../../csharp/language-reference/keywords/event.md) definice musí mít tělo.</span><span class="sxs-lookup"><span data-stu-id="7a975-104">An **add** or **remove** keyword in an [event](../../csharp/language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="7a975-105">Další informace najdete v tématu [události](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="7a975-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="7a975-106">Následující ukázka generuje CS0073:</span><span class="sxs-lookup"><span data-stu-id="7a975-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();

{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
