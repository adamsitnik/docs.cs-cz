# <a name="compiler-error-cs0014"></a>Compiler Error CS0014
Požadovaný soubor 'file' nebyl nalezen.  
  
 Soubor je potřeba v kompilátoru, ale není v systému. Ujistěte se, že cesta je správná. Pokud je soubor sady Visual Studio systémový soubor, pak může musíte opravit instalaci nebo odeberte a přeinstalujte Visual Studio úplně.
