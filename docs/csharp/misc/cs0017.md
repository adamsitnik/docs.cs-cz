# <a name="compiler-error-cs0017"></a>Compiler Error CS0017
Program 'název výstupního souboru' má definovaný více než jeden vstupní bod. Kompilovat s/Main určující typ obsahující vstupní bod.  
  
 Program může mít pouze jeden [hlavní](../../csharp/programming-guide/main-and-command-args/index.md) metody.  
  
 Chcete-li vyřešit tuto chybu, můžete buď odstranit všechny hlavní metody v kódu, s výjimkou jednoho, nebo můžete použít [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) – možnost kompilátoru stanovit, jakou metodu Main, kterou chcete použít.  
  
 Následující ukázka generuje CS0017:  
  
```csharp  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
