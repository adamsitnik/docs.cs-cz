# <a name="compiler-error-cs0533"></a>Compiler Error CS0533
'member odvozené třídy' skrývá zděděný abstraktní člen 'member základní třídy.  
  
 Základní [třídy](../../csharp/language-reference/keywords/class.md) metoda je skrytá. Zkontrolujte syntaxi vašeho prohlášení, jestli je správný.  
  
 Další informace najdete v tématu [základní](../../csharp/language-reference/keywords/base.md).  
  
 Následující ukázka generuje CS0533:  
  
```csharp  
// CS0533.cs  
namespace x  
{  
   abstract public class a  
   {  
      abstract public void f();  
   }  
  
   abstract public class b : a  
   {  
      new abstract public void f();   // CS0533  
      // try the following lines instead  
      // override public void f()  
      // {  
      // }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
