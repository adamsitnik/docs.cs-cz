# <a name="compiler-error-cs0636"></a><span data-ttu-id="26b00-102">CS0636 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="26b00-102">Compiler Error CS0636</span></span>
<span data-ttu-id="26b00-103">Atribut FieldOffset lze umístit pouze pro členy typů s deklarací StructLayout(LayoutKind.Explicit)</span><span class="sxs-lookup"><span data-stu-id="26b00-103">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="26b00-104">Je nutné použít **deklarací StructLayout(LayoutKind.Explicit)** atribut v deklaraci struktury, pokud neobsahuje žádné členy označené **FieldOffset** atribut.</span><span class="sxs-lookup"><span data-stu-id="26b00-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="26b00-105">Další informace najdete v tématu [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="26b00-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="26b00-106">Následující ukázka generuje CS0636:</span><span class="sxs-lookup"><span data-stu-id="26b00-106">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636   
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
