# <a name="compiler-error-cs1510"></a>Compiler Error CS1510
Ref nebo out argument musí být přiřaditelný proměnná  
  
 Pouze proměnné mohou být předány jako [ref](../../csharp/language-reference/keywords/ref.md) parametr ve volání metody. A `ref` hodnota je například předání ukazatele.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1510:  
  
```csharp  
// CS1510.cs  
public class C  
{  
   public static int j = 0;  
  
   public static void M(ref int j)  
   {  
      j++;  
   }  
  
   public static void Main ()  
   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error  
      // M (ref j);  
   }  
}  
```
