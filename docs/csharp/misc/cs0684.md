# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="dfb46-102">Kompilátor CS0684 upozornění (úroveň 1)</span><span class="sxs-lookup"><span data-stu-id="dfb46-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="dfb46-103">rozhraní 'rozhraní' označena s deklarací 'CoClassAttribute' není označena jako deklaraci comimportattribute.</span><span class="sxs-lookup"><span data-stu-id="dfb46-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="dfb46-104">Pokud zadáte **CoClassAttribute** v rozhraní, musíte zadat také **deklaraci ComImportAttribute**.</span><span class="sxs-lookup"><span data-stu-id="dfb46-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="dfb46-105">Následující ukázka generuje CS0684:</span><span class="sxs-lookup"><span data-stu-id="dfb46-105">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
