# <a name="compiler-warning-level-1-cs0684"></a>Kompilátor CS0684 upozornění (úroveň 1)
rozhraní 'rozhraní' označena s deklarací 'CoClassAttribute' není označena jako deklaraci comimportattribute.  
  
 Pokud zadáte **CoClassAttribute** v rozhraní, musíte zadat také **deklaraci ComImportAttribute**.  
  
 Následující ukázka generuje CS0684:  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
