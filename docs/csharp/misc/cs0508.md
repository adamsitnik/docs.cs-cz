# <a name="compiler-error-cs0508"></a>Compiler Error CS0508
"Zadejte 1.: návratový typ musí být typu 2 aby odpovídal přepsanému členu 'název člena.  
  
 Chcete-li změnit návratový typ v přepsání metody byl proveden pokus o. Chcete-li vyřešit tuto chybu, ujistěte se, že obě metody deklarovat, že stejný návratový typ.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0508.  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
