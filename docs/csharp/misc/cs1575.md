# <a name="compiler-error-cs1575"></a>Compiler Error CS1575
Výraz stackalloc vyžaduje za typem]  
  
 Velikost požadované alokace s [stackalloc](../language-reference/operators/stackalloc.md), musí být zadán v hranatých závorkách.  
  
 Následující ukázka generuje CS1575:  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
