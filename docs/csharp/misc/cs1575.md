# <a name="compiler-error-cs1575"></a><span data-ttu-id="cc7aa-102">Compiler Error CS1575</span><span class="sxs-lookup"><span data-stu-id="cc7aa-102">Compiler Error CS1575</span></span>
<span data-ttu-id="cc7aa-103">Výraz stackalloc vyžaduje za typem]</span><span class="sxs-lookup"><span data-stu-id="cc7aa-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="cc7aa-104">Velikost požadované alokace s [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), musí být zadán v hranatých závorkách.</span><span class="sxs-lookup"><span data-stu-id="cc7aa-104">The size of the requested allocation, with [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="cc7aa-105">Následující ukázka generuje CS1575:</span><span class="sxs-lookup"><span data-stu-id="cc7aa-105">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
