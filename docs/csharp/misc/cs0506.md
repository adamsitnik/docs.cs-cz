# <a name="compiler-error-cs0506"></a><span data-ttu-id="24014-102">CS0506 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="24014-102">Compiler Error CS0506</span></span>
<span data-ttu-id="24014-103">'function1': zděděný člen 'function2' protože je označené jako "virtuální", "abstraktní" nebo "přepisuje"</span><span class="sxs-lookup"><span data-stu-id="24014-103">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="24014-104">Metoda byla přepsána, která nebyla výslovně označena jako **virtuální**, **abstraktní**, nebo `override`.</span><span class="sxs-lookup"><span data-stu-id="24014-104">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="24014-105">Následující ukázka generuje CS0506:</span><span class="sxs-lookup"><span data-stu-id="24014-105">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
