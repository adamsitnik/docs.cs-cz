# <a name="compiler-error-cs0506"></a>CS0506 chyby kompilátoru
'function1': zděděný člen 'function2' protože je označené jako "virtuální", "abstraktní" nebo "přepisuje"  
  
 Metoda byla přepsána, která nebyla výslovně označena jako **virtuální**, **abstraktní**, nebo `override`.  
  
 Následující ukázka generuje CS0506:  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
