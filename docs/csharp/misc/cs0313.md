# <a name="compiler-error-cs0313"></a><span data-ttu-id="1cc37-102">Chyba kompilátoru CS0313</span><span class="sxs-lookup"><span data-stu-id="1cc37-102">Compiler Error CS0313</span></span>
<span data-ttu-id="1cc37-103">Typ typ1 nelze použít jako parametr typu parametru name v obecném typu nebo metodě ' typ2 '.</span><span class="sxs-lookup"><span data-stu-id="1cc37-103">The type 'type1' cannot be used as type parameter 'parameter name' in the generic type or method 'type2'.</span></span> <span data-ttu-id="1cc37-104">Typ s možnou hodnotou null ' typ1 ' nesplňuje omezení ' typ2 '.</span><span class="sxs-lookup"><span data-stu-id="1cc37-104">The nullable type 'type1' does not satisfy the constraint of 'type2'.</span></span> <span data-ttu-id="1cc37-105">Typy s možnou hodnotou null nemohou vyhovět žádným omezením rozhraní.</span><span class="sxs-lookup"><span data-stu-id="1cc37-105">Nullable types cannot satisfy any interface constraints.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="1cc37-108">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="1cc37-108">To correct this error</span></span>  
  
1. <span data-ttu-id="1cc37-109">Pomocí kódu, který následuje jako příklad, je jedno řešení určit běžný `ImplStruct` jako první argument typu v volání metody `TestMethod`.</span><span class="sxs-lookup"><span data-stu-id="1cc37-109">Using the code that follows as an example, one solution is to specify an ordinary `ImplStruct` as the first type argument in the call to `TestMethod`.</span></span> <span data-ttu-id="1cc37-110">Pak upravte `TestMethod` pro vytvoření verze `Implstruct` s možnou hodnotou null v příkazu return:</span><span class="sxs-lookup"><span data-stu-id="1cc37-110">Then modify `TestMethod` to create a nullable version of `Implstruct` in its return statement:</span></span>  
  
    ```csharp  
    return new Nullable<T>(t);  
    ```  
  
## <a name="example"></a><span data-ttu-id="1cc37-111">Příklad</span><span class="sxs-lookup"><span data-stu-id="1cc37-111">Example</span></span>

<span data-ttu-id="1cc37-112">Následující kód generuje CS0313:</span><span class="sxs-lookup"><span data-stu-id="1cc37-112">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
