## <a name="to-correct-this-error"></a><span data-ttu-id="f4563-108">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="f4563-108">To correct this error</span></span>  
  
1. <span data-ttu-id="f4563-109">Pomocí kódu, který následuje jako příklad, je jedno řešení zadat běžný `ImplStruct` jako první argument typu v volání metody `TestMethod`.</span><span class="sxs-lookup"><span data-stu-id="f4563-109">Using the code that follows as an example, one solution is to specify an ordinary `ImplStruct` as the first type argument in the call to `TestMethod`.</span></span> <span data-ttu-id="f4563-110">Pak změňte `TestMethod` pro vytvoření verze `Implstruct` s možnou hodnotou null v příkazu return:</span><span class="sxs-lookup"><span data-stu-id="f4563-110">Then modify `TestMethod` to create a nullable version of `Implstruct` in its return statement:</span></span>

  
## <a name="example"></a><span data-ttu-id="f4563-111">Příklad</span><span class="sxs-lookup"><span data-stu-id="f4563-111">Example</span></span>

<span data-ttu-id="f4563-112">Následující kód generuje CS0313:</span><span class="sxs-lookup"><span data-stu-id="f4563-112">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
