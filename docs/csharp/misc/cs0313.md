# <a name="compiler-error-cs0313"></a>Chyba kompilátoru CS0313
Typ typ1 nelze použít jako parametr typu parametru name v obecném typu nebo metodě ' typ2 '. Typ s možnou hodnotou null ' typ1 ' nesplňuje omezení ' typ2 '. Typy s možnou hodnotou null nemohou vyhovět žádným omezením rozhraní.  
  
 Typ hodnoty s možnou hodnotou null není ekvivalentní ke svému protějšku, který neumožňuje hodnotu null. V následujícím příkladu `ImplStruct` splňuje omezení `BaseInterface`, ale `ImplStruct?` není, protože `Nullable<ImplStruct>` neimplementuje `BaseInterface`.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Pomocí kódu, který následuje jako příklad, je jedno řešení zadat běžný `ImplStruct` jako první argument typu v volání metody `TestMethod`. Pak změňte `TestMethod` pro vytvoření verze `Implstruct` s možnou hodnotou null v příkazu return:  
  
    ```csharp  
    return new Nullable<T>(t);  
    ```  
  
## <a name="example"></a>Příklad

Následující kód generuje CS0313:  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
