## <a name="to-correct-this-error"></a><span data-ttu-id="68a40-108">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="68a40-108">To correct this error</span></span>  
  
1. <span data-ttu-id="68a40-109">Pomocí kódu, který následuje jako příklad, je jedno řešení určeno `ImplStruct` jako první argument typu v `TestMethod`volání.</span><span class="sxs-lookup"><span data-stu-id="68a40-109">Using the code that follows as an example, one solution is to specify an ordinary `ImplStruct` as the first type argument in the call to `TestMethod`.</span></span> <span data-ttu-id="68a40-110">Pak upravte `TestMethod` , aby se `Implstruct` v návratovém příkazu vytvořila verze s možnou hodnotou null:</span><span class="sxs-lookup"><span data-stu-id="68a40-110">Then modify `TestMethod` to create a nullable version of `Implstruct` in its return statement:</span></span>  
  
    ```csharp  
    return new Nullable<T>(t);  
    ```  
  
## <a name="example"></a><span data-ttu-id="68a40-111">Příklad</span><span class="sxs-lookup"><span data-stu-id="68a40-111">Example</span></span>  
 <span data-ttu-id="68a40-112">Následující kód generuje CS0313:</span><span class="sxs-lookup"><span data-stu-id="68a40-112">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
