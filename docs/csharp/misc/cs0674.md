<span data-ttu-id="ffbf3-103">Nepoužívejte "System.ParamArrayAttribute".</span><span class="sxs-lookup"><span data-stu-id="ffbf3-103">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="ffbf3-104">Místo toho použijte klíčové slovo 'params'.</span><span class="sxs-lookup"><span data-stu-id="ffbf3-104">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="ffbf3-105">C# Kompilátoru neumožňuje použití <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; použijte [params](../../csharp/language-reference/keywords/params.md) místo.</span><span class="sxs-lookup"><span data-stu-id="ffbf3-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../../csharp/language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="ffbf3-106">Následující ukázka generuje CS0674:</span><span class="sxs-lookup"><span data-stu-id="ffbf3-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
