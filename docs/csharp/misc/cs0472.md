<span data-ttu-id="dd25b-103">Výsledek výrazu je vždy "hodnota1" protože se nikdy nerovná hodnotě 'null' typu 'hodnota3' Hodnota typu "hodnota2"</span><span class="sxs-lookup"><span data-stu-id="dd25b-103">The result of the expression is always 'value1' since a value of type 'value2' is never equal to 'null' of type 'value3'</span></span>

## <a name="example"></a><span data-ttu-id="dd25b-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="dd25b-105">Example</span></span>  
 <span data-ttu-id="dd25b-106">Následující ukázka generuje CS0472.</span><span class="sxs-lookup"><span data-stu-id="dd25b-106">The following sample generates CS0472.</span></span>  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;   
        return counter;  
    }  
}  
```
