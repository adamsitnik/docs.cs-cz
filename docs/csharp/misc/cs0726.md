# <a name="compiler-error-cs0726"></a>Compiler Error CS0726
"specifikátor formátu" není platný specifikátor formátu  
  
 K této chybě dochází v ladicím programu. Když zadáte název proměnné do jednoho okna ladicího programu, můžete postupovat podle čárku a specifikátor formátu. Mezi příklady patří: `myInt, h` nebo `myString,nq`. Tato chyba nastane, když kompilátor nemůže rozpoznat [specifikátory formátu v C# ](/visualstudio/debugger/format-specifiers-in-csharp).
