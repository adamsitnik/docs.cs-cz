# <a name="compiler-error-cs1662"></a><span data-ttu-id="9a8ec-102">CS1662 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="9a8ec-102">Compiler Error CS1662</span></span>

<span data-ttu-id="9a8ec-103">Blok anonymní metody na delegáta typu typ delegáta', protože některé typy vrácených hodnot v bloku nejsou implicitně převeditelné na návratový typ delegáta nelze převést.</span><span class="sxs-lookup"><span data-stu-id="9a8ec-103">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>

<span data-ttu-id="9a8ec-104">K této chybě dochází, pokud blok anonymní metody návratový typ, který nebyl implicitně převeditelné na návratový typ delegáta.</span><span class="sxs-lookup"><span data-stu-id="9a8ec-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>

<span data-ttu-id="9a8ec-105">Následující ukázka generuje CS1662:</span><span class="sxs-lookup"><span data-stu-id="9a8ec-105">The following sample generates CS1662:</span></span>

```csharp
// CS1662.cs

delegate int MyDelegate(int i);

class C
{

  public static void Main()
  {
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662
     // Try this instead:
     // MyDelegate d = delegate(int i) { return (int)1.0; };
  }
}
```
