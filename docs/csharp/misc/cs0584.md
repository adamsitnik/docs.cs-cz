# <a name="compiler-error-cs0584"></a>Chyba kompilátoru CS0584

Vnitřní chyba kompilátoru: fáze ' Stage ' symbol ' symbol '
  
 Pokuste se zjistit, zda se kompilátor nezdařil z důvodu neschopnosti analyzovat neočekávanou syntaxi. Pokud tomu tak není, zkuste [nás kontaktovat](/visualstudio/ide/talk-to-us).

> [!NOTE]
> Tato chyba kompilátoru se už v Roslyn nepoužívá.
