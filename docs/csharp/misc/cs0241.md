# <a name="compiler-error-cs0241"></a>CS0241 chyby kompilátoru
Výchozí parametr specifikátory nejsou povolené.  
  
 [Parametry metody](../../csharp/language-reference/keywords/method-parameters.md) nemůže mít výchozí hodnoty. Použijte přetížení metody, pokud k dosažení stejného výsledku. Další informace najdete v tématu [předání parametrů](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0241. Kromě toho vzorek ukazuje, jak simulovat pomocí přetížení metody s výchozími argumenty.  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {   
      B x = new B();  
      x.Test();  
   }  
}  
```
