# <a name="compiler-error-cs0209"></a>Compiler Error CS0209
Typ lokální proměnná deklarovaná v příkazu fixed musí být typu ukazatel.  
  
 Proměnná deklarovaná v [oprava příkazu](../../csharp/language-reference/keywords/fixed-statement.md) musí být ukazatel. Další informace najdete v tématu [nezabezpečený kód a ukazatele](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Následující ukázka generuje CS0209:  
  
```csharp  
// CS0209.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0209  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```
