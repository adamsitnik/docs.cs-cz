# <a name="compiler-error-cs1556"></a>Chyba kompilátoru CS1556
klíčové slovo CONSTRUCT zadané pro metodu Main musí být platná třída nebo struktura.  
  
 Možnost kompilátoru [/Main](../language-reference/compiler-options/main-compiler-option.md) byla předána identifikátoru, který nebyl názvem třídy.
