# <a name="compiler-error-cs1955"></a>Compiler Error CS1955
Nevyvolatelného člena 'name' nelze použít jako metodu.  
  
 Lze vyvolat pouze metod a delegátů. Tato chyba se vygeneruje, když se pokusíte použít prázdných kulatých závorek k volání něco jiného než metody nebo delegáta.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte závorky z výrazu.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje CS1955 kód se pokouší o vyvolání pole a vlastnosti pomocí operátoru volání metody [()](../../csharp/language-reference/operators/invocation-operator.md). Pole nebo vlastnost nelze volat, ale můžete přistupovat k hodnotě uloží s použitím operátor přístupu členů ( [.](../../csharp/language-reference/operators/member-access-operator.md) ).  
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
