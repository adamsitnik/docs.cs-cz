# <a name="compiler-warning-level-1-cs3004"></a>Upozornění kompilátoru (úroveň 1) CS3004
Smíšené a rozložené znaky Unicode nejsou kompatibilní se specifikací CLS.  
  
 Pouze složené znaky Unicode jsou povoleny ve [veřejných](../language-reference/keywords/public.md), [chráněných](../language-reference/keywords/protected.md)nebo `protected internal` identifikátorech, aby byly kompatibilní se specifikací CLS (Common Language Specification). Další informace o kompatibilitě se specifikací CLS naleznete v tématu [nezávislost jazyků a jazykově nezávislé komponenty](../../standard/language-independence.md).
