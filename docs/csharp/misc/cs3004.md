# <a name="compiler-warning-level-1-cs3004"></a>Kompilátor CS3004 upozornění (úroveň 1)
Smíšené a rozložené znaky Unicode nejsou kompatibilní se Specifikací CLS  
  
 Jsou povoleny pouze složené znaky znakové sady UNICODE v [veřejné](../../csharp/language-reference/keywords/public.md), [chráněné](../../csharp/language-reference/keywords/protected.md), nebo `protected internal` identifikátory splňuje požadavky s specifikace CLS (Common Language). Další informace o dodržování specifikace CLS naleznete v tématu [jazyková nezávislost a jazykově nezávislé komponenty](../../standard/language-independence.md).
