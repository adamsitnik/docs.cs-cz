# <a name="compiler-error-cs0541"></a>Compiler Error CS0541
"deklarace": explicitní deklaraci rozhraní lze deklarovat pouze ve třídě nebo struktuře  
  
 Explicitní [rozhraní](../../csharp/language-reference/keywords/interface.md) deklarace byl nalezen mimo [třídy](../../csharp/language-reference/keywords/class.md) nebo [struktura](../../csharp/language-reference/keywords/struct.md).  
  
 Následující ukázka generuje CS0541:  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```
