# <a name="compiler-warning-level-3-cs0219"></a>Kompilátor CS0219 upozornění (úroveň 3)
Proměnné 'variable' je přiřazena, ale jeho hodnota se nikdy nepoužívá.  
  
 Kompilátor vyvolá upozornění úrovně 3, při deklaraci a přiřazení proměnné, ale nepoužívejte ho.  
  
 Následující ukázka generuje CS0219:  
  
```csharp  
// CS0219.cs  
// compile with: /W:3  
public class MyClass  
{  
   public static void Main()  
   {  
      int a = 0;   // CS0219  
   }  
}  
```
