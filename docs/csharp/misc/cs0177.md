 <span data-ttu-id="34446-104">Na parametr označený klíčovým slovem [out](../language-reference/keywords/out-parameter-modifier.md) se nepřiřadila hodnota v těle metody.</span><span class="sxs-lookup"><span data-stu-id="34446-104">A parameter marked with the [out](../language-reference/keywords/out-parameter-modifier.md) keyword was not assigned a value in the method body.</span></span> <span data-ttu-id="34446-105">Další informace najdete v tématu [předávání parametrů](../programming-guide/classes-and-structs/passing-parameters.md) .</span><span class="sxs-lookup"><span data-stu-id="34446-105">For more information, see [Passing Parameters](../programming-guide/classes-and-structs/passing-parameters.md)</span></span>
