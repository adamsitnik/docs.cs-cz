# <a name="compiler-error-cs0594"></a>Compiler Error CS0594
Konstanty s plovoucí desetinnou čárkou je mimo rozsah typu 'type'  
  
 Hodnota byla přiřazena k proměnné s plovoucí desetinnou čárkou, která je příliš velká pro tento datový typ proměnné. Zobrazit [integrální typy tabulky](../../csharp/language-reference/builtin-types/integral-numeric-types.md) informace o rozsahu hodnot povoleno v datové typy.  
  
 Následující ukázka generuje CS0594:  
  
```csharp  
// CS0594.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         float f = 6.77777777777E400;   // CS0594, value too large  
      }  
   }  
}  
```
