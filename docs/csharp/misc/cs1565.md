# <a name="compiler-error-cs1565"></a>Chyba kompilátoru CS1565
Jsou zadané konfliktní možnosti: Soubor prostředků Win32; Ikona Win32  
  
 Není platný pro určení jak možnost kompilátoru [parametr/win32res](../language-reference/compiler-options/win32res-compiler-option.md) , tak i možnost kompilátoru [/win32ico](../language-reference/compiler-options/win32icon-compiler-option.md) ve stejné kompilaci.
