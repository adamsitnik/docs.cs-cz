# <a name="compiler-error-cs1565"></a>Chyba kompilátoru CS1565
Jsou zadané konfliktní možnosti: soubor prostředků Win32; Ikona Win32  
  
 Není platný pro určení možnosti kompilátoru [-win32res](../language-reference/compiler-options/win32res-compiler-option.md) a možnost kompilátoru [-win32ico](../language-reference/compiler-options/win32icon-compiler-option.md) ve stejné kompilaci.
