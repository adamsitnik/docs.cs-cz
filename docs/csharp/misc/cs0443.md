# <a name="compiler-error-cs0443"></a>Compiler Error CS0443
Chyba syntaxe, byla očekávána hodnota  
  
 K této chybě dochází, když odkazujete pole bez zadání hodnoty indexu pole.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje CS0443.  
  
```csharp  
// CS0443.cs   
using System;   
class MyClass   
{  
    public static void Main()      
    {  
        int[,] x = new int[1,5];  
        if (x[] == 5) {} // CS0443  
        // if (x[0, 0] == 5) {}   
    }  
}  
```
