# <a name="compiler-error-cs0068"></a><span data-ttu-id="c0242-102">Chyba kompilátoru CS0068</span><span class="sxs-lookup"><span data-stu-id="c0242-102">Compiler Error CS0068</span></span>
<span data-ttu-id="c0242-103">' Event ': událost v rozhraní nemůže mít inicializátor.</span><span class="sxs-lookup"><span data-stu-id="c0242-103">'event': event in interface cannot have initializer</span></span>  
  
 <span data-ttu-id="c0242-104">Událost v rozhraní nemůže mít inicializátor.</span><span class="sxs-lookup"><span data-stu-id="c0242-104">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="c0242-105">Další informace naleznete v tématu [rozhraní](../programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="c0242-105">For more information, see [Interfaces](../programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="c0242-106">Následující ukázka generuje CS0068:</span><span class="sxs-lookup"><span data-stu-id="c0242-106">The following sample generates CS0068:</span></span>  
  
```csharp  
// CS0068.cs  
  
delegate void MyDelegate();  
  
interface I  
{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068  
   // try the following line instead  
   // event MyDelegate d2;  
}  
  
class M  
{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
