# <a name="compiler-error-cs0068"></a>CS0068 chyby kompilátoru
'událost': událost v rozhraní nemůže mít inicializátor.  
  
 Událost v rozhraní nemůže mít inicializátor. Další informace najdete v tématu [rozhraní](../../csharp/programming-guide/interfaces/index.md).  
  
 Následující ukázka generuje CS0068:  
  
```csharp  
// CS0068.cs  
  
delegate void MyDelegate();  
  
interface I  
{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068  
   // try the following line instead  
   // event MyDelegate d2;  
}  
  
class M  
{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
