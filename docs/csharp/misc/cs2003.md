# <a name="compiler-error-cs2003"></a>CS2003 chyby kompilátoru
Soubor odpovědí 'file' byl zahrnut několikrát  
  
 Soubor odpovědí byl předán kompilátoru více než jednou. Soubor odpovědí může být pouze předán kompilátoru jednou za výstupní soubor.  
  
 Další informace o souborech odpovědí najdete v tématu [@ (určení souboru odezvy)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md).
