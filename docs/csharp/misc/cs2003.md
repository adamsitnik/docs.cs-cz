# <a name="compiler-error-cs2003"></a>Chyba kompilátoru CS2003
Soubor odpovědi soubor byl zahrnut několikrát.  
  
 Do kompilátoru byl předán soubor odpovědí více než jednou. Soubor odpovědi může být do kompilátoru předán pouze jednou pro výstupní soubor.  
  
 Další informace o souborech odpovědí naleznete v tématu [@ (určení souboru odezvy)](../language-reference/compiler-options/response-file-compiler-option.md).
