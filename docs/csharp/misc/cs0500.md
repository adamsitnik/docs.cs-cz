# <a name="compiler-error-cs0500"></a>Compiler Error CS0500
'člena třídy' nemůže deklarovat tělo, protože je označené jako abstraktní  
  
 [Abstraktní](../../csharp/language-reference/keywords/abstract.md) metoda nemůže obsahovat jeho implementace.  
  
 Následující ukázka generuje CS0500:  
  
```csharp  
// CS0500.cs  
namespace x  
{  
   abstract public class clx  
   {  
      abstract public void f(){}   // CS0500  
      // try the following line instead  
      // abstract public void f();  
   }  
  
   public class cly  
   {  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
