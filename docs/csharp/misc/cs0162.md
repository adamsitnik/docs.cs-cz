# <a name="compiler-warning-level-2-cs0162"></a>Kompilátor CS0162 upozornění (úroveň 2)
Zjistil se nedosažitelný kód  
  
 Kompilátor zjistil kód, který se nikdy proveden.  
  
 Následující ukázka generuje CS0162:  
  
```csharp  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```
