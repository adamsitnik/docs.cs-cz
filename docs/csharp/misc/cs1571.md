# <a name="compiler-warning-level-2-cs1571"></a>Kompilátor CS1571 upozornění (úroveň 2)
Komentář XML v 'konstrukce' obsahuje duplicitní značku param pro 'parametru'  
  
 Při použití [/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md) – možnost kompilátoru, pro stejný parametr metody nebyly nalezeny více komentářů. Odeberte jeden z duplicitních řádků.  
  
 Následující ukázka generuje CS1571:  
  
```csharp  
// CS1571.cs  
// compile with: /W:2 /doc:x.xml  
  
/// <summary>help text</summary>  
public class MyClass  
{  
   /// <param name='Int1'>Used to indicate status.</param>  
   /// <param name='Char1'>An initial.</param>  
   /// <param name='Int1'>Used to indicate status.</param> // CS1571  
   public static void MyMethod(int Int1, char Char1)  
   {  
   }  
  
   /// <summary>help text</summary>  
   public static void Main ()  
   {  
   }  
}  
```
