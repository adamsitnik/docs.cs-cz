# <a name="compiler-error-cs0077"></a><span data-ttu-id="da5f4-102">Chyba kompilátoru CS0077</span><span class="sxs-lookup"><span data-stu-id="da5f4-102">Compiler Error CS0077</span></span>
<span data-ttu-id="da5f4-103">Operátor as se musí používat s odkazovým typem nebo typem s možnou hodnotou null (int je typ hodnoty, která není null).</span><span class="sxs-lookup"><span data-stu-id="da5f4-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="da5f4-104">Operátor [as](../language-reference/operators/type-testing-and-cast.md#as-operator) byl předán jako [typ hodnoty](../language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="da5f4-104">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="da5f4-105">Vzhledem k tomu, že `as` může vracet [hodnotu null](../language-reference/keywords/null.md), může být pouze předán pouze [odkazový typ](../language-reference/keywords/reference-types.md) nebo [typ hodnoty s možnou hodnotou null](../language-reference/builtin-types/nullable-value-types.md).</span><span class="sxs-lookup"><span data-stu-id="da5f4-105">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed a [reference type](../language-reference/keywords/reference-types.md) or a [nullable value type](../language-reference/builtin-types/nullable-value-types.md).</span></span>
  
 <span data-ttu-id="da5f4-106">Následující ukázka generuje CS0077:</span><span class="sxs-lookup"><span data-stu-id="da5f4-106">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
