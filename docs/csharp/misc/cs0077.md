# <a name="compiler-error-cs0077"></a>Compiler Error CS0077
As – operátor musí být použit s typem odkazu nebo typ připouštějící hodnotu Null ('int' je typ hodnotu Null).  
  
 [Jako](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator) operátor byl předán [typ hodnoty](../../csharp/language-reference/keywords/value-types.md). Protože `as` může vrátit [null](../../csharp/language-reference/keywords/null.md), lze pouze předat [referenční typy](../../csharp/language-reference/keywords/reference-types.md) nebo typ připouštějící hodnotu Null. Další informace o typech s povolenou hodnotou Null, naleznete v tématu [typy připouštějící hodnotu Null](../../csharp/programming-guide/nullable-types/index.md).  
  
 Následující ukázka generuje CS0077:  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
