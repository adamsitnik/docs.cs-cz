# <a name="compiler-error-cs0077"></a>Chyba kompilátoru CS0077
Operátor as se musí používat s odkazovým typem nebo typem s možnou hodnotou null (int je typ hodnoty, která není null).  
  
 Operátor [as](../language-reference/operators/type-testing-and-cast.md#as-operator) byl předán jako [typ hodnoty](../language-reference/keywords/value-types.md). Protože `as` může vracet [hodnotu null](../language-reference/keywords/null.md), může být pouze předána [odkazovým typům](../language-reference/keywords/reference-types.md) nebo typům s možnou hodnotou null. Další informace o typech hodnot s možnou hodnotou null naleznete v tématu [typy hodnot s možnou hodnotou null](../programming-guide/nullable-types/index.md).
  
 Následující ukázka generuje CS0077:  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
