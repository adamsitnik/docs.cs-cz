# <a name="compiler-error-cs0077"></a><span data-ttu-id="d707b-102">Chyba kompilátoru CS0077</span><span class="sxs-lookup"><span data-stu-id="d707b-102">Compiler Error CS0077</span></span>
<span data-ttu-id="d707b-103">Operátor as se musí používat s odkazovým typem nebo typem s možnou hodnotou null (int je typ hodnoty, která není null).</span><span class="sxs-lookup"><span data-stu-id="d707b-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="d707b-104">Operátor [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) byl předán jako [typ hodnoty](../../csharp/language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="d707b-104">The [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="d707b-105">Vzhledem `as` k tomu, že může vracet [hodnotu null](../../csharp/language-reference/keywords/null.md), může být pouze předána [odkazovým typům](../../csharp/language-reference/keywords/reference-types.md) nebo typem Nullable.</span><span class="sxs-lookup"><span data-stu-id="d707b-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="d707b-106">Další informace o typech Nullable naleznete v tématu [typy s možnou hodnotou null](../../csharp/programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="d707b-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="d707b-107">Následující ukázka generuje CS0077:</span><span class="sxs-lookup"><span data-stu-id="d707b-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
