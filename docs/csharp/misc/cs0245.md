# <a name="compiler-error-cs0245"></a>Compiler Error CS0245
Destruktory a objekt. Dokončení nelze volat přímo. Zvažte možnost volání metody IDisposable.Dispose, pokud je k dispozici.  
  
 Další informace najdete v tématu [základy programování pro uvolňování paměti](../../standard/garbage-collection/index.md) a [destruktory](../../csharp/programming-guide/classes-and-structs/destructors.md).  
  
 Následující ukázka generuje CS0245:  
  
```csharp  
// CS0245.cs  
using System;  
using System.Collections;  
  
class MyClass // : IDisposable  
{  
   /*  
   public void Dispose()  
   {  
      // cleanup code goes here  
   }  
   */  
  
   void m()  
   {  
      this.Finalize();   // CS0245  
      // this.Dispose();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
