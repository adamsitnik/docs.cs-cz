# <a name="compiler-error-cs2013"></a>Chyba kompilátoru CS2013
Neplatné základní číslo obrázku Value  
  
 Možnosti kompilátoru [/BaseAddress](../language-reference/compiler-options/baseaddress-compiler-option.md) byla předána neplatná hodnota (není číslo).  
  
 Následující ukázka generuje CS2013:  
  
```csharp  
// CS2013.cs  
// compile with: /target:library /baseaddress:x  
// CS2013 expected  
class MyClass  
{  
}  
```
