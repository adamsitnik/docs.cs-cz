# <a name="compiler-error-cs2013"></a>CS2013 chyby kompilátoru
Neplatné základní číslo obrázku 'value'  
  
 Byla předána neplatná hodnota (není číslo) [/BaseAddress](../../csharp/language-reference/compiler-options/baseaddress-compiler-option.md) – možnost kompilátoru.  
  
 Následující ukázka generuje CS2013:  
  
```csharp  
// CS2013.cs  
// compile with: /target:library /baseaddress:x  
// CS2013 expected  
class MyClass  
{  
}  
```
