# <a name="compiler-error-cs0449"></a><span data-ttu-id="0d2d1-102">Compiler Error CS0449</span><span class="sxs-lookup"><span data-stu-id="0d2d1-102">Compiler Error CS0449</span></span>
<span data-ttu-id="0d2d1-103">Omezení 'class' nebo 'struct' musí být uvedena před všemi ostatními omezeními.</span><span class="sxs-lookup"><span data-stu-id="0d2d1-103">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="0d2d1-104">Omezení u obecného typu nebo metody parametru typu se musí vyskytovat v určitém pořadí: `class` nebo `struct` musí být první, pokud jsou k dispozici, pak žádným omezením rozhraní a nakonec omezeními konstruktoru.</span><span class="sxs-lookup"><span data-stu-id="0d2d1-104">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="0d2d1-105">Tato chyba je způsobená `class` nebo `struct` omezení nezobrazují první.</span><span class="sxs-lookup"><span data-stu-id="0d2d1-105">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="0d2d1-106">Chcete-li vyřešit tuto chybu, změnit pořadí klauzulí omezení.</span><span class="sxs-lookup"><span data-stu-id="0d2d1-106">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0d2d1-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="0d2d1-107">Example</span></span>  
 <span data-ttu-id="0d2d1-108">Následující ukázka generuje CS0449.</span><span class="sxs-lookup"><span data-stu-id="0d2d1-108">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4   
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
