# <a name="compiler-error-cs0035"></a><span data-ttu-id="6d92a-102">CS0035 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="6d92a-102">Compiler Error CS0035</span></span>
<span data-ttu-id="6d92a-103">Operátor 'operator' je nejednoznačný na operandu typu 'type'</span><span class="sxs-lookup"><span data-stu-id="6d92a-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
<span data-ttu-id="6d92a-104">Kompilátor má více než jeden dostupný převodu a neví, kterého mohou vybírat před použitím operátoru.</span><span class="sxs-lookup"><span data-stu-id="6d92a-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span>
  
<span data-ttu-id="6d92a-105">Následující ukázka generuje CS0035:</span><span class="sxs-lookup"><span data-stu-id="6d92a-105">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
