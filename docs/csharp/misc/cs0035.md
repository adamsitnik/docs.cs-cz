# <a name="compiler-error-cs0035"></a><span data-ttu-id="33117-102">CS0035 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="33117-102">Compiler Error CS0035</span></span>
<span data-ttu-id="33117-103">Operátor 'operator' je nejednoznačný na operandu typu 'type'</span><span class="sxs-lookup"><span data-stu-id="33117-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
 <span data-ttu-id="33117-104">Kompilátor má více než jeden dostupný převodu a neví, kterého mohou vybírat před použitím operátoru.</span><span class="sxs-lookup"><span data-stu-id="33117-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span> <span data-ttu-id="33117-105">Další informace najdete v tématu [bez vizuálního vzhledu uživatele definované převody](/cpp/misc/templated-user-defined-conversions) a [operátory převodu](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="33117-105">For more information, see [Templated User Defined Conversions](/cpp/misc/templated-user-defined-conversions) and [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="33117-106">Následující ukázka generuje CS0035:</span><span class="sxs-lookup"><span data-stu-id="33117-106">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
