# <a name="compiler-error-cs0035"></a>CS0035 chyby kompilátoru
Operátor 'operator' je nejednoznačný na operandu typu 'type'  
  
 Kompilátor má více než jeden dostupný převodu a neví, kterého mohou vybírat před použitím operátoru. Další informace najdete v tématu [bez vizuálního vzhledu uživatele definované převody](/cpp/misc/templated-user-defined-conversions) a [operátory převodu](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).  
  
 Následující ukázka generuje CS0035:  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
