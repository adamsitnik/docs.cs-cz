# <a name="compiler-error-cs1503"></a>Compiler Error CS1503
Argument 'number' nelze převést z TypeA TypeB  
  
 Typ jednoho z argumentů v metodě neodpovídá typu, který byl předán, pokud byla vytvořena instance třídy. Tato chyba obvykle se zobrazí spolu s CS1502. Zobrazit [CS1502](../../csharp/language-reference/compiler-messages/cs1502.md) diskuzi o tom, jak vyřešit tuto chybu.
