# <a name="compiler-error-cs1503"></a>Chyba kompilátoru CS1503
Argument ' Number ' nemůže být převeden z typu ' na TypeB  
  
 Typ jednoho argumentu v metodě se neshoduje s typem, který byl předán při vytváření instance třídy. Tato chyba se obvykle objevuje společně s CS1502. Diskuzi o tom, jak tuto chybu vyřešit, najdete v tématu [CS1502](../language-reference/compiler-messages/cs1502.md) .
