# <a name="compiler-error-cs0196"></a>Compiler Error CS0196
Ukazatel může být indexován jenom jednu hodnotu  
  
 Ukazatel nemůže mít více indexů. Další informace najdete v tématu [typy ukazatelů](../../csharp/programming-guide/unsafe-code-pointers/pointer-types.md)  
  
 Následující ukázka generuje CS0196:  
  
```csharp  
// CS0196.cs  
public class MyClass  
{  
   public static void Main ()  
   {  
      int *i = null;  
      int j = 0;  
      j = i[1,2];   // CS0196  
      // try the following line instead  
      // j = i[1];  
   }  
}  
```
