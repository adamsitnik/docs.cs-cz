# <a name="compiler-error-cs0569"></a>CS0569 chyby kompilátoru
"method2": "– metoda1" nelze přepsat, protože není jazykem podporován  
  
 Tato chyba nastane, pokud odvozujete od základní třídu, která byla zapsána v jiném jazyce a kompilátor nepodporuje metodu, která se pokoušíte přepsat.
