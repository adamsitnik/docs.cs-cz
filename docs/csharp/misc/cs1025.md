# <a name="compiler-error-cs1025"></a>Compiler Error CS1025
Jednořádkový komentář nebo ukončení řádku. byl očekáván  
  
 Řádek s [direktivy preprocesoru](../../csharp/language-reference/preprocessor-directives/index.md) nemůže mít Víceřádkové komentáře.  
  
 Následující ukázka generuje CS1025:  
  
```csharp  
#if true /* hello  
*/   // CS1025  
#endif   // this is a good comment  
```  
  
 CS1025 by se mohl vyskytnout, pokud se pokusíte některé Neplatná direktiva preprocesoru, následujícím způsobem:  
  
```csharp  
// CS1025.cs  
#define a  
  
class Sample  
{  
   static void Main()  
   {  
      #if a 1   // CS1025, invalid syntax  
         System.Console.WriteLine("Hello, World!");  
      #endif  
   }  
}  
```
