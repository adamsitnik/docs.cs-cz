## <a name="example"></a><span data-ttu-id="cde40-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="cde40-110">Example</span></span>  
 <span data-ttu-id="cde40-111">Následující kód vygeneruje CS0736, protože `Program.testMethod` je deklarováno jako statické:</span><span class="sxs-lookup"><span data-stu-id="cde40-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>

```csharp  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
