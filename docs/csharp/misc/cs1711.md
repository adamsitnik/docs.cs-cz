# <a name="compiler-warning-level-2-cs1711"></a>Kompilátor CS1711 upozornění (úroveň 2)
Komentář XML na 'type' má značku typeparam pro 'parametru', ale neexistuje parametr typu s tímto názvem.  
  
 Dokumentace ke službě obecného typu obsahuje značku pro parametr typu, který má nesprávný název.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje varování CS1711.  
  
```csharp  
// cs1711.cs  
// compile with: /doc:cs1711.xml  
// CS1711 expected  
using System;  
///<typeparam name="WrongName">can be an int</typeparam>  
class CMain  
{  
    public static void Main() { }  
}  
```
