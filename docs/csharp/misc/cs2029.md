# <a name="compiler-warning-level-1-cs2029"></a>Kompilátor CS2029 upozornění (úroveň 1)
Neplatná hodnota pro '/ define'; 'identifier' není platný identifikátor.  
  
 K tomuto upozornění dochází, pokud hodnota, která se používá v [/ define](../../csharp/language-reference/compiler-options/define-compiler-option.md) možnost obsahuje neplatné znaky.  
  
 Nelze toto upozornění potlačit [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) možnost.
