# <a name="compiler-warning-level-1-cs2029"></a>Upozornění kompilátoru (úroveň 1) CS2029
Neplatná hodnota pro '/define '; identifikátor není platným identifikátorem.  
  
 K tomuto upozornění dochází, pokud hodnota použitá v možnosti [/define](../language-reference/compiler-options/define-compiler-option.md) obsahuje neplatné znaky.  
  
 Toto upozornění nelze potlačit pomocí možnosti [/nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) .
