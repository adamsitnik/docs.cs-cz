# <a name="compiler-error-cs0037"></a>Compiler Error CS0037
Nelze převést na 'typu', protože je typ hodnot neumožňující hodnotu null  
  
 Kompilátor nelze přiřadit hodnotu null na typ hodnoty; Hodnota Null se dá přiřadit jenom k [odkazovat na typ](../../csharp/language-reference/keywords/reference-types.md) nebo typ připouštějící hodnotu Null. [Struktura](../../csharp/language-reference/keywords/struct.md) je typ hodnoty. Další informace najdete v tématu [typy připouštějící hodnotu Null](../../csharp/programming-guide/nullable-types/index.md).  
  
 Následující ukázka generuje CS0037:  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
