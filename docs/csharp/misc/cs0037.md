# <a name="compiler-error-cs0037"></a>Chyba kompilátoru CS0037
Hodnotu null nelze převést na typ, protože se jedná o typ, který nepovoluje hodnotu null.  
  
 Kompilátor nemůže přiřadit hodnotu null typu hodnoty; hodnotu null lze přiřadit pouze [typu odkazu](../language-reference/keywords/reference-types.md) nebo typu s možnou hodnotou null. [Struktura](../language-reference/keywords/struct.md) je typ hodnoty. Další informace naleznete v tématu [typy s možnou hodnotou null](../programming-guide/nullable-types/index.md).  
  
 Následující ukázka generuje CS0037:  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
