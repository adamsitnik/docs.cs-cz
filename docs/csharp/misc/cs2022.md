# <a name="compiler-error-cs2022"></a>CS2022 chyby kompilátoru
Přepínače ' / out' a '/ target' musí být uvedena před názvy zdrojových souborů  
  
 [/Out (nastavení název výstupního souboru)](../../csharp/language-reference/compiler-options/out-compiler-option.md) a [/Target (určení formátu výstupního souboru)](../../csharp/language-reference/compiler-options/target-compiler-option.md) možnosti kompilátoru, pokud zadaný na příkazovém řádku, musí předcházet soubory zdrojového kódu.
