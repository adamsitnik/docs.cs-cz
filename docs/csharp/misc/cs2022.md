# <a name="compiler-error-cs2022"></a>Chyba kompilátoru CS2022
Možnosti '/out ' a '/Target ' musí být uvedeny před názvy zdrojových souborů  
  
 Možnosti kompilátoru [/out (nastavit název výstupního souboru)](../language-reference/compiler-options/out-compiler-option.md) a [/target (určení formátu výstupního souboru)](../language-reference/compiler-options/target-compiler-option.md) , jsou-li zadány na příkazovém řádku, musí předcházet soubory zdrojového kódu.
