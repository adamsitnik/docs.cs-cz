# <a name="compiler-error-cs2022"></a>Chyba kompilátoru CS2022
Možnosti '/out ' a '/Target ' musí být uvedeny před názvy zdrojových souborů  
  
 Možnost [-out (nastavit výstupní název souboru)](../language-reference/compiler-options/out-compiler-option.md) a [-target (určení formátu výstupního souboru)](../language-reference/compiler-options/target-compiler-option.md) možnosti kompilátoru, pokud jsou zadány na příkazovém řádku, musí předcházet soubory zdrojového kódu.
