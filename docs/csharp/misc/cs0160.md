<span data-ttu-id="9f723-103">Předchozí klauzule catch už zachytává všechny výjimky vyvolávané tímto nebo super typu (typ)</span><span class="sxs-lookup"><span data-stu-id="9f723-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="9f723-104">Řada [catch](../../csharp/language-reference/keywords/try-catch.md) příkazy musí být v sestupném pořadí podle odvození.</span><span class="sxs-lookup"><span data-stu-id="9f723-104">A series of [catch](../../csharp/language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="9f723-105">Například nejvíce odvozené objekty musí být uvedena první.</span><span class="sxs-lookup"><span data-stu-id="9f723-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="9f723-106">Další informace najdete v tématu [výjimek a zpracování výjimek](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="9f723-106">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="9f723-107">Následující ukázka generuje CS0160:</span><span class="sxs-lookup"><span data-stu-id="9f723-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
