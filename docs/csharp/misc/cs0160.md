# <a name="compiler-error-cs0160"></a>Compiler Error CS0160
Předchozí klauzule catch už zachytává všechny výjimky vyvolávané tímto nebo super typu (typ)  
  
Řada [catch](../../csharp/language-reference/keywords/try-catch.md) příkazy musí být v sestupném pořadí podle odvození. Například nejvíce odvozené objekty musí být uvedena první.
  
 Další informace najdete v tématu [výjimek a zpracování výjimek](../../csharp/programming-guide/exceptions/index.md).  
  
 Následující ukázka generuje CS0160:  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
