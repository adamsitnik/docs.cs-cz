# <a name="compiler-warning-level-3-cs0642"></a>Kompilátor CS0642 upozornění (úroveň 3)
Možná chybný prázdný příkaz  
  
 Středník po podmíněném příkazu může způsobit, že váš kód běžel odlišně, než bylo zamýšleno.  
  
 Můžete použít **/nowarn** – možnost kompilátoru nebo `#pragmas warning` zakázat toto upozornění, najdete v článku [/nowarn (C# – možnosti kompilátoru)](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) nebo [varování #pragma](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) Další informace informace.  
  
 Následující ukázka generuje CS0642:  
  
```csharp  
// CS0642.cs  
// compile with: /W:3  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
