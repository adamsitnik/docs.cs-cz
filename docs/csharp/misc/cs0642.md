 <span data-ttu-id="73691-106">Následující ukázka generuje CS0642:</span><span class="sxs-lookup"><span data-stu-id="73691-106">The following sample generates CS0642:</span></span>

```csharp  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?

         System.Console.WriteLine (i);  
      }  
   }  
}  
```
