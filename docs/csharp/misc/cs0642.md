# <a name="compiler-warning-level-3-cs0642"></a>Upozornění kompilátoru (úroveň 3) CS0642
Možná chybný prázdný příkaz  
  
 Středník po podmíněném příkazu může způsobit, že se váš kód spustí jinak, než je určeno.  
  
 Můžete použít možnost kompilátoru **/nowarn** `#pragmas warning` nebo zakázat toto upozornění; Další informace naleznete v tématu [/nowarn (C# možnosti kompilátoru)](../language-reference/compiler-options/nowarn-compiler-option.md) nebo [upozornění #pragma](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) .  
  
 Následující ukázka generuje CS0642:  
  
```csharp  
// CS0642.cs  
// compile with: /W:3  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
