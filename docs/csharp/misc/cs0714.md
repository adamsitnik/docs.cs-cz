# <a name="compiler-error-cs0714"></a>Compiler Error CS0714
'statického typu': statické třídy nemůžou implementovat rozhraní  
  
 Rozhraní může definovat nestatických metod na objekty a proto nemusí být implementováno statické třídy. Chcete-li tuto chybu vyřešit, ujistěte se, že vaše třída nebude pokoušet o implementaci všem rozhraním.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0714:  
  
```csharp  
// CS0714.cs  
interface I  
{  
}  
  
public static class C : I  // CS0714  
{  
    public static void Main()  
    {  
    }  
}  
```
