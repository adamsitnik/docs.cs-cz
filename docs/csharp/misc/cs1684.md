 <span data-ttu-id="a84fa-104">Tato chyba může být způsobena odkazem v jednom oboru názvů odkazující na typ, který se uvádí, že existuje v rámci druhého oboru názvů, ale typ neexistuje.</span><span class="sxs-lookup"><span data-stu-id="a84fa-104">This error can be caused by a reference inside one namespace referring to a type that it says exists inside a second namespace, but the type does not exist.</span></span> <span data-ttu-id="a84fa-105">Například mydll.dll říká typu `A` existuje v rámci yourdll.dll, ale neexistuje žádný takový typ uvnitř yourdll.dll.</span><span class="sxs-lookup"><span data-stu-id="a84fa-105">For example, mydll.dll says that type `A` exists inside yourdll.dll, but no such type exists inside yourdll.dll.</span></span> <span data-ttu-id="a84fa-106">Jednou z možných příčin této chyby je, že je příliš stará verze yourdll.dll používáte a `A` ještě nebyl definován.</span><span class="sxs-lookup"><span data-stu-id="a84fa-106">One possible cause of this error is that the version of yourdll.dll you are using is too old and `A` has not yet been defined.</span></span>  
  
 <span data-ttu-id="a84fa-107">Následující ukázka generuje CS1684.</span><span class="sxs-lookup"><span data-stu-id="a84fa-107">The following sample generates CS1684.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a84fa-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="a84fa-108">Example</span></span>  
  
```csharp  
// CS1684_a.cs  
// compile with: /target:library /keyfile:CS1684.key  
public class A {  
   public void Test() {}  
}  
  
public class C2 {}  
```  
  
## <a name="example"></a><span data-ttu-id="a84fa-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="a84fa-109">Example</span></span>  
  
```csharp  
// CS1684_b.cs  
// compile with: /target:library /r:cs1684_a.dll  
// post-build command: del /f CS1684_a.dll  
using System;  
public class Ref   
{  
   public static A GetA() { return new A(); }  
   public static C2 GetC() { return new C2(); }  
}  
```
