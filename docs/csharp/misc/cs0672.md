 <span data-ttu-id="a6685-105">Kompilátor najít `override` metodě označené jako `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="a6685-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="a6685-106">Ale přepsání metody nebyl samotné označené jako zastaralé.</span><span class="sxs-lookup"><span data-stu-id="a6685-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="a6685-107">Přepsání metody bude dál generovat [CS0612](../../csharp/misc/cs0612.md), pokud je volána.</span><span class="sxs-lookup"><span data-stu-id="a6685-107">The overriding method will still generate [CS0612](../../csharp/misc/cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="a6685-108">Zkontrolujte vaše deklarace metod a explicitně určit, zda by měla být označena metody (a všechny její přepsání) `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="a6685-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="a6685-109">Následující ukázka generuje CS0672:</span><span class="sxs-lookup"><span data-stu-id="a6685-109">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
