# <a name="compiler-warning-level-1-cs0672"></a><span data-ttu-id="d9cb1-102">Upozornění kompilátoru (úroveň 1) CS0672</span><span class="sxs-lookup"><span data-stu-id="d9cb1-102">Compiler Warning (level 1) CS0672</span></span>
<span data-ttu-id="d9cb1-103">Člen "Člen1" Přepisuje zastaralý člen "member2.</span><span class="sxs-lookup"><span data-stu-id="d9cb1-103">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="d9cb1-104">Přidat zastaralý atribut do "Člen1"</span><span class="sxs-lookup"><span data-stu-id="d9cb1-104">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="d9cb1-105">Kompilátor nalezl výjimku `override` pro metodu označenou jako `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="d9cb1-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="d9cb1-106">Překrytá metoda ale sama o sobě neoznačila jako zastaralou.</span><span class="sxs-lookup"><span data-stu-id="d9cb1-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="d9cb1-107">Přepsání metody bude stále generovat [CS0612](./cs0612.md), pokud je volána.</span><span class="sxs-lookup"><span data-stu-id="d9cb1-107">The overriding method will still generate [CS0612](./cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="d9cb1-108">Zkontrolujte deklarace metod a explicitně určete, zda by měla být označena `obsolete`metoda (a všechna její přepsání).</span><span class="sxs-lookup"><span data-stu-id="d9cb1-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="d9cb1-109">Následující ukázka generuje CS0672:</span><span class="sxs-lookup"><span data-stu-id="d9cb1-109">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
