# <a name="compiler-warning-level-1-cs0672"></a>Kompilátor CS0672 upozornění (úroveň 1)
Člen 'member1' přepisuje zastaralý člen ' člen2. Přidat Obsolete atribut 'member1'  
  
 Kompilátor najít `override` metodě označené jako `obsolete`. Ale přepsání metody nebyl samotné označené jako zastaralé. Přepsání metody bude dál generovat [CS0612](../../csharp/misc/cs0612.md), pokud je volána.  
  
 Zkontrolujte vaše deklarace metod a explicitně určit, zda by měla být označena metody (a všechny její přepsání) `obsolete`.  
  
 Následující ukázka generuje CS0672:  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
