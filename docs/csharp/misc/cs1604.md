# <a name="compiler-error-cs1604"></a>Compiler Error CS1604
Nelze přiřadit 'variable', protože je jen pro čtení  
  
 Přiřazení byla vytvořena na proměnnou jen pro čtení. K této chybě předejít, přiřadit ani pokus o změnu na tuto proměnnou v tomto kontextu.
