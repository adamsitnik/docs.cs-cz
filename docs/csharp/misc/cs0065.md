# <a name="compiler-error-cs0065"></a><span data-ttu-id="a830a-102">CS0065 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="a830a-102">Compiler Error CS0065</span></span>

<span data-ttu-id="a830a-103">'událost': vlastnost události musí obsahovat i přidat nebo odebrat přístupové objekty</span><span class="sxs-lookup"><span data-stu-id="a830a-103">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="a830a-104">Událost, která není pole musí mít obě metody přístupu.</span><span class="sxs-lookup"><span data-stu-id="a830a-104">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="a830a-105">Následující ukázka generuje CS0065:</span><span class="sxs-lookup"><span data-stu-id="a830a-105">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
