# <a name="compiler-error-cs0065"></a><span data-ttu-id="86b6a-102">Chyba kompilátoru CS0065</span><span class="sxs-lookup"><span data-stu-id="86b6a-102">Compiler Error CS0065</span></span>

<span data-ttu-id="86b6a-103">' Event ': vlastnost události musí obsahovat přistupující objekty Add i Remove.</span><span class="sxs-lookup"><span data-stu-id="86b6a-103">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="86b6a-104">Událost, která není pole, musí mít obě metody přístupu.</span><span class="sxs-lookup"><span data-stu-id="86b6a-104">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="86b6a-105">Následující ukázka generuje CS0065:</span><span class="sxs-lookup"><span data-stu-id="86b6a-105">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
