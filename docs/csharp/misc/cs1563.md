# <a name="compiler-error-cs1563"></a>Chyba kompilátoru CS1563
Výstup výstupního souboru nemá žádné zdrojové soubory.  
  
 Byla zadána možnost kompilátoru [-out](../language-reference/compiler-options/out-compiler-option.md) , ale nenásledují soubory zdrojového kódu. Neměli byste použít možnost **/out**, nebo byste měli zadat soubory zdrojového kódu po použití **/out**.
