# <a name="compiler-error-cs1563"></a>Compiler Error CS1563
Výstup 'výstupní soubor' nemá žádné zdrojové soubory  
  
 [/Out](../../csharp/language-reference/compiler-options/out-compiler-option.md) byla zadána možnost kompilátoru, ale podle žádné soubory zdrojového kódu. Neměli byste používat buď **/out**, nebo byste měli zadat soubory zdrojového kódu po **/out**.
