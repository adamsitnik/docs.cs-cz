# <a name="compiler-error-cs0609"></a>Compiler Error CS0609
Nelze nastavit IndexerName atribut označené přepsání  
  
 Atribut názvu (**IndexerNameAttribute**) nejde použít u indexovanou vlastnost, která je přepsání. Další informace najdete v tématu [indexery](../../csharp/programming-guide/indexers/index.md).  
  
 Následující ukázka generuje CS0609:  
  
```csharp  
// CS0609.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public class idx  
{  
   public virtual int this[int iPropIndex]  
   {  
      get  
      {  
         return 0;  
      }  
      set  
      {  
      }  
   }  
}  
  
public class MonthDays : idx  
{  
   [IndexerName("MonthInfoIndexer")]   // CS0609, delete to resolve this CS0609  
   public override int this[int iPropIndex]  
   {  
      get  
      {  
         return 0;  
      }  
      set  
      {  
      }  
   }  
}  
  
public class test  
{  
   public static void Main( string[] args )  
   {  
   }  
}  
```
