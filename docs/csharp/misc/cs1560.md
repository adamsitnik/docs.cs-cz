# <a name="compiler-error-cs1560"></a>Compiler Error CS1560
Neplatný název souboru zadaný pro direktivu preprocesoru. Název souboru je příliš dlouhý nebo není platný název souboru  
  
 Název souboru, který byl zadán s [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) překročil _MAX_PATH (256 znaků) nebo řádek, na které `#line` byl nalezen překročil 2000 znaků.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1560.  
  
```csharp  
// cs1560.cs   
using System;   
class MyClass   
{     
   public static void Main()      
   {        
      Console.WriteLine("Normal line #1.");     
      #line 21 "MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890.txt"   // CS1560  
    }  
}  
```
