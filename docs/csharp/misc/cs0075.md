# <a name="compiler-error-cs0075"></a><span data-ttu-id="2a876-102">Compiler Error CS0075</span><span class="sxs-lookup"><span data-stu-id="2a876-102">Compiler Error CS0075</span></span>
<span data-ttu-id="2a876-103">Přetypovat záporná hodnota, je nutné uzavřít hodnotu v závorkách</span><span class="sxs-lookup"><span data-stu-id="2a876-103">To cast a negative value, you must enclose the value in parentheses</span></span>  
  
 <span data-ttu-id="2a876-104">Pokud jsou přetypování použití klíčového slova, která identifikuje předdefinovaný typ, není nutné závorky.</span><span class="sxs-lookup"><span data-stu-id="2a876-104">If you are casting using a keyword that identifies a predefined type, then you do not need parentheses.</span></span> <span data-ttu-id="2a876-105">V opačném případě je nutné protože umístit závorky (x) – y, se nepovažují za výraz přetypování.</span><span class="sxs-lookup"><span data-stu-id="2a876-105">Otherwise, you must put the parentheses because (x) –y will not be considered a cast expression.</span></span> <span data-ttu-id="2a876-106">Od specifikace jazyka C# části 7.6.6:</span><span class="sxs-lookup"><span data-stu-id="2a876-106">From the C# Specification, Section 7.6.6:</span></span>  
  
 <span data-ttu-id="2a876-107">*Z pravidla mnohoznačnosti vyplývá, že pokud x a y jsou identifikátory (x), y, (x)(y), a (x)(-y) jsou výrazy přetypování, ale (x)-y není, i v případě, že x identifikuje typ. Nicméně pokud x je klíčové slovo, který identifikuje předdefinovaný typ (například int), pak všechny čtyři způsoby jsou výrazy přetypování (protože takové klíčové slovo nebylo možné pravděpodobně výraz samostatně).*</span><span class="sxs-lookup"><span data-stu-id="2a876-107">*From the disambiguation rule it follows that, if x and y are identifiers, (x)y, (x)(y), and (x)(-y) are cast-expressions, but (x)-y is not, even if x identifies a type. However, if x is a keyword that identifies a predefined type (such as int), then all four forms are cast-expressions (because such a keyword could not possibly be an expression by itself).*</span></span>  
  
 <span data-ttu-id="2a876-108">Následující kód vygeneruje CS0075:</span><span class="sxs-lookup"><span data-stu-id="2a876-108">The following code generates CS0075:</span></span>  
  
```csharp  
// CS0075  
namespace MyNamespace  
{  
    enum MyEnum { }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            // To fix the error, place the negative  
            // values below in parentheses  
            int i = (System.Int32) - 4; //CS0075  
            MyEnum e = (MyEnum) - 1;    //CS0075  
            System.Console.WriteLine(i); //to avoid warning  
            System.Console.WriteLine(e); //to avoid warning  
        }  
    }  
}  
```
