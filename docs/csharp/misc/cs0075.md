# <a name="compiler-error-cs0075"></a><span data-ttu-id="4f575-102">Chyba kompilátoru CS0075</span><span class="sxs-lookup"><span data-stu-id="4f575-102">Compiler Error CS0075</span></span>
<span data-ttu-id="4f575-103">Chcete-li přetypovat zápornou hodnotu, je nutné uzavřít hodnotu v závorkách.</span><span class="sxs-lookup"><span data-stu-id="4f575-103">To cast a negative value, you must enclose the value in parentheses</span></span>  
  
 <span data-ttu-id="4f575-104">Pokud předáváte pomocí klíčového slova, které identifikuje předdefinovaný typ, nepotřebujete závorky.</span><span class="sxs-lookup"><span data-stu-id="4f575-104">If you are casting using a keyword that identifies a predefined type, then you do not need parentheses.</span></span> <span data-ttu-id="4f575-105">V opačném případě je nutné umístit závorky, protože (x) – y nebude považován za výraz přetypování.</span><span class="sxs-lookup"><span data-stu-id="4f575-105">Otherwise, you must put the parentheses because (x) –y will not be considered a cast expression.</span></span> <span data-ttu-id="4f575-106">Ve C# specifikaci 7.6.6 oddílu:</span><span class="sxs-lookup"><span data-stu-id="4f575-106">From the C# Specification, Section 7.6.6:</span></span>  
  
 <span data-ttu-id="4f575-107">*Následující pravidlo nejednoznačnosti sleduje, že pokud x a y jsou identifikátory, (x) y, (x) (y) a (x) (-y), jsou přetypování výrazy, ale (x)-y není, i když x identifikuje typ. Pokud je však x klíčové slovo, které identifikuje předdefinovaný typ (například int), pak všechny čtyři formuláře jsou funkce CAST-Expressions (protože takové klíčové slovo by nebylo pravděpodobně výrazem samotného).*</span><span class="sxs-lookup"><span data-stu-id="4f575-107">*From the disambiguation rule it follows that, if x and y are identifiers, (x)y, (x)(y), and (x)(-y) are cast-expressions, but (x)-y is not, even if x identifies a type. However, if x is a keyword that identifies a predefined type (such as int), then all four forms are cast-expressions (because such a keyword could not possibly be an expression by itself).*</span></span>  
  
 <span data-ttu-id="4f575-108">Následující kód generuje CS0075:</span><span class="sxs-lookup"><span data-stu-id="4f575-108">The following code generates CS0075:</span></span>  
  
```csharp  
// CS0075  
namespace MyNamespace  
{  
    enum MyEnum { }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            // To fix the error, place the negative  
            // values below in parentheses  
            int i = (System.Int32) - 4; //CS0075  
            MyEnum e = (MyEnum) - 1;    //CS0075  
            System.Console.WriteLine(i); //to avoid warning  
            System.Console.WriteLine(e); //to avoid warning  
        }  
    }  
}  
```
