# <a name="compiler-error-cs0462"></a><span data-ttu-id="6f9a0-102">Compiler Error CS0462</span><span class="sxs-lookup"><span data-stu-id="6f9a0-102">Compiler Error CS0462</span></span>
<span data-ttu-id="6f9a0-103">Zděděné členy 'member1' a 'členu member2' mají před spuštěním akce stejný podpis v typu 'type', takže je nejde přepsat</span><span class="sxs-lookup"><span data-stu-id="6f9a0-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="6f9a0-104">Tato chyba nastane po zavedení obecných typů.</span><span class="sxs-lookup"><span data-stu-id="6f9a0-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="6f9a0-105">Za normálních okolností nemůžete mít dvě verze metody ve třídě se stejným podpisem.</span><span class="sxs-lookup"><span data-stu-id="6f9a0-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="6f9a0-106">Pomocí obecných typů, můžete ale zadat obecné metody, která může být duplicitní jinou metodu, pokud je vytvořena instance s určitého typu.</span><span class="sxs-lookup"><span data-stu-id="6f9a0-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6f9a0-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="6f9a0-107">Example</span></span>  
 <span data-ttu-id="6f9a0-108">Když `C<int>` je vytvořena instance, dvě verze metody `F` vytvářejí se stejnou signaturou, takže přepsání ve třídě `D` nelze rozhodnout, který z nich se má přepsání vztahovat.</span><span class="sxs-lookup"><span data-stu-id="6f9a0-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="6f9a0-109">Následující ukázka generuje CS0462.</span><span class="sxs-lookup"><span data-stu-id="6f9a0-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>   
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>   
{  
   public override void F(int t) {}   // CS0462  
}  
```
