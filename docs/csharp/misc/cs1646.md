# <a name="compiler-error-cs1646"></a>Compiler Error CS1646
Klíčové slovo, identifikátor nebo po specifikátoru doslovného řetězce: \@  
  
 Řetězcové literály pro používání specifikátoru doslovného naleznete v tématu "\@". Specifikátoru verbatim se povoluje jenom před řetězec, klíčové slovo nebo identifikátor. Chcete-li tuto chybu vyřešit, odeberte symbol z jakékoli nevhodný @ umístit nebo přidat odpovídající řetězec, – klíčové slovo nebo identifikátor.  
  
 Následující ukázka generuje CS1646:  
  
```csharp  
// CS1646  
class C  
{  
   int i = @5;  // CS1646  
   // Try this line instead:  
   // int i = 5;  
}  
```
