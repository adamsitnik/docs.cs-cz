# <a name="compiler-error-cs0837"></a>Compiler Error CS0837
První operand na "je" nebo "jako" operátor nesmí být výraz lambda nebo anonymní metodu.  
  
 Výrazy lambda a anonymní metody nemusí použít na levé straně [je](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#is-operator) nebo [jako](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator).
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pokud chyba zahrnuje `is` operátoru, nezapomeňte, že `is` přebírá hodnotu a typ a zjistíte, zda hodnota mohou být provedeny do typu odkazem, zabalení a rozbalení převodu. Protože výrazy lambda nejsou hodnoty a mít žádný odkaz, zabalení a rozbalení převody, výrazy lambda nejsou kandidáty pro `is`.  
  
- Pokud kód nevhodnému použití `as`, opravy je pravděpodobně ho změnit přetypování.  
  
## <a name="example"></a>Příklad

Následující příklad generuje CS0837:  
  
```csharp  
// cs0837.cs  
namespace TestNamespace  
{  
    public delegate void Del();  
  
    class Test  
    {  
        static int Main()  
        {  
            bool b1 = (() => { }) is Del;   // CS0837  
            bool b2 = delegate() { } is Del;// CS0837  
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837  
            return 1;  
        }  
    }  
}  
```
