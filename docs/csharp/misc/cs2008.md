# <a name="compiler-error-cs2008"></a>CS2008 chyby kompilátoru
Nezadali jste žádné vstupy  
  
 Byl vyvolán kompilátor a byly zadány možnosti kompilátoru, ale byly předány žádné soubory zdrojového kódu.
