# <a name="compiler-error-cs1561"></a>Compiler Error CS1561
Název výstupního souboru je příliš dlouhý nebo neplatný  
  
 Název výstupního souboru nemůže být delší než 256 znaků a nesmí obsahovat neplatné znaky, jako je například symbol eurech, otazník nebo zpětné lomítko, mimo jiné.
