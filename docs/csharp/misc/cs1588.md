# <a name="compiler-error-cs1588"></a>Compiler Error CS1588
Nelze určit common language runtime adresář – 'důvod:  
  
 Kompilátor nemůže načíst adresář, který je nainstalovaný modul common language runtime rozhraní .NET Framework. Tato chyba označuje, že je nesprávně nainstalovaný modul common language runtime.
