## <a name="example"></a><span data-ttu-id="a45a3-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="a45a3-106">Example</span></span>  
 <span data-ttu-id="a45a3-107">Tento příklad ukazuje použití CS0633 <xref:System.Diagnostics.ConditionalAttribute>.</span><span class="sxs-lookup"><span data-stu-id="a45a3-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="a45a3-108">Následující ukázka generuje CS0633.</span><span class="sxs-lookup"><span data-stu-id="a45a3-108">The following sample generates CS0633.</span></span>  
  
```csharp  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="a45a3-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="a45a3-109">Example</span></span>  
 <span data-ttu-id="a45a3-110">Tento příklad ukazuje použití CS0633 <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span><span class="sxs-lookup"><span data-stu-id="a45a3-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
