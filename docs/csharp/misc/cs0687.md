# <a name="compiler-error-cs0687"></a>CS0687 chyby kompilátoru
Kvalifikátor aliasu oboru názvů '::' vždy překládal na typ nebo obor názvů proto není zde. Zvažte použití '.' místo.  
  
 K této chybě dochází, pokud jste použili něco, co analyzátor interpretován jako typ neočekávané místě. Typ nebo obor názvů je platný jenom v výraz přístupu členů pomocí přístup ke členu (**.**) – operátor. Tato situace může nastat, pokud jste použili globální rozsah – operátor (:) v jiném kontextu.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0687:  
  
```csharp  
// CS0687.cs  
  
using M = Test;  
using System;  
  
public class Test   
{  
    public static int x = 77;  
  
    public static void Main()   
    {  
        Console.WriteLine(M::x); // CS0687  
        // To resolve use the following line instead:  
        // Console.WriteLine(M.x);  
    }  
}  
```
