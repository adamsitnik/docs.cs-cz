 <span data-ttu-id="504dc-105">Tato chyba je generována, pokud je definováno více implementací metody dotazu a kompilátor nemůže určit, který z nich je nejvhodnější použít pro dotaz.</span><span class="sxs-lookup"><span data-stu-id="504dc-105">This error is generated when multiple implementations of a query method are defined and the compiler cannot disambiguate which one is best to use for the query.</span></span> <span data-ttu-id="504dc-106">V následujícím příkladu obě verze `Select` mají stejný podpis, protože oba akceptují jeden `int` jako vstupní parametr a mají `int` jako návratovou hodnotu.</span><span class="sxs-lookup"><span data-stu-id="504dc-106">In the following example, both versions of `Select` have the same signature, because they both accept one `int` as an input parameter and have `int` as a return value.</span></span>

## <a name="example"></a><span data-ttu-id="504dc-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="504dc-109">Example</span></span>  
 <span data-ttu-id="504dc-110">Následující kód generuje CS1940:</span><span class="sxs-lookup"><span data-stu-id="504dc-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```
