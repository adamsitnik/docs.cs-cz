# <a name="compiler-error-cs0573"></a>Compiler Error CS0573
'pole deklaraci': nelze mít Inicializátory polí instance ve strukturách  
  
 Nelze inicializovat pole instance [struktura](../../csharp/language-reference/keywords/struct.md). Pole typů hodnot se inicializuje na výchozí hodnoty a pole typu odkazu bude inicializována tak `null`.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0573:  
  
```csharp  
// CS0573.cs  
namespace x  
{  
    public class clx  
    {  
        public static void Main()  
        {  
        }  
    }  
  
    public struct cly  
    {  
        clx a = new clx();   // CS0573  
        // clx a;            // OK  
        int i = 7;           // CS0573  
        // int i;            // OK  
    }  
}  
```
