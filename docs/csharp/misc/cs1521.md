# <a name="compiler-error-cs1521"></a>Compiler Error CS1521
Neplatný základní typ  
  
 A [základní](../../csharp/language-reference/keywords/base.md) třídy specifikace byl chybně vytvořen.  
  
 Následující ukázka generuje CS1521:  
  
```csharp  
// CS1521.cs  
class CMyClass  
{  
   public static void Main()  
   {  
   }  
}  
  
class CMyClass1 : CMyClass     // OK  
{  
}  
  
class CMyClass2 : CMyClass[]   // CS1521  
{  
}  
  
class CMyClass3 : CMyClass*    // CS1521  
{  
}  
```
