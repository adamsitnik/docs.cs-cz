# <a name="compiler-error-cs1101"></a><span data-ttu-id="a1028-102">Chyba kompilátoru CS1101</span><span class="sxs-lookup"><span data-stu-id="a1028-102">Compiler Error CS1101</span></span>
<span data-ttu-id="a1028-103">Modifikátor parametru REF nelze použít s parametrem this.</span><span class="sxs-lookup"><span data-stu-id="a1028-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="a1028-104">`this` Když klíčové slovo změní první parametr statické metody, signalizuje kompilátoru, že metoda je metoda rozšíření.</span><span class="sxs-lookup"><span data-stu-id="a1028-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="a1028-105">V prvním parametru metody rozšíření nejsou vyžadovány ani povoleny žádné jiné modifikátory.</span><span class="sxs-lookup"><span data-stu-id="a1028-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a1028-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="a1028-106">Example</span></span>  
 <span data-ttu-id="a1028-107">Následující příklad generuje CS1101:</span><span class="sxs-lookup"><span data-stu-id="a1028-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
