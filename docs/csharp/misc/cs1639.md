# <a name="compiler-error-cs1639"></a>Compiler Error CS1639
Podpis třídy obálky coclass spravované 'podpis"rozhraní"rozhraní"není platný podpis názvu třídy  
  
 K této chybě dochází, pokud jsou neplatná metadata pro rozhraní. To by nikdy neměly vzniknout Pokud rozhraní se vygeneroval pomocí C# nebo podporovaný jazyk .NET. Obraťte se na dodavatele rozhraní nebo ověřte, že sestavení rozhraní se úspěšně vygeneroval.
