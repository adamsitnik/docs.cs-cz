# <a name="compiler-error-cs0432"></a><span data-ttu-id="e7715-102">Compiler Error CS0432</span><span class="sxs-lookup"><span data-stu-id="e7715-102">Compiler Error CS0432</span></span>
<span data-ttu-id="e7715-103">Alias 'identifier' nebyl nalezen</span><span class="sxs-lookup"><span data-stu-id="e7715-103">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="e7715-104">K této chybě dochází, když použijete "::" napravo od identifikátor, který není alias.</span><span class="sxs-lookup"><span data-stu-id="e7715-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="e7715-105">Chcete-li chybu vyřešit, použijte "." místo.</span><span class="sxs-lookup"><span data-stu-id="e7715-105">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="e7715-106">Následující příklad generuje CS0432:</span><span class="sxs-lookup"><span data-stu-id="e7715-106">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
