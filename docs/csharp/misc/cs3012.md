# <a name="compiler-warning-level-1-cs3012"></a>Upozornění kompilátoru (úroveň 1) CS3012

U modulu, který se liší od atributu CLSCompliant sestavení, nelze zadat atribut CLSCompliant

Aby modul byl kompatibilní se specifikací CLS (Common Language Specification) `[module:System.CLSCompliant(true)]`, musí být sestaven pomocí možnosti [/target: Module](../language-reference/compiler-options/target-module-compiler-option.md) kompilátoru. Další informace o specifikaci CLS naleznete v tématu [nezávislost jazyka a jazykově nezávislé komponenty](../../standard/language-independence-and-language-independent-components.md).

## <a name="example"></a>Příklad

Následující příklad, pokud je sestaven bez `/target:module`, generuje CS3012:

```csharp
// CS3012.cs
// compile with: /W:1

[module:System.CLSCompliant(true)]   // CS3012
public class C
{
    public static void Main()
    {
    }
}
```
