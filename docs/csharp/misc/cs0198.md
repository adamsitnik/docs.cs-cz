 <span data-ttu-id="23e1e-104">Proměnná [jen pro čtení](../language-reference/keywords/readonly.md) musí mít stejné [statické](../language-reference/keywords/static.md) použití jako konstruktor, ve kterém ji chcete inicializovat.</span><span class="sxs-lookup"><span data-stu-id="23e1e-104">A [readonly](../language-reference/keywords/readonly.md) variable must have the same [static](../language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="23e1e-105">Další informace naleznete v tématu [statické konstruktory](../programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="23e1e-105">For more information, see [Static Constructors](../programming-guide/classes-and-structs/static-constructors.md).</span></span>

```csharp

{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
