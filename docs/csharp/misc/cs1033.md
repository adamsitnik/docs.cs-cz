# <a name="compiler-error-cs1033"></a>Compiler Error CS1033
Zdrojový soubor byl překročen limit 16,707,565 řádků, které v souboru .pdb; ladicí informace budou nesprávné  
  
 Soubor zdrojového kódu překročil maximální povolený počet řádků, které kompilátor může zpracovat. Chcete-li vyřešit tuto chybu, vytvořte dva nebo více souborů zdrojového kódu z původního souboru. Maximální počet řádků, které je 268,435,454 řádky. Pokud používáte **/debug**, pomocí více než 16,707,566 bude mít za následek poškozená ladicí informace.
