# <a name="compiler-warning-level-2-cs1698"></a>Kompilátor CS1698 upozornění (úroveň 2)
Cyklický odkaz na sestavení "AssemblyName1" neodpovídá názvu sestavení výstupu "AssemblyName2". Zkuste přidání odkazu na "AssemblyName1" nebo změnit název výstupu sestavení tak, aby odpovídaly.  
  
 CS1698 nastane, pokud odkaz na sestavení je nesprávný. To může nastat, když je přepsán v odkazovaném sestavení. Pokud chcete vyřešit, proveďte ne nahradit sestavení, která je závislost, kterou je odkazováno na sestavení.  
  
## <a name="example"></a>Příklad  
  
```csharp  
// CS1698_a.cs  
// compile with: /target:library /keyfile:mykey.snk  
[assembly:System.Reflection.AssemblyVersion("2")]  
public class CS1698_a {}  
```  
  
## <a name="example"></a>Příklad  
  
```csharp  
// CS1698_b.cs  
// compile with: /target:library /reference:CS1698_a.dll /keyfile:mykey.snk  
public class CS1698_b : CS1698_a {}  
```  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1698.  
  
```csharp  
// CS1698_c.cs  
// compile with: /target:library /out:cs1698_a.dll /reference:cs1698_b.dll /keyfile:mykey.snk  
// CS1698 expected  
[assembly:System.Reflection.AssemblyVersion("3")]  
public class CS1698_c : CS1698_b {}  
public class CS1698_a {}  
```
