# <a name="compiler-error-cs1017"></a>Compiler Error CS1017
Klauzule catch nemůže následovat za obecnou klauzulí catch příkazu try.  
  
 A `catch` blok, který nepřijímá žádné parametry musí být poslední v řadě `catch` bloky. Další informace o výjimkách, naleznete v tématu [výjimek a zpracování výjimek](../../csharp/programming-guide/exceptions/index.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1017:  
  
```csharp  
// CS1017.cs  
using System;  
  
namespace x  
{  
    public class b : Exception  
    {  
    }  
  
    public class a  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
  
            catch   // CS1017, must be last catch  
            {  
            }  
  
            catch(b)  
            {  
                throw;  
            }  
        }  
    }  
}  
```
