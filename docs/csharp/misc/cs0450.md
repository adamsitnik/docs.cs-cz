# <a name="compiler-error-cs0450"></a><span data-ttu-id="345e6-102">Compiler Error CS0450</span><span class="sxs-lookup"><span data-stu-id="345e6-102">Compiler Error CS0450</span></span>
<span data-ttu-id="345e6-103">'Zadejte název parametru': nelze zadat třídu omezení a omezení 'class' nebo 'struct'</span><span class="sxs-lookup"><span data-stu-id="345e6-103">'Type Parameter Name': cannot specify both a constraint class and the 'class' or 'struct' constraint</span></span>  
  
 <span data-ttu-id="345e6-104">Pokud je parametr typu je omezená omezení typu Struktura, je logicky odporuje, aby se také omezovat určitý typ třídy, protože struktury a třídy jsou vzájemně se vylučující kategorií.</span><span class="sxs-lookup"><span data-stu-id="345e6-104">If a type parameter is constrained by the struct type constraint, it is logically contradictory for it to also be constrained by a specific class type, since struct and class are mutually exclusive categories.</span></span> <span data-ttu-id="345e6-105">Pokud je parametr typu je omezená specifické omezení typu třídy, pak je podle definice omezeny omezení typu třídy a tak určení omezení typu třídy je redundantní.</span><span class="sxs-lookup"><span data-stu-id="345e6-105">If a type parameter is constrained by a specific class type constraint, then it is by definition constrained by the class type constraint, and so specifying the class type constraint is redundant.</span></span>  
  
## <a name="example"></a><span data-ttu-id="345e6-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="345e6-106">Example</span></span>  
  
```csharp  
// CS0450.cs  
// compile with: /t:library  
public class GenericsErrors   
{  
    public class B { }  
    public class G3<T> where T : struct, B { } // CS0450  
// To resolve, use the following line instead:  
// public class G3<T> where T : B { }  
}  
```
