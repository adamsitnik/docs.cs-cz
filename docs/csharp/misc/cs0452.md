# <a name="compiler-error-cs0452"></a><span data-ttu-id="1ec83-102">Chyba kompilátoru CS0452</span><span class="sxs-lookup"><span data-stu-id="1ec83-102">Compiler Error CS0452</span></span>
<span data-ttu-id="1ec83-103">Typ type name musí být odkazovým typem, aby ho bylo možné použít jako parametr názvu parametru v obecném typu nebo metodě ' identifikátoru obecného '.</span><span class="sxs-lookup"><span data-stu-id="1ec83-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>  
  
 <span data-ttu-id="1ec83-104">K této chybě dochází, Pokud předáte typ hodnoty, jako `struct` je `int` například nebo jako parametr pro obecný typ nebo metodu, která má omezení typu odkazu.</span><span class="sxs-lookup"><span data-stu-id="1ec83-104">This error occurs when you pass a value type such as a `struct` or `int` as a parameter to a generic type or method that has a reference type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1ec83-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="1ec83-105">Example</span></span>
