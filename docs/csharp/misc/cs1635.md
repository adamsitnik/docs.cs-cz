# <a name="compiler-warning-level-1-cs1635"></a><span data-ttu-id="2d852-102">Upozornění kompilátoru (úroveň 1) CS1635</span><span class="sxs-lookup"><span data-stu-id="2d852-102">Compiler Warning (level 1) CS1635</span></span>
<span data-ttu-id="2d852-103">Upozornění na kód upozornění se nedá obnovit, protože je globálně zakázané.</span><span class="sxs-lookup"><span data-stu-id="2d852-103">Cannot restore warning 'warning code' because it was disabled globally</span></span>  
  
 <span data-ttu-id="2d852-104">K tomuto upozornění dochází, pokud použijete možnost příkazového řádku **/nowarn** nebo nastavení projektu pro vypnutí upozornění pro celou jednotku kompilace, ale použijete `#pragma warning restore` ji k pokusu o obnovení tohoto upozornění.</span><span class="sxs-lookup"><span data-stu-id="2d852-104">This warning occurs if you use the **/nowarn** command line option or project setting to disable a warning for the entire compilation unit, but you use `#pragma warning restore` to attempt to restore that warning.</span></span> <span data-ttu-id="2d852-105">Chcete-li tuto chybu vyřešit, odeberte možnost příkazového řádku/nowarn nebo nastavení projektu nebo odeberte `#pragma warning restore` všechna upozornění, která zakážete prostřednictvím příkazového řádku nebo nastavení projektu.</span><span class="sxs-lookup"><span data-stu-id="2d852-105">To resolve this error, remove the /nowarn command line option or project setting, or remove the `#pragma warning restore` for any warnings you are disabling via the command line or project settings.</span></span> <span data-ttu-id="2d852-106">Další informace najdete v tématu [#pragma upozornění](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) .</span><span class="sxs-lookup"><span data-stu-id="2d852-106">For more information, see the [#pragma warning](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) topic.</span></span>  
  
 <span data-ttu-id="2d852-107">Následující ukázka generuje CS1635:</span><span class="sxs-lookup"><span data-stu-id="2d852-107">The following sample generates CS1635:</span></span>  
  
```csharp  
// CS1635.cs  
// compile with: /w:1 /nowarn:162  
  
enum MyEnum {one=1,two=2,three=3};  
  
class MyClass  
{  
    public static void Main()  
    {  
#pragma warning disable 162  
  
    if (MyEnum.three == MyEnum.two)  
        System.Console.WriteLine("Duplicate");  
  
#pragma warning restore 162  
    }  
}  
```
