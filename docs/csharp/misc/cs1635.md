# <a name="compiler-warning-level-1-cs1635"></a>Kompilátor CS1635 upozornění (úroveň 1)
Kód upozornění"upozornění" nelze obnovit, protože bylo globálně zakázané.  
  
 K tomuto upozornění dochází, pokud použijete **/nowarn** příkazového řádku možnost nebo nastavení zakázat upozornění pro celý kompilační jednotky projektu, ale použít `#pragma warning restore` pokus o obnovení tohoto upozornění. Chcete-li vyřešit tuto chybu, odebrání/nowarn – možnost příkazového řádku nebo nastavení projektu nebo odebrání `#pragma warning restore` pro všechna upozornění Chystáte se zakázat prostřednictvím příkazového řádku nebo nastavení projektu. Další informace najdete v tématu [varování #pragma](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) tématu.  
  
 Následující ukázka generuje CS1635:  
  
```csharp  
// CS1635.cs  
// compile with: /w:1 /nowarn:162  
  
enum MyEnum {one=1,two=2,three=3};  
  
class MyClass  
{  
    public static void Main()  
    {  
#pragma warning disable 162  
  
    if (MyEnum.three == MyEnum.two)  
        System.Console.WriteLine("Duplicate");  
  
#pragma warning restore 162  
    }  
}  
```
