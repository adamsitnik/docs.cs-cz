# <a name="compiler-error-cs0744"></a><span data-ttu-id="d20b3-102">Chyba kompilátoru CS0744</span><span class="sxs-lookup"><span data-stu-id="d20b3-102">Compiler Error CS0744</span></span>
<span data-ttu-id="d20b3-103">Očekávalo se kontextové klíčové slovo Equals.</span><span class="sxs-lookup"><span data-stu-id="d20b3-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="d20b3-104">Vzor klauzule `join` je `join`...`in`...`on`...`equals`, jak je znázorněno v následujícím příkladu:</span><span class="sxs-lookup"><span data-stu-id="d20b3-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

```csharp  
            join y in array2 on x equals y  
            select x;  
```

## <a name="example"></a><span data-ttu-id="d20b3-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="d20b3-107">Example</span></span>  
 <span data-ttu-id="d20b3-108">Následující kód generuje CS0744:</span><span class="sxs-lookup"><span data-stu-id="d20b3-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
