# <a name="compiler-warning-level-3-cs0169"></a>Kompilátor CS0169 upozornění (úroveň 3)
Soukromé pole 'člena třídy' se nikdy nepoužívá.  
  
 Privátní proměnná byla deklarovaná, ale nikdy odkazována. Běžný způsob, jak generovat toto upozornění je při deklarování soukromý člen třídy a nepoužívejte ho.  
  
 Následující ukázka generuje CS0169:  
  
```csharp  
// compile with: /W:3  
using System;  
public class ClassX  
{  
   int i;   // CS0169, i is not used anywhere  
  
   public static void Main()  
   {  
   }  
}  
```
