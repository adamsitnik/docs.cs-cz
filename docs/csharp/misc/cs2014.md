# <a name="compiler-warning-level-1-cs2014"></a>Upozornění kompilátoru (úroveň 1) CS2014
Možnost kompilátoru Old Option je zastaralá. místo toho použijte možnost New Option.  
  
 Forma možnosti kompilátoru je zastaralá. Další informace najdete v tématu [ C# možnosti kompilátoru](../language-reference/compiler-options/index.md) .
