# <a name="compiler-error-cs1032"></a>Compiler Error CS1032
Nelze definovat/nedefinované symboly preprocesoru po prvním tokenu v souboru  
  
 `#define` a `#undef` [direktivy preprocesoru](../../csharp/language-reference/preprocessor-directives/index.md) musí být použita na začátku programu, než všechny ostatní klíčová slova, jako jsou ty používané v deklaraci oboru názvů.  
  
 Následující ukázka generuje CS1032:  
  
```csharp  
// CS1032.cs  
namespace x  
{  
   public class clx  
   {  
      #define a   // CS1032, put before namespace  
      public static void Main()  
      {  
      }  
   }  
}  
```
