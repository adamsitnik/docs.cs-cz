# <a name="compiler-error-cs1606"></a>Compiler Error CS1606
Podpis sestavení se nezdařil; výstup nemusí být podepsán--důvod  
  
 Sestavení se vytvořil, ale když kompilátor se pokusil dokončit podepisování, došlo k chybě.
