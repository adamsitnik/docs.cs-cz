# <a name="compiler-error-cs0577"></a>Compiler Error CS0577
Atribut Conditional není platný na 'function', protože se jedná o konstruktor, destruktor, operátor nebo explicitní implementace rozhraní  
  
 `Conditional` nejde použít u určených metod.  
  
 Například nelze použít některé atributy na definici explicitního rozhraní. Následující ukázka generuje CS0577:  
  
```csharp  
// CS0577.cs  
// compile with: /target:library  
interface I  
{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
