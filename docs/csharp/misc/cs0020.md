# <a name="compiler-error-cs0020"></a>Chyba kompilátoru CS0020
Dělení nulovou konstantou  
  
 Výraz používá v jmenovateli operace dělení hodnotu literálu (ne proměnné) nula. Dělení nulou není definováno a je tedy neplatné.  
  
 Následující ukázka generuje CS0020:  
  
```csharp  
// CS0020.cs  
namespace x  
{  
   public class b  
   {  
      public static void Main()  
      {  
         1 / 0;   // CS0020  
      }  
   }  
}  
```
