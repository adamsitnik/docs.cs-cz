# <a name="compiler-error-cs1520"></a><span data-ttu-id="d5656-102">Chyba kompilátoru CS1520</span><span class="sxs-lookup"><span data-stu-id="d5656-102">Compiler Error CS1520</span></span>
<span data-ttu-id="d5656-103">Metoda musí mít návratový typ.</span><span class="sxs-lookup"><span data-stu-id="d5656-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="d5656-104">Metoda, která je deklarována ve třídě, struktuře nebo rozhraní, musí mít explicitní návratový typ.</span><span class="sxs-lookup"><span data-stu-id="d5656-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="d5656-105">V následujícím příkladu má čtvercová metoda návratovou hodnotu typu [String](../language-reference/keywords/string.md):</span><span class="sxs-lookup"><span data-stu-id="d5656-105">In the following example, the Square method has a return value of [string](../language-reference/keywords/string.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="d5656-106">Následující ukázka generuje CS1520:</span><span class="sxs-lookup"><span data-stu-id="d5656-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="d5656-107">Alternativně se tato chyba může vyskytnout, pokud se případ názvu konstruktoru liší od deklarace třídy nebo struktury, jako v následující ukázce.</span><span class="sxs-lookup"><span data-stu-id="d5656-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="d5656-108">Vzhledem k tomu, že název není přesně stejný jako název třídy, kompilátor ho interpretuje jako regulární metodu, nikoli konstruktor a vytvoří chybu:</span><span class="sxs-lookup"><span data-stu-id="d5656-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
