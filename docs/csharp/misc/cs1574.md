# <a name="compiler-warning-level-1-cs1574"></a>Kompilátor CS1574 upozornění (úroveň 1)
Komentář XML v 'konstrukce' obsahuje syntakticky nesprávný atribut 'name' cref  
  
 Předán řetězec cref značku, například v rámci \<výjimky > značky, označuje člena, který není k dispozici v aktuálním prostředí sestavení. Řetězec, který předáte cref značky musí být syntakticky správný název člena nebo pole.  
  
 Další informace najdete v tématu [doporučené značky pro dokumentační komentáře](../../csharp/programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).  
  
 Následující ukázka generuje CS1574:  
  
```csharp  
// CS1574.cs  
// compile with: /W:1 /doc:x.xml  
using System;  
  
/// <exception cref="System.Console.WriteLin">An exception class.</exception>   // CS1574  
// instead, uncomment and try the following line  
// /// <exception cref="System.Console.WriteLine">An exception class.</exception>  
class EClass : Exception  
{  
}  
  
class TestClass  
{  
   public static void Main()  
   {  
      try  
      {  
      }  
      catch(EClass)  
      {  
      }  
   }  
}  
```
