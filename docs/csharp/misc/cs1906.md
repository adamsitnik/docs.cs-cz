# <a name="compiler-error-cs1906"></a>Chyba kompilátoru CS1906
Neplatná možnost Option; Viditelnost prostředku musí být buď Public, nebo Private.  
  
 Tato chyba označuje neplatný parametr [/Resource (vložit soubor prostředků do výstupu)](../language-reference/compiler-options/resource-compiler-option.md) nebo [/linkresource (odkaz na .NET Framework prostředek)](../language-reference/compiler-options/linkresource-compiler-option.md) příkazového řádku. Zkontrolujte syntaxi možnosti příkazového řádku **/Resource** nebo **/linkresource** a ujistěte se, že použitý modifikátor přístupnosti je buď **veřejný** , nebo `private`.
