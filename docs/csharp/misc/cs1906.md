# <a name="compiler-error-cs1906"></a>Compiler Error CS1906
Neplatná možnost: '; Viditelnost prostředků musí být 'public' nebo 'private'  
  
 Tato chyba označuje neplatnou [/Resource (vložení zdrojového souboru pro výstup)](../../csharp/language-reference/compiler-options/resource-compiler-option.md) nebo [/linkresource (odkaz na prostředek rozhraní .NET Framework)](../../csharp/language-reference/compiler-options/linkresource-compiler-option.md) možnost příkazového řádku. Zkontrolujte syntaxi **/Resource** nebo **/linkresource** příkazového řádku a ujistěte se, že je použit modifikátor dostupnosti buď **veřejné** nebo `private`.
