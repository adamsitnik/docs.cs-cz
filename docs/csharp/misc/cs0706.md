# <a name="compiler-error-cs0706"></a>Compiler Error CS0706
Neplatné omezení typu. Typ použitý jako omezení musí být rozhraní, nezapečetěná třída nebo parametr typu.  
  
 Tato chyba nastane, pokud neplatná konstrukce použita ve klauzule omezení. Lze vyvarovat této chyby, použijte rozhraní nebo nezapečetěné třídy místo konstrukci, která způsobila chybu.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0706.  
  
```csharp  
// CS0706.cs  
// compile with: /target:library  
class A {}  
class C<T> where T : int[] {}  // CS0706  
class D<T> where T : A {}  // OK  
```
