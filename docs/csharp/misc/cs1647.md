# <a name="compiler-error-cs1647"></a>Compiler Error CS1647
Výraz je moc dlouhý nebo složitý ke kompilaci téměř "kód"  
  
 V kompilátoru zpracování kódu došlo k přetečení zásobníku. Chcete-li vyřešit tuto chybu, Zjednodušte váš kód. Pokud váš kód je platný, získat nápovědu v tématu [komunity vývojářů](https://developercommunity.visualstudio.com/spaces/61/index.html) lokality nebo [StackOverflow](https://stackoverflow.com/).
