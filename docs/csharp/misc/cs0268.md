# <a name="compiler-error-cs0268"></a>CS0268 chyby kompilátoru
Importovaný typ 'type' je neplatný. Obsahuje cyklickou závislost základních tříd.  
  
 K této chybě dochází, pokud má typ, importovaný z jiného jazyka cyklickou závislost základních tříd. Takový typ nelze použít v C# programu. Chcete-li tuto chybu vyřešit, zkontrolujte importovaných z jiných jazyků ve všech odkazovaných sestavení nebo modulů typů.
