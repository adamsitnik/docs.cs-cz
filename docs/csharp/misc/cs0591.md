# <a name="compiler-error-cs0591"></a>Compiler Error CS0591
Neplatná hodnota pro argument pro atribut' attribute '.  
  
 Atribut byl předán neplatný argument nebo dva argumenty vzájemně vylučují.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0591:  
  
```csharp  
// CS0591.cs  
using System;  
  
[AttributeUsage(0)]   // CS0591  
class I: Attribute  
{  
}  
  
public class a  
{  
    public static void Main()  
    {  
    }  
}  
```
