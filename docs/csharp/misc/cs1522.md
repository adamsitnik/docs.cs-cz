# <a name="compiler-warning-level-1-cs1522"></a>Kompilátor CS1522 upozornění (úroveň 1)
Prázdný blok switch  
  
 Kompilátor zjistil [přepnout](../../csharp/language-reference/keywords/switch.md) bloku bez **případ** nebo **výchozí** příkazu. A `switch` bloku musí mít jeden nebo více **případ** nebo **výchozí** příkazy.  
  
 Následující ukázka generuje CS1522:  
  
```csharp  
// CS1522.cs  
// compile with: /W:1  
using System;  
class x  
{  
   public static void Main()  
   {  
      int i = 6;  
  
      switch(i)   // CS1522  
      {  
         // add something to the switch block, for example:  
         /*  
         case (5):  
            Console.WriteLine("5");  
            return;  
         default:  
            Console.WriteLine("not 5");  
            return;  
         */  
      }  
   }  
}  
```
