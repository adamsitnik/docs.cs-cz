<span data-ttu-id="91f56-103">Prázdný blok switch</span><span class="sxs-lookup"><span data-stu-id="91f56-103">Empty switch block</span></span>  
  
 <span data-ttu-id="91f56-104">Kompilátor zjistil [přepnout](../../csharp/language-reference/keywords/switch.md) bloku bez **případ** nebo **výchozí** příkazu.</span><span class="sxs-lookup"><span data-stu-id="91f56-104">The compiler detected a [switch](../../csharp/language-reference/keywords/switch.md) block with no **case** or **default** statement.</span></span> <span data-ttu-id="91f56-105">A `switch` bloku musí mít jeden nebo více **případ** nebo **výchozí** příkazy.</span><span class="sxs-lookup"><span data-stu-id="91f56-105">A `switch` block must have one or more **case** or **default** statements.</span></span>

```csharp

class x  
{  
   public static void Main()  
   {  
      int i = 6;  
  
      switch(i)   // CS1522  
      {  
         // add something to the switch block, for example:  
         /*  
         case (5):  
            Console.WriteLine("5");  
            return;  
         default:  
            Console.WriteLine("not 5");  
            return;  
         */  
      }  
   }  
}  
```
