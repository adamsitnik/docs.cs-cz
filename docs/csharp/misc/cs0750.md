 <span data-ttu-id="fc313-104">Z důvodu jejich zvláštní chování částečné metody jsou v souladu s časovým omezením modifikátory, který může přijmout.</span><span class="sxs-lookup"><span data-stu-id="fc313-104">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>

1. <span data-ttu-id="fc313-106">Odebrání modifikátoru neoprávněné deklarace metody.</span><span class="sxs-lookup"><span data-stu-id="fc313-106">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fc313-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="fc313-107">Example</span></span>  
 <span data-ttu-id="fc313-108">Následující příklad generuje CS0750:</span><span class="sxs-lookup"><span data-stu-id="fc313-108">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
