# <a name="compiler-error-cs1541"></a>Compiler Error CS1541
Neplatný parametr reference: 'symbol': nelze odkazovat na adresáře  
  
 Kompilátor zjistil pokus o určit adresář, nikoli konkrétní soubor. Například při použití [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) – možnost kompilátoru, je nutné zadat soubor; není možné určit adresář.  
  
 Například předávání `/reference:c:\` kompilátoru vygeneruje CS1541.
