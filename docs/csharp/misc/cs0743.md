# <a name="compiler-error-cs0743"></a><span data-ttu-id="79133-102">Compiler Error CS0743</span><span class="sxs-lookup"><span data-stu-id="79133-102">Compiler Error CS0743</span></span>

 <span data-ttu-id="79133-104">Vzor pro `join` je klauzule `join`... `in`... `on`... `equals`, jak je znázorněno v tomto příkladu:</span><span class="sxs-lookup"><span data-stu-id="79133-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

## <a name="example"></a><span data-ttu-id="79133-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="79133-107">Example</span></span>  
 <span data-ttu-id="79133-108">Následující kód vygeneruje CS0743:</span><span class="sxs-lookup"><span data-stu-id="79133-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
