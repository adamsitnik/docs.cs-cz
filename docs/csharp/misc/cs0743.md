# <a name="compiler-error-cs0743"></a>Chyba kompilátoru CS0743
Očekávalo se kontextové klíčové slovo on.  
  
 Vzor `join` klauzule je `join`... `in`... `on`... `equals`, jak je znázorněno v následujícím příkladu:  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Přidejte klíčové slovo do `join`klauzule. `on`  
  
## <a name="example"></a>Příklad  
 Následující kód generuje CS0743:  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
