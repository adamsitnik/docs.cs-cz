# <a name="compiler-error-cs0743"></a><span data-ttu-id="3b512-102">Chyba kompilátoru CS0743</span><span class="sxs-lookup"><span data-stu-id="3b512-102">Compiler Error CS0743</span></span>
<span data-ttu-id="3b512-103">Očekávalo se kontextové klíčové slovo on.</span><span class="sxs-lookup"><span data-stu-id="3b512-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="3b512-104">Vzor klauzule `join` je `join`...`in`...`on`...`equals`, jak je znázorněno v následujícím příkladu:</span><span class="sxs-lookup"><span data-stu-id="3b512-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3b512-105">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="3b512-105">To correct this error</span></span>  
  
1. <span data-ttu-id="3b512-106">Přidejte klíčové slovo `on` do klauzule `join`.</span><span class="sxs-lookup"><span data-stu-id="3b512-106">Add the `on` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3b512-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="3b512-107">Example</span></span>  
 <span data-ttu-id="3b512-108">Následující kód generuje CS0743:</span><span class="sxs-lookup"><span data-stu-id="3b512-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
