# <a name="compiler-error-cs1542"></a>Compiler Error CS1542
'dll' nelze přidat do tohoto sestavení, protože už je sestavení; Použijte ' / R' místo něj parametr  
  
 Soubor, který se odkazovalo se [/addmodule](../../csharp/language-reference/compiler-options/addmodule-compiler-option.md) – možnost kompilátoru se nesestavil s [/target: Module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md); použijte [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) účelem odkazu na soubor v této kompilaci.
