# <a name="compiler-error-cs1542"></a>Chyba kompilátoru CS1542
' dll ' nelze přidat do tohoto sestavení, protože se již jedná o sestavení; místo toho použijte možnost/R  
  
 Soubor, na který se odkazovala možnost kompilátoru [-addmodule –](../language-reference/compiler-options/addmodule-compiler-option.md) , se nesestavil s parametrem [-target: Module](../language-reference/compiler-options/target-module-compiler-option.md); pomocí [odkazu](../language-reference/compiler-options/reference-compiler-option.md) na soubor v této kompilaci.
