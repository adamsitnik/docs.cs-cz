# <a name="compiler-warning-level-1-cs2002"></a>Kompilátor CS2002 upozornění (úroveň 1)
Zdrojový soubor 'file' zadaný několikrát.  
  
 Název zdrojového souboru byl předán kompilátoru více než jednou. Můžete určit pouze jednou souboru kompilátoru k sestavení výstupní soubor.  
  
 Nelze toto upozornění potlačit [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) možnost.  
  
 Následující ukázka generuje CS2002:  
  
```csharp  
// CS2002.cs  
// compile with: CS2002.cs  
public class A  
{  
public static void Main(){}  
}  
```  
  
 K vygenerování chyby kompilace v příkladu s příkazovým řádkem:  
  
```console  
csc CS2002.cs CS2002.cs  
```
