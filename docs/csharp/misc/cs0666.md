# <a name="compiler-error-cs0666"></a>CS0666 chyby kompilátoru
'member': ve struktuře je deklarovaný nový chráněný člen  
  
 A [struktura](../../csharp/language-reference/keywords/struct.md) nemůže být [abstraktní](../../csharp/language-reference/keywords/abstract.md) a je vždy implicitně [zapečetěné](../../csharp/language-reference/keywords/sealed.md). Protože struktury nepodporují dědičnosti, konceptu [chráněné](../../csharp/language-reference/keywords/protected.md) člen struktury nemá smysl. Další informace najdete v tématu [dědičnosti](../../csharp/programming-guide/classes-and-structs/inheritance.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0666:  
  
```csharp  
// CS0666.cs  
class M  
{  
    static void Main()  
    {  
    }  
}  
  
struct S  
{  
    protected int x;   // CS0666  
}  
```
