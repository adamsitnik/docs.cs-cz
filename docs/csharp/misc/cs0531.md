# <a name="compiler-error-cs0531"></a>Compiler Error CS0531
'member': členy rozhraní nemůžou obsahovat definice  
  
 Metody, které jsou deklarovány v [rozhraní](../../csharp/language-reference/keywords/interface.md) musí být implementované ve třídě, která dědí z něj a ne v samotným rozhraním.  
  
 Následující ukázka generuje CS0531:  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
