# <a name="compiler-error-cs0513"></a>Compiler Error CS0513
'function' je abstraktní, ale je obsažena v neabstraktní třídě 'class'  
  
 Metoda nemůže být [abstraktní](../../csharp/language-reference/keywords/abstract.md) člen neabstraktní třídy.  
  
 Následující ukázka generuje CS0513:  
  
```csharp  
// CS0513.cs  
namespace x  
{  
   public class clx  
   {  
      abstract public void f();   // CS0513, abstract member of nonabstract class  
   }  
}  
```
