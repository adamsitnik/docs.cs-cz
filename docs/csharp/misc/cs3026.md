# <a name="compiler-warning-level-1-cs3026"></a>Kompilátor CS3026 upozornění (úroveň 1)

Pole kompatibilní se Specifikací CLS 'pole' nemůže být typu volatile

Volatile proměnné by neměla být kompatibilní se Specifikací CLS.

## <a name="example"></a>Příklad

Následující příklad generuje CS3026.

```csharp
// CS3026.cs
[assembly:System.CLSCompliant(true)]
public class Test
{
    public volatile int v0 =0;   // CS3026
    // To resolve remove the CLS-Compliant attribute.
    public static void Main() { }
}
```
