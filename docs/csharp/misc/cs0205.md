# <a name="compiler-error-cs0205"></a>CS0205 chyby kompilátoru
Nelze volat abstraktní základní člen: 'metody'  
  
 Nelze volat [abstraktní](../../csharp/language-reference/keywords/abstract.md) metoda protože nemá tělo metody. Další informace najdete v tématu [abstraktní a zapečetěné třídy a členové](../../csharp/programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).  
  
 Následující ukázka generuje CS0205:  
  
```csharp  
// CS0205.cs  
abstract public class MyClass  
{  
   abstract public void M();  
}  
  
public class MyClass2 : MyClass  
{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
