 <span data-ttu-id="50440-106">Následující ukázka generuje CS0205:</span><span class="sxs-lookup"><span data-stu-id="50440-106">The following sample generates CS0205:</span></span>  
  
```csharp  
// CS0205.cs  
abstract public class MyClass  
{  
   abstract public void M();  
}  
  
public class MyClass2 : MyClass  
{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
