## <a name="example"></a><span data-ttu-id="7704a-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="7704a-108">Example</span></span>  

<span data-ttu-id="7704a-109">Následující C++ příklad kompiluje do souboru. dll.</span><span class="sxs-lookup"><span data-stu-id="7704a-109">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="7704a-110">Všimněte si, že atribut [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) je zakomentováný, aby se vygenerovala chyba.</span><span class="sxs-lookup"><span data-stu-id="7704a-110">Note that the the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  

<span data-ttu-id="7704a-111">Následující C# příklad volá soubor. dll.</span><span class="sxs-lookup"><span data-stu-id="7704a-111">The following C# example calls the .dll file.</span></span> <span data-ttu-id="7704a-112">Pokusí se získat přístup ke třídě přes indexer, ale vzhledem k tomu, že žádný člen nebyl deklarován jako výchozí indexer, je vygenerována chyba.</span><span class="sxs-lookup"><span data-stu-id="7704a-112">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="7704a-113">Tuto chybu můžete vyřešit tak, že Odkomentujete řádek `[DefaultMember("myItem")]`.</span><span class="sxs-lookup"><span data-stu-id="7704a-113">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>

{  
    public static void Main()

    }
