# <a name="compiler-error-cs0021"></a>Compiler Error CS0021
Nelze použít indexování pomocí []. ve výrazu typu 'type'  
  
 Byl proveden pokus o přistupovat k hodnotě prostřednictvím indexeru na datový typ, který nepodporuje [indexery](../../csharp/programming-guide/indexers/index.md).  
  
 Při pokusu použít indexer v, může se zobrazit CS0021 C++ sestavení. V takovém případě uspořádání C++ třídy s `DefaultMember` atribut proto C# kompilátor ví, které indexeru je výchozí nastavení. Následující ukázka generuje CS0021.  
  
## <a name="example"></a>Příklad  
 Tento soubor kompiluje soubor .dll – s `DefaultMember` atribut označené jako komentář, aby se vygenerovala chybu.  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a>Příklad  
 Tady je C# soubor, který volá soubor .dll. Tento soubor se pokusí o přístup ke třídě prostřednictvím indexeru, ale vzhledem k tomu, že žádný člen byl deklarován jako výchozímu indexeru, který se má použít, je generována chyba.  
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
