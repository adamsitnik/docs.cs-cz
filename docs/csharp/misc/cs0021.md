# <a name="compiler-error-cs0021"></a>Chyba kompilátoru CS0021
Nejde použít indexování s [] na výraz typu Type.  
  
 Byl proveden pokus o přístup k hodnotě prostřednictvím indexeru na datovém typu, který nepodporuje indexery. [](../programming-guide/indexers/index.md)  
  
 Při pokusu o použití indexeru v C++ sestavení můžete získat CS0021. V tomto případě seupravte C++ třídu `DefaultMember` atributem tak, aby C# kompilátor věděl, který indexer je výchozí. Následující ukázka generuje CS0021.  
  
## <a name="example"></a>Příklad  
 Tento soubor se zkompiluje do souboru. dll – s `DefaultMember` atributem s komentářem, aby se vygenerovala chyba.  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a>Příklad  
 Následuje C# soubor, který volá soubor. dll. Tento soubor se pokusí získat přístup ke třídě prostřednictvím indexeru, ale vzhledem k tomu, že žádný člen nebyl deklarován jako výchozí indexer, který má být použit, je vygenerována chyba.  
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
