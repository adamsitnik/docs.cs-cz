# <a name="compiler-error-cs0721"></a>Compiler Error CS0721
'type': statické typy nejde používat jako parametry  
  
 Statický typ není smysluplné jako parametr. Protože mohou být vytvořeny žádné instance statické typy, žádná instance může někdy předán jako parametr.  
  
 Následující ukázka generuje CS0721:  
  
```csharp  
// CS0721.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public void F(SC sc)  // CS0721  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
