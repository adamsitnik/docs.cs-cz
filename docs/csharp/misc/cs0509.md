# <a name="compiler-error-cs0509"></a>CS0509 chyby kompilátoru
'class1': nejde odvozovat ze zapečetěného typu 'class2'  
  
 A [zapečetěné](../../csharp/language-reference/keywords/sealed.md) třídy nemůže fungovat jako [základní](../../csharp/language-reference/keywords/base.md) třídy. Ve výchozím nastavení jsou zapečetěné struktury.  
  
 Následující ukázka generuje CS0509:  
  
```csharp  
// CS0509.cs  
// compile with: /target:library  
sealed public class clx {}  
public class cly : clx {}   // CS0509  
```
