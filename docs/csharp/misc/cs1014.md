# <a name="compiler-error-cs1014"></a>Compiler Error CS1014
Get nebo set očekával se přistupující objekt.  
  
 V deklaraci vlastnosti nebyla nalezena deklarace metody. Je možné deklarovat pouze `get` a `set` metody ve vlastnosti.  
  
 Další informace o vlastnostech najdete v tématu [pomocí vlastnosti](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1014.  
  
```csharp  
// CS1014.cs  
// compile with: /target:library  
class Sample  
{  
   public int TestProperty  
   {  
      get  
      {  
         return 0;  
      }  
      int z;   // CS1014  not get or set  
   }  
}  
```
