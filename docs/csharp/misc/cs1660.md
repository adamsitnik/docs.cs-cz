# <a name="compiler-error-cs1660"></a><span data-ttu-id="68c38-102">Compiler Error CS1660</span><span class="sxs-lookup"><span data-stu-id="68c38-102">Compiler Error CS1660</span></span>
<span data-ttu-id="68c38-103">Blok anonymní metody na typ 'type', protože to není typ delegáta nelze převést.</span><span class="sxs-lookup"><span data-stu-id="68c38-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="68c38-104">K této chybě dochází, pokud se pokusíte přiřadit nebo jinak blok anonymní metody převést na typ, který není typ delegáta.</span><span class="sxs-lookup"><span data-stu-id="68c38-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="68c38-105">Následující ukázka generuje CS1660:</span><span class="sxs-lookup"><span data-stu-id="68c38-105">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
