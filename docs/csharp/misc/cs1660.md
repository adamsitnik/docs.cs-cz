# <a name="compiler-error-cs1660"></a>Compiler Error CS1660
Blok anonymní metody na typ 'type', protože to není typ delegáta nelze převést.  
  
 K této chybě dochází, pokud se pokusíte přiřadit nebo jinak blok anonymní metody převést na typ, který není typ delegáta.  
  
 Následující ukázka generuje CS1660:  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
