# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="99e7b-102">Upozornění kompilátoru (úroveň 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="99e7b-102">Compiler Warning (level 1) CS0612</span></span>
<span data-ttu-id="99e7b-103">člen je zastaralý.</span><span class="sxs-lookup"><span data-stu-id="99e7b-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="99e7b-104">Návrhář tříd označil člena s [zastaralým](../programming-guide/concepts/attributes/common-attributes.md#Obsolete) atributem.</span><span class="sxs-lookup"><span data-stu-id="99e7b-104">The class designer marked a member with the [Obsolete](../programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute.</span></span> <span data-ttu-id="99e7b-105">To znamená, že člen nemusí být podporován v budoucí verzi třídy.</span><span class="sxs-lookup"><span data-stu-id="99e7b-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="99e7b-106">Následující příklad ukazuje, jak přistupovat k zastaralému členu generuje CS0612:</span><span class="sxs-lookup"><span data-stu-id="99e7b-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
