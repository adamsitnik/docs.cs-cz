# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="842a9-102">Kompilátor CS0612 upozornění (úroveň 1)</span><span class="sxs-lookup"><span data-stu-id="842a9-102">Compiler Warning (level 1) CS0612</span></span>
<span data-ttu-id="842a9-103">'member' je zastaralý</span><span class="sxs-lookup"><span data-stu-id="842a9-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="842a9-104">Návrhář tříd označené se členem [zastaralé](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) atribut.</span><span class="sxs-lookup"><span data-stu-id="842a9-104">The class designer marked a member with the [Obsolete](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute.</span></span> <span data-ttu-id="842a9-105">To znamená, že nemusí být člen podporován v budoucí verzi třídy.</span><span class="sxs-lookup"><span data-stu-id="842a9-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="842a9-106">Následující příklad ukazuje, jak přístup k zastaralý člen generuje CS0612:</span><span class="sxs-lookup"><span data-stu-id="842a9-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
