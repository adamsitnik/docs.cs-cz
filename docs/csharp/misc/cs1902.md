# <a name="compiler-error-cs1902"></a>Compiler Error CS1902
Neplatná možnost "možnost" pro/Debug; musí být full nebo pdbonly  
  
 Byl předán neplatný přepínač [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) – možnost kompilátoru.  
  
 Následující ukázka generuje CS1902:  
  
```csharp  
// CS1902.cs  
// compile with: /debug:x  
// CS1902 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
