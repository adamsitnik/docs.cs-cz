# <a name="compiler-error-cs1902"></a>Chyba kompilátoru CS1902
Neplatná možnost Option pro/Debug; musí být úplné nebo pdbonly.  
  
 Možnosti kompilátoru [/Debug](../language-reference/compiler-options/debug-compiler-option.md) byla předána neplatná možnost.  
  
 Následující ukázka generuje CS1902:  
  
```csharp  
// CS1902.cs  
// compile with: /debug:x  
// CS1902 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
