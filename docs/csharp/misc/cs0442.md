# <a name="compiler-error-cs0442"></a>Compiler Error CS0442
'Property': abstraktní vlastnosti nemůžou mít privátní přístupové objekty  
  
 Tato chyba nastane, pokud použijete modifikátor přístupu "privátní" upravit abstraktní přístupový objekt. Pokud chcete vyřešit, použijte modifikátor různý přístup nebo nastavení vlastnosti neabstraktní.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0442:  
  
```csharp  
// CS0442.cs  
public abstract class MyClass   
{  
    public abstract int AbstractProperty   
    {  
        get;  
        private set;   // CS0442  
        // Try this instead:  
        // set;  
    }  
}  
```
