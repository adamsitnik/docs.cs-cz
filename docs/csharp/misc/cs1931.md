 <span data-ttu-id="5a131-104">Deklarace proměnné rozsahu, stejně jako každá jiná deklarace, musí mít identifikátor, který je v rámci prostoru deklarace proměnné jedinečný.</span><span class="sxs-lookup"><span data-stu-id="5a131-104">The declaration of a range variable, just like every other declaration, must have an identifier which is unique within the variable's declaration space.</span></span>

## <a name="example"></a><span data-ttu-id="5a131-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="5a131-107">Example</span></span>  
 <span data-ttu-id="5a131-108">Následující kód vygeneruje CS1931, protože identifikátor `x` je použit jako místní proměnná v `Main` a jako proměnná rozsahu ve výrazu dotazu:</span><span class="sxs-lookup"><span data-stu-id="5a131-108">The following code generates CS1931 because the identifier `x` is used both as a local variable in `Main` and as the range variable in the query expression:</span></span>

```csharp  
class Test  
    {  
        static void Main()  
        {  
            int x = 1;  
            var y = from x in Enumerable.Range(1, 100) // CS1931  
                    select x;  
        }  
    }  
```
