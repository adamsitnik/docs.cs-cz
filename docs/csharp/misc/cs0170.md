# <a name="compiler-error-cs0170"></a>Compiler Error CS0170
Použití pravděpodobně nepřiřazené pole 'pole.  
  
 Pole ve struktuře se použil bez první během inicializace. Chcete-li tento problém vyřešit, nejprve určit, jaké pole se neinicializované, kterou následně inicializujete před pokusem o přístup. Další informace o inicializaci struktury naleznete v tématu [struktury](../../csharp/programming-guide/classes-and-structs/structs.md) a [pomocí struktury](../../csharp/programming-guide/classes-and-structs/using-structs.md).  
  
 Následující ukázka generuje CS0170:  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
