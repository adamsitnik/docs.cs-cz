## <a name="example"></a><span data-ttu-id="a7331-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="a7331-108">Example</span></span>  
 <span data-ttu-id="a7331-109">Následující kód vygeneruje CS0738, protože metoda třídy vrací `void` hodnotu a člen rozhraní se stejným názvem vrátí: `int`</span><span class="sxs-lookup"><span data-stu-id="a7331-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```csharp  
{  
    int TestMethod();  
}  
public class Test: ITest  
{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();  
}  
```
