# <a name="compiler-error-cs0133"></a>Compiler Error CS0133
Výraz přiřazovaný proměnné 'variable' musí být konstantou  
  
 A [const](../../csharp/language-reference/keywords/const.md) proměnnou nelze převzít jako svou hodnotu, která není konstantní výraz. Další informace najdete v tématu [konstanty](../../csharp/programming-guide/classes-and-structs/constants.md).  
  
 Následující ukázka generuje CS0133:  
  
```csharp  
// CS0133.cs  
public class MyClass  
{  
   public const int i = c;   // CS0133, c is not constant  
   public static int c = i;  
   // try the following line instead  
   // public const int i = 6;  
  
   public static void Main()  
   {  
   }  
}  
```
