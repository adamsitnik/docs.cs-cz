# <a name="compiler-error-cs2001"></a>CS2001 chyby kompilátoru
Zdrojový soubor 'file' nebyl nalezen.  
  
 Název zdrojového souboru byl předán kompilátoru, ale nebyla nalezena. Zkontrolujte správnost názvu souboru a umístění souboru.
