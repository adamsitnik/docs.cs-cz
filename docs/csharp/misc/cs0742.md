# <a name="compiler-error-cs0742"></a>Chyba kompilátoru CS0742
Tělo dotazu musí končit klauzulí SELECT nebo klauzulí Group.  
  
 Výraz dotazu musí být `select` ukončen klauzulí `group` nebo klauzulí bez pokračování.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Přidejte do dotazu [klauzuli SELECT](../language-reference/keywords/select-clause.md) nebo [klauzuli Group](../language-reference/keywords/group-clause.md) .  
  
## <a name="example"></a>Příklad  
 Následující kód generuje CS0742:  
  
```csharp  
// cs0742.cs  
using System.Linq;  
public class Test  
{  
    public static int Main()  
    {  
        int[] array = { 1, 2, 3 };  
        var query = from num in array; // CS0742  
        return 1;  
    }  
}  
```  
  
 Pokud klauzule používá klíčové slovo into k uložení výsledků seskupení do dočasného identifikátoru, nemůže to být poslední klauzule v dotazu. [](../language-reference/keywords/into.md) `group` Klauzule`group`orsepořádvyžaduje. `select`
