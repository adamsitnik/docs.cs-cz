# <a name="compiler-error-cs0154"></a>Compiler Error CS0154
Vlastnost nebo indexovací člen 'property' nelze použít v tomto kontextu protože neobsahuje přistupující objekt get  
  
 Pokus o použití [vlastnost](../../csharp/programming-guide/classes-and-structs/using-properties.md) se nezdařila, protože žádné metody přístupového objektu get byla definována ve vlastnosti. Další informace najdete v tématu [pole](../../csharp/programming-guide/classes-and-structs/fields.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0154:  
  
```csharp  
// CS0154.cs  
public class MyClass2  
{  
    public int i  
    {  
        set  
        {  
        }  
        // uncomment the get method to resolve this error  
        /*  
        get  
        {  
            return 0;  
        }  
        */  
    }  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClass2 myClass2 = new MyClass2();  
        int j = myClass2.i;   // CS0154, no get method  
    }  
}  
```
