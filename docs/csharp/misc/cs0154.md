# <a name="compiler-error-cs0154"></a>Chyba kompilátoru CS0154
Vlastnost nebo indexer ' Property ' nelze v tomto kontextu použít, protože neobsahuje přistupující objekt get  
  
 Pokus o použití [vlastnosti](../programming-guide/classes-and-structs/using-properties.md) se nezdařil, protože ve vlastnosti nebyla definována žádná metoda přístupu Get. Další informace najdete v tématu [pole](../programming-guide/classes-and-structs/fields.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0154:  
  
```csharp  
// CS0154.cs  
public class MyClass2  
{  
    public int i  
    {  
        set  
        {  
        }  
        // uncomment the get method to resolve this error  
        /*  
        get  
        {  
            return 0;  
        }  
        */  
    }  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClass2 myClass2 = new MyClass2();  
        int j = myClass2.i;   // CS0154, no get method  
    }  
}  
```
