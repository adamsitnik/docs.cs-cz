# <a name="compiler-error-cs1535"></a>Compiler Error CS1535
Přetěžovaný unární operátor 'operator' používá jeden parametr  
  
 Definice unární [operátor](../../csharp/language-reference/operators/operator-overloading.md) musí přijímat jeden parametr.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1535:  
  
```csharp  
// CS1535.cs  
class MyClass  
{  
    // uncomment the method parameter to resolve CS1535  
    public static MyClass operator ++ (/*MyClass MC1*/)   // CS1535  
    {  
        return new MyClass();  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
