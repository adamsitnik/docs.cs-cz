# <a name="compiler-warning-level-1-cs1589"></a>Kompilátor CS1589 upozornění (úroveň 1)
Nejde zahrnout fragment XML fragment souboru 'file'--důvod  
  
 Syntaxe (*fragment*) z [ \<zahrnout >](../../csharp/programming-guide/xmldoc/include.md) značky, který odkazuje soubor (`file`), bylo pro zadaný správný ***důvod***.  
  
 Poškozená řádku budou umístěny do generovaného souboru XML.  
  
 Následující ukázka generuje CS1589:  
  
```csharp  
// CS1589.cs  
// compile with: /W:1 /doc:CS1589_out.xml  
  
/// <include file='CS1589.doc' path='MyDocs/MyMembers[@name="test"]/' />   // CS1589  
// try the following line instead  
// /// <include file='CS1589.doc' path='MyDocs/MyMembers[@name="test"]/*' />  
class Test  
{  
   public static void Main()  
   {  
   }  
}  
```
