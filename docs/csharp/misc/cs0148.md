# <a name="compiler-error-cs0148"></a>Chyba kompilátoru CS0148
Delegát Delegate nemá platný konstruktor.  
  
 Naimportovali a použili jste spravovaný program (ten, který používá .NET Framework Common Language Runtime), který byl vytvořen s jiným kompilátorem. Tento kompilátor povoluje konstruktor delegáta s [](../language-reference/keywords/delegate.md) chybnou strukturou. Další informace najdete v tématu [Delegáti](../programming-guide/delegates/index.md).
