# <a name="compiler-error-cs0148"></a>CS0148 chyby kompilátoru
Delegát 'delegáta' nemá platný konstruktor  
  
 Importovat a používat spravované aplikace (jedna, která používá rozhraní .NET Framework common language runtime), který byl vytvořen s jiným kompilátorem. Tento kompilátor povolené chybně vytvořený [delegovat](../../csharp/language-reference/keywords/delegate.md) konstruktoru. Další informace najdete v tématu [delegáti](../../csharp/programming-guide/delegates/index.md).
