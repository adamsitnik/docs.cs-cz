# <a name="compiler-error-cs1039"></a>Compiler Error CS1039
Neukončený řetězcový literál  
  
 Kompilátor zjistil nesprávném [řetězec](../../csharp/language-reference/keywords/string.md) literálu.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1039. Chcete-li chybu vyřešit, přidejte ukončující znak uvozovek.  
  
```csharp  
// CS1039.cs  
public class MyClass  
{  
    public static void Main()  
    {  
        string b = @"hello, world;   // CS1039  
    }  
}  
```
