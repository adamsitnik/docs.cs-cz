# <a name="compiler-error-cs0027"></a><span data-ttu-id="c2a15-102">Chyba kompilátoru CS0027</span><span class="sxs-lookup"><span data-stu-id="c2a15-102">Compiler Error CS0027</span></span>
<span data-ttu-id="c2a15-103">Klíčové slovo this není v aktuálním kontextu k dispozici.</span><span class="sxs-lookup"><span data-stu-id="c2a15-103">Keyword 'this' is not available in the current context</span></span>  
  
 <span data-ttu-id="c2a15-104">Klíčové slovo [This](../language-reference/keywords/this.md) bylo nalezeno mimo vlastnost, metodu nebo konstruktor.</span><span class="sxs-lookup"><span data-stu-id="c2a15-104">The [this](../language-reference/keywords/this.md) keyword was found outside of a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="c2a15-105">Chcete-li tuto chybu opravit, upravte příkaz tak, aby vyloučil použití `this` klíčového slova, nebo přesuňte část nebo celý příkaz do vlastnosti, metody nebo konstruktoru.</span><span class="sxs-lookup"><span data-stu-id="c2a15-105">To fix this error, either modify the statement to eliminate use of the `this` keyword, and/or move part or all of the statement inside a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="c2a15-106">Následující příklad generuje CS0027:</span><span class="sxs-lookup"><span data-stu-id="c2a15-106">The following example generates CS0027:</span></span>  
  
```csharp  
using System;  
using System.Collections.Generic;  
using System.Text;  
  
namespace ConsoleApplication3  
{  
    class MyClass  
    {  
  
        int err1 = this.Fun() + 1;  // CS0027   
  
        public int Fun()  
        {  
            return 10;  
        }  
  
        public void Test()  
        {  
            // valid use of this  
            int err = this.Fun() + 1;  
            Console.WriteLine(err);  
        }  
  
        public static void Main()  
        {  
            MyClass c = new MyClass();  
            c.Test();  
        }  
    }  
}  
```
