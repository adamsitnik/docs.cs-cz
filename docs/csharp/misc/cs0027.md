# <a name="compiler-error-cs0027"></a>Compiler Error CS0027
Klíčové slovo, 'this' není k dispozici v rámci aktuálního kontextu  
  
 [To](../../csharp/language-reference/keywords/this.md) – klíčové slovo byl nalezen mimo vlastnosti, metody nebo konstruktoru.  
  
 Chcete-li vyřešit tuto chybu, buď upravte příkaz tak, aby eliminovat použití `this` – klíčové slovo, a/nebo přesunutí část nebo všechny výroku uvnitř vlastnosti, metody nebo konstruktoru.  
  
 Následující příklad generuje CS0027:  
  
```csharp  
using System;  
using System.Collections.Generic;  
using System.Text;  
  
namespace ConsoleApplication3  
{  
    class MyClass  
    {  
  
        int err1 = this.Fun() + 1;  // CS0027   
  
        public int Fun()  
        {  
            return 10;  
        }  
  
        public void Test()  
        {  
            // valid use of this  
            int err = this.Fun() + 1;  
            Console.WriteLine(err);  
        }  
  
        public static void Main()  
        {  
            MyClass c = new MyClass();  
            c.Test();  
        }  
    }  
}  
```
