# <a name="compiler-error-cs0225"></a>Compiler Error CS0225
Parametr params musí být jednorozměrné pole  
  
 Při použití [params](../../csharp/language-reference/keywords/params.md) – klíčové slovo, je nutné zadat jednorozměrné pole datového typu. Další informace najdete v tématu [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0225:  
  
```csharp  
// CS0225.cs  
public class MyClass  
{  
    public static void TestParams(params int a)   // CS0225  
    // try the following line instead  
    // public static void TestParams(params int[] a)  
    {  
    }  
  
    public static void Main()  
    {  
        TestParams(1);  
    }  
}  
```
