# <a name="compiler-error-cs0191"></a>Compiler Error CS0191
Vlastnost nebo indexovací člen, který 'name' nelze přiřazovat--je jen pro čtení  
  
 A [jen pro čtení](../../csharp/language-reference/keywords/readonly.md) pole jde převzít jenom přiřazení v konstruktoru nebo v deklaraci. Další informace najdete v tématu [konstruktory](../../csharp/programming-guide/classes-and-structs/constructors.md).  
  
 CS0191 také výsledků, pokud `readonly` pole je [statické](../../csharp/language-reference/keywords/static.md) a konstruktor není označený jako `static`.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0191.  
  
```csharp  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
