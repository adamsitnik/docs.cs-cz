## <a name="example"></a><span data-ttu-id="bb953-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="bb953-107">Example</span></span>  
 <span data-ttu-id="bb953-108">Následující ukázka generuje CS0191.</span><span class="sxs-lookup"><span data-stu-id="bb953-108">The following sample generates CS0191.</span></span>  
  
```csharp  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
