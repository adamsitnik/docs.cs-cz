# <a name="compiler-warning-level-2-cs1587"></a>Kompilátor CS1587 upozornění (úroveň 2)
Komentář XML není umístěný na platném prvku jazyka.  
  
 Doporučené značky pro dokumentační komentáře nejsou povoleny na všechny prvky jazyka. Například klíčové slovo není povolena u oboru názvů. Další informace o komentáře XML, naleznete v tématu [doporučené značky pro dokumentační komentáře](../../csharp/programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1587:  
  
```csharp  
// CS1587.cs  
// compile with: /W:2 /doc:x.xml  
  
/// <summary>test</summary>   // CS1587, tag not allowed on namespace  
namespace MySpace  
{  
    class MyClass  
    {  
        public static void Main()  
        {  
        }  
    }  
}  
```
