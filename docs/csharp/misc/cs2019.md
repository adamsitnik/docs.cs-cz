# <a name="compiler-error-cs2019"></a>CS2019 chyby kompilátoru
Neplatný cílový typ pro parametr/target: musí určit 'exe', 'winexe', 'library' nebo 'module'  
  
 [/Target](../../csharp/language-reference/compiler-options/target-compiler-option.md) byla použita možnost kompilátoru, ale byl předán neplatný parametr. Chcete-li vyřešit tuto chybu, znovu zkompilovat programu pomocí formátu **/target** možnost je vhodná pro výstupní soubor.  
  
 Následující ukázka generuje CS2017:  
  
```csharp  
// CS2019.cs  
// compile with: /target:libra  
// CS2019 expected  
class MyClass  
{  
}  
```
