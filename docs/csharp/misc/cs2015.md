# <a name="compiler-error-cs2015"></a>CS2015 chyby kompilátoru
'file' je binární soubor místo textového souboru  
  
 Soubor byl předán kompilátoru, který byl do binárního souboru. Kompilátor očekává, že soubor zdrojového kódu.
