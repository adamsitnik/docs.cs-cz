# <a name="compiler-error-cs0025"></a>Compiler Error CS0025
Standardní knihovna soubor 'file' nebyl nalezen.  
  
 Nebyl nalezen soubor, který je potřeba pro kompilátor. Zkontrolujte, že cesta je správná a zda soubor existuje.  
  
 Pokud je soubor systémový soubor sady Visual Studio, budete muset opravit instalaci sady Visual Studio, nebo znovu nainstalovat úplně.
