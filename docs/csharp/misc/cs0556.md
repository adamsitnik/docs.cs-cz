# <a name="compiler-error-cs0556"></a>Compiler Error CS0556
Uživatelem definovaný převod musí převádět do nebo z nadřazeného typu.  
  
 Rutina uživatelsky definovaný převod musí převést do nebo z třídy, která obsahuje rutiny.  
  
 Následující ukázka generuje CS0556:  
  
```csharp  
// CS0556.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         public static implicit operator int(byte aa)   // CS0556  
         // try the following line instead  
         // public static implicit operator int(iii aa)  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
