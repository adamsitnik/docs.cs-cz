# <a name="compiler-error-cs0662"></a>CS0662 chyby kompilátoru

"metoda" nelze určit jenom atribut u parametru ref Out. Použijte i v a atributy nebo ani jeden.  
  
 Metody rozhraní má parametr, který používá [ref](../../csharp/language-reference/keywords/ref.md) s jenom [si](xref:System.Runtime.InteropServices.OutAttribute) atribut. A `ref` parametr, který se používá **si** také musíte použít atribut [v](xref:System.Runtime.InteropServices.InAttribute) atribut.  
  
 Následující ukázka generuje CS0662:  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
