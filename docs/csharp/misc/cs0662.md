 <span data-ttu-id="ff966-105">Metody rozhraní má parametr, který používá [ref](../../csharp/language-reference/keywords/ref.md) s jenom [si](xref:System.Runtime.InteropServices.OutAttribute) atribut.</span><span class="sxs-lookup"><span data-stu-id="ff966-105">An interface method has a parameter that uses [ref](../../csharp/language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="ff966-106">A `ref` parametr, který se používá **si** také musíte použít atribut [v](xref:System.Runtime.InteropServices.InAttribute) atribut.</span><span class="sxs-lookup"><span data-stu-id="ff966-106">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>

```csharp
{  
   void method([Out] ref int i);   // CS0662

   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}

{  
   public static void Main()  
   {  
   }  
}  
```
