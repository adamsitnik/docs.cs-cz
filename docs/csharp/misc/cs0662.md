# <a name="compiler-error-cs0662"></a>Chyba kompilátoru CS0662

Metoda method nemůže pro parametr ref určovat pouze atribut out. Použijte atributy in i out, nebo ani jeden z nich.  
  
 Metoda rozhraní má parametr, který používá [ref](../language-reference/keywords/ref.md) s pouze atributem [out](xref:System.Runtime.InteropServices.OutAttribute) . Parametr, který používá atribut **out** , musí také použít atribut [in.](xref:System.Runtime.InteropServices.InAttribute) `ref`  
  
 Následující ukázka generuje CS0662:  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
