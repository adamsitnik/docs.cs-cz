 <span data-ttu-id="50d34-105">Metoda rozhraní má parametr, který používá [ref](../language-reference/keywords/ref.md) s pouze atributem [out](xref:System.Runtime.InteropServices.OutAttribute) .</span><span class="sxs-lookup"><span data-stu-id="50d34-105">An interface method has a parameter that uses [ref](../language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="50d34-106">Parametr, který používá atribut **out** , musí také použít atribut [in.](xref:System.Runtime.InteropServices.InAttribute) `ref`</span><span class="sxs-lookup"><span data-stu-id="50d34-106">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>

```csharp
{  
   public static void Main()  
   {  
   }  
}  
```
