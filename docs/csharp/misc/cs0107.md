# <a name="compiler-error-cs0107"></a>Compiler Error CS0107
Více než jeden modifikátor ochrany  
  
 Modifikátor přístupu pouze jeden ([veřejné](../../csharp/language-reference/keywords/public.md), [privátní](../../csharp/language-reference/keywords/private.md), [chráněné](../../csharp/language-reference/keywords/protected.md), nebo [interní](../../csharp/language-reference/keywords/internal.md)) je povolený pro člen třídy, s výjimkou z s výjimkou **interní chráněné**.  
  
 Následující ukázka generuje CS0107:  
  
```csharp  
// CS0107.cs  
public class C  
{  
   private internal void f()   // CS0107, delete private or internal  
   {  
   }  
  
   public static int Main()  
   {  
      return 1;  
   }  
}  
```
