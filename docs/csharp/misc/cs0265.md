# <a name="compiler-error-cs0265"></a>CS0265 chyby kompilátoru
Částečné deklarace 'type' mají nekonzistentní omezení pro typ parametru 'parametr typu.  
  
 K této chybě dochází při definování obecné třídy jako částečné třídy tak, aby jeho částečné definice vyskytují ve více než jednom místě a omezení u obecného typu jsou nekonzistentní nebo jiné ve dvou nebo více míst. Pokud chcete zadat omezení na více než jednom místě, se musí být všechny stejné. Nejjednodušším řešením je zadat omezení na jednom místě a vynechat, nechte je všude, kde jiný. Další informace najdete v tématu [částečné třídy a metody](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md) a [omezení parametrů typů](../../csharp/programming-guide/generics/constraints-on-type-parameters.md).  
  
 Následující kód vygeneruje chybu CS0265.  
  
## <a name="example"></a>Příklad  
 V tomto kódu definicí částečné třídy jsou všechno v jediném souboru, ale může být také rozložit mezi více souborů.  
  
```csharp  
// CS0265.cs  
public class GenericsErrors   
{  
    interface IFace1 { }  
    interface IFace2 { }  
    partial class PartialBadBounds<T> where T : IFace1 { } // CS0265  
    partial class PartialBadBounds<T> where T : IFace2 { }   
}  
```
