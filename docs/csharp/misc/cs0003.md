# <a name="compiler-error-cs0003"></a>Chyba kompilátoru CS0003
Nedostatek paměti  
  
 Kompilátor nemohl přidělit dostatek virtuální paměti pro dokončení kompilace. Zavřete všechny nepotřebné aplikace a znovu zkompilujte.  
  
 Možná budete chtít zvětšit velikost stránkovacího souboru a ujistit se, že máte volné místo na disku.  
  
 Tato chyba se může také zobrazit jako výsledek neodpovídající verze Windows SDK a C# kompilátoru nebo jednoho nebo více poškozených souborů, které podporují C# kompilátor. přeinstalujte sadu Visual Studio.
