# <a name="compiler-error-cs0003"></a>CS0003 chyby kompilátoru
Nedostatek paměti  
  
 Kompilátor nemohla přidělit dostatek virtuální paměti k dokončení kompilace. Ukončete všechny nepotřebné aplikace a znovu zkompilovat.  
  
 Můžete také zvýšit velikost stránkovacího souboru a ujistěte se, že máte volné místo na disku.  
  
 Této chybě může také dojít v důsledku s Neshoda verzí sady Windows Software Development Kit (SDK) a C# kompilátoru nebo jeden nebo více poškozený soubory, které podporují C# kompilátoru; Přeinstalujte Visual Studio.
