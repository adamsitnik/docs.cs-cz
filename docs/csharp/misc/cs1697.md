# <a name="compiler-warning-level-1-cs1697"></a>Kompilátor CS1697 upozornění (úroveň 1)
Různé hodnoty kontrolního součtu pro "název souboru.  
  
 Zadali jste víc než jeden kontrolní součet daného souboru. Ladicí program používá k určení soubor, který chcete ladit, když je více než jeden soubor v projektu se stejným názvem hodnota kontrolního součtu. Většina uživatelů se k této chybě dojde, ale pokud vytváříte aplikaci, která generuje kód, můžete narazit na něj. Chcete-li vyřešit tuto chybu, ujistěte se, generovat kontrolního součtu pouze jednou pro všechny soubory daného kódu.
