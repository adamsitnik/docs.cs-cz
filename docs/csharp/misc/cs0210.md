# <a name="compiler-error-cs0210"></a>Compiler Error CS0210
Je nutné zadat inicializátor v deklaracích příkazů fixed a using  
  
 Musíte deklarovat a inicializovat proměnnou v [oprava příkazu](../../csharp/language-reference/keywords/fixed-statement.md). Další informace najdete v tématu [nezabezpečený kód a ukazatele](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Následující ukázka generuje CS0210:  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 Následující ukázka rovněž vygeneruje CS0210 vzhledem k tomu, [příkaz using](../../csharp/language-reference/keywords/using-statement.md) nemá žádný inicializátor.  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test   
{  
   static void Main()   
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))   
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
