# <a name="compiler-error-cs0179"></a>CS0179 chyby kompilátoru
'member' nemůže být extern a deklarovat tělo  
  
 Když je členem třídy označena [extern](../../csharp/language-reference/keywords/extern.md), znamená to, že definice člena se nachází v jiném souboru. Proto, je členem třídy označené jako **extern** nemůže být definovaná ve třídě. Buď odeberte `extern` – klíčové slovo nebo odstranit definici. Další informace najdete v tématu [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Následující ukázka generuje CS0179:  
  
```csharp  
// CS0179.cs  
public class MyClass  
{  
   public extern int ExternMethod(int aa)   // CS0179  
   {  
      return 0;  
   }  
   // try the following line instead  
   // public extern int ExternMethod(int aa);  
  
   public static void Main()  
   {  
   }  
}  
```
