# <a name="compiler-error-cs1583"></a>Compiler Error CS1583
'file' není platný soubor prostředků Win32  
  
 K této chybě dochází, když zadáte název souboru se [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) – možnost kompilátoru, který není platný nebo správně formátovaného souboru prostředků Win32. V sadě Visual Studio je název souboru zadaný v podokně aplikace Návrháře projektu.
