# <a name="compiler-error-cs0242"></a>CS0242 chyby kompilátoru
Příslušná operace není definovaná pro ukazatele typu void  
  
 Zvyšování ukazatel void není povolený. Další informace najdete v tématu [nezabezpečený kód a ukazatele](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Následující ukázka generuje CS0242:  
  
```csharp  
// CS0242.cs  
// compile with: /unsafe  
class TestClass  
{  
   public unsafe void Test()  
   {  
      void * p = null;  
      p++;   // CS0242, incrementing a void pointer not allowed  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
