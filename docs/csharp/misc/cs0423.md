# <a name="compiler-error-cs0423"></a>Compiler Error CS0423
Protože 'class' obsahuje atribut ComImport, 'metody' musí být externí nebo abstraktní  
  
 Určení atributem ComImport znamená, že má být importován z modulu COM implementace třídy. Další metody nemusí být definován.  
  
 Následující ukázka generuje CS0423:  
  
```csharp  
// CS0423.cs  
  
using System.Runtime.InteropServices;  
  
[  
  ComImport,  
  Guid("7ab770c7-0e23-4d7a-8aa2-19bfad479829")  
]  
class ImageProperties  
{  
  public static void Main()  // CS0423  
  {  
    ImageProperties i = new ImageProperties();  
  }  
}  
```
