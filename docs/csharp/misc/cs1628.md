# <a name="compiler-error-cs1628"></a>Compiler Error CS1628
Nelze použít v ref nebo out parametr 'parametru' uvnitř anonymní metody, výrazu lambda nebo výrazu dotazu  
  
 K této chybě dochází, pokud používáte `in`, `ref`, nebo `out` parametr uvnitř anonymní metody bloku. Lze vyvarovat této chyby, použijte místní proměnné nebo některé jiné konstrukce.  
  
 Následující ukázka generuje CS1628:  
  
```csharp  
// CS1628.cs  
  
delegate int MyDelegate();  
  
class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
