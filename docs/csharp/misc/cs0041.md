# <a name="compiler-error-cs0041"></a>Compiler Error CS0041
Plně kvalifikovaný název pro 'type' je příliš dlouhý pro informace o ladění. Proveďte kompilaci bez možnost "/ debug".  
  
 Této chybě může dojít při použití [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) – možnost kompilátoru. Pokud dojde k této chybě, zkuste odstranit soubory PDB do adresáře bin a opětovné kompilace. Pokud stále dochází k této chybě, budete muset opravte nebo přeinstalujte Visual Studio.
