# <a name="compiler-error-cs0146"></a>Compiler Error CS0146
Cyklickou závislost základních tříd zahrnující 'class1' a 'class2'  
  
 Seznam dědičnost třídy obsahuje přímý nebo nepřímý odkaz na sebe sama. Třída nemůže dědit samo od sebe. Další informace najdete v tématu [dědičnosti](../../csharp/programming-guide/classes-and-structs/inheritance.md).  
  
 Následující ukázka generuje CS0146:  
  
```csharp  
// CS0146.cs  
namespace MyNamespace  
{  
   public interface InterfaceA  
   {  
   }  
  
   public class MyClass : InterfaceA, MyClass2  
   {  
      public void Main()  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass   // CS0146  
   {  
   }  
}  
```
