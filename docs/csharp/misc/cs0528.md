# <a name="compiler-error-cs0528"></a>Compiler Error CS0528
"rozhraní" je již uveden v seznamu rozhraní.  
  
 V seznamu rozhraní dědičnosti obsahuje duplicitní. [Rozhraní](../../csharp/language-reference/keywords/interface.md) lze zadat pouze jednou v seznamu dědičnosti.  
  
 Následující ukázka generuje CS0528:  
  
```csharp  
// CS0528.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public class b : a, a   // CS0528  
   {  
      public void Main()  
      {  
      }  
   }  
}  
```
