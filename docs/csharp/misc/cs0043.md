# <a name="compiler-error-cs0043"></a>CS0043 chyby kompilátoru
Soubor PDB 'file' má nesprávný nebo zastaralý formát. Odstranit a znovu sestavit.  
  
 Odstraňte soubor .pdb pro tato kompilace a opětovné kompilace.
