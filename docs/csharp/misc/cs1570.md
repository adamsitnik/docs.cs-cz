# <a name="compiler-warning-level-1-cs1570"></a>Kompilátor CS1570 upozornění (úroveň 1)
Komentář XML v 'konstrukce' má chybně vytvořený kód – 'důvod:  
  
 Při použití [/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md), všechny komentáře ve zdrojovém kódu musí být ve formátu XML. Všechny chyby se váš kód XML vygeneruje CS1570. Příklad:  
  
- Pokud předáte řetězec **cref**, například jako v [ \<výjimky >](../../csharp/programming-guide/xmldoc/exception.md) značky, řetězec musí být uzavřen do dvojitých uvozovek.  
  
- Pokud použijete značku, například [ \<seealso >](../../csharp/programming-guide/xmldoc/seealso.md), který nemá uzavírací značku, je nutné zadat lomítkem před ukončovací lomená závorka.  
  
- Pokud je třeba použít větší – menší než nebo – než symbolů v textu popis, budete muset pomocí představují **&gt;** nebo **&lt;**.  
  
- Atribut souboru nebo cesta [ \<zahrnout >](../../csharp/programming-guide/xmldoc/include.md) značky se chybějící nebo nesprávně formováno.  
  
 Následující ukázka generuje CS1570:  
  
```csharp  
// CS1570.cs  
// compile with: /W:1  
namespace ns  
{  
   // the following line generates CS1570  
   /// <summary> returns true if < 5 </summary>  
   // try this instead  
   // /// <summary> returns true if <5 </summary>  
  
   public class MyClass  
   {  
      public static void Main ()  
      {  
      }  
   }  
}  
```
