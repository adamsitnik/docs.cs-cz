# <a name="compiler-warning-level-2-cs1668"></a>Upozornění kompilátoru (úroveň 2) CS1668
Neplatná cesta hledání ' cesta ' zadaná v ' řetězec cesty '--' systémová chybová zpráva '  
  
 Cesta dodaná k [/lib](../language-reference/compiler-options/lib-compiler-option.md) na příkazovém řádku nebyla platná nebo cesta v proměnné prostředí LIB není platná. Zkontrolujte cestu, která se používá k ověření, zda existuje a zda má být k dispozici. Chybová zpráva v jednoduchých uvozovkách představuje chybu vrácenou z operačního systému.
