# <a name="compiler-warning-level-4-cs0078"></a><span data-ttu-id="38369-102">Kompilátor CS0078 upozornění (úroveň 4)</span><span class="sxs-lookup"><span data-stu-id="38369-102">Compiler Warning (level 4) CS0078</span></span>
<span data-ttu-id="38369-103">Přípona "l" je snadno zaměnitelná s číslicí 1. – použijte "L" pro přehlednost</span><span class="sxs-lookup"><span data-stu-id="38369-103">The 'l' suffix is easily confused with the digit '1' -- use 'L' for clarity</span></span>  
  
 <span data-ttu-id="38369-104">Kompilátor vás upozorní, když zjistí přetypování na malé písmeno l dlouho místo velké písmeno L.</span><span class="sxs-lookup"><span data-stu-id="38369-104">The compiler warns when it detects a cast to long using a lowercase l instead of an uppercase L.</span></span>  
  
 <span data-ttu-id="38369-105">Následující ukázka generuje CS0078:</span><span class="sxs-lookup"><span data-stu-id="38369-105">The following sample generates CS0078:</span></span>  
  
```csharp  
// CS0078.cs  
// compile with: /W:4  
class test {  
   public static void TestL(long i)  
   {  
   }  
  
   public static void TestL(int i)  
   {  
   }  
  
   public static void Main()  
   {  
      TestL(25l);   // CS0078  
      // try the following line instead  
      // TestL(25L);  
   }  
}  
```
