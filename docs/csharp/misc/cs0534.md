# <a name="compiler-error-cs0534"></a><span data-ttu-id="1b71c-102">Chyba kompilátoru CS0534</span><span class="sxs-lookup"><span data-stu-id="1b71c-102">Compiler Error CS0534</span></span>
<span data-ttu-id="1b71c-103">' Function1 ' neimplementuje zděděný abstraktní člen ' function2 '</span><span class="sxs-lookup"><span data-stu-id="1b71c-103">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="1b71c-104">Třída je vyžadována pro implementaci všech abstraktních [](../language-reference/keywords/abstract.md) členů v základní třídě, pokud není třída také abstraktní.</span><span class="sxs-lookup"><span data-stu-id="1b71c-104">A class is required to implement all the [abstract](../language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="1b71c-105">Následující ukázka generuje CS0534:</span><span class="sxs-lookup"><span data-stu-id="1b71c-105">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
