<span data-ttu-id="dfa99-103">'type' nelze označit jako kompatibilní se Specifikací CLS, protože se jedná o člen typu kompatibilním se Specifikací 'type'</span><span class="sxs-lookup"><span data-stu-id="dfa99-103">'type' cannot be marked as CLS-Compliant because it is a member of non CLS-compliant type 'type'</span></span>  
  
 <span data-ttu-id="dfa99-104">K tomuto upozornění dochází-li vnořené třídy s atributem CLSCompliant. nastavte na `true` je deklarováno jako člen třídy deklarované s atributem CLSCompliant. nastavte na `false`.</span><span class="sxs-lookup"><span data-stu-id="dfa99-104">This warning occurs if a nested class with the CLSCompliant attribute set to `true` is declared as a member of a class declared with the CLSCompliant attribute set to `false`.</span></span> <span data-ttu-id="dfa99-105">To není povoleno, protože vnořená třída nemůže být kompatibilní se Specifikací CLS, pokud se jedná o člen z vnější třídy, která není kompatibilní se Specifikací CLS.</span><span class="sxs-lookup"><span data-stu-id="dfa99-105">This is not allowed, since a nested class cannot be CLS-compliant if it is a member of an outer class that is not CLS-compliant.</span></span> <span data-ttu-id="dfa99-106">Pokud chcete vyřešit toto upozornění, odeberte atribut CLSCompliant z vnořené třídy nebo změňte z `true` k `false`.</span><span class="sxs-lookup"><span data-stu-id="dfa99-106">To resolve this warning, remove the CLSCompliant attribute from the nested class, or change it from `true` to `false`.</span></span> <span data-ttu-id="dfa99-107">Další informace o dodržování specifikace CLS naleznete v tématu [jazyková nezávislost a jazykově nezávislé komponenty](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="dfa99-107">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="dfa99-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="dfa99-108">Example</span></span>  
 <span data-ttu-id="dfa99-109">Následující ukázka generuje CS3018.</span><span class="sxs-lookup"><span data-stu-id="dfa99-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
