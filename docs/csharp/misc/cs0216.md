# <a name="compiler-error-cs0216"></a><span data-ttu-id="81e4e-102">CS0216 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="81e4e-102">Compiler Error CS0216</span></span>
<span data-ttu-id="81e4e-103">Operátor 'operator' vyžaduje odpovídající operátor 'missing_operator' aby byl rovněž definován</span><span class="sxs-lookup"><span data-stu-id="81e4e-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="81e4e-104">Uživatelem definované [true](../language-reference/keywords/true-false-operators.md) operátor vyžaduje uživatelem definované [false](../language-reference/keywords/true-false-operators.md) operátorů a naopak.</span><span class="sxs-lookup"><span data-stu-id="81e4e-104">A user-defined [true](../language-reference/keywords/true-false-operators.md) operator requires a user-defined [false](../language-reference/keywords/true-false-operators.md) operator, and vice versa.</span></span>
  
 <span data-ttu-id="81e4e-105">Následující ukázka generuje CS0216:</span><span class="sxs-lookup"><span data-stu-id="81e4e-105">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
