# <a name="compiler-error-cs0216"></a><span data-ttu-id="17bd6-102">Compiler Error CS0216</span><span class="sxs-lookup"><span data-stu-id="17bd6-102">Compiler Error CS0216</span></span>
<span data-ttu-id="17bd6-103">Operátor 'operator' vyžaduje odpovídající operátor 'missing_operator' aby byl rovněž definován</span><span class="sxs-lookup"><span data-stu-id="17bd6-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="17bd6-104">Uživatelem definované [true](../language-reference/operators/true-false-operators.md) operátor vyžaduje uživatelem definované [false](../language-reference/operators/true-false-operators.md) operátorů a naopak.</span><span class="sxs-lookup"><span data-stu-id="17bd6-104">A user-defined [true](../language-reference/operators/true-false-operators.md) operator requires a user-defined [false](../language-reference/operators/true-false-operators.md) operator, and vice versa.</span></span>
  
 <span data-ttu-id="17bd6-105">Následující ukázka generuje CS0216:</span><span class="sxs-lookup"><span data-stu-id="17bd6-105">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
