<span data-ttu-id="7f8ea-103">Vyrovnávací paměti pevné velikosti obsažené ve volném výrazu nejde použít.</span><span class="sxs-lookup"><span data-stu-id="7f8ea-103">You cannot use fixed size buffers contained in unfixed expressions.</span></span> <span data-ttu-id="7f8ea-104">Zkuste použít příkaz fixed.</span><span class="sxs-lookup"><span data-stu-id="7f8ea-104">Try using the fixed statement.</span></span>  
  
 <span data-ttu-id="7f8ea-105">K této chybě dochází, pokud používáte vyrovnávací paměť pevné velikosti ve výrazu zahrnující třídu, která sama oprava není.</span><span class="sxs-lookup"><span data-stu-id="7f8ea-105">This error occurs if you use the fixed sized buffer in an expression involving a class that is not itself fixed.</span></span> <span data-ttu-id="7f8ea-106">Modul runtime je volné nevyřešené třídy přesouvat optimalizovat přístup do paměti, což může vést k chybám při použití pevné velikosti vyrovnávací paměti.</span><span class="sxs-lookup"><span data-stu-id="7f8ea-106">The runtime is free to move the unfixed class around to optimize memory access, which could lead to errors when using the fixed sized buffer.</span></span> <span data-ttu-id="7f8ea-107">Chcete-li této chybě předejít, použijte `fixed` – klíčové slovo u příkazu.</span><span class="sxs-lookup"><span data-stu-id="7f8ea-107">To avoid this error, use the `fixed` keyword on the statement.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7f8ea-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="7f8ea-108">Example</span></span>  
 <span data-ttu-id="7f8ea-109">Následující ukázka generuje CS1666.</span><span class="sxs-lookup"><span data-stu-id="7f8ea-109">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);   
   }   
}  
```
