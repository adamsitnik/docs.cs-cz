# <a name="compiler-error-cs0682"></a>CS0682 chyby kompilátoru
'type1' nemůže implementovat 'type2', protože není jazykem podporován  
  
 K této chybě dochází, když se pokusí pro implementaci rozhraní v jiném jazyce a kompilátor nepodporuje rozhraní.
