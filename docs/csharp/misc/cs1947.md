# <a name="compiler-error-cs1947"></a>Chyba kompilátoru CS1947
Proměnné rozsahu s názvem proměnné nelze přiřadit – je jen pro čtení.  
  
 Proměnná rozsahu je jako iterační proměnná v `foreach` příkazu. Nedá se přiřadit do výrazu dotazu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte přiřazení k proměnné rozsahu.  
  
2. V případě potřeby zaveďte novou proměnnou rozsahu pomocí `let` klauzule a použijte ji k uložení hodnoty.  
  
## <a name="example"></a>Příklad  
 Následující kód generuje CS1947:  
  
```csharp  
// cs1947.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array  
                let k = i  
                select i = 5; // CS1947  
        x.ToList();  
    }  
}  
```
