## <a name="example"></a><span data-ttu-id="1d594-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="1d594-108">Example</span></span>  
 <span data-ttu-id="1d594-109">Následující kód vygeneruje CS1917:</span><span class="sxs-lookup"><span data-stu-id="1d594-109">The following code generates CS1917:</span></span>  
  
```csharp  
// cs1917.cs  
class CS1917  
{  
    public struct TestStruct  
    {  
        public int i;  
    }  
    public class C  
    {  
        public readonly TestStruct str = new TestStruct();  
        public static int Main()  
        {  
            C c = new C { str = { i = 1 } }; // CS1917  
            return 0;  
        }  
    }  
}  
```
