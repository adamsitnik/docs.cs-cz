# <a name="compiler-error-cs1594"></a>Chyba kompilátoru CS1594
Delegát Delegate má neplatné argumenty.  
  
 Typ argumentu předaného vyvolání [delegáta](../language-reference/keywords/delegate.md) nesouhlasí s typem parametru v deklaraci delegáta.  
  
 Následující ukázka generuje CS1594:  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
