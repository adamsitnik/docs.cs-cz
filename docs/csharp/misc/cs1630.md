# <a name="compiler-error-cs1630"></a>Compiler Error CS1630
Neplatná možnost "možnost" pro/errorreport; musí být řádku, odeslat, queue nebo none  
  
 Možnost příkazového řádku [/errorreport](../../csharp/language-reference/compiler-options/errorreport-compiler-option.md) musí být následován znakem **řádku**, **odeslat**, **fronty**, nebo **žádný**, určující jakou akci chcete provedené při výskytu vnitřní chyby kompilátoru.
