# <a name="compiler-error-cs1630"></a>Chyba kompilátoru CS1630
Neplatná možnost Option pro/errorreport; musí být Prompt, Send, Queue nebo None.  
  
 Parametr příkazového řádku [/errorreport](../language-reference/compiler-options/errorreport-compiler-option.md) musí následovat po příkazu **prompt**, **Send**, **Queue**nebo **none**a určení toho, jakou akci chcete provést při výskytu vnitřní chyby kompilátoru.
