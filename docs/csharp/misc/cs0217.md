# <a name="compiler-error-cs0217"></a>Compiler Error CS0217
Aby bylo možné použitelný jako operátor zkráceného vyhodnocení musí mít logický operátor ('operator') uživatelem definované vracet hodnotu stejného typu jako jeho 2 parametry.  
  
 Je-li definovat operátor pro typ definovaný uživatelem a pak zkuste použít operátor jako zkrácenou operátor, uživatelem definovaný operátor musí mít parametry a návratové hodnoty stejného typu. Další informace o zkrácenou operátory, naleznete v tématu [& & – operátor](../../csharp/language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) a [ &#124; &#124; operátor](../../csharp/language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-). Další informace o uživatelem definované zkrácenou, nebo podmíněného, operátory, najdete [podmíněné logické operátory definované uživatelem](~/_csharplang/spec/expressions.md#user-defined-conditional-logical-operators) část [ C# specifikace jazyka](~/_csharplang/spec/introduction.md).
  
 Následující ukázka generuje CS0217:  
  
```csharp  
// CS0217.cs  
using System;  
  
public class MyClass  
{  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static int operator & (MyClass f1, MyClass f2)   // CS0217  
   // try the following line instead  
   // public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;  
   }  
}  
```
