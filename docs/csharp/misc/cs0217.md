# <a name="compiler-error-cs0217"></a><span data-ttu-id="b071f-102">CS0217 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="b071f-102">Compiler Error CS0217</span></span>
<span data-ttu-id="b071f-103">Aby bylo možné použitelný jako operátor zkráceného vyhodnocení musí mít logický operátor ('operator') uživatelem definované vracet hodnotu stejného typu jako jeho 2 parametry.</span><span class="sxs-lookup"><span data-stu-id="b071f-103">In order to be applicable as a short circuit operator a user-defined logical operator ('operator') must have the same return type as the type of its 2 parameters.</span></span>  
  
 <span data-ttu-id="b071f-104">Je-li definovat operátor pro typ definovaný uživatelem a pak zkuste použít operátor jako zkrácenou operátor, uživatelem definovaný operátor musí mít parametry a návratové hodnoty stejného typu.</span><span class="sxs-lookup"><span data-stu-id="b071f-104">If you define an operator for a user-defined type, and then try to use the operator as a short-circuit operator, the user-defined operator must have parameters and return values of the same type.</span></span> <span data-ttu-id="b071f-105">Další informace o zkrácenou operátory, naleznete v tématu [& & – operátor](../../csharp/language-reference/operators/conditional-and-operator.md) a [ &#124; &#124; operátor](../../csharp/language-reference/operators/conditional-or-operator.md).</span><span class="sxs-lookup"><span data-stu-id="b071f-105">For more information on short-circuit operators, see [&& Operator](../../csharp/language-reference/operators/conditional-and-operator.md) and [&#124;&#124; Operator](../../csharp/language-reference/operators/conditional-or-operator.md).</span></span>  
  
 <span data-ttu-id="b071f-106">Následující ukázka generuje CS0217:</span><span class="sxs-lookup"><span data-stu-id="b071f-106">The following sample generates CS0217:</span></span>  
  
```csharp  
// CS0217.cs  
using System;  
  
public class MyClass  
{  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static int operator & (MyClass f1, MyClass f2)   // CS0217  
   // try the following line instead  
   // public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;  
   }  
}  
```
