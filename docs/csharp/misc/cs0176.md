# <a name="compiler-error-cs0176"></a>Compiler Error CS0176
Statického člena 'member' nelze přistupovat pomocí odkazu na instanci; Místo toho ho kvalifikovat s názvem typu  
  
 Pouze název třídy lze použít k vyfiltrování [statické](../../csharp/language-reference/keywords/static.md) proměnné; Název instance nemůže být kvalifikátoru. Další informace najdete v tématu [statické třídy a statické členy třídy](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md).  
  
 Následující ukázka generuje CS0176:  
  
```csharp  
// CS0176.cs  
public class MyClass2  
{  
    public static int num;  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        MyClass2 mc2 = new MyClass2();  
        int i = mc2.num;   // CS0176  
        // try the following line instead  
        // int i = MyClass2.num;  
    }  
}  
```
