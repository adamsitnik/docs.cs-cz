# <a name="compiler-error-cs0155"></a><span data-ttu-id="1c33e-102">Compiler Error CS0155</span><span class="sxs-lookup"><span data-stu-id="1c33e-102">Compiler Error CS0155</span></span>
<span data-ttu-id="1c33e-103">Zachycený nebo vyvolaný typ musí být odvozen od třídy System.Exception.</span><span class="sxs-lookup"><span data-stu-id="1c33e-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="1c33e-104">Byl proveden pokus o předání datový typ, který není odvozen od **System.Exception** do [catch](../../csharp/language-reference/keywords/try-catch.md) bloku.</span><span class="sxs-lookup"><span data-stu-id="1c33e-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../../csharp/language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="1c33e-105">Pouze datové typy, které jsou odvozeny z **System.Exception** mohou být předány do **catch** bloku.</span><span class="sxs-lookup"><span data-stu-id="1c33e-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="1c33e-106">Další informace najdete v tématu [příkazy zpracování výjimek](../../csharp/language-reference/keywords/exception-handling-statements.md) a [výjimek a zpracování výjimek](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="1c33e-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="1c33e-107">Následující ukázka generuje CS0155:</span><span class="sxs-lookup"><span data-stu-id="1c33e-107">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
