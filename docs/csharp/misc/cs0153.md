# <a name="compiler-error-cs0153"></a>Compiler Error CS0153
Příkaz goto case je platný jenom uvnitř příkazu switch  
  
 Byl proveden pokus o použití [přepnout](../../csharp/language-reference/keywords/switch.md) syntaxe mimo **přepnout** příkazu. Další informace najdete v tématu [přepnout](../../csharp/language-reference/keywords/switch.md).  
  
 Následující ukázka generuje CS0153:  
  
```csharp  
// CS0153.cs  
public class a  
{  
   public static void Main()  
   {  
      goto case 5;   // CS0153  
   }  
}  
```
