<span data-ttu-id="cd7b5-103">Uživatelem definovaný operátor nemůže převzít objekt nadřazeného typu a proveďte převod na objekt nadřazeného typu.</span><span class="sxs-lookup"><span data-stu-id="cd7b5-103">User-defined operator cannot take an object of the enclosing type and convert to an object of the enclosing type</span></span>

{

   {
