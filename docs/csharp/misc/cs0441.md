# <a name="compiler-error-cs0441"></a>Compiler Error CS0441
'class': třída nemůže být zároveň statická a zapečetěná  
  
 Tato chyba nastane, pokud deklarujete třídu, která je statická a zapečetěná. Statické třídy jsou ze své podstaty zapečetěné, takže není nutné zapečetěným modifikátorem zabráněno. Pokud chcete vyřešit, použijte pouze jeden modifikátor.  
  
 Následující příklad generuje CS0441:  
  
```csharp  
// CS0441.cs  
sealed static class MyClass { }   // CS0441  
```
