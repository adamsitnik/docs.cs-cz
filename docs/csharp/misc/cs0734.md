# <a name="compiler-error-cs0734"></a>Compiler Error CS0734
Možnost/moduleassemblyname jde zadat jenom při sestavování s cílovým typem 'module'  
  
 Možnost kompilátoru **/moduleassemblyname** by měla sloužit pouze při sestavování .netmodule. Zobrazit [/moduleassemblyname (C# – možnost kompilátoru)](../../csharp/language-reference/compiler-options/moduleassemblyname-compiler-option.md) Další informace.  
  
 Další informace o vytváření .netmodule, naleznete v tématu [/target: module (C# – možnosti kompilátoru)](../../csharp/language-reference/compiler-options/target-module-compiler-option.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0734. Chcete-li vyřešit, přidejte **/target: Module** do kompilace.  
  
```csharp  
// CS0734.cs  
// compile with: /moduleassemblyname:A  
// CS0734 expected  
public class Test {}  
```
