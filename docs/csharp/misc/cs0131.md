 <span data-ttu-id="26462-104">V příkazu přiřazení je přiřazena hodnota pravé straně na levé straně.</span><span class="sxs-lookup"><span data-stu-id="26462-104">In an assignment statement, the value of the right-hand side is assigned to the left-hand side.</span></span> <span data-ttu-id="26462-105">Levá strana musí být proměnná, vlastnost nebo indexer.</span><span class="sxs-lookup"><span data-stu-id="26462-105">The left-hand side must be a variable, property, or indexer.</span></span>  
  
 <span data-ttu-id="26462-106">Pokud chcete tuto chybu opravit, ujistěte se, že všechny operátory jsou na pravé straně a zda na levé straně proměnná, vlastnost nebo indexer.</span><span class="sxs-lookup"><span data-stu-id="26462-106">To fix this error, make sure that all operators are on the right-hand side and that the left-hand side is a variable, property, or indexer.</span></span> <span data-ttu-id="26462-107">Další informace najdete v tématu [příkazy, výrazy a operátory](../../csharp/programming-guide/statements-expressions-operators/index.md).</span><span class="sxs-lookup"><span data-stu-id="26462-107">For more information, see [Statements, Expressions, and Operators](../../csharp/programming-guide/statements-expressions-operators/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="26462-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="26462-108">Example</span></span>  
 <span data-ttu-id="26462-109">Následující ukázka generuje CS0131.</span><span class="sxs-lookup"><span data-stu-id="26462-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="26462-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="26462-110">Example</span></span>  
 <span data-ttu-id="26462-111">K této chybě může dojít také při pokusu o provedení aritmetické operace na levé straně operátoru přiřazení, jako v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="26462-111">This error can also occur if you attempt to perform arithmetic operations on the left hand side of an assignment operator, as in the following example.</span></span>

```csharp  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)

        return 1;  
    }  
}  
```
