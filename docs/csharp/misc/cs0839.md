# <a name="compiler-error-cs0839"></a>Compiler Error CS0839
Chybí argument.  
  
 Argument nebyl nalezen ve volání metody.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Double podpisu metody a vyhledejte a zadejte chybí argument.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0839:  
  
```csharp  
// cs0839.cs  
using System;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Add(int i, int j)  
        {  
            return i + j;  
        }  
  
        static int Main()   
        {  
            int i = Test.Add( , 5); // CS0839  
            return 1;  
  
        }  
    }  
}  
```
