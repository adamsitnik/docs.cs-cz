# <a name="compiler-error-cs1648"></a>Compiler Error CS1648
Členy pole jen pro čtení 'identifier' nelze upravovat (kromě v konstruktoru nebo inicializátoru proměnné)  
  
 Tato chyba nastane, pokud při pokusu upravit o člen pole, která je jen pro čtení ve kterém není povoleno upravovat. Chcete-li vyřešit tuto chybu, omezení přiřazení pole jen pro čtení do konstruktoru nebo inicializátoru proměnné nebo odeberte readonly – klíčové slovo z deklarace pole.  
  
 Následující ukázka generuje CS1648:  
  
```csharp  
// CS1648.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{    
  public readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void Main()  
   {  
      Outer outer = new Outer();  
      outer.inner.i = 1;  // CS1648  
   }  
}  
```
