# <a name="compiler-error-cs1648"></a><span data-ttu-id="d386b-102">Compiler Error CS1648</span><span class="sxs-lookup"><span data-stu-id="d386b-102">Compiler Error CS1648</span></span>
<span data-ttu-id="d386b-103">Členy pole jen pro čtení 'identifier' nelze upravovat (kromě v konstruktoru nebo inicializátoru proměnné)</span><span class="sxs-lookup"><span data-stu-id="d386b-103">Members of readonly field 'identifier' cannot be modified (except in a constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="d386b-104">Tato chyba nastane, pokud při pokusu upravit o člen pole, která je jen pro čtení ve kterém není povoleno upravovat.</span><span class="sxs-lookup"><span data-stu-id="d386b-104">This error occurs when you attempt to modify a member of a field which is readonly where it is not allowed to be modified.</span></span> <span data-ttu-id="d386b-105">Chcete-li vyřešit tuto chybu, omezení přiřazení pole jen pro čtení do konstruktoru nebo inicializátoru proměnné nebo odeberte readonly – klíčové slovo z deklarace pole.</span><span class="sxs-lookup"><span data-stu-id="d386b-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the readonly keyword from the declaration of the field.</span></span>  
  
 <span data-ttu-id="d386b-106">Následující ukázka generuje CS1648:</span><span class="sxs-lookup"><span data-stu-id="d386b-106">The following sample generates CS1648:</span></span>  
  
```csharp  
// CS1648.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{    
  public readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void Main()  
   {  
      Outer outer = new Outer();  
      outer.inner.i = 1;  // CS1648  
   }  
}  
```
