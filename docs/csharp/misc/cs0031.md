# <a name="compiler-error-cs0031"></a><span data-ttu-id="4741a-102">Chyba kompilátoru CS0031</span><span class="sxs-lookup"><span data-stu-id="4741a-102">Compiler Error CS0031</span></span>
<span data-ttu-id="4741a-103">Konstantní hodnotu Value nelze převést na typ.</span><span class="sxs-lookup"><span data-stu-id="4741a-103">Constant value 'value' cannot be converted to a 'type'.</span></span> <span data-ttu-id="4741a-104">(k přepsání použijte syntaxi ' unchecked ')</span><span class="sxs-lookup"><span data-stu-id="4741a-104">(use 'unchecked' syntax to override)</span></span>  
  
 <span data-ttu-id="4741a-105">Byl proveden pokus o přiřazení hodnoty proměnné, jejíž typ nemůže uložit hodnotu.</span><span class="sxs-lookup"><span data-stu-id="4741a-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="4741a-106">Další informace najdete v tématu [typy](../programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="4741a-106">For more information, see [Types](../programming-guide/types/index.md).</span></span>  
  
 <span data-ttu-id="4741a-107">Následující ukázka generuje CS0031 v zaškrtnutých i nekontrolovaných kontextech:</span><span class="sxs-lookup"><span data-stu-id="4741a-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>  
  
```csharp  
// CS0031.cs  
namespace CS0031  
{  
   public class a  
   {  
      public static void Main()  
      {  
         int num = (int)2147483648M; //CS0031  
         // Try using a larger numeric type instead:  
         // long num = (long)2147483648M; //CS0031  
  
         const decimal d = -10M; // Decimal literal  
            unchecked  
            {  
                const byte b = (byte)d; // CS0031  
                // For small values try using a signed byte instead:  
                // const sbyte b = (sbyte)d;  
            }  
      }  
   }  
}  
```
