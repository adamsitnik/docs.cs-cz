# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="b945e-102">Kompilátor CS1720 upozornění (úroveň 1)</span><span class="sxs-lookup"><span data-stu-id="b945e-102">Compiler Warning (level 1) CS1720</span></span>
<span data-ttu-id="b945e-103">Výraz způsobí výjimku System.NullReferenceException protože 'obecného typu' výchozí hodnota je null.</span><span class="sxs-lookup"><span data-stu-id="b945e-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="b945e-104">Pokud píšete výrazem zahrnujícím výchozí hodnotu proměnné obecného typu, který je typem odkazu (například třídy), dojde k této chybě.</span><span class="sxs-lookup"><span data-stu-id="b945e-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="b945e-105">Vezměte v úvahu následující výraz:</span><span class="sxs-lookup"><span data-stu-id="b945e-105">Consider the following expression:</span></span>  
  
```csharp  
default(T).ToString()  
```  
  
 <span data-ttu-id="b945e-106">Protože `T` je typem odkazu, jeho výchozí hodnota je null a to při pokusu o použití <xref:System.Object.ToString%2A> vyvolá metodu k němu <xref:System.NullReferenceException>.</span><span class="sxs-lookup"><span data-stu-id="b945e-106">Since `T` is a reference type, its default value is null, and so attempting to apply the <xref:System.Object.ToString%2A> method to it will throw a <xref:System.NullReferenceException>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b945e-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="b945e-107">Example</span></span>

 <span data-ttu-id="b945e-109">Následující ukázka generuje CS1720.</span><span class="sxs-lookup"><span data-stu-id="b945e-109">The following sample generates CS1720.</span></span>  
  
```csharp  
// CS1720.cs  
using System;  
public class Tester   
{  
    public static void GenericClass<T>(T t1) where T : class   
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
