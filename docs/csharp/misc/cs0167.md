# <a name="compiler-error-cs0167"></a>Chyba kompilátoru CS0167
V delegátu Delegate chybí metoda Invoke.  
  
 Naimportovali a použili jste spravovaný program (ten, který používá .NET Framework Common Language Runtime), který byl vytvořen s jiným kompilátorem. Tento kompilátor povoluje [delegáta](../language-reference/builtin-types/reference-types.md)s chybnou strukturou. Proto metoda `Invoke` nebyla k dispozici. Další informace najdete v tématu [Delegáti](../programming-guide/delegates/index.md).
