# <a name="compiler-error-cs0167"></a>CS0167 chyby kompilátoru
Delegáta 'delegát' neobsahuje metodu Invoke  
  
 Importovat a používat spravované aplikace (jedna, která používá rozhraní .NET Framework common language runtime), který byl vytvořen s jiným kompilátorem. Tento kompilátor povolené chybně vytvořený [delegovat](../../csharp/language-reference/keywords/delegate.md). Proto `Invoke` metode nebyla k dispozici. Další informace najdete v tématu [delegáti](../../csharp/programming-guide/delegates/index.md).
