# <a name="compiler-error-cs1553"></a>Compiler Error CS1553
Deklarace není platná. Použijte ' operátor modifikátor \<dest-type > (...' Místo toho  
  
Návratový typ [operátor převodu](../../csharp/language-reference/operators/user-defined-conversion-operators.md) musí bezprostředně předcházet seznam parametrů a *modifikátor* je buď `implicit` nebo `explicit`.
  
Následující ukázka generuje CS1553:  
  
```csharp  
// CS1553.cs  
class MyClass  
{  
   public static int implicit operator (MyClass f)   // CS1553  
   // try the following line instead  
   // public static implicit operator int (MyClass f)  
   {  
      return 6;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
