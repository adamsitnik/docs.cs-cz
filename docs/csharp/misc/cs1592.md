# <a name="compiler-warning-level-1-cs1592"></a>Upozornění kompilátoru (úroveň 1) CS1592
Chybně vytvořený kód XML v zahrnutém souboru komentářů – "důvod"  
  
 Problém, který byl hlášen jako ***důvod***, byl nalezen v souboru určeném pomocí [ \<tagu > include](../programming-guide/xmldoc/include.md) .
