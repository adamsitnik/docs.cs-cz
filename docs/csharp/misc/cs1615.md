# <a name="compiler-error-cs1615"></a>CS1615 chyby kompilátoru
Argument 'number' by neměl být předáván s klíčovým slovem '– klíčové slovo.  
  
 Jeden z klíčových slov `ref` nebo **si** byla použita při funkci nebere `ref` nebo **si** parametr pro tento argument. Chcete-li tuto chybu vyřešit, odeberte nesprávné – klíčové slovo a použijte odpovídající klíčové slovo, který odpovídá deklaraci funkce, pokud existuje.  
  
 Následující ukázka generuje CS1615:  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
