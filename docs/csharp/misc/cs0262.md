# <a name="compiler-error-cs0262"></a>CS0262 chyby kompilátoru
Částečné deklarace 'type' mají konfliktní modifikátory dostupnosti  
  
 K této chybě dochází, pokud částečného typu má nekonzistentní modifikátory, jako jsou veřejné, privátní, chráněný, interní nebo abstraktní. Tyto modifikátory musí být konzistentní vzhledem k aplikacím ve všech částečné deklarace pro daný typ. Další informace najdete v tématu [částečné třídy a metody](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0262:  
  
```csharp  
// CS0262.cs  
class A  
{  
    public partial class C  // CS0262  
    {  
    }  
    private partial class C  
    {  
    }  
}  
```
