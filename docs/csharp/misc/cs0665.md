# <a name="compiler-warning-level-3-cs0665"></a>Kompilátor CS0665 upozornění (úroveň 3)
Je přiřazení v podmíněných výrazech vždy konstantní. nechtěli jste použít operátor == místo = operátoru?  
  
 Podmíněný výraz používá [= – operátor](../../csharp/language-reference/operators/assignment-operator.md) a ne [== – operátor](../../csharp/language-reference/operators/equality-operators.md#equality-operator-).
  
 Následující ukázka generuje CS0665:  
  
```csharp  
// CS0665.cs  
// compile with: /W:3  
class Test  
{  
   public static void Main()  
   {  
      bool i = false;  
  
      if (i = true)   // CS0665  
      // try the following line instead  
      // if (i == true)  
      {  
      }  
  
      System.Console.WriteLine(i);  
   }  
}  
```
