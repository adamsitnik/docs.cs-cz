# <a name="compiler-error-cs0544"></a>Compiler Error CS0544
'přepsání vlastnosti': nelze přepsat, protože 'bez property' se nenachází ve vlastnosti  
  
 Byl proveden pokus o přepsání nonproperty datový typ jako [vlastnost](../../csharp/programming-guide/classes-and-structs/properties.md), což není povoleno.  
  
 Následující ukázka generuje CS0544:  
  
```csharp  
// CS0544.cs  
// compile with: /target:library  
public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
