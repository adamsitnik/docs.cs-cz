# <a name="compiler-error-cs5001"></a>Chyba kompilátoru CS5001

Program neobsahuje statickou metodu Main vhodnou pro vstupní bod.

K této chybě dochází, pokud v kódu, který vytváří spustitelný soubor, není nalezena žádná statická metoda `Main` se správným podpisem. K tomuto problému dochází, pokud je funkce vstupního bodu `Main` definována s nesprávným případem, například s malým případem `main`. Informace o pravidlech, která se vztahují na metodu `Main` naleznete v tématu [Main () a argumenty příkazového řádku](../programming-guide/main-and-command-args/index.md).

Pokud má metoda `Main` modifikátor `async`, ujistěte se, že je [ C# vybraná jazyková verze](../language-reference/configure-language-version.md) 7,1 nebo vyšší.

Metoda `Main` je požadována pouze při kompilování spustitelného souboru, tj. je-li zadána možnost kompilátoru [-target: exe](/dotnet/csharp/language-reference/compiler-options/target-exe-compiler-option) nebo [-target: winexe](/dotnet/csharp/language-reference/compiler-options/target-winexe-compiler-option) . Následující typy projektů sady Visual Studio určují ve výchozím nastavení jednu z těchto možností:

- Konzolová aplikace
- ASP.NET Core aplikace
- aplikace WPF
- Windows Forms – aplikace

## <a name="example"></a>Příklad

Následující příklad generuje CS5001:
  
```csharp
// CS5001.cs
// CS5001 expected when compiled with -target:exe or -target:winexe
public class Program
{
   // Uncomment the following line to resolve.
   // static void Main() {}
}
```
