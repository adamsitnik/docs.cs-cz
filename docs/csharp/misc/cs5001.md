# <a name="compiler-error-cs5001"></a>Chyba kompilátoru CS5001

Program neobsahuje statickou metodu Main vhodnou pro vstupní bod.  
  
K této chybě dochází, pokud `Main` v kódu, který vytváří spustitelný soubor, není nalezena žádná statická metoda se správným podpisem. K této chybě dochází také v případě, `Main`že funkce vstupního bodu je definována s nesprávným případem, například malými písmeny. `main` Informace o pravidlech, která platí pro `Main` metodu, naleznete v tématu [Main () a argumenty příkazového řádku](../programming-guide/main-and-command-args/index.md).

Pokud má metoda modifikátor, ujistěte se, že je [ C# vybraná verze jazyka](../language-reference/configure-language-version.md) 7,1 nebo vyšší. `async` `Main`

Metoda je vyžadována pouze při kompilování spustitelného souboru, to znamená, že je zadána možnost kompilátoru when [-target: exe](/dotnet/csharp/language-reference/compiler-options/target-exe-compiler-option) nebo [-target: winexe.](/dotnet/csharp/language-reference/compiler-options/target-winexe-compiler-option) `Main` Následující typy projektů sady Visual Studio určují ve výchozím nastavení jednu z těchto možností:

- Konzolová aplikace
- ASP.NET Core aplikace
- aplikace WPF
- Windows Forms – aplikace

## <a name="example"></a>Příklad  
 Následující příklad generuje CS5001:  
  
```csharp  
// CS5001.cs  
// CS5001 expected when compiled with -target:exe or -target:winexe 
public class a  
{  
   // Uncomment the following line to resolve.  
   // static void Main() {}  
}  
```
