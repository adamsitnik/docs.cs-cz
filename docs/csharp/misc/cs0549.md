# <a name="compiler-error-cs0549"></a><span data-ttu-id="2eff1-102">Chyba kompilátoru CS0549</span><span class="sxs-lookup"><span data-stu-id="2eff1-102">Compiler Error CS0549</span></span>
<span data-ttu-id="2eff1-103">' function ' je nový virtuální člen v zapečetěné třídě ' class '</span><span class="sxs-lookup"><span data-stu-id="2eff1-103">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="2eff1-104">[Zapečetěná](../language-reference/keywords/sealed.md)[Třída](../language-reference/keywords/class.md) se nedá použít jako základní třída.</span><span class="sxs-lookup"><span data-stu-id="2eff1-104">A [sealed](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="2eff1-105">Proto je nevhodné mít virtuální metodu v zapečetěné třídě.</span><span class="sxs-lookup"><span data-stu-id="2eff1-105">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="2eff1-106">Následující ukázka generuje CS0549:</span><span class="sxs-lookup"><span data-stu-id="2eff1-106">The following sample generates CS0549:</span></span>

```csharp  
{  
   virtual public void TestMethod() {}   // CS0549

}  
```
