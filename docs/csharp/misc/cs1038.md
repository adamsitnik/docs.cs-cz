# <a name="compiler-error-cs1038"></a>Compiler Error CS1038
\#očekávala se direktiva endregion  
  
 A [#region](../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) – direktiva nemá odpovídající [#endregion](../../csharp/language-reference/preprocessor-directives/preprocessor-endregion.md) směrnice.  
  
 Následující ukázka generuje CS1038:  
  
```csharp  
// CS1038.cs  
#region testing  
  
public class clx  
{  
   public static void Main()  
   {  
   }  
}  
// CS1038  
// uncomment the next line to resolve  
// #endregion  
```
