# <a name="compiler-error-cs0274"></a>Compiler Error CS0274
Nelze zadat modifikátory dostupnosti pro přistupující objekty jak vlastnost nebo indexovací člen 'vlastnost nebo indexer.  
  
 K této chybě dochází, když deklarujete vlastnost nebo indexer s modifikátory přístupu na jejich přístupových objektů. Chcete-li vyřešit tuto chybu, použijte modifikátor přístupu na jenom jednom dva přistupující objekty. Další informace najdete v tématu [přístupnosti přístupového objektu](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md).  
  
 Následující příklad generuje CS0274:  
  
```csharp  
// CS0274.cs  
public class MyClass  
{  
    public int Property   // CS0274  
    {  
        public get { return 0; }  
        protected set { }  
    }  
}  
```
