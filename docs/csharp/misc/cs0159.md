# <a name="compiler-error-cs0159"></a>CS0159 chyby kompilátoru
Popisek "štítek" v rámci oboru příkazu goto  
  
 Popisek, který odkazuje [goto](../../csharp/language-reference/keywords/goto.md) příkaz nebyl nalezen v rámci oboru `goto` příkazu.  
  
 Následující ukázka generuje CS0159:  
  
```csharp  
// CS0159.cs  
public class Class1  
{  
   public static void Main()  
   {  
      int i = 0;  
  
      switch (i)  
      {  
         case 1:  
            goto case 3;   // CS0159, case 3 label does not exist  
         case 2:  
            break;  
      }  
      goto NOWHERE;   // CS0159, NOWHERE label does not exist  
   }  
}  
```
