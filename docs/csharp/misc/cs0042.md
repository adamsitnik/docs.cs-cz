# <a name="compiler-error-cs0042"></a>CS0042 chyby kompilátoru
Neočekávaná chyba při vytváření souboru 'file' ladicí informace – 'důvod:  
  
 Kompilátor nemohl vytvořit ladicí informace; *důvod* obsahuje další informace o chybového stavu.
