# <a name="compiler-error-cs0811"></a>Compiler Error CS0811
Plně kvalifikovaný název pro "name" je příliš dlouhý pro informace o ladění. Proveďte kompilaci bez možnost "/ debug".  
  
 Existují omezení velikosti pro proměnnou a zadejte názvy v ladicí informace.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Pokud není možné Změna názvu, jedinou možností je pro kompilaci bez [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) možnost.
