# <a name="compiler-error-cs1020"></a>Compiler Error CS1020
Očekával se přetěžovatelný binární operátor  
  
 Byl proveden pokus o definování přetížení operátoru, ale nebyla operátor, který se očekával se přetěžovatelný binární operátor, který přebírá dva parametry. Seznam přetížitelné operátory, najdete v článku [přetížitelné operátory](../../csharp/language-reference/operators/operator-overloading.md#overloadable-operators) část [přetížení operátoru](../../csharp/language-reference/operators/operator-overloading.md) článku.
  
 Následující ukázka generuje CS1020:  
  
```csharp  
// CS1020.cs  
public class iii  
{  
   public static int operator ++(iii aa, int bb)   // CS1020, change ++ to +  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
