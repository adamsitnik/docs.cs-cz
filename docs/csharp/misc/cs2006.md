# <a name="compiler-error-cs2006"></a>Chyba kompilátoru CS2006
Chyba syntaxe příkazového řádku: Chybí parametr text pro možnost Option.  
  
 Syntaxe *Možnosti* vyžaduje další text. Informace naleznete v tématu [Možnosti kompilátoru](../language-reference/compiler-options/index.md).
