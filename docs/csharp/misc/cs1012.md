# <a name="compiler-error-cs1012"></a>CS1012 chyby kompilátoru
Příliš moc znaků ve znakovém literálu  
  
 Byl proveden pokus o inicializaci [char](../../csharp/language-reference/keywords/char.md) konstanty s více než jeden znak.  
  
 CS1012 může vzniknout také při vytváření datových vazeb. Například následující řádek vám poskytne chybu:  
  
 `<%# DataBinder.Eval(Container.DataItem, 'doctitle') %>`  
  
 Místo toho vyzkoušejte následující řádek:  
  
 `<%# DataBinder.Eval(Container.DataItem, "doctitle") %>`  
  
 Následující ukázka generuje CS1012:  
  
```csharp  
// CS1012.cs  
class Sample  
{  
   static void Main()  
   {  
      char a = 'xx';   // CS1012  
      char a2 = 'x';   // OK  
      System.Console.WriteLine(a2);  
   }  
}  
```
