# <a name="compiler-error-cs2024"></a>Chyba kompilátoru CS2024
Neplatné číslo zarovnání oddílu souboru #  
  
 Možnosti kompilátoru [/filealign](../language-reference/compiler-options/filealign-compiler-option.md) byla předána neplatná hodnota.  
  
 Následující ukázka generuje CS2024:  
  
```csharp  
// CS2024.cs  
// compile with: /filealign:ex  
// CS2024 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
