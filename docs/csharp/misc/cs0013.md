# <a name="compiler-error-cs0013"></a>Compiler Error CS0013
Neočekávaná chyba při zápisu metadat do souboru 'ModelStore.dll'--' již není logické místo vytvoření dalších uživatelských řetězců.  
  
 Modul common language runtime rozhraní .NET Framework není generování metadat. Zkontrolujte, zda je cesta správná a zda není disk plný. Pokud se problém nevyřeší, budete muset opravte nebo přeinstalujte Visual Studio a/nebo rozhraní .NET Framework.
