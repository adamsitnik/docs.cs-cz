# <a name="compiler-error-cs0200"></a>CS0200 chyby kompilátoru
Vlastnost nebo indexovací člen 'property' nelze přiřadit – je jen pro čtení  
  
 Byl proveden pokus o přiřazení hodnoty k [vlastnost](../../csharp/programming-guide/classes-and-structs/using-properties.md), ale vlastnost nemá přistupující objekt set. Vyřešte chybu tak, že přidáte přístupový objekt set. Další informace najdete v tématu [jak: Deklarování a použití vlastností čtení/zápisu](../../csharp/programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0200:  
  
```csharp  
// CS0200.cs  
public class MainClass  
{  
    // private int _mi;  
    int I  
    {  
        get  
        {  
            return 1;  
        }  
  
        // uncomment the set accessor and declaration for _mi  
        /*  
        set  
        {  
            _mi = value;  
        }  
        */  
    }  
  
    public static void Main ()  
    {  
        MainClass II = new MainClass();  
        II.I = 9;   // CS0200  
    }  
}  
```
