 <span data-ttu-id="84384-104">K tomuto upozornění dochází `[CLSCompliant(false)]` , pokud se zobrazí u třídy v sestavení, které nemá atribut CLSCompliant na úrovni sestavení nastaven na hodnotu true (tj. řádek `[assembly: CLSCompliant(true)]`).</span><span class="sxs-lookup"><span data-stu-id="84384-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="84384-105">Vzhledem k tomu, že sestavení neprovádí deklaraci samotné specifikace CLS, není potřeba nic v rámci sestavení, aby bylo možné deklarovat nekompatibilní, protože se předpokládá, že nedodržuje předpisy.</span><span class="sxs-lookup"><span data-stu-id="84384-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="84384-106">Další informace o kompatibilitě se specifikací CLS naleznete v tématu [nezávislost jazyků a jazykově nezávislé komponenty](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="84384-106">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>

## <a name="example"></a><span data-ttu-id="84384-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="84384-108">Example</span></span>  
 <span data-ttu-id="84384-109">Následující příklad generuje CS3021:</span><span class="sxs-lookup"><span data-stu-id="84384-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
