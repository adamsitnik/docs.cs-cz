## <a name="example"></a><span data-ttu-id="df006-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="df006-106">Example</span></span>  
 <span data-ttu-id="df006-107">Následující ukázka generuje CS1511.</span><span class="sxs-lookup"><span data-stu-id="df006-107">The following sample generates CS1511.</span></span>  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
