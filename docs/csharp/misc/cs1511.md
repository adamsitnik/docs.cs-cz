# <a name="compiler-error-cs1511"></a>Compiler Error CS1511
Klíčové slovo "základní" není k dispozici uvnitř statické metody.  
  
 [Základní](../../csharp/language-reference/keywords/base.md) klíčové slovo se používá v [statické](../../csharp/language-reference/keywords/static.md) metody. `base` lze volat pouze v konstruktoru instance, metoda instance nebo přístupový objekt instance.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1511.  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
