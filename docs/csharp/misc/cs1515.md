# <a name="compiler-error-cs1515"></a>Chyba kompilátoru CS1515
očekávalo se klíčové slovo in.  
  
 V příkazu [foreach v](../language-reference/keywords/foreach-in.md) příkazu, chybí část in.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1515:  
  
```csharp 
using System;  
  
class Driver  
{  
   static void Main()  
   {  
      int[] arr = new int[] {1, 2, 3};  
  
      // try the following line instead  
      // foreach (int x in arr)  
      foreach (int x arr)      // CS1515, "in" is missing  
      {  
         Console.WriteLine(x);  
      }  
   }  
}  
```
