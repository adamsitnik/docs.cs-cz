# <a name="compiler-error-cs1515"></a>Compiler Error CS1515
očekáváno "in"  
  
 V [foreach v](../../csharp/language-reference/keywords/foreach-in.md) prohlášení, chybí část "in".  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1515:  
  
```csharp 
using System;  
  
class Driver  
{  
   static void Main()  
   {  
      int[] arr = new int[] {1, 2, 3};  
  
      // try the following line instead  
      // foreach (int x in arr)  
      foreach (int x arr)      // CS1515, "in" is missing  
      {  
         Console.WriteLine(x);  
      }  
   }  
}  
```
