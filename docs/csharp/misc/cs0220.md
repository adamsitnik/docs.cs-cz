# <a name="compiler-error-cs0220"></a>Compiler Error CS0220
Došlo k přetečení v době kompilace v režimu kontroly  
  
 Byla zjištěna operace [zaškrtnutí](../../csharp/language-reference/keywords/checked.md), což je výchozí a výsledkem je ztráta dat. Buď opravte vstupy přiřazení nebo použití [Nekontrolovaná](../../csharp/language-reference/keywords/unchecked.md) pro vyřešení této chyby. Další informace najdete v tématu [zaškrtnuto a nezaškrtnuto](../../csharp/language-reference/keywords/checked-and-unchecked.md).  
  
 Následující ukázka generuje CS0220:  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
