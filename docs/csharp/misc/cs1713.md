# <a name="compiler-error-cs1713"></a>CS1713 chyby kompilátoru
Došlo k neočekávané chybě vytváření názvu metadat pro typ NázevTypu1 – 'důvod:  
  
 Tato chyba je často způsobeno vnitřní chyba kompilátoru. Provedení nějaké drobné změny kódu, například zkrácení délka názvu a poté opětovné kompilaci, může tuto chybu vyřešit.
