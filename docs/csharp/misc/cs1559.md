# <a name="compiler-error-cs1559"></a>Chyba kompilátoru CS1559
Klíčové slovo Object nejde použít pro metodu Main, protože je importovaný.  
  
 Pro možnost kompilátoru [/Main](../language-reference/compiler-options/main-compiler-option.md) byla zadána neplatná třída; třídu nelze použít jako umístění pro metodu [Main](../programming-guide/main-and-command-args/index.md) .
