# <a name="compiler-error-cs1504"></a>Compiler Error CS1504
Zdrojový soubor 'file' nelze otevřít ('z důvodu")  
  
 Zdrojový soubor nelze otevřít nebo číst kompilátorem. Soubor je pravděpodobně uzamčen jiná aplikace nebo může být nějaký jiný problém operačního systému. Zpráva obsahuje operační systém důvod proč kompilátor nelze otevřít nebo číst soubor.
