## <a name="example"></a><span data-ttu-id="a46ba-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="a46ba-107">Example</span></span>  
 <span data-ttu-id="a46ba-108">Následující kód vygeneruje CS1912, protože `memberA` je inicializován dvakrát:</span><span class="sxs-lookup"><span data-stu-id="a46ba-108">The following code generates CS1912 because `memberA` is initialized two times:</span></span>  
  
```csharp  
// cs1912.cs  
using System.Linq;  
  
public class TestClass  
{  
    public int memberA { get; set; }  
    public int memberB { get; set; }  
}

{  
    static void Main()  
    {  
        TestClass tc = new TestClass() { memberA = 5, memberA = 6, memberB = 2}; // CS1912  
    }  
}  
```
