# <a name="compiler-error-cs0061"></a>CS0061 chyby kompilátoru
Nekonzistentní dostupnost: základní rozhraní takto: interface 1' je méně dostupný než rozhraní "2 rozhraní.  
  
 A [veřejné](../../csharp/language-reference/keywords/public.md) konstrukce musí vrátit objekt veřejně přístupná.  
  
 Usnadnění přístupu rozhraní nelze zúžit odvozené rozhraní. Další informace najdete v tématu [rozhraní](../../csharp/programming-guide/interfaces/index.md) a [modifikátory přístupu](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
 Následující ukázka generuje CS0061.  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
