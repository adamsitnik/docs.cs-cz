 <span data-ttu-id="ed0f4-104">[Veřejná](../language-reference/keywords/public.md) konstrukce musí vracet veřejně přístupný objekt.</span><span class="sxs-lookup"><span data-stu-id="ed0f4-104">A [public](../language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="ed0f4-107">Následující ukázka generuje CS0061.</span><span class="sxs-lookup"><span data-stu-id="ed0f4-107">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
