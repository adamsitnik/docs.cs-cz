# <a name="compiler-error-cs1008"></a>Compiler Error CS1008
Typ byte, sbyte, short, ushort, int, uint, long nebo ulong očekávání  
  
 Některé datové typy, jako například [výčty](../../csharp/language-reference/keywords/enum.md), lze deklarovat pouze k uložení dat určené typy.  
  
 Následující ukázka generuje CS1008:  
  
```csharp  
// CS1008.cs  
abstract public class clx  
{  
   enum splitch : char   // CS1008, char not valid type for enums  
   {  
      x, y, z  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
