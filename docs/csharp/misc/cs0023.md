# <a name="compiler-error-cs0023"></a>Compiler Error CS0023
Operátor 'operator' nelze použít pro operand typu 'type'  
  
 Byl proveden pokus o použití operátoru na proměnnou, jejíž typ nebyl navržen pro práci s operátorem. Další informace najdete v tématu [typy](../../csharp/programming-guide/types/index.md) a [ C# operátory](../../csharp/language-reference/operators/index.md).  
  
 Následující ukázka generuje CS0023:  
  
```csharp  
// CS0023.cs  
namespace x  
{  
   public class a  
   {  
      public static void Main()  
      {  
         string s = "hello";  
         s = -s;   // CS0023, minus operator not allowed on strings  
      }  
   }  
}  
```
