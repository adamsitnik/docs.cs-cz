# <a name="compiler-error-cs0448"></a>Compiler Error CS0448
Návratový typ pro ++ a--– operátor musí být nadřazeného typu nebo odvozený z nadřazeného typu.  
  
 Při přepsání `++` nebo `--` operátory, které musí vracet stejný typ jako obsahující typ, nebo návratový typ, který je odvozen z nadřazeného typu.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0448.  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0448.  
  
```csharp  
// CS0448_b.cs  
public struct S  
{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
