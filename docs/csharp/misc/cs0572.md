# <a name="compiler-error-cs0572"></a>Compiler Error CS0572
'type': nelze odkazovat na typ prostřednictvím výrazu. Místo toho vyzkoušet "path_to_type.  
  
 Došlo pokusu o přístup ke členu třídy prostřednictvím identifikátor, která není v této situaci.  
  
 Následující ukázka generuje CS0572:  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
