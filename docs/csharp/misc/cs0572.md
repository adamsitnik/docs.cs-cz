# <a name="compiler-error-cs0572"></a><span data-ttu-id="9a128-102">Compiler Error CS0572</span><span class="sxs-lookup"><span data-stu-id="9a128-102">Compiler Error CS0572</span></span>
<span data-ttu-id="9a128-103">'type': nelze odkazovat na typ prostřednictvím výrazu. Místo toho vyzkoušet "path_to_type.</span><span class="sxs-lookup"><span data-stu-id="9a128-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="9a128-104">Došlo pokusu o přístup ke členu třídy prostřednictvím identifikátor, která není v této situaci.</span><span class="sxs-lookup"><span data-stu-id="9a128-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="9a128-105">Následující ukázka generuje CS0572:</span><span class="sxs-lookup"><span data-stu-id="9a128-105">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
