# <a name="compiler-error-cs0202"></a>CS0202 chyby kompilátoru
Příkaz foreach vyžaduje, aby návratový typ 'type' z ' typu. GetEnumerator()' měl vhodnou veřejnou metodu MoveNext a veřejnou vlastnost Current  
  
 A <xref:System.Collections.IEnumerable.GetEnumerator%2A> funkce, používá k povolení použití příkazu foreach příkazy, nemůže vracet ukazatel nebo pole, musí vracet instanci třídy, která může fungovat jako enumerátor. Správné požadavky, která bude sloužit jako enumerátor zahrnují veřejnou vlastnost Current a veřejnou metodu MoveNext.  
  
> [!NOTE]
>  V C# 2.0 bude kompilátor automaticky generovat aktuální a MoveNext za vás. Další informace najdete v příkladu kódu v [obecných rozhraní](../../csharp/programming-guide/generics/generic-interfaces.md).  
  
 Následující ukázka generuje CS0202:  
  
```csharp  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
