# <a name="compiler-warning-level-1-cs0602"></a>Kompilátor CS0602 upozornění (úroveň 1)
Funkce "old_feature" je zastaralá. Místo toho použijte "new_feature.  
  
 Funkce jazyka použití ve vašem kódu (*old_feature*) je stále podporovány, ale které nemusí být podporovány v budoucí verzi. Místo toho používejte doporučené syntaxe (*new_feature*).
