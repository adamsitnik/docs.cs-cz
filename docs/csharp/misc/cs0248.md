# <a name="compiler-error-cs0248"></a>CS0248 chyby kompilátoru
Nejde vytvořit pole se zápornou velikostí.  
  
 Pro velikost pole byl zadán s záporné číslo. Další informace najdete v tématu [pole](../../csharp/programming-guide/arrays/index.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0248:  
  
```csharp  
// CS0248.cs  
class MyClass  
{  
    public static void Main()  
    {  
        int[] myArray = new int[-3] {1,2,3};   // CS0248, pass a nonnegative number  
    }  
}  
```
