# <a name="compiler-error-cs1609"></a>Compiler Error CS1609
Modifikátory nejde umístit na deklarace přistupujícího objektu události  
  
 Modifikátory lze umístit pouze na deklarace události a ne na deklarace přistupujícího objektu události. Další informace najdete v tématu [pomocí vlastnosti](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1609.  
  
```csharp  
// CS1609.cs  
// compile with: /target:library  
delegate int Del();  
class A  
{  
   public event Del MyEvent   
   {  
      private add {}   // CS1609  
      // try the following line instead  
      // add {}  
      remove {}  
   }  
}  
```
