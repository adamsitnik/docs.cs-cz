# <a name="compiler-warning-level-1-cs1590"></a>Kompilátor CS1590 upozornění (úroveň 1)
Kód XML je neplatný prvek direktivy include--chybí atribut souboru  
  
 Atribut cesty nebo doc předán [ \<zahrnout >](../../csharp/programming-guide/xmldoc/include.md) značku, chybí nebo je neúplná.  
  
 Následující ukázka generuje CS1590:  
  
```csharp  
// CS1590.cs  
// compile with: /W:1 /doc:x.xml  
  
/// <include path='MyDocs/MyMembers[@name="test"]/*' />   // CS1590  
// try the following line instead  
// /// <include file='x.doc' path='MyDocs/MyMembers[@name="test"]/*' />  
class Test  
{  
   public static void Main()  
   {  
   }  
}  
```
