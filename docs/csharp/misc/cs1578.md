 <span data-ttu-id="dd67d-104">Po direktivě [#line](../language-reference/preprocessor-directives/preprocessor-line.md) je povolen pouze název souboru (v uvozovkách) nebo Jednořádkový komentář.</span><span class="sxs-lookup"><span data-stu-id="dd67d-104">After a [#line](../language-reference/preprocessor-directives/preprocessor-line.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>  
  
 <span data-ttu-id="dd67d-105">Následující ukázka generuje CS1578:</span><span class="sxs-lookup"><span data-stu-id="dd67d-105">The following sample generates CS1578:</span></span>  
  
```csharp

{  
   static void Main()  
   {  
      #line 101 abc.cs   // CS1578  
      // try the following line instead  
      //#line 101 "abc.cs"  
      intt i;          // error will be reported on line 101  
   }  
}  
```
