# <a name="compiler-error-cs1578"></a>Compiler Error CS1578
Název souboru, jednořádkový komentář nebo ukončení řádku. byl očekáván  
  
 Po [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) direktiv, pouze název souboru (v uvozovkách) ani jednořádkový komentář se nepovoluje.  
  
 Následující ukázka generuje CS1578:  
  
```csharp  
// CS1578.cs  
class MyClass  
{  
   static void Main()  
   {  
      #line 101 abc.cs   // CS1578  
      // try the following line instead  
      //#line 101 "abc.cs"  
      intt i;          // error will be reported on line 101  
   }  
}  
```
