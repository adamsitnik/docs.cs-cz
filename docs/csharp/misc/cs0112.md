# <a name="compiler-error-cs0112"></a><span data-ttu-id="c319d-102">Chyba kompilátoru CS0112</span><span class="sxs-lookup"><span data-stu-id="c319d-102">Compiler Error CS0112</span></span>
<span data-ttu-id="c319d-103">Statický člen Function nemůže být označený jako override, Virtual nebo abstract.</span><span class="sxs-lookup"><span data-stu-id="c319d-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="c319d-104">Jakákoliv deklarace metody, která používá `override`klíčové slovo, **Virtual**nebo **abstract** , nemůže také použít klíčové slovo **static** .</span><span class="sxs-lookup"><span data-stu-id="c319d-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="c319d-105">Další informace naleznete v tématu [metody](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="c319d-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="c319d-106">Následující ukázka generuje CS0112:</span><span class="sxs-lookup"><span data-stu-id="c319d-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
