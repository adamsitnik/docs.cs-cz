# <a name="compiler-error-cs0127"></a>Chyba kompilátoru CS0127
Vzhledem k tomu, že funkce Function vrací typ void, nesmí za klíčovým slovem Return následovat výraz objektu.  
  
 Metoda s návratovým typem [void](../language-reference/keywords/void.md) nemůže vracet hodnotu. Další informace naleznete v tématu [metody](../programming-guide/classes-and-structs/methods.md).  
  
 Následující ukázka generuje CS0127:  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
