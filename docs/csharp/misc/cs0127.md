# <a name="compiler-error-cs0127"></a><span data-ttu-id="2e062-102">Chyba kompilátoru CS0127</span><span class="sxs-lookup"><span data-stu-id="2e062-102">Compiler Error CS0127</span></span>
<span data-ttu-id="2e062-103">Vzhledem k tomu, že funkce Function vrací typ void, nesmí za klíčovým slovem Return následovat výraz objektu.</span><span class="sxs-lookup"><span data-stu-id="2e062-103">Since 'function' returns void, a return keyword must not be followed by an object expression</span></span>  
  
 <span data-ttu-id="2e062-104">Metoda s návratovým typem [void](../language-reference/keywords/void.md) nemůže vracet hodnotu.</span><span class="sxs-lookup"><span data-stu-id="2e062-104">A method with a [void](../language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="2e062-105">Další informace naleznete v tématu [metody](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="2e062-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="2e062-106">Následující ukázka generuje CS0127:</span><span class="sxs-lookup"><span data-stu-id="2e062-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
