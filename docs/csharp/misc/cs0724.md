# <a name="compiler-error-cs0724"></a><span data-ttu-id="5fbe3-102">Chyba kompilátoru CS0724</span><span class="sxs-lookup"><span data-stu-id="5fbe3-102">Compiler Error CS0724</span></span>

<span data-ttu-id="5fbe3-103">Příkaz throw bez argumentů není povolený v klauzuli finally, která je vnořená do nejbližší nadřazené klauzule catch.</span><span class="sxs-lookup"><span data-stu-id="5fbe3-103">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>

## <a name="example"></a><span data-ttu-id="5fbe3-104">Příklad</span><span class="sxs-lookup"><span data-stu-id="5fbe3-104">Example</span></span>

<span data-ttu-id="5fbe3-105">Následující příklad generuje CS0724 z důvodu `throw` příkazu v bloku klauzule `finally`:</span><span class="sxs-lookup"><span data-stu-id="5fbe3-105">The following example generates CS0724 because of the `throw` statement inside the `finally` clause block:</span></span>

```csharp  
// CS0724.cs  
using System;  
  
class Program
{
    static void Test()
    {
        try
        {
            throw new Exception();
        }
        catch
        {
            try
            {
            }
            finally
            {
                throw; // CS0724
            }
        }
    }

    static void Main()
    {
    }
}
```
