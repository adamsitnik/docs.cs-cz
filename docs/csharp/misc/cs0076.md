# <a name="compiler-error-cs0076"></a>Chyba kompilátoru CS0076
Název enumerátoru value__ je rezervovaný a nedá se použít.  
  
 Položka ve [výčtu](../language-reference/keywords/enum.md) nemůže mít identifikátor nazvaný **value__** .
