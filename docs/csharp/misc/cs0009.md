# <a name="compiler-error-cs0009"></a>Chyba kompilátoru CS0009
Soubor metadat ' file ' nelze otevřít – ' Description '  
  
 Soubor určený parametrem [-reference](../language-reference/compiler-options/reference-compiler-option.md) kompilátoru neobsahuje platná metadata.
