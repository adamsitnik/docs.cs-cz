# <a name="compiler-error-cs1601"></a>Compiler Error CS1601
Parametr delegáta nebo metody nemůže být typu 'type'  
  
 Některé typy v rozhraní .NET Framework knihovny tříd rozhraní, například <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> a <xref:System.ArgIterator> nelze použít jako [v](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) nebo [si](../../csharp/language-reference/keywords/out-parameter-modifier.md) Parametry vzhledem k tomu může být potenciálně zneužity k provádění operací unsafe.  
  
 Následující ukázka generuje CS1601:  
  
```csharp  
// CS1601.cs  
using System;

class MyClass
{
    public void Test1(in TypedReference t)   // CS1601  
    {
    }

    public void Test2(ref TypedReference t)   // CS1601  
    {
    }

    public void Test3(out ArgIterator t)   // CS1601  
    {
        t = default;
    }
}
```
