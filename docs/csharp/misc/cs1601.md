# <a name="compiler-error-cs1601"></a><span data-ttu-id="19e8e-102">Chyba kompilátoru CS1601</span><span class="sxs-lookup"><span data-stu-id="19e8e-102">Compiler Error CS1601</span></span>
<span data-ttu-id="19e8e-103">Parametr metody nebo delegáta nemůže být typu Type.</span><span class="sxs-lookup"><span data-stu-id="19e8e-103">Method or delegate parameter cannot be of type 'type'</span></span>  
  
 <span data-ttu-id="19e8e-104">Některé typy v knihovně tříd <xref:System.TypedReference>.NET Framework, například, <xref:System.RuntimeArgumentHandle> a <xref:System.ArgIterator> nelze je použít jako [in](../language-reference/keywords/in-parameter-modifier.md), parametry [ref](../language-reference/keywords/ref.md) nebo [out](../language-reference/keywords/out-parameter-modifier.md) , protože by mohly být použity k provádění nebezpečných operací.</span><span class="sxs-lookup"><span data-stu-id="19e8e-104">Some types in the .NET Framework class library, for example, <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> and <xref:System.ArgIterator> cannot be used as [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameters because they could potentially be used to perform unsafe operations.</span></span>  
  
 <span data-ttu-id="19e8e-105">Následující ukázka generuje CS1601:</span><span class="sxs-lookup"><span data-stu-id="19e8e-105">The following sample generates CS1601:</span></span>  
  
```csharp  
// CS1601.cs  
using System;

class MyClass
{
    public void Test1(in TypedReference t)   // CS1601  
    {
    }

    public void Test2(ref TypedReference t)   // CS1601  
    {
    }

    public void Test3(out ArgIterator t)   // CS1601  
    {
        t = default;
    }
}
```
