# <a name="compiler-error-cs1597"></a>Compiler Error CS1597
Středník po metodou nebo blokem přistupujícího objektu není platný  
  
 Středníky nejsou potřeba (nebo povolené) na konec metodu nebo přistupující objekt bloku.  
  
 Následující ukázka generuje CS1597:  
  
```csharp  
// CS1597.cs  
class TestClass  
{  
   public static void Main()  
   {  
   };   // CS1597, remove semicolon  
}  
```
