# <a name="compiler-error-cs0681"></a>CS0681 chyby kompilátoru
Modifikátor "abstraktní" není platný pro pole. Zkuste místo toho použít vlastnosti  
  
 Pole nemůže být abstraktní. Máte však abstraktní vlastnost, která přistupuje k poli.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0681:  
  
```csharp  
// CS0681.cs  
// compile with: /target:library  
abstract class C  
{  
    abstract int num;  // CS0681  
}  
```  
  
## <a name="example"></a>Příklad  
 Místo toho vyzkoušejte následující kód:  
  
```csharp  
// CS0681b.cs  
// compile with: /target:library  
abstract class C  
{  
    public abstract int num  
    {  
       get;  
       set;  
    }  
}  
```
