# <a name="compiler-error-cs1619"></a>CS1619 chyby kompilátoru
Nelze vytvořit dočasný soubor 'filename'--důvod  
  
 Kompilátor se nepodařilo vytvořit dočasný soubor pro daný důvod (například disk je plný).
