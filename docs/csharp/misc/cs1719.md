# <a name="compiler-error-cs1719"></a>Compiler Error CS1719
Chyba při čtení souboru prostředků Win32 "Název souboru" – "důvod.  
  
 Pokus o čtení souboru prostředků Win32, se nezdařila z důvodu uvedený v chybě, obvykle něco jako "soubor nebyl nalezen" nebo "přístup byl odepřen." Tuto chybu vyřeší tím, že opraví popsal z důvodu problému.
