# <a name="compiler-error-cs1056"></a>Compiler Error CS1056
Neočekávaný znak "znak.  
  
 C# Kompilátoru vyskytl se neočekávaný znak a nemůže určit token, který právě zpracovává. Například pokud kompilátor narazí na znak eura uprostřed zpracování identifikátoru, je schopen klasifikovat identifikátor, protože znak eura by být platný v řetězci, a kompilátor ví, že nezpracovává řetězec.
