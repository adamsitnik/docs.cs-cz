# <a name="compiler-error-cs1055"></a>Compiler Error CS1055
Panelu Přidat nebo odebrat očekával se přistupující objekt  
  
 Pokud vaše [události](../../csharp/language-reference/keywords/event.md) není deklarován jako pole, musí definovat, jak **přidat** a **odebrat** přístupové funkce.  
  
 Následující ukázka generuje CS1055:  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
