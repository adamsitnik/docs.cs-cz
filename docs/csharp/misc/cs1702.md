# <a name="compiler-warning-level-3-cs1702"></a>Kompilátor CS1702 upozornění (úroveň 3)
Za předpokladu, že odkaz na sestavení "Název sestavení #1" odpovídá "Název sestavení #2", budete muset zadat zásady pro běh  
  
 Odkazy na dvě sestavení mají rozdílné buildu a/nebo číslo revize, tak nebude automaticky sjednocení. Budete muset zadat zásady za běhu vynutit sjednocení pomocí direktivy v souboru .config aplikace.
