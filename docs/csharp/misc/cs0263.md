# <a name="compiler-error-cs0263"></a>CS0263 chyby kompilátoru
Částečné deklarace 'type' nesmí určovat různé základní třídy  
  
 Při definování typu v částečné deklarace, zadejte stejnou základní typy ve všech částečné deklarace. Další informace najdete v tématu [částečné třídy a metody](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
 Následující ukázka generuje CS0263:  
  
```csharp  
// CS0263.cs  
// compile with: /target:library  
class B1  
{  
}  
  
class B2  
{  
}  
partial class C : B1  // CS0263 – is the base class B1 or B2?  
{  
}  
  
partial class C : B2  
{  
}  
```
