# <a name="compiler-error-cs0272"></a>Compiler Error CS0272
Vlastnost nebo indexovací člen 'vlastnost nebo indexer' nelze v tomto kontextu použít, protože přistupující objekt Jet není dostupný.  
  
 Této chybě dochází, když `set` přistupující objekt není dostupný na kód programu. Chcete-li tuto chybu vyřešit, zvyšte přístupnost přístupového objektu nebo změňte umístění volání. Další informace najdete v tématu [omezení přístupnosti přístupového objektu](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0272:  
  
```csharp  
// CS0272.cs  
public class MyClass  
{  
    public int Property  
    {  
        get { return 0; }  
        private set { }  
    }  
}  
  
public class Test  
{  
    static void Main()  
    {  
        MyClass c = new MyClass();  
        c.Property = 10;      // CS0272  
        // To resolve, remove the previous line   
        // or use an appropriate modifier on the set accessor.  
    }  
}  
```
