# <a name="compiler-error-cs1714"></a>Compiler Error CS1714
Základní třídu nebo rozhraní Názevtypu1 nelze vyhodnotit nebo jsou neplatné  
  
 Při implementaci Názevtypu1 z třídy nebo rozhraní, že buď se nedá přeložit, například kompilátor nemůže najít nebo je neplatný. Řešení je určit, které z těchto dvou případech je a více správně zadat umístění typu, nebo opravte všechny chyby kompilátoru v základní třídě.
