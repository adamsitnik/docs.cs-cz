# <a name="compiler-error-cs0211"></a>Compiler Error CS0211
Nejde převzít adresu daného výrazu.  
  
 Můžete převzít adresu proměnné pole, místní proměnné a dereference ukazatele, ale je nelze provést, například adresu součet dvou místních proměnných. Další informace najdete v tématu [nezabezpečený kód a ukazatele](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Následující ukázka generuje CS0211:  
  
```csharp  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
