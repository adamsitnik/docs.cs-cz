 <span data-ttu-id="39b5b-104">Místní proměnné v [nebezpečné](../../csharp/language-reference/keywords/unsafe.md) metody nebo parametr již vyřešen (v zásobníku), takže nelze převzít adresu proměnné některý z těchto dvou proměnných ve [oprava](../../csharp/language-reference/keywords/fixed-statement.md) výrazu.</span><span class="sxs-lookup"><span data-stu-id="39b5b-104">A local variable in an [unsafe](../../csharp/language-reference/keywords/unsafe.md) method or a parameter is already fixed (on the stack), so you cannot take the address of either of these two variables in a [fixed](../../csharp/language-reference/keywords/fixed-statement.md) expression.</span></span> <span data-ttu-id="39b5b-105">Další informace najdete v tématu [nezabezpečený kód a ukazatele](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="39b5b-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="39b5b-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="39b5b-106">Example</span></span>  
 <span data-ttu-id="39b5b-107">Následující ukázka generuje CS0213.</span><span class="sxs-lookup"><span data-stu-id="39b5b-107">The following sample generates CS0213.</span></span>  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
