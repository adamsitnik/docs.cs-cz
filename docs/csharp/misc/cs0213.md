<span data-ttu-id="62af6-103">Příkaz fixed nelze použít k převzetí adresy již pevného výrazu.</span><span class="sxs-lookup"><span data-stu-id="62af6-103">You cannot use the fixed statement to take the address of an already fixed expression</span></span>  
  
 <span data-ttu-id="62af6-104">Lokální proměnná v nezabezpečené [](../language-reference/keywords/unsafe.md) metodě nebo parametru je již opravena (v zásobníku), takže nelze adresovat jednu z těchto dvou proměnných v rámci [fixního](../language-reference/keywords/fixed-statement.md) výrazu.</span><span class="sxs-lookup"><span data-stu-id="62af6-104">A local variable in an [unsafe](../language-reference/keywords/unsafe.md) method or a parameter is already fixed (on the stack), so you cannot take the address of either of these two variables in a [fixed](../language-reference/keywords/fixed-statement.md) expression.</span></span> <span data-ttu-id="62af6-105">Další informace naleznete v tématu [nezabezpečený kód a ukazatele](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="62af6-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="62af6-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="62af6-106">Example</span></span>  
 <span data-ttu-id="62af6-107">Následující ukázka generuje CS0213.</span><span class="sxs-lookup"><span data-stu-id="62af6-107">The following sample generates CS0213.</span></span>  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
