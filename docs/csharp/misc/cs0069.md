# <a name="compiler-error-cs0069"></a>CS0069 chyby kompilátoru
Událost v rozhraní nemůže mít přidat nebo odebrat přístupové objekty  
  
 Nejde definovat funkce přístupového objektu události v [rozhraní](../../csharp/language-reference/keywords/interface.md). Další informace najdete v tématu [události](../../csharp/programming-guide/events/index.md) a [rozhraní](../../csharp/programming-guide/interfaces/index.md).  
  
 Následující ukázka generuje CS0069:  
  
```csharp  
// CS0069.cs  
// compile with: /target:library  
  
public delegate void EventHandler();  
  
public interface a  
{  
   event EventHandler Click { remove {} }   // CS0069  
   event EventHandler Click2;   // OK  
}  
```
