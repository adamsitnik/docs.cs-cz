# <a name="compiler-error-cs0113"></a>Compiler Error CS0113
Člen 'function' označené jako override nejde označit jako new nebo virtual  
  
 Je vzájemně se vylučující k označení metody s [nové](../../csharp/language-reference/keywords/new.md) a [přepsat](../../csharp/language-reference/keywords/override.md) klíčová slova.  
  
 Následující ukázka generuje CS0113:  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
