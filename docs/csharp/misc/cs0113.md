# <a name="compiler-error-cs0113"></a><span data-ttu-id="a178c-102">Compiler Error CS0113</span><span class="sxs-lookup"><span data-stu-id="a178c-102">Compiler Error CS0113</span></span>
<span data-ttu-id="a178c-103">Člen 'function' označené jako override nejde označit jako new nebo virtual</span><span class="sxs-lookup"><span data-stu-id="a178c-103">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="a178c-104">Je vzájemně se vylučující k označení metody s [nové](../../csharp/language-reference/keywords/new-modifier.md) a [přepsat](../../csharp/language-reference/keywords/override.md) klíčová slova.</span><span class="sxs-lookup"><span data-stu-id="a178c-104">It is mutually exclusive to mark a method with the [new](../../csharp/language-reference/keywords/new-modifier.md) and [override](../../csharp/language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="a178c-105">Následující ukázka generuje CS0113:</span><span class="sxs-lookup"><span data-stu-id="a178c-105">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
