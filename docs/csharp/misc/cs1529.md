## <a name="example"></a><span data-ttu-id="b4e01-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="b4e01-105">Example</span></span>  
 <span data-ttu-id="b4e01-106">Následující ukázka generuje CS1529:</span><span class="sxs-lookup"><span data-stu-id="b4e01-106">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}

```
