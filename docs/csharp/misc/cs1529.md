# <a name="compiler-error-cs1529"></a>Compiler Error CS1529
Pomocí klauzule musí předcházet všem ostatním prvkům definovaným v oboru názvů s výjimkou deklarací externích aliasů  
  
 A [pomocí](../../csharp/language-reference/keywords/using.md) klauzule musí být nejdříve uveden v oboru názvů.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1529:  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
