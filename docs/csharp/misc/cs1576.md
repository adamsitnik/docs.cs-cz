# <a name="compiler-error-cs1576"></a>Compiler Error CS1576
Číslo řádku zadané v direktivě #line je chybějící nebo neplatný  
  
 Kompilátor zjistil chybu s hodnotu předanou [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) směrnice.  
  
 Následující ukázka generuje CS1576:  
  
```csharp  
// CS1576.cs  
public class MyClass  
{  
   static void Main()  
   {  
      #line "abc.sc"         // CS1576  
      // try the following line instead  
      //#line  101 "abc.sc"  
      intt i;  // error will be reported on line 101  
   }  
}  
```
