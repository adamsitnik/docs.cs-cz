 <span data-ttu-id="5b2d0-105">Metoda označena `extern` by měl být také označen atributem, například, [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) atribut.</span><span class="sxs-lookup"><span data-stu-id="5b2d0-105">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>

   }
