# <a name="compiler-warning-level-1-cs3005"></a>Kompilátor CS3005 upozornění (úroveň 1)
Identifikátor 'identifier' lišící se pouze velikostí písmen není kompatibilní se Specifikací CLS  
  
 A [veřejné](../../csharp/language-reference/keywords/public.md), [chráněné](../../csharp/language-reference/keywords/protected.md), nebo `protected internal` identifikátor, který se liší od jiného `public`, `protected`, nebo `protected internal` je identifikátor pouze v případě jedním nebo více písmeny. není kompatibilní s specifikace CLS (Common Language). Další informace o dodržování specifikace CLS naleznete v tématu [jazyková nezávislost a jazykově nezávislé komponenty](../../standard/language-independence.md).
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS3003:  
  
```csharp  
// CS3005.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public class a  
{  
    public static int a1 = 0;  
    public static int A1 = 1;   // CS3005  
  
    public static void Main()  
    {  
        Console.WriteLine(a1);  
        Console.WriteLine(A1);  
    }  
}  
```
