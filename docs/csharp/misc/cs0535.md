# <a name="compiler-error-cs0535"></a><span data-ttu-id="24c02-102">Chyba kompilátoru CS0535</span><span class="sxs-lookup"><span data-stu-id="24c02-102">Compiler Error CS0535</span></span>
<span data-ttu-id="24c02-103">klíčové slovo class neimplementuje člen rozhraní.</span><span class="sxs-lookup"><span data-stu-id="24c02-103">'class' does not implement interface member 'member'</span></span>  
  
 <span data-ttu-id="24c02-104">[Třída](../language-reference/keywords/class.md) odvozená z [rozhraní](../language-reference/keywords/interface.md), ale třída neimplementovala jeden nebo více členů rozhraní.</span><span class="sxs-lookup"><span data-stu-id="24c02-104">A [class](../language-reference/keywords/class.md) derived from an [interface](../language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="24c02-105">Třída musí implementovat všechny členy rozhraní, ze kterých je odvozena nebo jinak deklarována `abstract`.</span><span class="sxs-lookup"><span data-stu-id="24c02-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="24c02-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="24c02-106">Example</span></span>  
 <span data-ttu-id="24c02-107">Následující ukázka generuje CS0535.</span><span class="sxs-lookup"><span data-stu-id="24c02-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
