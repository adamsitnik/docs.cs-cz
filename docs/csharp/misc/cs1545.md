# <a name="compiler-error-cs1545"></a>Compiler Error CS1545
Vlastnost, indexer nebo událost 'property' není podporován jazyk; Zkuste přímo volat přístupové metody "přístupový objekt set" nebo "přistupující objekt get.  
  
 Kód využívá objekt, který má jinou než výchozí [indexer](../../csharp/programming-guide/indexers/index.md) a nepokusili se syntaxí indexované. Chcete-li vyřešit tuto chybu, zavolejte vlastnosti `get` nebo `set` přístupové metody.  
  
## <a name="example"></a>Příklad  
  
```cpp  
// CPP1545.cpp  
// compile with: /clr /LD  
// a Visual C++ program  
using namespace System;  
public ref struct Employee {  
   Employee( String^ s, int d ) {}  
  
   property String^ name {  
      String^ get() {  
         return nullptr;  
      }  
   }  
};  
  
public ref struct Manager {  
   property Employee^ Report [String^] {  
      Employee^ get(String^ s) {  
         return nullptr;  
      }  
  
      void set(String^ s, Employee^ e) {}  
   }  
};  
```  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1545.  
  
```csharp  
// CS1545.cs  
// compile with: /r:CPP1545.dll  
  
class x {  
   public static void Main() {  
      Manager Ed = new Manager();  
      Employee Bob = new Employee("Bob Smith", 12);  
      Ed.Report[ Bob.name ] = Bob;   // CS1545  
      Ed.set_Report( Bob.name, Bob);   // OK  
   }  
}  
```
