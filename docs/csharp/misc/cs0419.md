# <a name="compiler-warning-level-3-cs0419"></a><span data-ttu-id="2e702-102">Kompilátor CS0419 upozornění (úroveň 3)</span><span class="sxs-lookup"><span data-stu-id="2e702-102">Compiler Warning (level 3) CS0419</span></span>
<span data-ttu-id="2e702-103">Nejednoznačný odkaz v atributu cref: "Metoda název1".</span><span class="sxs-lookup"><span data-stu-id="2e702-103">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="2e702-104">Za předpokladu, že "Metoda název2", ale shoda s jinými přetíženími, včetně "Metoda název3".</span><span class="sxs-lookup"><span data-stu-id="2e702-104">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="2e702-105">V komentáři XML dokumentace v kódu nelze rozpoznat odkaz.</span><span class="sxs-lookup"><span data-stu-id="2e702-105">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="2e702-106">Tato situace může nastat, pokud je přetížena metoda nebo pokud se nacházejí dvě různé identifikátory se stejným názvem.</span><span class="sxs-lookup"><span data-stu-id="2e702-106">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="2e702-107">Chcete-li vyřešit upozornění, pomocí kvalifikovaného názvu k rozlišení odkaz nebo použít konkrétního přetížení v závorkách.</span><span class="sxs-lookup"><span data-stu-id="2e702-107">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="2e702-108">Následující ukázka generuje CS0419.</span><span class="sxs-lookup"><span data-stu-id="2e702-108">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
