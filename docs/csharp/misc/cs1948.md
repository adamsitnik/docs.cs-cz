# <a name="compiler-error-cs1948"></a>Compiler Error CS1948
Proměnná rozsahu 'name' nesmí mít stejný název jako parametr typů metod  
  
 Na stejném místě deklarace nesmí obsahovat dvě deklarace stejný identifikátor.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Změňte název proměnné rozsahu nebo parametr typu.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS1948, protože identifikátor `T` se používá pro proměnnou rozsahu a pro parametr typu v metodě `TestMethod`:  
  
```csharp  
// cs1948.cs  
using System.Linq;  
class Test  
{  
    public void TestMethod<T>(T t)  
    {  
        var x = from T in Enumerable.Range(1, 100) // CS1948  
                select T;  
    }  
}  
```
