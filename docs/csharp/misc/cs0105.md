# <a name="compiler-warning-level-3-cs0105"></a>Kompilátor CS0105 upozornění (úroveň 3)
Using – direktiva pro atribut namespace' se objevil dřív v tomto oboru názvů  
  
 A [obor názvů](../../csharp/language-reference/keywords/namespace.md), které by se dají deklarovat jenom jednou, byl deklarován více než jednou; odebrat všechny duplicitní deklarace oboru názvů.  
  
 Následující ukázka generuje CS0105:  
  
```csharp  
// CS0105.cs  
// compile with: /W:3  
using System;  
using System;   // CS0105  
  
public class a  
{  
   public static void Main()  
   {  
   }  
}  
```
