# <a name="compiler-error-cs0588"></a>Compiler Error CS0588
Vnitřní chyba kompilátoru: fáze 'LEX'  
  
 Pokuste se zjistit, zda kompilátor nedaří kvůli neschopnost analyzovat neočekávaná syntaxe. Pokud to není tento případ, zkuste [kontaktujte nás](/visualstudio/ide/talk-to-us).
