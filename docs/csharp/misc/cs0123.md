# <a name="compiler-error-cs0123"></a>Compiler Error CS0123
Žádná přetížená metoda 'metody' neodpovídá delegáta 'delegát.  
  
 Pokus o vytvoření delegáta se nezdařila, protože se nepoužil správný podpis. Instance delegáta musí být deklarována se stejným podpisem jako deklaraci delegáta.  
  
 Můžete tuto chybu vyřešit buď nastavení metody nebo delegáta podpis. Další informace najdete v tématu [delegáti](../../csharp/programming-guide/delegates/index.md).  
  
 Následující ukázka generuje CS0123.  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
