# <a name="compiler-error-cs0026"></a>Compiler Error CS0026
Klíčové slovo 'this' není platné ve statické vlastnosti, statickou metodu nebo statickém inicializátoru pole.  
  
 [To](../../csharp/language-reference/keywords/this.md) – klíčové slovo odkazuje na objekt, který je instancí typu. Protože statické metody jsou nezávislé na jakoukoli instanci obsahující třídy, klíčové slovo "Tento" nemá význam a proto není povolena. Další informace najdete v tématu [statické třídy a statické členy třídy](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md) a [objekty](../../csharp/programming-guide/classes-and-structs/objects.md).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0026:  
  
```csharp  
// CS0026.cs  
public class A  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
// CS0026  
      this.i = this.i + 1;     
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
