# <a name="compiler-error-cs1676"></a><span data-ttu-id="6fc4c-102">CS1676 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="6fc4c-102">Compiler Error CS1676</span></span>
<span data-ttu-id="6fc4c-103">Parametr 'number' musí být deklarováno s klíčovým slovem '– klíčové slovo'</span><span class="sxs-lookup"><span data-stu-id="6fc4c-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="6fc4c-104">Tato chyba nastane, pokud se liší od, který používá v deklaraci delegáta, které jsou přetypování metody, která modifikátor parametrů typu v anonymní metodě.</span><span class="sxs-lookup"><span data-stu-id="6fc4c-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="6fc4c-105">Následující ukázka generuje CS1676:</span><span class="sxs-lookup"><span data-stu-id="6fc4c-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
