# <a name="compiler-error-cs1022"></a>Compiler Error CS1022
Definice typu nebo oboru názvů nebo end souboru očekávání  
  
 Soubor zdrojového kódu nemá odpovídající složené závorky.  
  
 Následující ukázka generuje CS1022:  
  
```csharp  
// CS1022.cs  
namespace x  
{  
}  
}   // CS1022  
```
