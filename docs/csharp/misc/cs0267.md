# <a name="compiler-error-cs0267"></a>CS0267 chyby kompilátoru
Částečný modifikátor se může objevit jenom bezprostředně před 'class', 'struct' nebo 'rozhraní'  
  
 Umístění **částečné** modifikátor byl nesprávný v deklaraci třídy, struktury nebo rozhraní. Oprava chyby, změnit pořadí modifikátory. Další informace najdete v tématu [částečné třídy a metody](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
 Následující ukázka generuje CS0267:  
  
```csharp  
// CS0267.cs  
public partial class MyClass  
{  
   public MyClass()  
   {  
   }  
}  
  
partial public class MyClass  // CS0267  
// Try this line instead:  
// public partial class MyClass  
{  
   public void Foo()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
