# <a name="compiler-error-cs0471"></a>Compiler Error CS0471
Metoda "name" není obecná metoda. Pokud jste chtěli použít seznam, použijte závorky kolem < výrazu.  
  
 Tato chyba se vygeneruje, když váš kód obsahuje seznam výraz bez závorek.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0471.  
  
```csharp  
// CS0471.cs  
// compile with: /t:library  
class Test  
{  
    public void F(bool x, bool y) {}  
    public void F1()  
    {  
        int a = 1, b = 2, c = 3;  
        F(a<b, c>(3));    // CS0471  
        // To resolve, try the following instead:  
        // F((a<b), c>(3));  
    }  
}  
```
