 <span data-ttu-id="364ec-104">K tomuto upozornění dochází, když typ nebo člen, který má <xref:System.CLSCompliantAttribute> atribut, není viditelný z jiného sestavení.</span><span class="sxs-lookup"><span data-stu-id="364ec-104">This warning occurs when a type or a member that has the <xref:System.CLSCompliantAttribute> attribute is not visible from another assembly.</span></span> <span data-ttu-id="364ec-105">Chcete-li tuto chybu vyřešit, odeberte atribut u všech tříd nebo členů, které nejsou viditelné v jiném sestavení, nebo nastavte typ nebo členy jako viditelné.</span><span class="sxs-lookup"><span data-stu-id="364ec-105">To resolve this error, remove the attribute on any classes or members that are not visible from the other assembly, or make the type or members visible.</span></span> <span data-ttu-id="364ec-106">Další informace o kompatibilitě se specifikací CLS naleznete v tématu [nezávislost jazyků a jazykově nezávislé komponenty](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="364ec-106">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="364ec-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="364ec-107">Example</span></span>  
 <span data-ttu-id="364ec-108">Následující ukázka generuje CS3019:</span><span class="sxs-lookup"><span data-stu-id="364ec-108">The following sample generates CS3019:</span></span>  
  
```csharp  
// CS3019.cs  
// compile with: /W:2  
  
using System;  
  
[assembly: CLSCompliant(true)]  
  
// To fix the error, remove the next line  
[CLSCompliant(true)]  // CS3019  
class C  
{  
    [CLSCompliant(false)]  // CS3019  
    void Foo()  
    {  
    }  
  
    static void Main()  
    {  
    }  
}  
```
