## <a name="example"></a><span data-ttu-id="25895-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="25895-107">Example</span></span>

```csharp

// compile with: /W:2  
  
using System;  
  
[assembly: CLSCompliant(true)]  
  
// To fix the error, remove the next line

{  
    [CLSCompliant(false)]  // CS3019

    }

    {  
    }  
}  
```
