## <a name="example"></a><span data-ttu-id="0b407-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="0b407-108">Example</span></span>  
 <span data-ttu-id="0b407-109">Následující kód vygeneruje CS1928 a CS1929:</span><span class="sxs-lookup"><span data-stu-id="0b407-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
