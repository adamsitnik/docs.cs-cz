 <span data-ttu-id="b22ea-104">Tato chyba je generována při pokusu o vyvolání rozšiřující metody ze třídy, kterou nerozšiřuje.</span><span class="sxs-lookup"><span data-stu-id="b22ea-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="b22ea-105">V následujícím příkladu je metoda rozšíření definována pro odvozenou třídu `A`, ale ne pro základní třídu. `B`</span><span class="sxs-lookup"><span data-stu-id="b22ea-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>

1. <span data-ttu-id="b22ea-107">Vytvořte novou metodu rozšíření pro typ, kde je nutné ji vyvolat, nebo jinak přesuňte volání do objektu typu, který existující metoda rozšiřuje.</span><span class="sxs-lookup"><span data-stu-id="b22ea-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b22ea-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="b22ea-108">Example</span></span>  
 <span data-ttu-id="b22ea-109">Následující kód generuje CS1928 a CS1929:</span><span class="sxs-lookup"><span data-stu-id="b22ea-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
