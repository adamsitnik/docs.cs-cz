# <a name="compiler-error-cs1900"></a>Chyba kompilátoru CS1900
Úroveň upozornění musí být v rozsahu 0-4.  
  
 Možnost kompilátoru [/warn](../language-reference/compiler-options/warn-compiler-option.md) může převzít jenom jednu z pěti možných hodnot (0, 1, 2, 3 nebo 4). Všechny další hodnoty předané do **/warn** budou mít za následek CS1900.  
  
 Následující ukázka generuje CS1900:  
  
```csharp  
// CS1900.cs  
// compile with: /W:5  
// CS1900 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
