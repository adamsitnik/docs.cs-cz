# <a name="compiler-error-cs1900"></a>Compiler Error CS1900
Úroveň upozornění musí být v rozsahu 0-4  
  
 [/ Warn](../../csharp/language-reference/compiler-options/warn-compiler-option.md) – možnost kompilátoru může mít pouze jeden z pěti možných hodnot (0, 1, 2, 3 nebo 4). Jakákoli jiná hodnota předána **/ warn** způsobí CS1900.  
  
 Následující ukázka generuje CS1900:  
  
```csharp  
// CS1900.cs  
// compile with: /W:5  
// CS1900 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
