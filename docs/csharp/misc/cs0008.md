# <a name="compiler-error-cs0008"></a>Chyba kompilátoru CS0008
Neočekávaná chyba při čtení metadat ze souboru ' file ' – ' Description '  
  
 Knihovna DLL byla úspěšně otevřena pro načítání metadat, ale je poškozena, takže z ní nelze číst data. Další informace naleznete v tématu [-Reference (C# možnosti kompilátoru)](../language-reference/compiler-options/reference-compiler-option.md).
