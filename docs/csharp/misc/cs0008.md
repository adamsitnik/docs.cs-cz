# <a name="compiler-error-cs0008"></a>CS0008 chyby kompilátoru
Neočekávaná chyba při čtení metadat ze souboru 'file', 'popis'  
  
 Knihovny DLL byl úspěšně otevřen pro načtení metadat, ale je poškozený, takže jej nelze načíst data. Další informace najdete v tématu [/Reference (možnosti kompilátoru C#)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).
