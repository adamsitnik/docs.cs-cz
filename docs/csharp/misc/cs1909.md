# <a name="compiler-error-cs1909"></a>Compiler Error CS1909
DefaultValue – atribut se nedá použít pro parametry typu 'type'  
  
 CS1909 se vygeneruje, když použijete `DefaultValue` atribut, který se nedá použít pro typ parametru.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1909.  
  
```csharp  
// CS1909.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909  
  
   void Test2([DefaultParameterValue("Test String")] string s);   // OK  
}  
```
