## <a name="example"></a><span data-ttu-id="46122-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="46122-105">Example</span></span>

```csharp  
// CS1909.cs

public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909

}  
```
