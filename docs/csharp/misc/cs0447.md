# <a name="compiler-error-cs0447"></a>Compiler Error CS0447
Atributy nelze použít pro argumenty typu pouze u parametrů typu  
  
 K této chybě dochází při použití atributu na argument typu, který se nachází v vyvolání příkazu. Je přijatelné použijte atribut na parametr typu v příkazu deklarace třídy nebo metody například následující:  
  
```csharp  
class C<[some attribute] T> {…}  
```  
  
 Následující kód vygeneruje tuto chybu. Předpokládá se, že třída `C`definovaná v předchozím řádku kódu, je volána statická metoda `MyStaticMethod`.  
  
```csharp  
C<[some attribute] T>.MyStaticMethod();  
```  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje chybu CS0447.  
  
```csharp  
// CS0447.cs  
using System;  
namespace Test41  
{  
    public interface I<A>   
    {  
        void Meth<B>();  
    }  
    public class B : I<int>   
    {  
        void I<[Test] int>.Meth<X>() { }  // CS0447  
    }  
}  
```
