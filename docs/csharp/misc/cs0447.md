# <a name="compiler-error-cs0447"></a><span data-ttu-id="30598-102">Chyba kompilátoru CS0447</span><span class="sxs-lookup"><span data-stu-id="30598-102">Compiler Error CS0447</span></span>

<span data-ttu-id="30598-103">Atributy nejde použít pro argumenty typu, jenom u parametrů typu.</span><span class="sxs-lookup"><span data-stu-id="30598-103">Attributes cannot be used on type arguments, only on type parameters</span></span>

 <span data-ttu-id="30598-104">K této chybě dochází, pokud použijete atribut na argument typu, který se nachází v příkazu vyvolání.</span><span class="sxs-lookup"><span data-stu-id="30598-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="30598-105">Je přijatelné použít atribut pro parametr typu v příkazu třídy nebo deklarace metody, jako je například následující:</span><span class="sxs-lookup"><span data-stu-id="30598-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

```csharp
class C<[some attribute] T> {…}
```

 <span data-ttu-id="30598-106">Tato chyba se vygeneruje v následujícím řádku kódu.</span><span class="sxs-lookup"><span data-stu-id="30598-106">The following line of code will generate this error.</span></span> <span data-ttu-id="30598-107">Předpokládá se, že třída `C` definovaná v předchozím řádku kódu má statickou metodu nazvanou `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="30598-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

```csharp
C<[some attribute] T>.MyStaticMethod();
```

## <a name="example"></a><span data-ttu-id="30598-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="30598-108">Example</span></span>

 <span data-ttu-id="30598-109">Následující kód generuje chybu CS0447:</span><span class="sxs-lookup"><span data-stu-id="30598-109">The following code generates error CS0447:</span></span>

```csharp
// CS0447.cs
using System;

namespace Test41
{
    public interface I<A>
    {
        void Meth<B>();
    }
    public class B : I<int>
    {
        void I<[Test] int>.Meth<X>() { }  // CS0447
    }
}
```
