# <a name="compiler-error-cs0236"></a>CS0236 chyby kompilátoru
Inicializátor pole nemůže odkazovat na nestatické pole, metodu nebo vlastnost pole  
  
 Pole instance nelze použít k inicializaci jiné pole instancí mimo metodu. Pokud se pokoušíte inicializovat proměnné mimo metodu, zvažte provedení inicializace v konstruktoru třídy. Další informace najdete v tématu [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Následující ukázka generuje CS0236:  
  
```csharp  
// CS0236.cs  
public class MyClass  
{  
   public int i = 5;  
   public int j = i;  // CS0236  
   public int k;      // initialize in constructor  
  
   MyClass()  
   {  
      k = i;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
