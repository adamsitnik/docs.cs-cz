# <a name="compiler-warning-level-1-cs1723"></a>Kompilátor CS1723 upozornění (úroveň 1)
Komentář XML v 'param' má atribut cref 'attribute', který odkazuje na parametr typu.  
  
 Tato chyba je generován, který odkazuje na parametr typu komentáře XML.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS1723.  
  
```csharp  
// CS1723.cs  
// compile with: /t:library /doc:filename.XML  
///<summary>A generic list class.</summary>  
///<see cref="T" />   // CS1723  
// To resolve comment the previous line.  
public class List<T>   
{  
}  
```
