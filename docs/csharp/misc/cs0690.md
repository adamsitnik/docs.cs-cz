# <a name="compiler-error-cs0690"></a>Chyba kompilátoru CS0690
Vstupní soubor ' file ' obsahuje neplatná metadata.  
  
 Můžete otevřít soubor metadat, ale kvůli problému s lokalizací byla metadata poškozena. To se podobá chybě [CS0009](./cs0009.md)s tím rozdílem, že je možné otevřít soubor metadat.  
  
 Prostředí PE (spustitelný soubor procesu) je spustitelný soubor.
