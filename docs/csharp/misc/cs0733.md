# <a name="compiler-error-cs0733"></a>Compiler Error CS0733
Nelze předat obecného typu, "GenericType <>.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0733. První soubor jako knihovna zkompilovat a pak na ni odkazujte při kompilaci druhý soubor.  
  
```csharp  
// CS0733a.cs  
// compile with: /target:library  
public class GenericType<T>   
{  
}  
```  
  
```csharp  
// CS0733.cs  
// compile with: /target:library /r:CS0733a.dll  
[assembly: System.Runtime.CompilerServices.TypeForwardedTo(typeof(GenericType<int>))]   // CS0733  
```
