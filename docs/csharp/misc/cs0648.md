# <a name="compiler-error-cs0648"></a>CS0648 chyby kompilátoru
'type' je typ není jazykem podporován  
  
 Metadata, která byla vygenerována z jiného jazyka, případně C++, obsažen typ, který nebyl označen jako spravovaný. Informace o tomto typu je zahrnut v metadatech, ale typ není k dispozici pro programy napsané v C#.
