# <a name="compiler-error-cs0254"></a>Compiler Error CS0254
Pravá strana přiřazení příkazu fixed nemůže být výrazem přetypování.  
  
 Pravé straně [oprava](../../csharp/language-reference/keywords/fixed-statement.md) výrazu nemusíte používat přetypování. Další informace najdete v tématu [nezabezpečený kód a ukazatele](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Následující ukázka generuje CS0254:  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```
