# <a name="compiler-error-cs0254"></a><span data-ttu-id="7f766-102">Chyba kompilátoru CS0254</span><span class="sxs-lookup"><span data-stu-id="7f766-102">Compiler Error CS0254</span></span>
<span data-ttu-id="7f766-103">Pravá strana přiřazení příkazu fixed nemůže být výrazem přetypování.</span><span class="sxs-lookup"><span data-stu-id="7f766-103">The right hand side of a fixed statement assignment may not be a cast expression</span></span>  
  
 <span data-ttu-id="7f766-104">Pravá strana [pevného](../language-reference/keywords/fixed-statement.md) výrazu nesmí používat přetypování.</span><span class="sxs-lookup"><span data-stu-id="7f766-104">The right side of a [fixed](../language-reference/keywords/fixed-statement.md) expression may not use a cast.</span></span> <span data-ttu-id="7f766-105">Další informace naleznete v tématu [nezabezpečený kód a ukazatele](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="7f766-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="7f766-106">Následující ukázka generuje CS0254:</span><span class="sxs-lookup"><span data-stu-id="7f766-106">The following sample generates CS0254:</span></span>  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```
