# <a name="compiler-error-cs0117"></a>Compiler Error CS0117
'type' neobsahuje definici pro 'identifier'  
  
- Této chybě dochází v některých situacích při odkazu na člen, který pro datový typ neexistuje.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0117:  
  
```csharp  
// CS0117.cs  
public class BaseClass { }  
  
    public class base021 : BaseClass  
    {  
        public void TestInt()  
        {  
            int i = base.someMember; //CS0117  
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
