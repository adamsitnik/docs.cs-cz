# <a name="compiler-error-cs0514"></a>Compiler Error CS0514
'konstruktor': statický konstruktor nemůže používat explicitní 'this' nebo 'base' volání konstruktoru  
  
 Volání `this` v statický konstruktor není povolená, protože statický konstruktor je automaticky volána před vytvořením jakékoli instance třídy. Také statické konstruktory nedědí a nelze ji vyvolat přímo.  
  
 Další informace najdete v tématu [to](../../csharp/language-reference/keywords/this.md) a [základní](../../csharp/language-reference/keywords/base.md).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0514:  
  
```csharp  
// CS0514.cs  
class A  
{  
    static A() : base(0) // CS0514  
    {  
    }  
  
    public A(object o)  
    {  
    }  
}  
  
class B  
{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
