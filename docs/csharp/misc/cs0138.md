# <a name="compiler-error-cs0138"></a>CS0138 chyby kompilátoru
Using – direktiva obor názvů jde použít jedině do oborů názvů. 'type' je typ není obor názvů  
  
 A [pomocí](../../csharp/language-reference/keywords/using.md) – direktiva můžete provést pouze na název oboru názvů jako parametr. Další informace najdete v tématu [obory názvů](../../csharp/programming-guide/namespaces/index.md).  
  
 Následující ukázka generuje CS0138:  
  
```csharp  
// CS0138.cs  
using System.Object;   // CS0138  
```
