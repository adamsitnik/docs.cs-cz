# <a name="compiler-error-cs0247"></a>CS0247 chyby kompilátoru
Ve výrazu stackalloc nejde použít zápornou velikost.  
  
 Byl předán záporné číslo [stackalloc](../language-reference/operators/stackalloc.md) příkazu.
  
 Následující ukázka generuje CS0247:  
  
```csharp  
// CS0247.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int[-30];   // CS0247  
   }  
}  
```
