# <a name="compiler-error-cs1508"></a>Chyba kompilátoru CS1508
Identifikátor prostředku s identifikátorem se už v tomto sestavení používá.  
  
 V kompilaci byl stejný identifikátor (***identifikátor***) předaný na dvě nebo více možností kompilátoru [/Resource](../language-reference/compiler-options/resource-compiler-option.md) nebo [-linkresource –](../language-reference/compiler-options/linkresource-compiler-option.md) .  
  
 Například následující možnosti by vygenerovaly CS1508:  
  
```console  
/resource:anyfile.bmp,DuplicatIdent /linkresource:a.bmp,DuplicatIdent  
```
