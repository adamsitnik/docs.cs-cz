 <span data-ttu-id="8890a-104">Vzor klauzule `group` je `group...by` následovaný nepovinným `into`, jak je znázorněno v následujícím příkladu:</span><span class="sxs-lookup"><span data-stu-id="8890a-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>

## <a name="example"></a><span data-ttu-id="8890a-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="8890a-108">Example</span></span>  
 <span data-ttu-id="8890a-109">Následující kód generuje CS0745:</span><span class="sxs-lookup"><span data-stu-id="8890a-109">The following code generates CS0745:</span></span>

using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };
