# <a name="compiler-error-cs0745"></a>Chyba kompilátoru CS0745
Očekávalo se kontextové klíčové slovo by.  
  
 Vzor `group` klauzule je následován volitelnou `group...by` `into`, jak je znázorněno v následujícím příkladu:  
  
```csharp  
string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
var query = from name in names  
            group name by name[0];  
```  
  
 or  
  
```csharp  
var query2 = from name in names  
             group name by name[0] into g  
             //...additional query clauses  
```  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Přidejte klíčové slovo do `group`klauzule. `by`  
  
## <a name="example"></a>Příklad  
 Následující kód generuje CS0745:  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
