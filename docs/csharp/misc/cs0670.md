# <a name="compiler-error-cs0670"></a>Compiler Error CS0670
Pole nemůže být typu void.  
  
 Pole byl deklarován jako typ [void](../../csharp/language-reference/keywords/void.md).  
  
 Následující ukázka generuje CS0670:  
  
```csharp  
// CS0670.cs  
class C  
{  
   void f;   // CS0670  
   // try the following line instead  
   // public int f;  
  
   public static void Main()  
   {  
      C myc = new C();  
      myc.f = 0;  
   }  
}  
```
