# <a name="compiler-error-cs1530"></a>Compiler Error CS1530
Klíčového slova 'new' není povolen v prvcích definovaných v oboru názvů  
  
 Není potřeba zadávat [nové](../../csharp/language-reference/keywords/new.md) – klíčové slovo v žádné konstrukci, která je v [obor názvů](../../csharp/language-reference/keywords/namespace.md).  
  
 Následující ukázka generuje CS1530:  
  
```csharp  
// CS1530.cs  
namespace a  
{  
   new class i   // CS1530  
   {  
   }  
  
   // try the following instead  
   class ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
