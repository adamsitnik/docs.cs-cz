# <a name="compiler-error-cs0040"></a>Chyba kompilátoru CS0040
Neočekávaná chyba při vytváření souboru s ladicími informacemi – důvod  
  
 K této chybě může dojít při použití možnosti kompilátoru [-Debug](../language-reference/compiler-options/debug-compiler-option.md) a označuje, že kompilátor nemohl zapisovat do souboru. pdb. Možná řešení této chyby zahrnují přeinstalaci sady Visual Studio, která zajišťuje, že má kompilátor oprávnění k zápisu do souboru nebo adresáře, nebo není zkompilován pomocí/Debug.
