# <a name="compiler-error-cs0040"></a>CS0040 chyby kompilátoru
Neočekávaná chyba při vytváření souboru informace ladění – 'důvod:  
  
 Této chybě může dojít při použití [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) – možnost kompilátoru a označuje, že kompilátor se nepodařilo zapsat do souboru pdb. Opětovná instalace sady Visual Studio, zajištění, že kompilátor má oprávnění k zápisu do souboru nebo adresáře nebo není kompilaci pomocí/Debug jsou uvedena možná řešení této chyby.
