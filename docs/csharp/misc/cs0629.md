# <a name="compiler-error-cs0629"></a><span data-ttu-id="c2aff-102">Compiler Error CS0629</span><span class="sxs-lookup"><span data-stu-id="c2aff-102">Compiler Error CS0629</span></span>
<span data-ttu-id="c2aff-103">Podmíněný člen 'member' nemůže implementovat člen rozhraní 'členu základní třídy' v typu 'Type Name'</span><span class="sxs-lookup"><span data-stu-id="c2aff-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="c2aff-104">[Podmíněného](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) atributu nelze použít na implementaci rozhraní.</span><span class="sxs-lookup"><span data-stu-id="c2aff-104">The [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="c2aff-105">Následující ukázka generuje CS0629:</span><span class="sxs-lookup"><span data-stu-id="c2aff-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
