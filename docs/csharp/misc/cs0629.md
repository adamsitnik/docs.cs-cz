# <a name="compiler-error-cs0629"></a><span data-ttu-id="1a0aa-102">Chyba kompilátoru CS0629</span><span class="sxs-lookup"><span data-stu-id="1a0aa-102">Compiler Error CS0629</span></span>
<span data-ttu-id="1a0aa-103">Člen podmíněného členu nemůže implementovat člen rozhraní Base Class v typu Type Name.</span><span class="sxs-lookup"><span data-stu-id="1a0aa-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="1a0aa-104">[Podmíněný](../programming-guide/concepts/attributes/common-attributes.md#Conditional) atribut nelze použít při implementaci rozhraní.</span><span class="sxs-lookup"><span data-stu-id="1a0aa-104">The [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="1a0aa-105">Následující ukázka generuje CS0629:</span><span class="sxs-lookup"><span data-stu-id="1a0aa-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
