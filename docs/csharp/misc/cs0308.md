# <a name="compiler-error-cs0308"></a><span data-ttu-id="a6846-102">CS0308 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="a6846-102">Compiler Error CS0308</span></span>
<span data-ttu-id="a6846-103">Neobecný typ nebo metodu 'identifier' nelze použít s argumenty typů.</span><span class="sxs-lookup"><span data-stu-id="a6846-103">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="a6846-104">Metoda nebo typ není obecný, ale byl použit s argumenty typů.</span><span class="sxs-lookup"><span data-stu-id="a6846-104">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="a6846-105">Lze vyvarovat této chyby, odeberte ostrými závorkami a argumenty typu nebo předeklarovat metody nebo typu jako typu nebo obecné metody.</span><span class="sxs-lookup"><span data-stu-id="a6846-105">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="a6846-106">Následující příklad generuje CS0308:</span><span class="sxs-lookup"><span data-stu-id="a6846-106">The following example generates CS0308:</span></span>  
  
```csharp  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="a6846-107">Následující příklad generuje také CS0308.</span><span class="sxs-lookup"><span data-stu-id="a6846-107">The following example also generates CS0308.</span></span> <span data-ttu-id="a6846-108">Chcete chybu vyřešit, použijte direktivu "pomocí System.Collections.Generic."</span><span class="sxs-lookup"><span data-stu-id="a6846-108">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```csharp  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
