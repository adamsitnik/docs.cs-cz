# <a name="compiler-error-cs1509"></a>Compiler Error CS1509
Odkazovaný soubor 'file' není sestavení; Použijte ' / addmodule' místo něj parametr  
  
 Výstupní soubor (výstupní soubor 1), vytvořilo v kompilaci, která používá [/target: Module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md) (nemá manifest sestavení), byl určen jako [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md). Takže místo přidávání sestavení k sestavení pro aktuální program, informace metadat ve výstupním souboru 1 se přidají do sestavení pro aktuální program.
