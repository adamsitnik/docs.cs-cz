# <a name="compiler-error-cs1509"></a>Chyba kompilátoru CS1509
Odkazovaný soubor ' file ' není sestavením. místo toho použijte možnost/addmodule  
  
 Výstupní soubor (výstupní soubor 1) vytvořený v kompilaci, která používá [/target: Module](../language-reference/compiler-options/target-module-compiler-option.md) (nemá manifest sestavení), byl zadán pro [/reference](../language-reference/compiler-options/reference-compiler-option.md). Takže místo připojení sestavení k sestavení pro aktuální program budou informace o metadatech ve výstupním souboru 1 přidány do sestavení pro aktuální program.
