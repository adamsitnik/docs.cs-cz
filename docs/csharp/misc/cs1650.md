# <a name="compiler-error-cs1650"></a><span data-ttu-id="3b3e3-102">Compiler Error CS1650</span><span class="sxs-lookup"><span data-stu-id="3b3e3-102">Compiler Error CS1650</span></span>
<span data-ttu-id="3b3e3-103">Pole statického pole 'identifier' nelze přiřazovat (kromě statického konstruktoru nebo inicializátoru proměnné)</span><span class="sxs-lookup"><span data-stu-id="3b3e3-103">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="3b3e3-104">Tato chyba nastane, pokud při pokusu upravit o člen pole, která je jen pro čtení a statické kde není povoleno upravovat.</span><span class="sxs-lookup"><span data-stu-id="3b3e3-104">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="3b3e3-105">Chcete-li vyřešit tuto chybu, omezení přiřazení pole jen pro čtení do konstruktoru nebo inicializátoru proměnné nebo odebrat `readonly` – klíčové slovo z deklarace pole.</span><span class="sxs-lookup"><span data-stu-id="3b3e3-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
