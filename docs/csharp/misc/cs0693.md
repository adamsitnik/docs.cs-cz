# <a name="compiler-warning-level-3-cs0693"></a>Kompilátor CS0693 upozornění (úroveň 3)
Parametr typu 'parametr typu' má stejný název jako parametr typu z vnějšího typu 'type'  
  
 K této chybě dochází v případě, že máte obecný člen, jako je například metoda v generické třídy. Protože parametr typu metody není nutně stejné jako parametr typu třídy, nelze je poskytnout stejný název. Další informace najdete v tématu [obecné metody](../../csharp/programming-guide/generics/generic-methods.md).  
  
 Abyste předešli této situaci, použijte jiný název pro jeden z parametrů typu.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0693.  
  
```csharp  
// CS0693.cs  
// compile with: /W:3 /target:library  
class Outer<T>  
{  
   class Inner<T> {}   // CS0693  
   // try the following line instead  
   // class Inner<U> {}  
}  
```
