# <a name="compiler-error-cs1621"></a><span data-ttu-id="b76f2-102">Chyba kompilátoru CS1621</span><span class="sxs-lookup"><span data-stu-id="b76f2-102">Compiler Error CS1621</span></span>
<span data-ttu-id="b76f2-103">Příkaz yield se nedá použít uvnitř anonymní metody nebo výrazu lambda.</span><span class="sxs-lookup"><span data-stu-id="b76f2-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="b76f2-104">Nemůžete použít příkaz [yield](../../csharp/language-reference/keywords/yield.md) v [anonymní metodě](../language-reference/operators/delegate-operator.md) nebo [výrazu lambda](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="b76f2-104">You cannot use the [yield](../../csharp/language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="b76f2-105">Následující ukázka generuje CS1621:</span><span class="sxs-lookup"><span data-stu-id="b76f2-105">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
