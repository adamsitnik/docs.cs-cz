# <a name="compiler-error-cs1621"></a>Chyba kompilátoru CS1621
Příkaz yield se nedá použít uvnitř anonymní metody nebo výrazu lambda.  
  
Nemůžete použít příkaz [yield](../../csharp/language-reference/keywords/yield.md) v [anonymní metodě](../language-reference/operators/delegate-operator.md) nebo [výrazu lambda](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md).
  
Následující ukázka generuje CS1621:
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
