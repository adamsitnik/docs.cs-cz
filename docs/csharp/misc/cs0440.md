<span data-ttu-id="e9ce7-103">Definování aliasu s názvem "globální" se nedoporučuje, protože ' global:: "vždy odkazuje na globální obor názvů, ne na alias</span><span class="sxs-lookup"><span data-stu-id="e9ce7-103">Defining an alias named 'global' is ill-advised since 'global::' always references the global namespace and not an alias</span></span>

## <a name="example"></a><span data-ttu-id="e9ce7-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="e9ce7-105">Example</span></span>  
 <span data-ttu-id="e9ce7-106">Následující příklad generuje CS0440:</span><span class="sxs-lookup"><span data-stu-id="e9ce7-106">The following example generates CS0440:</span></span>  
  
```csharp  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```
