# <a name="compiler-error-cs1655"></a>Compiler Error CS1655
Nelze předat pole "proměnné" ref nebo out argument vzhledem k tomu, že je "jen pro čtení typu proměnné.  
  
 K této chybě dochází, pokud se pokoušíte předat členem [foreach](../../csharp/language-reference/keywords/foreach-in.md) proměnnou, [pomocí](../../csharp/language-reference/keywords/using-statement.md) proměnnou, nebo [oprava](../../csharp/language-reference/keywords/fixed-statement.md) funkci ref nebo out argument proměnné. Protože tyto proměnné jsou považovány za jen pro čtení v těchto kontextech, to není povoleno.  
  
 Následující ukázka generuje CS1655:  
  
```csharp  
// CS1655.cs  
struct S   
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
