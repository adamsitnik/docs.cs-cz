# <a name="compiler-error-cs1011"></a>Chyba kompilátoru CS1011
Prázdný znakový literál  
  
 Typ [char](../language-reference/keywords/char.md) byl deklarován a inicializován na hodnotu null. Inicializace a `char` musí určovat znak.  
  
 Následující ukázka generuje CS1011:  
  
```csharp  
// CS1011.cs  
class Sample  
{  
   public char CharField = '';   // CS1011  
}  
```
