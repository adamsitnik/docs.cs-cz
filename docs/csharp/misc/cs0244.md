# <a name="compiler-error-cs0244"></a>Chyba kompilátoru CS0244
Typ ukazatele není platný ani jako.  
  
 Operátory [is](../../csharp/language-reference/operators/type-testing-and-cast.md#is-operator) a [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) nejsou platné pro použití na typech ukazatelů. Další informace naleznete v tématu [nezabezpečený kód a ukazatele](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Následující ukázka generuje CS0244:  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
