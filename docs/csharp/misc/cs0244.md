 <span data-ttu-id="913dc-104">Operátory [is](../../csharp/language-reference/operators/type-testing-and-cast.md#is-operator) a [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) nejsou platné pro použití na typech ukazatelů.</span><span class="sxs-lookup"><span data-stu-id="913dc-104">The [is](../../csharp/language-reference/operators/type-testing-and-cast.md#is-operator) and [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) operators are not valid for use on pointer types.</span></span> <span data-ttu-id="913dc-105">Další informace naleznete v tématu [nezabezpečený kód a ukazatele](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="913dc-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="913dc-106">Následující ukázka generuje CS0244:</span><span class="sxs-lookup"><span data-stu-id="913dc-106">The following sample generates CS0244:</span></span>  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
