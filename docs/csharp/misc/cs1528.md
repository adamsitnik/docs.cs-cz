# <a name="compiler-error-cs1528"></a>Compiler Error CS1528
Očekával se znak; nebo = (nejde zadat argumenty konstruktoru v deklaraci)  
  
 Odkaz na třídu byl vytvořen, jako kdyby byl právě vytvořen objekt do třídy. Například byl pokus předat proměnnou do konstruktoru. Použití [nové](../../csharp/language-reference/operators/new-operator.md) operátoru pro vytvoření objektu třídy.  
  
 Následující ukázka generuje CS1528:  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
