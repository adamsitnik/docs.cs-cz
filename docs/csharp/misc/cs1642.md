## <a name="example"></a><span data-ttu-id="08ac8-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="08ac8-106">Example</span></span>  
 <span data-ttu-id="08ac8-107">Následující ukázka generuje CS1642.</span><span class="sxs-lookup"><span data-stu-id="08ac8-107">The following sample generates CS1642.</span></span>

```csharp  
unsafe class C  
{  
   fixed int a[10];   // CS1642

{  
    fixed int a[10];  
}

{  
   public int[] a = null;  
}  
```
