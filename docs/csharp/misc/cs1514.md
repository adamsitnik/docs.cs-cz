# <a name="compiler-error-cs1514"></a>Compiler Error CS1514
očekáván znak {}  
  
 Kompilátor očekává otevírající složenou závorku (`{`), který nebyl nalezen.  
  
 Následující ukázka generuje CS1514:  
  
```csharp  
// CS1514.cs  
namespace x  
// CS1514, no open curly brace  
```
