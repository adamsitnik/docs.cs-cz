# <a name="compiler-error-cs0677"></a>CS0677 chyby kompilátoru
'variable': pole s modifikátorem volatile nemůže být typu 'type'  
  
 Pole deklarované s `volatile` – klíčové slovo musí být jedna z následujících typů:  
  
- jakéhokoliv odkazového typu  
  
- Libovolný typ ukazatele (v `unsafe` kontextu)  
  
- Typy `sbyte`, **bajtů**, **krátký**, `ushort`, `int`, `uint`, `char`, **float**, `bool`  
  
- Výčtové typy založené na některé z výše uvedených typů  
  
 Následující ukázka generuje CS0677:  
  
```csharp  
// CS0677.cs  
class TestClass  
{  
   private volatile long i;   // CS0677  
  
   public static void Main()  
   {  
   }  
}  
```
