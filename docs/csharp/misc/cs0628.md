# <a name="compiler-warning-level-4-cs0628"></a>Kompilátor CS0628 upozornění (úroveň 4)
'member': v zapečetěné třídě je deklarovaný nový chráněný člen  
  
 A [zapečetěné](../../csharp/language-reference/keywords/sealed.md) třídy nelze zavést [chráněné](../../csharp/language-reference/keywords/protected.md) člen protože žádná třída se nemůže dědit z `sealed` třídy a použít `protected` člena.  
  
 Následující ukázka generuje CS0628:  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
