# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="fe5fb-102">Upozornění kompilátoru (úroveň 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="fe5fb-102">Compiler Warning (level 4) CS0628</span></span>
<span data-ttu-id="fe5fb-103">člen: v zapečetěné třídě je deklarovaný nový chráněný člen.</span><span class="sxs-lookup"><span data-stu-id="fe5fb-103">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="fe5fb-104">[Zapečetěná](../language-reference/keywords/sealed.md) třída nemůže zavést [chráněného](../language-reference/keywords/protected.md) člena, protože žádná jiná třída nebude moci `sealed` dědit ze třídy a použít `protected` člena.</span><span class="sxs-lookup"><span data-stu-id="fe5fb-104">A [sealed](../language-reference/keywords/sealed.md) class cannot introduce a [protected](../language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="fe5fb-105">Následující ukázka generuje CS0628:</span><span class="sxs-lookup"><span data-stu-id="fe5fb-105">The following sample generates CS0628:</span></span>  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
