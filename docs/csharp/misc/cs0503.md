# <a name="compiler-error-cs0503"></a>Compiler Error CS0503
Abstraktní metoda "method" nemohou být označeny virtuální  
  
 Je redundantní. pro označení metody člen obě [abstraktní](../../csharp/language-reference/keywords/abstract.md) a [virtuální](../../csharp/language-reference/keywords/virtual.md) protože **abstraktní** znamená **virtuální**.  
  
 Následující ukázka generuje CS0503:  
  
```csharp  
// CS0503.cs  
namespace x  
{  
   abstract public class clx  
   {  
      abstract virtual public void f();   // CS0503  
   }  
}  
```
