# <a name="compiler-warning-level-2-cs0114"></a>Kompilátor CS0114 upozornění (úroveň 2)
'function1' skrývá zděděný člen "function2". Chcete-li aktuální metoda tuto implementaci přepsat, přidejte klíčové slovo override. Jinak přidejte klíčové slovo new.  
  
 Deklaraci ve třídě v konfliktu s deklarací v základní třídě tak, aby se skryjí členu základní třídy.  
  
 Další informace najdete v tématu [základní](../../csharp/language-reference/keywords/base.md).  
  
 Následující ukázka generuje CS0114:  
  
```csharp  
// CS0114.cs  
// compile with: /W:2 /warnaserror  
abstract public class clx  
{  
   public abstract void f();  
}  
  
public class cly : clx  
{  
   public void f() // CS0114, hides base class member  
   // try the following line instead  
   // override public void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
