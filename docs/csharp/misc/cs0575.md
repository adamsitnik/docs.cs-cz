# <a name="compiler-error-cs0575"></a>Compiler Error CS0575
Pouze typy tříd mohou obsahovat destruktory.  
  
 A [struktura](../../csharp/language-reference/keywords/struct.md) nemůže obsahovat destruktor.  
  
 Následující ukázka generuje CS0575:  
  
```csharp  
// CS0575.cs  
namespace x  
{  
   public struct iii  
   {  
      ~iii()   // CS0575  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
