# <a name="compiler-error-cs1605"></a>Chyba kompilátoru CS1605
Klíčové slovo var nelze předat jako argument ref nebo out, protože je jen pro čtení.  
  
 Očekává se, že proměnná předaná jako parametr [ref](../language-reference/keywords/ref.md) nebo [out](../language-reference/keywords/out-parameter-modifier.md) se upraví v volané metodě. Proto není možné předat parametr jen pro čtení jako `ref` nebo. `out`
