# <a name="compiler-error-cs1605"></a>Compiler Error CS1605
Nelze předat "var" ref nebo out argument protože je jen pro čtení  
  
 Proměnná se předává jako [ref](../../csharp/language-reference/keywords/ref.md) nebo [si](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametr očekává se dají upravit v volané metody. Proto není možné předat jako parametr jen pro čtení `ref` nebo `out`.
