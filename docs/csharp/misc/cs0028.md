# <a name="compiler-warning-level-4-cs0028"></a>Kompilátor CS0028 upozornění (úroveň 4)
'Deklarace funkce' má nesprávný podpis na vstupní bod  
  
 Deklarace metody pro `Main` byl neplatný: byl deklarován s neplatným podpisem. `Main` musí být deklarované jako statické a musí vrátit buď [int](../../csharp/language-reference/builtin-types/integral-numeric-types.md) nebo [void](../../csharp/language-reference/keywords/void.md). Další informace najdete v tématu [Main() a argumenty příkazového řádku](../../csharp/programming-guide/main-and-command-args/index.md).  
  
 Následující ukázka generuje CS0028:  
  
```csharp  
// CS0028.cs  
// compile with: /W:4 /warnaserror  
public class a  
{  
    public static double Main(int i)   // CS0028  
    // Try the following line instead:  
    // public static void Main()  
    {  
    }  
}  
```
