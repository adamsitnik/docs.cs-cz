# <a name="compiler-warning-level-1-cs3000"></a>Kompilátor CS3000 upozornění (úroveň 1)
Metody s proměnnými argumenty nejsou kompatibilní se Specifikací CLS  
  
 Argumenty použité v metodě vystavit funkce, které nejsou v specifikací CLS (Common Language). Další informace o dodržování specifikace CLS naleznete v tématu [jazyková nezávislost a jazykově nezávislé komponenty](../../standard/language-independence.md).
  
 Následující příklad generuje upozornění CS3000.  
  
```csharp  
// CS3000.cs  
// compile with: /target:library  
// CS3000 expected  
[assembly:System.CLSCompliant(true)]  
  
public class Test  
{  
   public void AddABunchOfInts( __arglist ) {}   // CS3000  
}  
```
