# <a name="compiler-error-cs4009"></a>Chyba kompilátoru CS4009

' Type. Method ': vstupní bod nemůže být označen `async` modifikátorem.

`async` Klíčové slovo nelze použít v vstupním bodě aplikace ( `Main` obvykle metoda).  

> [!IMPORTANT]
> Počínaje C# 7,1 `Main` může `async` metoda mít modifikátor. Další informace najdete v článku věnovaném [asynchronním hlavním](../whats-new/csharp-7-1.md#async-main) částem [novinky v C# 7,1](../whats-new/csharp-7-1.md) . Informace o tom, jak vybrat C# jazykovou verzi, najdete v článku věnovaném [ C# jazykové verzi](../language-reference/configure-language-version.md) .

Následující příklad vytvoří CS4009:

```csharp
using System;
using System.Threading.Tasks;

public class Example
{
   public static async void Main()
   {
       Console.WriteLine("About to wait two seconds");
       await WaitTwoSeconds();
       Console.WriteLine("About to exit the program");
   }

   private static async Task WaitTwoSeconds()
   {
      await Task.Delay(2000);
      Console.WriteLine("Returning from an asynchronous method");
   } 
}
```

## <a name="to-correct-this-error"></a>Oprava této chyby

[Aktualizujte C# jazykovou verzi](../language-reference/configure-language-version.md) , kterou projekt používá, na 7,1 nebo vyšší.

Pokud používáte C# 7,0 nebo nižší, odeberte `async` klíčové slovo z podpisu vstupního bodu aplikace.  Také odeberte všechna `await` klíčová slova, která jste použili k čekání asynchronních metod v vstupním bodě aplikace. 

Nicméně stále musíte počkat na dokončení asynchronní metody, než vaše vstupní bod obnoví provádění. V opačném případě kompilace vygeneruje upozornění kompilátoru CS4014 a aplikace skončí před dokončením asynchronní operace. Následující příklad ukazuje tento problém.
