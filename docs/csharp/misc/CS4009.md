# <a name="compiler-error-cs4009"></a>Chyba kompilátoru CS4009

' Type. Method ': vstupní bod nemůže být označen modifikátorem `async`.

V vstupním bodě aplikace nemůžete použít klíčové slovo `async` (obvykle metoda `Main`).

> [!IMPORTANT]
> Počínaje C# 7,1 může metoda `Main` obsahovat modifikátor `async`. Další informace najdete v článku věnovaném [asynchronním hlavním](../whats-new/csharp-7-1.md#async-main) částem [ C# novinky v 7,1](../whats-new/csharp-7-1.md) . Informace o tom, jak vybrat C# jazykovou verzi, najdete v článku věnovaném [ C# jazykové verzi](../language-reference/configure-language-version.md) .

## <a name="example"></a>Příklad

Následující příklad vytvoří CS4009:

```csharp
using System;
using System.Threading.Tasks;

public class Example
{
    public static async void Main()
    {
        Console.WriteLine("About to wait two seconds");
        await WaitTwoSeconds();
        Console.WriteLine("About to exit the program");
    }

    private static async Task WaitTwoSeconds()
    {
        await Task.Delay(2000);
        Console.WriteLine("Returning from an asynchronous method");
    } 
}
```

## <a name="to-correct-this-error"></a>Oprava této chyby

[Aktualizujte C# jazykovou verzi](../language-reference/configure-language-version.md) , kterou projekt používá, na 7,1 nebo vyšší.

Pokud používáte C# 7,0 nebo nižší, odeberte klíčové slovo `async` z podpisu vstupního bodu aplikace. Odeberte také všechna klíčová slova `await`, která jste použili k čekání na asynchronní metody v vstupním bodě aplikace.
