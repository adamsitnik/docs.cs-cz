# <a name="compiler-error-cs0586"></a>Compiler Error CS0586
Vnitřní chyba kompilátoru: fáze 'fázi.  
  
 Pokuste se zjistit, pokud kompilátor selhává kvůli neschopnost analyzovat neočekávaná syntaxe. Pokud to není tento případ, zkuste [kontaktujte nás](/visualstudio/ide/talk-to-us).
