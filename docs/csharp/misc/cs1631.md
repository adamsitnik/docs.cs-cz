# <a name="compiler-error-cs1631"></a>Compiler Error CS1631
Nejde používat příkaz yield hodnotu v těle klauzule catch.  
  
 Příkaz yield není povoleno z v těle klauzule catch. Chcete-li vyhnout se této chybě, přesuňte příkaz yield mimo tělo klauzule catch.  
  
 Následující ukázka generuje CS1631:  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
