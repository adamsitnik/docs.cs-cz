# <a name="compiler-error-cs0312"></a>Chyba kompilátoru CS0312
Typ typ1 nejde použít jako parametr typu Name v obecném typu nebo metodě Name. Typ s možnou hodnotou null ' typ1 ' nesplňuje omezení ' typ2 '.  
  
 Typ s povolenou hodnotou null se liší od jeho protějšku, který neumožňuje hodnotu null; neexistuje žádný implicitní převod odkazu ani mezi nimi není určen převod. Převod zabalení s možnou hodnotou null nevyhovuje omezení obecného typu. V následujícím příkladu je `Nullable<int>` prvním parametrem typu a druhým parametrem typu `System.Int32`je.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte omezení.  
  
2. V následujícím příkladu nastavte druhý argument typu buď `int?` nebo. `object`  
  
## <a name="example"></a>Příklad  
 Následující kód generuje CS0312:  
  
```csharp  
// cs0312.cs  
class Program  
{  
    static void MTyVar<T, U>() where T : U { }  
  
    static int Main()  
    {  
        MTyVar<int?, int>(); // CS0312  
        return 1;  
    }  
}  
```  
  
 I když je typ s povolenou hodnotou null odlišný od typu, který nepovoluje hodnotu null, jsou povolené různé druhy převodů mezi hodnotou null a hodnotami, které neumožňují hodnotu null.
