# <a name="compiler-error-cs0439"></a>Compiler Error CS0439
Deklarace externího aliasu musí předcházet všem ostatním prvkům definovaným v oboru názvů  
  
 K této chybě dochází při `extern` prohlášení, přichází po něco jiného, například `using` deklarace ve stejném oboru názvů. `extern` Deklarace musí předcházet všem ostatním prvkům oboru názvů.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0439:  
  
```csharp  
// CS0439.cs  
using System;  
  
extern alias MyType;   // CS0439  
// To resolve the error, make the extern alias the first line in the file.  
  
public class Test   
{  
    public static void Main()   
    {  
    }  
}  
```
