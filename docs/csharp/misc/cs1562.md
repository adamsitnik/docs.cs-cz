# <a name="compiler-error-cs1562"></a>Chyba kompilátoru CS1562
Pro výstupy bez zdroje musí být zadaná možnost/out.  
  
 Kompilace by mohla vytvořit výstupní soubor, ale nebyl nalezen žádný soubor zdrojového kódu jako vstup, ze kterého by bylo možné implicitně zadat název výstupního souboru. Například se pokoušíte kompilovat soubor s metadaty nebo pouze prostředky.  
  
 Pomocí možnosti kompilátoru [-out](../language-reference/compiler-options/out-compiler-option.md) zadejte název výstupního souboru.
