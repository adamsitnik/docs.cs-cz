# <a name="compiler-error-cs0192"></a>Compiler Error CS0192
Pole statického pole 'name' nelze předat jako parametr ref nebo out (kromě statického konstruktoru)  
  
 Pole (proměnnou) označené [jen pro čtení](../../csharp/language-reference/keywords/readonly.md) – klíčové slovo nelze předat buď [ref](../../csharp/language-reference/keywords/ref.md) nebo [si](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametr s výjimkou v konstruktoru. Další informace najdete v tématu [pole](../../csharp/programming-guide/classes-and-structs/fields.md).  
  
 CS0192 také výsledků, pokud `readonly` pole je [statické](../../csharp/language-reference/keywords/static.md) a konstruktor není označený jako `static`.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0192.  
  
```csharp 
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
