 <span data-ttu-id="427ca-104">Pole (proměnnou) označené [jen pro čtení](../../csharp/language-reference/keywords/readonly.md) – klíčové slovo nelze předat buď [ref](../../csharp/language-reference/keywords/ref.md) nebo [si](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametr s výjimkou v konstruktoru.</span><span class="sxs-lookup"><span data-stu-id="427ca-104">A field (variable) marked with the [readonly](../../csharp/language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="427ca-105">Další informace najdete v tématu [pole](../../csharp/programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="427ca-105">For more information, see [Fields](../../csharp/programming-guide/classes-and-structs/fields.md).</span></span>

## <a name="example"></a><span data-ttu-id="427ca-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="427ca-107">Example</span></span>  
 <span data-ttu-id="427ca-108">Následující ukázka generuje CS0192.</span><span class="sxs-lookup"><span data-stu-id="427ca-108">The following sample generates CS0192.</span></span>  
  
```csharp 
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
