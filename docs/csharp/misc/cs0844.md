## <a name="example"></a><span data-ttu-id="eea9a-111">Příklad</span><span class="sxs-lookup"><span data-stu-id="eea9a-111">Example</span></span>  
 <span data-ttu-id="eea9a-112">Následující kód vygeneruje CS0844:</span><span class="sxs-lookup"><span data-stu-id="eea9a-112">The following code generates CS0844:</span></span>  
  
```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }
```

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="eea9a-113">Opravte chybu pomocí `this.num` o pole třídy</span><span class="sxs-lookup"><span data-stu-id="eea9a-113">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
