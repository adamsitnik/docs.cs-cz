# <a name="compiler-warning-level-3-cs0067"></a>Kompilátor CS0067 upozornění (úroveň 3)
Událost 'událost' se nikdy nepoužívá.  
  
 [Události](../../csharp/language-reference/keywords/event.md) byla deklarovaná, ale nikdy se nepoužívá ve třídě, ve kterém byl deklarován.  
  
 Následující ukázka generuje CS0067:  
  
```csharp  
// CS0067.cs  
// compile with: /W:3  
using System;  
delegate void MyDelegate();  
  
class MyClass  
{  
   public event MyDelegate evt;   // CS0067  
   // uncomment TestMethod to resolve this CS0067  
/*  
   private void TestMethod()  
   {  
      if (evt != null)  
         evt();  
   }  
*/  
   public static void Main()  
   {  
   }  
}  
```
