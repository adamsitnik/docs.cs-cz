# <a name="compiler-error-cs2018"></a>CS2018 chyby kompilátoru
Nepovedlo se najít soubor cscmsgs.dll zprávy  
  
 Soubor .dll, který obsahuje chyby a upozornění kompilátoru nebyl nalezen. Tento soubor se musí nacházet ve stejném adresáři jako další podpůrné soubory kompilátoru.
