# <a name="compiler-error-cs0404"></a>Compiler Error CS0404
"<" neočekávaný: atributy nemohou být obecné  
  
 Parametry obecného typu nejsou povoleny v atributech. Odeberte parametr typu a ostrými závorkami.  
  
 Následující ukázka generuje CS0404:  
  
```csharp  
// CS0404.cs  
[MyAttrib<int>]  // CS0404  
class C  
{  
   public static void Main()  
   {  
  
   }  
}  
```
