# <a name="compiler-error-cs2011"></a>CS2011 chyby kompilátoru
Nelze otevřít soubor 'file' odpovědi  
  
 Byl zadán soubor odpovědí v kompilaci, ale kompilátor se nepodařilo najít a otevřít soubor.  
  
 Další informace o souborech odpovědí najdete v tématu [@ (určení souboru odezvy)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md).
