# <a name="compiler-error-cs2011"></a>Chyba kompilátoru CS2011
Nepovedlo se otevřít soubor odpovědí.  
  
 V kompilaci byl zadán soubor odezvy, ale kompilátor nemohl soubor najít a otevřít.  
  
 Další informace o souborech odpovědí naleznete v tématu [@ (určení souboru odezvy)](../language-reference/compiler-options/response-file-compiler-option.md).
