# <a name="compiler-error-cs1027"></a>Compiler Error CS1027
\#byl očekáván endif – direktiva  
  
 Odpovídající `#endif` [direktivy preprocesoru](../../csharp/language-reference/preprocessor-directives/index.md) nebyla nalezena pro určitou vlastnost `#if` směrnice. Nebo může kompilátor našli `#endregion` směrnice, i když se žádné odpovídající `#region` direktivy uvnitř `#if` bloku.  
  
 Následující ukázka generuje CS1027:  
  
```csharp  
// CS1027.cs  
#if true   // CS1027, uncomment next line to resolve  
// #endif  
  
namespace x  
{  
   public class clx  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
