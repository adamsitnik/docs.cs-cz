# <a name="compiler-error-cs1107"></a>Compiler Error CS1107
Parametr může mít jenom jeden modifikátor 'modifikátor name'.  
  
 Jedná se o chybu pro modifikátorech parametrů, jako `this`, `ref`, `in`, a `out` objevit více než jednou v definici parametru.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS1107:  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{ 
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
