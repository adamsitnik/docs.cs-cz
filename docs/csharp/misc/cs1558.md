# <a name="compiler-error-cs1558"></a>Compiler Error CS1558
'class' nemá vhodnou statickou metodu Main  
  
 [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) – možnost kompilátoru zadána třída, ve kterém chcete hledat **hlavní** metody. Ale [hlavní](../../csharp/programming-guide/main-and-command-args/index.md) metoda nebyl správně definován.  
  
 Následující příklad generuje CS1558 z důvodu neplatné návratovým typem.  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
