# <a name="compiler-error-cs1558"></a>Chyba kompilátoru CS1558
Třída nemá vhodnou statickou metodu Main.  
  
 Možnost kompilátoru [/Main](../language-reference/compiler-options/main-compiler-option.md) určila třídu, ve které chcete vyhledat metodu **Main** . Metoda [Main](../programming-guide/main-and-command-args/index.md) však nebyla definována správně.  
  
 Následující příklad generuje CS1558 z důvodu neplatného návratového typu.  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
