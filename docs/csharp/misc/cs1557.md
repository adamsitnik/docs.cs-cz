# <a name="compiler-error-cs1557"></a>Chyba kompilátoru CS1557
Klíčové slovo class nelze použít pro metodu Main, protože je v jiném výstupním souboru.  
  
 Možnost kompilátoru [/Main](../language-reference/compiler-options/main-compiler-option.md) byla zadána pro jeden výstupní soubor ve službě kompilace více výstupních souborů. Třída však nebyla nalezena ve zdrojovém kódu pro kompilaci/Main; byl nalezen v souboru zdrojového kódu pro jeden z ostatních výstupních souborů v kompilaci.
