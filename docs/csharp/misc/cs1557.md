# <a name="compiler-error-cs1557"></a>Compiler Error CS1557
'Class' nelze použít pro metodu Main, protože je v jiném výstupním souboru  
  
 [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) – možnost kompilátoru byl zadán pro jednu výstupní soubor v kompilaci více výstupní soubor. Nicméně třída nebyla nalezena ve zdrojovém kódu pro / Main kompilaci. byl nalezen v souboru zdrojového kódu z jednoho z ostatních výstupních souborů v kompilaci.
