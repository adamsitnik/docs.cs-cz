# <a name="compiler-error-cs1015"></a>Compiler Error CS1015
Objektu, řetězec nebo byl očekáván typ třída  
  
 Byl proveden pokus o předání předdefinované datové typy do [catch](../../csharp/language-reference/keywords/try-catch.md) bloku. Pouze datové typy, které jsou odvozeny z <xref:System.Exception?displayProperty=nameWithType> mohou být předány do `catch` bloku. Další informace o výjimkách, naleznete v tématu [výjimek a zpracování výjimek](../../csharp/programming-guide/exceptions/index.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1015:  
  
```csharp  
// CS1015.cs  
class Sample  
{  
    static void Main()  
    {  
        try   
        {  
        }  
        catch(int)   // CS1015, int is not derived from System.Exception  
        {  
        }  
    }  
}  
```
