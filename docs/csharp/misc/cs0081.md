# <a name="compiler-error-cs0081"></a><span data-ttu-id="f983b-102">Chyba kompilátoru CS0081</span><span class="sxs-lookup"><span data-stu-id="f983b-102">Compiler Error CS0081</span></span>
<span data-ttu-id="f983b-103">Deklarace parametru typu musí být identifikátor, ne typ.</span><span class="sxs-lookup"><span data-stu-id="f983b-103">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="f983b-104">Pokud deklarujete obecnou metodu nebo typ, zadejte parametr typu jako identifikátor, například "T" nebo "inputType".</span><span class="sxs-lookup"><span data-stu-id="f983b-104">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="f983b-105">Když kód klienta volá metodu, dodá typ, který nahradí všechny výskyty identifikátoru v těle metody nebo třídy.</span><span class="sxs-lookup"><span data-stu-id="f983b-105">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="f983b-106">Další informace naleznete v tématu [parametry obecného typu](../programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="f983b-106">For more information, see [Generic Type Parameters](../programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
