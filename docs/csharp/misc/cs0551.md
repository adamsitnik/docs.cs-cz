 <span data-ttu-id="c39e3-104">Třída, která explicitně implementuje vlastnost rozhraní, musí implementovat všechny přistupující objekty, které definuje rozhraní.</span><span class="sxs-lookup"><span data-stu-id="c39e3-104">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>

## <a name="example"></a><span data-ttu-id="c39e3-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="c39e3-106">Example</span></span>  
 <span data-ttu-id="c39e3-107">Následující ukázka generuje CS0551.</span><span class="sxs-lookup"><span data-stu-id="c39e3-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
