# <a name="compiler-error-cs0551"></a>Compiler Error CS0551
Explicitní implementace rozhraní 'implementace' nebyl nalezen přístupový objekt 'přístupového objektu'  
  
 Všechny přistupující objekty, které definuje rozhraní musí implementovat třídu, která implementuje explicitně rozhraní vlastnost.  
  
 Další informace najdete v tématu [pomocí vlastnosti](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0551.  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
