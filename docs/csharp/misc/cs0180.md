# <a name="compiler-error-cs0180"></a>Compiler Error CS0180
'member' nemůže být extern i abstract  
  
 [Abstraktní](../../csharp/language-reference/keywords/abstract.md) a [extern](../../csharp/language-reference/keywords/extern.md) klíčová slova se vzájemně vylučují. `extern` – Klíčové slovo znamená, že člen je definovaný mimo soubor, a **abstraktní** znamená, že je k dispozici implementace v odvozené třídě. Další informace najdete v tématu [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Následující ukázka generuje CS0180:  
  
```csharp  
// CS0180.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public extern abstract int Foo(int a);   // CS0180  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
