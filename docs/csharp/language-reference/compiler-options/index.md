# <a name="c-compiler-options"></a>Možnosti kompilátoru C#
Kompilátor vytvoří spustitelné soubory (.exe), dynamické knihovny (DLL) nebo moduly kódu (.netmodule).  
  
 Každý – možnost kompilátoru je k dispozici ve dvou formách: **– možnost** a **/možnost**. V dokumentaci se zobrazují jenom **– možnost** formuláře.  
  
 V sadě Visual Studio můžete nastavit možnosti kompilátoru v souboru web.config. Další informace najdete v tématu [ \<kompilátoru > Element](../../../framework/configure-apps/file-schema/compiler/compiler-element.md).
