## <a name="remarks"></a><span data-ttu-id="2881a-130">Poznámky</span><span class="sxs-lookup"><span data-stu-id="2881a-130">Remarks</span></span>  
  
- <span data-ttu-id="2881a-131">Explicitní číselný převod může způsobit ztrátu přesnosti nebo výsledek v došlo k výjimce, obvykle <xref:System.OverflowException>.</span><span class="sxs-lookup"><span data-stu-id="2881a-131">The explicit numeric conversion may cause loss of precision or result in throwing an exception, typically an <xref:System.OverflowException>.</span></span>  

- <span data-ttu-id="2881a-132">Při převodu hodnoty celočíselného typu na jiný celočíselný typ, výsledek závisí na přetečení [kontrola kontextu](checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="2881a-132">When you convert a value of an integral type to another integral type, the result depends on the overflow [checking context](checked-and-unchecked.md).</span></span> <span data-ttu-id="2881a-133">Ve zkontrolovaném kontextu převod je úspěšný, pokud je hodnota zdrojového rozsahu cílového typu.</span><span class="sxs-lookup"><span data-stu-id="2881a-133">In a checked context, the conversion succeeds if the source value is within the range of the destination type.</span></span> <span data-ttu-id="2881a-134">V opačném případě <xref:System.OverflowException> je vyvolána výjimka.</span><span class="sxs-lookup"><span data-stu-id="2881a-134">Otherwise, an <xref:System.OverflowException> is thrown.</span></span> <span data-ttu-id="2881a-135">V nekontrolovaném kontextu převod vždy úspěšná a probíhá následujícím způsobem:</span><span class="sxs-lookup"><span data-stu-id="2881a-135">In an unchecked context, the conversion always succeeds, and proceeds as follows:</span></span>

  - <span data-ttu-id="2881a-136">Pokud typ zdroje je větší než cílový typ, pak je rozdělená do zdrojové hodnoty se zahodí jeho "navíc" nejvýznamnější bity.</span><span class="sxs-lookup"><span data-stu-id="2881a-136">If the source type is larger than the destination type, then the source value is truncated by discarding its "extra" most significant bits.</span></span> <span data-ttu-id="2881a-137">Výsledek je pak považován za hodnotu cílového typu.</span><span class="sxs-lookup"><span data-stu-id="2881a-137">The result is then treated as a value of the destination type.</span></span>

  - <span data-ttu-id="2881a-138">Pokud typ zdroje je menší než cílový typ, pak zdrojová hodnota je rozšířena o znaménko nebo nulou tak, aby se stejnou velikostí jako typ cíle.</span><span class="sxs-lookup"><span data-stu-id="2881a-138">If the source type is smaller than the destination type, then the source value is either sign-extended or zero-extended so that it is the same size as the destination type.</span></span> <span data-ttu-id="2881a-139">Rozšířením znaménka se používá, pokud typ zdroje je podepsaný; nulové rozšíření se používá, pokud se zdrojový typ není podepsaný.</span><span class="sxs-lookup"><span data-stu-id="2881a-139">Sign-extension is used if the source type is signed; zero-extension is used if the source type is unsigned.</span></span> <span data-ttu-id="2881a-140">Výsledek je pak považován za hodnotu cílového typu.</span><span class="sxs-lookup"><span data-stu-id="2881a-140">The result is then treated as a value of the destination type.</span></span>

  - <span data-ttu-id="2881a-141">Pokud se stejnou velikostí jako cílový typ je typ zdrojového, zdrojová hodnota je považován za hodnotu cílového typu.</span><span class="sxs-lookup"><span data-stu-id="2881a-141">If the source type is the same size as the destination type, then the source value is treated as a value of the destination type.</span></span>
  
- <span data-ttu-id="2881a-142">Při převodu `decimal` hodnota na integrální typ., tato hodnota zaokrouhlena směrem k nule na nejbližší celočíselnou hodnotu.</span><span class="sxs-lookup"><span data-stu-id="2881a-142">When you convert a `decimal` value to an integral type, this value is rounded towards zero to the nearest integral value.</span></span> <span data-ttu-id="2881a-143">Pokud je výsledný celočíselné hodnoty mimo rozsah cílového typu <xref:System.OverflowException> je vyvolána výjimka.</span><span class="sxs-lookup"><span data-stu-id="2881a-143">If the resulting integral value is outside the range of the destination type, an <xref:System.OverflowException> is thrown.</span></span>  
  
- <span data-ttu-id="2881a-144">Při převodu `double` nebo `float` hodnota na integrální typ., tato hodnota zaokrouhlena směrem k nule na nejbližší celočíselnou hodnotu.</span><span class="sxs-lookup"><span data-stu-id="2881a-144">When you convert a `double` or `float` value to an integral type, this value is rounded towards zero to the nearest integral value.</span></span> <span data-ttu-id="2881a-145">Pokud výsledná celočíselné hodnoty je mimo rozsah cílového typu, výsledek závisí na přetečení [kontrola kontextu](checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="2881a-145">If the resulting integral value is outside the range of the destination type, the result depends on the overflow [checking context](checked-and-unchecked.md).</span></span> <span data-ttu-id="2881a-146">Ve zkontrolovaném kontextu <xref:System.OverflowException> je vyvolána, když v nekontrolovaném kontextu, výsledek je neurčená hodnota cílového typu.</span><span class="sxs-lookup"><span data-stu-id="2881a-146">In a checked context, an <xref:System.OverflowException> is thrown, while in an unchecked context, the result is an unspecified value of the destination type.</span></span>  
  
- <span data-ttu-id="2881a-147">Při převodu `double` k `float`, `double` hodnota je zaokrouhlená na nejbližší `float` hodnotu.</span><span class="sxs-lookup"><span data-stu-id="2881a-147">When you convert `double` to `float`, the `double` value is rounded to the nearest `float` value.</span></span> <span data-ttu-id="2881a-148">Pokud `double` hodnota je příliš malá nebo příliš velký a nevejde do cílového typu, výsledkem bude nula nebo nekonečno.</span><span class="sxs-lookup"><span data-stu-id="2881a-148">If the `double` value is too small or too large to fit into the destination type, the result will be zero or infinity.</span></span>  
  
- <span data-ttu-id="2881a-149">Při převodu `float` nebo `double` k `decimal`, zdrojová hodnota je převedena na `decimal` vyjádření a zaokrouhlí na nejbližší číslo po 28 desetinné místo podle potřeby.</span><span class="sxs-lookup"><span data-stu-id="2881a-149">When you convert `float` or `double` to `decimal`, the source value is converted to `decimal` representation and rounded to the nearest number after the 28th decimal place if required.</span></span> <span data-ttu-id="2881a-150">V závislosti na hodnotě zdrojovou hodnotu může dojít jednu z následujících výsledků:</span><span class="sxs-lookup"><span data-stu-id="2881a-150">Depending on the value of the source value, one of the following results may occur:</span></span>  

  - <span data-ttu-id="2881a-151">Pokud zdrojová hodnota je příliš malá, aby se dala vyjádřit jako `decimal`, výsledkem bude nule.</span><span class="sxs-lookup"><span data-stu-id="2881a-151">If the source value is too small to be represented as a `decimal`, the result becomes zero.</span></span>  

  - <span data-ttu-id="2881a-152">Pokud zdrojová hodnota NaN (není číslo), nekonečno, nebo příliš velký, aby se dala vyjádřit jako `decimal`, <xref:System.OverflowException> je vyvolána výjimka.</span><span class="sxs-lookup"><span data-stu-id="2881a-152">If the source value is NaN (not a number), infinity, or too large to be represented as a `decimal`, an <xref:System.OverflowException> is thrown.</span></span>  
  
- <span data-ttu-id="2881a-153">Při převodu `decimal` k `float` nebo `double`, `decimal` hodnota je zaokrouhlená na nejbližší `double` nebo `float` hodnotu.</span><span class="sxs-lookup"><span data-stu-id="2881a-153">When you convert `decimal` to `float` or `double`, the `decimal` value is rounded to the nearest `double` or `float` value.</span></span>
