```csharp

```

<span data-ttu-id="9334b-126">Použijte operátor [typeof](../operators/type-testing-and-cast.md#typeof-operator) k získání instance <xref:System.Type?displayProperty=nameWithType>, která představuje zadaný typ:</span><span class="sxs-lookup"><span data-stu-id="9334b-126">Use the [typeof](../operators/type-testing-and-cast.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp

Type doubleType = typeof(System.Double);
Console.WriteLine(doubleType.FullName);

// Output:
// System.String
// System.Double
```
