
<span data-ttu-id="2a115-126">Použití [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) operátor zobrazíte <xref:System.Type?displayProperty=nameWithType> instanci, která představuje zadaný typ:</span><span class="sxs-lookup"><span data-stu-id="2a115-126">Use the [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
// System.String
// System.Double
```
