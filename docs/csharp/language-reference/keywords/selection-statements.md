# <a name="selection-statements-c-reference"></a>Příkazy výběru (C# odkaz)

Následující klíčová slova se používají v příkazech výběr:

- [if](if-else.md)

- [else](if-else.md)

- [switch](switch.md)

- [case](switch.md)
