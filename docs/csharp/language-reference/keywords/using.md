# <a name="using-c-reference"></a>using (Referenční dokumentace jazyka C#)

`using` – Klíčové slovo má dvě hlavní využití:

- Jako směrnice, když se používá k vytvoření aliasu pro obor názvů a importu typy definované v jiných oborech názvů. Zobrazit [direktiva using](using-directive.md).

- Jako příkaz, pokud ho definuje obor na konci, z nichž bude uvolněn objekt. Zobrazit [příkaz using](using-statement.md).

Kromě toho [pomocí statické](using-static.md) umožňuje definovat typ obsahující statickou máte přístup bez určení názvu typu.
