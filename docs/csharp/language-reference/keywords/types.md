# <a name="types-c-reference"></a>Typy (Referenční dokumentace jazyka C#)

Zadáním systému C# obsahuje následující kategorie:

- [Typy hodnot](value-types.md)

- [Odkazové typy](reference-types.md)

- [Typy ukazatelů](../../programming-guide/unsafe-code-pointers/pointer-types.md)

 Proměnné, které jsou typy hodnot ukládání dat, a ty, které jsou odkazové typy ukládání odkazů na skutečná data. Instance typy odkazů jsou také označovány jako objekty. Typy ukazatelů lze použít pouze ve [nebezpečné](unsafe.md) režimu.

 Je možné převést typ hodnoty na odkazový typ a zpět na typ hodnoty, s použitím [zabalení a rozbalení](../../../csharp/programming-guide/types/boxing-and-unboxing.md). S výjimkou zabalený typ hodnoty nelze převést typ odkazu na typ hodnoty.

 Tato část také zavádí [void](void.md).
