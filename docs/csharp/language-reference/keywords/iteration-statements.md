# <a name="iteration-statements-c-reference"></a>Příkazy iterace (Referenční dokumentace jazyka C#)

Vytvoření smyček pomocí příkazy iterace. Příkazy příčina vložené příkazy iterace bude spuštěn s počtem opakování můžou kritéria ukončení smyčky. Tyto příkazy jsou spouštěny v pořadí, kromě případů, kdy [příkaz skoku](jump-statements.md) dochází.

Následující klíčová slova se používají v příkazech iterace:

- [do](do.md)

- [for](for.md)

- [foreach, in](foreach-in.md)

- [while](while.md)
