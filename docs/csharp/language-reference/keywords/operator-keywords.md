# <a name="operator-keywords-c-reference"></a>Klíčová slova operátorů (Referenční dokumentace jazyka C#)

Používat různé akce, třeba vytvořit objekty, kontrolu za běhu typ objektu, získávání velikosti, typu a provádět jiné akce. Tato část představuje následující klíčová slova:

- [jako](as.md) převede objekt na kompatibilní typ.

- [operátor await](await.md) pozastaví [asynchronní](async.md) metodu, dokud nebude dokončena očekávaná úloha.

- [je](is.md) kontroly za běhu typ objektu, nebo (počínaje C# 7.0) testuje výraz se vzorem.

- [new](new.md)

  - [Operátor new](new-operator.md) vytvoří objekty.

  - [New – modifikátor](new-modifier.md) skrývá zděděný člen.

  - [New – omezení](new-constraint.md) kvalifikuje parametr typu.

- [nameof](nameof.md) získá jednoduchého (nekvalifikovaného) řetězcového názvu proměnné, typ nebo člen.

- [operátor sizeof:](sizeof.md) získá velikost nespravovaným typem.  

- [typeof](typeof.md) obdrželi <xref:System.Type?displayProperty=nameWithType> pro typ objektu.  

- [stackalloc](stackalloc.md) přiděluje blok paměti v zásobníku.  

Následující klíčová slova, které je možné použít jako operátory a jako příkazy, jsou popsané v [příkazy](statement-keywords.md) části:

- [checked](checked.md) kontextu určuje zaškrtnuto.  

- [unchecked](unchecked.md) určuje nezkontrolovaném kontextu.
