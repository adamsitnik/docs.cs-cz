# <a name="compiler-error-cs1644"></a>Chyba kompilátoru CS1644
Funkce 'funkce' není součástí standardizované ISO C# specifikace jazyka a možná ji nepůjde použít ostatních kompilátorech  
  
 K této chybě dochází, pokud jste zadali [/langversion](../../../csharp/language-reference/compiler-options/langversion-compiler-option.md) možnost ISO-1 a při kompilaci kódu používá funkce, které nejsou součástí standardu ISO 1.0. Chcete-li vyřešit tuto chybu, nepoužívejte některý C# 2.0 funkce kompilátoru, například obecné typy nebo anonymní metody s možností kompatibility ISO-1.  
  
 Následující ukázka generuje CS1644:  
  
```csharp  
// CS1644.cs  
// compile with: /langversion:ISO-1 /target:library  
class C<T> {}   // CS1644  
```
