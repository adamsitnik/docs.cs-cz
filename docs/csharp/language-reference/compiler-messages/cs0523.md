# <a name="compiler-error-cs0523"></a>Chyba kompilátoru CS0523
Člen struktury 'strukt2"typ 'struct1' způsobuje cyklus v rozložení struktury  
  
 Definice dvou struktur zahrnout rekurzivních odkazů. Změnit [struktura](../../../csharp/language-reference/keywords/struct.md) definice tak, aby každý nebyla definována na druhý. Toto omezení se vztahuje pouze na struktury, protože struktury jsou typy hodnot. Pokud používáte rekurzivní odkazy, deklarujte jako třídy typů.  
  
 Následující ukázka generuje CS0523:  
  
```csharp  
// CS0523.cs  
// compile with: /target:library  
struct RecursiveLayoutStruct1  
{  
   public RecursiveLayoutStruct2 field;  
}  
  
struct RecursiveLayoutStruct2  
{  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
