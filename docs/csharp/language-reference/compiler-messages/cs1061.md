## <a name="example"></a><span data-ttu-id="d9b52-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="d9b52-105">Example</span></span>  
 <span data-ttu-id="d9b52-106">Následující příklad generuje CS1061, protože `TestClass1` `DisplaySomething` nemá metodu.</span><span class="sxs-lookup"><span data-stu-id="d9b52-106">The following example generates CS1061 because `TestClass1` does not have a `DisplaySomething` method.</span></span> <span data-ttu-id="d9b52-107">Má metodu, která je volána `WriteSomething`.</span><span class="sxs-lookup"><span data-stu-id="d9b52-107">It does have a method that is called `WriteSomething`.</span></span> <span data-ttu-id="d9b52-108">Možná to znamená, že autor tohoto zdrojového kódu chtěl zapisovat.</span><span class="sxs-lookup"><span data-stu-id="d9b52-108">Perhaps that is what the author of this source code meant to write.</span></span>  
  
```csharp  
// cs1061.cs  
public class TestClass1  
{  
    // TestClass1 has one method, called WriteSomething.  
    public void WriteSomething(string s)  
    {  
        System.Console.WriteLine(s);  
    }  
}  
  
public class TestClass2  
{  
    // TestClass2 has one method, called DisplaySomething.  
    public void DisplaySomething(string s)  
    {  
        System.Console.WriteLine(s);  
    }  
}  
  
public class TestTheClasses  
{  
    public static void Main()  
    {  
        TestClass1 tc1 = new TestClass1();  
        TestClass2 tc2 = new TestClass2();  
        // The following call fails because TestClass1 does not have   
        // a method called DisplaySomething.  
        tc1.DisplaySomething("Hello");      // CS1061  
  
        // To correct the error, change the method call to either   
        // tc1.WriteSomething or tc2.DisplaySomething.  
        tc1.WriteSomething("Hello from TestClass1");  
        tc2.DisplaySomething("Hello from TestClass2");  
    }  
}  
```
