# <a name="compiler-error-cs1579"></a><span data-ttu-id="74201-102">Chyba kompilátoru CS1579</span><span class="sxs-lookup"><span data-stu-id="74201-102">Compiler Error CS1579</span></span>
<span data-ttu-id="74201-103">Příkaz foreach nejde použít pro proměnné typu 'type1', protože 'type2' neobsahuje veřejnou definici pro 'identifier'</span><span class="sxs-lookup"><span data-stu-id="74201-103">foreach statement cannot operate on variables of type 'type1' because 'type2' does not contain a public definition for 'identifier'</span></span>

<span data-ttu-id="74201-104">K iteraci v rámci kolekce pomocí [foreach](../keywords/foreach-in.md) prohlášení kolekce musí splňovat následující požadavky:</span><span class="sxs-lookup"><span data-stu-id="74201-104">To iterate through a collection using the [foreach](../keywords/foreach-in.md) statement, the collection must meet the following requirements:</span></span>

- <span data-ttu-id="74201-105">Jeho typ musí obsahovat veřejný konstruktor bez parametrů `GetEnumerator` metoda, jejíž návratový typ je třída, struktura nebo typu rozhraní.</span><span class="sxs-lookup"><span data-stu-id="74201-105">Its type must include a public parameterless `GetEnumerator` method whose return type is either class, struct, or interface type.</span></span>
- <span data-ttu-id="74201-106">Návratový typ `GetEnumerator` metoda musí obsahovat veřejnou vlastnost s názvem `Current` a veřejný konstruktor bez parametrů metodu s názvem `MoveNext` jehož návratový typ je <xref:System.Boolean>.</span><span class="sxs-lookup"><span data-stu-id="74201-106">The return type of the `GetEnumerator` method must contain a public property named `Current` and a public parameterless method named `MoveNext` whose return type is <xref:System.Boolean>.</span></span>

## <a name="example"></a><span data-ttu-id="74201-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="74201-107">Example</span></span>

<span data-ttu-id="74201-108">Následující ukázka generuje CS1579, protože `MyCollection` třída neobsahuje veřejnou `GetEnumerator` metody:</span><span class="sxs-lookup"><span data-stu-id="74201-108">The following sample generates CS1579 because the `MyCollection` class doesn't contain the public `GetEnumerator` method:</span></span>

```csharp  
// CS1579.cs  
using System;  
public class MyCollection   
{  
   int[] items;  
   public MyCollection()   
   {  
      items = new int[5] {12, 44, 33, 2, 50};  
   }  
  
   // Delete the following line to resolve.  
   MyEnumerator GetEnumerator()  
  
   // Uncomment the following line to resolve:  
   // public MyEnumerator GetEnumerator()   
   {  
      return new MyEnumerator(this);  
   }  
  
   // Declare the enumerator class:  
   public class MyEnumerator   
   {  
      int nIndex;  
      MyCollection collection;  
      public MyEnumerator(MyCollection coll)   
      {  
         collection = coll;  
         nIndex = -1;  
      }  
  
      public bool MoveNext()   
      {  
         nIndex++;  
         return (nIndex < collection.items.Length);  
      }  
  
      public int Current => collection.items[nIndex];
   }  
  
   public static void Main()   
   {  
      MyCollection col = new MyCollection();  
      Console.WriteLine("Values in the collection are:");  
      foreach (int i in col)   // CS1579  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
