# <a name="compiler-error-cs0518"></a>Chyba kompilátoru CS0518
Předdefinovaný typ 'type' není definován ani importován  
  
 Hlavní příčinou tohoto problému je, že projekt není import mscorlib.dll, která definuje obor názvů celého systému. Může to být způsobeno jedním z následujících akcí:  
  
- [/Nostdlib](../../../csharp/language-reference/compiler-options/nostdlib-compiler-option.md) byla zadána možnost z příkazového řádku kompilátoru. Možnost/nostdlib brání import souboru mscorlib.dll. Tuto možnost použijte, pokud chcete definovat nebo vytvoření oboru názvů specifických pro uživatele systému.  
  
- Se odkazuje na nesprávnou knihovnu mscorlib.dll.  
  
- Existuje poškozenou Visual Studio .NET nebo rozhraní .NET Framework common language runtime instalaci.  
  
- Zbývající součásti z dřívější instalace, které jsou kompatibilní s nejnovější instalace zůstane.  
  
 Chcete-li vyřešit tento problém, proveďte jednu z následujících akcí:  
  
- Nezadávejte možnosti/nostdlib z příkazového řádku kompilátoru.  
  
- Ujistěte se, že se projekt odkazuje na správný soubor mscorlib.dll.  
  
- Přeinstalujte rozhraní .NET Framework common language runtime (Pokud předchozích řešení nepodaří problém vyřešit).
