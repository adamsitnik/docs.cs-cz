# <a name="compiler-error-cs1946"></a><span data-ttu-id="7c44a-102">Chyba kompilátoru CS1946</span><span class="sxs-lookup"><span data-stu-id="7c44a-102">Compiler Error CS1946</span></span>

<span data-ttu-id="7c44a-103">Výraz anonymní metody nejde převést na strom výrazu.</span><span class="sxs-lookup"><span data-stu-id="7c44a-103">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="7c44a-104">[Anonymní metoda](../operators/delegate-operator.md) představuje sadu příkazů, ale strom výrazu nesmí obsahovat příkaz.</span><span class="sxs-lookup"><span data-stu-id="7c44a-104">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="7c44a-105">Anonymní metodu proto nemůže být reprezentována stromem výrazu.</span><span class="sxs-lookup"><span data-stu-id="7c44a-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="7c44a-106">Chcete-li tuto chybu opravit, změňte anonymní metodu na [výraz lambda](../../programming-guide/statements-expressions-operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="7c44a-106">To correct this error, change the anonymous method to a [lambda expression](../../programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="7c44a-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="7c44a-107">Example</span></span>

<span data-ttu-id="7c44a-108">Následující příklad generuje CS1946:</span><span class="sxs-lookup"><span data-stu-id="7c44a-108">The following example generates CS1946:</span></span>

```csharp
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
