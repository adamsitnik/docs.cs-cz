# <a name="compiler-error-cs0007"></a>Chyba kompilátoru CS0007
Neočekávané common language runtime Chyba při inicializaci – "popisu"  
  
 K této chybě dochází, pokud modul runtime nelze načíst. Tato situace může nastat, pokud není k dispozici na počítači verze common language runtime, který kompilátor se pokusí načíst, nebo pokud common language runtime instalaci nebo konfiguraci je poškozen.  
  
 K tomu může dojít, pokud `csc.exe.config` soubor se změnil. Tento soubor je nakonfigurovaná v průběhu instalace a neměl měnit. Pokud je možnost, která `csc.exe.config` soubor byl změněn, najdete v souboru, abyste měli jistotu, že verze runtime zadané v souboru je na počítači. Pokud je k dispozici správná verze, může být poškozený. Znovu nainstalujte modul common language runtime.
