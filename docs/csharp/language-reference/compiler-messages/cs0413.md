# <a name="compiler-error-cs0413"></a>Chyba kompilátoru CS0413
Parametr typu 'parametr typu' nelze použít s operátorem 'as', protože nemá omezení typu Třída ani omezení 'class'  
  
K této chybě dochází, pokud používá obecného typu [jako](../../language-reference/operators/type-testing-and-conversion-operators.md#as-operator) operátoru, ale, že obecného typu nemá třídu omezení typu. `as` Operátor je povolen pouze v odkazu a typy s možnou hodnotou Null, tak parametr typu musí být omezený zaručí, že není typem hodnoty. Lze vyvarovat této chyby, použijte omezení typu třída nebo omezení typu odkazu.
  
Je to proto, `as` operátor může vrátit `null`, což není možná hodnota pro typ hodnoty, a parametr typu musí zacházet jako typ hodnoty, pokud se omezení typu Třída ani omezení typu odkazu.
  
## <a name="example"></a>Příklad

Následující ukázka generuje CS0413.  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
