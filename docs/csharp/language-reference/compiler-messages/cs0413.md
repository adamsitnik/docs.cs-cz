# <a name="compiler-error-cs0413"></a><span data-ttu-id="5f7d8-102">Chyba kompilátoru CS0413</span><span class="sxs-lookup"><span data-stu-id="5f7d8-102">Compiler Error CS0413</span></span>
<span data-ttu-id="5f7d8-103">Parametr typu typu Parameter nejde použít s operátorem as, protože nemá omezení typu třída ani omezení class.</span><span class="sxs-lookup"><span data-stu-id="5f7d8-103">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint</span></span>  
  
<span data-ttu-id="5f7d8-104">K této chybě dochází, pokud obecný typ používá operátor [as](../operators/type-testing-and-cast.md#as-operator) , ale tento obecný typ nemá omezení typu třída.</span><span class="sxs-lookup"><span data-stu-id="5f7d8-104">This error occurs if a generic type uses the [as](../operators/type-testing-and-cast.md#as-operator) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="5f7d8-105">`as` Operátor je povolen pouze s typy hodnot s možnou hodnotou null, takže parametr typu musí být omezen, aby bylo zaručeno, že se nejedná o hodnotový typ.</span><span class="sxs-lookup"><span data-stu-id="5f7d8-105">The `as` operator is only allowed with reference and nullable value types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="5f7d8-106">Chcete-li se této chybě vyhnout, použijte omezení typu třídy nebo omezení typu odkazu.</span><span class="sxs-lookup"><span data-stu-id="5f7d8-106">To avoid this error, use a class type constraint or a reference type constraint.</span></span>
  
<span data-ttu-id="5f7d8-107">Důvodem je, že `as` operátor může vracet `null`, což není možná hodnota pro typ hodnoty a parametr typu musí být považován za typ hodnoty, pokud se nejedná o omezení typu třídy nebo omezení typu odkazu.</span><span class="sxs-lookup"><span data-stu-id="5f7d8-107">This is because the `as` operator could return `null`, which is not a possible value for a value type, and the type parameter must be treated as a value type unless it is a class type constraint or a reference type constraint.</span></span>
  
## <a name="example"></a><span data-ttu-id="5f7d8-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="5f7d8-108">Example</span></span>

<span data-ttu-id="5f7d8-109">Následující ukázka generuje CS0413.</span><span class="sxs-lookup"><span data-stu-id="5f7d8-109">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
