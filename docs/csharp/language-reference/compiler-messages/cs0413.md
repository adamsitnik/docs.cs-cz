# <a name="compiler-error-cs0413"></a><span data-ttu-id="44ce9-102">Chyba kompilátoru CS0413</span><span class="sxs-lookup"><span data-stu-id="44ce9-102">Compiler Error CS0413</span></span>
<span data-ttu-id="44ce9-103">Parametr typu 'parametr typu' nelze použít s operátorem 'as', protože nemá omezení typu Třída ani omezení 'class'</span><span class="sxs-lookup"><span data-stu-id="44ce9-103">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint</span></span>  
  
 <span data-ttu-id="44ce9-104">K této chybě dochází, pokud používá obecného typu [jako](../../../csharp/language-reference/keywords/as.md) operátoru, ale, že obecného typu nemá třídu omezení typu.</span><span class="sxs-lookup"><span data-stu-id="44ce9-104">This error occurs if a generic type uses the [as](../../../csharp/language-reference/keywords/as.md) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="44ce9-105">`as` Operátor je povolen pouze v případě typů odkazu, tedy parametr typu musí být omezený zaručí, že není typem hodnoty.</span><span class="sxs-lookup"><span data-stu-id="44ce9-105">The `as` operator is only allowed with reference types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="44ce9-106">Lze vyvarovat této chyby, použijte omezení typu třída nebo omezení typu odkazu.</span><span class="sxs-lookup"><span data-stu-id="44ce9-106">To avoid this error, use a class type constraint or a reference type constraint.</span></span>  
  
 <span data-ttu-id="44ce9-107">Je to proto, `as` operátor může vrátit `null`, což není možná hodnota pro typ hodnoty, a parametr typu musí zacházet jako typ hodnoty, pokud se omezení typu Třída ani omezení typu odkazu.</span><span class="sxs-lookup"><span data-stu-id="44ce9-107">This is because the `as` operator could return `null`, which is not a possible value for a value type, and the type parameter must be treated as a value type unless it is a class type constraint or a reference type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="44ce9-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="44ce9-108">Example</span></span>  
 <span data-ttu-id="44ce9-109">Následující ukázka generuje CS0413.</span><span class="sxs-lookup"><span data-stu-id="44ce9-109">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
