# <a name="compiler-error-cs0188"></a>Chyba kompilátoru CS0188

"Tento" objekt nejde použít, dokud všechna jeho pole Přiřazeno  
  
 Všechna pole v `struct` mají být přiděleny pomocí konstruktoru, než konstruktor může volat metodu v `struct`.  
  
 Pokud tato chyba se zobrazí při pokusu o inicializaci vlastností v konstruktoru strukturu, je řešení změňte parametr konstruktoru určit místo samotné vlastnosti pole zálohování. Automaticky implementované vlastnosti, mělo by se vyhnout ve strukturách, protože nemají žádné pole zálohování a proto jej nelze inicializovat jakýmkoli způsobem z konstruktoru.  
  
 Další informace najdete v tématu [pomocí struktury](../../../csharp/programming-guide/classes-and-structs/using-structs.md).  
  
## <a name="example"></a>Příklad

 Následující ukázka generuje CS0188:  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
