## <a name="example"></a><span data-ttu-id="9d331-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="9d331-108">Example</span></span>

 <span data-ttu-id="9d331-109">Následující ukázka generuje CS0188:</span><span class="sxs-lookup"><span data-stu-id="9d331-109">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
