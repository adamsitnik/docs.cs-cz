 <span data-ttu-id="bca2c-104">Všechna pole v `struct` musí být přiřazena konstruktorem před tím, než konstruktor může volat metodu `struct`v.</span><span class="sxs-lookup"><span data-stu-id="bca2c-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="bca2c-105">Pokud se zobrazí tato chyba při pokusu o inicializaci vlastnosti v konstruktoru struktury, je třeba změnit parametr konstruktoru tak, aby místo samotné vlastnosti určila pole pro zálohování.</span><span class="sxs-lookup"><span data-stu-id="bca2c-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="bca2c-106">Automaticky implementované vlastnosti by se měly vyhnout ve strukturách, protože nemají žádné zálohovací pole, a proto je nelze z konstruktoru inicializovat jakýmkoli způsobem.</span><span class="sxs-lookup"><span data-stu-id="bca2c-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>

## <a name="example"></a><span data-ttu-id="bca2c-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="bca2c-108">Example</span></span>

 <span data-ttu-id="bca2c-109">Následující ukázka generuje CS0188:</span><span class="sxs-lookup"><span data-stu-id="bca2c-109">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
