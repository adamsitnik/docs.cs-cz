# <a name="compiler-error-cs0134"></a>Chyba kompilátoru CS0134
'variable' je typu 'type'. Pole const odkazového typu než string lze inicializovat pouze s hodnotou null.  
  
 Konstantní výraz je výraz, který může být plně vyhodnocen v době kompilace. Jelikož jediný způsob, jak vytvořit hodnotu než null typu odkazu, je použít operátor new a operátor new není povolen v konstantní výraz, jediná možná hodnota pro konstanty odkazových typů než řetězec má hodnotu null.  
  
 Pokud jste k této chybě dojde při pokusu o vytvoření [const](../../../csharp/language-reference/keywords/const.md) pole řetězců, řešení je, aby pole [jen pro čtení](../../../csharp/language-reference/keywords/readonly.md)a inicializace v konstruktoru.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0134.  
  
```csharp  
// CS0134.cs  
// compile with: /target:library  
class MyTest {}   
  
class MyClass  
{  
   const MyTest test = new MyTest();   // CS0134  
  
   //OK  
   const MyTest test2 = null;  
   const System.String test3 = "test";  
}  
```
