## <a name="example"></a><span data-ttu-id="b1c4e-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="b1c4e-109">Example</span></span>

<span data-ttu-id="b1c4e-110">Následující kód generuje CS1942:</span><span class="sxs-lookup"><span data-stu-id="b1c4e-110">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="b1c4e-111">CS1942 se vztahuje k CS1949, protože použití `var` s proměnnou rozsahu způsobí selhání základní `Cast<T>` operace, protože `var` se nejedná o typ.</span><span class="sxs-lookup"><span data-stu-id="b1c4e-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
