# <a name="compiler-warning-level-4-cs1610"></a>Upozornění kompilátoru (úroveň 4) CS1610
Nepovedlo se odstranit dočasný soubor 'file' používá pro výchozí prostředek Win32--prostředků  
  
 Při použití [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md) – možnost kompilátoru a kdy vaší **% TEMP %** adresář nemá oprávnění odstranit, toto upozornění signalizuje, že kompilátor nemůže odstranit dočasný soubor, který je vytvořen.  
  
 Ujistěte se, že máte oprávnění ke čtení/zápis/delete **% TEMP %** adresáře.  
  
 V případě potřeby můžete tyto soubory ručně odstranit a budou mít žádný vliv na C# nebo libovolného z programů.
