# <a name="compiler-error-cs1112"></a>Chyba kompilátoru CS1112
Nepoužívejte "System.Runtime.CompilerServices.ExtensionAttribute". Místo toho použijte klíčové slovo 'this'.  
  
 Tato chyba je generována při <xref:System.Runtime.CompilerServices.ExtensionAttribute> se používá na nestatické třídy, která obsahuje metody rozšíření. Pokud tento atribut je použit ve statické třídě, jiné chyby, jako je například je CS0708: Může dojít k "Nelze deklarovat členy instance ve statické třídě".  
  
 V C#metody rozšíření musí být definován ve statické třídě a první parametr metody se mění `this` – klíčové slovo. Nepoužívejte atribut vůbec ve zdrojovém kódu. Další informace najdete v tématu [rozšiřující metody](../../../csharp/programming-guide/classes-and-structs/extension-methods.md).  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte atribut a použít `this` Modifikátor pro první parametr metody.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS1112:  
  
```csharp  
// cs1112.cs  
[System.Runtime.CompilerServices.ExtensionAttribute] // CS1112  
public class Extensions  
{  
    public bool A(bool b) { return b; }  
}  
  
class A { }   
```
