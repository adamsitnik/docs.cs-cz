 <span data-ttu-id="a8013-104">[Příkaz using](../../../csharp/language-reference/keywords/using-statement.md) je určena pro použití k zajištění integrity objekt na konci `using` blokovat se proto mohou používat pouze typy, které jsou na jedno použití takových příkazu.</span><span class="sxs-lookup"><span data-stu-id="a8013-104">The [using Statement](../../../csharp/language-reference/keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="a8013-105">Například nejsou uvolnitelné typy hodnot a parametrů typu, které nejsou omezeny na třídy nemusí být předpokládá se, že na jedno použití.</span><span class="sxs-lookup"><span data-stu-id="a8013-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a8013-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="a8013-106">Example</span></span>

## <a name="example"></a><span data-ttu-id="a8013-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="a8013-108">Example</span></span>  
 <span data-ttu-id="a8013-109">Následující ukázka generuje CS1674.</span><span class="sxs-lookup"><span data-stu-id="a8013-109">The following sample generates CS1674.</span></span>  
  
```csharp

using System;

   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}

   void IDisposable.Dispose() {}  
   public void Dispose() {}

      using (D d = new D()) {}  
   }

```
