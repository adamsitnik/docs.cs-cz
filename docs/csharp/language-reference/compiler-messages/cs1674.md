## <a name="example"></a><span data-ttu-id="fa3e9-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="fa3e9-108">Example</span></span>  
 <span data-ttu-id="fa3e9-109">Následující ukázka generuje CS1674.</span><span class="sxs-lookup"><span data-stu-id="fa3e9-109">The following sample generates CS1674.</span></span>  
  
```csharp

   public void Test() {  
      using (C c = new C()) {}   // CS1674

   void IDisposable.Dispose() {}

```
