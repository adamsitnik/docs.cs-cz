# <a name="compiler-error-cs0233"></a>Chyba kompilátoru CS0233
identifikátor nemá předdefinovanou velikost, takže sizeof se dá použít jenom v nezabezpečeném kontextu.
  
 Bez [](../keywords/unsafe.md) nebezpečného kontextu lze operátor [sizeof](../../../csharp/language-reference/operators/sizeof.md) použít pouze pro typy, jejichž velikost je konstanta v čase kompilace. Pokud se tato chyba zobrazuje, použijte nezabezpečený kontext.
  
Následující příklad generuje CS0233:
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
