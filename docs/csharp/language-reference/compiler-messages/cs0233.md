# <a name="compiler-error-cs0233"></a><span data-ttu-id="995e4-102">Chyba kompilátoru CS0233</span><span class="sxs-lookup"><span data-stu-id="995e4-102">Compiler Error CS0233</span></span>
<span data-ttu-id="995e4-103">identifikátor nemá předdefinovanou velikost, takže sizeof se dá použít jenom v nezabezpečeném kontextu.</span><span class="sxs-lookup"><span data-stu-id="995e4-103">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context</span></span>
  
 <span data-ttu-id="995e4-104">Bez [](../keywords/unsafe.md) nebezpečného kontextu lze operátor [sizeof](../operators/sizeof.md) použít pouze pro typy, jejichž velikost je konstanta v čase kompilace.</span><span class="sxs-lookup"><span data-stu-id="995e4-104">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="995e4-105">Pokud se tato chyba zobrazuje, použijte nezabezpečený kontext.</span><span class="sxs-lookup"><span data-stu-id="995e4-105">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="995e4-106">Následující příklad generuje CS0233:</span><span class="sxs-lookup"><span data-stu-id="995e4-106">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
