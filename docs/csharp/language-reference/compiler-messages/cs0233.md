# <a name="compiler-error-cs0233"></a>Chyba kompilátoru CS0233
'identifier' nemá předdefinovanou velikost, proto sizeof jde použít jenom v kontextu unsafe (zvažte možnost použití operátoru System.Runtime.InteropServices.Marshal.SizeOf)  
  
 [Sizeof](../../../csharp/language-reference/keywords/sizeof.md) operátor jde použít jenom pro typy, které jsou konstanty z doby kompilace. Pokud se zobrazuje tato chyba, ujistěte se, že velikost identifikátor můžete určit v době kompilace. Pokud nemůžete, pak použít <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A> místo `sizeof`.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0233:  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```
