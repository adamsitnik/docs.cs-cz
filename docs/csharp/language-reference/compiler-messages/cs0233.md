# <a name="compiler-error-cs0233"></a><span data-ttu-id="afad9-102">Chyba kompilátoru CS0233</span><span class="sxs-lookup"><span data-stu-id="afad9-102">Compiler Error CS0233</span></span>

 <span data-ttu-id="afad9-104">Bez [](../keywords/unsafe.md) nebezpečného kontextu lze operátor [sizeof](../../../csharp/language-reference/operators/sizeof.md) použít pouze pro typy, jejichž velikost je konstanta v čase kompilace.</span><span class="sxs-lookup"><span data-stu-id="afad9-104">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../../../csharp/language-reference/operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="afad9-105">Pokud se tato chyba zobrazuje, použijte nezabezpečený kontext.</span><span class="sxs-lookup"><span data-stu-id="afad9-105">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="afad9-106">Následující příklad generuje CS0233:</span><span class="sxs-lookup"><span data-stu-id="afad9-106">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
