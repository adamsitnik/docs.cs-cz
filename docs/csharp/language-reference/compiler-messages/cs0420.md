# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="ec013-102">Upozornění kompilátoru (úroveň 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="ec013-102">Compiler Warning (level 1) CS0420</span></span>
<span data-ttu-id="ec013-103">' identifier ': odkaz na pole s modifikátorem volatile nebude zpracován jako volatile</span><span class="sxs-lookup"><span data-stu-id="ec013-103">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="ec013-104">Pole s modifikátorem volatile by se obvykle nemělo `ref` předávat pomocí parametru nebo, protože se v rámci rozsahu funkce nepovažuje za volatili.</span><span class="sxs-lookup"><span data-stu-id="ec013-104">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="ec013-105">Existují výjimky, například při volání propojeného rozhraní API.</span><span class="sxs-lookup"><span data-stu-id="ec013-105">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="ec013-106">Stejně jako u všech upozornění můžete použít [upozornění #pragma](../preprocessor-directives/preprocessor-pragma-warning.md) k zakázání tohoto upozornění ve výjimečných případech, kdy jste záměrně použili pole typu volatile jako referenční parametr.</span><span class="sxs-lookup"><span data-stu-id="ec013-106">As with any warning, you may use the [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="ec013-107">Následující ukázka generuje CS0420:</span><span class="sxs-lookup"><span data-stu-id="ec013-107">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420   
   }  
}  
```
