# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="c9e0c-102">Upozornění kompilátoru (úroveň 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="c9e0c-102">Compiler Warning (level 1) CS0420</span></span>
<span data-ttu-id="c9e0c-103">'identifier': odkaz na pole s modifikátorem volatile se nezpracuje jako volatile</span><span class="sxs-lookup"><span data-stu-id="c9e0c-103">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="c9e0c-104">Pole s modifikátorem volatile by neměl být předávaná pomocí `ref` nebo **si** parametr, protože se nezpracuje jako volatile v rámci oboru funkce.</span><span class="sxs-lookup"><span data-stu-id="c9e0c-104">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="c9e0c-105">Existují výjimky, například při volání propojeného API.</span><span class="sxs-lookup"><span data-stu-id="c9e0c-105">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="c9e0c-106">Jako u jakékoli upozornění, můžete použít [varování #pragma](../../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) zakázat toto upozornění v těchto výjimečných případech, kdy úmyslně použijete pole s modifikátorem volatile jako odkaz na parametr.</span><span class="sxs-lookup"><span data-stu-id="c9e0c-106">As with any warning, you may use the [#pragma warning](../../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="c9e0c-107">Následující ukázka generuje CS0420:</span><span class="sxs-lookup"><span data-stu-id="c9e0c-107">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420   
   }  
}  
```
