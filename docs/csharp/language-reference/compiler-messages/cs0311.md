## <a name="example"></a><span data-ttu-id="afbcb-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="afbcb-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }

{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}
```  
