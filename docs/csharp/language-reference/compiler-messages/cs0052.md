# <a name="compiler-error-cs0052"></a><span data-ttu-id="fa8d2-102">Chyba kompilátoru CS0052</span><span class="sxs-lookup"><span data-stu-id="fa8d2-102">Compiler Error CS0052</span></span>

<span data-ttu-id="fa8d2-103">Nekonzistentní dostupnost: typ pole 'type' je méně dostupný než pole.</span><span class="sxs-lookup"><span data-stu-id="fa8d2-103">Inconsistent accessibility: field type 'type' is less accessible than field 'field'</span></span>  
  
 <span data-ttu-id="fa8d2-104">Typ pole nemůže být míň dostupný než pole samotného, protože všechny veřejné konstruktory musí vrátit objekt veřejně přístupná.</span><span class="sxs-lookup"><span data-stu-id="fa8d2-104">The type of a field cannot be less accessible than the field itself because all public constructs must return a publicly accessible object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fa8d2-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="fa8d2-105">Example</span></span>

 <span data-ttu-id="fa8d2-106">Následující ukázka generuje CS0052:</span><span class="sxs-lookup"><span data-stu-id="fa8d2-106">The following sample generates CS0052:</span></span>  

```csharp
// CS0052.cs  
public class MyClass2  
{  
    // The following line causes an error because the field, M, is declared
    // as public, but the type, MyClass, is private. Therefore the type is
    // less accessible than the field.  
    public MyClass M;   // CS0052  
  
    private class MyClass  
    {  
    }  
    // One way to resolve the error is to change the accessibility of the type
    // to public.
    //public class MyClass
    // Another solution is to change the accessibility of the field to private.
    // private MyClass M;
}  
  
    public class MainClass  
    {  
        public static void Main()  
        {  
        }  
    }  
```
