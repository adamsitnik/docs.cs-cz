# <a name="compiler-error-cs0234"></a>Chyba kompilátoru CS0234
Typ nebo obor názvů názvu 'název' neexistuje v oboru názvů atribut namespace. (nechybí odkaz na sestavení?)  
  
 Byl očekáván typ. Možné příčiny této chyby patří:  
  
- Neexistuje odkaz sestavení, který obsahuje definici typu v kompilaci. použít [– reference (Importovat Metadata)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md) pro zadání sestavení  
  
- Předaný název proměnné, do [typeof](../../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#typeof-operator) operátor.  
  
- Pokusili jste se referenční sestavení, které není součástí cílové profilu rozhraní .NET Framework. Další informace najdete v tématu [řešení chyb cílí na .NET Framework](/visualstudio/msbuild/troubleshooting-dotnet-framework-targeting-errors).  
  
 Pokud tato chyba zobrazí po přesunutí kódu z jednoho vývojového počítače do druhého, ujistěte se, že projekt na nový počítač má správné odkazy. proto, že verze sestavení jsou stejné jako na původní počítač. Prohlížeč objektů lze také použít ke kontrole sestavení a ověřte, zda obsahuje typy, které očekáváte, že bude obsahovat.  
  
 Následující ukázka generuje CS0234:  
  
```csharp  
// CS0234.cs  
public class C  
{  
   public static void Main()  
   {  
      System.DateTime x = new System.DateTim();   // CS0234  
      // try the following line instead  
      // System.DateTime x = new System.DateTime();  
   }  
}  
```
