# <a name="compiler-error-cs0507"></a><span data-ttu-id="80272-102">Chyba kompilátoru CS0507</span><span class="sxs-lookup"><span data-stu-id="80272-102">Compiler Error CS0507</span></span>
<span data-ttu-id="80272-103">'function1': Při přepisu "přístup" zděděného člena function2 nelze měnit modifikátory přístupu</span><span class="sxs-lookup"><span data-stu-id="80272-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="80272-104">Došlo k pokusu o změnu specifikace přístupu v přepsání metody.</span><span class="sxs-lookup"><span data-stu-id="80272-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="80272-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="80272-105">Example</span></span>  
 <span data-ttu-id="80272-106">Následující ukázka generuje CS0507.</span><span class="sxs-lookup"><span data-stu-id="80272-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="80272-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="80272-107">Example</span></span>  
 <span data-ttu-id="80272-108">CS0507 může také dojít, pokud třída pokusí o přepsání metody označené jako `protected internal` definované v metadatech odkazovaný.</span><span class="sxs-lookup"><span data-stu-id="80272-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="80272-109">V takovém případě by měl být označeny přepsání metody `protected`.</span><span class="sxs-lookup"><span data-stu-id="80272-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs

abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="80272-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="80272-110">Example</span></span>  
 <span data-ttu-id="80272-111">Následující ukázka generuje CS0507.</span><span class="sxs-lookup"><span data-stu-id="80272-111">The following sample generates CS0507.</span></span>
