# <a name="compiler-error-cs0071"></a>Chyba kompilátoru CS0071
Explicitní implementace rozhraní události musí používat syntaxi přistupujícího objektu události  
  
 Při implementaci explicitně [události](../../../csharp/language-reference/keywords/event.md) , který byl deklarován v rozhraní, je nutné použít ručně zadejte `add` a `remove` přístupových objektů událostí, které jsou obvykle poskytován kompilátorem. Kód přístupového kódu můžete připojit události rozhraní k jiné událost ve své třídě (viz dále v tomto tématu) nebo na vlastní typ delegáta. Další informace najdete v tématu [jak:  Implementace událostí rozhraní](../../../csharp/programming-guide/events/how-to-implement-interface-events.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0071.  
  
```csharp  
// CS0071.cs  
public delegate void MyEvent(object sender);  
  
interface ITest  
{  
    event MyEvent Clicked;  
}  
  
class Test : Itest  
{  
    event MyEvent ITest.Clicked;  // CS0071  
  
    // try the following code instead  
/*  
private MyEvent clicked;  
  
    event MyEvent Itest.Clicked  
    {  
        add  
        {  
            clicked += value;  
        }  
        remove  
        {  
            clicked -= value;  
        }  
    }  
*/  
    public static void Main() { }  
}  
```
