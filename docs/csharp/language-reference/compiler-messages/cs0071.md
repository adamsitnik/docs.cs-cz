# <a name="compiler-error-cs0071"></a>Chyba kompilátoru CS0071
Explicitní implementace rozhraní události musí používat syntaxi přistupujícího objektu události.  
  
 Při explicitní implementaci [události](../keywords/event.md) , která byla deklarována v rozhraní, je nutné použít ruční zadání `add` přístupových objektů události a `remove` , které jsou obvykle poskytovány kompilátorem. Přístupový kód může připojit událost rozhraní k jiné události ve třídě (viz dále v tomto tématu) nebo k vlastnímu typu delegáta. Další informace najdete v tématu [jak:  Implementujte události](../../programming-guide/events/how-to-implement-interface-events.md)rozhraní.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0071.  
  
```csharp  
// CS0071.cs  
public delegate void MyEvent(object sender);  
  
interface ITest  
{  
    event MyEvent Clicked;  
}  
  
class Test : Itest  
{  
    event MyEvent ITest.Clicked;  // CS0071  
  
    // try the following code instead  
/*  
private MyEvent clicked;  
  
    event MyEvent Itest.Clicked  
    {  
        add  
        {  
            clicked += value;  
        }  
        remove  
        {  
            clicked -= value;  
        }  
    }  
*/  
    public static void Main() { }  
}  
```
