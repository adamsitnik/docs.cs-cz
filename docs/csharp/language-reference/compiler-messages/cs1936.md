<span data-ttu-id="ded15-105">Aby bylo možné dotazovat zdrojového typu, který typ musí implementovat standardní metody operátoru dotazu, které jsou volání v dotazu.</span><span class="sxs-lookup"><span data-stu-id="ded15-105">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="ded15-106">Implementace může být buď v podobě členy třídy nebo metody rozšíření, které jsou přeneseny do rozsahu příslušnou `using` směrnice.</span><span class="sxs-lookup"><span data-stu-id="ded15-106">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>

## <a name="example"></a><span data-ttu-id="ded15-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="ded15-110">Example</span></span>

<span data-ttu-id="ded15-111">Následující příklad vytvoří CS1936:</span><span class="sxs-lookup"><span data-stu-id="ded15-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="ded15-112">K této chybě obvykle dochází, když se omylem pokusíte dotazovat objektu nějakého typu místo kolekce těchto objektů.</span><span class="sxs-lookup"><span data-stu-id="ded15-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
