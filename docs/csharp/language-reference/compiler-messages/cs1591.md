# <a name="compiler-warning-level-4-cs1591"></a>Upozornění kompilátoru (úroveň 4) CS1591
Komentář XML pro veřejně viditelný typ nebo člen "typ nebo člen.  
  
 [/Doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) byla zadána možnost kompilátoru, ale jeden nebo více konstrukce neměla komentáře.  
  
 Následující ukázka generuje CS1591:  
  
```csharp  
// CS1591.cs  
// compile with: /W:4 /doc:x.xml  
  
/// text  
public class Test  
{  
   // /// text  
   public static void Main()   // CS1591, remove "//" from previous line  
   {  
   }  
}  
```
