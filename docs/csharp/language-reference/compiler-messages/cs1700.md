# <a name="compiler-warning-level-3-cs1700"></a>Upozornění kompilátoru (úroveň 3) CS1700
Odkaz na sestavení název je neplatný a nelze ho přeložit  
  
 Toto upozornění signalizuje, že atribut, například <xref:System.Runtime.CompilerServices.InternalsVisibleToAttribute>, nebyl správně zadán.  
  
 Další informace najdete v tématu [přátelských sestavení](../../../standard/assembly/friend-assemblies.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1700.  
  
```csharp  
// CS1700.cs  
// compile with: /target:library  
using System.Runtime.CompilerServices;  
[assembly:InternalsVisibleTo("app2, Retargetable=f")]   // CS1700  
[assembly:InternalsVisibleTo("app2")]   // OK  
```
