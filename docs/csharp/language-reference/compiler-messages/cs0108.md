# <a name="compiler-warning-level-2-cs0108"></a>Upozornění kompilátoru (úroveň 2) CS0108

'member1' skrývá zděděný člen 'členu member2'. Použití new – klíčové slovo je-li skrytí úmyslné.  
  
 Proměnná byla deklarovaná se stejným názvem jako proměnná v základní třídě. Ale [nové](../../../csharp/language-reference/keywords/new-modifier.md) – klíčové slovo se nepoužil. Toto upozornění vás informuje, že by měl používat **nové**; proměnná je deklarovaná, jako kdyby **nové** byly použity v deklaraci.  
  
 Následující ukázka generuje CS0108:  

```csharp
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
