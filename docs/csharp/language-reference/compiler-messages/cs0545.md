# <a name="compiler-error-cs0545"></a>Chyba kompilátoru CS0545
'function': nelze přepsat, protože 'property' nemá přepsatelný přistupující objekt get  
  
 Byl proveden pokus definovat přepsání pro přistupující objekt vlastnosti při základní třída nemá žádná taková definice přepsání. Je-li vyřešit tuto chybu:  
  
- Přidání `set` přístupového objektu v základní třídě.  
  
- Odebírá `set` přistupující objekt z odvozené třídy.  
  
- Skrytí vlastnost základní třídy tak, že přidáte [nové](../../../csharp/language-reference/keywords/new.md) – klíčové slovo na vlastnost v odvozené třídě.  
  
- Provádění vlastnost základní třídy [virtuální](../../../csharp/language-reference/keywords/virtual.md).  
  
 Další informace najdete v tématu [pomocí vlastnosti](../../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0545.  
  
```csharp  
// CS0545.cs  
// compile with: /target:library  
// CS0545  
public class a  
{  
   public virtual int i  
   {  
      set {}  
  
      // Uncomment the following line to resolve.  
      // get { return 0; }  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      get { return 0; }  
      set {}   // OK  
   }  
}  
```
