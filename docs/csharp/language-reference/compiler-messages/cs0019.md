# <a name="compiler-error-cs0019"></a>Chyba kompilátoru CS0019

Operátor ' operator ' nelze použít na operandy typu ' type ' a ' type '

 Binární operátor je použit na datové typy, které ho nepodporují. [ &#124; ](../operators/boolean-logical-operators.md#conditional-logical-or-operator-) Například nemůžete použít operátor na řetězcích, nemůžete použít operátory [+](../operators/addition-operator.md), [-](../operators/subtraction-operator.md), [\<](../operators/comparison-operators.md#less-than-operator-)nebo [0](../operators/comparison-operators.md#greater-than-operator-) u proměnných [bool](../keywords/bool.md) a nemůžete použít [3](../operators/equality-operators.md#equality-operator-) . operátor s typem 4, pokud typ explicitně nepřetěžuje operátor.

 Operátor lze přetížit, aby podporoval operandy určitých typů. Další informace naleznete v tématu [přetížení operátoru](../operators/operator-overloading.md).

## <a name="example"></a>Příklad

 V následujícím příkladu je CS0019 vygenerováno na dvou místech, [](../keywords/bool.md) protože bool C# in není převoditelná na [int](../builtin-types/integral-numeric-types.md). CS0019 se také generuje při použití operátoru odčítání na řetězec. Operátor sčítání (+) lze použít s řetězcovými operandy, protože tento operátor je přetížen třídou `String` k provedení zřetězení řetězců.

```csharp
static void Main()
{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }
    
    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```

## <a name="example"></a>Příklad

 V následujícím příkladu je nutné zadat podmínkovou logiku mimo <xref:System.Diagnostics.ConditionalAttribute>. Do <xref:System.Diagnostics.ConditionalAttribute> můžete předat pouze jeden předdefinovaný symbol.

 Následující ukázka generuje CS0019:

```csharp
// CS0019_a.cs
// compile with: /target:library
using System.Diagnostics;

public class MyClass
{
   [ConditionalAttribute("DEBUG" || "TRACE")]   // CS0019
   public void TestMethod() {}

   // OK
   [ConditionalAttribute("DEBUG"), ConditionalAttribute("TRACE")]
   public void TestMethod2() {}
}
```
