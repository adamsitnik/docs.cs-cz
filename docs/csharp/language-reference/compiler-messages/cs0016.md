# <a name="compiler-error-cs0016"></a>Chyba kompilátoru CS0016

Nejde zapisovat do výstupního souboru ' file ' – důvod '

 Kompilátor nemůže zapisovat do výstupního souboru. Zkontrolujte cestu k souboru a ujistěte se, že existuje. Pokud už dříve sestavený soubor je v umístění, ujistěte se, že je zapisovatelný a že tento soubor v současné době nemá žádný proces zámek. Ujistěte se například, že váš spustitelný soubor není spuštěn při pokusu o sestavení.
