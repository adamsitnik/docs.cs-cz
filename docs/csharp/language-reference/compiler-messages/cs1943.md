## <a name="example"></a><span data-ttu-id="1d4e3-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="1d4e3-109">Example</span></span>  
 <span data-ttu-id="1d4e3-110">Následující kód vygeneruje CS1943:</span><span class="sxs-lookup"><span data-stu-id="1d4e3-110">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
