# <a name="compiler-error-cs0504"></a>Chyba kompilátoru CS0504
Konstanta 'variable' nemůže být označena jako statická  
  
 Pokud je proměnná [const](../../../csharp/language-reference/keywords/const.md), je také [statické](../../../csharp/language-reference/keywords/static.md). Chcete-li **const** a **statické** proměnnou, stačí deklarování této proměnné jako **const**; Chcete-li všechny je **statické** proměnné, stačí označit **statické**.  
  
 Následující ukázka generuje CS0504:  
  
```csharp  
// CS0504.cs  
namespace x  
{  
   abstract public class clx  
   {  
      static const int i = 0;   // CS0504, cannot be both static and const  
      abstract public void f();  
   }  
}  
```
