# <a name="compiler-error-cs0504"></a><span data-ttu-id="aff95-102">Chyba kompilátoru CS0504</span><span class="sxs-lookup"><span data-stu-id="aff95-102">Compiler Error CS0504</span></span>
<span data-ttu-id="aff95-103">Konstanta 'variable' nemůže být označena jako statická</span><span class="sxs-lookup"><span data-stu-id="aff95-103">The constant 'variable' cannot be marked static</span></span>  
  
 <span data-ttu-id="aff95-104">Pokud je proměnná [const](../../../csharp/language-reference/keywords/const.md), je také [statické](../../../csharp/language-reference/keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="aff95-104">If a variable is [const](../../../csharp/language-reference/keywords/const.md), it is also [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="aff95-105">Chcete-li **const** a **statické** proměnnou, stačí deklarování této proměnné jako **const**; Chcete-li všechny je **statické** proměnné, stačí označit **statické**.</span><span class="sxs-lookup"><span data-stu-id="aff95-105">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="aff95-106">Následující ukázka generuje CS0504:</span><span class="sxs-lookup"><span data-stu-id="aff95-106">The following sample generates CS0504:</span></span>  
  
```csharp  
// CS0504.cs  
namespace x  
{  
   abstract public class clx  
   {  
      static const int i = 0;   // CS0504, cannot be both static and const  
      abstract public void f();  
   }  
}  
```
