 <span data-ttu-id="4f3c9-104">Nová funkce v C# 2.0 je schopnost definovat pole v řádku uvnitř `struct`.</span><span class="sxs-lookup"><span data-stu-id="4f3c9-104">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="4f3c9-105">Takovými poli je přístupný jenom prostřednictvím lokálních proměnných nebo polí a nemusí být odkazováno jako zprostředkující hodnoty na levé straně výrazu.</span><span class="sxs-lookup"><span data-stu-id="4f3c9-105">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="4f3c9-106">Navíc pole přístupné pole, která jsou `static` nebo `readonly`.</span><span class="sxs-lookup"><span data-stu-id="4f3c9-106">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>

## <a name="example"></a><span data-ttu-id="4f3c9-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="4f3c9-109">Example</span></span>  
 <span data-ttu-id="4f3c9-110">Následující ukázka generuje CS1708.</span><span class="sxs-lookup"><span data-stu-id="4f3c9-110">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
