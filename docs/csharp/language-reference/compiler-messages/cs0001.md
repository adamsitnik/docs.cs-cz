# <a name="compiler-error-cs0001"></a>Chyba kompilátoru CS0001
Vnitřní chyba kompilátoru  
  
 Pokuste se zjistit, zda kompilátor nedaří kvůli neschopnost analyzovat neočekávaná syntaxe. Pokud tato chyba se zobrazí opakovaně, kontaktujte společnost Microsoft.
