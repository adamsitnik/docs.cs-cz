# <a name="compiler-error-cs1009"></a><span data-ttu-id="9896a-102">Chyba kompilátoru CS1009</span><span class="sxs-lookup"><span data-stu-id="9896a-102">Compiler Error CS1009</span></span>
<span data-ttu-id="9896a-103">Nerozpoznaná řídicí sekvence</span><span class="sxs-lookup"><span data-stu-id="9896a-103">Unrecognized escape sequence</span></span>  
  
 <span data-ttu-id="9896a-104">Neočekávaný znak následuje zpětné lomítko (\\) v [řetězci](../keywords/string.md).</span><span class="sxs-lookup"><span data-stu-id="9896a-104">An unexpected character follows a backslash (\\) in a [string](../keywords/string.md).</span></span> <span data-ttu-id="9896a-105">Kompilátor očekává jeden z platných řídicích znaků.</span><span class="sxs-lookup"><span data-stu-id="9896a-105">The compiler expects one of the valid escape characters.</span></span> <span data-ttu-id="9896a-106">Další informace naleznete v tématu [znakové řídicí znaky](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="9896a-106">For more information, see [Character Escapes](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span></span>  
  
 <span data-ttu-id="9896a-107">Následující ukázka generuje CS1009.</span><span class="sxs-lookup"><span data-stu-id="9896a-107">The following sample generates CS1009.</span></span>  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";     
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 <span data-ttu-id="9896a-108">Běžnou příčinou této chyby je použití znaku zpětného lomítka v názvu souboru, jak ukazuje následující příklad.</span><span class="sxs-lookup"><span data-stu-id="9896a-108">A common cause of this error is using the backslash character in a file name, as the following example shows.</span></span>  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 <span data-ttu-id="9896a-109">Chcete-li vyřešit tuto chybu,\\použijte "\\" @-quoted nebo řetězcový literál, jak ukazuje následující příklad.</span><span class="sxs-lookup"><span data-stu-id="9896a-109">To resolve this error, use "\\\\" or the @-quoted string literal, as the following example shows.</span></span>  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";     
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```
