# <a name="compiler-error-cs1009"></a>Chyba kompilátoru CS1009
Nerozpoznaná řídicí sekvence  
  
 Neočekávaný znak následuje zpětné lomítko (\\) v [řetězci](../keywords/string.md). Kompilátor očekává jeden z platných řídicích znaků. Další informace naleznete v tématu [znakové řídicí znaky](../../../standard/base-types/character-escapes-in-regular-expressions.md).  
  
 Následující ukázka generuje CS1009.  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";     
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 Běžnou příčinou této chyby je použití znaku zpětného lomítka v názvu souboru, jak ukazuje následující příklad.  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 Chcete-li vyřešit tuto chybu,\\použijte "\\" @-quoted nebo řetězcový literál, jak ukazuje následující příklad.  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";     
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```
