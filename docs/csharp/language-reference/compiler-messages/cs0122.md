## <a name="example"></a><span data-ttu-id="fd4cc-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="fd4cc-108">Example</span></span>

```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}

{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
