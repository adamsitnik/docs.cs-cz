# <a name="compiler-error-cs0122"></a>Chyba kompilátoru CS0122

člen není dostupný z důvodu úrovně ochrany.

 [Modifikátor přístupu](../keywords/modifiers.md) pro člena třídy brání přístupu ke členu. Další informace najdete v tématu [modifikátory přístupu](../../programming-guide/classes-and-structs/access-modifiers.md).

 Jedna příčina (není uvedená v ukázce níže) může vynechat příznak kompilátoru **/out** v cíli sestavení typu Friend. Další informace naleznete v tématu [Friend Assemblies](../../../standard/assembly/friend.md) and [/out (C# možnosti kompilátoru)](../compiler-options/out-compiler-option.md).

## <a name="example"></a>Příklad

 Následující ukázka generuje CS0122:

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
