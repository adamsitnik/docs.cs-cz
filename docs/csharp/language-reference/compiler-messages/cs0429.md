 <span data-ttu-id="870eb-104">Této chybě dochází vždy součástí výrazu v kódu nedostupný.</span><span class="sxs-lookup"><span data-stu-id="870eb-104">This error occurs whenever part of an expression in your code is unreachable.</span></span> <span data-ttu-id="870eb-105">V následujícím příkladu je podmínka `false && myTest()` splňuje tato kritéria, protože `myTest()` metoda nebude nikdy vyhodnocena vzhledem k tomu, že na levé straně `&&` operace má vždy hodnotu false.</span><span class="sxs-lookup"><span data-stu-id="870eb-105">In the following example, the condition `false && myTest()` meets this criteria because the `myTest()` method will never get evaluated due to the fact that the left side of the `&&` operation is always false.</span></span> <span data-ttu-id="870eb-106">Poté, co `&&` operátor vyhodnotí `false` příkaz hodnotu false, vyhodnocování se zastaví a nikdy vyhodnotí pravé straně.</span><span class="sxs-lookup"><span data-stu-id="870eb-106">As soon as the `&&` operator evaluates the `false` statement as false, it stops the evaluation, and will never evaluate the right side.</span></span>
