# <a name="compiler-error-cs0006"></a>Chyba kompilátoru CS0006
Soubor metadat ' DLL_name ' nebyl nalezen.  
  
 Program byl zkompilován a explicitně předaný název souboru, který obsahoval metadata; Knihovna DLL však nebyla nalezena. Další informace naleznete v tématu [/Reference (C# možnosti kompilátoru)](../compiler-options/reference-compiler-option.md).
