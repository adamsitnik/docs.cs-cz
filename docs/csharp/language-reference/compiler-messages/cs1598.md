# <a name="compiler-warning-level-1-cs1598"></a>Upozornění kompilátoru (úroveň 1) CS1598
Analyzátor XML nelze načíst z následujícího důvodu: ' důvod '. Soubor dokumentace XML "File" nebude vygenerován.  
  
 Byla zadána možnost [-doc](../compiler-options/doc-compiler-option.md) , ale kompilátor nemohl najít a načíst knihovnu Msxml3. dll. Přesvědčte se, zda je soubor Msxml3. dll nainstalován a zaregistrován.
