# <a name="compiler-warning-level-1-cs1598"></a>Upozornění kompilátoru (úroveň 1) CS1598
Analyzátor XML nemůže být zaveden z následujícího důvodu: "důvod". Soubor dokumentace XML 'file' nebude vygenerován.  
  
 [/Doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) byla zadána možnost, ale kompilátor nemůže najít a načíst msxml3.dll. Ujistěte se, že je soubor msxml3.dll instalaci a registraci.
