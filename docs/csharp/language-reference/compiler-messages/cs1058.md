# <a name="compiler-warning-level-1-cs1058"></a>Upozornění kompilátoru (úroveň 1) CS1058
Předchozí klauzule catch už zachytává všechny výjimky. Všechny výjimky vyvolané budou zabalené v System.Runtime.CompilerServices.RuntimeWrappedException  
  
 Tento atribut vygeneruje CS1058, pokud `catch()` nemá žádný zadaný typ výjimky po bloku `catch (System.Exception e)` bloku. Vás upozornění informuje o tom, `catch()` bloku nezachytí žádné výjimky.  
  
 A `catch()` blokovat po `catch (System.Exception e)` bloku může zachytit výjimky neodpovídající specifikaci CLS, pokud `RuntimeCompatibilityAttribute` je nastavena na hodnotu false v souboru AssemblyInfo.cs: `[assembly: RuntimeCompatibilityAttribute(WrapNonExceptionThrows = false)]`. Pokud tento atribut není explicitně nastavena na hodnotu false, všechny výjimky vyvolané neodpovídající specifikaci CLS jsou zabaleny jako výjimky a `catch (System.Exception e)` bloku je zachytí. Další informace najdete v tématu [jak: Catch – kompatibilní se Specifikací výjimky nekompatibilní](../../../csharp/programming-guide/exceptions/how-to-catch-a-non-cls-exception.md).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS1058.  
  
```csharp  
// CS1058.cs  
// CS1058 expected  
using System.Runtime.CompilerServices;  
  
// the following attribute is set to true by default in the C# compiler  
// set to false in your source code to resolve CS1058  
[assembly: RuntimeCompatibilityAttribute(WrapNonExceptionThrows = true)]  
  
class TestClass   
{  
   static void Main()   
   {  
      try {}  
  
      catch (System.Exception e) {   
         System. Console.WriteLine("Caught exception {0}", e);  
      }  
  
      catch {}   // CS1058. This line will never be reached.  
   }  
}  
```
