# <a name="compiler-error-cs0103"></a><span data-ttu-id="ad6e9-102">Chyba kompilátoru CS0103</span><span class="sxs-lookup"><span data-stu-id="ad6e9-102">Compiler Error CS0103</span></span>
<span data-ttu-id="ad6e9-103">Název 'identifier' v aktuálním kontextu neexistuje.</span><span class="sxs-lookup"><span data-stu-id="ad6e9-103">The name 'identifier' does not exist in the current context</span></span>  
  
 <span data-ttu-id="ad6e9-104">Použít název, který neexistuje ve třídě, byl proveden pokus o [obor názvů](../../../csharp/language-reference/keywords/namespace.md), nebo rozsah.</span><span class="sxs-lookup"><span data-stu-id="ad6e9-104">An attempt was made to use a name that does not exist in the class, [namespace](../../../csharp/language-reference/keywords/namespace.md), or scope.</span></span> <span data-ttu-id="ad6e9-105">Zkontrolujte, zda název a zkontrolovat vaše pomocí direktivy a odkazy na sestavení, abyste měli jistotu, že název, který se pokoušíte použít je k dispozici.</span><span class="sxs-lookup"><span data-stu-id="ad6e9-105">Check the spelling of the name and check your using directives and assembly references to make sure that the name that you are trying to use is available.</span></span>  
  
 <span data-ttu-id="ad6e9-106">K této chybě často dochází, pokud deklarujete proměnnou ve smyčce nebo `try` nebo `if` blokovat a pak se pokusíte k ní z z vnějšího bloku kódu nebo samostatného kódu bloku, jak je znázorněno v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="ad6e9-106">This error frequently occurs if you declare a variable in a loop or a `try` or `if` block and then attempt to access it from an enclosing code block or a separate code block, as shown in the following example.</span></span>  
  
```csharp  
using System;  
  
class MyClass1  
{  
    public static void Main()  
    {  
        try  
        {  
            // The following declaration is only available inside the try block.  
            MyClass1 conn = new MyClass1();  
        }  
        catch (Exception e)  
        {  
            // The following expression causes error CS0103, because variable  
            // conn only exists in the try block.  
            if (conn != null)   
                Console.WriteLine("{0}", e);  
        }  
    }  
}  
```  
  
 <span data-ttu-id="ad6e9-107">V následujícím příkladu řeší chyby.</span><span class="sxs-lookup"><span data-stu-id="ad6e9-107">The following example resolves the error.</span></span>  
  
```csharp  
using System;  
  
class MyClass2  
{  
    public static void Main()  
    {  
        // To resolve the error in the example, the first step is to   
        // move the declaration of conn out of the try block. The following  
        // declaration is available throughout the Main method.  
        MyClass2 conn = null;  
        try  
        {  
            // Inside the try block, use the conn variable that you declared  
            // previously.  
            conn = new MyClass2();  
        }  
        catch (Exception e)  
        {  
            // The following expression no longer causes an error, because  
            // the declaration of conn is in scope.  
            if (conn != null)   
                Console.WriteLine("{0}", e);  
        }  
    }  
}  
```
