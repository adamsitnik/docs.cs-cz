# <a name="compiler-error-cs1502"></a><span data-ttu-id="53eb7-102">Chyba kompilátoru CS1502</span><span class="sxs-lookup"><span data-stu-id="53eb7-102">Compiler Error CS1502</span></span>

 <span data-ttu-id="53eb7-104">Tato chyba nastane, pokud se předá metodě typy argumentů se neshodují s typy parametrů této metody.</span><span class="sxs-lookup"><span data-stu-id="53eb7-104">This error occurs when the argument types being passed to the method do not match the parameter types of that method.</span></span> <span data-ttu-id="53eb7-105">Pokud je přetížena volané metody, žádný z přetížené verze nemá podpis, který se shoduje s typy argumentů předaných.</span><span class="sxs-lookup"><span data-stu-id="53eb7-105">If the called method is overloaded, then none of the overloaded versions has a signature that matches the argument types being passed.</span></span>  
  
 <span data-ttu-id="53eb7-106">Chcete-li vyřešit tento problém, proveďte jednu z následujících akcí:</span><span class="sxs-lookup"><span data-stu-id="53eb7-106">To resolve this problem, do one of the following:</span></span>  
  
- <span data-ttu-id="53eb7-107">Zkontrolujte typy argumenty předané.</span><span class="sxs-lookup"><span data-stu-id="53eb7-107">Double-check the types of the arguments being passed.</span></span> <span data-ttu-id="53eb7-108">Ujistěte se, že budou odpovídat argumentů volané metody.</span><span class="sxs-lookup"><span data-stu-id="53eb7-108">Make sure that they match the arguments of the method being called.</span></span>

- <span data-ttu-id="53eb7-111">V případě potřeby, definujte další verze přetížené metody, která odpovídají typům parametrů, které jsou odesílány.</span><span class="sxs-lookup"><span data-stu-id="53eb7-111">If appropriate, define another overloaded version of the method to match the parameter types that are being sent.</span></span>

      }
