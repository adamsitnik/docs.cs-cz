# <a name="compiler-error-cs0163"></a>Chyba kompilátoru CS0163
Ovládací prvek nemůže být předáno z jednoho návěstí příkazu case (popisek) do jiného  
  
 Když [switch – příkaz](../../../csharp/language-reference/keywords/switch.md) obsahuje více než jeden přepínač oddíl, musí explicitně neukončíte každého oddílu, včetně poslední z nich, pomocí jedné z následujících klíčových slov:  
  
- [return](../../../csharp/language-reference/keywords/return.md)  
  
- [goto](../../../csharp/language-reference/keywords/goto.md)  
  
- [break](../../../csharp/language-reference/keywords/break.md)  
  
- [throw](../../../csharp/language-reference/keywords/throw.md)  
  
- [continue](../../../csharp/language-reference/keywords/continue.md)  
  
 Pokud chcete implementovat chování "předat" z jednoho oddílu na další, použijte `goto case #`. Další informace a příklady najdete v tématu [přepnout](../../../csharp/language-reference/keywords/switch.md).  
  
 Následující ukázka generuje CS0163.  
  
```csharp  
// CS0163.cs  
public class MyClass  
{  
    public static void Main()  
    {  
        int i = 0;  
  
        switch (i)   // CS0163  
        {  
            // Compiler error CS0163 is reported on the following line.  
            case 1:  
                i++;  
            // To resolve the error, uncomment one of the following example statements.  
            // return;  
            // break;  
            // goto case 3;  
  
            case 2:  
                i++;  
                return;  
  
            case 3:  
                i = 0;  
                return;  
  
            // Compiler error CS0163 is reported on the following line.  
            default:  
                Console.WriteLine("Default");  
                // To resolve the error, uncomment the following line:  
            //break;  
        }  
    }  
```
