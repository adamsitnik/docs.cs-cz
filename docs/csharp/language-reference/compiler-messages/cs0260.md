# <a name="compiler-error-cs0260"></a><span data-ttu-id="bef30-102">Chyba kompilátoru CS0260</span><span class="sxs-lookup"><span data-stu-id="bef30-102">Compiler Error CS0260</span></span>

<span data-ttu-id="bef30-103">Nebyl nalezen částečný modifikátor deklarace typu 'type'; existuje jiná částečná deklarace tohoto typu</span><span class="sxs-lookup"><span data-stu-id="bef30-103">Missing partial modifier on declaration of type 'type'; another partial declaration of this type exists</span></span>  
  
 <span data-ttu-id="bef30-104">Tato chyba označuje, že je deklarován více tříd, které mají stejný název.</span><span class="sxs-lookup"><span data-stu-id="bef30-104">This error indicates that you have declared multiple classes that have the same name.</span></span> <span data-ttu-id="bef30-105">Kromě toho na nejméně jedna, ale ne všechny deklarace obsahuje `partial` modifikátor.</span><span class="sxs-lookup"><span data-stu-id="bef30-105">In addition, at least one but not all of the declarations contains the `partial` modifier.</span></span> <span data-ttu-id="bef30-106">Pokud chcete definovat třídu v několika částí, každá část musíte deklarovat pomocí klíčového slova `partial`.</span><span class="sxs-lookup"><span data-stu-id="bef30-106">If you want to define a class in several parts, you must declare each part by using the keyword `partial`.</span></span>  
  
 <span data-ttu-id="bef30-107">K této chybě dojde, pokud deklaraci třídy a neúmyslně jí stejný název jako částečné třídy jinde deklarovaný ve stejném oboru názvů.</span><span class="sxs-lookup"><span data-stu-id="bef30-107">This error also occurs if you declare a class and accidentally give it the same name as a partial class that's declared elsewhere in the same namespace.</span></span>  
  
 <span data-ttu-id="bef30-108">Následující ukázka generuje CS0260:</span><span class="sxs-lookup"><span data-stu-id="bef30-108">The following sample generates CS0260:</span></span>  

```csharp
// CS0260.cs  
// You must mark both parts of the definition of class C
// by using the partial keyword.  
  
// The following line causes CS0260. To resolve the error, add  
// the 'partial' keyword to the declaration.  
class C
{  
}  
  
partial class C  
{  
}  
```
