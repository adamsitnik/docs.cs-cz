# <a name="compiler-error-cs1567"></a>Chyba kompilátoru CS1567
Chyba při generování prostředku Win32: 'file'  
  
 Kompilaci použita buď [/win32icon](../../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) – možnost kompilátoru nebo nepoužívá [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md), což způsobí, že kompilátor generuje soubor, který obsahuje informace o zdroji, ale nemohl kompilátor Vytvoření souboru z důvodu nedostatku místa na disku nebo některé jiné chyby.  
  
 Pokud nemůžete vyřešit problém při generování souboru, můžete použít [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md), který nevygeneruje soubor, který obsahuje informace o prostředku.
