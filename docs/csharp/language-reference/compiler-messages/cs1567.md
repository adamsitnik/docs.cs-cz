# <a name="compiler-error-cs1567"></a>Chyba kompilátoru CS1567
Chyba při generování prostředku Win32: ' file '  
  
 Vaše kompilace buď použila možnost kompilátoru [/win32icon](../compiler-options/win32icon-compiler-option.md) , nebo nepoužívala [parametr/win32res](../compiler-options/win32res-compiler-option.md), což způsobí, že kompilátor vygeneruje soubor, který obsahuje informace o prostředku, ale kompilátor nedokázal vytvořit soubor z důvodu nedostatku místa na disku. nebo došlo k nějaké chybě.  
  
 Pokud nemůžete vyřešit problém generace souborů, můžete použít [parametr/win32res](../compiler-options/win32res-compiler-option.md), který negeneruje soubor, který obsahuje informace o zdroji.
