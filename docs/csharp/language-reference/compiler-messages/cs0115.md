## <a name="example"></a><span data-ttu-id="2cefc-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="2cefc-106">Example</span></span>  
 <span data-ttu-id="2cefc-107">Následující ukázka generuje CS0115.</span><span class="sxs-lookup"><span data-stu-id="2cefc-107">The following sample generates CS0115.</span></span> <span data-ttu-id="2cefc-108">Je-li vyřešit CS0115 v jednom ze dvou způsobů:</span><span class="sxs-lookup"><span data-stu-id="2cefc-108">You can resolve CS0115 in one of two ways:</span></span>  
  
- <span data-ttu-id="2cefc-109">Odeberte `override` – klíčové slovo z metody v `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="2cefc-109">Remove the `override` keyword from the method in `MyClass2`.</span></span>  
  
- <span data-ttu-id="2cefc-110">Použití `MyClass1` jako základní třída pro `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="2cefc-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>  
  
```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
