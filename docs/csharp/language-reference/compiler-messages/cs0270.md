# <a name="compiler-error-cs0270"></a>Chyba kompilátoru CS0270
Velikost pole nelze zadat v deklaraci proměnné (zkuste inicializaci pomocí výrazu 'new')  
  
 K této chybě dochází, když je velikost zadaná jako součást deklarace pole. Chcete-li vyřešit, použijte [operátor new](../../../csharp/language-reference/keywords/new-operator.md) výrazu.  
  
 Následující příklad generuje CS0270:  
  
```csharp  
// CS0270.cs  
// compile with: /t:module  
  
public class Test  
{  
   int[10] a;   // CS0270  
   // To resolve, use the following line instead:  
   // int[] a = new int[10];  
}  
```
