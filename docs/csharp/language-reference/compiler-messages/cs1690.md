# <a name="compiler-warning-level-1-cs1690"></a>Upozornění kompilátoru (úroveň 1) CS1690
Přístup ke členovi na 'member' může způsobit běhovou výjimku. protože se jedná o pole třídy marshal-by-reference  
  
 K tomuto upozornění dochází při pokusu o volání metoda, vlastnost nebo indexer u členu třídy, která je odvozena z <xref:System.MarshalByRefObject>, a je člen typu hodnoty. Objekty, které dědí z `MarshalByRefObject` obvykle mají být zařazen podle odkazu napříč doménou aplikace. Pokud žádný kód někdy pokusí získat přímo přístup člena typu hodnota takového objektu napříč doménou aplikace, dojde k výjimce modulu runtime. Chcete-li vyřešit upozornění, nejprve zkopírovat člena do místní proměnné a volat metodu na tuto proměnnou.  
  
 Následující ukázka generuje CS1690:  
  
```csharp  
// CS1690.cs  
using System;  
  
class WarningCS1690: MarshalByRefObject  
{  
   int i = 5;  
  
   public static void Main()   
   {  
     WarningCS1690 e = new WarningCS1690();  
     e.i.ToString();   // CS1690  
  
     // OK  
     int i = e.i;  
     i.ToString();  
     e.i = i;  
   }  
}  
```
