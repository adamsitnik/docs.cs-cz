# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="149be-102">Upozornění kompilátoru (úroveň 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="149be-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="149be-103">Přístup ke členovi na členovi může způsobit výjimku za běhu, protože se jedná o pole třídy Marshal-by-reference.</span><span class="sxs-lookup"><span data-stu-id="149be-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="149be-104">K tomuto upozornění dochází při pokusu o volání metody, vlastnosti nebo indexeru na členu třídy, která je odvozena z <xref:System.MarshalByRefObject> a člen je typ hodnoty.</span><span class="sxs-lookup"><span data-stu-id="149be-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="149be-105">Objekty, které dědí z `MarshalByRefObject` jsou obvykle určeny k zařazení pomocí odkazu v doméně aplikace.</span><span class="sxs-lookup"><span data-stu-id="149be-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="149be-106">Pokud se nějaký kód někdy pokusí přímo získat přístup k členu typu hodnoty takového objektu napříč doménou aplikace, dojde k běhu <xref:System.InvalidOperationException>.</span><span class="sxs-lookup"><span data-stu-id="149be-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime <xref:System.InvalidOperationException> will occur.</span></span> <span data-ttu-id="149be-107">Chcete-li vyřešit upozornění, nejprve zkopírujte člena do místní proměnné a zavolejte metodu na tuto proměnnou.</span><span class="sxs-lookup"><span data-stu-id="149be-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>
  
 <span data-ttu-id="149be-108">Následující ukázka generuje CS1690:</span><span class="sxs-lookup"><span data-stu-id="149be-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;

class WarningCS1690 : MarshalByRefObject
{
    int i = 5;

    public static void Main()
    {            
        AppDomain domain = AppDomain.CreateDomain("MyDomain");                
        Type t = typeof(WarningCS1690);
        WarningCS1690 e = (WarningCS1690)domain.CreateInstanceAndUnwrap(t.Assembly.FullName,t.FullName);

        e.i.ToString(); // CS1690    

        // OK  
        int i = e.i;
        i.ToString();
        e.i = i;        
    }
}
```
