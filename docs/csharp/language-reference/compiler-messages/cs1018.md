# <a name="compiler-error-cs1018"></a><span data-ttu-id="77fff-102">Chyba kompilátoru CS1018</span><span class="sxs-lookup"><span data-stu-id="77fff-102">Compiler Error CS1018</span></span>

 <span data-ttu-id="77fff-104">Kompilátor narazil na neúplnou deklaraci konstruktoru.</span><span class="sxs-lookup"><span data-stu-id="77fff-104">The compiler encountered an incomplete constructor declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="77fff-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="77fff-105">Example</span></span>  
 <span data-ttu-id="77fff-106">Následující příklad generuje CS1018 a navrhne několik způsobů, jak vyřešit chybu:</span><span class="sxs-lookup"><span data-stu-id="77fff-106">The following example generates CS1018, and suggests several ways to resolve the error:</span></span>  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // possible resolutions:  
    // public a () resolves by removing the colon  
    // public a () : base() calls C's parameterless constructor  
    // public a () : this(1) calls the assignment constructor of class a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
