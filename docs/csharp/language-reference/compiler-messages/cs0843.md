# <a name="compiler-error-cs0843"></a>Chyba kompilátoru CS0843
Pomocné pole automaticky implementované proměnné 'name' musí být plně přiřazeno před vrácením řízení volajícímu. Zvažte možnost volání konstruktoru bez parametrů z inicializátoru konstruktoru.  
  
 Chcete-li přiřadit hodnotu pro automaticky implementovanou vlastnost z konstruktoru, je nutné nejprve vyvolat konstruktor k vytvoření objektu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Přidejte volání pro konstruktor bez parametrů v inicializátoru konstruktoru, jak je znázorněno v následujícím příkladu. Všimněte si použití `: this()`. Další informace najdete v tématu [to](../../../csharp/language-reference/keywords/this.md).  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje CS0843:  
  
```csharp  
// cs0843.cs  
struct S  
{  
    public int AIProp { get; set; }  
    public S(int i){} //CS0843  
    // Try the following lines instead.  
   // public S(int i) : this()  
   // {  
   //     AIProp = i;  
   // }  
}  
  
class Test  
{  
    static int Main()  
    {  
        return 1;  
    }  
}  
```
