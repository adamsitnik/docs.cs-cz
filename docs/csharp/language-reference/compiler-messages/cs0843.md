# <a name="compiler-error-cs0843"></a>Chyba kompilátoru CS0843
Pomocné pole automaticky implementované proměnné 'name' musí být plně přiřazeno před vrácením řízení volajícímu. Zvažte možnost volání výchozího konstruktoru z inicializátoru konstruktoru.  
  
 Přiřadí hodnotu pro automaticky implementovanou vlastnost z konstruktor, musí nejdřív vyvolání výchozího konstruktoru pro vytvoření objektu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. V inicializátoru konstruktoru, jak je znázorněno v následujícím příkladu přidáte volání výchozího konstruktoru. Všimněte si použití `: this()`. Další informace najdete v tématu [to](../../../csharp/language-reference/keywords/this.md).  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje CS0843:  
  
```csharp  
// cs0843.cs  
struct S  
{  
    public int AIProp { get; set; }  
    public S(int i){} //CS0843  
    // Try the following lines instead.  
   // public S(int i) : this()  
   // {  
   //     AIProp = i;  
   // }  
}  
  
class Test  
{  
    static int Main()  
    {  
        return 1;  
    }  
}  
```
