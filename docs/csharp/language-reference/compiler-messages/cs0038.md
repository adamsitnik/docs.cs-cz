 <span data-ttu-id="c1d96-104">Pole ve třídě není automaticky k dispozici pro vnořenou třídu.</span><span class="sxs-lookup"><span data-stu-id="c1d96-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="c1d96-105">K dispozici pro vnořenou třídu musí být pole [statické](../keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="c1d96-105">To be available to a nested class, the field must be [static](../keywords/static.md).</span></span> <span data-ttu-id="c1d96-106">V opačném případě je nutné vytvořit instanci vnější třídy.</span><span class="sxs-lookup"><span data-stu-id="c1d96-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="c1d96-107">Další informace naleznete v tématu [vnořené typy](../../programming-guide/classes-and-structs/nested-types.md).</span><span class="sxs-lookup"><span data-stu-id="c1d96-107">For more information, see [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="c1d96-108">Následující ukázka generuje CS0038:</span><span class="sxs-lookup"><span data-stu-id="c1d96-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;

      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
