# <a name="compiler-error-cs0038"></a>Chyba kompilátoru CS0038
Nelze přistupovat k nestatickému členu vnějšího typu 'type1' vnořeného typu 'type2'  
  
 Pole v třídě není automaticky dostupné pro vnořené třídy. Aby bylo dostupné pro vnořené třídy, musí být pole [statické](../../../csharp/language-reference/keywords/static.md). V opačném případě musí vytvořit instanci z vnější třídy. Další informace najdete v tématu [vnořené typy](../../../csharp/programming-guide/classes-and-structs/nested-types.md).  
  
 Následující ukázka generuje CS0038:  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
