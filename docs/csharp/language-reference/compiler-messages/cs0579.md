# <a name="compiler-error-cs0579"></a><span data-ttu-id="24582-102">Chyba kompilátoru CS0579</span><span class="sxs-lookup"><span data-stu-id="24582-102">Compiler Error CS0579</span></span>
<span data-ttu-id="24582-103">Duplicitní atribut' attribute '.</span><span class="sxs-lookup"><span data-stu-id="24582-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="24582-104">Není možné zadat stejný atribut více než jednou, pokud atribut určuje **AllowMultiple = true** v jeho [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md).</span><span class="sxs-lookup"><span data-stu-id="24582-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="24582-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="24582-105">Example</span></span>  
 <span data-ttu-id="24582-106">Následující příklad generuje CS0579.</span><span class="sxs-lookup"><span data-stu-id="24582-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
