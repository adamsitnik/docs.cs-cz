# <a name="compiler-warning-level-1-cs1691"></a>Upozornění kompilátoru (úroveň 1) CS1691
'number' není platné číslo upozornění  
  
 Číslo, které bylo předáno [varování #pragma](../../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) direktivy preprocesoru není platné číslo upozornění. Ověřte, že číslo představuje varování, není chybu nebo jiný posloupnost znaků.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS1691.  
  
```csharp  
// CS1691.cs  
public class C  
{  
    int i = 1;  
    public static void Main()  
    {  
        C myC = new C();  
#pragma warning disable 151  // CS1691  
// Try the following line instead:  
// #pragma warning disable 1645    
        myC.i++;  
#pragma warning restore 151  // CS1691  
// Try the following line instead:  
// #pragma warning restore 1645    
    }  
}  
```
