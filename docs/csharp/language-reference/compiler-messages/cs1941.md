## <a name="example"></a><span data-ttu-id="5b6cd-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="5b6cd-108">Example</span></span>

<span data-ttu-id="5b6cd-109">Následující kód vygeneruje CS1941, protože operátor `equals` je požádán o porovnání `int` s `string`.</span><span class="sxs-lookup"><span data-stu-id="5b6cd-109">The following code generates CS1941 because the `equals` operator is being asked to compare an `int` to a `string`.</span></span>  

```csharp
// cs1941.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        var nums = new[] { 1, 2, 3, 4, 5, 6 };  
        var words = new string[] { "lake", "mountain", "sky" };  
        IEnumerable e = from n in nums  
                        join w in words on n equals w // CS1941  
                        select w;  
        return 0;  
    }  
}  
```
