## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="0f7de-106">Příklad nejednoznačný převod</span><span class="sxs-lookup"><span data-stu-id="0f7de-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="0f7de-107">Této chybě může dojít, pokud neexistuje žádný převod nebo pokud je k dispozici implicitní převody za následek dvojznačný situaci.</span><span class="sxs-lookup"><span data-stu-id="0f7de-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="0f7de-108">Následující příklad generuje CS0151:</span><span class="sxs-lookup"><span data-stu-id="0f7de-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```
