## <a name="example"></a><span data-ttu-id="59bb3-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="59bb3-108">Example</span></span>

 <span data-ttu-id="59bb3-109">Následující ukázka generuje CS0592:</span><span class="sxs-lookup"><span data-stu-id="59bb3-109">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
