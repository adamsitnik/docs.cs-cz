# <a name="compiler-error-cs1019"></a><span data-ttu-id="a80eb-102">Chyba kompilátoru CS1019</span><span class="sxs-lookup"><span data-stu-id="a80eb-102">Compiler Error CS1019</span></span>
<span data-ttu-id="a80eb-103">Očekával se přetížený unární operátor.</span><span class="sxs-lookup"><span data-stu-id="a80eb-103">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="a80eb-104">Něco, co vypadá jako přetížený unární operátor, je deklarované, ale operátor chybí nebo je v signatuře v nesprávném umístění.</span><span class="sxs-lookup"><span data-stu-id="a80eb-104">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="a80eb-105">*Unární operátor* je operátor, který pracuje na jednom operandu.</span><span class="sxs-lookup"><span data-stu-id="a80eb-105">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="a80eb-106">Například `++` je unární operátor.</span><span class="sxs-lookup"><span data-stu-id="a80eb-106">For example, `++` is a unary operator.</span></span> <span data-ttu-id="a80eb-107">Můžete přetížit některé unární operátory pomocí `operator` klíčového slova a zadáním jednoho parametru typu, na kterém operátor pracuje.</span><span class="sxs-lookup"><span data-stu-id="a80eb-107">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="a80eb-108">Například pokud chcete přetížit operátor `++` pro uživatelsky definovanou třídu `Temperature` , abyste mohli napsat `Temperature++`, můžete ho definovat tímto způsobem:</span><span class="sxs-lookup"><span data-stu-id="a80eb-108">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="a80eb-109">Při obdržení této chyby jste deklarovali něco, co vypadá jako přetížený unární operátor s tím rozdílem, že samotný operátor chybí nebo je v signatuře v nesprávném umístění.</span><span class="sxs-lookup"><span data-stu-id="a80eb-109">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="a80eb-110">Pokud odeberete `++` z signatury v předchozím příkladu, vygenerujete CS1019.</span><span class="sxs-lookup"><span data-stu-id="a80eb-110">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="a80eb-111">Následující kód generuje CS1019:</span><span class="sxs-lookup"><span data-stu-id="a80eb-111">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)     
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;   
   }  
  
   public static void Main()  
   {  
   }  
}  
```
