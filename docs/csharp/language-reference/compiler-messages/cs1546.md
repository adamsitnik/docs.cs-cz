# <a name="compiler-error-cs1546"></a>Chyba kompilátoru CS1546
Vlastnost, indexer nebo událost 'property' není podporován jazyk; Zkuste přímo volat metodu přístupového objektu 'přistupujícího objektu.  
  
 Kód využívá objekt, který má výchozí indexovanou vlastnost a pokusili indexované syntaxí. Chcete-li vyřešit tuto chybu, volejte metodu přistupujícího objektu vlastnosti. Další informace o indexování a vlastnostech najdete v tématu [indexery](../../../csharp/programming-guide/indexers/index.md).  
  
 Následující ukázka generuje CS1546.  
  
## <a name="example"></a>Příklad  
 Tento vzorový kód obsahuje soubor .cpp, který se kompiluje soubor .dll, a .cs soubor, který používá tento .dll. Následující kód je soubor .dll a definuje vlastnost, která má být přistupováno z kódu v souboru .cs.  
  
```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```  
  
## <a name="example"></a>Příklad  
 Toto je C# souboru.  
  
```csharp  
// CS1546.cs  
// compile with: /r:CPP1546.dll   
using System;  
public class Test  
{  
    public static void Main()  
    {  
        int i = 0;  
        MCPP mcpp = new MCPP();  
        i = mcpp.Prop(1,1); // CS1546  
        // Try the following line instead:  
        // i = mcpp.get_Prop(1,1);  
    }  
}  
```
