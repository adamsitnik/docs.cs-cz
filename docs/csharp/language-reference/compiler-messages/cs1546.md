## <a name="example"></a><span data-ttu-id="22e8d-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="22e8d-108">Example</span></span>  
 <span data-ttu-id="22e8d-109">Tento vzorový kód obsahuje soubor .cpp, který se kompiluje soubor .dll, a .cs soubor, který používá tento .dll.</span><span class="sxs-lookup"><span data-stu-id="22e8d-109">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="22e8d-110">Následující kód je soubor .dll a definuje vlastnost, která má být přistupováno z kódu v souboru .cs.</span><span class="sxs-lookup"><span data-stu-id="22e8d-110">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp

// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
