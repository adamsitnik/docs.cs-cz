## <a name="example"></a><span data-ttu-id="b0d64-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="b0d64-106">Example</span></span>  
 <span data-ttu-id="b0d64-107">Následující příklad generuje CS0039.</span><span class="sxs-lookup"><span data-stu-id="b0d64-107">The following example generates CS0039.</span></span>  
  
```csharp  
// CS0039.cs  
using System;  
class A  
{  
}  
class B: A  
{  
}  
class C: A  
{  
}  
class M  
{  
    static void Main()  
    {  
        A a = new C();  
        B b = new B();  
        C c;  
  
        // This is valid; there is a built-in reference  
        // conversion from A to C.  
        c = a as C;    
  
        //The following generates CS0039; there is no  
        // built-in reference conversion from B to C.  
        c = b as C;  // CS0039  
    }  
}  
```
