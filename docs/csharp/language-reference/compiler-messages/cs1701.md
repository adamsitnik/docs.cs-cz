# <a name="compiler-warning-level-2-cs1701"></a>Upozornění kompilátoru (úroveň 2) CS1701
Za předpokladu, že odkaz na sestavení "Název sestavení #1" odpovídá "Název sestavení #2", budete muset zadat zásady pro běh  
  
 Dvě sestavení se liší číslem vydání nebo verze. Pro sjednocení pravděpodobnější musíte zadat direktivy v souboru .config aplikace a musíte poskytnout správný silný název sestavení, jak je ukázáno v následujícím příkladu kódu.  
  
## <a name="example"></a>Příklad  
 Následující ukázka odkazuje na sestavení pomocí dvou různých externích aliasů. Tento první příklad vytvoří starší verzi kódu, který se vytvoří sestavení CS1701_d.  
  
```csharp  
// CS1701_a.cs  
// compile with: /target:library /out:cs1701_d.dll /keyfile:mykey.snk  
using System.Reflection;  
[assembly:AssemblyVersion("1.0")]  
public class A {  
   public void M1() {}  
}  
  
public class C1 {}  
```  
  
## <a name="example"></a>Příklad  
 Toto je kód, který vytvoří na novější verzi sestavení CS1701_d. Všimněte si, že se zkompiluje do jiného adresáře než starší verzi, která je potřebná protože výstupní soubory mají stejné názvy.  
  
```csharp  
// CS1701_b.cs  
// compile with: /target:library /out:c:\\cs1701_d.dll /keyfile:mykey.snk  
using System.Reflection;  
[assembly:AssemblyVersion("2.0")]  
public class A {  
   public void M2() {}  
   public void M1() {}  
}  
  
public class C2 {}  
public class C1 {}  
```  
  
## <a name="example"></a>Příklad  
 Tato ukázka nastaví externích aliasů A1 a A2.  
  
```csharp  
// CS1701_c.cs  
// compile with: /target:library /reference:A2=c:\\cs1701_d.dll /reference:A1=cs1701_d.dll  
  
extern alias A1;  
extern alias A2;  
// using System;  
using a1 = A1::A;  
using a2 = A2::A;  
  
public class Ref {  
   public static a1 A1() { return new a1(); }  
   public static a2 A2() { return new a2(); }  
  
   public static A1::C1 M1() { return new A1::C1(); }  
   public static A2::C2 M2() { return new A2::C2(); }  
}  
```  
  
## <a name="example"></a>Příklad  
 Tato ukázka volá metody, pomocí dvou různých aliasů A. Následující ukázka generuje C1701.  
  
```csharp  
// CS1701_d.cs  
// compile with: /reference:c:\\CS1701_d.dll /reference:CS1701_c.dll  
// CS1701 expected  
class Tester {  
   public static void Main() {  
      Ref.A1().M1();  
      Ref.A2().M2();  
   }  
}  
```
