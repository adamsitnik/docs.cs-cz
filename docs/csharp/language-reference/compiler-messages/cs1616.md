# <a name="compiler-warning-level-1-cs1616"></a>Upozornění kompilátoru (úroveň 1) CS1616
' Možnost ' přepíše atribut 'attribute' předaný ve zdrojovém souboru nebo přidaném modulu  
  
 K tomuto upozornění dochází, pokud se atributy sestavení <xref:System.Reflection.AssemblyKeyFileAttribute> nebo <xref:System.Reflection.AssemblyKeyNameAttribute> součástí zdroje je v konfliktu s [/keyfile](../../../csharp/language-reference/compiler-options/keyfile-compiler-option.md) nebo [/keycontainer](../../../csharp/language-reference/compiler-options/keycontainer-compiler-option.md) název souboru příkazového řádku možnost nebo klíč nebo kontejneru klíčů určený ve vlastnostech projektu.  
  
 Níže uvedený příklad předpokládá, že jste soubor klíče s názvem `cs1616.snk`. Vygenerování tohoto souboru z příkazového řádku:  
  
```console  
sn -k CS1616.snk  
```  
  
 Následující ukázka generuje CS1616:  
  
```csharp  
// CS1616.cs  
// compile with: /keyfile:cs1616.snk  
using System.Reflection;  
  
// To fix the error, remove the next line  
[assembly: AssemblyKeyFile("cs1616b.snk")]  // CS1616  
  
class C  
{  
  public static void Main()  
  {  
  }  
}  
```
