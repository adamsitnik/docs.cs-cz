# <a name="compiler-warning-level-1-cs1616"></a>Upozornění kompilátoru (úroveň 1) CS1616
Možnost Option Přepisuje atribut Attribute zadaný ve zdrojovém souboru nebo přidaném modulu.  
  
 K tomuto upozornění dochází, pokud atributy <xref:System.Reflection.AssemblyKeyFileAttribute> sestavení <xref:System.Reflection.AssemblyKeyNameAttribute> nebo nalezené ve zdroji jsou v konfliktu s možností [/keyfile](../compiler-options/keyfile-compiler-option.md) nebo [/keycontainer](../compiler-options/keycontainer-compiler-option.md) příkazového řádku nebo s názvem souboru klíče nebo kontejnerem klíčů zadaným ve vlastnostech projektu.  
  
 V následujícím příkladu Předpokládejme, že máte soubor klíče s názvem `cs1616.snk`. Vygenerujte tento soubor pomocí příkazového řádku:  
  
```console  
sn -k CS1616.snk  
```  
  
 Následující ukázka generuje CS1616:  
  
```csharp  
// CS1616.cs  
// compile with: /keyfile:cs1616.snk  
using System.Reflection;  
  
// To fix the error, remove the next line  
[assembly: AssemblyKeyFile("cs1616b.snk")]  // CS1616  
  
class C  
{  
  public static void Main()  
  {  
  }  
}  
```
