# <a name="compiler-error-cs0445"></a><span data-ttu-id="391ff-102">Chyba kompilátoru CS0445</span><span class="sxs-lookup"><span data-stu-id="391ff-102">Compiler Error CS0445</span></span>
<span data-ttu-id="391ff-103">Nejde změnit výsledek unboxingového převodu.</span><span class="sxs-lookup"><span data-stu-id="391ff-103">Cannot modify the result of an unboxing conversion</span></span>  
  
 <span data-ttu-id="391ff-104">Výsledek unboxingového převodu je dočasná proměnná.</span><span class="sxs-lookup"><span data-stu-id="391ff-104">The result of an unboxing conversion is a temporary variable.</span></span> <span data-ttu-id="391ff-105">Kompilátor zabrání upravovat tyto proměnné, protože jakákoli změna by odchází při zmizet, dočasná proměnná.</span><span class="sxs-lookup"><span data-stu-id="391ff-105">The compiler prevents you from modifying such variables because any modification would go away when the temporary variable goes away.</span></span> <span data-ttu-id="391ff-106">Tento problém můžete deklarovat nový typ hodnoty proměnnou pro uložení zprostředkující výrazu a přiřadit výsledek převodu rozbalení pro danou proměnnou.</span><span class="sxs-lookup"><span data-stu-id="391ff-106">To fix this, declare a new value-type variable to store the intermediate expression, and assign the result of the unboxing conversion to that variable.</span></span>  
  
 <span data-ttu-id="391ff-107">Následující kód vygeneruje CS0455.</span><span class="sxs-lookup"><span data-stu-id="391ff-107">The following code generates CS0455.</span></span>  
  
```csharp  
// CS0445.CS  
class UnboxingTest  
{  
    public static void Main()  
    {  
        Point p;  
        p.x = 1;  
        p.y = 2;  
        object obj = p;  
        // The following line generates CS0445, because the result  
        // of unboxing obj is a temporary variable.  
        ((Point)obj).x = 2;  
  
        // The following lines resolve the error.  
  
        // Store the result of the unboxing conversion in p2.  
        Point p2;       
        p2 = (Point)obj;  
        // Then you can modify the unboxed value.  
        p2.x = 2;  
    }  
}  
  
struct Point  
{  
    public int x, y;  
}  
```
