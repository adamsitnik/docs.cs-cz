# <a name="returns-c-programming-guide"></a>\<Vrátí > (C# Programming Guide)
## <a name="syntax"></a>Syntaxe  
  
```xml  
<returns>description</returns>  
```  
  
#### <a name="parameters"></a>Parametry  
 `description`  
 Popis návratovou hodnotu.
