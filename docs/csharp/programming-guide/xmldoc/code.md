# <a name="ltcodegt-c-programming-guide"></a>&lt;kód&gt; (C# Programming Guide)
## <a name="syntax"></a>Syntaxe  
  
```xml  
<code>content</code>  
```  
  
#### <a name="parameters"></a>Parametry  
 `content`  
 Text, který je označený jako kód.  
  
## <a name="remarks"></a>Poznámky  
 \<Kód > značky poskytuje způsob, jak určit více řádků jako kód. Použití [ \<c >](../../../csharp/programming-guide/xmldoc/code-inline.md) k označení, že text v popisu musí být označené jako kód.  
  
 Kompilovat s [/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) pro zpracování dokumentačních komentářů do souboru.
