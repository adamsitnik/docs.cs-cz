<span data-ttu-id="84288-109">Použijete typ připouštějící hodnotu Null, když budete potřebovat k reprezentaci nedefinovaná hodnota základního typu.</span><span class="sxs-lookup"><span data-stu-id="84288-109">You use a nullable type when you need to represent the undefined value of an underlying type.</span></span> <span data-ttu-id="84288-110">Proměnné typu Boolean může mít jenom dvě hodnoty: true a false.</span><span class="sxs-lookup"><span data-stu-id="84288-110">A Boolean variable can have only two values: true and false.</span></span> <span data-ttu-id="84288-111">Neexistuje žádná hodnota "undefined".</span><span class="sxs-lookup"><span data-stu-id="84288-111">There is no "undefined" value.</span></span> <span data-ttu-id="84288-112">V mnoha programování aplikací zejména interakce databáze, hodnotu proměnné lze nedefinované nebo chybí.</span><span class="sxs-lookup"><span data-stu-id="84288-112">In many programming applications, most notably database interactions, a variable value can be undefined or missing.</span></span> <span data-ttu-id="84288-113">Například pole v databázi může obsahovat hodnoty true nebo false, nebo může obsahovat vůbec žádnou hodnotu.</span><span class="sxs-lookup"><span data-stu-id="84288-113">For example, a field in a database may contain the values true or false, or it may contain no value at all.</span></span> <span data-ttu-id="84288-114">Můžete použít `Nullable<bool>` v takovém případě zadejte.</span><span class="sxs-lookup"><span data-stu-id="84288-114">You use a `Nullable<bool>` type in that case.</span></span>

- <span data-ttu-id="84288-116">Typy s možnou hodnotou Null představují proměnné typu hodnoty, které je možné přiřadit `null` hodnotu.</span><span class="sxs-lookup"><span data-stu-id="84288-116">Nullable types represent value-type variables that can be assigned the `null` value.</span></span> <span data-ttu-id="84288-117">Nelze vytvořit typ připouštějící hodnotu Null na základě typu odkazu.</span><span class="sxs-lookup"><span data-stu-id="84288-117">You cannot create a nullable type based on a reference type.</span></span> <span data-ttu-id="84288-118">(Referenční typy již podporují `null` hodnota.)</span><span class="sxs-lookup"><span data-stu-id="84288-118">(Reference types already support the `null` value.)</span></span>  
  
- <span data-ttu-id="84288-119">Syntaxe `T?` představuje zkratku pro `Nullable<T>`.</span><span class="sxs-lookup"><span data-stu-id="84288-119">The syntax `T?` is shorthand for `Nullable<T>`.</span></span> <span data-ttu-id="84288-120">Dvě různými formami jsou zaměnitelné.</span><span class="sxs-lookup"><span data-stu-id="84288-120">The two forms are interchangeable.</span></span>

- <span data-ttu-id="84288-123">Použití <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> a <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType> vlastnosti jen pro čtení pro testování pro hodnotu null a načtení hodnoty, jak je znázorněno v následujícím příkladu: `if (x.HasValue) y = x.Value;`</span><span class="sxs-lookup"><span data-stu-id="84288-123">Use the <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> and <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType> readonly properties to test for null and retrieve the value, as shown in the following example: `if (x.HasValue) y = x.Value;`</span></span>  
  
  - <span data-ttu-id="84288-124"><xref:System.Nullable%601.HasValue%2A> Vrátí vlastnost `true` Pokud proměnná obsahuje hodnotu, nebo `false` jde `null`.</span><span class="sxs-lookup"><span data-stu-id="84288-124">The <xref:System.Nullable%601.HasValue%2A> property returns `true` if the variable contains a value, or `false` if it's `null`.</span></span>
  
  - <span data-ttu-id="84288-125"><xref:System.Nullable%601.Value%2A> Vlastnost vrací hodnotu, pokud <xref:System.Nullable%601.HasValue%2A> vrátí `true`.</span><span class="sxs-lookup"><span data-stu-id="84288-125">The <xref:System.Nullable%601.Value%2A> property returns a value if <xref:System.Nullable%601.HasValue%2A> returns `true`.</span></span> <span data-ttu-id="84288-126">V opačném případě <xref:System.InvalidOperationException> je vyvolána výjimka.</span><span class="sxs-lookup"><span data-stu-id="84288-126">Otherwise, an <xref:System.InvalidOperationException> is thrown.</span></span>  
  
- <span data-ttu-id="84288-127">Můžete také použít `==` a `!=` operátory s povolenou hodnotou Null typu, jak je znázorněno v následujícím příkladu: `if (x != null) y = x.Value;`.</span><span class="sxs-lookup"><span data-stu-id="84288-127">You can also use the `==` and `!=` operators with a nullable type, as shown in the following example: `if (x != null) y = x.Value;`.</span></span> <span data-ttu-id="84288-128">Pokud `a` a `b` mají obě hodnotu null, `a == b` vyhodnotí jako `true`.</span><span class="sxs-lookup"><span data-stu-id="84288-128">If `a` and `b` are both null, `a == b` evaluates to `true`.</span></span>
