Toto téma obsahuje odkazy na LINQ to XML referenční témata.

 Referenční dokumentace pro funkci LINQ to XML tříd, naleznete v tématu <xref:System.Xml.Linq>.  
  
 Referenční dokumentace pro rozšiřující metody, které vám pomůžou ověřit stromů XML proti souboru XSD, naleznete v tématu <xref:System.Xml.Schema.Extensions?displayProperty=nameWithType>.
