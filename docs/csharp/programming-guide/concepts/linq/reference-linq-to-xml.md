# <a name="reference-linq-to-xml"></a>Referenční informace (LINQ to XML)
Toto téma obsahuje odkazy na referenční témata LINQ to XML.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 Referenční dokumentaci pro třídy LINQ to XML naleznete v tématu <xref:System.Xml.Linq>.  
  
 Referenční dokumentaci pro metody rozšíření, které vám pomůžou ověřit stromy XML proti souboru XSD, naleznete v <xref:System.Xml.Schema.Extensions?displayProperty=nameWithType>tématu.  
  
 Referenční dokumentaci pro metody rozšíření, které umožňují vyhodnotit dotazy XPath ve stromu XML, naleznete v tématu <xref:System.Xml.XPath.Extensions?displayProperty=nameWithType>.
