```csharp
    // Suspends GetDotNetCountAsync() to allow the caller (the web server)

```

<span data-ttu-id="0398d-164">Zde je stejný scénář napsané pro univerzální aplikace pro Windows, který provádí stejnou úlohu, když se stiskne tlačítko:</span><span class="sxs-lookup"><span data-stu-id="0398d-164">Here's the same scenario written for a Universal Windows App, which performs the same task when a Button is pressed:</span></span>

```csharp
    // Capture the task handle here so we can await the background task later.
    var getDotNetFoundationHtmlTask = _httpClient.GetStringAsync("https://www.dotnetfoundation.org");

    // Any other work on the UI thread can be done here, such as enabling a Progress Bar.
    // This is important to do here, before the "await" call, so that the user
```

<span data-ttu-id="0398d-166">Může být pro vás sami v situaci, kde je potřeba načíst data na více místech současně.</span><span class="sxs-lookup"><span data-stu-id="0398d-166">You may find yourself in a situation where you need to retrieve multiple pieces of data concurrently.</span></span>  <span data-ttu-id="0398d-167">`Task` Rozhraní API obsahuje dvě metody `Task.WhenAll` a `Task.WhenAny` který umožňuje psát asynchronní kód, který provádí čekání bez blokování na víc úloh na pozadí.</span><span class="sxs-lookup"><span data-stu-id="0398d-167">The `Task` API contains two methods, `Task.WhenAll` and `Task.WhenAny` which allow you to write asynchronous code which performs a non-blocking wait on multiple background jobs.</span></span>

<span data-ttu-id="0398d-168">Tento příklad ukazuje, jak může vzít `User` data pro sadu `userId`s.</span><span class="sxs-lookup"><span data-stu-id="0398d-168">This example shows how you might grab `User` data for a set of `userId`s.</span></span>

```csharp
public async Task<User> GetUserAsync(int userId)
{
    // Code omitted:
    //
    // Given a user Id {userId}, retrieves a User object corresponding
    // to the entry in the database with {userId} as its Id.
}

public static async Task<IEnumerable<User>> GetUsersAsync(IEnumerable<int> userIds)
{
    var getUserTasks = new List<Task<User>>();

    foreach (int userId in userIds)

        getUserTasks.Add(GetUserAsync(userId));
    }

    return await Task.WhenAll(getUserTasks);
}
```
