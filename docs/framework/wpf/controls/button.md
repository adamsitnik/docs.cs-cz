# <a name="button"></a>Tlačítko
A <xref:System.Windows.Controls.Button> ovládací prvek reaguje na uživatelský vstup z myši, klávesnice, pera, nebo jiné vstupní zařízení a vyvolá <xref:System.Windows.Controls.Primitives.ButtonBase.Click> událostí. A <xref:System.Windows.Controls.Button> je základní [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] součást, která může obsahovat jednoduchý obsah, jako je například text a může také obsahovat složitým obsahem, jako jsou obrázky a <xref:System.Windows.Controls.Panel> ovládací prvky.
