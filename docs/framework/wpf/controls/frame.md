# <a name="frame"></a>Rámec
<xref:System.Windows.Controls.Frame> Ovládací prvek podporuje navigaci v obsahu v rámci obsahu. <xref:System.Windows.Controls.Frame> mohou být hostovány systémem kořenový element jako <xref:System.Windows.Window>, <xref:System.Windows.Navigation.NavigationWindow>, <xref:System.Windows.Controls.Page>, <xref:System.Windows.Controls.UserControl>, <xref:System.Windows.Documents.FlowDocument>, nebo jako ostrůvku v rámci stromu obsahu, který patří do kořenového prvku.
