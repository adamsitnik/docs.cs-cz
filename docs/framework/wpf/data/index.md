# <a name="data"></a>Data
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] datové vazby poskytují jednoduchý a konzistentní způsob, jak aplikace prezentovat a interagovat s daty.
