# <a name="1143---flowchartnextnull"></a>1143 – FlowchartNextNull
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|1143|  
|klíčová slova|WFActivities|  
|úroveň|Informace o|  
|Kanál|Aplikace Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Označuje, že ve vývojovém diagramu další uzel je null. Spuštění vývojového diagramu bude ukončeno.  
  
## <a name="message"></a>Zpráva  
 Vývojový diagram "%1'/FlowStep - další uzel je null. Spuštění vývojového diagramu bude ukončeno.
