# <a name="net-framework-45-built-in-activity-library"></a>Knihovna předdefinovaných aktivit rozhraní .NET framework 4.5

[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] obsahuje novou knihovnu aktivit rozšířené funkce. Typy aktivit, patří:

- [Tok řízení](../../../docs/framework/windows-workflow-foundation/control-flow-activities-in-wf.md)

- [Vývojový diagram](../../../docs/framework/windows-workflow-foundation/flowchart-activities-in-wf.md)

- [Stavový stroj](../../../docs/framework/windows-workflow-foundation/state-machine-activities-in-wf.md)

- [Aktivity zasílání zpráv](../../../docs/framework/wcf/feature-details/messaging-activities.md)

- [Modul runtime](../../../docs/framework/windows-workflow-foundation/runtime-activities-in-wf.md)

- [Primitivní elementy](../../../docs/framework/windows-workflow-foundation/primitives-activities-in-wf.md)

- [Transakce](../../../docs/framework/windows-workflow-foundation/transaction-activities-in-wf.md)

- [Kolekce](../../../docs/framework/windows-workflow-foundation/collection-activities-in-wf.md)

- [Zpracování chyb](../../../docs/framework/windows-workflow-foundation/error-handling-activities-in-wf.md)

- [Migrace](../../../docs/framework/windows-workflow-foundation/migration-activity-in-wf.md)

Aktivity v knihovně integrovaných aktivit můžete složený do pracovních postupů, nebo ho používáte spolu se vlastní aktivity. Aktivity v knihovně integrovaných aktivit jsou zapečetěné; nejsou určeny pro použití k vytvoření nové funkce prostřednictvím dědičnosti.
