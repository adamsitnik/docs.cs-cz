# <a name="windows-workflow-foundation-data-model"></a>Datový model Windows Workflow Foundation
Datový model Windows Workflow Foundation se skládá ze tří koncepty: proměnné, argumenty a výrazy. Proměnné představují úložiště dat a argumenty představují tok dat do a z aktivity. Argumenty jsou vázána (přiřazena hodnota) pomocí výrazů, které mohou odkazovat na proměnné.
