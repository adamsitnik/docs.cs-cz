     <span data-ttu-id="60a77-112">Hodnoty ve slovníku readWriteValues jsou dodávány jako **InstanceValue** objekty.</span><span class="sxs-lookup"><span data-stu-id="60a77-112">The values in the readWriteValues dictionary are packaged as **InstanceValue** objects.</span></span> <span data-ttu-id="60a77-113">Hodnoty ve slovníku jen pro zápis, které jsou dodávány jako **InstanceValue** objekty s nastavenou InstanceValueOptions.Optional a InstanceValueOption.WriteOnly.</span><span class="sxs-lookup"><span data-stu-id="60a77-113">The values in the write-only dictionary are packaged as **InstanceValue** objects with InstanceValueOptions.Optional and InstanceValueOption.WriteOnly set.</span></span> <span data-ttu-id="60a77-114">Každý **InstanceValue** poskytované **CollectValues** implementace napříč všichni účastníci trvalosti musí mít jedinečný název.</span><span class="sxs-lookup"><span data-stu-id="60a77-114">Each **InstanceValue** provided by the **CollectValues** implementations across all persistence participants must have a unique name.</span></span>

  
     <span data-ttu-id="60a77-121"><xref:System.Activities.Persistence.PersistenceParticipant.MapValues%2A> Metoda poskytuje funkce, která <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> tak není, v tom, že umožňuje pro závislost na jiná hodnota poskytnutá jiným účastníkem trvalého, která nebyla zpracována <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> ještě.</span><span class="sxs-lookup"><span data-stu-id="60a77-121">The <xref:System.Activities.Persistence.PersistenceParticipant.MapValues%2A> method provides functionality that <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> does not, in that it allows for a dependency on another value provided by another persistence participant that hasn’t been processed by <xref:System.Activities.Persistence.PersistenceParticipant.CollectValues%2A> yet.</span></span>
