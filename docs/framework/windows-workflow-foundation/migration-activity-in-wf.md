# <a name="migration-activity-in-wf"></a>Aktivity migrace v WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] poskytuje <xref:System.Activities.Statements.Interop> aktivity pro spuštění aktivity, které jsou odvozeny z aktivity v pracovním postupu, který je založen na <xref:System.Activities.Activity>. Další informace najdete v tématu [pokyny k migraci](migration-guidance.md) oddílu.  
  
> [!NOTE]
>  <xref:System.Activities.Statements.Interop> Aktivity se nezobrazí v panelu nástrojů návrháře postupu, pokud projekt pracovního postupu neobsahuje jeho **Cílová architektura** nastavení **rozhraní .net Framework 4** nebo vyšší.
