# <a name="migration-activity-in-wf"></a>Aktivita migrace v WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)]poskytuje aktivitu pro provádění aktivit, které jsou odvozeny z aktivity v rámci pracovního postupu, <xref:System.Activities.Activity>na kterém je založena. <xref:System.Activities.Statements.Interop> Další informace najdete v části [pokyny k migraci](migration-guidance.md) .  
  
> [!NOTE]
> Tato <xref:System.Activities.Statements.Interop> aktivita se nezobrazí v sadě nástrojů návrháře pracovních postupů, pokud nemá projekt pracovního postupu nastavenou **cílovou architekturu** na **.NET Framework 4** nebo vyšší.
