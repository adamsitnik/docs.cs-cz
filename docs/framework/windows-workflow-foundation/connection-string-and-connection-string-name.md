# <a name="connection-string-and-connection-string-name"></a>Připojovací řetězec a název připojovacího řetězce
**Připojovací řetězec** vlastnost určuje připojovací řetězec, který Store Instance pracovního postupu SQL má používat pro připojení k databázi trvalosti. Tento parametr je volitelný parametr. **Název připojovacího řetězce** vlastnost určuje název pojmenovaného připojovacího řetězce, který Store Instance pracovního postupu SQL má používat pro připojení k databázi trvalosti. Tento parametr je volitelný parametr. By měl určit hodnotu pro vlastnost název připojovacího řetězce nebo vlastnost připojovací řetězec, pokud nechcete, aby Store Instance pracovního postupu SQL chcete použít výchozí připojovací řetězec s názvem **DefaultSqlWorkflowInstanceStoreConnectionString** .
