# <a name="runtime-activities-in-wf"></a>Modul runtime aktivit v WF
[!INCLUDE[netfx_current_short](../../../includes/netfx-current-short-md.md)] poskytuje několik aktivit poskytované systémem pro přístup k funkcím modulu runtime pracovního postupu, jako je například trvalost a ukončení.  
  
|Aktivita|Popis|  
|--------------|-----------------|  
|<xref:System.Activities.Statements.Persist>|Výslovně požaduje, aby pracovní postup zachování stavu.|  
|<xref:System.Activities.Statements.TerminateWorkflow>|Ukončí běžící instance pracovního postupu.|  
|<xref:System.Activities.Statements.NoPersistScope>|Aktivita kontejneru, který brání uchování podřízené aktivity.|
