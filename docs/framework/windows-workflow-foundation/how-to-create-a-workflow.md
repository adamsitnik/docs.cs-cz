# <a name="how-to-create-a-workflow"></a>Postupy: Vytvoření pracovního postupu
Pracovní postupy mohou být vytvořeny z vestavěných aktivit i z vlastních aktivit. Témata v této části popisují postup vytvoření pracovního postupu, který používá předdefinované aktivity, jako je <xref:System.Activities.Statements.Flowchart> aktivita, a vlastní aktivity z předchozího [postupu: Vytvoření tématu aktivity](how-to-create-an-activity.md) . Pracovní postup modeluje číslo odhadující hru. K dokončení kurzu je nutné pouze jedno z témat v této části. měli byste vybrat styl, který vás zajímá, a postupovat podle tohoto kroku. V případě potřeby však můžete všechna témata dokončit.  
  
> [!NOTE]
> Každé téma v kurzu Začínáme závisí na předchozích tématech. Chcete-li dokončit toto téma, je nutné [nejprve provést následující kroky: Vytvoří aktivitu](how-to-create-an-activity.md).  
  
> [!NOTE]
> Pokud si chcete stáhnout dokončenou verzi kurzu, přečtěte si [kurz programovací model Windows Workflow Foundation (WF45) – začínáme](https://go.microsoft.com/fwlink/?LinkID=248976).
