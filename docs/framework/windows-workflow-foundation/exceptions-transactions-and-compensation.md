# <a name="exceptions-transactions-and-compensation"></a>Výjimky, transakce a kompenzace
[!INCLUDE[wf1](../../../includes/wf1-md.md)] poskytuje několik různých mechanismy pro zpracování chyb za běhu v pracovních postupech. Pracovní postupy můžete použít kombinaci obslužné rutiny výjimek, zrušení, transakce a kompenzace ke zpracování a provést řádné zotavení chybové stavy.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [Výjimky](exceptions.md)  
 Popisuje způsob použití <xref:System.Activities.Statements.TryCatch> aktivity pro zpracování výjimek v pracovním postupu.  
  
 [Transakce](workflow-transactions.md)  
 Popisuje způsob použití <xref:System.Activities.Statements.TransactionScope> aktivity pro použití transakcí v pracovním postupu.  
  
 [Kompenzace](compensation.md)  
 Popisuje kompenzace v pracovních postupech a ukazuje, jak použít kompenzace aktivity, jako <xref:System.Activities.Statements.CompensableActivity>, <xref:System.Activities.Statements.Compensate>, a <xref:System.Activities.Statements.Confirm>.  
  
 [Zrušení](modeling-cancellation-behavior-in-workflows.md)  
 Popisuje postup při zrušení zpracování v pracovní postupy pomocí integrovaných aktivit, stejně jako vlastní aktivity.  
  
 [Ladění pracovních postupů](debugging-workflows.md)
