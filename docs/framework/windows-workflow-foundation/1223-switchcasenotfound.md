# <a name="1223---switchcasenotfound"></a>1223 – SwitchCaseNotFound
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|1223|  
|klíčová slova|WFActivities|  
|úroveň|Informace o|  
|Kanál|Aplikace Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Označuje, že v aktivita Switch nelze nalézt aktivitu case odpovídající výsledku výrazu.  
  
## <a name="message"></a>Zpráva  
 Aktivita Switch '%1' nenalezla aktivitu Case odpovídající výsledku výrazu.
