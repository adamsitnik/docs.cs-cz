# <a name="1148---flowchartswitchcasenotfound"></a>1148 – FlowchartSwitchCaseNotFound
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|1148|  
|klíčová slova|WFActivities|  
|úroveň|Informace o|  
|Kanál|Aplikace Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Označuje, že nebyl nalezen odpovídající case ani výchozí případ přepínače vývojový diagram. Spuštění vývojového diagramu bude ukončeno.  
  
## <a name="message"></a>Zpráva  
 Vývojový diagram "%1'/FlowSwitch - najít aktivitu Case ani výchozí příkaz Case odpovídající výsledku výrazu. Spuštění vývojového diagramu bude ukončeno.  
  
## <a name="details"></a>Podrobnosti  
  
|Název položky dat|Datový typ položky|Popis|  
|--------------------|--------------------|-----------------|  
|FlowChart|xs:string|Zobrazovaný název vývojový diagram.|  
|AppDomain|xs:string|Řetězec vrácený funkcí AppDomain.CurrentDomain.FriendlyName.|
