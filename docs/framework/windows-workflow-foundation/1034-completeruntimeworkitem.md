# <a name="1034---completeruntimeworkitem"></a>1034 – CompleteRuntimeWorkItem
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|1034|  
|klíčová slova|WFRuntime|  
|úroveň|Podrobnosti|  
|Kanál|Aplikace Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Označuje, že že runtimeworkitem byla dokončena.  
  
## <a name="message"></a>Zpráva  
 Běhová pracovní položka byla dokončena pro aktivitu %1, DisplayName: %2, InstanceId: '%3'.  
  
## <a name="details"></a>Podrobnosti  
  
|Název položky dat|Datový typ položky|Popis|  
|--------------------|--------------------|-----------------|  
|Aktivita|xs:string|Název typu aktivity.|  
|displayName|xs:string|Zobrazovaný název aktivity.|  
|InstanceId|xs:string|Id instance aktivity.|  
|AppDomain|xs:string|Řetězec vrácený funkcí AppDomain.CurrentDomain.FriendlyName.|
