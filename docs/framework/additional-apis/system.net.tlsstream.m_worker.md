# <a name="tlsstreamm_worker-field"></a>TlsStream. m_Worker – pole

Představuje stav datového proudu SSL.

## <a name="syntax"></a>Syntaxe

```csharp
private SslState m_Worker;
```

## <a name="field-value"></a>Hodnota pole

`System.Net.Security.SslState`  
Stav datového proudu SSL.

## <a name="remarks"></a>Poznámky

> [!WARNING]
> Pole `TlsStream.m_Worker` je soukromé a není určeno pro použití přímo v kódu.
>
> Společnost Microsoft v žádné situaci nepodporuje použití tohoto pole v produkční aplikaci.
