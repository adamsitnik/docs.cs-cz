 <span data-ttu-id="3a23e-179">Element `<Assembly>` definuje zásady modulu runtime pro všechny typy v sestavení.</span><span class="sxs-lookup"><span data-stu-id="3a23e-179">The `<Assembly>` element defines runtime policy for all the types in an assembly.</span></span> <span data-ttu-id="3a23e-180">Liší se od [knihovny\<](library-element-net-native.md) elementu, který určuje knihovnu, ale závisí na jejích podřízených prvcích k definování zásad reflexe modulu runtime.</span><span class="sxs-lookup"><span data-stu-id="3a23e-180">It differs from the [\<Library>](library-element-net-native.md) element, which specifies a library but depends on its child elements to define runtime reflection policy.</span></span> <span data-ttu-id="3a23e-181">Element `<Assembly>` se vztahuje na všechny typy v sestavení, pokud nejsou přepsány podřízeným elementem.</span><span class="sxs-lookup"><span data-stu-id="3a23e-181">The `<Assembly>` element applies to all the types in an assembly unless they are overridden by a child element.</span></span>
