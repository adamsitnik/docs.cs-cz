 <span data-ttu-id="ec030-115">A `conceptual model` je konkrétní reprezentace strukturu dat jako entit a vztahů a je obecně podle jazyka specifického pro doménu (DSL), který implementuje koncepty škálovatelnosti.</span><span class="sxs-lookup"><span data-stu-id="ec030-119">Meanwhile storage schema designers can focus on the efficiency of data access, storage, and scalability.</span></span>
