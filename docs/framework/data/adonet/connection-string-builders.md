 <span data-ttu-id="c0ae4-115">Třídy tvůrce řetězec připojení jsou navrženy tak, aby obsahoval jenom izolovat a chránit proti chyby syntaxe a ohrožení zabezpečení.</span><span class="sxs-lookup"><span data-stu-id="c0ae4-115">The connection string builder classes are designed to eliminate guesswork and protect against syntax errors and security vulnerabilities.</span></span> <span data-ttu-id="c0ae4-116">Poskytuje metody a vlastnosti odpovídající dvojice klíč/hodnota známé od každého poskytovatele dat.</span><span class="sxs-lookup"><span data-stu-id="c0ae4-116">They provide methods and properties corresponding to the known key/value pairs permitted by each data provider.</span></span> <span data-ttu-id="c0ae4-117">Každá třída udržuje pevné kolekce synonym a může překládat z synonymum pro odpovídající dobře známý název klíče.</span><span class="sxs-lookup"><span data-stu-id="c0ae4-117">Each class maintains a fixed collection of synonyms and can translate from a synonym to the corresponding well-known key name.</span></span> <span data-ttu-id="c0ae4-118">Kontroly jsou prováděny pro páry klíč/hodnota platný a vyvolá výjimku, neplatný pár.</span><span class="sxs-lookup"><span data-stu-id="c0ae4-118">Checks are performed for valid key/value pairs and an invalid pair throws an exception.</span></span> <span data-ttu-id="c0ae4-119">Kromě toho jsou zpracovány vloženého hodnoty bezpečným způsobem.</span><span class="sxs-lookup"><span data-stu-id="c0ae4-119">In addition, injected values are handled in a safe manner.</span></span>  
  
 <span data-ttu-id="c0ae4-120">Následující příklad ukazuje, jak <xref:System.Data.SqlClient.SqlConnectionStringBuilder> zpracovává vložený hodnotu navíc pro `Initial Catalog` nastavení.</span><span class="sxs-lookup"><span data-stu-id="c0ae4-120">The following example demonstrates how the <xref:System.Data.SqlClient.SqlConnectionStringBuilder> handles an inserted extra value for the `Initial Catalog` setting.</span></span>
