# <a name="connection-pooling"></a>Sdružování připojení
Připojení ke zdroji dat může být časově náročné. Pro minimalizaci nákladů na otevírání připojení používá ADO.NET techniku optimalizace nazývanou *sdružování připojení*, která minimalizuje náklady na opakované otevírání a uzavírání připojení. Sdružování připojení se pro poskytovatele .NET Frameworkch dat zpracovává jinak.
