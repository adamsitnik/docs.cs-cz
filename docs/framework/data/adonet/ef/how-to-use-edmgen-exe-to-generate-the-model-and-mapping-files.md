### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="9a59c-116">Generování modelu školní pro projekt jazyka Visual Basic pomocí EdmGen.exe</span><span class="sxs-lookup"><span data-stu-id="9a59c-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>

2.  <span data-ttu-id="9a59c-119">Na příkazovém řádku spusťte následující příkaz bez konce řádků:</span><span class="sxs-lookup"><span data-stu-id="9a59c-119">At the command prompt, execute the following command without line breaks:</span></span>

