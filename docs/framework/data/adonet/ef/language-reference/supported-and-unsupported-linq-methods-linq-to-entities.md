# <a name="supported-and-unsupported-linq-methods-linq-to-entities"></a>Podporované a nepodporované metody LINQ (LINQ to Entities)
V této části najdete informace o standardních operátorech dotazů jazyka LINQ (Language-Integrated Query), které jsou podporované nebo nepodporované v LINQ to Entitiesch dotazech. Mnoho standardních operátorů dotazů LINQ má přetíženou verzi, která přijímá celočíselný argument. Celočíselný argument odpovídá indexu založenému na nule v sekvenci, která je provozována, <xref:System.Collections.Generic.IEqualityComparer%601>nebo. <xref:System.Collections.Generic.IComparer%601> Není-li uvedeno jinak, nejsou podporovány tyto přetížené verze operátorů dotazů LINQ Standard a pokus o jejich použití vyvolá výjimku.
