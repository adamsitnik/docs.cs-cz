# <a name="-greater-than-entity-sql"></a>> (Větší) (Entity SQL)
Porovná dva výrazy k určení, zda levý výraz má hodnotu větší než pravý výraz.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
expression > expression  
```  
  
## <a name="arguments"></a>Arguments  
 `expression`  
 Libovolný platný výraz. Implicitně převést datové typy musí mít oba výrazy.  
  
## <a name="result-types"></a>Typy výsledků  
 `true` Pokud levý výraz má hodnotu větší než pravý výraz; v opačném případě `false`.
