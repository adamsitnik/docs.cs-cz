# <a name="multiple-active-result-sets-mars"></a>Více aktivních sad výsledků (MARS)
Několik aktivních sad výsledků dotazu (MARS) je funkce, která umožňuje spuštění více dávek v jednom připojení. V předchozích verzích bylo možné v jednom okamžiku provést pouze jednu dávku s jedním připojením. Spuštění více dávek se službou MARS neznamená souběžné provádění operací.
