# <a name="stored-procedures"></a>Uložené procedury
[!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] metody v objektovém modelu používá k reprezentaci uložené procedury v databázi. Určíte jako uložené procedury pro metody s použitím <xref:System.Data.Linq.Mapping.FunctionAttribute> atribut a v případě potřeby <xref:System.Data.Linq.Mapping.ParameterAttribute> atribut. Další informace najdete v tématu [The LINQ to SQL objektový Model](../../../../../../docs/framework/data/adonet/sql/linq/the-linq-to-sql-object-model.md).  
  
 Vývojáři, kteří používají Visual Studio by obvykle pomocí Návrháře relací objektů mapě uložených procedur. Témata v této části ukazují, jak formulář a volat tyto metody ve své aplikaci psát kód sami.
