# <a name="how-to-specify-database-names"></a>Postupy: Zadání názvů databází
<xref:System.Data.Linq.Mapping.DatabaseAttribute.Name%2A> Pomocí vlastnosti<xref:System.Data.Linq.Mapping.DatabaseAttribute> u atributu určete název databáze, pokud není název dodán připojením.  
  
 Ukázky kódu naleznete v tématu <xref:System.Data.Linq.Mapping.DatabaseAttribute.Name%2A>.  
  
### <a name="to-specify-the-name-of-the-database"></a>Určení názvu databáze  
  
1. <xref:System.Data.Linq.Mapping.DatabaseAttribute> Přidejte atribut do deklarace třídy pro databázi.  
  
2. <xref:System.Data.Linq.Mapping.DatabaseAttribute.Name%2A> Přidejte vlastnost<xref:System.Data.Linq.Mapping.DatabaseAttribute> do atributu.  
  
3. Nastavte hodnotu <xref:System.Data.Linq.Mapping.DatabaseAttribute.Name%2A> vlastnosti na název, který chcete zadat.
