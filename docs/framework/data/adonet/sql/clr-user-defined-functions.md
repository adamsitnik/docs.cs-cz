# <a name="clr-user-defined-functions"></a>Uživatelem definované funkce CLR
Uživatelsky definované funkce jsou rutiny, které mohou přijímat parametry, provádět výpočty nebo jiné akce a vracet výsledek. Uživatelsky definované funkce můžete psát v libovolném programovacím jazyce Microsoft .NET Framework, jako je Microsoft Visual Basic .NET nebo Microsoft Visual C#.  
  
 Podrobnější informace najdete v tématu [uživatelsky definované funkce CLR](/sql/relational-databases/clr-integration-database-objects-user-defined-functions/clr-user-defined-functions).
