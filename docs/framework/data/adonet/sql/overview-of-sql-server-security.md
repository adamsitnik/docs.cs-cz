- <span data-ttu-id="f0ca9-113">Ověřování je proces přihlášení k systému SQL Server, pomocí kterého požaduje objekt zabezpečení přístup odesláním přihlašovací údaje, které vyhodnotí server.</span><span class="sxs-lookup"><span data-stu-id="f0ca9-113">Authentication is the process of logging on to SQL Server by which a principal requests access by submitting credentials that the server evaluates.</span></span> <span data-ttu-id="f0ca9-114">Ověření určí identitu uživatele nebo proces ověřuje.</span><span class="sxs-lookup"><span data-stu-id="f0ca9-114">Authentication establishes the identity of the user or process being authenticated.</span></span>  
  
- <span data-ttu-id="f0ca9-115">Autorizace je procesu, který určuje přístup k objektu zabezpečení zabezpečitelné prostředky a operace, které jsou povoleny pro tyto prostředky.</span><span class="sxs-lookup"><span data-stu-id="f0ca9-115">Authorization is the process of determining which securable resources a principal can access, and which operations are allowed for those resources.</span></span>
