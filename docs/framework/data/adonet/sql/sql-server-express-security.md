 <span data-ttu-id="bed1a-118">Uživatelská instance je vygenerována z primární instance SQL Server nebo SQL Server Express jménem uživatele.</span><span class="sxs-lookup"><span data-stu-id="bed1a-118">A user instance is generated from a primary instance of SQL Server or SQL Server Express on behalf of a user.</span></span> <span data-ttu-id="bed1a-119">Spouští se jako proces uživatele v kontextu zabezpečení systému Windows uživatele, nikoli jako služba.</span><span class="sxs-lookup"><span data-stu-id="bed1a-119">It runs as a user process under the Windows security context of the user, not as a service.</span></span> <span data-ttu-id="bed1a-120">Přihlášení SQL Server nejsou povolena. podporují se jenom přihlašovací údaje systému Windows.</span><span class="sxs-lookup"><span data-stu-id="bed1a-120">SQL Server logins are disallowed; only Windows logins are supported.</span></span> <span data-ttu-id="bed1a-121">Tím se zabrání tomu, aby software spuštěný na uživatelské instanci prováděl změny v rámci systému, které by uživatel neměl mít oprávnění k provedení.</span><span class="sxs-lookup"><span data-stu-id="bed1a-121">This prevents software executing on a user instance from making system-wide changes that the user would not have permissions to make.</span></span> <span data-ttu-id="bed1a-122">Uživatelská instance je také označována jako podřízená nebo klientská instance a je někdy označována pomocí zkratky RANU ("spustit jako normální uživatel").</span><span class="sxs-lookup"><span data-stu-id="bed1a-122">A user instance is also known as a child or client instance, and is sometimes referred to by using the RANU acronym ("run as normal user").</span></span>  
  
 <span data-ttu-id="bed1a-123">Jednotlivé uživatelské instance jsou izolované od své nadřazené instance a z jiných instancí uživatelů spuštěných ve stejném počítači.</span><span class="sxs-lookup"><span data-stu-id="bed1a-123">Each user instance is isolated from its parent instance and from other user instances running on the same computer.</span></span> <span data-ttu-id="bed1a-124">Databáze nainstalované v uživatelských instancích jsou otevřené jenom v jednouživatelském režimu. k nim se nelze připojit více uživatelů.</span><span class="sxs-lookup"><span data-stu-id="bed1a-124">Databases installed on user instances are opened in single-user mode only; multiple users cannot connect to them.</span></span> <span data-ttu-id="bed1a-125">Replikace, distribuované dotazy a vzdálená připojení jsou pro uživatelské instance zakázané.</span><span class="sxs-lookup"><span data-stu-id="bed1a-125">Replication, distributed queries and remote connections are disabled for user instances.</span></span> <span data-ttu-id="bed1a-126">Při připojení k uživatelské instanci nemají uživatelé žádná zvláštní oprávnění pro nadřazenou instanci SQL Server Express.</span><span class="sxs-lookup"><span data-stu-id="bed1a-126">When connected to a user instance, users do not have any special privileges on the parent SQL Server Express instance.</span></span>
