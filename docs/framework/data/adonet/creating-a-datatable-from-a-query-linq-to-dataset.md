 <span data-ttu-id="b7493-115">Všimněte si, že <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> volání metody způsobí provedení dotazu vázaného na zdrojovou tabulku.</span><span class="sxs-lookup"><span data-stu-id="b7493-115">Note that calling the <xref:System.Data.DataTableExtensions.CopyToDataTable%2A> method will cause the query bound to the source table to execute.</span></span>
