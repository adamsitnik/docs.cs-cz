 <span data-ttu-id="3d641-108">V konceptuálním modelu můžete vytvořit hierarchie dědičnosti, ve kterých odvozený typ dědí z jiného odvozeného typu.</span><span class="sxs-lookup"><span data-stu-id="3d641-108">In a conceptual model you can build inheritance hierarchies in which a derived type inherits from another derived type.</span></span> <span data-ttu-id="3d641-109">Typ v horní části hierarchie (jeden typ v hierarchii, která není odvozeným typem) se nazývá *kořenový typ*.</span><span class="sxs-lookup"><span data-stu-id="3d641-109">The type at the top of the hierarchy (the one type in the hierarchy that is not a derived type) is called the *root type*.</span></span> <span data-ttu-id="3d641-110">V hierarchii dědičnosti [klíč entity](../../../../docs/framework/data/adonet/entity-key.md) na kořenový typ musí být definován.</span><span class="sxs-lookup"><span data-stu-id="3d641-110">In an inheritance hierarchy, the [entity key](../../../../docs/framework/data/adonet/entity-key.md) must be defined on the root type.</span></span>  
  
 <span data-ttu-id="3d641-111">Hierarchie dědičnosti, ve kterých odvozený typ dědí z více než jeden typ nelze vytvořit.</span><span class="sxs-lookup"><span data-stu-id="3d641-111">You cannot build inheritance hierarchies in which a derived type inherits from more than one type.</span></span> <span data-ttu-id="3d641-112">Například v konceptuálním modelu s `Book` typ entity, můžete definovat odvozené typy `FictionBook` a `NonFictionBook` každý dědit z `Book`.</span><span class="sxs-lookup"><span data-stu-id="3d641-112">For example, in a conceptual model with a `Book` entity type, you could define derived types `FictionBook` and `NonFictionBook` that each inherit from `Book`.</span></span> <span data-ttu-id="3d641-113">Však nelze definovat poté typ, který zdědí vlastnosti z obou `FictionBook` a `NonFictionBook` typy.</span><span class="sxs-lookup"><span data-stu-id="3d641-113">However, you could not then define a type that inherits from both the `FictionBook` and `NonFictionBook` types.</span></span>
