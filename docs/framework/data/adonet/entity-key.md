<span data-ttu-id="70015-103">*Klíč entity* je [vlastnost](../../../../docs/framework/data/adonet/property.md) nebo sada vlastností [typ entity](../../../../docs/framework/data/adonet/entity-type.md) , který slouží k určení identity.</span><span class="sxs-lookup"><span data-stu-id="70015-103">An *entity key* is a [property](../../../../docs/framework/data/adonet/property.md) or a set of properties of an [entity type](../../../../docs/framework/data/adonet/entity-type.md) that are used to determine identity.</span></span> <span data-ttu-id="70015-104">Vlastnosti, které tvoří klíč entity jsou vybrán v době návrhu.</span><span class="sxs-lookup"><span data-stu-id="70015-104">The properties that make up an entity key are chosen at design time.</span></span> <span data-ttu-id="70015-105">Hodnoty vlastnosti klíče entity musí jednoznačně identifikovat instanci entity typu v rámci [sadu entit](../../../../docs/framework/data/adonet/entity-set.md) v době běhu.</span><span class="sxs-lookup"><span data-stu-id="70015-105">The values of entity key properties must uniquely identify an entity type instance within an [entity set](../../../../docs/framework/data/adonet/entity-set.md) at run time.</span></span> <span data-ttu-id="70015-106">Vlastnosti, které tvoří klíč entity, je třeba zvolit pro zajištění jedinečnosti instancí v sadu entit.</span><span class="sxs-lookup"><span data-stu-id="70015-106">The properties that make up an entity key should be chosen to guarantee uniqueness of instances in an entity set.</span></span>  
  
 <span data-ttu-id="70015-107">Následují požadavky na sadu vlastností pro být klíč entity:</span><span class="sxs-lookup"><span data-stu-id="70015-107">The following are the requirements for a set of properties to be an entity key:</span></span>  
  
-   <span data-ttu-id="70015-108">Žádné klíče dvě entity v rámci sady entit může být identické.</span><span class="sxs-lookup"><span data-stu-id="70015-108">No two entity keys within an entity set can be identical.</span></span> <span data-ttu-id="70015-109">To znamená pro dvě entity v rámci sady entit, hodnoty pro všechny vlastnosti, které tvoří klíč nemůže být stejné.</span><span class="sxs-lookup"><span data-stu-id="70015-109">That is, for any two entities within an entity set, the values for all of the properties that constitute a key cannot be the same.</span></span> <span data-ttu-id="70015-110">Nicméně některé (ale ne všech) hodnot, které tvoří entitu klíč může být stejný.</span><span class="sxs-lookup"><span data-stu-id="70015-110">However, some (but not all) of the values that make up an entity key can be the same.</span></span>  
  
-   <span data-ttu-id="70015-111">Klíč entity musí obsahovat sadu Null, nezměnitelný, [primitivní typ vlastnosti](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span><span class="sxs-lookup"><span data-stu-id="70015-111">An entity key must consist of a set of non-nullable, immutable, [primitive type properties](../../../../docs/framework/data/adonet/entity-data-model-primitive-data-types.md).</span></span>  
  
-   <span data-ttu-id="70015-112">Vlastnosti, které tvoří klíč entity pro danou entitu typu nelze změnit.</span><span class="sxs-lookup"><span data-stu-id="70015-112">The properties that make up an entity key for a given entity type cannot change.</span></span> <span data-ttu-id="70015-113">Více než jeden klíč možné entity nelze povolit pro danou entitu typu; náhradní klíče nejsou podporovány.</span><span class="sxs-lookup"><span data-stu-id="70015-113">You cannot allow more than one possible entity key for a given entity type; surrogate keys are not supported.</span></span>  
  
-   <span data-ttu-id="70015-114">Když entitu je zahrnuta v hierarchii dědičnosti, kořenové entity musí obsahovat všechny vlastnosti, které tvoří klíč entity a klíč entity musí být definováno na kořenový typ entity.</span><span class="sxs-lookup"><span data-stu-id="70015-114">When an entity is involved in an inheritance hierarchy, the root entity must contain all the properties that make up the entity key, and the entity key must be defined on the root entity type.</span></span> <span data-ttu-id="70015-115">Další informace najdete v tématu [modelu Entity Data Model: Dědičnost](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="70015-115">For more information, see [Entity Data Model: Inheritance](../../../../docs/framework/data/adonet/entity-data-model-inheritance.md).</span></span>
