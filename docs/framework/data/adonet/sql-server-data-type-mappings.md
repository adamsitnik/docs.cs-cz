# <a name="sql-server-data-type-mappings"></a>Mapování datových typů SQL Serveru
SQL Server a .NET Framework jsou založené na různých systémech typů. Například struktura .NET Framework <xref:System.Decimal> má maximální měřítko 28, zatímco SQL Server Desítková a číselná datový typ mají maximální měřítko 38. Aby se zajistila integrita dat při čtení a zápisu <xref:System.Data.SqlClient.SqlDataReader> dat, zpřístupňuje SQL Server – specifické metody přístupového objektu <xref:System.Data.SqlTypes> , které vracejí objekty, i metody přístupového objektu, které vracejí .NET Framework typy. SQL Server typy i .NET Framework typy jsou také reprezentovány výčty v <xref:System.Data.DbType> třídách a <xref:System.Data.SqlDbType> , které lze použít při zadávání <xref:System.Data.SqlClient.SqlParameter> datových typů.
