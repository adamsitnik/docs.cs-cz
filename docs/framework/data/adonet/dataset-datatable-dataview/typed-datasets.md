# <a name="typed-datasets"></a>Typové datové sady
Spolu s pozdní vazbou přístup k hodnotám prostřednictvím slabě typované proměnné <xref:System.Data.DataSet> poskytuje přístup k datům prostřednictvím metafora silného typu. Tabulky a sloupce, které jsou součástí **datovou sadu** lze přistupovat pomocí uživatelsky přívětivých názvů a silně typované proměnné.  
  
 Zadaný **datovou sadu** je třída, která je odvozena z **datovou sadu**. V důsledku toho dědí všechny metody, události a vlastnosti **datovou sadu**. Kromě toho typovaného **datovou sadu** poskytuje silného typu metody, události a vlastnosti. To znamená, že se zobrazí tabulky a sloupce podle názvu, namísto použití metody založené na kolekci. Kromě lepší čitelnost kódu, typovaného **datovou sadu** také umožňuje kódu sady Visual Studio .NET editoru při psaní automaticky dokončit řádky.  
  
 Kromě toho silnými typy **datovou sadu** poskytuje přístup k hodnotám jako správného typu v době kompilace. Se silnými typy **datovou sadu**, jsou zachyceny chyby neshoda typu, pokud kód je zkompilován spíše než v čas spuštění.
