        userName="MyDomain\UserAccount"   
        password="*****" />

 <span data-ttu-id="93fee-116">Účet pevné identity musí být účet s nízkou úrovní oprávnění, který má oprávnění pouze nezbytné v databázi.</span><span class="sxs-lookup"><span data-stu-id="93fee-116">The fixed identity account should be a low-privilege account that has been granted only necessary permissions in the database.</span></span> <span data-ttu-id="93fee-117">Kromě toho by měla šifrovat konfiguračního souboru tak, aby uživatelské jméno a heslo nejsou zveřejněné ve formátu prostého textu.</span><span class="sxs-lookup"><span data-stu-id="93fee-117">In addition, you should encrypt the configuration file so that the user name and password are not exposed in clear text.</span></span>
