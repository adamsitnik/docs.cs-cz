# <a name="xshared-attribute"></a><span data-ttu-id="e6335-102">x:Shared – atribut</span><span class="sxs-lookup"><span data-stu-id="e6335-102">x:Shared Attribute</span></span>
<span data-ttu-id="e6335-103">Pokud je nastavena na `false`, upravuje chování načtení prostředku WPF, takže požadavky s atributy prostředku vytvoření nové instance pro každý požadavek místo sdílení stejné instanci pro všechny požadavky.</span><span class="sxs-lookup"><span data-stu-id="e6335-103">When set to `false`, modifies WPF resource-retrieval behavior so that requests for the attributed resource create a new instance for each request instead of sharing the same instance for all requests.</span></span>

```xaml  
</ResourceDictionary>  
```
