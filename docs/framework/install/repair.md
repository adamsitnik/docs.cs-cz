# <a name="repair-the-net-framework"></a>Oprava rozhraní .NET Framework

Instalace rozhraní .NET Framework v některých situacích může dojít k poškození a vyžadují opravy. To může být v případě, pokud aplikace spadne hned po pokusu o spuštění, nebo nelze-li nainstalovat novější verze rozhraní .NET Framework.

Můžete opravit pomocí instalace rozhraní .NET Framework [opravit nástroje .NET Framework](https://download.microsoft.com/download/2/B/D/2BDE5459-2225-48B8-830C-AE19CAF038F1/NetFxRepairTool.exe).

Pokud vaše aplikace se pořád ještě spouští po dokončení opravy rozhraní .NET Framework, může aplikace došlo k potížím. V takovém případě by měl kontaktujte vydavatele aplikace.
