# <a name="errorprovider-component-windows-forms"></a>ErrorProvider – komponenta (Windows Forms)
Windows Forms `ErrorProvider` komponenty se používá k zobrazení uživatele nevtíravým způsobem, že se něco stalo. To se obvykle používá ve spojení s ověřování uživatelského vstupu ve formuláři nebo zobrazování chyb v prvku dataset.
