# <a name="printdocument-component-windows-forms"></a>PrintDocument – komponenta (Windows Forms)
Windows Forms `PrintDocument` komponenty se používá k nastavení vlastnosti, které popisují, jak vytisknout a pak se dokument vytiskne na v rámci aplikace pro systém Windows. Je možné použít ve spojení s <xref:System.Windows.Forms.PrintDialog> součást má být v příkazu všechny aspekty tisk dokumentu.
