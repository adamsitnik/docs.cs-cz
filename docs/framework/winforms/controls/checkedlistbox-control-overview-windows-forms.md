# <a name="checkedlistbox-control-overview-windows-forms"></a>CheckedListBox – přehled ovládacího prvku (Windows Forms)
Windows Forms <xref:System.Windows.Forms.CheckedListBox> rozšiřuje ovládací prvek <xref:System.Windows.Forms.ListBox> ovládacího prvku. Provádí téměř všechno, co seznamu nemá a také můžete zobrazit zaškrtávací políčko vedle položky v seznamu. Jsou rozdíly mezi dvou ovládacích prvků, který zkontrolovat seznamem pouze podporu <xref:System.Windows.Forms.DrawMode.Normal?displayProperty=nameWithType>; a pole se seznamem může mít pouze jednu položku nebo není vybrán žádný, který zkontrolovat. Mějte na paměti, že vybrané položky se zobrazí ve formuláři zvýrazněné a není stejný jako zaškrtnuté položky.  
  
 Zaškrtnuto pole se seznamem může mít položky přidané na pomocí návrhu **Editor kolekce řetězců** nebo může jejich položky z kolekce za běhu dynamicky přidávat pomocí <xref:System.Windows.Forms.CheckedListBox.Items%2A> vlastnost. Další informace najdete v tématu [jak: Přidání a odebrání položek z Windows Forms ComboBox, ListBox nebo CheckedListBox – ovládací prvek](add-and-remove-items-from-a-wf-combobox.md).
