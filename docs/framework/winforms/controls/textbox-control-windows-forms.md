# <a name="textbox-control-windows-forms"></a>TextBox – ovládací prvek (Windows Forms)
Windows Forms textová pole se používají k získání vstupy od uživatele nebo k zobrazení textu. `TextBox` Ovládací prvek se obecně používají pro upravitelný text, i když ji můžete také nastavit jen pro čtení. Textová pole můžete zobrazit více řádků, zalamovat text, který má velikost ovládacího prvku a přidat základní formátování. `TextBox` Ovládací prvek umožňuje jeden formát pro text zobrazuje nebo je zadaný v ovládacím prvku.
