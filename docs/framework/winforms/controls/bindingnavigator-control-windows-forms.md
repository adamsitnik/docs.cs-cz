# <a name="bindingnavigator-control-windows-forms"></a>BindingNavigator – ovládací prvek (Windows Forms)
`BindingNavigator` Ovládací prvek je navigaci a manipulaci s uživatelského rozhraní (UI) pro ovládací prvky, které jsou vázány na data. `BindingNavigator` Ovládací prvek umožňuje uživatelům procházet a manipulaci s daty ve formuláři Windows.  
  
 Témata v této části poskytují přehled o `BindingNavigator` řídit a nabízí podrobné pokyny, jak používat ovládací prvek navigace daty a přechod <xref:System.Data.DataSet>.
