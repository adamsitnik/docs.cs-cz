# <a name="fontdialog-component-windows-forms"></a>FontDialog – komponenta (Windows Forms)
Windows Forms <xref:System.Windows.Forms.FontDialog> komponenta je předem nakonfigurované dialogového okna. Jde o stejný **písmo** dialogové okno vystavené operačního systému Windows. Součást dědí z <xref:System.Windows.Forms.CommonDialog> třídy.
