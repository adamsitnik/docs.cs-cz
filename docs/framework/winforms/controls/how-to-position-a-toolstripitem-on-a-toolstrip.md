# <a name="how-to-position-a-toolstripitem-on-a-toolstrip"></a>Postupy: Umístění prvku ToolStripItem do prvku ToolStrip
Můžete přesunout nebo přidat <xref:System.Windows.Forms.ToolStripItem> levé nebo pravé straně <xref:System.Windows.Forms.ToolStrip>.  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-left-side-of-a-toolstrip"></a>Přesunutí nebo přidání prvku ToolStripItem do levé prvku ToolStrip  
  
1.  Nastavte <xref:System.Windows.Forms.ToolStripItemAlignment> vlastnost <xref:System.Windows.Forms.ToolStripItem> k <xref:System.Windows.Forms.ToolStripItemAlignment.Left>.  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-right-side-of-a-toolstrip"></a>Přesunutí nebo přidání prvku ToolStripItem do pravé straně prvku ToolStrip  
  
1.  Nastavte <xref:System.Windows.Forms.ToolStripItemAlignment> vlastnost <xref:System.Windows.Forms.ToolStripItem> k <xref:System.Windows.Forms.ToolStripItemAlignment.Right>.
