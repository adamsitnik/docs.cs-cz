# <a name="how-to-set-the-image-displayed-by-a-windows-forms-control-using-the-designer"></a>Postupy: Nastavení obrázku zobrazovaného ovládacím prvkem model Windows Forms pomocí návrháře

Obrázek může zobrazit několik ovládacích prvků model Windows Forms. Obrázek může být ikona, která vysvětluje účel ovládacího prvku, jako je například ikona disku na tlačítku, které označuje příkaz **Uložit** . Nebo může být ikona obrázek na pozadí, aby ovládací prvek měl požadovaný vzhled.

## <a name="to-set-the-image-displayed-by-a-control"></a>Nastavení obrázku zobrazovaného ovládacím prvkem

1. V okně **vlastnosti** vyberte vlastnost **Image** nebo **BackgroundImage** ovládacího prvku a potom vyberte tři tečky (![se třemi tečkami v aplikaci Visual Studio](./media/visual-studio-ellipsis-button.png)), aby se zobrazil prostředek pro **Výběr** . Dialogové okno.

2. Vyberte obrázek, který chcete zobrazit.
