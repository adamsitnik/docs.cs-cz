# <a name="splitter-control-windows-forms"></a>Rozdělovač – ovládací prvek (Windows Forms)
Ovládací `Splitter` prvky model Windows Forms slouží k změně velikosti ukotvených ovládacích prvků za běhu. Tento `Splitter` ovládací prvek se často používá ve formulářích s ovládacími prvky, které mají různou délku dat, jako je Průzkumník Windows, jejichž podokna dat obsahují informace o různých šířkách v různých časech.  
  
> [!NOTE]
> I <xref:System.Windows.Forms.SplitContainer> když nahrazuje a přidává funkce `Splitter` do kontroly nad předchozími verzemi `Splitter` , je uchována pro zpětnou kompatibilitu i pro budoucí použití, pokud zvolíte.
