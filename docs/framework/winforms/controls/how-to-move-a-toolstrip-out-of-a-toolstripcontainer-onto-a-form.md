# <a name="how-to-move-a-toolstrip-out-of-a-toolstripcontainer-onto-a-form"></a>Postupy: Přesunutí ToolStrip mimo prvek ToolStripContainer ve formuláři
Chcete-li přesunout <xref:System.Windows.Forms.ToolStrip> výstup <xref:System.Windows.Forms.ToolStripContainer> do formuláře, použijte následující postup.

## <a name="to-move-a-toolstrip-out-of-a-toolstripcontainer-onto-a-form"></a>Přesunutí ovládacího prvku ToolStrip z prvku ToolStripContainer do formuláře

1. <xref:System.Windows.Forms.ToolStrip>Vyberte.

2. Vyjměte stisknutím kombinace kláves CTRL + X nebo <xref:System.Windows.Forms.ToolStrip> kliknutím pravým tlačítkem myši a zvolením možnosti **Vyjmout** z místní nabídky. <xref:System.Windows.Forms.ToolStrip>

3. Vyberte formulář.

4. Stiskněte klávesovou zkratku CTRL + V nebo v nabídce **Upravit** vyberte možnost **Vložit.** <xref:System.Windows.Forms.ToolStrip>

5. <xref:System.Windows.Forms.ToolStrip.Dock%2A> Nastavte vlastnost <xref:System.Windows.Forms.ToolStrip> na **Top**.
