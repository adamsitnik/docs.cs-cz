# <a name="splitter-control-overview-windows-forms"></a>Přehled ovládacího prvku Rozdělovač (Windows Forms)
> [!IMPORTANT]
>  I když <xref:System.Windows.Forms.SplitContainer> nahradí a přidá funkce, které <xref:System.Windows.Forms.Splitter> ovládacího prvku z předchozích verzí <xref:System.Windows.Forms.Splitter> se zachovává kvůli zpětné kompatibilitě a budoucí použití, pokud se rozhodnete.  
  
 Windows Forms <xref:System.Windows.Forms.Splitter> ovládací prvky se používají ke změně velikosti ukotvených ovládacích prvků za běhu. <xref:System.Windows.Forms.Splitter> Ovládací prvek se často používá ve formulářích s ovládacími prvky, které mají různých délek data k prezentaci, jako je Průzkumník Windows, jejichž podokna data obsahují informace o různých šířkách v různých časech.  
  
## <a name="working-with-the-splitter-control"></a>Práce s ovládacím prvku rozdělovač  
 Když uživatel ukazuje ukazatel myši na neukotvené okrajem ovládacího prvku, který se dá změnit pomocí ovládacího prvku rozdělovač, ukazatel se změní jeho vzhled, který indikuje, že můžete změnit velikost ovládacího prvku. Pomocí ovládacího prvku rozdělovač uživatel můžete změnit velikost ukotvených ovládací prvek, který je bezprostředně před. Proto umožní uživateli, změna velikosti ukotveného ovládací prvek v době běhu, ukotvit ovládací prvek změnit velikost na okraj kontejneru a potom ukotvěte ovládací prvek splitter na jedné straně tohoto kontejneru.
