# <a name="how-to-create-a-basic-windows-forms-toolstrip-with-standard-items-using-the-designer"></a>Postupy: Vytvoření základní model Windows Forms ToolStrip se standardními položkami pomocí návrháře
Následující postup ukazuje, jak vytvořit <xref:System.Windows.Forms.ToolStrip> a přidat sedm <xref:System.Windows.Forms.ToolStripButton> ovládacích prvků, které představují typické úkoly.

## <a name="to-add-standard-items-in-the-designer"></a>Přidání standardních položek v Návrháři

1. <xref:System.Windows.Forms.ToolStrip> Vytvořte ovládací prvek.

2. V pravém horním rohu okna <xref:System.Windows.Forms.ToolStrip>klikněte na šipku inteligentního úkolu, aby se zobrazilo podokno **úlohy ToolStrip** .

3. V podokně **úlohy ToolStrip** vyberte možnost **Vložit standardní položky**.
