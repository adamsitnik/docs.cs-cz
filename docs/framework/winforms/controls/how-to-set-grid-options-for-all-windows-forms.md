# <a name="how-to-set-grid-options-for-all-windows-forms"></a>Postupy: Nastavení možností mřížky pro všechny model Windows Forms

Jak se budete používat pro práci ve vývojovém prostředí sady Visual Studio, můžete nastavit předvolby pro všechny formuláře a projekty, se kterými pracujete v Návrhář formulářů.

## <a name="set-global-windows-forms-options"></a>Nastavení možností globálních model Windows Forms

1. V aplikaci Visual Studio v nabídce **nástroje** vyberte možnost **Možnosti**.

2. V levém podokně dialogového okna **Možnosti** klikněte na **Návrhář formulářů**.

   V pravém podokně v záhlaví **nastavení rozložení** můžete nastavit výchozí nastavení mřížky pro všechny nové formuláře, které vytvoříte. Tato nastavení zahrnují velikost mřížky, zda se ovládací prvky přichytí k ní a zda je ve výchozím nastavení zapnutá. Kromě toho můžete vybrat mezi režimy rozložení **SnapToGrid** a **zarovnávacím čárám** . Další informace o zarovnávacím čárám najdete v tématu [Názorný postup: Uspořádání ovládacích prvků na model Windows Forms pomocí](walkthrough-arranging-controls-on-windows-forms-using-snaplines.md)zarovnávacím čárám.
