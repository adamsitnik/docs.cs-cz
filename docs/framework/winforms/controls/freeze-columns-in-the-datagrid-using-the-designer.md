### <a name="to-freeze-a-column-using-the-designer"></a><span data-ttu-id="c7bda-115">Chcete-li ukotvit sloupce pomocí návrháře</span><span class="sxs-lookup"><span data-stu-id="c7bda-115">To freeze a column using the designer</span></span>  
  
1.  <span data-ttu-id="c7bda-116">Klikněte na inteligentní označit piktogram (![piktogram inteligentní](./media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) v pravém horním rohu <xref:System.Windows.Forms.DataGridView> ovládací prvek a potom vyberte **upravit sloupce**.</span><span class="sxs-lookup"><span data-stu-id="c7bda-116">Click the smart tag glyph (![Smart Tag Glyph](./media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) on the upper-right corner of the <xref:System.Windows.Forms.DataGridView> control, and then select **Edit Columns**.</span></span>  
  
2.  <span data-ttu-id="c7bda-117">Vyberte sloupec **vybrané sloupce** seznamu.</span><span class="sxs-lookup"><span data-stu-id="c7bda-117">Select a column from the **Selected Columns** list.</span></span>  
  
3.  <span data-ttu-id="c7bda-118">V **vlastnosti sloupce** mřížky, nastavte <xref:System.Windows.Forms.DataGridViewColumn.Frozen%2A> vlastnost `true`.</span><span class="sxs-lookup"><span data-stu-id="c7bda-118">In the **Column Properties** grid, set the <xref:System.Windows.Forms.DataGridViewColumn.Frozen%2A> property to `true`.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="c7bda-119">Můžete také ukotvit sloupec při přidávání tak, že vyberete **Frozen** pole **přidat sloupec** dialogové okno.</span><span class="sxs-lookup"><span data-stu-id="c7bda-119">You can also freeze a column when adding it by selecting the **Frozen** box in the **Add Column** dialog box.</span></span>
