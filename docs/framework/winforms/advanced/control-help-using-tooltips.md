# <a name="control-help-using-tooltips"></a>Nápověda ovládacího prvku pomocí ToolTips
Můžete použít <xref:System.Windows.Forms.ToolTip> komponenty, chcete-li zobrazit stručný, specializované nápovědu pro jednotlivé ovládacích prvků ve Windows Forms. <xref:System.Windows.Forms.ToolTip> Komponenta obsahuje vlastnost, která určuje text zobrazený pro každého ovládacího prvku na formuláři. Další informace o práci s <xref:System.Windows.Forms.ToolTip> komponenty tímto způsobem, naleznete v tématu [jak: Nastavení ToolTips pro ovládací prvky ve formuláři Windows Forms v době návrhu](../controls/how-to-set-tooltips-for-controls-on-a-windows-form-at-design-time.md). Kromě toho můžete nakonfigurovat <xref:System.Windows.Forms.ToolTip> komponentu tak, že dochází ke zpoždění, než se zobrazí. Další informace najdete v tématu [jak: Změna zpoždění komponenty Windows Forms ToolTip](../controls/how-to-change-the-delay-of-the-windows-forms-tooltip-component.md).
