# <a name="constructing-and-drawing-curves"></a>Sestavování a kreslení křivek
Rozhraní GDI + podporuje několik typů křivek: symbol tří teček, elipsy, základní křivky vyhlazení a Bézierovy křivky. Elipsa je definován ve své ohraničující obdélník; oblouku je část určené počáteční úhel a úhel oblouku elipsy. Pole bodů a parametr napětí je definován křivky mohutnosti – křivka plynule prochází každý bod v poli, a parametr napětí ovlivňuje způsob, jakým zatáčkách křivky. Bézierovy křivky je definován tak, že dva koncové body a dva ovládací prvek body, které neprochází přes kontrolních bodů křivky, ale kontrolní body ovlivnit směru a ohybem jako křivku přechází z jednoho koncového bodu na druhý.
