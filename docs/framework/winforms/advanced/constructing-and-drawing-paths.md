# <a name="constructing-and-drawing-paths"></a>Sestavování a kreslení cest
Cesta je posloupnost primitivy grafiky (řádky, obdélníky, křivek, text a podobně), které mohou zpracovat a vykreslit jako jeden celek. Cesta je možné rozdělit do *obrázky* , které jsou otevřeno nebo Uzavřeno. Obrázek může obsahovat několik primitiv.  
  
 Cestu můžete nakreslit pomocí volání <xref:System.Drawing.Graphics.DrawPath%2A> metodu <xref:System.Drawing.Graphics> třídy kde můžete zadat cestu voláním <xref:System.Drawing.Graphics.FillPath%2A> metodu <xref:System.Drawing.Graphics> třídy.
