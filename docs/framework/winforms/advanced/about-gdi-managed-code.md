# <a name="about-gdi-managed-code"></a>Informace o spravovaném kódu GDI+

GDI+ je část operačního systému Windows, která poskytuje dvourozměrnou vektorovou grafiku, obrázky a typografii. Rozhraní GDI+ vylepšuje rozhraní GDI (GDI je součástí dřívějších verzí Windows) přidáním nových funkcí a optimalizací stávajících funkcí.

Rozhraní spravované třídy GDI+ (sada obálek) je součástí .NET Framework, prostředí pro sestavování, nasazování a spouštění webových služeb XML a dalších aplikací.

Tato část poskytuje informace o rozhraní GDI+ API pro programátory pomocí spravovaného kódu.
