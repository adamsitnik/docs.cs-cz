## <a name="implementation-guidelines-and-conventions"></a><span data-ttu-id="74fb6-109">Pokyny pro implementaci a konvence</span><span class="sxs-lookup"><span data-stu-id="74fb6-109">Implementation Guidelines and Conventions</span></span>

-   <span data-ttu-id="74fb6-111">Podpora pro tuto – vzor ovládacích prvků není omezeno pouze na objekty v klientských počítačích.</span><span class="sxs-lookup"><span data-stu-id="74fb6-111">Support for this control pattern is not limited to objects on the desktop.</span></span> <span data-ttu-id="74fb6-112">Tento vzor ovládacích prvků musí také podporovat podřízené objekty daného objektu kontejneru Pokud podřízené objekty lze přesunout, se změněnou velikostí ani otočen volně v rámci hranice kontejneru.</span><span class="sxs-lookup"><span data-stu-id="74fb6-112">This control pattern must also be supported by the children of a container object if the children can be moved, resized, or rotated freely within the boundaries of the container.</span></span>

-   <span data-ttu-id="74fb6-116">Všechny parametry a hodnoty vlastností jsou absolutní a nezávislý na národním prostředí.</span><span class="sxs-lookup"><span data-stu-id="74fb6-116">All parameters and property values are absolute and independent of locale.</span></span>
