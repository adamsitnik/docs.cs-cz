## <a name="service"></a><span data-ttu-id="ebd38-108">Služba</span><span class="sxs-lookup"><span data-stu-id="ebd38-108">Service</span></span>  
 <span data-ttu-id="ebd38-109">Tento projekt obsahuje službu kalkulačky v místním prostředí.</span><span class="sxs-lookup"><span data-stu-id="ebd38-109">This project contains a self-hosted calculator service.</span></span> <span data-ttu-id="ebd38-110">V `Main` metody vytvoření hostitele služby a koncového bodu služby se přidá do ní.</span><span class="sxs-lookup"><span data-stu-id="ebd38-110">In the `Main` method, a service host is created and a service endpoint is added to it.</span></span> <span data-ttu-id="ebd38-111">Další, <xref:System.ServiceModel.Discovery.ServiceDiscoveryBehavior> se vytvoří.</span><span class="sxs-lookup"><span data-stu-id="ebd38-111">Next, a <xref:System.ServiceModel.Discovery.ServiceDiscoveryBehavior> is created.</span></span> <span data-ttu-id="ebd38-112">Pokud chcete povolit oznámení, musí koncový bod oznámení přidaný do <xref:System.ServiceModel.Discovery.ServiceDiscoveryBehavior>.</span><span class="sxs-lookup"><span data-stu-id="ebd38-112">To enable announcements, an announcement endpoint must be added to the <xref:System.ServiceModel.Discovery.ServiceDiscoveryBehavior>.</span></span> <span data-ttu-id="ebd38-113">V tomto případě je standardní koncový bod, používat UDP, vícesměrového vysílání se přidá jako koncový bod oznámení.</span><span class="sxs-lookup"><span data-stu-id="ebd38-113">In this case a standard endpoint, using UDP multicast is added as the announcement endpoint.</span></span> <span data-ttu-id="ebd38-114">Toto oznámení vysílá prostřednictvím dobře známé adrese UDP.</span><span class="sxs-lookup"><span data-stu-id="ebd38-114">This broadcasts the announcements over a well-known UDP address.</span></span>  
  
```csharp
Uri baseAddress = new Uri("http://localhost:8000/" + Guid.NewGuid().ToString());  
  
// Create a ServiceHost for the CalculatorService type.  
using (ServiceHost serviceHost = new ServiceHost(typeof(CalculatorService), baseAddress))  
{  
     serviceHost.AddServiceEndpoint(typeof(ICalculatorService), new WSHttpBinding(), String.Empty);  
  
     ServiceDiscoveryBehavior serviceDiscoveryBehavior = new ServiceDiscoveryBehavior();  
  
     // Announce the availability of the service over UDP multicast  
    serviceDiscoveryBehavior.AnnouncementEndpoints.Add(new UdpAnnouncementEndpoint());  
  
    // Make the service discoverable over UDP multicast.  
    serviceHost.Description.Behaviors.Add(serviceDiscoveryBehavior);                  
    serviceHost.AddServiceEndpoint(new UdpDiscoveryEndpoint());  
    serviceHost.Open();  
    // ...  
}  
```

```csharp
// Subscribe the announcement events

announcementService.OfflineAnnouncementReceived += OnOfflineEvent;  
```
