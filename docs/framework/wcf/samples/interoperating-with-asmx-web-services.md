# <a name="interoperating-with-asmx-web-services"></a><span data-ttu-id="4cbd5-102">Spolupráce s webovými službami ASMX</span><span class="sxs-lookup"><span data-stu-id="4cbd5-102">Interoperating with ASMX Web Services</span></span>
<span data-ttu-id="4cbd5-103">Tato ukázka předvádí, jak integrovat klientskou aplikaci Windows Communication Foundation (WCF) s existující webovou službou ASMX.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-103">This sample demonstrates how to integrate a Windows Communication Foundation (WCF) client application with an existing ASMX Web service.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="4cbd5-104">Postup nastavení a pokyny pro sestavení pro tuto ukázku najdete na konci tohoto tématu.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-104">The setup procedure and build instructions for this sample are located at the end of this topic.</span></span>  
  
 <span data-ttu-id="4cbd5-105">Tato ukázka se skládá z programu klientské konzoly (. exe) a knihovny služeb (. dll) hostované službou Internetová informační služba (IIS).</span><span class="sxs-lookup"><span data-stu-id="4cbd5-105">This sample consists of a client console program (.exe) and a service library (.dll) hosted by Internet Information Services (IIS).</span></span> <span data-ttu-id="4cbd5-106">Služba je webová služba ASMX, která implementuje kontrakt definující způsob komunikace požadavek-odpověď.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-106">The service is an ASMX Web Service that implements a contract that defines a request-reply communication pattern.</span></span> <span data-ttu-id="4cbd5-107">Služba zpřístupňuje matematické operace (`Add`, `Subtract`, `Multiply` `Divide`a).</span><span class="sxs-lookup"><span data-stu-id="4cbd5-107">The service exposes math operations (`Add`, `Subtract`, `Multiply`, and `Divide`).</span></span> <span data-ttu-id="4cbd5-108">Klient provádí synchronní požadavky na matematickou operaci a ve výsledku odpovídá službě.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-108">The client makes synchronous requests to a math operation and the service replies with the result.</span></span> <span data-ttu-id="4cbd5-109">Aktivita klienta se zobrazí v okně konzoly.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-109">Client activity is visible in the console window.</span></span>  
  
 <span data-ttu-id="4cbd5-110">Implementace webové služby ASMX zobrazená v následujícím ukázkovém kódu vypočítá a vrátí příslušný výsledek.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-110">The ASMX Web service implementation shown in the following sample code calculates and returns the appropriate result.</span></span>  
  
```csharp  
[WebService(Namespace="http://Microsoft.ServiceModel.Samples")]  
public class CalculatorService : System.Web.Services.WebService  
    {  
        [WebMethod]  
        public double Add(double n1, double n2)  
        {  
            return n1 + n2;  
        }  
        [WebMethod]  
        public double Subtract(double n1, double n2)  
        {  
            return n1 - n2;  
        }  
        [WebMethod]  
        public double Multiply(double n1, double n2)  
        {  
            return n1 * n2;  
        }  
        [WebMethod]  
        public double Divide(double n1, double n2)  
        {  
            return n1 / n2;  
        }  
    }  
```  
  
 <span data-ttu-id="4cbd5-111">Jak je nakonfigurováno, může být služba k `http://localhost/servicemodelsamples/service.asmx` dispozici klientovi ve stejném počítači.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-111">As configured, the service can be accessed at `http://localhost/servicemodelsamples/service.asmx` by a client on the same machine.</span></span> <span data-ttu-id="4cbd5-112">Aby mohli klienti na vzdálených počítačích přistupovat ke službě, je třeba zadat kvalifikovaný název domény namísto názvu localhost.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-112">For clients on remote machines to access the service, a qualified domain name must be specified instead of localhost.</span></span>  
  
 <span data-ttu-id="4cbd5-113">Komunikace se provádí prostřednictvím klienta generovaného nástrojem pro dodávání [metadat ServiceModel (Svcutil. exe)](../../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md).</span><span class="sxs-lookup"><span data-stu-id="4cbd5-113">Communication is done through a client generated by the [ServiceModel Metadata Utility Tool (Svcutil.exe)](../../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md).</span></span> <span data-ttu-id="4cbd5-114">Klient nástroje je obsažen v souboru generatedClient.cs.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-114">The client is contained in the file generatedClient.cs.</span></span> <span data-ttu-id="4cbd5-115">Služba ASMX musí být k dispozici, aby vygenerovala kód proxy, protože se používá k načtení aktualizovaných metadat.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-115">The ASMX service must be available to generate the proxy code, because it is used to retrieve the updated metadata.</span></span> <span data-ttu-id="4cbd5-116">Spusťte následující příkaz z příkazového řádku v adresáři klienta pro vygenerování typovaného proxy serveru.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-116">Run the following command from a command prompt in the client directory to generate the typed proxy.</span></span>  
  
```  
svcutil.exe /n:http://Microsoft.ServiceModel.Samples,Microsoft.ServiceModel.Samples http://localhost/servicemodelsamples/service.svc?wsdl /out:generatedClient.cs  
```  
  
 <span data-ttu-id="4cbd5-117">Pomocí vygenerovaného klienta můžete získat přístup ke koncovému bodu služby nakonfigurováním příslušné adresy a vazby.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-117">By using the generated client, you can access a service endpoint by configuring the appropriate address and binding.</span></span> <span data-ttu-id="4cbd5-118">Podobně jako služba Klient používá konfigurační soubor (App. config) k určení koncového bodu ke komunikaci.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-118">Like the service, the client uses a configuration file (App.config) to specify the endpoint to communicate with.</span></span> <span data-ttu-id="4cbd5-119">Konfigurace koncového bodu klienta se skládá z absolutní adresy koncového bodu služby, vazby a kontraktu, jak je znázorněno v následující ukázkové konfiguraci.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-119">The client endpoint configuration consists of an absolute address for the service endpoint, the binding, and the contract, as shown in the following sample configuration.</span></span>  
  
```xml  
<client>  
   <endpoint   
      address="http://localhost/ServiceModelSamples/service.asmx"   
      binding="basicHttpBinding"   
      contract="Microsoft.ServiceModel.Samples.CalculatorServiceSoap" />  
</client>  
```  
  
 <span data-ttu-id="4cbd5-120">Implementace klienta vytvoří instanci vygenerovaného klienta.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-120">The client implementation constructs an instance of the generated client.</span></span> <span data-ttu-id="4cbd5-121">Vygenerovaný klient se pak může použít ke komunikaci se službou.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-121">The generated client can then be used to communicate with the service.</span></span>  
  
```csharp  
// Create a client.  
CalculatorServiceSoapClient client = new CalculatorServiceSoapClient();  
  
// Call the Add service operation.  
double value1 = 100.00D;  
double value2 = 15.99D;  
double result = client.Add(value1, value2);  
Console.WriteLine("Add({0},{1}) = {2}", value1, value2, result);  
  
// Call the Subtract service operation.  
value1 = 145.00D;  
value2 = 76.54D;  
result = client.Subtract(value1, value2);  
Console.WriteLine("Subtract({0},{1}) = {2}", value1, value2, result);  
  
// Call the Multiply service operation.  
value1 = 9.00D;  
value2 = 81.25D;  
result = client.Multiply(value1, value2);  
Console.WriteLine("Multiply({0},{1}) = {2}", value1, value2, result);  
  
// Call the Divide service operation.  
value1 = 22.00D;  
value2 = 7.00D;  
result = client.Divide(value1, value2);  
Console.WriteLine("Divide({0},{1}) = {2}", value1, value2, result);  
  
//Closing the client gracefully closes the connection and cleans up resources.  
client.Close();  
  
Console.WriteLine();  
Console.WriteLine("Press <ENTER> to terminate client.");  
Console.ReadLine();  
```  
  
 <span data-ttu-id="4cbd5-122">Při spuštění ukázky se v okně konzoly klienta zobrazí požadavky na operace a odpovědi.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-122">When you run the sample, the operation requests and responses are displayed in the client console window.</span></span> <span data-ttu-id="4cbd5-123">V okně klienta stiskněte klávesu ENTER pro vypnutí klienta.</span><span class="sxs-lookup"><span data-stu-id="4cbd5-123">Press ENTER in the client window to shut down the client.</span></span>  
  
