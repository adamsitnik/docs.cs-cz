Tato část obsahuje ukázky, které předvádějí vlastní vazby Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>V tomto oddílu  
 <xref:System.ServiceModel.NetHttpBinding>  
 Ukazuje, jak implementovat vazbu, která zásobníků <xref:System.ServiceModel.Channels.HttpTransportBindingElement> nebo <xref:System.ServiceModel.Channels.HttpsTransportBindingElement> nahoře <xref:System.ServiceModel.Channels.BinaryMessageEncodingBindingElement>.  
  
 [WSStreamedHttpBinding](wsstreamedhttpbinding.md)  
 Ukazuje, jak vytvořit vazbu, která je navržena pro podporu streamování scénáře, když se používá přenos pomocí protokolu HTTP.
