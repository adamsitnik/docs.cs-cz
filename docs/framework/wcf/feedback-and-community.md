# <a name="feedback-and-community"></a>Názory a komunita
Děkujeme za vaše komentáře a pochybnostmi k dokumentaci k Windows Communication Foundation (WCF) nebo služba CardSpace. V dolní části každé stránky v sadě SDK je větu "Odeslat komentáře v tomto tématu společnosti Microsoft." Klikněte na odkaz "komentář" k zobrazení nebo návrhy týkající se konkrétního tématu.  
  
 Pokud hledáte další informace nebo chcete sdílet s jinými uživateli, kteří mají zájem o WCF techniky, zkuste [fórum MSDN Windows Communication Foundation](https://go.microsoft.com/fwlink/?LinkId=186463).
