# <a name="wcf-system-requirements"></a>WCF – požadavky na systém

Windows Communication Foundation (WCF) je komunikační infrastruktura, která se používá k vytváření distribuovaných aplikací. Následující seznam obsahuje požadavky, které umožňují službě WCF běžet.

## <a name="system-requirements"></a>Požadavky na systém

Služba WCF je nainstalována ve výchozím [!INCLUDE[wv](../../../includes/wv-md.md)]nastavení v systému.

WCF je také možné nainstalovat na [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]nebo [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].

> [!NOTE]
> Funkce služby Řízení front zpráv (MSMQ) WCF je podporována pouze v [!INCLUDE[wv](../../../includes/wv-md.md)]systémech [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)], [!INCLUDE[wxppro](../../../includes/wxppro-md.md)]a.
