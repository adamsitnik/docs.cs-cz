# <a name="wcf-system-requirements"></a>WCF – požadavky na systém

Windows Communication Foundation (WCF) je komunikační infrastruktura, která se používá k vytváření distribuovaných aplikací. Následující seznam obsahuje požadavky, které umožňují službě WCF běžet.

## <a name="system-requirements"></a>Požadavky na systém

Ve výchozím nastavení je služba WCF nainstalována v [!INCLUDE[wv](../../../includes/wv-md.md)].

WCF je také možné nainstalovat na [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], Windows Server 2003 R2 nebo [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].

> [!NOTE]
> Funkce řízení front zpráv (MSMQ) WCF je podporovaná jenom na [!INCLUDE[wv](../../../includes/wv-md.md)], Windows serveru 2003 R2, [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] a Windows XP Professional.
