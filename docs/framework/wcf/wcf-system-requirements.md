# <a name="wcf-system-requirements"></a>WCF – požadavky na systém
Windows Communication Foundation (WCF) se komunikační infrastruktura, která se používá k vytvoření distribuované aplikace. Následuje seznam požadavků, které umožňují WCF ke spuštění.  
  
## <a name="system-requirements"></a>Požadavky na systém  
 WCF je ve výchozím nastavení nainstalované na [!INCLUDE[wv](../../../includes/wv-md.md)].  
  
 WCF je možné nainstalovat i na [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], nebo [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].  
  
 **Poznámka:** The služby Řízení front zpráv (MSMQ) funkce technologie WCF je podporován pouze v [!INCLUDE[wv](../../../includes/wv-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)], a [!INCLUDE[wxppro](../../../includes/wxppro-md.md)].
