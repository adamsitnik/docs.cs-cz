# <a name="extending-bindings"></a>Rozšiřování vazeb
Vazby určují přenos, kódování a protokol vyžadovaný pro připojení ke koncovému bodu. Rozšíření vazby a vlastní vazby implementují vlastní komunikační funkce vyžadované pro podporu funkcí aplikace.
