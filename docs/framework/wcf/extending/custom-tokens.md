# <a name="custom-tokens"></a>Vlastní tokeny
I když Windows Communication Foundation (WCF) podporuje nativně certifikáty X.509, tokeny kontextu zabezpečení, lístky protokolu Kerberos a název tokeny uživatele jako přihlašovací údaje, je dostatečně flexibilní, aby bylo možné používat vlastní přihlašovací údaje.
