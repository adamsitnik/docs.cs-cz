# <a name="developing-service-oriented-applications-with-wcf"></a>Vývoj aplikací orientovaných na služby s použitím technologie WCF
Tato část dokumentace obsahuje informace o Windows Communication Foundation (WCF), což je jednotný programovací model pro vytváření aplikací orientovaných na služby. Umožňuje vývojářům vytvářet zabezpečená, spolehlivá a počet řešení, které se integrují napříč platformami vzájemnou spolupráci se službami stávajících investic.
