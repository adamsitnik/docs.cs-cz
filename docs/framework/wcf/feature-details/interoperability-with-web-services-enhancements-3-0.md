# <a name="interoperability-with-web-services-enhancements-30"></a>Interoperabilita s Web Services Enhancements 3.0
Klienty Windows Communication Foundation (WCF) a služby jsou přenosový kompatibilní s 3.0 rozšíření webové služby pro klienty Microsoft .NET (WSE) a služby, když klienti WCF a služeb jsou nakonfigurovány pro použití verzi ze srpna 2004 Specifikace WS-Addressing.
