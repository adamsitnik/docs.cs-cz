# <a name="partial-trust"></a>Částečná důvěryhodnost
Počínaje [!INCLUDE[netfx35_long](../../../../includes/netfx35-long-md.md)]částečně důvěryhodné volající může přístup k veřejné typy a metody implementované v <xref:System.ServiceModel>, <xref:System.Runtime.Serialization>, a <xref:System.ServiceModel.Web>. Tato část popisuje podporované scénáře použití služby Windows Communication Foundation (WCF) v rámci částečně důvěryhodné aplikace, stejně jako omezenou podmnožinou WCF funkce, které jsou k dispozici pro aplikace spuštěné s zabezpečení přístupu ke snížení kódu (CAS) oprávnění.
