## <a name="untyped-message-programming"></a><span data-ttu-id="fc8d9-115">Programování bez typu zprávy</span><span class="sxs-lookup"><span data-stu-id="fc8d9-115">Untyped Message Programming</span></span>  
 <span data-ttu-id="fc8d9-116">Při práci přímo s objekty netypové zpráv, musíte explicitně nastavit vlastnosti netypové zprávy ho serializovat jako JSON.</span><span class="sxs-lookup"><span data-stu-id="fc8d9-116">When working directly with untyped Message objects, you must explicitly set the properties on the untyped message to serialize it as JSON.</span></span> <span data-ttu-id="fc8d9-117">Následující fragment kódu ukazuje, jak to provést.</span><span class="sxs-lookup"><span data-stu-id="fc8d9-117">The following code snippet shows how to do this.</span></span>  
  
```  
 Message response = Message.CreateMessage(  
                  MessageVersion.None,    // No SOAP message version  
                             "*",                     // SOAP action, ignored since this is JSON  
                             "Response string: JSON format specified", // Message body  
                             new DataContractJsonSerializer(typeof(string))); // Specify DataContractJsonSerializer  
      response.Properties.Add( WebBodyFormatMessageProperty.Name,   
                    new WebBodyFormatMessageProperty(WebContentFormat.Json)); // Use JSON format  
```
