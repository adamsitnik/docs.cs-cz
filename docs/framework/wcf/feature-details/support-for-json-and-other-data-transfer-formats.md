# <a name="support-for-json-and-other-data-transfer-formats"></a>Podpora formátu JSON a dalších formátů přenosu dat
JSON (JavaScript Object Notation) je formát kódování efektivní data, která umožňuje rychlé výměny malé množství dat mezi prohlížeči klientů a AJAX (asynchronní JavaScript a XML)-povolené webové služby.

 [Postupy: Serializace a deserializace dat protokolu JSON](../../../../docs/framework/wcf/feature-details/how-to-serialize-and-deserialize-json-data.md)  
 Ukazuje, jak serializovat objekty typu .NET na data zakódovaná ve formátu JSON a potom deserializovat data ve formátu JSON zpět do instance typů .NET pomocí <xref:System.Runtime.Serialization.Json.DataContractJsonSerializer>.  
  
 [Mapování mezi JSON a XML](../../../../docs/framework/wcf/feature-details/mapping-between-json-and-xml.md)  
 Popisuje souvztažnost mezi kódování zápis JSON (JavaScript Object) a informační sadu XML, který se používá interně ve Windows Communication Foundation (WCF) představující data zakódovaná ve formátu JSON a dokumenty.
