# <a name="wcf-discovery"></a>Zjišťování WCF
Windows Communication Foundation (WCF) poskytuje podporu pro povolení služby být zjistitelné za běhu pomocí protokolu WS-Discovery interoperabilní způsobem. Služby WCF můžete oznamujeme jejich dostupnosti, síť pomocí vícesměrového vysílání zprávy nebo na proxy serveru zjišťování. Klientské aplikace můžete vyhledávat v síti nebo server proxy zjišťování k vyhledání služby, které splňují sadu kritérií. Témata v této části poskytují přehled a programovací model pro tuto funkci podrobně popisují.
