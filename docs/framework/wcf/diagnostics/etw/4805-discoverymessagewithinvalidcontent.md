|||  
|-|-|  
|ID|4805|  
|klíčová slova|Zjišťování|  
|úroveň|Upozornění|  
|Kanál|Aplikace Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Tato událost je vygenerován při zjišťování zpráva obsahuje neplatný obsah.  
  
## <a name="message"></a>Zpráva  
 Zpráva %1 s parametrem messageId = %2 byla zahozena, protože její obsah byl neplatný.
