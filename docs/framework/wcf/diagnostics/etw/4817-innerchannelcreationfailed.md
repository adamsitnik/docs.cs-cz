# <a name="4817---innerchannelcreationfailed"></a>4817 – InnerChannelCreationFailed
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|4817|  
|klíčová slova|Zjišťování|  
|úroveň|Upozornění|  
|Kanál|Aplikace Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Tato událost je vygenerován při objektu DiscoveryClientChannel se nepodařilo vytvořit kanál se zjistitelným koncovým. Objektu DiscoveryClientChannel se pokusí použít další dostupné nalezený koncový bod.  
  
## <a name="message"></a>Zpráva  
 Objektu DiscoveryClientChannel se nepodařilo vytvořit kanál se zjistitelným koncovým s parametry EndpointAddress = '%1' a Via = '%2'. Objektu DiscoveryClientChannel se pokusí použít další dostupné nalezený koncový bod.
