## <a name="description"></a>Popis  
 Tato událost je vygenerován, pokud objekt pro vytváření kanálů byl stará z mezipaměti, protože mezipaměť dosáhla svého limitu.  
  
## <a name="message"></a>Zpráva  
 Objekt pro vytváření kanálů byl z mezipaměti zastaralá, protože mezipaměť dosáhla svého limitu '%1'.
