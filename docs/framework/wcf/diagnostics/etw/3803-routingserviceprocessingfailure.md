|úroveň|Chyba|

## <a name="description"></a>Popis  
 Tato událost je vygenerován při směrovací služba skončila chybou při zpracování zprávy v koncovém bodě.  
  
## <a name="message"></a>Zpráva  
 Směrovací službě se nepodařilo zpracovat zprávu na koncovém bodu s adresou '%1'.
