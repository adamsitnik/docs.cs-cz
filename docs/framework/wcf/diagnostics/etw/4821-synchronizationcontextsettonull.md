# <a name="4821---synchronizationcontextsettonull"></a>4821 - SynchronizationContextSetToNull
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|4821|  
|klíčová slova|Zjišťování|  
|úroveň|Informace o|  
|Kanál|Aplikace Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Tato událost je vygenerován, když třída SynchronizationContext byla nastavena na hodnotu null objektem DiscoveryClientChannel před inicializací operace Find.  
  
## <a name="message"></a>Zpráva  
 Třída SynchronizationContext byla nastavena na hodnotu null objektem DiscoveryClientChannel před inicializací operace Find.
