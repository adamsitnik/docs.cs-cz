# <a name="4812---discoverymessagewithnullreplyto"></a>4812 – DiscoveryMessageWithNullReplyTo
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|4812|  
|klíčová slova|Zjišťování|  
|úroveň|Upozornění|  
|Kanál|Aplikace Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Tato událost je vygenerován při zjišťování zpráva byla objektem DiscoveryClient zahozena, protože neobsahovala adresu ReplyTo.  
  
## <a name="message"></a>Zpráva  
 Zpráva požadavku na zjišťování s parametrem messageId = %1 byla zahozena, protože neobsahovala adresu ReplyTo.
