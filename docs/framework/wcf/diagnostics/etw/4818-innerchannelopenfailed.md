# <a name="4818---innerchannelopenfailed"></a>4818 – InnerChannelOpenFailed
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|4818|  
|klíčová slova|Zjišťování|  
|úroveň|Upozornění|  
|Kanál|Aplikace Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Tato událost je vygenerován při objektu DiscoveryClientChannel se nepodařilo otevřít kanál se zjistitelným koncovým. Objektu DiscoveryClientChannel se pokusí použít další dostupné nalezený koncový bod.  
  
## <a name="message"></a>Zpráva  
 Objektu DiscoveryClientChannel se nepodařilo otevřít kanál se zjistitelným koncovým s parametry EndpointAddress = '%1' a Via = '%2'. Objektu DiscoveryClientChannel se pokusí použít další dostupné nalezený koncový bod.
