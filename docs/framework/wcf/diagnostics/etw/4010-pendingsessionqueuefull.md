## <a name="description"></a>Popis  
 Tato událost je vygenerován při odeslání relace se nezdařila, protože fronta čekající relací je plná.  
  
## <a name="message"></a>Zpráva  
 Zpracování relace pro '%1' se nezdařilo, protože fronta nedokončených relací je plná a obsahuje '%2' nedokončených položek.
