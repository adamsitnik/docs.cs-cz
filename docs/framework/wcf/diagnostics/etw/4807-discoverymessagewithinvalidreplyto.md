# <a name="4807---discoverymessagewithinvalidreplyto"></a>4807 – DiscoveryMessageWithInvalidReplyTo
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|4807|  
|klíčová slova|Zjišťování|  
|úroveň|Upozornění|  
|Kanál|Aplikace Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Tato událost je vygenerován při zjišťování zpráva byla zahozena, protože obsahovala neplatnou adresu ReplyTo.  
  
## <a name="message"></a>Zpráva  
 Zpráva požadavku na zjišťování s parametrem messageId = %1 byla zahozena, protože obsahovala neplatnou adresu ReplyTo.
