# <a name="debugging-on-the-client"></a>Ladění na klientovi
Chcete-li usnadnit uživatelům psaní klientských aplikací pro služby WCF, můžete přidat [ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) služeb chování do konfiguračního souboru služby. Toto chování slouží k publikování stránek nápovědy a vrátí informace o spravované výjimce v podrobností chyb SOAP vrácena klientovi.
