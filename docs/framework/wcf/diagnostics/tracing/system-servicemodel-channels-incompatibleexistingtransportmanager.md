# <a name="systemservicemodelchannelsincompatibleexistingtransportmanager"></a>System.ServiceModel.Channels.IncompatibleExistingTransportManager
Pro zadaný identifikátor URI byl nalezen stávající nekompatibilní správce přenosu.  
  
## <a name="description"></a>Popis  
 Tento informační trasování označuje, že pro naslouchací proces se otevře, ale není kompatibilní stávající správce přenosu nebyl nalezen odpovídající registraci. Naslouchací proces se pokusí použít další správce přenosu a vyvolání výjimky, pokud není úspěšné.
