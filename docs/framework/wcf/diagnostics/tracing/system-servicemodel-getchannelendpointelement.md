# <a name="systemservicemodelgetchannelendpointelement"></a>System.ServiceModel.GetChannelEndpointElement
System.ServiceModel.GetChannelEndpointElement  
  
## <a name="description"></a>Popis  
 Trasování je vygenerován při vytváření objektu pro vytváření kanálů, na úrovni informace. Poskytuje popis koncového bodu, který klient se mluví (například Vzdálená adresa, vazba, název smlouvy).
