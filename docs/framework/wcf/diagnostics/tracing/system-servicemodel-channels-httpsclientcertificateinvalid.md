# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid
Klientský certifikát je neplatný.  
  
## <a name="description"></a>Popis  
 Trasování označuje, že certifikát dodaný klientem byla nalezena neplatná naslouchací proces HTTPS. Naslouchací proces HTTPS k pokusu o ověření pravosti klienta pomocí tohoto certifikátu. Certifikát může být neplatný, pokud jeho certifikační autorita není rozpoznána serverem, který je hostitelem služby.
