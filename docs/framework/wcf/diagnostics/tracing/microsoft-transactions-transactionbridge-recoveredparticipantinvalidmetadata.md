# <a name="microsofttransactionstransactionbridgerecoveredparticipantinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RecoveredParticipantInvalidMetadata
Reference koncového bodu nalezená v položce protokolu obnovení účastníka obsahovala neplatná nebo neslučitelná data.  
  
## <a name="description"></a>Popis  
 Systém nemůže vytvořit proxy pro obnovené zapsání účastníka.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Pokud se problém opakuje, obraťte se na Microsoft.
