# <a name="systemservicemodelchannelsfailedpipeconnect"></a>System.ServiceModel.Channels.FailedPipeConnect
Pokus o připojení ke koncovému bodu pojmenovaného kanálu se nepovedlo. Další pokus se uskuteční v rámci určeného časového limitu.  
  
## <a name="description"></a>Popis  
 Tento informační trasování indikuje selhání připojení ke koncovému bodu pojmenovaného kanálu. Může k tomu dojít, pokud koncovému bodu pojmenovaného kanálu nebyl nalezen nebo je zaneprázdněný. Další pokusy, oddělených krátkém čase, dokud jeden neuspěje nebo OpenTimeout vyprší platnost.
