# <a name="systemservicemodeltxcompletionstatuscompletedforsetcomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForSetComplete
Zadanou transakci pro zadanou operaci byla dokončena kvůli volání na SetTransactionComplete.  
  
## <a name="description"></a>Popis  
 Trasovaná, pokud uživatel se rozhodl explicitně dokončete aktuální transakci prostřednictvím OperationContext.
