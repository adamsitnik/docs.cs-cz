# <a name="microsofttransactionstransactionbridgeregistercoordinator"></a>Microsoft.Transactions.TransactionBridge.RegisterCoordinator
Služba protokolu WS-Atomic Transactions úspěšně registrovala se svým koordinátorem pro kontrolní protokol.  
  
## <a name="description"></a>Popis  
 Trasovaná při místní TransactionManager úspěšně zaregistrovala s jeho nadřízeného TransactionManage
