# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>Popis  
 Byla přijata zpráva odpovědi, ale neexistují žádné požadavky čekající na tuto zprávu. Zpráva se vyřazuje modulem runtime Windows Communication Foundation (WCF).  
  
 Trasování můžete použít pro ladění ztracené zprávy.
