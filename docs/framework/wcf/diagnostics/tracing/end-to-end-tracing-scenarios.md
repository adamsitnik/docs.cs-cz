# <a name="end-to-end-tracing-scenarios"></a>Scénáře komplexního trasování
Tento oddíl obsahuje témata, která popisují různé scénáře použití trasování.  
  
 Trasování aktivit je implementován pro odpovědi nebo odpovědi, OneWay a duplexní scénáře následujícím způsobem  
  
-   Synchronní nebo asynchronní  
  
-   BasicHttpBinding, netNamedPipeBinding, netTcpBinding, wsHttpBinding or netMsmqBinding  
  
-   Služby modelu COM  
  
-   Šíření id aktivity uživatele
