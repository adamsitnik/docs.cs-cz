# <a name="systemservicemodeltxcompletionstatuscompletedforasyncabort"></a>System.ServiceModel.TxCompletionStatusCompletedForAsyncAbort
Zadanou transakci pro zadanou operaci byla dokončena kvůli asynchronnímu přerušení.  
  
## <a name="description"></a>Popis  
 Aktuální transakce byla přerušena z důvodu jiné účastníka hlasování pro přerušení, vypršení časových limitů, ke kterým dochází, nebo jiná chyba uvnitř účastník transakce.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Pokud toto přerušení neočekávaná, zkontrolujte všechny protokoly systému a zjistěte jeho důvod skutečné pro přerušení.
