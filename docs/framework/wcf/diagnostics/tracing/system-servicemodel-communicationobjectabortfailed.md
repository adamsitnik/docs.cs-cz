# <a name="systemservicemodelcommunicationobjectabortfailed"></a>System.ServiceModel.CommunicationObjectAbortFailed
Objekt komunikace se nepodařilo zrušit.  
  
## <a name="description"></a>Popis  
 Objekt komunikace nepovedlo se přerušit, protože došlo k výjimce během provádění metody její přerušení. Další okolní trasování může obsahovat podrobnosti o výjimce.
