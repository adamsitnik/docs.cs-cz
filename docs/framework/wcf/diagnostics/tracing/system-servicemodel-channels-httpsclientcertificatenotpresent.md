# <a name="systemservicemodelchannelshttpsclientcertificatenotpresent"></a>System.ServiceModel.Channels.HttpsClientCertificateNotPresent
Je požadován klientský certifikát. V požadavku nebyl nalezen žádný certifikát.  
  
## <a name="description"></a>Popis  
 Trasování označuje, že naslouchací proces HTTPS přijala požadavek HTTPS, který se nepřidružil k klientský certifikát. Protože naslouchací proces byl nakonfigurovaný k vyžadování klientské certifikáty pro všechny požadavky HTTPS, naslouchací proces se nepovedlo ověřit pravost klienta.
