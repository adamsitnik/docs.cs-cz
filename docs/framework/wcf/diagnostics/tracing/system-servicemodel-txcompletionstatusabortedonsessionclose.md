# <a name="systemservicemodeltxcompletionstatusabortedonsessionclose"></a>System.ServiceModel.TxCompletionStatusAbortedOnSessionClose
Zadaný transakce byla přerušena, protože při zavření relace, a atribut TransactionAutoCompleteOnSessionClose OperationBehaviorAttribute byl nastaven na hodnotu false.  
  
## <a name="description"></a>Popis  
 Sledovat aktuální aktivní relace byla ukončena a transakce nebyla dokončena a TransactionAutoCompleteOnSessionClose nastavená na `false`.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Trasování označuje potenciální chyby aplikace, které by mělo být vypátráno.
