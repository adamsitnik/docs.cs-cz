# <a name="microsofttransactionstransactionbridgevolatileparticipantindoubt"></a>Microsoft.Transactions.TransactionBridge.VolatileParticipantInDoubt
Služba protokolu WS-AT obdržela zprávu připravit nebo odpovědět od rozpoznaného nestálého účastníka. Chyba byla vrácena účastníkovi, deklaruje, že výsledek transakce je zpochybněn.  
  
## <a name="description"></a>Popis  
 Trasovaná při místní správce transakcí přijme zprávu připravit nebo odpovědět od těkavých zařazení, které už se zapomene.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Prošetření možných příčin opožděné zprávy přicházející z nestálého účastníka.
