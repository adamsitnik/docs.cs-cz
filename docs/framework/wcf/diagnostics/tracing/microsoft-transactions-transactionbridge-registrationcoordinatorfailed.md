# <a name="microsofttransactionstransactionbridgeregistrationcoordinatorfailed"></a>Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorFailed
Služba protokolu WS-AT nepovedlo se poslat zprávu svému koordinátorovi registrační  
  
## <a name="description"></a>Popis  
 Trasovaná Pokud místní TransactionManager není schopen se registrují v jeho nadřízeného TransactionManager lokalizovat odeslat zprávu.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Zkontrolujte zprávu trasování pro výjimky, které způsobují selhání odesílání zpráv
