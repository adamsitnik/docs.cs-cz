# <a name="systemservicemodelcommunicationobjectopenfailed"></a>System.ServiceModel.CommunicationObjectOpenFailed
Objekt komunikace se nepodařilo otevřít.  
  
## <a name="description"></a>Popis  
 Objekt komunikace se nepodařilo otevřít, protože došlo k výjimce během zpracování jeho metodu Open. Další okolní trasování může obsahovat podrobnosti o výjimce.
