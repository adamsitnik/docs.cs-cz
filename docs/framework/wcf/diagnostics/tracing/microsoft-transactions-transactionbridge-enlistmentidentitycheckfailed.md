# <a name="microsofttransactionstransactionbridgeenlistmentidentitycheckfailed"></a>Microsoft.Transactions.TransactionBridge.EnlistmentIdentityCheckFailed
Totožnost odesílatele příchozí zprávy nesouhlasí s totožností odvozenou z reference koncového bodu.  
  
## <a name="description"></a>Popis  
 Trasovaná, pokud ověření zabezpečení zprávy se nezdařilo.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Ujistěte se, že je zabezpečení transakcí správně nakonfigurovaný pro všechny počítače účastnící se transakce.
