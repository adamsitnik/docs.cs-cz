# <a name="systemservicemodelchannelshttpchannelrequestaborted"></a>System.ServiceModel.Channels.HttpChannelRequestAborted
Nepovedlo se poslat zprávu požadavku prostřednictvím protokolu HTTP.  
  
## <a name="description"></a>Popis  
 Úroveň trasování varování indikuje, že požadavek kanálu HTTP byla přerušena. Další informace v tomto trasování bude odhalit příčinu nebo zprávu předá metodě přerušení.
