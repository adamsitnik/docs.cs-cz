# <a name="systemservicemodelservicehostfaulted"></a>System.ServiceModel.ServiceHostFaulted
System.ServiceModel.ServiceHostFaulted  
  
## <a name="description"></a>Popis  
 Došlo k chybě hostitele ServiceHost.  
  
 Trasování označuje, že jeden naslouchací procesy došlo k chybě. Trasování je užitečné při ladění.
