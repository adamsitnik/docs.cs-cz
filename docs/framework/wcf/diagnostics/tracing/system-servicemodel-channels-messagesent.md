# <a name="systemservicemodelchannelsmessagesent"></a>System.ServiceModel.Channels.MessageSent
System.ServiceModel.Channels.MessageSent  
  
## <a name="description"></a>Popis  
 Byla odeslána zpráva přes kanál.  
  
 Trasování je vygenerován při odesílání zprávy WCF, pokud `System.ServiceModel` je povolený zdroj trasování na podrobné úrovni nebo informace. Trasování je nutné zobrazit šipka korelace zprávy v zobrazení grafu aktivit.
