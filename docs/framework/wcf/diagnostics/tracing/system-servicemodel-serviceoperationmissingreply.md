# <a name="systemservicemodelserviceoperationmissingreply"></a>System.ServiceModel.ServiceOperationMissingReply
System.ServiceModel.ServiceOperationMissingReply  
  
## <a name="description"></a>Popis  
 Operace požadavku/odpovědi nemá žádnou zprávu s odpovědí.  
  
 To je obousměrný operace; neexistuje ale žádná odpověď k odeslání do klienta. Trasování můžete použít pro ladění během vývoje.
