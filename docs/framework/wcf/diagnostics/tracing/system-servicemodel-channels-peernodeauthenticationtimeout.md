# <a name="systemservicemodelchannelspeernodeauthenticationtimeout"></a>System.ServiceModel.Channels.PeerNodeAuthenticationTimeout
Bezpečnostní ověření typu handshake se nezdařilo během zadaného časového limitu.  
  
## <a name="description"></a>Popis  
 Vyvolá se při pokusu o navázání připojení k sousedovi zabezpečené tohoto trasování.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Ujistěte se, že všechny okolí mají příslušná zabezpečovací přihlašovací údaje a reagovat na všechny zprávy v méně než 1 minuta.
