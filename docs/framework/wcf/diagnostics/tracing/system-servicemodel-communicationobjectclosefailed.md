# <a name="systemservicemodelcommunicationobjectclosefailed"></a>System.ServiceModel.CommunicationObjectCloseFailed
Objekt komunikace se nepodařilo zavřít.  
  
## <a name="description"></a>Popis  
 Objekt komunikace se nepodařilo zavřít, protože došlo k výjimce během zpracování její metoda Close. Další okolní trasování může obsahovat podrobnosti o výjimce.
