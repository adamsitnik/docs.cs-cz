# <a name="systemservicemodelactivationwebhostcompilation"></a>System.ServiceModel.Activation.WebHostCompilation
System.ServiceModel.Activation.WebHostCompilation  
  
## <a name="description"></a>Popis  
 Vyvolá se v případě hostované Internetové informační služby (IIS) nebo Windows Process Activation Service (WAS), že služba je zkompilován z vloženého kódu, který se nachází v souboru SVC.
