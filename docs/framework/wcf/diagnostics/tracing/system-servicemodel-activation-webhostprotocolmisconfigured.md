# <a name="systemservicemodelactivationwebhostprotocolmisconfigured"></a>System.ServiceModel.Activation.WebHostProtocolMisconfigured
Protokol není správně nakonfigurováno v Windows WAS Process Activation Service ().  
  
## <a name="description"></a>Popis  
 Tato chyba nastane, pokud přenosový protokol je nesprávně nakonfigurována.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Zkontrolujte a opravte přenos konfigurační parametry v souboru web.config.
