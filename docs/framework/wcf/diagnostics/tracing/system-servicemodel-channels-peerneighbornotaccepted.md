# <a name="systemservicemodelchannelspeerneighbornotaccepted"></a>System.ServiceModel.Channels.PeerNeighborNotAccepted
Rovnocenný soused nebyl přijat.  
  
## <a name="description"></a>Popis  
 Trasování nastane při připojení k souseda zpracování. Mezi možné důvody patří nastavení neodpovídající zabezpečení, překročen časový limit, souseda, které nemůžou přijímat připojení nebo přechodný síťový problémy.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Zkontrolujte nastavení neodpovídající zabezpečení, vypršel časový limit, od souseda nemožnost tak, aby přijímal připojení nebo přechodný síťový problémy.
