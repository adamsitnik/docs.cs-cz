# <a name="message-logging"></a>Protokolování zpráv
Windows Communication Foundation (WCF) poskytuje možnost protokoluovat příchozí a odchozí zprávy pro offline spotřebu. Protokolování zpráv umožňuje zobrazit informace o tom, jak zpráva a tělo zprávy vypadají. Tento typ protokolování je obzvlášť užitečný v tom, že vám dává informace o tom, jaké argumenty byly předány, a jak přijímající koncový bod viděl argumenty vyjádřené jako XML. Protokolování zprávy, jak byla přijata, umožňuje také diagnostikovat poškozené zprávy a zjistit, jak byla zpráva doručena. Můžete také prostudovat používané tokeny zabezpečení, části šifrované a podepsané a části, které jsou ponechány beze změny.
