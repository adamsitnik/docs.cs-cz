# <a name="core-communications-named-pipe-transport-channels"></a>Základní komunikace: Přenosové kanály pojmenovaného kanálu
Toto téma uvádí všechny výjimky generované přenosové kanály pojmenovaného kanálu.  
  
## <a name="exception-list"></a>Seznam výjimek  
  
|Kód zdroje|Řetězec prostředku|  
|-------------------|---------------------|  
|PipeCantCloseWithPendingWrite|Čeká na vyřízení operace zápisu do kanálu nelze zavřít kanálu.|  
|PipeReadPending|Operace čtení se u tohoto kanálu.|  
|PipeShutdownReadError|Operace čtení kanálu indikátoru "vypnout" se nezdařilo.|  
|PipeShutdownWriteError|Operace zápisu, indikátoru "vypnout" kanálu se nezdařila.|  
|PipeWritePending|Operace zápisu se u tohoto kanálu.|
