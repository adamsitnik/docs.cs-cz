# <a name="peer-channel"></a>Rovnocenný kanál
Toto téma uvádí všechny výjimky generované rovnocenného kanálu Windows Communication Foundation (WCF).  
  
## <a name="exception-list"></a>Seznam výjimek  
  
|Kód zdroje|Řetězec prostředku|  
|-------------------|---------------------|  
|InsufficientCredentials|Zadaná pověření nejsou dostatečná k provedení požadované operace. Zadejte prosím platnou hodnotu pro zadanou operaci|  
|InvalidResolverMode|Zadaná hodnota třídy PeerResolverMode je neplatná. Zadejte třídu PeerResolveMode.Auto, výchozí nebo PNRP.|  
|PeerNodeAborted|Třídu PeerNode nelze otevřít, protože operace byla ukončena.|  
|PeerNullRegistrationInfo|Informace o registraci nemůže mít hodnotu null. Zkontrolujte, zda je operace registrace vyvolána s platným objektem RegistrationInfo.|  
|PeerNullResolveInfo|Informace nelze vyřešit `null`. Zkontrolujte, zda je operace řešení vyvolána s platným objektem ResolveInfo.|
