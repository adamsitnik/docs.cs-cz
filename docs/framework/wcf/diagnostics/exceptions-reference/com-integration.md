# <a name="com-integration"></a>Integrace COM+
Toto téma uvádí všechny výjimky generované integrace modelu COM +.  
  
## <a name="exception-list"></a>Seznam výjimek  
  
|Kód zdroje|Řetězec prostředku|  
|-------------------|---------------------|  
|CannotResolveTypeForParamInMessageDescription|Typ parametru zadané v rámci zadaného oboru názvů se nedá přeložit.|  
|NoEqualSignFound|Zadané klíčové slovo má nenásleduje ho. Ujistěte se, že každý klíčovým slovem následovalo rovnítko a hodnota.|  
|UdtNotFoundInAssembly|Nebyl nalezen zadaný uživatelem definovaného typu. Ujistěte se, že jsou zaregistrované a zadaný správný typ a knihovna typů.|
