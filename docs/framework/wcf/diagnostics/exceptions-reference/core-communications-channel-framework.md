# <a name="core-communications-channel-framework"></a>Základní komunikace: Architektura kanálů
Toto téma uvádí všechny výjimky generované architektura kanálů Windows Communication Foundation (WCF).  
  
## <a name="exception-list"></a>Seznam výjimek  
  
|Kód zdroje|Řetězec prostředku|  
|-------------------|---------------------|  
|ReceiveRequestTimedOut|Byl přijat požadavek na zadané adrese místní vypršel po určeném čase. Čas přidělený této operaci pravděpodobně částí delšího časového limitu.|  
|ReceiveTimedOutNoLocalAddress|Operace obdržení vypršel po určeném čase. Čas přidělený této operaci pravděpodobně částí delšího časového limitu.|  
|UnableToDemuxChannel|Žádný kanál je k dispozici pro příjem zpráv s určenou akcí.|
