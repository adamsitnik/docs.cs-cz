# <a name="reliable-messaging"></a>Spolehlivé zasílání zpráv
Toto téma uvádí všechny výjimky generované spolehlivé zasílání zpráv.  
  
## <a name="exception-list"></a>Seznam výjimek  
  
|Kód zdroje|Řetězec prostředku|  
|-------------------|---------------------|  
|InvalidBufferRemaining|Zpráva byla přijata vyrovnávací paměť hodnotou zadanou zbývající velikost. Vyrovnávací paměť, která zůstává nemůže být menší než nula.|
