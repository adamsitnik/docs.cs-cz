# <a name="deploying-services"></a>Nasazení služeb
Toto téma popisuje, jak lze nasadit aplikaci Windows Communication Foundation (WCF) do prostředí Runtime.  
  
## <a name="choosing-the-hosting-environment-for-your-application"></a>Výběr hostitelského prostředí pro vaši aplikaci  
 Služby WCF jsou navržené tak, aby běžely v jakémkoli procesu Windows, který podporuje spravovaný kód. Aby bylo možné stát aktivní, musí být služba hostována v prostředí runtime, které je vytvoří a řídí její kontext a dobu života. Hostování možností v rámci nejjednodušších konzolových aplikací na serverová prostředí, jako je například služba systému Windows, Internetová informační služba (IIS) nebo v rámci pracovního procesu spravovaného aktivační službou systému Windows (WAS). Chcete-li si projít různé možnosti hostování vaší aplikace WCF, přečtěte si informace o [hostitelských službách](../hosting-services.md).
