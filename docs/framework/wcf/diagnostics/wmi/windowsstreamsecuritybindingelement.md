# <a name="windowsstreamsecuritybindingelement"></a>WindowsStreamSecurityBindingElement
WindowsStreamSecurityBindingElement  
  
## <a name="syntax"></a>Syntaxe  
  
```csharp
class WindowsStreamSecurityBindingElement : BindingElement  
{  
  string ProtectionLevel;  
};  
```  
  
## <a name="methods"></a>Metody  
 Třída WindowsStreamSecurityBindingElement nedefinuje žádné metody.  
  
## <a name="properties"></a>Vlastnosti  
 Třída WindowsStreamSecurityBindingElement má následující vlastnost:  
  
### <a name="protectionlevel"></a>Třída protectionLevel  
 Datový typ: řetězec  
  
 Typ přístupu: jen pro čtení  
  
 Úroveň ochrany pro datový proud TCP.
