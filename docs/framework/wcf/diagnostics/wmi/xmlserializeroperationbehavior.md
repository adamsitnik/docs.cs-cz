# <a name="xmlserializeroperationbehavior"></a>XmlSerializerOperationBehavior
XmlSerializerOperationBehavior  
  
## <a name="syntax"></a>Syntaxe  
  
```csharp
class XmlSerializerOperationBehavior : Behavior  
{  
  string Style;  
  string Use;  
};  
```  
  
## <a name="methods"></a>Metody  
 Třída XmlSerializerOperationBehavior nedefinuje žádné metody.  
  
## <a name="properties"></a>Vlastnosti  
 Třída XmlSerializerOperationBehavior má následující vlastnosti:  
  
### <a name="style"></a>Styl  
 Datový typ: řetězec  
  
 Typ přístupu: jen pro čtení  
  
 Definuje styl zprávy SOAP.  
  
### <a name="use"></a>Použití  
 Datový typ: řetězec  
  
 Typ přístupu: jen pro čtení  
  
 Určuje styl kódování SOAP.
