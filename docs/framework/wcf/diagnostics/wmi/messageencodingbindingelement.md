# <a name="messageencodingbindingelement"></a>MessageEncodingBindingElement
MessageEncodingBindingElement  
  
## <a name="syntax"></a>Syntaxe  
```csharp
class MessageEncodingBindingElement : BindingElement
{
    string MessageVersion;  
};  
 ```
  
## <a name="methods"></a>Metody  
 Tříd MessageEncodingBindingElement nedefinuje žádné metody.  
  
## <a name="properties"></a>Vlastnosti  
 Tříd MessageEncodingBindingElement má následující vlastnost:  
  
### <a name="messageversion"></a>MessageVersion  
 Datový typ: řetězec  
  
 Typ přístupu: jen pro čtení  
  
 Verze protokolu SOAP zprávy odesílané pomocí vazby.
