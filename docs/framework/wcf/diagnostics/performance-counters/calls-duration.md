# <a name="calls-duration"></a>Doba volání
Název čítače: Doba volání  
  
## <a name="description"></a>Popis  
 Průměrná doba trvání volání této operace. Průměrná doba trvání se počítá na základě rovnice: (N1-N0)/(D1-D0).  
  
> [!WARNING]
>  Při použití na asynchronní služby WCF čítač trvání volání vždy vrátí hodnotu -1.
