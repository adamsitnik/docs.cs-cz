# <a name="calls-duration"></a>Doba volání
Název čítače: Doba volání  
  
## <a name="description"></a>Popis  
 Průměrná doba volání této operace. Průměrná doba trvání se vypočítá na základě této rovnice: (N1-N0)/(D1-D0).  
  
> [!WARNING]
> Při použití v asynchronní službě WCF počítadlo trvání volání vrátí hodnotu-1.
