# <a name="endpoint-transactions-flowed"></a>Koncový bod: Počet plynoucích transakcí
Název čítače: Počet plynoucích transakcí.  
  
## <a name="description"></a>Popis  
 Počet transakcí, které byly převedeny do operací v tomto koncovém bodu. Hodnota tohoto čítače se zvýší pokaždé, když ID transakce je k dispozici v zpráva odeslaná do koncového bodu.
