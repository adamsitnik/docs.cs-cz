# <a name="endpoint-calls-faulted"></a>Koncový bod: Počet volání s chybou
Název čítače: Počet volání s chybou.  
  
## <a name="description"></a>Popis  
 Počet volání tohoto koncového bodu, které se mají vrátit chyby. V aplikacích Windows Communication Foundation (WCF) komunikují metod služby informace o chybě zpracování pomocí chybové zprávy protokolu SOAP. Typy zpráv, které jsou zahrnuty v metadatech pro operaci služby a proto vytvořit selhání kontrakt, který můžou klienti použít k jejich spuštění zkontrolujte robustní nebo interaktivní jsou chyby SOAP. Protože chyb SOAP jsou vyjádřeny klientům v podobě XML, jsou vysoce interoperabilní.
