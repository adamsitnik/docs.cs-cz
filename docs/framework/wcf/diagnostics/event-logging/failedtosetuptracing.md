# <a name="failedtosetuptracing"></a>FailedToSetupTracing
Id: 100  
  
 Závažnost: Chyba  
  
 Kategorie: Trasování  
  
## <a name="description"></a>Popis  
 Pokus o nastavení trasování se nezdařilo. Trasování je zakázáno.  
  
 Uvádí události výjimky, název procesu a ID procesu.
