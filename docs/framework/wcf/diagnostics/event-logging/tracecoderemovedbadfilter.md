# <a name="tracecoderemovedbadfilter"></a>TraceCodeRemovedBadFilter
Účet 114  
  
 Závažnost Chyba  
  
 Kategorií ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost indikuje špatný filtr protokolování zprávy, který měl přístup k textu zprávy byl odebrán. Událost obsahuje seznam XPath, název procesu a ID procesu.
