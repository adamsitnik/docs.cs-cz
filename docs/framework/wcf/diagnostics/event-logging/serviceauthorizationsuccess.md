# <a name="serviceauthorizationsuccess"></a>ServiceAuthorizationSuccess
Účet 176  
  
 Závažnost Informativní  
  
 Kategorií SecurityAudit  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že autorizace služby byla úspěšná. Událost obsahuje seznam služby, akce, identitu klienta, autorizační kontext, ID aktivity a ServiceAuthorizationManager.
