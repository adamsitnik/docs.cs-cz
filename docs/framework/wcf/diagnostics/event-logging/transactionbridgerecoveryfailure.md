# <a name="transactionbridgerecoveryfailure"></a>TransactionBridgeRecoveryFailure
Id: 146  
  
 Závažnost: Chyba  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost ukazuje na to, že se nepodařilo MSDTC TransactionBridge během zotavení. Toto je závažná podmínka, takže služba MSDTC byla ukončena. Uvádí události výjimky, název procesu a ID procesu. .
