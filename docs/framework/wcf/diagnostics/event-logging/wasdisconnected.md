# <a name="wasdisconnected"></a>WasDisconnected
Účet 162  
  
 Závažnost Chyba  
  
 Kategorií SMSvcHost  
  
## <a name="description"></a>Popis  
 Tato událost indikuje, že aktivace procesů systému Windows (WAS) byla odpojena. Událost zobrazí seznam HRESULT, název procesu a ID procesu.
