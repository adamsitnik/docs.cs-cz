# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
Id: 161  
  
 Závažnost: Chyba  
  
 Kategorie: SMSvcHost  
  
## <a name="description"></a>Popis  
 Tato událost ukazuje na to, že došlo k neočekávané chybě v adaptéru naslouchacího procesu při zpracování oznámení Windows Process Activation Service (WAS). Proces je ukončen. Uvádí události protokolu, výjimky, název procesu a ID procesu.
