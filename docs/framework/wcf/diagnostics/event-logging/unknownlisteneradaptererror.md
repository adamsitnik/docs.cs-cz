# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
Účet 161  
  
 Závažnost Chyba  
  
 Kategorií SMSvcHost  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že došlo k neočekávané chybě v adaptéru naslouchacího procesu během zpracování oznámení o aktivační službě procesů systému Windows (WAS). Proces je ukončen. Událost uvádí protokol, výjimku, název procesu a ID procesu.
