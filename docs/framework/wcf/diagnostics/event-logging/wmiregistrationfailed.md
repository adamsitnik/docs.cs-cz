# <a name="wmiregistrationfailed"></a>WmiRegistrationFailed
Účet 126  
  
 Závažnost Chyba  
  
 Kategorií ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že zprostředkovatel rozhraní WMI nebyl zaregistrován. Událost zobrazí seznam objektů rozhraní WMI, chybu, název procesu a ID procesu.
