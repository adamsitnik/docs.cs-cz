# <a name="wmiregistrationfailed"></a>WmiRegistrationFailed
Id: 126  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost ukazuje na to, že zprostředkovatel rozhraní WMI nebyla registrována. Událost obsahuje objekt rozhraní WMI, chyba, název procesu a ID procesu.
