# <a name="participantrecoverylogentrycorrupt"></a>ParticipantRecoveryLogEntryCorrupt
Účet 138  
  
 Závažnost Chyba  
  
 Kategorií TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost indikuje, že položka protokolu obnovení účastníka je poškozená a nedá se deserializovat. Tato chyba může být způsobena ztrátou dat. Událost uvádí ID transakce, data obnovení (kódovaná v kódování Base64), výjimku, název procesu a ID procesu.
