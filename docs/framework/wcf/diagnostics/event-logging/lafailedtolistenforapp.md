# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp
Id: 160  
  
 Závažnost: Chyba  
  
 Kategorie: SMSvcHost  
  
## <a name="description"></a>Popis  
 Tato událost ukazuje na to, že došlo k chybě zadaný aktivační služby protokolu zadaného při pokusu o naslouchání pro určenou lokalitu, že tedy protokolu je zakázaná pro web dočasně... Událost obsahuje adresu URL, stav, výjimky, název procesu a ID procesu.
