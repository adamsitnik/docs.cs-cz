# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp
Účet 160  
  
 Závažnost Chyba  
  
 Kategorií SMSvcHost  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že došlo k chybě v zadané aktivační službě zadaného protokolu, při pokusu o naslouchání pro zadanou lokalitu, proto je protokol pro web dočasně zakázán. Událost vypíše adresu URL, stav, výjimku, název procesu a ID procesu.
