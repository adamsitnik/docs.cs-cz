# <a name="failedtoinitializetracesource"></a>FailedToInitializeTraceSource
Id: 101  
  
 Závažnost: Chyba  
  
 Kategorie: Trasování  
  
## <a name="description"></a>Popis  
 Zdroj trasování nebyl inicializován. Trasování je zakázáno. Uvádí události výjimky, název procesu a ID procesu.
