# <a name="participantrecoverylogentrycreationfailure"></a>ParticipantRecoveryLogEntryCreationFailure
Účet 141  
  
 Závažnost Chyba  
  
 Kategorií TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že se pro zařazení účastníka nepovedlo vygenerovat položku protokolu arecovery. Transakce bude přerušena. Událost obsahuje seznam ID transakce, ID zařazení, výjimku, název procesu a ID procesu.
