# <a name="protocolrecoverybeginningfailure"></a>ProtocolRecoveryBeginningFailure
Účet 144  
  
 Závažnost Chyba  
  
 Kategorií TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že při zahájení obnovení se nezdařila Chyba protokolu MSDTC WS-Atomic Transaction Protocol. Výsledkem je, že funkce WS-Atomic Transactions je zakázaná. Událost obsahuje seznam ID protokolu, název protokolu, výjimku, název procesu a ID procesu.
