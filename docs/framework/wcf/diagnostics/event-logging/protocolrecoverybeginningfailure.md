# <a name="protocolrecoverybeginningfailure"></a>ProtocolRecoveryBeginningFailure
Id: 144  
  
 Závažnost: Chyba  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že protokol MSDTC WS-Atomic Transactions selhal na začátku zotavení. V důsledku toho funkce WS-Atomic Transactions zakázána. Událost obsahuje ID protokol, název protokolu, výjimky, název procesu a ID procesu.
