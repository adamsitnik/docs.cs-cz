# <a name="coordinatorrecoverylogentrycorrupt"></a>CoordinatorRecoveryLogEntryCorrupt
Účet 139  
  
 Závažnost Chyba  
  
 Kategorií TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že položka protokolu obnovení koordinátora byla poškozena a nelze ji deserializovat. Tato chyba může být způsobena ztrátou dat. Událost uvádí ID transakce, data obnovení (kódovaná v kódování Base64), výjimku, název procesu a ID procesu.
