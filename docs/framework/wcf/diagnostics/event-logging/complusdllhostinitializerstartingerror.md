# <a name="complusdllhostinitializerstartingerror"></a>ComPlusDllHostInitializerStartingError
Id: 131  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost ukazuje na to, že při spuštění DllHost inicializátoru došlo k chybě. Události jsou uvedeny AppID, výjimky, název procesu a ID procesu.
