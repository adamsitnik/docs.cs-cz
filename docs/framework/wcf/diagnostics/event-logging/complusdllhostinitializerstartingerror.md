# <a name="complusdllhostinitializerstartingerror"></a>ComPlusDllHostInitializerStartingError
Účet 131  
  
 Závažnost Chyba  
  
 Kategorií ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že při spouštění inicializátoru DllHost došlo k chybě. Událost obsahuje seznam AppID, výjimku, název procesu a ID procesu.
