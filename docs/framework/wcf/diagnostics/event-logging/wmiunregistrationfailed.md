# <a name="wmiunregistrationfailed"></a>WmiUnregistrationFailed
Id: 127  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost ukazuje na to, že zprostředkovatel rozhraní WMI nebyla zrušena registrace. Událost obsahuje objekt rozhraní WMI, chyba, název procesu a ID procesu.
