# <a name="messageloggingon"></a>MessageLoggingOn
Id: 116  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Popis  
 Přihlašování zprávy bylo zapnuto. Může být zaprotokolována citlivé informace, i v případě, že byla přenášena šifrovaná přenosu. Například zpráva těla. Události jsou uvedeny AppID, výjimky, název procesu a ID procesu.
