# <a name="messageloggingon"></a>MessageLoggingOn
Účet 116  
  
 Závažnost Chyba  
  
 Kategorií ServiceModel  
  
## <a name="description"></a>Popis  
 Bylo zapnuté protokolování zpráv. Citlivé údaje mohou být protokolovány i v případě, že byly zašifrovány na lince. Například tělo zprávy. Událost obsahuje seznam AppID, výjimku, název procesu a ID procesu.
