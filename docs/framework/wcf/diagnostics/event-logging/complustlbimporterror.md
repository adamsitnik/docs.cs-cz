Účet 132  
  
 Závažnost Chyba

## <a name="description"></a>Popis  
 Tato událost označuje, že při importu knihovny typů došlo k chybě. Událost obsahuje seznam identifikátorů IID, ID knihovny typů, výjimku, název procesu a ID procesu.
