# <a name="webhostfailedtolisten"></a>WebHostFailedToListen
Účet 112  
  
 Závažnost Chyba  
  
 Kategorií ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost indikuje, že při pokusu o naslouchat konkrétní adrese URL došlo k chybě. Tento pracovní proces je ukončen. Událost obsahuje seznam výjimek, názvu procesu a ID procesu.
