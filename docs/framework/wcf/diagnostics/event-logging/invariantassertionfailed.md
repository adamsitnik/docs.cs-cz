# <a name="invariantassertionfailed"></a>InvariantAssertionFailed
Id: 106  
  
 Závažnost: Chyba  
  
 Kategorie: Trasování  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že invariantní kontrolní výraz má hodnotu false. Událost obsahuje zprávy, název procesu a ID procesu.
