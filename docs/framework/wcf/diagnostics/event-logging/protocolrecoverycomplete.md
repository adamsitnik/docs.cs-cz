# <a name="protocolrecoverycomplete"></a>ProtocolRecoveryComplete
Účet 150  
  
 Závažnost Chyba  
  
 Kategorií TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že služba protokolu WS-Atomic Transaction Protocol úspěšně dokončila spuštění a obnovení. Událost obsahuje seznam ID protokolu, název protokolu, název procesu a ID procesu.
