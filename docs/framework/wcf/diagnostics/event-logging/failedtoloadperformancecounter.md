# <a name="failedtoloadperformancecounter"></a>FailedToLoadPerformanceCounter
Účet 118  
  
 Závažnost Chyba  
  
 Kategorií ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že čítač výkonu nebyl načten. Událost obsahuje seznam názvů kategorií, název čítače, název instance, výjimku, název procesu a ID procesu.
