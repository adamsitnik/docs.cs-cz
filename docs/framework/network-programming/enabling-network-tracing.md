 <span data-ttu-id="cbd6e-116">Chcete-li odeslat trasování do souboru protokolu, přidejte následující uzel do `<system.diagnostics>` uzlu příslušného konfiguračního souboru (aplikace nebo počítač).</span><span class="sxs-lookup"><span data-stu-id="cbd6e-116">To send traces to a log file, add the following node to the `<system.diagnostics>` node of the appropriate configuration file (application or machine).</span></span> <span data-ttu-id="cbd6e-117">Název souboru (Trace. log) můžete změnit tak, aby vyhovoval vašim potřebám.</span><span class="sxs-lookup"><span data-stu-id="cbd6e-117">You can change the name of the file (trace.log) to suit your needs.</span></span>
