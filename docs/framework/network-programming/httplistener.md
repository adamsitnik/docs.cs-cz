# <a name="httplistener"></a>HttpListener
<xref:System.Net.HttpListener> Třída poskytuje programově řízený naslouchací proces protokolu HTTP. Naslouchací proces je aktivní po dobu života <xref:System.Net.HttpListener> objektu a běží v rámci aplikace.  
  
## <a name="httpsys"></a>HTTP. TABULCE  
 <xref:System.Net.HttpListener> Třída je postavená na http. sys, což je naslouchací proces režimu jádra, který zpracovává veškerý provoz protokolu HTTP pro systém Windows. HTTP. sys poskytuje správu připojení, omezení šířky pásma a protokolování webového serveru. Pomocí tohoto `HttpCfg.exe` nástroje přidejte certifikáty SSL. Další informace najdete v dokumentaci k nástroji [Httpcfg. exe](https://go.microsoft.com/fwlink/?LinkID=178284) v dokumentaci k [serveru](https://go.microsoft.com/fwlink/?LinkID=178285) .
