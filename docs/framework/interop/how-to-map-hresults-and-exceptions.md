<span data-ttu-id="b5aaa-103">Metody modelu COM zprávy o chybách tak, že vrací výsledky HRESULT; metod rozhraní .NET dejte nám o nich vyvoláním výjimky.</span><span class="sxs-lookup"><span data-stu-id="b5aaa-103">COM methods report errors by returning HRESULTs; .NET methods report them by throwing exceptions.</span></span> <span data-ttu-id="b5aaa-104">Modul runtime zpracovává přechod mezi nimi.</span><span class="sxs-lookup"><span data-stu-id="b5aaa-104">The runtime handles the transition between the two.</span></span> <span data-ttu-id="b5aaa-105">Každá třída výjimky v rozhraní .NET Framework se mapuje na HRESULT.</span><span class="sxs-lookup"><span data-stu-id="b5aaa-105">Each exception class in the .NET Framework maps to an HRESULT.</span></span>  
  
 <span data-ttu-id="b5aaa-106">Třídy výjimek definované uživatelem můžete zadat jakékoli HRESULT je vhodné.</span><span class="sxs-lookup"><span data-stu-id="b5aaa-106">User-defined exception classes can specify whatever HRESULT is appropriate.</span></span> <span data-ttu-id="b5aaa-107">Tyto třídy výjimek můžete dynamicky měnit hodnota HRESULT, který se má vrátit, když výjimka je vygenerován pomocí nastavení **HResult** pole objektu výjimky.</span><span class="sxs-lookup"><span data-stu-id="b5aaa-107">These exception classes can dynamically change the HRESULT to be returned when the exception is generated by setting the **HResult** field on the exception object.</span></span> <span data-ttu-id="b5aaa-108">Další informace o výjimce jsou uvedeny do klienta prostřednictvím **IErrorInfo** rozhraní, které je implementované u objektu rozhraní .NET v nespravovaný proces.</span><span class="sxs-lookup"><span data-stu-id="b5aaa-108">Additional information about the exception is provided to the client through the **IErrorInfo** interface, which is implemented on the .NET object in the unmanaged process.</span></span>

### <a name="to-create-a-new-exception-class-and-map-it-to-an-hresult"></a><span data-ttu-id="b5aaa-114">Chcete-li vytvořit novou třídu výjimky a jejich mapování na HRESULT</span><span class="sxs-lookup"><span data-stu-id="b5aaa-114">To create a new exception class and map it to an HRESULT</span></span>  
  
1. <span data-ttu-id="b5aaa-115">Pomocí následujícího kódu vytvořte novou třídu výjimek s názvem `NoAccessException` a jejich mapování na HRESULT `E_ACCESSDENIED`.</span><span class="sxs-lookup"><span data-stu-id="b5aaa-115">Use the following code to create a new exception class called `NoAccessException` and map it to the HRESULT `E_ACCESSDENIED`.</span></span>

    ```cpp  
    {  
        NoAccessException () {  
        HResult = E_ACCESSDENIED;

    }

    {  
    throw new NoAccessException();  
    }  
    ```
