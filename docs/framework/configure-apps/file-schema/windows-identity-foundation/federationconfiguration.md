## <a name="example"></a><span data-ttu-id="dda5c-161">Příklad</span><span class="sxs-lookup"><span data-stu-id="dda5c-161">Example</span></span>  
 <span data-ttu-id="dda5c-162">Následující XML ukazuje `<federationConfiguration>` element, který určuje nastavení WSFAM a určuje, že soubor cookie obslužnou rutinu výchozí (instance <xref:System.IdentityModel.Services.ChunkedCookieHandler> třídy) využívat SAM.</span><span class="sxs-lookup"><span data-stu-id="dda5c-162">The following XML shows a `<federationConfiguration>` element that specifies settings for the WSFAM and specifies that the default cookie handler (an instance of the <xref:System.IdentityModel.Services.ChunkedCookieHandler> class) be used by the SAM.</span></span>

>  <span data-ttu-id="dda5c-163">V tomto příkladu obslužné rutiny souborů cookie ani WSFAM vyžadovaných pro použití protokolu HTTPS.</span><span class="sxs-lookup"><span data-stu-id="dda5c-163">In this example, neither the cookie handler nor WSFAM are required to use HTTPS.</span></span> <span data-ttu-id="dda5c-164">Důvodem je, že `requireHttps` atribut na `<wsFederation>` elementu a `requireSsl` atribut na `<cookieHandlerElement>` jsou `false`.</span><span class="sxs-lookup"><span data-stu-id="dda5c-164">This is because the `requireHttps` attribute on the `<wsFederation>` element and the `requireSsl` attribute on the `<cookieHandlerElement>` are `false`.</span></span> <span data-ttu-id="dda5c-165">Tato nastavení se nedoporučuje pro většinu produkčních prostředích, jako, může představovat bezpečnostní riziko.</span><span class="sxs-lookup"><span data-stu-id="dda5c-165">These settings are not recommended for most production environments as they may present a security risk.</span></span>
