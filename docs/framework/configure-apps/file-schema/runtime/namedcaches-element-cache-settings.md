 <span data-ttu-id="e3720-145">Následující příklad ukazuje, jak nastavit název mezipaměti, výchozí název položky mezipaměti tak, že nastavíte `name` atribut "Výchozí".</span><span class="sxs-lookup"><span data-stu-id="e3720-145">The following example shows how to set the name of the cache to the default cache entry name by setting the `name` attribute to "default".</span></span>
