 <span data-ttu-id="60c9d-200">Poskytuje stejnou podporu protokolů webové služby `WSHttpBinding`jako, ale pro použití s meziduplexní kontrakty. `WSDualHttpBinding`</span><span class="sxs-lookup"><span data-stu-id="60c9d-200">The `WSDualHttpBinding` provides the same support for Web Service protocols as the `WSHttpBinding`, but for use with duplex contracts.</span></span> <span data-ttu-id="60c9d-201">`WSDualHttpBinding`podporuje pouze zabezpečení SOAP a vyžaduje spolehlivé zasílání zpráv.</span><span class="sxs-lookup"><span data-stu-id="60c9d-201">`WSDualHttpBinding` only supports SOAP security and requires reliable messaging.</span></span> <span data-ttu-id="60c9d-202">Tato vazba vyžaduje, aby měl klient veřejný identifikátor URI, který pro službu poskytuje koncový bod zpětného volání.</span><span class="sxs-lookup"><span data-stu-id="60c9d-202">This binding requires that the client has a public URI that provides a callback endpoint for the service.</span></span> <span data-ttu-id="60c9d-203">Tato vlastnost je k dispozici v `clientBaseAddress` atributu.</span><span class="sxs-lookup"><span data-stu-id="60c9d-203">This is provided by the `clientBaseAddress` attribute.</span></span> <span data-ttu-id="60c9d-204">Duální vazba zpřístupňuje IP adresu klienta ke službě.</span><span class="sxs-lookup"><span data-stu-id="60c9d-204">A dual binding exposes the IP address of the client to the service.</span></span> <span data-ttu-id="60c9d-205">Klient by měl použít zabezpečení, aby zajistil, že se připojí pouze ke službám, kterým důvěřuje.</span><span class="sxs-lookup"><span data-stu-id="60c9d-205">The client should use security to ensure that it only connects to services it trusts.</span></span>  
  
 <span data-ttu-id="60c9d-206">Tato vazba se dá použít ke spolehlivé komunikaci prostřednictvím jednoho nebo více zprostředkovatelů SOAP.</span><span class="sxs-lookup"><span data-stu-id="60c9d-206">This binding can be used to communicate reliably through one or more SOAP intermediaries.</span></span>  
  
 <span data-ttu-id="60c9d-207">Ve výchozím nastavení tato vazba generuje zásobník modulu runtime s WS-ReliableMessaging pro spolehlivost, WS-Security pro zabezpečení a ověřování zpráv, HTTP pro doručování zpráv a kódování textu/XML.</span><span class="sxs-lookup"><span data-stu-id="60c9d-207">By default, this binding generates a runtime stack with WS-ReliableMessaging for reliability, WS-Security for message security and authentication, HTTP for message delivery, and a Text/XML message encoding.</span></span>
