 <span data-ttu-id="cab7a-207">Tato vazba je možné spolehlivě komunikovat prostřednictvím jednoho nebo více zprostředkovatelů SOAP.</span><span class="sxs-lookup"><span data-stu-id="cab7a-207">This binding can be used to communicate reliably through one or more SOAP intermediaries.</span></span>  
  
 <span data-ttu-id="cab7a-208">Ve výchozím nastavení, generuje tato vazba zásobník modulu runtime s WS-ReliableMessaging spolehlivosti, WS-Security zabezpečení zpráv a ověřování protokolu HTTP pro doručování zpráv a kódování zpráv Text/XML.</span><span class="sxs-lookup"><span data-stu-id="cab7a-208">By default, this binding generates a runtime stack with WS-ReliableMessaging for reliability, WS-Security for message security and authentication, HTTP for message delivery, and a Text/XML message encoding.</span></span>
