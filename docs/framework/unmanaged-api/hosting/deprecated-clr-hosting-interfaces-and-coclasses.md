# <a name="deprecated-clr-hosting-interfaces-and-coclasses"></a>Zastaralá rozhraní a třídy typu Coclass rozhraní hostování CLR
Tato část popisuje rozhraní, která nespravovaných hostitelů můžete použít k integraci common language runtime (CLR) v rozhraní .NET Framework verze 1.0 a 1.1 do svých aplikací. Tato rozhraní poskytuje metody pro hostitele konfigurace a načtení modulu runtime do procesu.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 IAppDomainSetup  
 Poskytuje metody pro hostitele konfigurace <xref:System.AppDomain>.  
  
 [ICeeFileGen – třída](../../../../docs/framework/unmanaged-api/hosting/iceefilegen-class.md)  
 (Nepoužívané) Poskytuje funkce pro vytváření nativní přenosný spustitelný soubor (PE).  
  
 [ICorRuntimeHost – rozhraní](../../../../docs/framework/unmanaged-api/hosting/icorruntimehost-interface.md)  
 Poskytuje metody pro hostitele a zadejte nastavení modulu CLR.
