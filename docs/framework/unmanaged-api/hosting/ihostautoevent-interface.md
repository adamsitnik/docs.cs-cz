# <a name="ihostautoevent-interface"></a>IHostAutoEvent – rozhraní
Poskytuje reprezentaci hostitele implementace události automatického obnovení.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[Set – metoda](../../../../docs/framework/unmanaged-api/hosting/ihostautoevent-set-method.md)|Nastaví aktuální `IHostAutoEvent` instance do signalizovaného stavu.|  
|[Wait – metoda](../../../../docs/framework/unmanaged-api/hosting/ihostautoevent-wait-method.md)|Způsobí, že aktuální `IHostAutoEvent` instance počkat, až vlastní události nebo zadaného množství času po uplynutí předem.|
