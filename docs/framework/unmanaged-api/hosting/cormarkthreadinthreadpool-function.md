# <a name="cormarkthreadinthreadpool-function"></a>CorMarkThreadInThreadPool – funkce
Označuje aktuálně spuštěné vlákno fondu vláken pro spuštění spravovaného kódu. Od verze .NET Framework 2,0 Tato funkce nemá žádný vliv. Není požadován a lze jej odebrat z vašeho kódu. Tato funkce je zastaralá v .NET Framework 4.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
void CorMarkThreadInThreadPool ();  
```
