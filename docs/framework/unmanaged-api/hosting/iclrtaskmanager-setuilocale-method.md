# <a name="iclrtaskmanagersetuilocale-method"></a>ICLRTaskManager::SetUILocale – metoda
Modul CLR (CLR), hostitele změnil národní prostředí uživatelského rozhraní (UI) nebo jazykovou verzi, upozorní na právě prováděnou úlohu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT SetUILocale (  
    [in] LCID lcid  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `lcid`  
 [in] Hodnota identifikátor národního prostředí, která mapuje pro nově přiřazenou zeměpisné jazykovou verzi a jazyk uživatelského rozhraní.  
  
## <a name="return-value"></a>Návratová hodnota  
  
|HRESULT|Popis|  
|-------------|-----------------|  
|S_OK|`SetUILocale` bylo úspěšně vráceno.|  
|HOST_E_CLRNOTAVAILABLE|Modul CLR se nenačetl do procesu nebo modul CLR je ve stavu, ve kterém nelze spouštět spravovaný kód nebo úspěšně zpracovat volání.|  
|HOST_E_TIMEOUT|Vypršel časový limit volání.|  
|HOST_E_NOT_OWNER|Volající není vlastníkem zámku.|  
|HOST_E_ABANDONED|Událost byla zrušena při zablokování vlákna nebo vlákénka čekal na něj.|  
|E_FAIL|Došlo k neznámé katastrofických selhání. Po návratu metody E_FAIL, modul CLR už nejsou použitelné v rámci procesu. Následující volání metody hostování vrací HOST_E_CLRNOTAVAILABLE.|  
  
## <a name="remarks"></a>Poznámky  
 `SetUILocale` nabízí příležitosti pro hostitele, chcete-li spustit některý z mechanismů, které může mít pro synchronizaci národní prostředí.
