# <a name="iappdomainbinding-interface"></a>IAppDomainBinding – rozhraní
Poskytuje metodu, která je volána modulem CLR (Common Language Runtime), aby upozornila hostitelskou aplikaci, že byla vytvořena doména aplikace.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[OnAppDomain – metoda](../../../../docs/framework/unmanaged-api/hosting/iappdomainbinding-onappdomain-method.md)|Volána modulem CLR (Common Language Runtime) pro oznamování hostitele, že byla vytvořena doména aplikace.|
