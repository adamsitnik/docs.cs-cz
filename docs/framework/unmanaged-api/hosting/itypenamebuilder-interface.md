# <a name="itypenamebuilder-interface"></a>ITypeNameBuilder – rozhraní
Poskytuje metody pro vytvoření názvu typu.  
  
 Toto rozhraní podporuje infrastrukturu rozhraní .NET Framework a není určena pro použití přímo v kódu.
