# <a name="hosting-structures"></a>Struktury pro hostování
Tato část popisuje nespravované struktury, které používá rozhraní API pro hostování.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [AssemblyBindInfo – struktura](../../../../docs/framework/unmanaged-api/hosting/assemblybindinfo-structure.md)  
 Poskytuje podrobné informace o odkazovaném sestavení.  
  
 [BucketParameters – struktura](../../../../docs/framework/unmanaged-api/hosting/bucketparameters-structure.md)  
 Ukládá název typu události a parametry pro aktuální výjimku, která je přidružena k události.  
  
 [COR_GC_STATS – struktura](../../../../docs/framework/unmanaged-api/hosting/cor-gc-stats-structure.md)  
 Poskytuje statistiku o mechanizmu uvolňování paměti modulu CLR (Common Language Runtime).  
  
 [COR_GC_THREAD_STATS – struktura](../../../../docs/framework/unmanaged-api/hosting/cor-gc-thread-stats-structure.md)  
 Obsahuje statistiku jednotlivých vláken, která souvisí s uvolňováním paměti.  
  
 [CustomDumpItem – struktura](../../../../docs/framework/unmanaged-api/hosting/customdumpitem-structure.md)  
 Popisuje položku, která se má přidat do vlastního výpisu paměti při zasílání zpráv o chybách.  
  
 [MDAInfo – struktura](../../../../docs/framework/unmanaged-api/hosting/mdainfo-structure.md)  
 Poskytuje podrobnosti o události `Event_MDAFired`, která aktivuje vytvoření pomocníka spravovaného ladění (MDA).  
  
 [ModuleBindInfo – struktura](../../../../docs/framework/unmanaged-api/hosting/modulebindinfo-structure.md)  
 Obsahuje podrobné informace o odkazovaném modulu a sestavení, které ho obsahuje.  
  
 [StackOverflowInfo – struktura](../../../../docs/framework/unmanaged-api/hosting/stackoverflowinfo-structure.md)  
 Ukládá typ přetečení, ke kterému došlo, a informace o výjimce, která byla vyvolána z důvodu přetečení.
