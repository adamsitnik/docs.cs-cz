# <a name="igchostcontrol-interface"></a>IGCHostControl – rozhraní
Poskytuje metodu, která umožňuje požádat o hostiteli, chcete-li změnit omezení virtuální paměti systému uvolňování paměti.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[RequestVirtualMemLimit – metoda](../../../../docs/framework/unmanaged-api/hosting/igchostcontrol-requestvirtualmemlimit-method.md)|Požadavky hostitele, chcete-li změnit omezení virtuální paměti.|
