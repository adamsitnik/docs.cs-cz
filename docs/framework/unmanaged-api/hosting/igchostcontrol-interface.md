# <a name="igchostcontrol-interface"></a>IGCHostControl – rozhraní
Poskytuje metodu, která umožňuje systému uvolňování paměti požádat hostitele o změnu omezení virtuální paměti.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[RequestVirtualMemLimit – metoda](../../../../docs/framework/unmanaged-api/hosting/igchostcontrol-requestvirtualmemlimit-method.md)|Požádá hostitele o změnu omezení virtuální paměti.|
