# <a name="itypenamefactory-interface"></a>ITypeNameFactory – rozhraní
Poskytuje metody pro dekonstrukce název typu.  
  
 Toto rozhraní podporuje infrastrukturu rozhraní .NET Framework a není určena pro použití přímo v kódu.
