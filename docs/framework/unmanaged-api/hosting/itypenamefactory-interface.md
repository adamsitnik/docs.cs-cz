# <a name="itypenamefactory-interface"></a>ITypeNameFactory – rozhraní
Poskytuje metody pro dekonstrukci názvu typu.  
  
 Toto rozhraní podporuje infrastrukturu .NET Framework a není určeno pro použití přímo v kódu.
