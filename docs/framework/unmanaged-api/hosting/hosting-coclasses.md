# <a name="hosting-coclasses"></a>Třídy typu coclass hostování
Tato část popisuje nespravované třídy API coclass, které používá rozhraní API pro hostování.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [CLRRuntimeHost – třída typu coclass](../../../../docs/framework/unmanaged-api/hosting/clrruntimehost-coclass.md)  
 Poskytuje rozhraní pro správu provádění kódu modulem runtime.  
  
 [ComCallUnmarshal – třída typu coclass](../../../../docs/framework/unmanaged-api/hosting/comcallunmarshal-coclass.md)  
 Poskytuje rozhraní pro správu zařazování ukazatelů rozhraní.  
  
 [CorRuntimeHost – třída typu coclass](../../../../docs/framework/unmanaged-api/hosting/corruntimehost-coclass.md)  
 Poskytuje rozhraní pro správu aplikací, které jsou spouštěny modulem CLR (Common Language Runtime).  
  
 [TypeNameFactory – třída typu coclass](../../../../docs/framework/unmanaged-api/hosting/typenamefactory-coclass.md)  
 Poskytuje rozhraní pro správu dekonstrukci názvu typu.
