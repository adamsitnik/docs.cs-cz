# <a name="itypelibresolver-interface"></a>ITypeLibResolver – rozhraní
Poskytuje [resolvetypelib – metoda](../../../../docs/framework/unmanaged-api/tlbexp/resolvetypelib-method.md), která se přeloží cestu k souboru knihovny typů.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[ResolveTypeLib – metoda](../../../../docs/framework/unmanaged-api/tlbexp/resolvetypelib-method.md)|Přeloží jednoduchý název knihovny typů tak, že vrací jeho úplnou cestu.|
