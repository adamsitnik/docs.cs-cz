# <a name="iassemblynameclone-method"></a>IAssemblyName::Clone – metoda

## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT Clone (  
    [out] IAssemblyName **pName

```  
  
## <a name="parameters"></a>Parametry  
 `pName`  
 [out] Vrácené kopie tohoto `IAssemblyName` objektu.
