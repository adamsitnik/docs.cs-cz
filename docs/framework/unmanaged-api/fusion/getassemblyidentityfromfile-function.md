# <a name="getassemblyidentityfromfile-function"></a>GetAssemblyIdentityFromFile – funkce
Získá ukazatel na `IUnknown` objekt, který je zadaný `IID` v sestavení v zadané cestě k souboru.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetAssemblyIdentityFromFile (  
    [in]  LPCWSTR   pwzFilePath,  
    [in]  REFIID    riid,  
    [out] IUnknown  **ppIdentity  
 );  
```  
  
## <a name="parameters"></a>Parametry  
 `pwzFilePath`  
 pro Platná cesta k požadovanému sestavení.  
  
 `riid`  
 pro `IID` Rozhraní, které se má vrátit.  
  
 `ppIdentity`  
 mimo Vrácený ukazatel rozhraní.
