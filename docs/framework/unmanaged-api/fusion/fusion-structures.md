 Obsahuje informace o sestavení, které je registrováno v globální mezipaměti sestavení (GAC).  
  
 [FUSION_INSTALL_REFERENCE – struktura](fusion-install-reference-structure.md)  
 Představuje odkaz, který aplikace provede na sestavení, které aplikace nainstalovala v globální mezipaměti sestavení (GAC).
