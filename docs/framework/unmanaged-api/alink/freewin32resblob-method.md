# <a name="freewin32resblob-method"></a>FreeWin32ResBlob – metoda
Uvolní objekt blob prostředků Win32 a související prostředky.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT FreeWin32ResBlob(  
    const void** ppResBlob  
) PURE;  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppResBlob`  
 Objekt blob prostředek uvolnit. Tato metoda přiřadí ukazatel objektu blob na hodnotu NULL.  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud metoda uspěje, vrátí hodnotu S_OK.  
  
## <a name="requirements"></a>Požadavky  
 Vyžaduje alink.h
