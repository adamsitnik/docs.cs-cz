# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory – metoda
Uloží všechna metadata v aktuálním oboru k zadané oblasti paměti.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT SaveToMemory (   
    [out]  void        *pbData,   
    [in]   ULONG       cbData   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pbData`  
 [out] Adresa, na kterém má být zápis metadat.  
  
 `cbData`  
 [in] Velikost v bajtech přidělené paměti.
