# <a name="imetadataimportgetsigfromtoken-method"></a>IMetaDataImport::GetSigFromToken – metoda
Získá metadata binární podpis přidružené k zadaným tokenu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetSigFromToken (   
   [in]   mdSignature        mdSig,   
   [out]  PCCOR_SIGNATURE    *ppvSig,   
   [out]  ULONG              *pcbSig   
);  
```  
  
## <a name="parameters"></a>Parametry  
 `mdSig`  
 [in] Token se vraťte podpis binárního metadat pro.  
  
 `ppvSig`  
 [out] Ukazatel na podpis vrácených metadat.  
  
 `pcbSig`  
 [out] Velikost v bajtech binárního metadat podpisu.
