# <a name="imetadataimportresetenum-method"></a>IMetaDataImport::ResetEnum – metoda
Zadaný čítač resetuje na určenou pozici.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,   
   [in] ULONG       ulPos  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `hEnum`  
 [in] Enumerátor resetovat.  
  
 `ulPos`  
 [in] Nové umístění, kam chcete umístit enumerátor.
