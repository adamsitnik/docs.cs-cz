|Metoda|Popis|  
|------------|-----------------|  
|[ValidateMetaData – metoda](../../../../docs/framework/unmanaged-api/metadata/imetadatavalidate-validatemetadata-method.md)|Ověří podpisy metadat objektů v aktuálním oboru metadat.|  
|[ValidatorInit – metoda](../../../../docs/framework/unmanaged-api/metadata/imetadatavalidate-validatorinit-method.md)|Nastaví příznak, který určuje typ modulu v aktuálním oboru metadata a zaregistruje zadaná metoda zpětného volání pro chyby ověření.|
