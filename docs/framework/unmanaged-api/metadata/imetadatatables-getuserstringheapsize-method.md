# <a name="imetadatatablesgetuserstringheapsize-method"></a>IMetaDataTables::GetUserStringHeapSize – metoda
Získá velikost v bajtech, řetězec haldy uživatele.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetUserStringHeapSize (  
    [out] ULONG   *pcbBlobs  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pcbBlobs`  
 [out] Ukazatel na velikost v bajtech, řetězec haldy uživatele.
