# <a name="imetadatainfo-interface"></a>IMetaDataInfo – rozhraní

|------------|-----------------|  
|[GetFileMapping – metoda](../../../../docs/framework/unmanaged-api/metadata/imetadatainfo-getfilemapping-method.md)|Získá paměťové oblasti mapovaný soubor a druh mapování.|
