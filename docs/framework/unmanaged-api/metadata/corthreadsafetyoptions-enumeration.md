# <a name="corthreadsafetyoptions-enumeration"></a>CorThreadSafetyOptions – výčet
Určuje příznaky a možnosti pro bezpečný přístup z více vláken.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
typedef enum CorThreadSafetyOptions {  
    MDThreadSafetyDefault       = 0x00000000,  
    MDThreadSafetyOff           = 0x00000000,  
    MDThreadSafetyOn            = 0x00000001,  
} CorThreadSafetyOptions;  
```  
  
## <a name="members"></a>Členové  
  
|Člen|Popis|  
|------------|-----------------|  
|`MDThreadSatetyDefault`|Výchozí hodnota. Stejné jako `MDThreadSatetyOff`.|  
|`MDThreadSatetyOff`|Označuje, že zámek čtení/zápis nelze nastavit.|  
|`MDThreadSatetyOn`|Označuje, že můžete nastavit zámek čtení/zápis.|
