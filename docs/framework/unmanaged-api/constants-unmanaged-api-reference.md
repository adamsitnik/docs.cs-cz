# <a name="constants-unmanaged-api-reference"></a>Konstanty (referenční dokumentace nespravovaného rozhraní API)
Toto téma popisuje typ jazyka, jazyk dodavatele a konstanty typu dokumentu, které jsou definovány v CorSym.idl.  
  
## <a name="language-type-constants"></a>Konstanty jazyka zadejte  
 Následující tabulka uvádí jazykové konstanty typů, které představují identifikátory GUID, které identifikují programovacích jazyků.  
  
|Symbol|Popis|  
|------------|-----------------|  
|CorSym_LanguageType_C|Určuje jazyk C.|  
|CorSym_LanguageType_CPlusPlus|Určuje jazyk C++.|  
|CorSym_LanguageType_CSharp|Označuje, C# jazyka.|  
|CorSym_LanguageType_Basic|Určuje základní jazyk.|  
|CorSym_LanguageType_Java|Určuje jazyk Java.|  
|CorSym_LanguageType_Cobol|Určuje jazyk, COBOL.|  
|CorSym_LanguageType_Pascal|Určuje jazyk, Pascal.|  
|CorSym_LanguageType_ILAssembly|Určuje kód sestavení Microsoft intermediate language (MSIL).|  
|CorSym_LanguageType_JScript|Určuje jazyk JScript.|  
|CorSym_LanguageType_SMC|Určuje jazyk, SMC.|  
|CorSym_LanguageType_MCPlusPlus|Určuje jazyk C++ povoleny pro rozhraní .NET Framework.|  
  
## <a name="language-vendor-constants"></a>Konstanty jazyka dodavatele  
 Následující tabulka uvádí jazykové dodavatele konstanty, které představují identifikátory GUID, které identifikují programovací jazyk dodavatelů.  
  
|Symbol|Popis|  
|------------|-----------------|  
|CorSym_LanguageVendor_Microsoft|Označuje společnosti Microsoft.|  
  
## <a name="document-type-constants"></a>Konstanty typu dokumentu  
 Následující tabulka uvádí dokument, typ konstanty, které představují identifikátory GUID, které identifikují typů dokumentů.  
  
|Symbol|Popis|  
|------------|-----------------|  
|CorSym_DocumentType_Text|Určuje textový dokument.|  
|CorSym_DocumentType_MC|Označuje než textovém dokumentu.|
