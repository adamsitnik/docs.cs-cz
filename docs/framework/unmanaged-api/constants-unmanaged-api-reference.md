# <a name="constants-unmanaged-api-reference"></a>Konstanty (referenční dokumentace nespravovaného rozhraní API)
Toto téma popisuje typ jazyka, dodavatele jazyka a konstanty typu dokumentu, které jsou definovány v souboru CorSym. idl.  
  
## <a name="language-type-constants"></a>Konstanty typu jazyka  
 V následující tabulce jsou uvedeny konstanty typu jazyka, které reprezentují identifikátory GUID, které identifikují programovací jazyky.  
  
|Písmeno|Popis|  
|------------|-----------------|  
|CorSym_LanguageType_C|Určuje jazyk C.|  
|CorSym_LanguageType_CPlusPlus|Určuje C++ jazyk.|  
|CorSym_LanguageType_CSharp|Určuje C# jazyk.|  
|CorSym_LanguageType_Basic|Označuje základní jazyk.|  
|CorSym_LanguageType_Java|Označuje jazyk Java.|  
|CorSym_LanguageType_Cobol|Určuje jazyk COBOL.|  
|CorSym_LanguageType_Pascal|Určuje jazyk Pascal.|  
|CorSym_LanguageType_ILAssembly|Označuje kód sestavení jazyka MSIL (Microsoft Intermediate Language).|  
|CorSym_LanguageType_JScript|Určuje jazyk JScript.|  
|CorSym_LanguageType_SMC|Označuje jazyk SMC.|  
|CorSym_LanguageType_MCPlusPlus|Určuje C++ jazyk povolený pro .NET Framework.|  
  
## <a name="language-vendor-constants"></a>Konstanty dodavatele jazyka  
 V následující tabulce jsou uvedeny konstanty dodavatelů jazyka, které reprezentují identifikátory GUID, které identifikují dodavatele programovacích jazyků.  
  
|Písmeno|Popis|  
|------------|-----------------|  
|CorSym_LanguageVendor_Microsoft|Označuje Microsoft.|  
  
## <a name="document-type-constants"></a>Konstanty typu dokumentu  
 V následující tabulce jsou uvedeny konstanty typu dokumentu, které reprezentují identifikátory GUID, které identifikují typy dokumentů.  
  
|Písmeno|Popis|  
|------------|-----------------|  
|CorSym_DocumentType_Text|Označuje textový dokument.|  
|CorSym_DocumentType_MC|Indikuje, že dokument není typu text.|
