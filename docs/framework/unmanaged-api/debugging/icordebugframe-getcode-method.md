## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetCode (  
    [out] ICorDebugCode      **ppCode  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `ppCode`  
 mimo Ukazatel na adresu objektu ICorDebugCode, který představuje kód přidružený k tomuto snímku.
