# <a name="icordebugframeenum-interface"></a>ICorDebugFrameEnum – rozhraní

Implementuje metody ICorDebugEnum a vytváří výčet polí ICorDebugFrame.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[Next – metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugframeenum-next-method.md)|Získá zadaný počet `ICorDebugFrame` instancí z výčtu od aktuální pozice.|
