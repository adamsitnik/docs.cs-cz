|Metoda|Popis|  
|------------|-----------------|  
|[Next – metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugvalueenum-next-method.md)|Získá zadaný počet `ICorDebugValue` instancí z výčtu od aktuální pozice.|
