# <a name="icordebugthreadgetregisterset-method"></a>ICorDebugThread::GetRegisterSet – metoda
Získá ukazatel rozhraní na sadu registrů, která je přidružena k aktivní části tohoto objektu ICorDebugThread.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetRegisterSet (  
    [out] ICorDebugRegisterSet **ppRegisters  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `ppRegisters`  
 mimo Ukazatel na adresu objektu rozhraní [ICorDebugRegisterSet](../../../../docs/framework/unmanaged-api/debugging/icordebugregisterset-interface.md) , která představuje sadu registrů pro aktivní část tohoto vlákna.
