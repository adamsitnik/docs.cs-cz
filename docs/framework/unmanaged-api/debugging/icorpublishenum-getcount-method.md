# <a name="icorpublishenumgetcount-method"></a>ICorPublishEnum::GetCount – metoda
Získá počet položek ve výčtu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetCount (  
    [out] ULONG   *pcelt  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pcelt`  
 [out] Ukazatel na počet položek ve výčtu.
