# <a name="icordebugprocess7-interface"></a>Rozhraní ICorDebugProcess7
[Podporované v .NET Framework 4.5.2 a novějších verzích]  
  
 Poskytuje metodu, která konfiguruje ladicí program pro zpracování aktualizací metadat v paměti v cílovém procesu.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[SetWriteableMetadataUpdateMode – metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugprocess7-setwriteablemetadataupdatemode-method.md)|Nastaví hodnotu, která určuje, jak ladicí program zpracovává aktualizace v paměti metadat v rámci cílového procesu.|
