Získá token metadat pro tuto funkci.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetToken (  
    [out] mdMethodDef *pMethodDef  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pMethodDef`  
 mimo Ukazatel na token `mdMethodDef`, který odkazuje na metadata této funkce.
