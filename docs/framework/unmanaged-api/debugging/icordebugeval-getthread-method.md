# <a name="icordebugevalgetthread-method"></a>ICorDebugEval::GetThread – metoda
Získá vláken, ve kterém toto testování je provádění nebo se spustí.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetThread (  
    [out] ICorDebugThread   **ppThread  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `ppThread`  
 [out] Ukazatel na adresu icordebugthread – objekt, který představuje vlákno.
