# <a name="attributes-visual-basic"></a>Atributy (Visual Basic)
Visual Basic poskytuje několik atributů, které umožňují že objekty spolupráce s nespravovaným kódem a jeden atribut, který umožňuje členům modulu přistupovat bez názvu modulu. Následující tabulka uvádí seznam atributů používá jazyka Visual Basic.  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.ComClassAttribute>|Instruuje kompilátor, chcete-li přidat metadata, která umožňuje vystavit jako objekt modelu COM třída.|  
|<xref:Microsoft.VisualBasic.HideModuleNameAttribute>|Umožňuje členům modulu přistupovat pomocí pouze kvalifikace potřebné pro modul.|  
|<xref:Microsoft.VisualBasic.VBFixedArrayAttribute>|Označuje, že pole ve struktuře nebo jiné než místní proměnné mají být považována za pole s pevnou délkou.|  
|<xref:Microsoft.VisualBasic.VBFixedStringAttribute>|Označuje, že řetězec by měl být zpracován jako opraveny délku.|
