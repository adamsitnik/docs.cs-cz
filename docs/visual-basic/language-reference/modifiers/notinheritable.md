# <a name="notinheritable-visual-basic"></a>NotInheritable (Visual Basic)
Určuje, zda třídu nelze použít jako základní třídu.  
  
## <a name="remarks"></a>Poznámky  
 **Alternativní podmínky**. Někdy se označuje jako třída, která nelze dědit *zapečetěné* třídy.  
  
 `NotInheritable` Modifikátor lze použít v tomto kontextu:
