# <a name="directories-and-files-summary-visual-basic"></a>Souhrn adresářů a souborů (Visual Basic)
Klíčová slova jazyka Visual Basic a členy běhové knihovny jsou uspořádány podle účelu a použití.  
  
 Použití `My` funkce vám poskytuje větší produktivitu a výkon operací vstupu a výstupu souborů než při použití těchto členů. Další informace naleznete v tématu <xref:Microsoft.VisualBasic.FileIO.FileSystem>.

|----------------|--------------------------|  
|Změna adresáře nebo složky.|<xref:Microsoft.VisualBasic.FileSystem.ChDir%2A>|

|Kopírování souboru.|<xref:Microsoft.VisualBasic.FileSystem.FileCopy%2A>|  
|Ujistěte se, adresáře nebo složky.|<xref:Microsoft.VisualBasic.FileSystem.MkDir%2A>|  
|Odeberte adresáře nebo složky.|<xref:Microsoft.VisualBasic.FileSystem.RmDir%2A>|
