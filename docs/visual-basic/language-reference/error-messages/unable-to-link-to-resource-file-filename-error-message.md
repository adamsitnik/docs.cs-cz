# <a name="unable-to-link-to-resource-file-filename-error-message"></a>Nelze vytvořit odkaz na soubor prostředků\<filename >: \<chybová zpráva >
Kompilátor Visual Basic volá linker sestavení (Al. exe, označovaný také jako ALink), aby vygeneroval sestavení s manifestem. Linker oznámil chybu propojení s nativním souborem prostředků modelu COM+ ze sestavení.  
  
 **ID chyby:** BC30144  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Prohlédněte si chybovou zprávu v uvozovkách a Projděte si téma [Al. exe](../../../framework/tools/al-exe-assembly-linker.md). pro další vysvětlení a Rady.  
  
2. Pokud chyba přetrvává, shromážděte informace o okolnostech a upozorněte služby podpory společnosti Microsoft.
