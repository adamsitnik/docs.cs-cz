# <a name="unable-to-create-strong-named-assembly-from-key-file-filename-error"></a>Nelze vytvořit sestavení se silným názvem z klíčového souboru '\<název souboru >': \<chyby >
Ze zadaného souboru klíče nelze vytvořit sestavení se silným názvem.  
  
 **ID chyby:** BC31026  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, že byl zadán správný soubor klíče, a že není uzamčený jinou aplikací.
