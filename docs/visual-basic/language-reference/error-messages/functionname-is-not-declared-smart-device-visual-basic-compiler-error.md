# <a name="39ltfunctionnamegt39-is-not-declared-smart-devicevisual-basic-compiler-error"></a>&#39;&lt;FunctionName&gt; &#39; není deklarována (inteligentního zařízení/jazyka Visual Basic chyba kompilátoru)
<`functionname`> není deklarován. Funkce pro vstupně-výstupních operací souborů jsou běžně k dispozici v `Microsoft.VisualBasic` obor názvů, ale cílové verzi rozhraní .NET Compact Framework je nepodporuje.  
  
 **ID chyby:** BC30766  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Provedení operace se soubory pomocí funkce definované v `System.IO` oboru názvů.
