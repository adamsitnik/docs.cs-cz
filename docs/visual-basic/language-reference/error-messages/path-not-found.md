# <a name="path-not-found"></a>Cesta nebyla nalezena.
Během operace přístupu k souborům nebo přístup k disku operačního systému se nepodařilo najít zadanou cestu. Cesta k souboru obsahuje specifikaci jednotky plus adresářů a podadresářů, které musí procházet a vyhledejte soubor. Cesta může být relativní nebo absolutní.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte a respecify cestu.
