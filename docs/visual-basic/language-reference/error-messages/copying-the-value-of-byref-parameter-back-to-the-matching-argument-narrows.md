 <span data-ttu-id="c04e1-109">Pokud `ByRef` hodnota argumentu je zkopírován do procedury a argumentu a parametru jsou stejného typu, je nezbytné žádný převod.</span><span class="sxs-lookup"><span data-stu-id="c04e1-109">If a `ByRef` argument value is copied into the procedure and the argument and parameter are of the same type, no conversion is necessary.</span></span> <span data-ttu-id="c04e1-110">Ale pokud se typy liší, musíte převést jazyka Visual Basic v obou směrech.</span><span class="sxs-lookup"><span data-stu-id="c04e1-110">But if the types are different, Visual Basic must convert in both directions.</span></span> <span data-ttu-id="c04e1-111">Pokud jeden z typů je typ třídy nebo struktury, musíte převést jazyka Visual Basic ji do i z jiného typu.</span><span class="sxs-lookup"><span data-stu-id="c04e1-111">If one of the types is your class or structure type, Visual Basic must convert it both to and from the other type.</span></span> <span data-ttu-id="c04e1-112">Pokud je jeden z těchto převodů rozšiřující, může být zúžení zpětný převod.</span><span class="sxs-lookup"><span data-stu-id="c04e1-112">If one of these conversions is widening, the reverse conversion might be narrowing.</span></span>
