# <a name="cannot-create-activex-component"></a>Součást prvku ActiveX nelze vytvořit.
Pokusili jste se umístit ovládacího prvku ActiveX ve formuláři v době návrhu nebo přidat formuláře do projektu s ovládacím prvkem ActiveX v něm, ale přidružené informace v registru nebyl nalezen.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Informace v registru byla odstraněna nebo poškozený. Znovu nainstalujte ovládací prvek ActiveX nebo se obraťte na dodavatele ovládacího prvku.
