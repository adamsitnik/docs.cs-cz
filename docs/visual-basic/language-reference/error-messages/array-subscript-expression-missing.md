# <a name="array-subscript-expression-missing"></a>Výraz dolního indexu pole chybí.
Inicializace pole zůstanou na jeden nebo více dolními indexy, které definují hranice pole. Příkaz může například obsahovat výraz `myArray (5,5,,10)`, což ponechá si třetí dolní index.  
  
 **ID chyby:** BC30306  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte chybí subscript.
