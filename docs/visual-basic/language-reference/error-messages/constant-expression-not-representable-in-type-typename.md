# <a name="constant-expression-not-representable-in-type-typename"></a>Konstantní výraz nejde reprezentovat typem '\<typename > "
Pokoušíte se vyhodnotit konstantu, která se nevejdou do cílového typu, obvykle, protože to je přetečení rozsahu.  
  
 **ID chyby:** BC30439  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Změna typu cíle, který dokáže zpracovat konstanty.
