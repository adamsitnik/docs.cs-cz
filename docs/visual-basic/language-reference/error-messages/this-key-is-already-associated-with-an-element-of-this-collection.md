# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>Tento klíč již je přidružen k elementu této kolekce.
Zadaný klíč pro člena kolekce již identifikuje jiného člena kolekce. Klíč je řetězec zadaný v poli `Add` metodu, která jednoznačně identifikuje konkrétního člena kolekce.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použijte jiný klíč pro tento člen.
