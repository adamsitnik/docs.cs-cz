# <a name="type-mismatch-visual-basic"></a>Neshoda typu (Visual Basic)
Pokusili jste se převést hodnotu na jiný typ způsobem, který není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte přiřazení, abyste měli jistotu, že je platný.  
  
2.  Zajistěte, aby že nebyla úspěšná objektu do procedury, která očekává, že jednu vlastnost nebo hodnota.  
  
3.  Ujistěte se, že jste nepoužili název modulu nebo projektu kde byl očekáván výraz.
