# <a name="expression-does-not-produce-a-value"></a>Výsledkem výrazu není hodnota.
Pokusili jste se použít výraz, který nevytváří hodnotu v kontextu vytváření hodnoty, jako je například volání `Sub` v kontextu kde `Function` očekává.  
  
 **ID chyby:** BC30491  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte výraz, který vytváří hodnotu.
