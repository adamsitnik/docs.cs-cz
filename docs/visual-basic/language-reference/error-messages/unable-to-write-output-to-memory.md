# <a name="unable-to-write-output-to-memory"></a>Nelze zapisovat výstup do paměti.
Došlo k potížím, zápis výstupu do paměti.  
  
 **ID chyby:** BC31020  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Kompilovat program znovu, abyste viděli, pokud chyba bude opakovat.  
  
2. Pokud chyba přetrvává, uložte si práci a restartujte aplikaci Visual Studio.  
  
3. Pokud se chyba objeví znovu, přeinstalujte Visual Basic.  
  
4. Pokud chyba přetrvává po přeinstalaci, upozorní Microsoft Product Support Services.
