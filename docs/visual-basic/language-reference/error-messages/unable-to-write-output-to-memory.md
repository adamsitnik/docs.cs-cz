# <a name="unable-to-write-output-to-memory"></a>Nelze zapisovat výstup do paměti.
Při zápisu výstupu do paměti došlo k potížím.  
  
 **ID chyby:** BC31020  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkompilujte program znovu, abyste viděli, zda se chyba opakuje.  
  
2. Pokud chyba přetrvává, uložte svou práci a restartujte aplikaci Visual Studio.  
  
3. Pokud se chyba opakuje, přeinstalujte Visual Basic.  
  
4. Pokud potíže potrvají i po přeinstalaci, upozorněte služby Microsoft Product Support Services.
