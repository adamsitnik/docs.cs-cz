# <a name="39ltmethodnamegt39-has-multiple-definitions-with-identical-signatures"></a>&#39;&lt;methodName&gt; &#39; má několik definic s identickými podpisy
A `Function` nebo `Sub` proceduru deklarace používá seznamu identické postupu názvem a argument jako předchozí deklarace. Jednou z možných příčin je pokus o přetížení původní procedury. Přetížené procedury musí mít seznamy argumentů s jinou.  
  
 **ID chyby:** BC30269  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte název procedury nebo seznamu argumentů nebo odeberte duplicitní deklarace.
