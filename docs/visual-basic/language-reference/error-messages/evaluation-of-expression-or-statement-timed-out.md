# <a name="evaluation-of-expression-or-statement-timed-out"></a>Časový limit vyhodnocení výrazu nebo příkazu vypršel.
Vyhodnocení výrazu nebyla dokončena v časovém limitu.  
  
 **ID chyby:** BC30722  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, že zadaný kód je správný.  
  
2.  Zjednodušte výraz tak, že bude trvat kratší dobu spuštění.
