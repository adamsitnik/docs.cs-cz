# <a name="evaluation-of-expression-or-statement-timed-out"></a>Časový limit vyhodnocení výrazu nebo příkazu vypršel.
Vyhodnocení výrazu se včas nedokončilo.  
  
 **ID chyby:** BC30722  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte správnost zadaného kódu.  
  
2. Zjednodušte svůj výraz tak, aby jeho spuštění trvalo méně času.
