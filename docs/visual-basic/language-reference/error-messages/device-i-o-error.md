# <a name="device-io-error"></a>Vstupně-výstupní chyba zařízení
Vstupní nebo výstupní došlo k chybě programu bylo pomocí zařízení, jako je například tiskárnu nebo diskovou jednotku.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že zařízení funguje správně a pak zkuste operaci zopakovat.
