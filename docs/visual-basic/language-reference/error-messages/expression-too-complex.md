# <a name="expression-too-complex"></a>Výraz je příliš složitý.
S plovoucí desetinnou čárkou výraz obsahuje příliš mnoho vnořených dílčích výrazů.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Podle potřeby chybu zabránit přerušit tolik samostatné výrazy výraz.
