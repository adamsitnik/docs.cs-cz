# <a name="textfieldparser-is-unable-to-complete-the-read-operation-because-maximum-buffer-size-has-been-exceeded"></a>TextFieldParser nemůže dokončit operaci čtení, protože byla překročena maximální velikost vyrovnávací paměti.
Operaci nelze dokončit, protože se překročila maximální velikost vyrovnávací paměti (10 000 000 bajtů).  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že v souboru nejsou žádná poškozená pole.
