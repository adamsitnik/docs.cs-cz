# <a name="line-is-too-long"></a>Řádek je příliš dlouhý.
Zdrojové řádky textu může mít maximálně 65535 znaků.  
  
 **ID chyby:** BC30494  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkraťte délku řádku do 65 535 znaků nebo méně.
