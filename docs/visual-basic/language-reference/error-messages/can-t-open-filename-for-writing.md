# <a name="cant-open-filename-for-writing"></a>Nelze otevřít '\<název souboru > "pro zápis.
Zadaný soubor nelze otevřít pro zápis, pravděpodobně protože je již otevřen.  
  
 **ID chyby:** BC2012  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zavřete a znovu ho otevřete.  
  
2. Zkontrolujte oprávnění.
