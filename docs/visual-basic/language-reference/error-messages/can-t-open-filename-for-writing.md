# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>Můžete&#39;t otevřít &#39; &lt;filename&gt; &#39; pro zápis.
Zadaný soubor nelze otevřít pro zápis, pravděpodobně protože je již otevřen.  
  
 **ID chyby:** BC2012  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zavřete a znovu ho otevřete.  
  
2.  Zkontrolujte oprávnění.
