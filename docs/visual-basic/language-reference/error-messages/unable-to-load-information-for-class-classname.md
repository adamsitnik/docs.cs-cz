# <a name="unable-to-load-information-for-class-classname"></a>Nepovedlo se načíst informace pro třídu\<ClassName >.
Byl proveden odkaz na třídu, která není k dispozici.  
  
 **ID chyby:** BC30712  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, zda je třída definována a zda jste název zadali správně.  
  
2. Zkuste získat přístup k jednomu z členů deklarovaným v modulu. V některých případech ladicí prostředí nemůže najít členy, protože moduly, ve kterých jsou deklarovány, ještě nebyly načteny.
