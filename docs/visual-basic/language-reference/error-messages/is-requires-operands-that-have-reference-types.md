# <a name="39is39-requires-operands-that-have-reference-types-but-this-operand-has-the-value-type-39lttypenamegt39"></a>&#39;Je&#39; vyžaduje operandy, které mají typy odkazů, ale tento operand má typ hodnoty &#39; &lt;typename&gt;&#39;
`Is` Operátor porovnání určuje, zda dva objektových proměnných odkazují na stejnou instanci. Toto porovnání není definován pro typy hodnot.  
  
 **ID chyby:** BC30020  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Operátor porovnání odpovídající aritmetické nebo `Like` operátor porovnání dvou typů hodnot.
