# <a name="unable-to-write-temporary-file-because-temporary-path-is-not-available"></a>Do dočasného souboru nelze zapisovat, protože dočasná cesta není k dispozici.
Visual Basic nemohl určit cestu, kam se ukládají dočasné soubory.  
  
 **ID chyby:** BC30698  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Restartujte sadu Visual Studio.  
  
2. Pokud se problém opakuje, přeinstalujte sadu Visual Studio.
