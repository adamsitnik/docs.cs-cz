# <a name="property-array-index-is-not-valid"></a>Index pole vlastností není platný.
Zadaná hodnota není platná pro index pole vlastností.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Abyste měli jistotu, že je indexu v platném rozsahu pro zadanou vlastnost v dokumentaci komponenty.
