# <a name="automation-error"></a>Chyba automatizace
Při provádění metody nebo získání nebo nastavení vlastnosti proměnné objektu došlo k chybě. Aplikace, který vytvořil objekt ohlásil chybu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte vlastnosti `Err` objektu určit zdroj a povaze chyby.  
  
2.  Použití `On Error Resume Next` příkaz bezprostředně před přístupem k příkazu a pak vyhledejte chyby ihned po přístupu k příkazu.
