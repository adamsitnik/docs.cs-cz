# <a name="cannot-convert-anonymous-type-to-an-expression-tree-because-a-property-of-the-type-is-used-to-initialize-another-property"></a><span data-ttu-id="36461-102">Anonymní typ nelze převést na strom výrazu, protože vlastnost tohoto typu je použita k inicializaci jiné vlastnosti.</span><span class="sxs-lookup"><span data-stu-id="36461-102">Cannot convert anonymous type to an expression tree because a property of the type is used to initialize another property</span></span>

<span data-ttu-id="36461-103">Kompilátor nepřijímá převod anonymního typu na strom výrazu, pokud je použita jedna vlastnost anonymního typu k inicializaci jiné vlastnosti anonymního typu.</span><span class="sxs-lookup"><span data-stu-id="36461-103">The compiler does not accept conversion of an anonymous type to an expression tree when one property of the anonymous type is used to initialize another property of the anonymous type.</span></span> <span data-ttu-id="36461-104">Například v následujícím kódu je `Prop1` deklarována v inicializačním seznamu a pak použita jako počáteční hodnota pro `Prop2`.</span><span class="sxs-lookup"><span data-stu-id="36461-104">For example, in the following code, `Prop1` is declared in the initialization list and then used as the initial value for `Prop2`.</span></span>

<span data-ttu-id="36461-105">**ID chyby:** BC36548</span><span class="sxs-lookup"><span data-stu-id="36461-105">**Error ID:** BC36548</span></span>

## <a name="example"></a><span data-ttu-id="36461-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="36461-106">Example</span></span>

```vb
Module M2

    Sub ExpressionExample(Of T)(x As Expressions.Expression(Of Func(Of T)))
    End Sub

    Sub Main()
        ' The following line causes the error.
        ExpressionExample(Function() New With {.Prop1 = 2, .Prop2 = .Prop1})
    End Sub
End Module
```

## <a name="to-correct-this-error"></a><span data-ttu-id="36461-107">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="36461-107">To correct this error</span></span>

<span data-ttu-id="36461-108">Přiřaďte počáteční hodnotu pro `Prop1` místní proměnné.</span><span class="sxs-lookup"><span data-stu-id="36461-108">Assign the initial value for `Prop1` to a local variable.</span></span> <span data-ttu-id="36461-109">Přiřaďte tuto proměnnou do obou `Prop1` a `Prop2`, jak je znázorněno v následujícím kódu:</span><span class="sxs-lookup"><span data-stu-id="36461-109">Assign that variable to both `Prop1` and `Prop2`, as shown in the following code:</span></span>

```vb
Sub Main()
    Dim temp = 2
    ExpressionExample(Function() New With {.Prop1 = temp, .Prop2 = temp})
End Sub
```
