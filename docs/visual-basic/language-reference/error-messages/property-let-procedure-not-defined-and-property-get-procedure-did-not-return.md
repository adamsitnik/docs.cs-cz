# <a name="property-let-procedure-not-defined-and-property-get-procedure-did-not-return-an-object"></a>Procedura Property Let není definována a procedura Property Get nevrátila objekt.
Některé operace, metody a vlastnosti lze použít pouze pro `Collection` objekty. Zadaná operace nebo vlastnost, která je výhradně pro kolekce, ale objekt není kolekce.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda název objektu nebo vlastnosti, nebo ověřte, zda je objekt `Collection` objektu.  
  
2.  Podívejte se na `Add` metoda používá k přidání objektu do kolekce, kterou chcete mít jistotu, syntaxe je správný a zda všechny identifikátory byly zadány správně.
