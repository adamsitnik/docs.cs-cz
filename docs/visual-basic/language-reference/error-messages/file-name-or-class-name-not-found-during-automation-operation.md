# <a name="file-name-or-class-name-not-found-during-automation-operation-visual-basic"></a>Název souboru nebo třídy nebyl v průběhu operace Automation nalezen. (Visual Basic)
Název stanovený pro název souboru nebo třídy ve volání `GetObject` funkce nebyla nalezena.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolovat názvy a zkuste to znovu. Ujistěte se, že je název sloužící k `class` parametr se shoduje s registrovaný v systému.
