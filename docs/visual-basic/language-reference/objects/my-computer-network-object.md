# <a name="mycomputernetwork-object"></a>My.Computer.Network – objekt
Poskytuje vlastnosti, události a metody pro práci se sítí, ke kterému je počítač připojený.  
  
## <a name="remarks"></a>Poznámky  
 Informace o tom, metody a vlastnosti `My.Computer.Network` objektu, najdete v článku <xref:Microsoft.VisualBasic.Devices.Network>.  
  
 Další informace najdete v tématu [provádění síťové operace](../../../visual-basic/developing-apps/programming/computer-resources/performing-network-operations.md).
