# <a name="mycomputerkeyboard-object"></a>My.Computer.Keyboard – objekt
Poskytuje vlastnosti pro přístup k aktuálním stavu klávesnice, jako je například co klíčů aktuálně stisknutí a představuje způsob, jak odeslat stisknutí kláves do aktivního okna.  
  
## <a name="remarks"></a>Poznámky  
 Informace o tom, metody a vlastnosti `My.Computer.Keyword` objektu, najdete v článku <xref:Microsoft.VisualBasic.Devices.Keyboard>.  
  
 Další informace najdete v tématu [přístup ke klávesnici](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-keyboard.md).  
  
## <a name="requirements"></a>Požadavky  
 **Namespace:** <xref:Microsoft.VisualBasic.Devices>  
  
 **Třída:** <xref:Microsoft.VisualBasic.Devices.Keyboard>  
  
 **Sestavení:** Visual Basic Runtime Library (v souboru Microsoft.VisualBasic.dll)
