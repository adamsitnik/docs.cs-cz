# <a name="myuser-object"></a>My.User – objekt
Poskytuje přístup k informacím o aktuálním uživateli.  
  
## <a name="remarks"></a>Poznámky  
 Informace o tom, metody a vlastnosti `My.User` objektu, najdete v článku <xref:Microsoft.VisualBasic.ApplicationServices.User?displayProperty=nameWithType>.  
  
 Další informace najdete v tématu [přístup k datům uživatele](../../../visual-basic/developing-apps/programming/accessing-user-data.md).
