# <a name="-visual-basic-reference"></a>$ (Referenční dokumentace jazyka Visual Basic)

Identifikuje řetězcového literálu jako [interpolovaný řetězec](../../programming-guide/language-features/strings/interpolated-strings.md). Interpolované řetězce je šablona jako řetězec, který obsahuje prostý text spolu s *interpolovaných výrazů*. Pokud interpolovaný řetězec nebude vyřešen, například v příkazu přiřazení nebo volání metody, jeho interpolovaných výrazů jsou nahrazené jejich řetězcové vyjádření ve výsledném řetězci. Interpolované řetězce se při nahrazování [složených formátovacích řetězců](../../../standard/base-types/composite-format.md) podporované rozhraním .NET Framework.
