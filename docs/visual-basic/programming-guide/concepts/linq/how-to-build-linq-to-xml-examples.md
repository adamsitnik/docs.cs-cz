# <a name="how-to-build-linq-to-xml-examples---visual-basic"></a>Postupy: sestavení LINQ to XML Příklady – Visual Basic

Jednotlivé fragmenty kódu a příklady v této dokumentaci využívají třídy a typy z různých oborů názvů. Při kompilování Visual Basic kódu je nutné dodat příslušné příkazy `Imports`.

## <a name="example"></a>Příklad

Následující kód obsahuje příkazy `Imports`, které Visual Basic příklady vyžadují pro sestavení a spuštění. Pro každý příklad nejsou vyžadovány všechny příkazy `Imports`.
  
```vb
Imports System.Diagnostics
Imports System.Collections
Imports System.Collections.Generic
Imports System.Collections.Specialized
Imports System.Text
Imports System.Linq
Imports System.Xml
Imports System.Xml.Linq
Imports System.Xml.Schema
Imports System.Xml.XPath
Imports System.Xml.Xsl
Imports System.IO
Imports System.Threading
Imports System.Reflection
Imports System.IO.Packaging
```
