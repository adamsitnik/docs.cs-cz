# <a name="language-integrated-query-linq-visual-basic"></a>Language-Integrated Query (LINQ) (Visual Basic)
LINQ je sada funkcí, které rozšiřuje výkonné možnosti dotazování syntaxe jazyků Visual Basic. Představuje standardní, snadno vzory pro dotazování a aktualizace dat LINQ a tato technologie je možné rozšířit na potenciálně jakéhokoli druhu úložiště dat podporují.  Rozhraní .NET Framework obsahuje sestavení zprostředkovatele LINQ, které umožňují použití technologie LINQ s kolekcemi rozhraní .NET Framework, databáze systému SQL Server, datovými sadami ADO.NET a dokumenty XML.
