# <a name="language-integrated-axes-in-visual-basic-linq-to-xml"></a>Osy integrované v jazyce v jazyce Visual Basic (LINQ to XML)
Tato část popisuje funkcí integrovaných přímo do jazyka Visual Basic k tomu, aby pro přístup k XML. Mnoho příkladů v LINQ k dokumentaci XML použít tyto integrované osy jazyka Visual Basic.  
  
## <a name="in-this-section"></a>V tomto oddílu  
  
|Téma|Popis|  
|-----------|-----------------|  
|[Vlastnost osy podřízeného XML](../../../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)|Poskytuje přístup k podřízené objekty daného jednu z následujících: <xref:System.Xml.Linq.XElement> objekt, <xref:System.Xml.Linq.XDocument> object, kolekce <xref:System.Xml.Linq.XElement> objekty nebo kolekci <xref:System.Xml.Linq.XDocument> objekty. Tato osa odpovídá <xref:System.Xml.Linq.XContainer.Elements%2A> osy.|  
|[Vlastnost osy nástupce XML](../../../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md)|Poskytuje přístup k následníky z následujících akcí: <xref:System.Xml.Linq.XElement> objekt, <xref:System.Xml.Linq.XDocument> objektu nebo kolekci <xref:System.Xml.Linq.XElement> objekty. Tato osa odpovídá <xref:System.Xml.Linq.XContainer.Descendants%2A> osy.|  
|[Vlastnost osy atributu XML](../../../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)|Poskytuje přístup k atributu <xref:System.Xml.Linq.XElement> objektu. Tato osa je zhruba odpovídá <xref:System.Xml.Linq.XElement.Attribute%2A> osy. Tato osa se liší od <xref:System.Xml.Linq.XElement.Attribute%2A> osy v, který vrátí hodnotu atributu, nikoli <xref:System.Xml.Linq.XAttribute> objektu.|  
|[Vlastnost indexeru rozšíření](../../../../visual-basic/language-reference/xml-axis/extension-indexer-property.md)|Poskytuje přístup k jednotlivým prvkům v kolekci.|
