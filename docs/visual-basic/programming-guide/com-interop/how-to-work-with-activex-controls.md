# <a name="how-to-work-with-activex-controls-visual-basic"></a>Postupy: Práce s ovládacími prvky ActiveX (Visual Basic)
Ovládací prvky ActiveX jsou součástí modelu COM nebo objekty, které můžete vložit do webové stránky nebo jiné aplikace pro opětovné použití zabalené funkce, na které má naprogramovat někdo jiný. Ovládací prvky ActiveX s vyvinutý pro Visual Basic 6.0 a starší verze můžete použít k přidání funkcí do **nástrojů** sady Visual Studio.  
  
### <a name="to-add-activex-controls-to-the-toolbox"></a>Přidání ovládacích prvků ActiveX do panelu nástrojů  
  
1. Na **nástroje** nabídky, klikněte na tlačítko **zvolit položky nástrojů**.  
  
     **Panelu nástrojů zvolte** zobrazí se dialogové okno.  
  
2. Klikněte na tlačítko **komponenty modelu COM** kartu.  
  
3. Zaškrtněte políčko vedle ovládacího prvku ActiveX, kterou chcete použít a potom klikněte na tlačítko **OK**.  
  
     Nový ovládací prvek se zobrazí s dalšími nástroji v **nástrojů**.  
  
    > [!NOTE]
    >  Ruční vytvoření sestavení vzájemné spolupráce pro ovládací prvky ActiveX lze použít nástroj Aximp. Další informace najdete v tématu [Aximp.exe (Importér ovládacích prvků ActiveX formulářů Windows)](../../../framework/tools/aximp-exe-windows-forms-activex-control-importer.md).
