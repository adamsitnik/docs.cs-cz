## <a name="passing-by-value"></a><span data-ttu-id="64b95-106">Předání hodnotou</span><span class="sxs-lookup"><span data-stu-id="64b95-106">Passing by Value</span></span>  
 <span data-ttu-id="64b95-107">Předat argument *podle hodnoty* zadáním [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) – klíčové slovo pro odpovídající parametr v definici procedury.</span><span class="sxs-lookup"><span data-stu-id="64b95-107">You pass an argument *by value* by specifying the [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) keyword for the corresponding parameter in the procedure definition.</span></span> <span data-ttu-id="64b95-108">Při použití tohoto mechanismu předávání jazyka Visual Basic kopíruje hodnotu základní programovací element do místní proměnné v postupu.</span><span class="sxs-lookup"><span data-stu-id="64b95-108">When you use this passing mechanism, Visual Basic copies the value of the underlying programming element into a local variable in the procedure.</span></span> <span data-ttu-id="64b95-109">Kód procedury nemá přístup k podkladové element ve volajícím kódu.</span><span class="sxs-lookup"><span data-stu-id="64b95-109">The procedure code does not have any access to the underlying element in the calling code.</span></span>
