# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="dcdb9-102">Rozlišení přetěžování (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="dcdb9-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="dcdb9-103">Když kompilátor jazyka Visual Basic dojde během volání procedury, která je definována v několika přetížené verze, kompilátor musí rozhodnout, které přetížení volání.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="dcdb9-104">Dělá to pomocí následujících kroků:</span><span class="sxs-lookup"><span data-stu-id="dcdb9-104">It does this by performing the following steps:</span></span>  
  
1.  <span data-ttu-id="dcdb9-105">**Přístupnost**</span><span class="sxs-lookup"><span data-stu-id="dcdb9-105">**Accessibility.**</span></span> <span data-ttu-id="dcdb9-106">Eliminuje žádného přetížení s úrovní přístupu, který brání volání volající kód.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2.  <span data-ttu-id="dcdb9-107">**Počet parametrů.**</span><span class="sxs-lookup"><span data-stu-id="dcdb9-107">**Number of Parameters.**</span></span> <span data-ttu-id="dcdb9-108">Eliminuje žádného přetížení, která definuje různý počet parametrů než je zadaný ve volání.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3.  <span data-ttu-id="dcdb9-109">**Datové typy parametrů.**</span><span class="sxs-lookup"><span data-stu-id="dcdb9-109">**Parameter Data Types.**</span></span> <span data-ttu-id="dcdb9-110">Kompilátor dává přednost metody instance prostřednictvím metody rozšíření.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="dcdb9-111">Pokud se najde libovolnou metodu instance, který se vyžaduje jenom na rozšiřující převody tak, aby odpovídaly volání procedury, se zahodí všechny metody rozšíření a kompilátor pokračuje pouze kandidáty metodu instance.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="dcdb9-112">Pokud se nenajde žádný takový metodu instance, bude pokračovat s instancí a metody rozšíření.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="dcdb9-113">V tomto kroku eliminuje žádného přetížení, pro které datové typy argumentů volání nelze převést na parametr typy definované v přetížení.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4.  <span data-ttu-id="dcdb9-114">**Zužující převody.**</span><span class="sxs-lookup"><span data-stu-id="dcdb9-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="dcdb9-115">Eliminuje žádného přetížení, která vyžaduje zúžení převodu z: volání typy argumentů pro typy definované parametrů.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="dcdb9-116">Je hodnota true Určuje, zda přepnout kontrola typu ([Option Strict – příkaz](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) je `On` nebo `Off`.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5.  <span data-ttu-id="dcdb9-117">**Rozšiřující nejnižší.**</span><span class="sxs-lookup"><span data-stu-id="dcdb9-117">**Least Widening.**</span></span> <span data-ttu-id="dcdb9-118">Kompilátor považuje za zbývající přetížení v párech.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="dcdb9-119">Pro každý pár porovná datové typy definované parametry.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="dcdb9-120">Pokud se typy v jednom ze všech přetížení rozšířit na odpovídající typy v jiném, kompilátor eliminuje ten.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="dcdb9-121">To znamená zachová přetížení, která vyžaduje nejnižší možné rozšíření.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6.  <span data-ttu-id="dcdb9-122">**Jeden Release Candidate.**</span><span class="sxs-lookup"><span data-stu-id="dcdb9-122">**Single Candidate.**</span></span> <span data-ttu-id="dcdb9-123">Pokračuje v considering přetížení v párech až do pouze jedno přetížení zůstane a přeloží volání tohoto přetížení.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="dcdb9-124">Pokud kompilátor nemůže snížit přetížení k jedné Release candidate, dojde k chybě.</span><span class="sxs-lookup"><span data-stu-id="dcdb9-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
