# <a name="constants-and-enumerations-in-visual-basic"></a>Konstanty a výčty v jazyce Visual Basic
Konstanty jsou způsob, jak použijte smysluplné názvy místo hodnotu, která se nezmění. Konstanty ukládání hodnot, které, jak již název napovídá, zůstanou neměnný po celou dobu spuštění aplikace. Konstanty slouží k poskytování smysluplné názvy, namísto čísla, aby váš kód lépe čitelný.  
  
 Výčty poskytují pohodlný způsob pro práci se sadami související s konstantami a přidružení konstantních hodnot s názvy. Můžete třeba deklarovat výčet sady konstanty typu integer, které jsou spojené s dny v týdnu a potom použít názvy dní, namísto jejich celočíselných hodnot ve vašem kódu.
