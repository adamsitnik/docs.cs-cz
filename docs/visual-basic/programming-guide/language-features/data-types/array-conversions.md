## <a name="conversion-to-an-object-array"></a><span data-ttu-id="6abb2-117">Převod na pole objektů</span><span class="sxs-lookup"><span data-stu-id="6abb2-117">Conversion to an Object Array</span></span>  
 <span data-ttu-id="6abb2-118">Pokud deklarujete `Object` pole bez inicializuje, jeho typ elementu je `Object` dlouho, dokud zůstává Neinicializovaný.</span><span class="sxs-lookup"><span data-stu-id="6abb2-118">When you declare an `Object` array without initializing it, its element type is `Object` as long as it remains uninitialized.</span></span> <span data-ttu-id="6abb2-119">Pokud ji nastavíte na pole určité třídy, převezme typu třídy.</span><span class="sxs-lookup"><span data-stu-id="6abb2-119">When you set it to an array of a specific class, it takes on the type of that class.</span></span> <span data-ttu-id="6abb2-120">Jeho základní typ je však stále `Object`, a můžete ho nastavit následně do jiného objektu array nesouvisejících třídy.</span><span class="sxs-lookup"><span data-stu-id="6abb2-120">However, its underlying type is still `Object`, and you can subsequently set it to another array of an unrelated class.</span></span> <span data-ttu-id="6abb2-121">Protože všechny třídy odvozovat z `Object`, typ elementu pole z jiné třídy, můžete změnit na jiné třídy.</span><span class="sxs-lookup"><span data-stu-id="6abb2-121">Since all classes derive from `Object`, you can change the array's element type from any class to any other class.</span></span>  
  
 <span data-ttu-id="6abb2-122">V následujícím příkladu, neexistuje žádný převod mezi typy `student` a `String`, ale oba jsou odvozeny z `Object`, takže platí všechna přiřazení.</span><span class="sxs-lookup"><span data-stu-id="6abb2-122">In the following example, no conversion exists between types `student` and `String`, but both derive from `Object`, so all assignments are valid.</span></span>  
  
```  
' Assume student has already been defined as a class.  
Dim testArray() As Object  
' testArray is still an Object array at this point.  
Dim names() As String = New String(3) {"Name0", "Name1", "Name2", "Name3"}  
testArray = New student(3) {}  
' testArray is now of type student().  
testArray = names  
' testArray is now a String array.  
```  
  
### <a name="underlying-type-of-an-array"></a><span data-ttu-id="6abb2-123">Základní typ pole</span><span class="sxs-lookup"><span data-stu-id="6abb2-123">Underlying Type of an Array</span></span>  
 <span data-ttu-id="6abb2-124">Pokud deklarujete původně pole s konkrétní třídou, je jeho základní typ elementu třídy.</span><span class="sxs-lookup"><span data-stu-id="6abb2-124">If you originally declare an array with a specific class, its underlying element type is that class.</span></span> <span data-ttu-id="6abb2-125">Pokud následně ji nastavíte na pole jiné třídy, musí být převod mezi dvěma třídami.</span><span class="sxs-lookup"><span data-stu-id="6abb2-125">If you subsequently set it to an array of another class, there must be a conversion between the two classes.</span></span>  
  
 <span data-ttu-id="6abb2-126">V následujícím příkladu `students` je `student` pole.</span><span class="sxs-lookup"><span data-stu-id="6abb2-126">In the following example, `students` is a `student` array.</span></span> <span data-ttu-id="6abb2-127">Protože neexistuje žádný převod mezi `String` a `student`, poslední příkaz se nezdaří.</span><span class="sxs-lookup"><span data-stu-id="6abb2-127">Since no conversion exists between `String` and `student`, the last statement fails.</span></span>  
  
```  
Dim students() As student  
Dim names() As String = New String(3) {"Name0", "Name1", "Name2", "Name3"}  
students = New Student(3) {}  
' The following statement fails at compile time.

```
