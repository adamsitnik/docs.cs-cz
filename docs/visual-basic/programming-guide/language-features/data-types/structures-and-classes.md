## <a name="differences"></a><span data-ttu-id="ea6f4-120">Rozdíly</span><span class="sxs-lookup"><span data-stu-id="ea6f4-120">Differences</span></span>

Třída může být úplně prázdná.</span><span class="sxs-lookup"><span data-stu-id="ea6f4-127">A structure must have at least one nonshared variable or nonshared, noncustom event element; a class can be completely empty.</span></span>

- <span data-ttu-id="ea6f4-133">Struktury nejsou odvoditelný; třídy jsou.</span><span class="sxs-lookup"><span data-stu-id="ea6f4-133">Structures are not inheritable; classes are.</span></span>
