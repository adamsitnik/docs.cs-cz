# <a name="redim-preserve-operand-cannot-be-nothing"></a>Operand příkazu ReDim nemůže být Nothing.
Příkaz se pokusí `Preserve` použít klíčové slovo pro změnu rozměru pole, které není poslední dimenzí, ale neposkytuje platnou hodnotu pro svůj operand. `ReDim`  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- `Preserve` Změňte operand na platnou hodnotu.
