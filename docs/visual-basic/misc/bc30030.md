# <a name="try-must-have-at-least-one-catch-or-a-finally"></a>Výraz Try musí mít aspoň jeden "Catch" nebo "Finally"
A `Try` bloku musí obsahovat buď `Finally` bloku nebo alespoň jeden `Catch` blokovat před `End Try` příkazu.  
  
 **ID chyby:** BC30030  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Catch` nebo `Finally` blokovat v rámci `Try` bloku.
