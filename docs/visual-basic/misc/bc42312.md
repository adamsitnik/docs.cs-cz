# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>Blok komentáře XML nelze přidružit libovolný element jazyka, který podporuje aplikace dokumentační komentáře XML
Blok komentáře XML nemůže být spojen s prvkem any platný jazyk. Komentář XML bude ignorován. K této chybě dojde, pokud značku XML v blok komentáře nemá žádný odpovídající element jazyka.  
  
 **ID chyby:** BC42312  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Najít a odebrat nadbytečné značku XML nebo zadejte prvek jazyka, který popisuje.
