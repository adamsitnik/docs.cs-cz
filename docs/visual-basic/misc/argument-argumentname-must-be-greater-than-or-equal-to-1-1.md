# <a name="argument-argumentname-must-be-greater-than-or-equal-to--1"></a>Argument '\<název argumentu argumentname >' musí být větší než nebo roven -1
Hodnota argumentu je menší než -1.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že hodnota argumentu spadá v platném rozsahu.  
  
2.  Pokud je argument zadaný ve výpočtu, zkontrolujte hodnoty procesy výpočtu.
