# <a name="unable-to-access-member"></a>Nelze získat přístup k člena
Zadaný člen není přístupný.  
  
 **ID chyby:** BC30749  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, že člen je definovaný a jestli mají název člena napsaný správně.  
  
2. Zkuste jednu z členy deklarované v modulu. V některých případech nelze ladicího prostředí najít členy, protože ještě nebyly načtené moduly, ve kterém jsou deklarovány.
