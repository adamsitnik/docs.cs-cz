# <a name="file-ltfilenamegt-cannot-be-deleted-because-it-is-open"></a>Soubor '&lt;filename&gt;"nejde odstranit, protože je otevřen
Na otevřený soubor Probíhá pokus o odstranění.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zavřete soubor před jeho odstraněním.
