# <a name="file-filename-cannot-be-deleted-because-it-is-open"></a>Soubor '\<název souboru > "nejde odstranit, protože je otevřen
Na otevřený soubor Probíhá pokus o odstranění.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zavřete soubor před jeho odstraněním.
