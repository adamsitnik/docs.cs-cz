# <a name="query-expressions-are-not-valid-in-debug-windows"></a>Výrazy dotazu nejsou platné v ladicích oknech
Byl zadán v dotazu LINQ **Watch** nebo **okamžité** okno během relace ladění. Dotazy nejsou k dispozici v ladicích oknech.  
  
 **ID chyby:** BC36609
