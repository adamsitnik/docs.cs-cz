# <a name="cast-from-string-quotltstringgtquot-to-type-lttypenamegt-is-not-valid"></a>Přetypování z řetězce &quot; &lt;řetězec&gt; &quot; na typ '&lt;typename&gt;' je neplatný
Pokus o převod proměnné řetězce k jinému typu není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte zapojených do převodu typů a ujistěte se, že se pokoušíte převést na typ smysluplné.
