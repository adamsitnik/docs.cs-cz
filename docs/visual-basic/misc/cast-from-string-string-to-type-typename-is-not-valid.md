# <a name="cast-from-string-quotstringquot-to-type-typename-is-not-valid"></a>Přetypování z řetězce &quot; \<řetězec >&quot; na typ '\<typename >' není platný
Pokus o převod proměnné řetězce k jinému typu není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte zapojených do převodu typů a ujistěte se, že se pokoušíte převést na typ smysluplné.
