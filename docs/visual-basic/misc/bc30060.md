# <a name="conversion-from-lttype1gt-to-lttype2gt-cannot-occur-in-a-constant-expression"></a>Převod z '&lt;type1&gt;"do"&lt;type2&gt;"nelze provést v konstantním výrazu
Výraz inicializace v `Const` vyhodnotí na datový typ, který nelze převést na deklarovaný typ konstanty.  
  
 **ID chyby:** BC30060  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Inicializujte – konstanta s výrazem datový typ, který lze převést na typ deklarovaný pro konstantu.
