# <a name="as-expected"></a>Podle očekávání
Dojde k deklaraci, která neposkytuje `As` klauzuli pro jeden nebo více prvků byl deklarován.  
  
 **ID chyby:** BC30197  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Poskytnout `As` klauzule podle potřeby, nebo vypněte `Option Strict Off`.
