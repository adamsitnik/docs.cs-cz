# <a name="stop-request-is-pending"></a>Požadavek na zastavení čeká na vyřízení.
V ladicím programu sady Visual Studio určuje výraz volání procedury, ale existuje žádost o zastavení vlákna.  
  
 Ladicí program se nepokouší volat proceduru v neaktivním vlákně.  
  
 **ID chyby:** BC30946  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pokud je to možné, určete zdroj žádosti o zastavení vlákna, aby bylo možné zabránit tomu, aby se znovu stalo.  
  
- Ukončí a znovu spustí ladění.
