# <a name="interface-interfacename-is-not-implemented-by-this-class"></a>Rozhraní '\<interfacename >' není touto třídou implementována.
Člena této třídy nebo struktury se pokouší implementovat člen rozhraní, třídy nebo struktury neimplementuje.  
  
 **ID chyby:** BC31035  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidat `Implements` příkaz na začátek třídy nebo struktury tak, že implementuje odpovídající rozhraní.  
  
- Odeberte `Implements` – klíčové slovo od člena, který způsobí, že k této chybě.
