# <a name="exit-operator-is-not-valid-use-return-to-exit-an-operator"></a>'Klíčové slovo exit operátor"není platný. K ukončení operátoru použijte příkaz 'Return'
`Exit Operator` Příkazu se zobrazí v `Operator` postup.  
  
 Je nutné použít [příkaz Return](../../visual-basic/language-reference/statements/return-statement.md) vrácení z `Operator` postup. [Příkaz Exit](../../visual-basic/language-reference/statements/exit-statement.md) nepřijímá `Operator` – klíčové slovo a `End Operator` příkaz nevrací řízení volajícímu kódu.  
  
 **ID chyby:** BC33008  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahradit `Exit Operator` příkazem `Return` příkazu.
