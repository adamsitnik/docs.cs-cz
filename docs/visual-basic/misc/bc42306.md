# <a name="xml-comment-tag-tag-is-not-permitted-on-a-element-language-element"></a>Značka komentáře XML "\<značka >' není povolená u"\<element >' prvek jazyka
Značka komentáře XML \<značka > není povolená u "\<element >' elementu. Komentář XML bude ignorován.  
  
 Značka komentáře XML není podporován jazyk elementu, na který se použije.  
  
 **ID chyby:** BC42306  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nepodporované značka komentáře XML.
