# <a name="end-function-expected"></a>Byl očekáván příkaz 'end funkce.
`End` Příkaz musí odpovídat bloku jeho ukončení. A `Function` definici procedury nebyl správně zavřen.  
  
 **ID chyby:** BC30027  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidat `End Function` příkaz na konec procedury.
