# <a name="cannot-determine-array-type-because-it-is-nothing"></a>Typ pole nelze určit, protože má hodnotu Nothing
Typ vaše pole nelze určit, protože byla nastavena na hodnotu `Nothing`.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zadejte hodnotu jiné než `Nothing` pro pole.
