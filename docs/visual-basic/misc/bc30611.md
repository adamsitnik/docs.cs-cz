# <a name="array-dimensions-cannot-have-a-negative-size"></a>Rozměry pole nemohou být záporná čísla
Jeden nebo více mezí pole zadané je záporné číslo. Záporný subscript můžete zadat, jenom když použijete horní hranici -1, chcete-li deklarovat prázdné pole. Tato pole má nulovým počtem elementů, ale to není `Nothing`.  
  
 **ID chyby:** BC30611  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte negativní pole navázaný kladné číslo.
