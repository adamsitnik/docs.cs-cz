# <a name="inheriting-from-systemltclassnamegt-is-not-valid"></a>Dědění z ' System. &lt;classname&gt;' je neplatný
Třída nemůže dědit od třídy `System.Array`, `System.Delegate`, `System.Enum`, nebo `System.ValueType`.  
  
 **ID chyby:** BC30015  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Inherits` příkaz nebo změňte ji na platný základní třídy dědí.
