# <a name="inheriting-from-systemclassname-is-not-valid"></a>Dědění z ' System. \<classname >' není platný
Třída nemůže dědit od třídy `System.Array`, `System.Delegate`, `System.Enum`, nebo `System.ValueType`.  
  
 **ID chyby:** BC30015  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Inherits` příkaz nebo změňte ji na platný základní třídy dědí.
