# <a name="modifier-is-not-valid-on-an-interface-declaration"></a>'\<modifikátor >' není platný v deklaraci rozhraní
Použili jste modifikátor, který není platný v `Interface` deklaraci. Jedinými platnými modifikátory `Sub`pro `Function`příkazy, `Property` , `Interface` nebo deklarované v deklaraci jsou `Overloads` klíčová slova a `Default` . Jiné modifikátory, `Public` `Private` jakonapříklad`Overrides`,,, `Shared` `Static`,, ,`MustOverride` a nejsouplatné`Overridable`. `Protected` `Friend`  
  
 **ID chyby:** BC30397  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte modifikátor.
