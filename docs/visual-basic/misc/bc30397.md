# <a name="modifier-is-not-valid-on-an-interface-declaration"></a>"\<modifikátor >' není platný v deklaraci rozhraní
Použili jste modifikátor, který není platný pro `Interface` deklarace. Platné pouze modifikátory pro `Sub`, `Function`, nebo `Property` příkazy deklarované v `Interface` deklarace jsou `Overloads` a `Default` klíčová slova. Další modifikátory, jako například `Public`, `Private`, `Friend`, `Protected`, `Shared`, `Static`, `Overrides`, `MustOverride` a `Overridable`, nejsou platné.  
  
 **ID chyby:** BC30397  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte modifikátor.
