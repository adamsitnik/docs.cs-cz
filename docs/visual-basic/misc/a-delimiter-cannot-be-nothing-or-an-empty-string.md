# <a name="a-delimiter-cannot-be-nothing-or-an-empty-string"></a>Oddělovač nemůže být Nothing nebo prázdný řetězec
`TextFieldParser` Nejsou schopny číst ze souboru, protože `Delimiters` je nastavena na `Nothing` nebo je prázdná `String` ("").  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte platnou hodnotu pro `Delimiters`.
