# <a name="method-call-did-not-return-a-value"></a>Volání metody nevrátil hodnotu
Metoda nevrátil hodnotu podle očekávání.  
  
 **ID chyby:** BC30723  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, že metoda volá funkci, která vrací hodnotu.
