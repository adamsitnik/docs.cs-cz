# <a name="method-call-did-not-return-a-value"></a>Volání metody nevrátilo hodnotu.
Metoda volala nevrátila hodnotu podle očekávání.  
  
 **ID chyby:** BC30723  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, že volaná metoda je funkce, která vrací hodnotu.
