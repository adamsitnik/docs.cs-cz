# <a name="argument-ltargumentnamegt-cannot-be-converted-to-type-date"></a>Argument '&lt;název argumentu argumentname&gt;' nelze převést na typ "Datum"
Převod se pokusila změnit proměnné na typ `Date`, ale hodnoty se nerozpoznal jako platná data.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Najdete převod, který má smysl.
