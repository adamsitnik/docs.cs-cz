 <span data-ttu-id="3e1b0-104">To může dojít, pokud nemá žádné konstruktory základní třídy určené nebo pokud má konstruktory s úrovní přístupu, které brání přístupu z jiné třídy.</span><span class="sxs-lookup"><span data-stu-id="3e1b0-104">This can happen if the intended base class has no constructors or if it has constructors with access levels that prevent access from another class.</span></span>
