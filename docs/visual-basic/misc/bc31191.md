# <a name="error-embedding-win32-manifest-ltmanifestgt"></a>Chyba při vkládání manifestu Win32: &lt;manifestu&gt;
Zadaný pro soubor `/win32manifest` – možnost kompilátoru nemůže být vložený. To může být výsledkem nedostatečná přístupová oprávnění k souboru manifestu.  
  
 **ID chyby:** BC31191  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že identita spuštěním příkazu kompilátoru jazyka Visual Basic má přístup k souboru manifestu Win32.
