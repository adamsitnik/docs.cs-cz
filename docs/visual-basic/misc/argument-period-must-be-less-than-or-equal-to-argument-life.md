# <a name="argument-period-must-be-less-than-or-equal-to-argument-life"></a>Argument období musí být menší než nebo rovná hodnotě argumentu 'život.
Hodnota `Period` argument, který určuje období, pro které prostředek odpisů počítá, je větší než hodnota `Life` argument.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, `Life` a `Period` argumenty jsou vyjádřeny ve stejných jednotkách. Například pokud `Life` se měří v měsících, `Period` by měl být také.
