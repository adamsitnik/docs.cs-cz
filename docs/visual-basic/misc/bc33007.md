# <a name="end-operator-must-be-preceded-by-a-matching-operator"></a>'End Operator' musí předcházet odpovídající 'Operator'
`End Operator` Příkazu se zobrazí s žádná odpovídající `Operator` před deklarací.  
  
 **ID chyby:** BC33007  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `End Operator` příkazu, pokud je redundantní.  
  
-   Zadat chybějící `Operator` postup, pokud chybí.  
  
-   Přesunout `End Operator` příkaz na příslušné místo v kódu.
