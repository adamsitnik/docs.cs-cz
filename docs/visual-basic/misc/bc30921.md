# <a name="ltderivedtypenamegt-cannot-inherit-from-lttypegt-ltconstructedbasetypenamegt-because-it-expands-the-access-of-type-ltinternaltypenamegt-to-ltregiongt-ltregionnamegt"></a><span data-ttu-id="6494b-102">"&lt;derivedtypename&gt;' nemůže dědit od třídy &lt;typ&gt; "&lt;constructedbasetypename&gt;'protože rozšiřuje přístup typu"&lt;internaltypename&gt;"k &lt;oblasti&gt; "&lt;regionname&gt;.</span><span class="sxs-lookup"><span data-stu-id="6494b-102">'&lt;derivedtypename&gt;' cannot inherit from &lt;type&gt; '&lt;constructedbasetypename&gt;' because it expands the access of type '&lt;internaltypename&gt;' to &lt;region&gt; '&lt;regionname&gt;'</span></span>
<span data-ttu-id="6494b-103">Odvozená třída nebo rozhraní se pokusí rozbalit úroveň přístupu vnitřního typu pomocí jako argument typu na základní třídu nebo rozhraní.</span><span class="sxs-lookup"><span data-stu-id="6494b-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="6494b-104">Následující kód lze vygenerovat tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="6494b-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="6494b-105">Kód mimo `containingClass` nemá povolený přístup k `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="6494b-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="6494b-106">Nicméně `derivedClass` je přístupná z libovolné kódu ve stejném sestavení.</span><span class="sxs-lookup"><span data-stu-id="6494b-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="6494b-107">Proto `derivedClass` nemůže dědit `baseClass` pokud používá `internalStructure` jako argument typu, protože může vystavit, který `internalStructure` v rámci oblasti definující kód.</span><span class="sxs-lookup"><span data-stu-id="6494b-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="6494b-108">**ID chyby:** BC30921</span><span class="sxs-lookup"><span data-stu-id="6494b-108">**Error ID:** BC30921</span></span>
