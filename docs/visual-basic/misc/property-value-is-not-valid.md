# <a name="property-value-is-not-valid"></a>Hodnota vlastnosti není platná
Jedna z vlastností objektu nebo ovládací prvek byl nastaven na hodnotu mimo rozsah.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte hodnotu vlastnosti na platné nastavení.
