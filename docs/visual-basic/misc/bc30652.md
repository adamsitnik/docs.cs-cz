# <a name="reference-required-to-assembly-assemblyname-containing-the-type-membername"></a>Odkaz vyžadoval sestavení '\<assemblyname >' obsahující typ '\<membername > "
Odkaz vyžadoval sestavení '\<assemblyname >' obsahující typ '\<membername >'. Přidejte jej do projektu.  
  
 Projekt obsahuje typ, který závisí na jiném typu ze sestavení, která není zahrnutá v odkazech projektu.  
  
 **ID chyby:** BC30652  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte odkaz na projekt u závislých typů v projektu.
