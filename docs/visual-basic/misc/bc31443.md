# <a name="name-of-type-parameter-lttypeparametername1gt-does-not-match-lttypeparametername2gt-the-corresponding-type-parameter-defined-on-the-partial-method-declaration-ltmethodnamegt"></a>Název parametru typu "&lt;typeparametername1&gt;"neodpovídá"&lt;typeparametername2&gt;', příslušného parametru definovaného v deklaraci částečné metody'&lt;methodname&gt;'
V částečné metody, která obsahuje jeden nebo více parametrů typu musí být názvy parametrů typu v deklaraci metody a v implementaci metody.  
  
 Například následující deklaraci a implementaci příčinou této chyby.  
  
```vb  
' Definition of the partial method signature with type parameter T.  
Partial Private Sub OnNameChanged(Of T)()  
End Sub  
```  
  
```vb  
'' Implementation of the partial method with type parameter N.  
'Private Sub OnNameChanged(Of N)()  
'    Console.WriteLine("Name was changed to " & Me.Name)  
'End Sub  
```  
  
 **ID chyby:** BC31443  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte parametry typu k určení, kde se neshodují. Změňte názvy podle potřeby tak, aby byly stejné.
