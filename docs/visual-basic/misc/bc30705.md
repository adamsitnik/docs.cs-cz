# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>Odkazovaný objekt "\<objectname >' s hodnotou 'Nothing'
Odkazovaný objekt nebyla nastavena nebo byla nastavena na hodnotu `Nothing`.  
  
 **ID chyby:** BC30705  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ověřte, že není správně napsaný název objektu a jestli je nastavená na objekt.
