# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>Odkazovaný objekt\<ObjectName > má hodnotu Nothing.
Odkazovaný objekt nebyl nastaven nebo byl nastaven na hodnotu `Nothing`.  
  
 **ID chyby:** BC30705  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda je název objektu správně zadán a zda byl objekt nastaven.
