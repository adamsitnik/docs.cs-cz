# <a name="operator-ltoperatornamegt-is-not-defined-for-type-lttypenamegt"></a>Operátor '&lt;operatorname&gt;'není definován pro typ'&lt;typename&gt;.
Pokusili jste se použít unární operátor (`+`, `-`, nebo `Not`) na typ, například `String`, pro který není definován.  
  
 **ID chyby:** BC30487  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahraďte příslušného operátoru pro výraz unární operátor.
