# <a name="method-methodname-does-not-have-the-same-signature-as-delegate-delegatename"></a>Metoda '\<methodname >' nemá stejný podpis jako delegát '\<vlastnost delegatename > '
Došlo k neshodě mezi podpisy metod a delegátů, který se pokoušíte použít. `Delegate` Prohlášení definuje typy parametrů a návratové typy třídy delegáta. Všechny procedury s odpovídajícími parametrů, typy a návratový typ může použít k vytvoření instance tohoto typu delegáta. Každá třída delegáta je definován konstruktor, který je předán specifikace metodu objektu.  
  
 **ID chyby:** BC30408  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Kontrola najít Neshoda podpisů a opraví ho.
