# <a name="type-lttypenamegt-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>Typ '&lt;typename&gt;' nelze použít jako atribut, protože není deklararovaný jako 'Veřejné'
Atributy musí být veřejné.  
  
 **ID chyby:** BC31516  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidat `Public` Modifikátor třídy atributů.
