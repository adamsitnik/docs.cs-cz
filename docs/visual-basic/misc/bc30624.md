# <a name="structure-statement-must-end-with-a-matching-end-structure"></a>Příkaz structure,' musí končit odpovídajícího "End konstrukce

 **ID chyby:** BC30624  
  
## <a name="to-correct-this-error"></a>Oprava této chyby
