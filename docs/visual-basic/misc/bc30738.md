# <a name="sub-main-is-declared-more-than-once-in-namespace-message"></a>'Procedura sub Main' je deklarován více než jednou v "\<oboru názvů >': \<zpráva >
`Sub Main` lze deklarovat pouze jednou v oboru názvů.  
  
 **ID chyby:** BC30738  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že existuje pouze jeden `Sub Main` postup ve vašem projektu.
