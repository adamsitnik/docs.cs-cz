# <a name="sub-main-is-declared-more-than-once-in-ltnamespacegt-ltmessagegt"></a>'Procedura sub Main' je deklarován více než jednou v "&lt;obor názvů&gt;': &lt;zprávy&gt;
`Sub Main` lze deklarovat pouze jednou v oboru názvů.  
  
 **ID chyby:** BC30738  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že existuje pouze jeden `Sub Main` postup ve vašem projektu.
