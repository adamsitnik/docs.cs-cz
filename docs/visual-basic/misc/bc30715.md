# <a name="catch-statements-are-not-valid-in-the-immediate-window"></a>'Příkaz catch' příkazy nejsou platné v příkazovém
`Catch` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30715  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `Catch` příkazy z ladění kódu.
