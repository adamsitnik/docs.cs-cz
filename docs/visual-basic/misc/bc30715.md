# <a name="catch-statements-are-not-valid-in-the-immediate-window"></a>Příkazy Catch nejsou platné v příkazovém podokně.
příkazy `Catch` lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30715  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte příkazy `Catch` z ladění kódu.
