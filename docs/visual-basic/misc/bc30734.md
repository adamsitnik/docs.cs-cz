# <a name="ltnamegt-is-already-declared-as-a-parameter-of-this-method"></a>"&lt;název&gt;' již byl deklarován jako parametr této metody
Zadaný název parametru je už deklarovaný pro tuto metodu.  
  
 **ID chyby:** BC30734  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přejmenujte parametr tak, aby byl jedinečný.
