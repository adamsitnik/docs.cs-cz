# <a name="insufficient-security-permissions-to-set-the-system-time"></a>Nedostatečná oprávnění zabezpečení k nastavení systémového času

## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Změňte oprávnění zabezpečení.
