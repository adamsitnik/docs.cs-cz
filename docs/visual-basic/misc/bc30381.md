# <a name="finally-can-only-appear-once-in-a-try-statement"></a>"Finally" může být pouze jednou v příkazu 'Try.
`Finally` slouží k dokončení `Try...Catch...Finally` blokovat; proto to může být pouze jednou na konci bloku.  
  
 **ID chyby:** BC30381  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Vyhledat a odebrat redundantní `Finally`.
