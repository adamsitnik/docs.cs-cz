# <a name="wrong-number-of-type-arguments-passed-to-method-genericprocedurename"></a>Nesprávný počet argumentů předaný metodě '\<genericprocedurename >'
Obecný postup se nazývá s počtem argumentů typu, který neodpovídá počtu parametrů typu, se kterými je definována.  
  
 **ID chyby:** BC30951  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte argument typu pro každý parametru typu definovaného pro obecné procedury.  
  
     -nebo-  
  
- Volání obecné procedury bez argumentů typu vůbec a nechte kompilátor pokusí odvodit argumenty typu.
