# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>Názvy jiných než vnitřních typů nejsou povoleny ve výrazech podmíněné kompilace
Podmíněná kompilace výraz odkazuje na jiných než vnitřních typů, ale jsou povolené jenom vnitřní typy.  
  
 Další informace o vnitřního typu v jazyce Visual Basic najdete v tématu [souhrn datových typů](../../visual-basic/language-reference/keywords/data-types-summary.md).  
  
 **ID chyby:** BC31426  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahraďte – vnitřní typ vnitřního typu.
