# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-error"></a>Kompilátor jazyka Visual Basic se nezotavil z následující chyby: \<chyby >
Kompilátor jazyka Visual Basic se nezotavil z následující chyby: <`error`>.  
  
 Byly zakázány některé funkce integrovaného vývojového prostředí.  
  
 V kompilátoru jazyka Visual Basic došlo k vnitřní chybě.  
  
 **ID chyby:** BC31021  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Uložte si práci a restartujte aplikaci Visual Studio.  
  
2.  Pokud se chyba objeví znovu, přeinstalujte Visual Studio.  
  
3.  Pokud chyba přetrvává po přeinstalaci, upozorní Microsoft Product Support Services.
