# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>Výraz nejde převést na strom výrazu
Většina výrazy můžete převést na stromy výrazů, ale některé nikoli. Tato chyba označuje, že byl proveden pokus o převodu výrazu, který nelze převést.  
  
 **ID chyby:** BC36534
