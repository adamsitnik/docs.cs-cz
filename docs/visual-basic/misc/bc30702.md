# <a name="index-ltindexnumbergt-for-dimension-ltdimensionnumbergt-is-out-of-range"></a>Index '&lt;Rejstříkové_číslo&gt;"pro dimenzi"&lt;dimensionnumber&gt;"je mimo rozsah.
Jedna z indexů překračuje rozsah definovaný.  
  
 **ID chyby:** BC30702  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte logiku procedury tak, aby se překročila hranice pole.  
  
-   Znovu definujte pole, které chcete povolit používání nástroje většímu počtu hodnot.
