# <a name="index-indexnumber-for-dimension-dimensionnumber-is-out-of-range"></a>Index '\<Rejstříkové_číslo > 'pro dimenzi'\<dimensionnumber >' je mimo rozsah.
Jedna z indexů překračuje rozsah definovaný.  
  
 **ID chyby:** BC30702  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte logiku procedury tak, aby se překročila hranice pole.  
  
-   Znovu definujte pole, které chcete povolit používání nástroje většímu počtu hodnot.
