# <a name="name-cannot-expose-type-type-used-in-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-to-specifier1-type1-through-specifier2-type2"></a>'\<název >' nemůže vystavovat typ '\<typu >' používá v nadřazeném typu delegáta '\<delegatetype > 'události, kterou implementuje do \<specifier1 >'\<type1 > 'prostřednictvím \<specifier2 > '\<type2 > '
Kód poskytuje typ soukromého mimo veřejnou třídu.  
  
 **ID chyby:** BC30923  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Deklaruje typ `Public` nebo pomocí jiného typu.
