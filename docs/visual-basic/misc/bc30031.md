# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-number-characters"></a>Plně kvalifikované názvy pro typy v oborech názvů nemůže být delší než \<číslo > znaků
Plně kvalifikovaný název typu nemůže být delší než zadaný počet znaků.  
  
 **ID chyby:** BC30031  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkraťte název typu.
