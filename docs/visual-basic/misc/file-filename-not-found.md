# <a name="file-ltfilenamegt-not-found"></a>Soubor '&lt;filename&gt;' nebyl nalezen
Název souboru je nesprávný nebo soubor neexistuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že je název souboru správný a zda existuje.
