# <a name="project-projectname-makes-an-indirect-reference-to-assembly-assemblyname-which-contains-classname"></a>Projekt '\<projectname >' vytváří nepřímý odkaz na sestavení '\<assemblyname >' obsahující '\<classname >'
Projekt '\<projectname > 'vytváří nepřímý odkaz na sestavení '\<assemblyname >' obsahující'\<classname >'. Přidejte odkaz na '\<assemblyname >' do projektu.  
  
 Tento projekt obsahuje členy, které závisí na sestavení, který není obsažen v odkazech projektu.  
  
 **ID chyby:** BC31515  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat odkaz na sestavení, kliknutím pravým tlačítkem myši **odkazy** v **Průzkumníka řešení** a pak levým na **přidat odkaz**.
