# <a name="end-tag-tagname-expected"></a>Koncová značka \</tagName > očekávání
Koncová značka v literál XML elementu se neshoduje s odpovídající počáteční značce.  
  
 **ID chyby:** BC31150  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Upravte koncový odpovídající počáteční značka XML pro literál XML elementu značky XML.
