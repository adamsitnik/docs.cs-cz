# <a name="cannot-convert-start-value-of-type-lttype1gt-and-step-value-of-type-lttype2gt-to-a-common-type"></a>Nelze převést počáteční hodnotu typu '&lt;type1&gt;"a hodnotu kroku typu"&lt;type2&gt;"na společný typ.
Zadaný `For...Next` hodnoty ovládacích prvků mají různé typy a proto nejsou platné vzhledem k tomu, že nelze převést na společný typ.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změňte typ jednoho nebo více hodnot, tak, aby všechny hodnoty lze převést na společný typ.
