# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Příkazy deklarace metod musí být první logického řádku
A `Function` nebo `Sub` příkaz následuje dvojtečka (:) oddělovač. `Function` nebo `Sub` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC32009  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na samostatné řádky.
