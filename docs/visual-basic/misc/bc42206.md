# <a name="maximum-number-of-warnings-has-been-exceeded"></a>Byl překročen maximální počet upozornění
Maximální počet varování kompilace je 100. Kompilátor ukládá tento limit, aby se zmenšila velikost chyby a upozornění sestav.  
  
 Ve výchozím nastavení tato zpráva je upozornění. Další informace o zobrazení nebo skrytí upozornění zpracování upozornění jako chyby, najdete v části [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID chyby:** BC42206  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přečtěte si další upozornění vygenerovaná tuto kompilaci a vyřešit co nejvíce. Pokud zdrojový kód obsahuje více než 100 upozornění, může se zobrazit další varování při další kompilaci.
