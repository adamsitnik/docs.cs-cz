# <a name="imports-alias-name-conflicts-with-name2-declared-in-the-root-namespace"></a>Alias importu\<name >' je v konfliktu s '\<name2 >' deklarované v kořenovém oboru názvů
Název aliasu používané `Imports` příkaz je v konfliktu s jiným názvem pro deklarované v kořenovém oboru názvů.  
  
 **ID chyby:** BC31403  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Vyberte jedinečný název pro aliasem, který se používá `Imports` příkazu.
