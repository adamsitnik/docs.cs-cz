# <a name="imports-alias-ltnamegt-conflicts-with-ltname2gt-declared-in-the-root-namespace"></a>Alias importu&lt;název&gt;"je v konfliktu s"&lt;name2&gt;"deklarované v kořenovém oboru názvů
Název aliasu používané `Imports` příkaz je v konfliktu s jiným názvem pro deklarované v kořenovém oboru názvů.  
  
 **ID chyby:** BC31403  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Vyberte jedinečný název pro aliasem, který se používá `Imports` příkazu.
