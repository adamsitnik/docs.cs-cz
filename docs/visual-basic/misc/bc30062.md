# <a name="me-cannot-be-the-target-of-an-assignment"></a>'Me' nemůže být cílem přiřazení.
`Me` – Klíčové slovo vždy odkazuje na konkrétní instanci třídy, kde právě spouští kód. Jeho hodnotu nelze změnit.  
  
 **ID chyby:** BC30062  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte příkaz, který přiřazuje hodnotu `Me`.
