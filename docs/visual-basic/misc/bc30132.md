# <a name="label-labelname-is-not-defined"></a>Popisek '\<labelname >' není definován
Popisek řádku nebo číslo řádku je uvedené ale není definovaný v rámci odkazu. Popis musí být v rámci procedury, která obsahuje odkaz.  
  
 **ID chyby:** BC30132  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změnit strukturu kódu tak, aby popisku řádku a odkazy byly ve stejné proceduře.
