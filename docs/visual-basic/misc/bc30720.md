# <a name="on-error-statements-are-not-valid-in-the-immediate-window"></a>Příkazy On Error nejsou platné v příkazovém podokně.
příkazy `On Error` lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30720  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte příkazy `On Error` z ladění kódu.
