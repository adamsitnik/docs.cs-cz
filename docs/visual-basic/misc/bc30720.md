# <a name="on-error-statements-are-not-valid-in-the-immediate-window"></a>"Při chybě" příkazy nejsou platné v příkazovém
`On Error` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30720  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `On Error` příkazy z ladění kódu.
