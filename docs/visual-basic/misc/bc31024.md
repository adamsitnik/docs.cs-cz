# <a name="unable-to-create-a-net-runtime-interface-error"></a>Nelze vytvořit rozhraní .NET Runtime: \<chyby >
Došlo k chybě vnitřního kompilátoru vytváření rozhraní nebo volání metody v rozhraní se nezdařilo.  
  
 **ID chyby:** BC31024  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Uložte si práci a restartujte aplikaci Visual Studio.  
  
2. Pokud se chyba objeví znovu, přeinstalujte Visual Studio.  
  
3. Pokud chyba přetrvává po přeinstalaci, upozorní Microsoft Product Support Services.
