# <a name="parameter-already-declared-with-name-ltargumentnamegt"></a>Parametr už deklarovaný s názvem "&lt;název argumentu argumentname&gt;.
Deklaraci procedury definuje dva argumenty se stejným názvem. Všechny názvy argumentů v deklaraci procedury musí být jedinečný.  
  
 **ID chyby:** BC30237  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte název jednoho z argumentů.
