# <a name="ltfunctiongt-is-not-declared"></a>"&lt;funkce&gt;' není deklarována
"\<funkce >' není deklarována. Funkce byla přesunuta do oboru názvů Microsoft.VisualBasic.  
  
 **ID chyby:** BC30818  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použijte příslušnou funkci z `Microsoft.VisualBasic` oboru názvů.
