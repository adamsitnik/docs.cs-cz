# <a name="members-in-a-structure-cannot-be-declared-specifier"></a>Členy ve struktuře nelze použít deklaraci '\<specifikátor > "
Použili jste specifikátor, který není platný pro člena v rámci `Structure` příkazu.  
  
 **ID chyby:** BC30435  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
