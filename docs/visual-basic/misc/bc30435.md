# <a name="members-in-a-structure-cannot-be-declared-ltspecifiergt"></a>Členy ve struktuře nelze použít deklaraci '&lt;specifikátor&gt;.
Použili jste specifikátor, který není platný pro člena v rámci `Structure` příkazu.  
  
 **ID chyby:** BC30435  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
