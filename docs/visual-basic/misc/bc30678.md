# <a name="end-statement-not-valid"></a>Příkaz 'End' není platný
`End` – Klíčové slovo se používá nesprávně nebo je nevhodná v aktuálním kontextu.  
  
 **ID chyby:** BC30678  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Použití `End` samostatně způsobit proceduru přijít o oboru.  
  
2.  Použití `End` s další klíčová slova k ukončení bloku kódu, jako je struktura nebo vlastnost.
