# <a name="project-projectname-cannot-reference-project-projectname-because-projectname-directly-or-indirectly-references-projectname"></a>Projekt "\<projectname >' nemůže odkazovat na projekt"\<projectname > "protože"\<projectname > "přímo nebo nepřímo odkazuje na"\<projectname > "
Projekt obsahuje odkaz na druhý projekt. Druhý projekt zase obsahuje odkaz na projekt, ve které se odkazuje.  
  
 **ID chyby:** BC30761  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte jeden z odkazů.
