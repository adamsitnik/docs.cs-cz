# <a name="project-ltprojectnamegt-cannot-reference-project-ltprojectnamegt-because-ltprojectnamegt-directly-or-indirectly-references-ltprojectnamegt"></a>Projekt '&lt;projectname&gt;'nemůže odkazovat na projekt'&lt;projectname&gt;'protože&lt;projectname&gt;"přímo nebo nepřímo odkazuje na"&lt;projectname &gt;'
Projekt obsahuje odkaz na druhý projekt. Druhý projekt zase obsahuje odkaz na projekt, ve které se odkazuje.  
  
 **ID chyby:** BC30761  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte jeden z odkazů.
