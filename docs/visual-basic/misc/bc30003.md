# <a name="next-expected"></a>Byl očekáván 'Další'
`On Error Resume` Konstrukce probíhá, aniž by závěrečném dílu seriálu `Next`.  
  
 **ID chyby:** BC30003  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Next` – klíčové slovo do konce `On Error` příkazu.
