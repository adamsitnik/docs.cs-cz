# <a name="if-operator-requires-either-two-or-three-operands"></a>Pokud operátor vyžaduje dva nebo tři operandy
`If` Operátor přijímá pouze dvě nebo tři argumenty.  
  
 **ID chyby:** BC33104  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte `If` výraz, který se zobrazí, zda počet operátorů lze snížit nebo rozšířit na dvě nebo tři argumenty.  
  
- Zvažte použití jiného podmíněné konstrukce, jako například `If...Then...Else` příkazu nebo vnořený `If...Then...Else` příkaz k uspořádání argumenty.
