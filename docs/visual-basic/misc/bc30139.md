# <a name="error-setting-assembly-manifest-option-lterror-messagegt"></a>Chyba při nastavování možnosti manifestu sestavení: &lt;chybová zpráva&gt;
Kompilátor jazyka Visual Basic volá Assembly Linker (Al.exe, označované také jako Alink) ke generování sestavení s manifestem. Linker ohlásil chybu hledání souboru klíče nebo název klíče, pomocí kterého se má sestavení podepsat.  
  
 **ID chyby:** BC30139  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Prověřením v uvozovkách chybové zprávy a prostudovat si chyby uvedené v [Al.exe (Linker sestavení)](../../framework/tools/al-exe-assembly-linker.md) pro další vysvětlení a poradenství.  
  
2.  Pokud potíže potrvají, shromážděte informace o okolnostech a upozornit Microsoft Product Support Services.
