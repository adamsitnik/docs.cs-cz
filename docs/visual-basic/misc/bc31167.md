# <a name="expected--for-xml-end-tag"></a>Byl očekáván ' /' koncové značky XML
Chybí koncová značka XML pro literál XML požadované "/" znak.  
  
 **ID chyby:** BC31167  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidáte znak "/" na začátku koncová značka XML pro literál XML.
