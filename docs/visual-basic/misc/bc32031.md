# <a name="end-sub-must-be-the-first-statement-on-a-line"></a>'End Sub' musí být prvním příkazem na řádku
`End Sub` Příkaz následuje dvojtečka (:) oddělovač. `End Sub` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC32031  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Oddělte více příkazů na samostatné řádky.
