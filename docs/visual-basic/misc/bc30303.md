# <a name="type-character-cannot-be-used-in-a-sub-declaration-because-a-sub-doesnt-return-a-value"></a>Znak typu nejde použít v deklaraci 'Sub', protože 'Sub nevrací hodnotu
A `Sub` proceduru, třeba `Function` postup, je samostatné procedury, která může přijmout argumenty a provádět řadu příkazů. Na rozdíl od `Function` postupu `Sub` nevrací hodnotu a proto nemůže obsahovat deklarace typu.  
  
 **ID chyby:** BC30303  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změnit `Sub` postup `Function` postup.
