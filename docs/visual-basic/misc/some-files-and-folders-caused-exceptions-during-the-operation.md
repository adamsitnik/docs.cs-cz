# <a name="some-files-and-folders-caused-exceptions-during-the-operation"></a>Některé soubory a složky způsobeno výjimky v průběhu operace
Výjimky může způsobit adresářů a souborů, když dojde k pokusu o operaci, ale uživatel nemá potřebná oprávnění.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte potřebná oprávnění pro přístup ke všem soubory a složky.
