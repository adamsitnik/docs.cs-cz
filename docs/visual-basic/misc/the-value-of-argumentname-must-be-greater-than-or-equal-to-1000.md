# <a name="the-value-of-argumentname-must-be-greater-than-or-equal-to-1000"></a>Hodnota \<název argumentu argumentname > musí být větší než nebo rovna 1000
Hodnota <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> vlastnost musí být větší než nebo rovna 1000.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> Vlastnost určuje maximální počet bajtů v souboru protokolu předtím, než se vytvoří nový soubor protokolu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nastavte <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> vlastnost na číslo větší než nebo rovna 1000.
