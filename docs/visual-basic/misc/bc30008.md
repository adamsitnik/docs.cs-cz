# <a name="reference-required-to-module-ltmodulenamegt-containing-the-base-class-ltclassnamegt"></a>Vyžaduje se odkaz na modul '&lt;modulename&gt;'obsahující základní třídu'&lt;classname&gt;"
Vyžaduje se odkaz na modul '\<modulename >' obsahující základní třídu\<classname >'. Přidejte jej do projektu.  
  
 Je třída definovaná v modulu, který není přímo odkazován ve vašem projektu. Kompilátor jazyka Visual Basic vyžaduje přidání odkazu na-li předejít nejednoznačnosti v případě, že je třída definovaná ve více než jeden modul.  
  
 **ID chyby:** BC30008  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Název modulu neodkazovaná zahrnout do odkazů projektu.
