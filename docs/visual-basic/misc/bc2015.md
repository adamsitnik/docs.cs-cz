# <a name="the-file-filename-is-not-a-text-file"></a>Soubor '\<název souboru >' není textový soubor
Zadaný soubor nelze zapisovat, protože je to binární soubor.  
  
 **ID chyby:** BC2015  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, že zadáváte správný název souboru a cestu.
