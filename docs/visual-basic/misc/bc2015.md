# <a name="the-file-ltfilenamegt-is-not-a-text-file"></a>Soubor '&lt;filename&gt;"není textový soubor
Zadaný soubor nelze zapisovat, protože je to binární soubor.  
  
 **ID chyby:** BC2015  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, že zadáváte správný název souboru a cestu.
