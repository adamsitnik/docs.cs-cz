# <a name="character-sequence----is-not-allowed-in-an-xml-comment"></a>Posloupnosti znaků '--' není povolen v komentáři XML
Literál komentáře XML obsahuje sekvence znaků '--', což není podporováno.  
  
 **ID chyby:** BC31171  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat sekvence znaků '--' z literál komentáře XML.
