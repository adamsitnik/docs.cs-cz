# <a name="file-io-with-type-typename-is-not-valid"></a>Soubor vstupně-výstupních operací s typem '\<typename >' není platný
Pokoušíte se zápis do souboru nepodporovaného typu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte strukturu určit, kde dochází k nepodporovaný typ a změňte ji na platného typu.
