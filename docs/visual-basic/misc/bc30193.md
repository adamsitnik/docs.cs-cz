# <a name="specifiers-and-attributes-are-not-valid-on-this-statement"></a>Specifikátory a atributy nejsou platné pro tento příkaz
Deklarace, která nelze upravit, jako například `Inherits`, obsahuje jeden nebo více atributů, nebo jednu nebo více klíčových slov, například `Public` nebo `ReadOnly`. Tato klíčová slova nebo atributy může obsahovat jenom upravitelné deklarace.  
  
 **ID chyby:** BC30193  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte deklaraci atributu nebo – klíčové slovo.
