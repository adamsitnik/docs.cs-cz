# <a name="specifiers-and-attributes-are-not-valid-on-this-statement"></a>Specifikátory a atributy nejsou pro tento příkaz platné.
Deklarace, kterou nelze změnit, například `Inherits`, obsahuje jeden nebo více atributů nebo jedno nebo více klíčových slov, `Public` například nebo `ReadOnly`. Taková klíčová slova nebo atributy můžou obsahovat jenom upravitelné deklarace.  
  
 **ID chyby:** BC30193  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte atribut nebo klíčové slovo z deklarace.
