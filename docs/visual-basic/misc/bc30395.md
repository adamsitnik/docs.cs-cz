# <a name="ltspecifiergt-is-not-valid-on-a-structure-declaration"></a>"&lt;specifikátor&gt;' není platný v deklaraci struktury
Specifikátor, jste už použili, jako `Dim` nebo `Override`, která je neplatná pro `Structure` deklarace.  
  
 **ID chyby:** BC30395  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
