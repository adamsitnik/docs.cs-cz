# <a name="picture-is-not-valid"></a>Obrázek není platný
Pokusili jste se přiřadit grafický formát než rastrový obrázek, ikona nebo Windows metafile k `Picture` vlastnost formuláře nebo ovládacího prvku.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že se pokoušíte načíst do souboru `Picture` vlastností je soubor platný grafiky podporované jazykem Visual Basic.
