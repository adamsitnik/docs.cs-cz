# <a name="method-ltmethodname1gt-must-be-declared-private-in-order-to-implement-partial-method-ltmethodname2gt"></a>Metoda "&lt;methodname1&gt;'musí být deklarován 'Privátní' aby mohla implementovat částečnou metodu'&lt;methodname2&gt;.
Implementace částečné metody musí být deklarován `Private`. Například následující kód způsobí, že k této chybě.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,   
    ' causes this error.   
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 **ID chyby:** BC31441  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Použít modifikátor přístupu `Private` v implementaci částečné metody, jak je znázorněno v následujícím příkladu.  
  
```vb  
Private Sub valueChanged()  
    MsgBox("Value was changed to " & Me.Quantity)  
End Sub  
```
