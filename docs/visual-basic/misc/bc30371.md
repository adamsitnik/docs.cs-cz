# <a name="module-ltmodulenamegt-cannot-be-used-as-a-type"></a>Modul '&lt;modulename&gt;' nelze použít jako typ.
Pokusili jste se použít jako typ modulu. Modul nelze použít s výjimkou jako `Get` typu.  
  
 **ID chyby:** BC30371  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahraďte něčím, který může sloužit jako platný typ modulu.
