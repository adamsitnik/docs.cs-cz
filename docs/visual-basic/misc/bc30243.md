# <a name="modifier-is-not-valid-on-an-event-declaration"></a>'\<modifikátor >' není platný v deklaraci události
`Event` Příkazu obsahuje neplatné klíčové slovo jako například `ReadOnly`.  
  
 **ID chyby:** BC30243  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte neplatné klíčové slovo z `Event` příkazu.
