# <a name="side-effects-not-valid-during-expression-evaluation-in-this-context"></a>Vedlejší efekty při vyhodnocení výrazu v tomto kontextu není platný.
V tomto kontextu nejsou povoleny příkazy, které ke změně stavu objektu.  
  
 **ID chyby:** BC30700  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte kód tak, aby příkazy vyhodnocují beze změny stavu další položky.
