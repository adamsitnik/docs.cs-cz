# <a name="unable-to-ping-because-a-network-connection-is-not-available"></a>Nelze pro odeslání příkazu ping, protože není k dispozici síťové připojení
Operaci nelze provést, protože je k dispozici žádné síťové připojení.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Připojení k síti a opakujte operaci.
