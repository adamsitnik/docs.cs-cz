# <a name="addhandler-declaration-must-end-with-a-matching-end-addhandler"></a>Deklarace 'AddHandler' musí končit odpovídajícího 'End AddHandler
`AddHandler` Deklarace musí končit `End AddHandler` příkazu.  
  
 **ID chyby:** BC31115  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, že `AddHandler` deklarace končí `End AddHandler` příkazu.
