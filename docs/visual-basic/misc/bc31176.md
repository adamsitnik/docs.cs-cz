# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>Znak 'znak' (0 x\<šestnáctkové >) není povolen v XML literál
Literál XML se použil nepodporovaný znak.  
  
 **ID chyby:** BC31176  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte nepodporovaný znak.
