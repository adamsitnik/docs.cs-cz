# <a name="inherits-statements-must-precede-all-declarations-in-an-interface"></a>Příkazy 'Inherits' musí předcházet před všemi deklaracemi v rozhraní
`Inherits` Příkaz musí být na prvním řádku neprázdné, mimo komentář v definici třídy.  
  
 **ID chyby:** BC30357  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesuňte vaše `Inherits` příkaz tak, že předchází všechny deklarace.
