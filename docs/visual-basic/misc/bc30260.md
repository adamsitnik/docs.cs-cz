# <a name="name-is-already-declared-as-declaration-in-this-declarationspace"></a>'\<název >' je již deklarován jako\<deklarace >' v tomto \<declarationspace >
Dva programovací prvky jsou deklarovány pomocí stejného názvu ve stejné deklaraci místa (třídy, modulu, rozhraní nebo struktura).  
  
 **ID chyby:** BC30260  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Jak přesunout některou z deklarací na znak jiný prohlášení, nebo změňte název jedné z deklarované elementy.
