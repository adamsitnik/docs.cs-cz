# <a name="ltnamegt-is-already-declared-as-ltdeclarationgt-in-this-ltdeclarationspacegt"></a>"&lt;název&gt;'již byl deklarován jako'&lt;deklarace&gt;" v tomto &lt;declarationspace&gt;
Dva programovací prvky jsou deklarovány pomocí stejného názvu ve stejné deklaraci místa (třídy, modulu, rozhraní nebo struktura).  
  
 **ID chyby:** BC30260  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Jak přesunout některou z deklarací na znak jiný prohlášení, nebo změňte název jedné z deklarované elementy.
