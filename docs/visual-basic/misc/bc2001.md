# <a name="file-ltfilenamegt-could-not-be-found"></a>Soubor &lt;filename&gt; nebyl nalezen
Zadaný soubor nebyl nalezen. To může být důsledek poskytnutí cestu k souboru.  
  
 **ID chyby:** BC2001  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda název souboru a cesta správná.
