# <a name="paramarray-parameter-must-be-an-array"></a>Parametrem ParamArray musí být pole
A `ParamArray` argument musí být deklarován jako typ jednorozměrné pole. Deklarace nezahrnuje závorek za `ParamArray` název argumentu.  
  
 **ID chyby:** BC30050  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte dvojici závorek za `ParamArray` název argumentu.
