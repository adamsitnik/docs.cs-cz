# <a name="goto-to-label-labelname-cannot-be-used-within-lambda-expressions"></a>Přejít na popisek "\<labelname >' nelze použít ve výrazech lambda
Hlavní část výrazu lambda musí být výraz, který vypočítá a vrátí jednu hodnotu. Nejsou povoleny žádné příkazy, včetně `GoTo` příkazy.  
  
 **ID chyby:** BC36597
