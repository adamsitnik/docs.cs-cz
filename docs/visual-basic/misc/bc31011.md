# <a name="unable-to-load-referenced-library-filename-error"></a>Nepovedlo se načíst odkazované knihovny '\<název souboru >': \<chyby >
Nelze načíst knihovnu odkazuje tento projekt.  
  
 **ID chyby:** BC31011  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, že soubor je v zadaném umístění a není uzamčený jinou aplikací.
