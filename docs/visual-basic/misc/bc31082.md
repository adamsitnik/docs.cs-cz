# <a name="variablename-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-catch-variable"></a>"\<NázevProměnné >' není lokální proměnná ani parametr a proto jej nelze použít jako proměnná 'Catch'
Proměnné v `Try...Catch...Finally` příkazy musí být deklarovaná místně, nebo být parametry pro aktuální proceduru.  
  
 **ID chyby:** BC31082  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Deklarujte místní proměnné nebo parametru pro `Try...Catch...Finally` příkazy.
