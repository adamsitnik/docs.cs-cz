<span data-ttu-id="7493f-103">Bylo odkazováno podřízený element XML pro objekt, který není typu <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, nebo `IEnumerable(Of XElement)`.</span><span class="sxs-lookup"><span data-stu-id="7493f-103">An XML child element has been referenced for an object that is not of type <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="7493f-104">Další informace najdete v tématu [vlastnost osy podřízeného souboru XML](../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md).</span><span class="sxs-lookup"><span data-stu-id="7493f-104">For more information, see [XML Child Axis Property](../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md).</span></span>  
  
```vb

Dim var = "sample text".<child>  
```  
  
 <span data-ttu-id="7493f-105">**ID chyby:** BC36807</span><span class="sxs-lookup"><span data-stu-id="7493f-105">**Error ID:** BC36807</span></span>
