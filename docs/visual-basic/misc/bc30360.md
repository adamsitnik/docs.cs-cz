# <a name="property-propertyname-must-be-declared-default-because-it-overrides-a-default-property-on-the-base-name"></a>Vlastnost '\<propertyname >' musí být deklarovány'Default', protože přepisuje výchozí vlastnost v základní třídě '\<název >'
A `Property` přepíše příkaz v kódu `Default` vlastnost základní třídy, struktury, nebo rozhraní a proto je potřeba deklarovat `Default`.  
  
 **ID chyby:** BC30360  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Deklarovat vlastnost `Default`.
