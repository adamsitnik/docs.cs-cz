# <a name="ltparameternamegt-is-already-declared-as-a-type-parameter-of-this-method"></a>"&lt;parametername&gt;' již byl deklarován jako parametr typu této metody
Obecný postup definuje normální parametr nebo místní proměnná se stejným názvem jako parametr typu.  
  
 Každý parametr procedury, včetně každý parametr typu obecný postup je popsán musí mít název liší od všechny ostatní parametry. Vzhledem k tomu, že parametry procedury slouží jako lokální proměnné, všechny místní proměnné deklarované v rámci postupu musí také mít název liší od všech parametrů a parametrů typů.  
  
 **ID chyby:** BC32089  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte název normální parametr nebo lokální proměnné.
