# <a name="ltspecifiergt-is-not-valid-on-a-withevents-declaration"></a>"&lt;specifikátor&gt;' není platný v deklaraci WithEvents
A `Dim` příkaz obsahuje klíčové slovo, jako například `ReadOnly`, která je neplatný v kombinaci s `WithEvents` – klíčové slovo.  
  
 **ID chyby:** BC30234  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte neplatné klíčové slovo z `Dim` příkazu.
