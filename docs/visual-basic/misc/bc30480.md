# <a name="shared-sub-new-cannot-be-declared-ltspecifiergt"></a>Sdílené proceduru Sub New nelze použít deklaraci '&lt;specifikátor&gt;.
Pokusili jste se použít specifikátor v `Shared Sub New` deklarace, která není platná.  
  
 **ID chyby:** BC30480  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
