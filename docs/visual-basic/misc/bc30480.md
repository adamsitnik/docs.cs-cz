# <a name="shared-sub-new-cannot-be-declared-specifier"></a>Sdílené proceduru Sub New nelze použít deklaraci '\<specifikátor > "
Pokusili jste se použít specifikátor v `Shared Sub New` deklarace, která není platná.  
  
 **ID chyby:** BC30480  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
