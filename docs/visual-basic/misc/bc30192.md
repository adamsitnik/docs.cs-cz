# <a name="end-of-parameter-list-expected"></a>Konec seznamu parametrů očekává
A `ParamArray` specifikace následuje jedna nebo více jiných argument specifikace v deklaraci procedury.  
  
 **ID chyby:** BC30192  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunout `ParamArray` na konec seznamu argumentů.
