# <a name="ltspecifier1gt-lttypegt-cannot-inherit-from-a-ltspecifier2gt-lttypegt-because-it-expands-the-access-of-the-base-lttypegt"></a>&lt;specifier1&gt; &lt;typ&gt; nemůže dědit od třídy &lt;specifier2&gt; &lt;typ&gt; protože rozšiřuje přístup základní třídy &lt;typu&gt;
Pokusili jste se mají veřejná třída dědí ze základní třídy označené jako `Private` nebo `Friend`.  
  
 **ID chyby:** BC30509  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Deklarace základní třídy `Public` nebo deklaraci dědičné třídě `Private` nebo `Friend`.
