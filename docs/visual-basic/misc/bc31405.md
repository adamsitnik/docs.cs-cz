# <a name="optional-parameters-cannot-have-structure-types"></a>Volitelné parametry nemůžou mít typy struktur
Volitelné parametry nemohou být typy struktur.  
  
 **ID chyby:** BC31405  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Deklarování parametrů typu Struktura podle potřeby.  
  
2. Deklarujte statickou třídu člen uskutečnit struktury.
