# <a name="unable-to-open-key-file-ltfilenamegt-lterrorgt"></a>Nelze otevřít soubor klíče '&lt;filename&gt;': &lt;chyba&gt;
Soubor klíče zadaný pro tento projekt nelze otevřít.  
  
 **ID chyby:** BC31025  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ověřte, zda existuje soubor s klíčem a není uzamčený jinou aplikací.
