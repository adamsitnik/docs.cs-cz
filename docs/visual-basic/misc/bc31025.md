# <a name="unable-to-open-key-file-filename-error"></a>Nelze otevřít soubor klíče '\<název souboru >': \<chyby >
Soubor klíče zadaný pro tento projekt nelze otevřít.  
  
 **ID chyby:** BC31025  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda existuje soubor s klíčem a není uzamčený jinou aplikací.
