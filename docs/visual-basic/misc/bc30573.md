# <a name="xml-namespace-prefix-prefix-is-already-declared"></a>Předpona oboru názvů XML 'předponu' je už deklarovaný.
Předponu oboru názvů XML určené `Imports` příkaz odpovídá dříve definovanou předponu oboru názvů XML.  
  
 **ID chyby:** BC30573  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použijte jedinečný název pro předponu oboru názvů XML.
