# <a name="array-dimensions-do-not-match-those-specified-in-the-vbfixedarray-attribute"></a>Dimenze pole neodpovídají hodnotám zadaným v atributu ' VBFixedArray '.
Dimenze pole neodpovídají rozměrům, které jste zadali v `VBFixedArray` atributu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nastavte rozměry pole v příkazu array `Dim` tak, aby odpovídaly rozměrům určeným `VBFixedArray` atributem.
