# <a name="expected--at-start-of-an-embedded-expression"></a>Byl očekáván ' % = "na začátku vloženého výrazu
Počáteční identifikátor vloženého výrazu `<%=`, nebyl správně kódovány. Všimněte si, že nelze použít znak procenta (%) v názvu elementu XML literál.  
  
 **ID chyby:** BC31179  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že je jako programový identifikátor začátku vloženého výrazu `<%=`.
