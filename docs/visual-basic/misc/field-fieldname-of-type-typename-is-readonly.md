# <a name="field-ltfieldnamegt-of-type-lttypenamegt-is-readonly"></a>Pole '&lt;fieldname&gt;"typu"&lt;typename&gt;"je"jen pro čtení.
Pokus o zápis do pole, která je deklarována `ReadOnly`.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte pole tak, že je již deklarován `ReadOnly`.  
  
-   —nebo—  
  
-   Nepokoušejte se zapisovat do tohoto pole.
