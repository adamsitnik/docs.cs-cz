# <a name="no-default-member-found-for-type-lttypenamegt"></a>Pro typ se nenašly žádné výchozí člen "&lt;typename&gt;.
Index s pozdní vazbou `Get` nebo `Set` pokusil volání.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda má objekt vlastnost výchozí `Get` nebo `Set`, v závislosti na tom, které se pokoušíte vytvořit.
