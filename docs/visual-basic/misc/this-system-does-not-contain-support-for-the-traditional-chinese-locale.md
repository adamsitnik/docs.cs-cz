# <a name="this-system-does-not-contain-support-for-the-traditional-chinese-locale"></a>Tento systém neobsahuje podporu pro tradiční čínské národní prostředí.
Pokusili jste se použít člena `VbStrConv.TraditionalChinese` výčtu, ale systém nepodporuje tradiční čínské národní prostředí.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Upgradujte počítač pro podporu tradičního čínského národního prostředí nebo spusťte aplikaci na počítači, který podporuje národní prostředí.
