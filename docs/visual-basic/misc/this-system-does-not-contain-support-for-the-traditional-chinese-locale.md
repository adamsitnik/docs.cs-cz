# <a name="this-system-does-not-contain-support-for-the-traditional-chinese-locale"></a>Tento systém neobsahuje podporu pro tradiční čínské národní prostředí
Pokusili jste se použít `VbStrConv.TraditionalChinese` člen výčtu, ale váš systém nepodporuje tradiční čínské národní prostředí.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Upgrade počítače pro podporu tradiční čínské národní prostředí nebo spusťte aplikaci na počítač, který podporuje národní prostředí.
