# <a name="addhandler-and-removehandler-method-parameters-cannot-be-declared-byref"></a>Parametry metody AddHandler a RemoveHandler nemůžou být deklarované jako ByRef.
Parametry `AddHandler` metod `RemoveHandler` a`ByRef` nelze deklarovat s modifikátorem.  
  
 **ID chyby:** BC31134  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- `ByRef` Odeberte klíčové slovo z parametru.
