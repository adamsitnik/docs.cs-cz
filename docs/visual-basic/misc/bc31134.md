# <a name="addhandler-and-removehandler-method-parameters-cannot-be-declared-byref"></a>Parametry metody "AddHandler" a 'RemoveHandler' nelze použít deklaraci 'ByRef'
Parametry `AddHandler` a `RemoveHandler` metody se nedá deklarovat pomocí `ByRef` modifikátor.  
  
 **ID chyby:** BC31134  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `ByRef` – klíčové slovo z parametru.
