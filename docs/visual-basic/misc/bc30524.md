# <a name="property-propertyname-is-writeonly"></a>Vlastnost '\<propertyname >' je'jen pro zápis.
Pokusili jste se načtení hodnoty z vlastnosti, která je deklarována `WriteOnly`.  
  
 **ID chyby:** BC30524  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `WriteOnly` specifikátor v deklaraci vlastnosti.
