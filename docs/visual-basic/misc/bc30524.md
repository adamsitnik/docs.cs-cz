# <a name="property-ltpropertynamegt-is-writeonly"></a>Vlastnost "&lt;propertyname&gt;" je "jen pro zápis.
Pokusili jste se načtení hodnoty z vlastnosti, která je deklarována `WriteOnly`.  
  
 **ID chyby:** BC30524  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `WriteOnly` specifikátor v deklaraci vlastnosti.
