# <a name="method-ltmethodname1gt-does-not-have-the-same-generic-constraints-as-the-partial-method-ltmethodname2gt"></a>Metoda "&lt;methodname1&gt;"nemá stejná obecná omezení jako částečná metoda"&lt;methodname2&gt;.
Definovali jste implementace částečné metody, která má obecná omezení, které se liší od omezení v deklaraci částečné metody. Následující kód ukazuje chybu.  
  
```vb  
Partial Class Class1  
  
    Partial Private Sub Test(Of T As Class)(ByVal arg As T)  
    End Sub  
  
End Class  
  
Partial Class Class1  
  
    '' The error occurs here, for Test.  
    'Private Sub Test(Of T As Structure)(ByVal arg As T)  
    'End Sub  
  
End Class  
```  
  
 **ID chyby:** BC31438
