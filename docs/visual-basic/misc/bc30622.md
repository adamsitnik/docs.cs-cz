# <a name="end-module-must-be-preceded-by-a-matching-module"></a>'End Module' musí předcházet odpovídající 'Module'
Modul bloky začínají `Module` příkazu a končit `End Module` vytvořit.  
  
 **ID chyby:** BC30622  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že modul blok začíná `Module` příkazu.
