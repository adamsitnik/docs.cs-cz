# <a name="set-not-permitted"></a>Metoda Set není povolena
Jste se pokusili získat změnit vlastnost, jejíž nastavení buď nelze nastavit v době běhu, jinak může být nastaven pouze za určitých podmínek. Například může Pokusili jste se změnit `Appearance`, `ControlBox`,`MinButton`, nebo `MaxButton` možná jste se pokusili nastavení vlastností pro formulář v době běhu, nebo můžete nastavit `Visible` vlastnost `False` za poslední zbývající viditelné podnabídce nadřazené nabídky.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte vlastnosti a určit, za jakých podmínek je možné nastavit.
