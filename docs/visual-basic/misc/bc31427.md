# <a name="syntax-error-in-conditional-compilation-expression"></a>Syntaktická chyba ve výrazu podmíněné kompilace
Výraz podmíněné kompilace generuje nespecifickou chybu syntaxe.  
  
 **ID chyby:** BC31427  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Projděte si dokumentaci pro podmíněnou kompilaci a pro klíčová slova použitá ve výrazu. Porovnejte vzorový kód v dokumentaci proti zdrojovému řádku, který generuje tuto chybu.  
  
2. Pokud nemůžete zjistit příčinu této chyby, shromážděte informace o okolnostech a upozorněte služby Microsoft Product Support Services.
