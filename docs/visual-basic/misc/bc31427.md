# <a name="syntax-error-in-conditional-compilation-expression"></a>Chyba syntaxe ve výrazu podmíněné kompilace
Výraz podmíněnou kompilaci vygeneruje chybu Obecná syntaxe.  
  
 **ID chyby:** BC31427  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte v dokumentaci pro podmíněnou kompilaci a klíčová slova používaná ve výrazu. Porovnejte kód příkladu v dokumentaci pro zdrojový řádek generování této chyby.  
  
2. Pokud nelze určit příčinu chyby, shromážděte informace o okolnostech a upozornit Microsoft Product Support Services.
