# <a name="file-name-already-declared-with-a-different-guid-and-checksum-value"></a>Název souboru již byl deklarován s jiným identifikátorem GUID a kontrolního součtu hodnotou
Chcete-li deklarovat souboru byl proveden pokus o, ale soubor už existuje s jiným identifikátorem GUID a kontrolního součtu hodnotou.  
  
 **ID chyby:** BC42034  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte název souboru a cestu k Ujistěte se, že všechny jsou správné.  
  
- Odstraňte redundantní soubor, pokud není potřeba.
