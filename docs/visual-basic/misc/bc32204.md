# <a name="event-name-length-cannot-exceed-1011-characters"></a>Délka názvu události nesmí překročit 1011 znaků
Název nebo identifikátor každé události deklarované pomocí `Event` příkaz je omezená na 1011 znaků.  
  
 **ID chyby:** BC32204  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkraťte identifikátoru.
