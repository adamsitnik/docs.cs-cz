# <a name="the-folder-cannot-be-created-since-a-file-already-exists-with-the-same-path"></a>Složku nelze vytvořit, protože soubor se stejnou cestou již existuje
Byl proveden pokus o vytvoření adresáře, ale soubor už existuje. pomocí této cesty. To může být důsledek nesprávně analýzy cesty, zejména při práci s relativní cesty.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, že zadáváte správnou cestu.  
  
- Pokud už je nepotřebujete, odstraňte redundantní soubor.
