# <a name="shared-sub-new-cannot-have-any-parameters"></a>Sdílené proceduru Sub New nemůže mít žádné parametry
Jste zahrnuli do sdílené parametrem nebo parametry `Sub New` příkaz, který není platný.  
  
 **ID chyby:** BC30479  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte parametrem nebo parametry.
