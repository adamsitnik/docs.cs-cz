# <a name="ltnamegt-is-ambiguous-between-declarations-in-modules-ltmodulename1gt-and-ltmodulename2gt"></a>"&lt;název&gt;je nejednoznačný mezi deklaracemi v modulech&lt;modulename1&gt;"a"&lt;modulename2&gt;.
Zadali jste název, který je nejednoznačný a proto je v konfliktu s jiným názvem. Kompilátor jazyka Visual Basic nemá žádná pravidla řešení konfliktů; rozlišení názvů musí být sami.  
  
 **ID chyby:** BC30562  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Plně kvalifikujte název.
