# <a name="membername1-cannot-override-membername2-because-they-differ-by-the-types-of-optional-parameters"></a>'\<membername1 >' nelze přepsat '\<membername2>' protože se liší typy nepovinných parametrů
Přetěžované členy musí mít různé datové typy pro jeden nebo více požadovaných parametrů.  
  
 **ID chyby:** BC30697  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Jako jiný datový typ definujte aspoň jeden povinný parametr.
