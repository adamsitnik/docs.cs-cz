# <a name="ltmembername1gt-cannot-override-ltmembername2gt-because-they-differ-by-the-types-of-optional-parameters"></a>"&lt;membername1&gt;nemůže přepsat&lt;membername2&gt;' protože se liší typy nepovinných parametrů
Přetěžované členy musí mít různé datové typy pro jeden nebo více požadovaných parametrů.  
  
 **ID chyby:** BC30697  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Jako jiný datový typ definujte aspoň jeden povinný parametr.
