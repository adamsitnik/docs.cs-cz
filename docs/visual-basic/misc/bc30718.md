# <a name="select-statements-are-not-valid-in-the-immediate-window"></a>Příkazy 'Select' nejsou platné v příkazovém
`Select` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30718  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `Select` příkazy z ladění kódu.
