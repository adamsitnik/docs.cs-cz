# <a name="on-goto-and-on-gosub-statements-are-no-longer-supported"></a>"Na GoTo" a "Na GoSub" příkazy již nejsou podporovány
Už nejsou platné. použijte hodnotu proměnné nebo výrazu `GoTo` a `GoSub` příkazů do ovládacího prvku programu toku.  
  
 **ID chyby:** BC30817  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změny struktury aplikace pomocí `If...Then...Else` nebo `Case` příkazy.
