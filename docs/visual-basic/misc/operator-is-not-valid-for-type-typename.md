# <a name="operator-is-not-valid-for-type-lttypenamegt"></a>Operátor není platný pro typ '&lt;typename&gt;.
Operátor není vhodná pro typy.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že operátor, který je vhodný pro typy, se kterou se pracuje na.
