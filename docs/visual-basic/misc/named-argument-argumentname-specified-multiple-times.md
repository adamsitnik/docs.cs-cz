# <a name="named-argument-ltargumentnamegt-specified-multiple-times"></a>Pojmenovaný argument '&lt;název argumentu argumentname&gt;"zadaný několikrát.
Pojmenovaný argument můžete zadat více než jednou.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte všechny kromě jednoho specifikací.
