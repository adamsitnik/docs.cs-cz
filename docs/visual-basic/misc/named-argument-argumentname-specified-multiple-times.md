# <a name="named-argument-argumentname-specified-multiple-times"></a>Pojmenovaný argument '\<název argumentu argumentname >' zadaný několikrát.
Pojmenovaný argument můžete zadat více než jednou.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte všechny kromě jednoho specifikací.
