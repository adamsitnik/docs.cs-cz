 **ID chyby:** BC30827  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použití `^` operátor nebo `E+` znaků k označení exponentu je k dispozici. Příklad:

    Const Mole = 6.02E+23 ' Same as 6.02D23
