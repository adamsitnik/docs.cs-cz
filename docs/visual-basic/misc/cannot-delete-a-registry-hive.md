# <a name="cannot-delete-a-registry-hive"></a>Nelze odstranit podregistr registru
Byl proveden pokus o odstranění podregistr registru. Podregistry jsou klíče registru nejvyšší úrovně, jako je například `HKEY_CURRENT_USER` a `HKEY_LOCAL_MACHINE`a nelze ji odstranit.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte klíč registru, který se pokoušíte odstranit a ujistěte se, že zadáváte správnou cestu.
