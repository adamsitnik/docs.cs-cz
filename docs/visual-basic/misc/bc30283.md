# <a name="sub-new-cannot-be-declared-overrides"></a>'Sub New' nelze použít deklaraci 'přepsání.
Konstruktor Určuje, že přepsání zděděného konstruktoru. Konstruktory nelze přepsat.  
  
 **ID chyby:** BC30283  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Overrides` – klíčové slovo z `Sub` deklarace.
