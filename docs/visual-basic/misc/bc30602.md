# <a name="interface-members-must-be-methods-properties-events-or-type-definitions"></a>Členy rozhraní musí být metody, vlastnosti, události nebo definice typů.
Pouze vnořené typy, metody, vlastnosti a události mohou být členy rozhraní.  
  
 **ID chyby:** BC30602  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že položka je metoda, vlastnost, událost nebo vnořený typ, jako je například výčet.
