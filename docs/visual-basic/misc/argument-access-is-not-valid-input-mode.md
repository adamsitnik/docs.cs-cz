# <a name="argument-access-is-not-valid-valid-values-for-input-mode-are-openaccessread-and-openaccessdefault"></a>Argument "Přístup" není platný. Platnými hodnotami režimu Input jsou "OpenAccess.Read" a "OpenAccess.Default.
Hodnotu přístupu není platný v `Input` režimu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte hodnotu na buď `OpenAccess.Read` nebo `OpenAccess.Default`.
