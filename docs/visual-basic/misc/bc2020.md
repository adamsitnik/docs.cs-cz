# <a name="the-maximum-number-of-errors-has-been-exceeded"></a>Byl překročen maximální počet chyb
Bylo dosaženo kompilátoru maximálního počtu 100 chyb.  
  
 **ID chyby:** BC2020  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ladění a adresování každý zase snížit počet vygenerované chyby.
