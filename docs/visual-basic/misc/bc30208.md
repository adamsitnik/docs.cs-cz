# <a name="compare-expected"></a>"Porovnání" očekávání
`Option` Příkaz určuje `Text` nebo `Binary`, ale ne `Compare`.  
  
 **ID chyby:** BC30208  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidat `Compare` klíčové slovo `Option` příkazu, například `Option Compare Binary`.
