# <a name="operator-operator-must-have-either-one-or-two-parameters"></a>Operátor '\<operátoru >' musí mít jeden nebo dva parametry
Operátor, který může být buď unární nebo binární soubor je definována bez parametrů nebo s více než dva parametry.  
  
 Operátor unární nebo binární musí mít jeden nebo dva parametry.  
  
 **ID chyby:** BC33016  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Upravte definici a určete jeden nebo dva parametry.  
  
- Pokud potřebujete žádné parametry nebo větší než dvě, je nutné použít [Function – příkaz](../../visual-basic/language-reference/statements/function-statement.md) k definování `Function` postupu namísto operátoru.
