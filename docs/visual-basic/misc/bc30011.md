# <a name="internal-compiler-error-code-generator-received-malformed-input"></a>Vnitřní chyba kompilátoru: Generátor kódu přijal poškozený vstup
Kompilátor jazyka Visual Basic nelze interpretovat úspěšně jeden z jeho interně napsané souborů.  
  
 **ID chyby:** BC30011  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Kompilovat program znovu, abyste viděli, pokud se chyba objeví znovu.  
  
2. Pokud se chyba objeví znovu, přeinstalujte kompilátor jazyka Visual Basic.  
  
3. Pokud chyba přetrvává po přeinstalaci, shromážděte informace o okolnostech a upozornit Microsoft Product Support Services.
