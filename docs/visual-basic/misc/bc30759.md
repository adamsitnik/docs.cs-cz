# <a name="referenced-membername-has-a-value-of-nothing"></a>Odkazované "\<membername >" má hodnotu Nothing"
Položky, na kterou se odkazuje buď není byla přiřazena hodnota, nebo jí byla přiřazena hodnota `Nothing`.  
  
 **ID chyby:** BC30759  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ověřte název této položky je napsaný a zařídit správné.
