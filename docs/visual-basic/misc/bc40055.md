# <a name="casing-of-namespace-name-namespacename1-does-not-match-casing-of-namespace-name-namespacename2-in-file-filepath"></a>Malá a velká písmena názvu oboru názvů '\<namespacename1 >' neodpovídají názvu oboru názvů '\<namespacename2 >' v souboru '\<filepath > '
Obor názvů se objevuje víckrát v projektu, ale s jinou písmen.  
  
 *Malá a velká písmena* odkazuje na velká a malá znaky názvu programovací element. Visual Basic je velká a malá písmena, ale modul CLR (CLR) je velká a malá písmena. Další informace najdete v tématu „Případ citlivosti v názvy“ v [deklarované názvy elementů](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 Ve výchozím nastavení tato zpráva je upozornění. Informace o zobrazení nebo skrytí upozornění zpracování upozornění jako chyby, najdete v části [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID chyby:** BC40055  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Jako preventivní opatření vždy používejte stejné malých a velkých písmen v všechny odkazy na obor názvů. To může zabránit špatného modulem common language runtime.
