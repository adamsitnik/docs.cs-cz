# <a name="goto-labelname-is-not-valid-because-labelname-is-inside-a-try-catch-or-finally-statement-that-does-not-contain-this-statement"></a>'GoTo \<labelname >' je neplatný protože'\<labelname > 'se nachází uvnitř bloku Try, 'příkaz Catch' nebo 'nakonec příkaz, který tento příkaz neobsahuje
Nelze větvit do `Try...Catch...Finally` bloku.  
  
 **ID chyby:** BC30754  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změny struktury kódu tak, aby předchází popisek `Try...Catch...Finally` bloku.
