# <a name="structures-cannot-have-inherits-statements"></a>Příkazy 'Inherits' nelze použít ve strukturách
Na rozdíl od třídy struktury nepodporují dědičnosti.  
  
 **ID chyby:** BC30628  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Inherits` příkaz ze struktury.  
  
- Změnit návrh aplikace místo toho použít třídy.
