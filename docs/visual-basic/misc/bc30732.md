# <a name="web-methods-cannot-be-evaluated-in-debug-windows"></a>Webové metody nelze vyhodnotit v ladicích oknech
V okně ladění nelze provést test metody webové služby XML.  
  
 **ID chyby:** BC30732  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Testovací metody webové služby XML z editoru kódu.
