# <a name="web-methods-cannot-be-evaluated-in-debug-windows"></a>Webové metody nelze vyhodnotit v ladicích oknech.
Metody webové služby XML nelze testovat v okně ladění.  
  
 **ID chyby:** BC30732  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Otestujte metody webové služby XML z editoru kódu.
