# <a name="expression-or-statement-is-not-valid-in-debug-windows"></a>Výraz nebo příkaz nejsou platné v ladicích oknech
`Try...Catch...Finally` Příkazy nelze použít v kontextu ladění.  
  
 **ID chyby:** BC30706  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odebrat `Try...Catch...Finally` příkazy z ladění kódu.
