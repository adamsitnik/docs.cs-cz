# <a name="readonly-and-writeonly-cannot-be-combined"></a>Nelze kombinovat 'ReadOnly' a 'jen pro zápis.
`ReadOnly` a `WriteOnly` modifikátory se vzájemně vylučují.  
  
 **ID chyby:** BC30635  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte buď `ReadOnly` nebo `WriteOnly`, ale ne obojí.
