# <a name="withevents-variable-does-not-raise-any-instance-events-that-are-accessible-to-containername"></a>'Proměnnou WithEvents' nevyvolává žádné události instance, které jsou k dispozici'\<containername > '
Objekt, na který odkazujete, nevyvolává přístupné události.  
  
 **ID chyby:** BC30600  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Upravte modifikátory přístupu pro událost nebo vyberte událost, která je k dispozici v aktuálním kontextu.
