# <a name="withevents-variable-does-not-raise-any-instance-events-that-are-accessible-to-containername"></a>"Proměnnou WithEvents' nevyvolává žádné události instance, které jsou k dispozici"\<containername > "
Objekt, který se odkazuje na nevyvolává dostupná událostem.  
  
 **ID chyby:** BC30600  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Upravit modifikátory přístupu pro událost nebo zvolit událost, která je přístupná v rámci aktuálního kontextu.
