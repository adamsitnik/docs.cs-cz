# <a name="simplifiedchinese-and-vbstrconvtraditionalchinese-cannot-be-combined"></a>SimplifiedChinese a VbStrConv. TraditionalChinese nelze kombinovat.
Vaše aplikace se pokouší kombinovat členy výčtu `VbStrConv` `SimplifiedChinese` a `TraditionalChinese`, které se vzájemně vylučují.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte buď `VbStrConv.SimplifiedChinese`, nebo `VbStrConv.TraditionalChinese`.
