# <a name="simplifiedchinese-and-vbstrconvtraditionalchinese-cannot-be-combined"></a>SimplifiedChinese a VbStrConv.TraditionalChinese nelze kombinovat
Vaše aplikace se pokouší o kombinování `VbStrConv` členy výčtu `SimplifiedChinese` a `TraditionalChinese`, které se vzájemně vylučují.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Buď odeberte `VbStrConv.SimplifiedChinese` nebo `VbStrConv.TraditionalChinese`.
