# <a name="-expected"></a>')' očekávání
Seznam, jako je například seznam argumentů v definici procedury nebo volání procedury, není ukončena pravou závorkou (`)`) znaků.  
  
 **ID chyby:** BC30198  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte pravou závorku znak na konci tohoto seznamu.
