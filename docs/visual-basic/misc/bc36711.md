# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>Argument odpovídající parametru '\<parametername > se zužuje na "\<typename >"
Bylo provedeno volání přetěžované metody, ale kompilátor nemůže najít metodu, kterou lze volat bez omezujícího převodu. A *zužující převod* změní hodnotu na datový typ, který nemusí být schopný uchovat některou z možných hodnot.  
  
 **ID chyby:** BC36711
