## <a name="example"></a><span data-ttu-id="2185c-111">Příklad</span><span class="sxs-lookup"><span data-stu-id="2185c-111">Example</span></span>

<span data-ttu-id="2185c-112">Následující příklad používá `AttributeUsage` atribut k určení, které prvky programu `ExtensionAttribute` může nová verze použít na.</span><span class="sxs-lookup"><span data-stu-id="2185c-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="2185c-113">Příklad `AttributeTargets` určuje tři členy výčtu: `Assembly`, `Class`, a `Method`.</span><span class="sxs-lookup"><span data-stu-id="2185c-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="2185c-114">Tato chyba způsobí Vynechání některého z těchto prvků.</span><span class="sxs-lookup"><span data-stu-id="2185c-114">The omission of any one of these elements will cause this error.</span></span>

```vb

        AttributeTargets.Class Or AttributeTargets.Method)>

        Inherits System.Attribute
        ' Definitions of methods, fields, and properties.
    End Class
End Namespace
```
