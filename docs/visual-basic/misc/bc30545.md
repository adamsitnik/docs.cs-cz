# <a name="property-access-must-assign-to-the-property-or-use-its-value"></a>Přístup k vlastnosti musí přiřadit vlastnosti nebo použijte její hodnotu
Pokusili jste se přístup k vlastnosti bez přiřazení k němu nebo pomocí jeho hodnotu.
  
 **ID chyby:** BC30545  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přiřaďte hodnotu k vlastnosti.  
  
     \- nebo –  
  
-   Použijte hodnotu vlastnosti.
