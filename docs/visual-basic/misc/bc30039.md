# <a name="loop-control-variable-cannot-be-a-property-or-a-late-bound-indexed-array"></a>Řídicí proměnná smyčky nemůže být vlastnost ani indexované pole s pozdní vazbou
Proměnné použité k iteraci v rámci `For` smyčky musí být číselného datového typu.  
  
 **ID chyby:** BC30039  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte deklaraci řídicí proměnná smyčky k určení `Integer`, `Short`, `Long`, `Byte`, `Single`, `Double`, nebo `Decimal`.
