# <a name="variables-in-modules-cannot-be-declared-specifier"></a>Proměnné v modulu nelze použít deklaraci '\<specifikátor > "
Specifikátor, jste už použili, jako `MustInherit`, na proměnnou `Module` příkazu. Moduly možné nikdy instancovat, nepodporují dědičnosti a nemůžou implementovat rozhraní.  
  
 **ID chyby:** BC30593  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
