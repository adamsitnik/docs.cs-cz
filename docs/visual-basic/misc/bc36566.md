# <a name="parameter-ltparameternamegt-already-has-a-matching-omitted-argument"></a>Parametr '&lt;parametername&gt;' již existuje odpovídající vynechaný argument
Volání procedury poskytuje po vynechání stejné argumentů podle pozice argumentu podle názvu. Tady je příklad:  
  
```vb  
Public Sub ABC(ByVal X As Byte, Optional ByVal Y As Byte = 0, _  
                                Optional ByVal Z As Byte = 0)  
' ...  
' Argument Y is omitted by position, but supplied by name.  
Call ABC(6, , Y:=3)     
```  
  
 **ID chyby:** BC36566  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadání argumentů podle pozice nebo odstraňte čárku, který se vynechá.
