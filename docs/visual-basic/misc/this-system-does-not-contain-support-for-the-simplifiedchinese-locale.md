# <a name="this-system-does-not-contain-support-for-the-simplifiedchinese-locale"></a>Tento systém neobsahuje podporu pro národní prostředí SimplifiedChinese.
Pokoušíte se použít `VbStrConv.SimplifiedChinese` člen výčtu, ale systém nepodporuje zjednodušené čínské národní prostředí.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Upgradujte počítač tak, aby podporoval zjednodušené čínské národní prostředí, nebo spusťte aplikaci na počítači, který podporuje národní prostředí.
