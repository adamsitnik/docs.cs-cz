# <a name="unable-to-sign-assembly-lterror-messagegt"></a>Sestavení nelze podepsat: &lt;chybová zpráva&gt;
Kompilátor jazyka Visual Basic volá Assembly Linker (Al.exe, označované také jako Alink) ke generování sestavení s manifestem. Linker ohlásilo chybu podepsání sestavení po jeho vytvoření. To může být z důvodu neplatného klíče.  
  
 **ID chyby:** BC30146  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte v uvozovkách chybovou zprávu a najdete v tématu [Al.exe (Linker sestavení)](../../framework/tools/al-exe-assembly-linker.md) pro další vysvětlení a poradenství.  
  
2.  Pokud potíže potrvají, shromážděte informace o okolnostech a upozornit Microsoft Product Support Services.
