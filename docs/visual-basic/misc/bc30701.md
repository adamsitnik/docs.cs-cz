# <a name="nothing-cannot-be-evaluated"></a>'Nothing' nemůže být vyhodnocena
`Nothing` Klíčové slovo se používá v okně ladění.  
  
 **ID chyby:** BC30701  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nepoužívejte `Nothing` během ladění.
