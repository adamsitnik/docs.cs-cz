# <a name="ltmethodgt-ltmethodname1gt-must-be-declared-overloads-because-another-ltmethodname2gt-is-declared-overloads"></a>&lt;– Metoda&gt; "&lt;methodname1&gt;' musí být deklarován 'Přetížení', protože jiný"&lt;methodname2&gt;' je deklarován "přetížení.
Jiné vlastnosti nebo metody se stejným názvem, je deklarována s `Overloads` modifikátor.  
  
 **ID chyby:** BC31409  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidat `Overloads` modifikátor k vlastnosti nebo metody.  
  
2.  Odeberte `Overloads` modifikátor všechny vlastnosti nebo metody se stejným názvem.
