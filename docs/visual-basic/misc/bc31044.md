# <a name="events-declared-with-an-as-clause-must-have-a-delegate-type"></a>Události deklarované s klauzulí 'Jako' musí mít typ delegáta
Typ byla zadána, který není delegate.  
  
 **ID chyby:** BC31044  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `As` klauzuli.  
  
2.  Zadejte delegát s `As` klauzuli.
