# <a name="classes-cannot-be-declared-specifier"></a>Třídy nejde použít deklaraci '\<specifikátor > "
Pokusili jste se deklarovat `Class` nebo `Module` s Neplatný specifikátor.  
  
 **ID chyby:** BC30461  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
