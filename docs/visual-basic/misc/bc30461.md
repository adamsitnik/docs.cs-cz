# <a name="classes-cannot-be-declared-ltspecifiergt"></a>Třídy nejde použít deklaraci '&lt;specifikátor&gt;.
Pokusili jste se deklarovat `Class` nebo `Module` s Neplatný specifikátor.  
  
 **ID chyby:** BC30461  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
