# <a name="array-initializer-has-too-many-dimensions"></a>Inicializátor pole má příliš mnoho rozměrů.
V inicializátoru pole jste zadali příliš mnoho dimenzí.  
  
 **ID chyby:** BC30566  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte inicializátor pole, abyste zjistili, kolik dimenzí je potřeba.  
  
- Odeberte další argument nebo argumenty.
