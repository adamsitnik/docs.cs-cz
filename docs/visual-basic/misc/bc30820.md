# <a name="lsetstmt-is-not-declared"></a>"\<lsetstmt >' není deklarována
"\<lsetstmt >' není deklarována. Příkazy LSet již nejsou podporovány. Použijte místo nich příkazy Microsoft.VisualBasic.LSet.  
  
 Několik funkcí, které byly vnitřní do jazyka Visual Basic v předchozích verzích se přesunuly do <xref:Microsoft.VisualBasic?displayProperty=nameWithType> oboru názvů. Díky jejich funkce obecně k dispozici pro všechny programovací jazyky.  
  
 **ID chyby:** BC30820  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `LSet` fungovat v `Microsoft.VisualBasic` obor názvů místo.
