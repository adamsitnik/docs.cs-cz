# <a name="assemblyname-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>'\<assemblyname >' je sestavení a nelze na něj odkazovat jako na modul
Byl proveden pokus o použití sestavení jako modul.  
  
 **ID chyby:** BC31077  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte odkaz na modul.
