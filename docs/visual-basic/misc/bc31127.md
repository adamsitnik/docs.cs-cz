# <a name="addhandler-is-already-declared"></a>'AddHandler' je už deklarovaný.
Více než jeden `AddHandler` deklarace se zobrazí v deklaraci vlastní událost. `AddHandler` Deklarace deklaruje postup pro přidání obslužné rutiny události.  
  
 **ID chyby:** BC31127  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat redundantní `AddHandler` příkazu.
