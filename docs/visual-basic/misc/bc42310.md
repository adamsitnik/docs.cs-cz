# <a name="xml-comment-tag-include-must-have-a-attribute-attribute"></a>Značka komentáře XML musí mít 'zahrnout' '\<atribut >' atribut
Značka komentáře XML musí mít 'zahrnout' \<atribut > atribut. Komentář XML bude ignorován.  
  
 A `file` nebo `path` atribut předat `<`zahrnují`>` značku, chybí nebo je neúplná.  
  
 **ID chyby:** BC42310  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidání nebo dokončení požadovaný atribut pro \<zahrnout > značky.
