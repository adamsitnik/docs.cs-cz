# <a name="xml-name-expected"></a>Očekával se název XML
XML literál nebo vlastnosti osy XML nebyl zadán bez názvu.  
  
 **ID chyby:** BC31146  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidání názvu pro literál XML nebo vlastnosti osy XML.
