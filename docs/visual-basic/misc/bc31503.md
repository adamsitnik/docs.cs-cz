# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>'\<typename>' nelze použít jako atribut, protože se nejedná o třídu
Byl proveden pokus o použití typu, který není třídou jako atribut.  
  
 **ID chyby:** BC31503  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Definujte vlastní atributy jako třídy, které jsou `System.Attribute`odvozeny z.
