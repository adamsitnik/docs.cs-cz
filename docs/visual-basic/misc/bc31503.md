# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>"\<typename >' nelze použít jako atribut, protože se nejedná o třídu
Došlo pokusu o použití typu, který se nejedná o třídu jako atribut.  
  
 **ID chyby:** BC31503  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Definovat vlastní atributy jako třídy, které jsou odvozeny z `System.Attribute`.
