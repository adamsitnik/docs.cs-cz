# <a name="unable-to-read-delimited-fields-because-delimiters-is-nothing-or-empty"></a>Nelze číst pole oddělená oddělovačem, protože parametr Delimiters má hodnotu Nothing nebo je prázdný
`TextFieldParser` Nejsou schopny číst ze souboru, protože `Delimiters` je nastavena na `Nothing` nebo je prázdný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte platnou hodnotu pro `Delimiters`.
