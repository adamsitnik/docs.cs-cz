# <a name="system-event-log-cannot-be-deleted"></a>Protokol událostí systému nelze odstranit.
Byl proveden pokus o odstranění protokolu událostí systému, kterou nejde odstranit. Systémového protokolu se sleduje systémové události, jako je například selhání spuštění a hardware systému.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zvažte, jestli by vaše aplikace napsat do aplikace nebo vlastního protokolu, nikoli protokol událostí systému.  
  
- Nepokoušejte se smazat protokol událostí systému.
