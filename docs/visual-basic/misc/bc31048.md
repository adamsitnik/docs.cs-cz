# <a name="properties-with-no-required-parameters-cannot-be-declared-default"></a>Vlastnosti bez požadovaných parametrů nelze použít deklaraci "Výchozí"
Vlastnost byl zadán bez požadovaných parametrů, ale s `Default` modifikátor.  
  
 **ID chyby:** BC31048  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Deklarujte požadované parametry pro výchozí vlastnosti.  
  
- Odeberte `Default` modifikátoru v definici vlastnosti.
