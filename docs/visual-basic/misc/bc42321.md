# <a name="badly-formed-xml-file-ltcommentfilegt-cannot-be-included"></a>Chybně vytvořený kód XML. Soubor '&lt;commentfile&gt;"nelze zahrnout
`<include>` Značky v jednom z komentářů XML odkazuje na neexistující soubor nebo soubor, který obsahuje chybně vytvořený kód XML.  
  
 **ID chyby:** BC42321  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, zda soubor existuje.  
  
2.  Ověřte, že je platný kód XML v souboru.
