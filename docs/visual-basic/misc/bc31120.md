# <a name="end-raiseevent-must-be-the-first-statement-on-a-line"></a>'End RaiseEvent' musí být prvním příkazem na řádku
`End RaiseEvent` Příkaz následuje dvojtečka (:) oddělovač. `End RaiseEvent` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC31120  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Oddělte více příkazů na samostatné řádky.
