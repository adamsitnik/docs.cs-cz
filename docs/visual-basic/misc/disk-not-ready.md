# <a name="disk-not-ready"></a>Disk není připraven
Pro vybranou jednotku není žádný disk nebo je otevřená dvířka jednotky.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte zadané jednotce a vložte disk v případě potřeby.  
  
2.  Zavřete dvířka jednotky.
