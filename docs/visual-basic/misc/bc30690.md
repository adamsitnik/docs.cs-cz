# <a name="structure-structurename-cannot-be-indexed-because-it-has-no-default-property"></a>Struktura '\<%{structurename/ >' nelze indexovat, protože nemá žádnou výchozí vlastnost
Byl proveden pokus o použití výchozí vlastnost syntaxe se strukturou, který nemá žádnou výchozí vlastnost.  
  
 **ID chyby:** BC30690  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přístup k vlastnostem strukturu, použijte syntaxi standardní vlastnosti.
