# <a name="warning-number-number-for-the-option-optionname-is-either-not-configurable-or-not-valid"></a>Číslo upozornění '\<číslo >' pro možnost '\<název_možnosti >' není konfigurovatelné nebo platný
ID upozornění není platný. Identifikátory musí být celočíselné hodnoty vyšší než 0.  
  
 **ID chyby:** BC2026  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte zadané abyste měli jistotu, že je platné ID upozornění.
