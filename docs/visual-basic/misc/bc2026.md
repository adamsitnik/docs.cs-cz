# <a name="warning-number-ltnumbergt-for-the-option-ltoptionnamegt-is-either-not-configurable-or-not-valid"></a>Číslo upozornění '&lt;číslo&gt;'parametru'&lt;název_možnosti&gt;"není konfigurovatelné nebo platný
ID upozornění není platný. Identifikátory musí být celočíselné hodnoty vyšší než 0.  
  
 **ID chyby:** BC2026  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte zadané abyste měli jistotu, že je platné ID upozornění.
