# <a name="keyword-is-not-valid-within-a-structure"></a>'\<– klíčové slovo >' není platný v rámci struktury
Struktury jsou typy hodnot, nikoli odkazové typy. Struktura není instancí vytvořenou z třídy, takže `Me`klíčová slova, `MyClass`a a `MyBase` nemají v rámci struktury žádný význam.  
  
 **ID chyby:** BC30044  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte strukturu na třídu nebo odeberte klíčové slovo z procedury.
