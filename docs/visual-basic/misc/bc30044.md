# <a name="ltkeywordgt-is-not-valid-within-a-structure"></a>"&lt;– klíčové slovo&gt;' není platný v rámci struktury
Struktury jsou typy hodnot, typy odkazů nikoli. Struktura se nejedná o instanci vytvořené ze třídy, proto `Me`, `MyClass`, a `MyBase` klíčová slova jsou bezvýznamné ve struktuře.  
  
 **ID chyby:** BC30044  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změnit strukturu pro třídu nebo odeberte klíčové slovo z procedury.
