# <a name="ltpropertygt-parameters-cannot-be-declared-byref"></a>&lt;Vlastnost&gt; parametry nejde použít deklaraci 'ByRef'
Parametry pro vlastnosti musí být deklarovány pomocí `ByVal` modifikátor.  
  
 **ID chyby:** BC30651  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Definovat vlastnost parametry jako `ByVal`.
