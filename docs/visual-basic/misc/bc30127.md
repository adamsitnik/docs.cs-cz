# <a name="attribute-attributename-is-not-valid-error"></a>Atribut '\<attributename >' není platný: \<chyby >
Modul common language runtime nepřijímá atributů v rámci aktuálního kontextu.  
  
 **ID chyby:** BC30127  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Atribut z deklarace ho odeberte nebo nahraďte atributem, který je platný pro daný kontext.  
  
2.  Prověřením chybové zprávy a přijmout vhodná opatření.
