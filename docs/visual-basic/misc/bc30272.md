# <a name="argumentname-is-not-a-parameter-of-methoddeclaration"></a>'\<název argumentu argumentname >' není parametr '\<methoddeclaration >'
Volání procedury poskytuje další argument, který není definován procedurou.  
  
 **ID chyby:** BC30272  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nadbytečný argument odeberte ze seznamu argumentů.
