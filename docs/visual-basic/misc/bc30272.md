# <a name="ltargumentnamegt-is-not-a-parameter-of-ltmethoddeclarationgt"></a>"&lt;název argumentu argumentname&gt;"není parametr"&lt;methoddeclaration&gt;.
Volání procedury poskytuje další argument, který není definován procedurou.  
  
 **ID chyby:** BC30272  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nadbytečný argument odeberte ze seznamu argumentů.
