# <a name="argument-ltargumentnamegt-must-be-greater-than-0-or-equal-to--1"></a>Argument '&lt;název argumentu argumentname&gt;' musí být větší než 0 nebo roven -1
Hodnota argumentu je menší než 0, ale není rovno -1.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že hodnota argumentu spadá v platném rozsahu.  
  
2.  Pokud je argument zadaný ve výpočtu, zkontrolujte hodnoty procesy výpočtu.
