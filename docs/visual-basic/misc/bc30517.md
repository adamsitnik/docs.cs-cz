# <a name="overload-resolution-failed-because-no-method-is-accessible"></a>Rozlišení přetěžování se nezdařilo, protože žádné '\<metoda >' je dostupný
Provedení volání přetížené metody, ale kompilátor nemůže najít dostupné metody se seznamem parametrů, který odpovídá z hlediska počtu parametrů.  
  
 **ID chyby:** BC30517  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte seznam argumentů, které jste zadali.  
  
2. Zkontrolujte metodu, kterou se pokoušíte volání za účelem ověření, zda je přístupný.
