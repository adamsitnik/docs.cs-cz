# <a name="attribute-value-is-not-valid-expecting-value1-or-value2"></a>Hodnota atributu není platná. byl očekáván "hodnota1" nebo "hodnota2"
Byla zadána neplatná hodnota pro atribut v dokumentu XML literál. Literál dokumentu XML platný atribut hodnoty jsou podle specifikace XML 1.0.  
  
 **ID chyby:** BC31182  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte hodnotu atributu na hodnotu, která je platný podle specifikace XML 1.0.
