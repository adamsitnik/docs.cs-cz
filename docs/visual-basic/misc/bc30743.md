# <a name="type-typename-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Typ '\<typename >' není definován nebo modulu, který ho obsahuje, není zaveden do ladicí relace
Typ odkazovalo, který není k dispozici nebo není definován.  
  
 **ID chyby:** BC30743  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, že název typu je napsán správně.  
  
2. Definování typu v rámci aktuální třídu nebo modul.
