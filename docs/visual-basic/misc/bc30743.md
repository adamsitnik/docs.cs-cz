# <a name="type-typename-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Typ\<TypeName > není definovaný nebo modul, který ho obsahuje, není načtený v relaci ladění.
Odkazovalo se na typ, který není k dispozici, nebo není definován.  
  
 **ID chyby:** BC30743  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, zda je název typu zadán správně.  
  
2. Definujte typ v rámci aktuální třídy nebo modulu.
