# <a name="sub-or-function-expected-after-delegate"></a>'Sub' nebo 'funkce"po"delegáta.
A `Delegate` příkaz neurčuje `Sub` nebo `Function` postup. `Sub` Nebo `Function` – klíčové slovo musí následovat bezprostředně `Delegate` – klíčové slovo.  
  
 **ID chyby:** BC30278  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidat `Sub` nebo `Function` – klíčové slovo ihned po `Delegate`.  
  
2.  Zadejte název procedury, seznam argumentů a návratový typ podle potřeby.
