# <a name="expression-name-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>Výrazu '\<name >' není procedura, ale vyskytuje se jako cíl volání procedury
Provedli jste postup volání výraz, který není procedura.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že píšete není název procedury.  
  
2. Zadejte proces, který <`name`> volání.
