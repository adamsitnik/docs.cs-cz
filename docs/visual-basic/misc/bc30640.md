# <a name="option-explicit-can-be-followed-only-by-on-or-off"></a>"Možnost Explicit" může následovat pouze 'Na' nebo 'Off'
`Option Explicit` příkazy musí určovat buď `On` nebo `Off`. Všechny ostatní hodnoty nejsou platné.  
  
 **ID chyby:** BC30640  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte buď `Option Explicit On` nebo `Option Explicit Off`.
