# <a name="at-least-one-parameter-type-of-this-binary-operator-must-be-the-containing-type-typename"></a>Alespoň jeden typ parametr tohoto binárního operátoru musí být nadřazeného typu "\<typename >"
Definice binárního operátoru určuje oba parametry s typy jiné než u třídy nebo struktury, ve kterém je operátor definován.  
  
 Když definujete ve třídě nebo struktuře operátor, nejméně jeden z parametrů musí být typu této třídě nebo struktuře.  
  
 **ID chyby:** BC33021  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změna typu jeden nebo oba parametry typu třídy nebo struktury, ve kterém je operátor definován.
