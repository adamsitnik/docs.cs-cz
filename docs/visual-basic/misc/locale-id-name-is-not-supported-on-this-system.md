# <a name="locale-id-ltnamegt-is-not-supported-on-this-system"></a>ID národního prostředí "&lt;název&gt;" není v tomto systému podporován.
Pokoušíte se použít `VbStrConv`, ale váš systém nepodporuje zadané národní prostředí.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Upgrade počítače pro podporu národní prostředí nebo spusťte aplikaci na počítač, který podporuje zadanému národnímu prostředí.
