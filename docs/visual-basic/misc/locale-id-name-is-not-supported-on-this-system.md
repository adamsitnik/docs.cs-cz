# <a name="locale-id-name-is-not-supported-on-this-system"></a>ID národního prostředí\<Name > se v tomto systému nepodporuje.
Pokoušíte se použít `VbStrConv`, ale váš systém nepodporuje zadané národní prostředí.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Upgradujte počítač pro podporu národního prostředí nebo spusťte aplikaci na počítači, který podporuje zadané národní prostředí.
