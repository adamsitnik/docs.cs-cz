# <a name="systemruntimeinteropservicesdllimportattribute-cannot-be-applied-to-a-get-or-set"></a>'System.Runtime.InteropServices.DllImportAttribute' nelze použít pro 'Get' nebo 'Set'
`DllImportAttribute` Atribut aplikováno `Get` nebo `Set` procedura property.  
  
 **ID chyby:** BC31524  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odebrat `DllImportAttribute` z `Get` a `Set` procedury vlastností.
