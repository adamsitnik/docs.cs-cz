# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>Příkaz nelze použít vně těla nebo víceřádkového výrazu lambda – metoda
Spustitelné příkazy musí využívat v rámci procedury.  
  
 **ID chyby:** BC30689  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesuňte příkaz do procedury nebo podprogramu.
