# <a name="unable-to-find-entry-point-name-in-dll-filename-error"></a>Nepovedlo se najít vstupní bod "\<název >" v knihovně DLL '\<název souboru >': \<chyby >
V zadané knihovně DLL se nenašel název procedury.  
  
 **ID chyby:** BC31014  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, jestli název procedury je napsaný správně a že malých a velkých písmen odpovídá, funkce v knihovně DLL.
