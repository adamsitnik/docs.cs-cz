# <a name="delegate-constructor-for-type-typename-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Konstruktor delegátů pro typ '\<typename >', je nezbytné pro implementaci delegátů nebyl nalezen.
Sestavení obsahující daný typ může být poškozený nebo není platný.  
  
 **ID chyby:** BC31074  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, zda je sestavení obsahující tento typ platný.  
  
2.  Kompilovat program znovu, abyste viděli, pokud se chyba objeví znovu.  
  
3.  Pokud chyba přetrvává, uložte si práci a restartujte aplikaci Visual Studio.
