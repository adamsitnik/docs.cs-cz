# <a name="property-missing-end-property"></a>Ve vlastnosti chybí 'End Property'
`End` Příkaz musí odpovídat bloku jeho ukončení. A `Property` definice nebyl správně zavřen.  
  
 **ID chyby:** BC30025  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidat `End Property` příkaz za účelem definice vlastnosti.
