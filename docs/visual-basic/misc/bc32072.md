<span data-ttu-id="db51c-103">Obsahuje deklarace třídy `Implements` příkaz, který určuje dvě nebo více rozhraní, ale alespoň jedno z rozhraní je obecný a dvě implementace by mohla konfliktu pro určité hodnoty argumentů typu.</span><span class="sxs-lookup"><span data-stu-id="db51c-103">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="db51c-104">Tato chyba může generovat následující příkazy.</span><span class="sxs-lookup"><span data-stu-id="db51c-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1  
    Sub testSub(ByVal arg As String)  
End Interface  
Public Interface iFace2(Of t)  
    Sub testSub(ByVal arg As t)  
End Interface  
Public Class testClass  
    Implements iFace1, iFace2(Of String)  
End Class  
```  
  
 <span data-ttu-id="db51c-105">Protože `iFace2` je vytvořená pomocí `String`, `testClass` musí implementovat dvě verze `testSub` s identickými podpisy.</span><span class="sxs-lookup"><span data-stu-id="db51c-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="db51c-106">To vytvoří nejednoznačnost o přístup k verzi.</span><span class="sxs-lookup"><span data-stu-id="db51c-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="db51c-107">**ID chyby:** BC32072</span><span class="sxs-lookup"><span data-stu-id="db51c-107">**Error ID:** BC32072</span></span>
