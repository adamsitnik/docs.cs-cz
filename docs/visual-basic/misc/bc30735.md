# <a name="type-in-a-module-cannot-be-declared-specifier"></a>Typ v modulu nejde použít deklaraci '\<specifikátor > "
Typ s neplatné modifikátory byl deklarován v rámci modulu.  
  
 **ID chyby:** BC30735  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nepoužívejte `Protected` nebo `Protected Friend` modifikátorů typy deklarované v rámci modulu.
