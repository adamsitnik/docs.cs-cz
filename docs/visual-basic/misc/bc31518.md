# <a name="assembly-assemblyname-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>Sestavení '\<assemblyname >' nelze vytvořit, protože příslušná cesta je delší než 259 znaků.
Cesta pro toto sestavení přesahuje maximální délku 259 znaků.  
  
 **ID chyby:** BC31518  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vytvoření s kratší cestou pomocí kratší názvy nebo výběrem shallower hierarchie souborů.
