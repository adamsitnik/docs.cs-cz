# <a name="resume-or-goto-expected"></a>'Pokračovat' nebo 'GoTo očekávání
`On Error` Příkaz neobsahuje buď `Resume` klauzule nebo `GoTo` klauzuli. Musí používat jednu klauzuli, nebo druhé.  
  
 **ID chyby:** BC32019  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat buď `Resume` klauzule nebo `GoTo` klauzule, která `On Error` příkaz.
