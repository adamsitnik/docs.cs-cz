# <a name="implicit-variable-variablename-is-invalid-because-of-message"></a>Implicitní proměnná\<Variable > je neplatná kvůli\<> zprávy.
Během ladění byla vygenerována neplatná implicitní proměnná.  
  
 **ID chyby:** BC36016  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Projděte citovanou zprávu o příčině a možné nápravě.  
  
2. Pokud chyba přetrvává, shromážděte informace o okolnostech a upozorněte služby podpory společnosti Microsoft.
