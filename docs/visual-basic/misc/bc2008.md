# <a name="no-input-sources-specified"></a>Nebyly zadány žádné vstupní zdroje.
Při kompilaci nejsou zadány žádné vstupní zdroje.  
  
 **ID chyby:** BC2008  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že jste zahrnuli soubor zdrojového kódu.
