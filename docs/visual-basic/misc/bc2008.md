# <a name="no-input-sources-specified"></a>Vstupní zdroje nebyly zadány
Při kompilaci, nebyly zadány žádné vstupní zdroje.  
  
 **ID chyby:** BC2008  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že jste zahrnuli souboru zdrojového kódu.
