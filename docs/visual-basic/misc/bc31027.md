# <a name="unable-to-open-file-filename-error"></a>Nelze otevřít soubor "\<název souboru >': \<chyby >
Došlo k chybě při otevírání souboru v tomto projektu.  
  
 **ID chyby:** BC31027  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, že soubor existuje a že nejsou poškozené nebo uzamčený jinou aplikací.
