# <a name="parameter-ltparameternamegt-already-has-a-matching-argument"></a>Parametr '&lt;parametername&gt;' již existuje odpovídající argument
Argument je předán podle názvu více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC36567  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda některý název argumentu je zadáno chybně.  
  
-   Duplicitní argument odeberte ze seznamu argumentů.
