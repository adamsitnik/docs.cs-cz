# <a name="comma-expected"></a>Byla očekávána čárka
Seznam argumentů je chybí čárka (,) mezi dvěma argumenty.  
  
 **ID chyby:** BC30196  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že všechny argumenty jsou odděleny čárkami.
