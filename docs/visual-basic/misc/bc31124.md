# <a name="end-addhandler-must-be-preceded-by-a-matching-addhandler-declaration"></a>'End AddHandler' musí předcházet párová deklarace "AddHandler.
`End AddHandler` Příkazu došlo k chybě bez odpovídající `AddHandler` příkazu. `End AddHandler` musí předcházet odpovídající `AddHandler` příkazu.  
  
 **ID chyby:** BC31124  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, že předchozí `AddHandler` příkazu je platný a správně zadané.
