# <a name="notinheritable-classes-cannot-have-members-declared-ltspecifiernamegt"></a>Třídy 'NotInheritable' nemůže obsahovat členy s deklarací '&lt;specifiername&gt;.
Přepsání Modifikátory nelze použít s `NotInheritable` třídy, protože jejich členy nelze přepsat.  
  
 **ID chyby:** BC30607  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat modifikátory přepsání, jako například `Overridable`, `NotOverridable`, nebo `MustOverride`, z definice třídy.
