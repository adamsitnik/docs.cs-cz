# <a name="membername-is-not-declared-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>"\<membername >' není deklarován nebo modul, který ho obsahuje, není zaveden do ladicí relace
Zadaný člen nebyl nalezen.  
  
 **ID chyby:** BC30699  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, že člen je definovaný a neobsahuje překlepy.  
  
2.  Přístup k některé z členy deklarované v modulu. V některých případech nelze ladicího prostředí najít členy, protože ještě nebyly načteny moduly, ve kterém jsou deklarovány.
