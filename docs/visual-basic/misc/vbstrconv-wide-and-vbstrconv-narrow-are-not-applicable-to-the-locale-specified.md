# <a name="vbstrconvwide-and-vbstrconvnarrow-are-not-applicable-to-the-locale-specified"></a>Argumenty VbStrConv. roztažitelné a VbStrConv. Narrow nelze použít pro zadané národní prostředí.
Aplikace se pokouší použít `VbStrConv` členů výčtu `Wide` nebo `Narrow`, které nelze použít pro zadané národní prostředí.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte buď `VbStrConv.Wide`, nebo `VbStrConv.Narrow`.
