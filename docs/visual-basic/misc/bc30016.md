# <a name="labels-are-not-valid-outside-methodsmultiline-lambdas"></a>Návěstí nejsou platné mimo metody nebo víceřádkových výrazů lambda
Popisek k příkazu můžete přidat pouze v rámci `Sub`, `Function`, vlastnost `Get`, nebo vlastnost `Set` postup. Pouze byl spustitelný příkaz může mít popisek a musí být všechny spustitelné příkazy uvnitř procedury.  
  
 **ID chyby:** BC30016  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odebrání popisku od příkazu nebo přesunout výroku uvnitř procedury.
