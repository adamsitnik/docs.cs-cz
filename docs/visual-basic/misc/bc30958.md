# <a name="object-created-by-edit-and-continue-is-not-yet-available-for-evaluation"></a>Objekt vytvořený procedurou Edit a Continue ještě není k dispozici pro vyhodnocení
V ladicím programu sady Visual Studio se vytváří nový objekt, ale nemůže být ještě dostupné.  
  
 Možné příčiny pro objekt do nedostupný: stav selhání a objekt přidělení paměti synchronizovány. Toto jsou dočasné interní podmínky a obvykle není výsledkem vstup do ladicího programu.  
  
 **ID chyby:** BC30958  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ukončit a znovu spusťte ladění.
