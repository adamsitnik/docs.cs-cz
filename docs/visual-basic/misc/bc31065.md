# <a name="set-parameter-cannot-be-declared-ltspecifiergt"></a>Parametr "Nastavení" nelze použít deklaraci '&lt;specifikátor&gt;.
Byl zadán neplatný modifikátor s parametrem `Set` procedura property.  
  
 **ID chyby:** BC31065  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte neplatné modifikátory, jako například `ByRef`, `ParamArray`, a `Optional`.
