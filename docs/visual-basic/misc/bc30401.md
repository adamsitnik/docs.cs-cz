# <a name="interfacename1-cannot-implement-methodname-because-there-is-no-matching-method-on-interface-interfacename2"></a>'\<interfacename1 >' nemůže implementovat '\<methodname >' protože neobsahuje odpovídající metodu \<– metoda > rozhraní '\<interfacename2 >'
Pokusili jste se implementovat metodu na rozhraní, když není k dispozici žádná metoda na rozhraní, ze kterého bylo odvozeno první rozhraní.  
  
 **ID chyby:** BC30401  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že jste nezadali název metody, nebo zadejte chybějící metodu.
