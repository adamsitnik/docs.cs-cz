# <a name="named-argument-cannot-match-a-paramarray-parameter"></a>Pojmenovaný argument nemůže odpovídat parametru ParamArray
Jste zadali pojmenovaný argument (zadaným argument deklarovaný název následovaný dvojtečkou a shodné znaménko, za nímž následuje hodnota argumentu); pole parametrů. nelze však předat podle názvu. Při volání postupu zadáte nekonečný počet oddělovači argumentů pole parametrů a kompilátor nelze přidružit k více než jeden argument jeden název.  
  
 **ID chyby:** BC30587  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Předání argumentu podle polohy, nikoli podle názvu.
