# <a name="properties-in-a-module-cannot-be-declared-ltspecifiergt"></a>Vlastnosti v modulu nelze použít deklaraci '&lt;specifikátor&gt;.
Pokusili jste se použít specifikátor, například `MustOverride`, `NotOverridable`, nebo `Overridable`, u vlastnosti v rámci modulu.  
  
 **ID chyby:** BC30503  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
