# <a name="conversions-from-typename1-to-typename2-must-be-explicit"></a>Převody z "\<NázevTypu1 >' k '\<NázevTypu2 >' musí být explicitní
Tento převod nelze provést implicitně.  
  
 **ID chyby:** BC30664  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `CType`, nebo jeden konkrétní převodu funkcí, jako například `CInt` nebo `CDec`.
