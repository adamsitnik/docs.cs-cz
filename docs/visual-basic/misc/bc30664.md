# <a name="conversions-from-lttypename1gt-to-lttypename2gt-must-be-explicit"></a>Převod z '&lt;NázevTypu1&gt;"do"&lt;NázevTypu2&gt;' musí být explicitní
Tento převod nelze provést implicitně.  
  
 **ID chyby:** BC30664  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `CType`, nebo jeden konkrétní převodu funkcí, jako například `CInt` nebo `CDec`.
