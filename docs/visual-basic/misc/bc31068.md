# <a name="delegate-in-an-interface-cannot-be-declared-specifier"></a>Pro delegáta v rozhraní nejde použít deklaraci '\<specifikátor > "
Byl zadán neplatný modifikátor delegáta deklarovat v rozhraní.  
  
 **ID chyby:** BC31068  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odebere modifikátory například `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, a `Overridable`.
