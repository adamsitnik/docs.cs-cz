# <a name="widening-and-narrowing-cannot-be-combined"></a>Nelze kombinovat "Rozšíření" a "zúžení.
[Operator – příkaz](../../visual-basic/language-reference/statements/operator-statement.md) Určuje [Widening](../../visual-basic/language-reference/modifiers/widening.md) a [Narrowing](../../visual-basic/language-reference/modifiers/narrowing.md).  
  
 Při definování operátor převodu, musíte ji deklarovat jako `Widening` nebo `Narrowing`. Toto jsou vzájemně se vylučující charakteristiky, proto nelze zadat současně.  
  
 **ID chyby:** BC33001  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Buď odeberte `Widening` nebo `Narrowing` – klíčové slovo z `Operator` příkazu. Je nutné zadat jeden z nich.
