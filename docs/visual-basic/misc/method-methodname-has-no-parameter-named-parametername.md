# <a name="method-ltmethodnamegt-has-no-parameter-named-ltparameternamegt"></a>Metoda "&lt;methodname&gt;"nemá žádný parametr s názvem"&lt;parametername&gt;.
Zadaný argument s názvem neodpovídá žádnému z argumentů cílové metody.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že byl správně zadán pojmenovaný argument.
