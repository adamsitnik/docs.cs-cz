# <a name="type-parameters-cannot-be-specified-on-this-declaration"></a>Parametry typu nelze zadat u této deklarace
Programovací element je deklarován s seznam parametrů typu, ale programovací element není můžou být obecného typu.  
  
 Programovací elementy, které nemají nárok na účast je obecný, zahrnují vlastnosti, operátory, události a konstruktory. Deklarace některé z těchto prvků s typem parametru seznamu výsledků v této chyby.  
  
 **ID chyby:** BC32065  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Of` – klíčové slovo a parametry typu v deklaraci.
