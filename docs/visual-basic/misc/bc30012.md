# <a name="if-block-must-end-with-a-matching-end-if"></a>Blok '#If' musí končit odpovídající '#End If'
`#If` je direktivy podmíněné kompilace. `#If` Bloku není ukončena `#End If` směrnice.  
  
 **ID chyby:** BC30012  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `#End If` na konci bloku podmíněné kompilace.
