# <a name="withevents-variables-cannot-be-typed-as-arrays"></a>Proměnné 'WithEvents' nemůže být typu pole
Pokusili jste se deklarovat pole pomocí `WithEvents`. Lze deklarovat libovolný počet jednotlivých proměnné jako je například `WithEvents`, ale nemůže deklarovat pole tímto způsobem.  
  
 **ID chyby:** BC30476  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Deklarujte proměnné jednotlivě.
