# <a name="security-attribute-ltattributenamegt-cannot-be-applied-to-a-module"></a>Atribut zabezpečení '&lt;attributename&gt;' nelze použít v modulu
Modul common language runtime nepřijímá atribut zabezpečení v modulu.

**ID chyby:** BC30131

## <a name="to-correct-this-error"></a>Oprava této chyby

- Odeberte atribut z deklarace.
