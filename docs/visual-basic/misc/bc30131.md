# <a name="security-attribute-attributename-cannot-be-applied-to-a-module"></a>Atribut zabezpečení attributeName\<> nejde použít pro modul.
Modul CLR (Common Language Runtime) nepřijímá atribut zabezpečení v modulu.

**ID chyby:** BC30131

## <a name="to-correct-this-error"></a>Oprava této chyby

- Odeberte atribut z deklarace.
