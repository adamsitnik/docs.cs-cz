# <a name="get-is-already-declared"></a>'Get' je už deklarovaný.
A `Get` příkazu uveden více než jednou v bloku kódu. `Get` Příkaz deklaruje `Get` vlastnost postup použitý k načtení hodnoty vlastnosti.  
  
 **ID chyby:** BC30443  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat redundantní `Get` příkazu.
