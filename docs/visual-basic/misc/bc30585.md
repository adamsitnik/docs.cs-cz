# <a name="event-eventname-cannot-be-handled-because-it-is-not-accessible-from-name"></a>Událost '\<eventname >' nelze zpracovat, protože není přístupná z '\<název >'
Pokusili jste se zpracovat událost, která není přístupná. Například pokud `Handles` je proměnná sdílená, musí se také sdílet metoda, která zpracovává události.  
  
 **ID chyby:** BC30585  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že je událost dostupná.
