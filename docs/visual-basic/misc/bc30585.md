# <a name="event-eventname-cannot-be-handled-because-it-is-not-accessible-from-name"></a>Událost '\<eventname >' nelze zpracovat, protože není přístupná z '\<název >'
Jste se pokusili získat zpracovat událost, která není přístupná. Například pokud `Handles` sdílené proměnné, musí se sdílet také metody zpracování událostí.  
  
 **ID chyby:** BC30585  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že událost je přístupný.
