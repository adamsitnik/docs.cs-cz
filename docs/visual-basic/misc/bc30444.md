# <a name="set-is-already-declared"></a>"Nastavení" je již deklarována.
A `Set` příkazu uveden více než jednou v bloku kódu. `Set` Příkaz deklaruje `Set` procedury vlastnosti použít pro přiřazení hodnoty k vlastnosti.  
  
 **ID chyby:** BC30444  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat redundantní `Set` příkazu.
