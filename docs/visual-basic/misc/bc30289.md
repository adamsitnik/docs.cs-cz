# <a name="statement-cannot-appear-within-a-method-body"></a>Příkaz nemůže být použit v těle metody
Postup obsahuje neplatný příkaz například jinou proceduru deklarace. Deklarace procedury nelze vnořit.  
  
 **ID chyby:** BC30289  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat deklaraci procedury nebo přesunout mimo ohraničující postup.
