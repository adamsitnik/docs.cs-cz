# <a name="no-accessible-overloaded-ltmethodnamegt-can-be-called-with-these-arguments-ltlistgt"></a>Žádná dostupná přetížení "&lt;methodname&gt;' lze volat s těmito argumenty: &lt;seznamu&gt;
Je volána přetížená metoda, ale žádná metoda by se shodovala se seznamem zadaných argumentů.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odpovídá seznamu argumentů pro přetížení, které máte v úmyslu volání.  
  
-   Upravte seznam argumentů tak, aby signatura odpovídá přetížení.
