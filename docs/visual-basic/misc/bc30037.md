# <a name="character-is-not-valid"></a>Znak není platný
Aktuálním kontextu není povoleno tento znak. Například tilda (`~`) nebo svislá čára (`|`) nemůže být součástí element deklarovaný název.  
  
 **ID chyby:** BC30037  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zaškrtněte, pokud chcete zobrazit, pokud znak navíc omylem byl vložen do programovací element.  
  
2. Zkontrolujte, že programovací element je zadán nebo vytvářeny pomocí jenom znaky, které jsou platné pro tento typ elementu.
