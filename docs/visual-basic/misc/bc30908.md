 <span data-ttu-id="8265a-106">Související omezení je, že typ nemůže dědit od typu, který ještě není definovaný.</span><span class="sxs-lookup"><span data-stu-id="8265a-106">A related restriction is that a type cannot inherit from a type that is not yet defined.</span></span> <span data-ttu-id="8265a-107">Dědičnost zahrnuje možnost opakovaného použití členů základní třídy, která zase vyžaduje, aby tyto členy byly definovány.</span><span class="sxs-lookup"><span data-stu-id="8265a-107">Inheritance involves the ability to reuse members of the base class, which in turn requires that these members be defined.</span></span> <span data-ttu-id="8265a-108">Proto Visual Basic nemůže kompilovat kód jako v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="8265a-108">Therefore, Visual Basic cannot compile code such as the following example.</span></span>

```vb  
    Public Class innerClass  
        Public Sub doSomething()

    End Class

```
