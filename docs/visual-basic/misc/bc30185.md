# <a name="enum-must-end-with-a-matching-end-enum"></a>"Výčet" musí končit odpovídajícího 'End Enum
`Enum` Vyvolá příkaz bez odpovídající `End Enum` příkazu. `End Enum` Příkaz musí použít na konec výčtu.  
  
 **ID chyby:** BC30185  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, že `Enum` členy jsou správně naformátované.  
  
2.  Přidat `End Enum` příkaz na konci výčtu.
