# <a name="unable-to-open-module-file-filename-error"></a>Nelze otevřít soubor modulu '\<název souboru >': \<chyby >
Soubor, který obsahuje modul pro tento projekt nelze otevřít.  
  
 **ID chyby:** BC31007  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, že soubor je v zadaném umístění a není uzamčený jinou aplikací.
