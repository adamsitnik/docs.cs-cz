# <a name="type-statements-are-no-longer-supported-use-structure-statements-instead"></a>Příkazy "Typ" již nejsou podporovány. Použijte místo nich příkazy Structure "
`Type` – Klíčové slovo je už nebude možné definovat uživatelem definované typy.  
  
 **ID chyby:** BC30802  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `Structure` příkaz k definování nového typu.
