# <a name="xml-comment-tag-tag-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>Značka komentáře XML "\<značka > se objevuje s identickými atributy více než jednou ve stejném bloku komentáře XML
Značka komentáře XML \<značka > objevuje s identickými atributy více než jednou ve stejném bloku komentáře XML. Komentář XML bude ignorován.  
  
 Komentář XML obsahuje více než jednu značku s identickými atributy.  
  
 **ID chyby:** BC42305  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte duplicitní značku.
