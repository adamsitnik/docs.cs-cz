# <a name="this-system-does-not-contain-support-for-the-locale-specified"></a>Tento systém neobsahuje podporu pro určené národní prostředí
Pokoušíte se použít `VbStrConv` výčtu, ale váš systém nepodporuje zadané národní prostředí.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Upgrade počítače pro podporu národní prostředí nebo spusťte aplikaci na počítači, který ji podporuje.
