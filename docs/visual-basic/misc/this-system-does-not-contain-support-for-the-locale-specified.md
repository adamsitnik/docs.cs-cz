# <a name="this-system-does-not-contain-support-for-the-locale-specified"></a>Tento systém neobsahuje podporu pro zadané národní prostředí.
Pokoušíte se použít výčet `VbStrConv`, ale systém nepodporuje zadané národní prostředí.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Upgradujte počítač pro podporu národního prostředí nebo spusťte aplikaci na počítači, který ho podporuje.
