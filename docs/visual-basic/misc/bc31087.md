# <a name="array-modifiers-cannot-be-specified-on-both-a-variable-and-its-type"></a>Modifikátory polí nelze zadat pro proměnnou a její typ
Modifikátory polí se nacházejí na proměnnou a její typ, která pole polí.  
  
 **ID chyby:** BC31087  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte modifikátor pole ze specifikátoru typu.
