# <a name="too-many-type-arguments-to-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>Příliš mnoho argumentů metody rozšíření '&lt;methodName&gt;"definované v"&lt;typeName&gt;.
Obecný rozšiřující metoda byla vyvolána s více argumentů, než je počet parametrů typu.  
  
 Při vyvolání obecné metody, je nutné zadat jeden argument typu pro jednotlivé parametry typu určené metody.  
  
 **ID chyby:** BC36591  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte argumenty typu ze seznamu argumentů typu tak, aby se jeden pro každý parametr typu určené obecnou metodu, kterou vyvoláváte.
