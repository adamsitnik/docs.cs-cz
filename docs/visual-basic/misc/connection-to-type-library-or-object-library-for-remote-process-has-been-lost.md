# <a name="connection-to-type-library-or-object-library-for-remote-process-has-been-lost"></a>Připojení ke knihovně typů nebo knihovně objektů pro vzdálené zpracování bylo ztraceno
Ztratili jste připojení k vzdálený proces objekt knihovny nebo knihovny typů.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Restartujte `Application` objektu.  
  
2.  V dialogovém okně chybové přes který jste zadali Toto téma nápovědy, klikněte na tlačítko **OK** zobrazíte **odkazy** dialogové okno. Slovem "CHYBÍ" na levé straně se zobrazí odkaz na ztracené.  
  
3.  Odstraňte odkaz na ztracené.  
  
4.  V **odkazy** dialogové okno, vyberte zaškrtávací políčko pro `Application` objektu.
