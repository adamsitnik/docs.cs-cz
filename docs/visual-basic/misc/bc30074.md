# <a name="constant-cannot-be-the-target-of-an-assignment"></a>Konstanta nemůže být cílem přiřazení.
Vyvolá se v kontextu, který přiřazuje hodnotu konstanty. Hodnoty přiřazené k jejich během provádění může mít pouze zapisovatelné proměnných, vlastností a prvky pole.  
  
 **ID chyby:** BC30074  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahraďte konstanty jeden zapisovatelný proměnná, vlastnost nebo pole elementu.
