# <a name="comma-or--expected-type-argument-list"></a>Čárka nebo ")' byl očekáván (seznam argumentů typu)
Seznam argumentů typu obsahuje neočekávaný element. Použít čárky (`,`) oddělit argumentů do tohoto seznamu, a použít pravou závorku (`)`) na konec seznamu.  
  
 **ID chyby:** BC32094  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte čárku a pokračovat v seznamu argumentů typu nebo uzavírací závorkou znak ukončení seznamu.
