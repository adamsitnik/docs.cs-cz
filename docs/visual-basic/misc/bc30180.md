# <a name="keyword-does-not-name-a-type"></a>Klíčové slovo neurčuje typu
Klíčové slovo se zobrazí, kde je požadován název typu. Kontext volá základní datový typ nebo element deklarovaný název představující typu.  
  
 **ID chyby:** BC30180  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Klíčové slovo nahraďte platný název typu.
