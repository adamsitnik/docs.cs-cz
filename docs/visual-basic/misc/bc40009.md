# <a name="possible-problem-detected-while-building-assembly-error"></a>Při sestavování sestavení se zjistily možné potíže: \<chyby >
Nástroj ALink, který je volán kompilátor jazyka Visual Basic, hlásí chybu vytváření sestavení. Jednou z možných příčin je podepsané sestavení odkazu na sestavení bez znaménka.  
  
 Tato zpráva je upozornění. Kompilátor bude pokračovat ke generování sestavení. Další informace o zobrazení nebo skrytí upozornění zpracování upozornění jako chyby, najdete v tématu [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID chyby:** BC40009  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte v uvozovkách chybovou zprávu a proveďte příslušné akce.  
  
2.  Kompilovat program znovu, abyste viděli, pokud se chyba objeví znovu.  
  
3.  Pokud se chyba objeví znovu, přeinstalujte kompilátor jazyka Visual Basic.  
  
4.  Pokud chyba přetrvává po přeinstalaci, shromážděte informace o okolnostech a upozornit Microsoft Product Support Services.
