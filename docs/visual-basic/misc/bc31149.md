# <a name="duplicate-xml-attribute-attributename"></a>Duplicitní atribut XML 'attributeName.
Atribut zadaný pro element XML literál odpovídá atributu, která dříve byla definována pro literál XML elementu. Názvy atributů v literálech XML elementu musí být jedinečný.  
  
 **ID chyby:** BC31149  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přejmenujte atribut pro název, který je jedinečný pro literál XML elementu.
