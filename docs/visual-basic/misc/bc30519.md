# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-can-be-called-without-a-narrowing-conversion-lterrorgt"></a>Rozlišení přetěžování se nezdařilo, protože žádnou dostupnou "&lt;metoda&gt;' lze volat bez omezujícího převodu: &lt;chyba&gt;
Provedení volání přetížené metody, ale kompilátor nemůže najít metodu, kterou lze volat bez omezujícího převodu. Zužující převod změní hodnotu na datový typ, který nemusí být schopný uchovat některou z možných hodnot přesně.  
  
 **ID chyby:** BC30519  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte `Option Strict Off`.
