# <a name="resource-name-ltresourcenamegt-cannot-be-used-more-than-once"></a>Název prostředku "&lt;resourcename&gt;' nelze použít více než jednou.
Název zadaný prostředek v propojenými dynamickými knihovna nebo spustitelný soubor byl použit více než jednou.  
  
 **ID chyby:** BC31502  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Jedinečné jméno prostředky.
