# <a name="mathfunction1-is-not-declared"></a>"\<mathfunction1 >' není deklarována
"\<mathfunction1 >' není deklarována. Funkce byla přesunuta do třídy System.Math a se teď nazývá "\<mathfunction2 >'.  
  
 Několik funkcí, které byly vnitřní do jazyka Visual Basic v předchozích verzích se přesunuly do <xref:System.Math?displayProperty=nameWithType> oboru názvů. Díky jejich funkce obecně k dispozici pro všechny programovací jazyky.  
  
 **ID chyby:** BC30819  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použijte metody deklarované v <xref:System.Math?displayProperty=nameWithType>.
