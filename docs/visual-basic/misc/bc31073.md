# <a name="ltname1gt-conflicts-with-a-lttypegt-by-the-same-name-defined-in-ltname2gt"></a>"&lt;name1&gt;' je v konfliktu s &lt;typ&gt; se stejným názvem, které jsou definované v"&lt;name2&gt;"
Název došlo, který je v konfliktu s typem se stejným názvem.  
  
 **ID chyby:** BC31073  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Deklarujte jedinečný název.  
  
2.  Použijte hranaté závorky kolem konfliktní název člena.
