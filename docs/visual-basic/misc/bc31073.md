# <a name="name1-conflicts-with-a-type-by-the-same-name-defined-in-name2"></a>"\<name1 >' je v konfliktu s \<typ > se stejným názvem, které jsou definované v"\<name2 > "
Název došlo, který je v konfliktu s typem se stejným názvem.  
  
 **ID chyby:** BC31073  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Deklarujte jedinečný název.  
  
2.  Použijte hranaté závorky kolem konfliktní název člena.
