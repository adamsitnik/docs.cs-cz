# <a name="new-cannot-be-used-on-class-classname-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>'Nové' jej nelze použít na třídu\<classname >' protože obsahuje člen 'MustOverride', který nebyl přepsán
Pokusili jste se použít `New` na třídu, která obsahuje `MustOverride` člena, který nebyl přepsán.  
  
 **ID chyby:** BC30376  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `New` příkazu.
