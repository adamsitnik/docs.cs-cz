# <a name="project-already-has-a-reference-to-assembly-ltassemblyidentitygt"></a>Projekt již obsahuje odkaz na sestavení &lt;assemblyidentity&gt;
Projekt již obsahuje odkaz na sestavení \<assemblyidentity >. Druhý odkaz na '\<cesta_k_souboru >' nelze přidat.  
  
 Projekt je více než jeden odkaz na stejné sestavení.  
  
 Identita sestavení obsahuje název sestavení, verze, veřejného klíče, pokud existuje a jazykové verze.  
  
 Jednou z možných příčin této chyby je odkazem na jinou kopii sestavení prostřednictvím jiné cestě než původní odkaz.  
  
 **ID chyby:** BC32208  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte odkaz na druhý. Není nutné, protože odkazuje na stejné sestavení.
