# <a name="events-in-interfaces-cannot-be-declared-implements"></a>Události v rozhraní nejde použít deklaraci '\<implementuje > "
Události deklarované v rozhraní nelze implementovat, události rozhraní.  
  
 **ID chyby:** BC30688  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Implements` příkazu.  
  
2.  Implementace události v rámci třídy nebo struktury.
