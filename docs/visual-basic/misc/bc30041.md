# <a name="maximum-number-of-errors-has-been-exceeded"></a>Byl překročen maximální počet chyb
Maximální počet chyb kompilace je 100. Kompilátor ukládá tento limit, aby se zmenšila velikost zprávy o chybách.  
  
 **ID chyby:** BC30041  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přečtěte si další chyby generované tuto kompilaci a vyřešit co nejvíce. Pokud zdrojový kód obsahuje více než 100 chyb, může se zobrazit další chyby při další kompilaci.
