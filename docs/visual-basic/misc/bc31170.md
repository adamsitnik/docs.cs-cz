# <a name="character-character-0xlthexadecimalgt-is-not-allowed-in-an-xml-name"></a>Znak "znak" (0 x&lt;šestnáctkové&gt;) v názvu XML není povoleno
Nepodporovaný znak je obsažen v názvu XML literál nebo vlastnosti osy XML.  
  
 **ID chyby:** BC31170  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte z XML literál nebo vlastnosti osy XML nepodporovaný znak.
