# <a name="no-method-ltmethodnamegt-can-accept-an-argument-of-type-lttypenamegt-for-parameter-ltparameternamegt"></a>Žádná metoda "&lt;methodname&gt;"může přijmout argument typu"&lt;typename&gt;'parametru'&lt;parametername&gt;.
Volání metody obsahuje argument typu, který není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte typy argumentů.  
  
2.  Shodovat s typy argumentů na typy parametrů metody, abyste měli jistotu, že jsou ve smlouvě. Podle potřeby změnit typy.
