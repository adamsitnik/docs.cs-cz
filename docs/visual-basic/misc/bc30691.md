# <a name="lttypenamegt-is-a-type-in-ltcontainernamegt-and-cannot-be-used-as-an-expression"></a>"&lt;typename&gt;'je typ v'&lt;containername&gt;" a nelze jej použít jako výraz
Jeden z názvů používá již byl deklarován jako název typu a nelze použít ve výrazu.  
  
 **ID chyby:** BC30691  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Název jednoznačně člena.  
  
-   Použijte hranaté závorky kolem názvu člena.
