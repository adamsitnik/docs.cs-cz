# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>Název agregační funkce nelze použít se znakem typu
Typ znak byla zahrnuty na konci názvu agregační funkci.  
  
 **ID chyby:** BC36617  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte znak typu z názvu agregační funkci.
