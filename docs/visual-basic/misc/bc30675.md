# <a name="synclock-statement-must-end-with-a-matching-end-synclock"></a>Příkaz 'SyncLock' musí končit odpovídajícího 'End SyncLock
`SyncLock` bloky začínají `SyncLock` – klíčové slovo a končit `End SyncLock` vytvořit.  
  
 **ID chyby:** BC30675  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že `SyncLock` bloku končí `End SyncLock` vytvořit.
