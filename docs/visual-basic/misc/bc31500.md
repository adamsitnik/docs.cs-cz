# <a name="shared-attribute-property-propertyfield-cannot-be-the-target-of-an-assignment"></a>Vlastnost 'Sdílené' atribut '\<propertyfield >' nemůže být cílem přiřazení.
Byl proveden pokus o přiřazení hodnoty k `ReadOnly` vlastnosti nebo `Shared` v atributu.  
  
 **ID chyby:** BC31500  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte příkaz pro přiřazení vlastnosti.  
  
2. Pokud používáte vlastnosti, které jste vyvinuli `ReadOnly` , `Shared` odeberte modifikátory nebo z vlastnosti atributu.
