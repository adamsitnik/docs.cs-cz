# <a name="shared-attribute-property-propertyfield-cannot-be-the-target-of-an-assignment"></a>Vlastnost 'Sdílené' atribut '\<propertyfield >' nemůže být cílem přiřazení.
Byl proveden pokus o přiřazení hodnoty k `ReadOnly` nebo `Shared` vlastnost v atributu.  
  
 **ID chyby:** BC31500  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte příkaz přiřazení vlastnosti.  
  
2.  Pokud pomocí vlastnosti, které jste vytvořili, odeberte `ReadOnly` nebo `Shared` modifikátory z vlastnost atributu.
