# <a name="unrecognized-option-optionname-ignored"></a>Nerozpoznaná možnost \<název_možnosti >; ignoruje
Byla zadána možnost, která není platná z příkazového řádku.  
  
 **ID chyby:** BC2007  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte zadané možnosti a vyberte ten, který není platný.
