# <a name="end-raiseevent-must-be-preceded-by-a-matching-raiseevent-declaration"></a>'End RaiseEvent' musí předcházet párová deklarace "RaiseEvent.
`End RaiseEvent` Příkazu došlo k chybě bez odpovídající `RaiseEvent` příkazu. `End RaiseEvent` musí předcházet odpovídající `RaiseEvent` příkazu.  
  
 **ID chyby:** BC31126  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, že předchozí `RaiseEvent` příkazu je platný a správně zadané.
