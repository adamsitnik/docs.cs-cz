# <a name="argument-value-ltpathnamegt-contains-characters-that-are-not-valid-in-a-path-name"></a>Hodnota argumentu "&lt;cesta&gt;' obsahuje znaky, které nejsou platné v názvu cesty
Cesta obsahuje jeden nebo více znaků, které nejsou platné.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte znak v názvu cesty.
