## <a name="to-correct-this-error"></a><span data-ttu-id="2bb17-105">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="2bb17-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="2bb17-106">Přidat `Join` – klíčové slovo chcete `Group Join` klauzule, jak je znázorněno v následujícím příkladu:</span><span class="sxs-lookup"><span data-stu-id="2bb17-106">Add the `Join` keyword to the `Group Join` clause, as shown in the following example:</span></span>
