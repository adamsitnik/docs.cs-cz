# <a name="set-method-cannot-have-more-than-one-parameter"></a>Metodu 'Set' nemůže mít více než jeden parametr
Pro vlastnost je definován více než jeden parametr `Set` postupy.  
  
 **ID chyby:** BC31063  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte všechny nadbytečné parametry z vlastnosti `Set` postupy.
