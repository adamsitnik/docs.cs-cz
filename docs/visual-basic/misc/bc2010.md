# <a name="compilation-failed-message"></a>Kompilace se nezdařila: \<zpráva >
Kompilace se nezdařila z důvodu okolností uvedených v chybové zprávě  
  
 **ID chyby:** BC2010  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Najdete v chybové zprávě, chcete-li zjistit přesnou příčinu selhání.
