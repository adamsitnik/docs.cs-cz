# <a name="code-resource-not-found"></a>Prostředek kódu nebyl nalezen
Došlo k volání do procedury v kódu prostředků, ale nepovedlo se najít prostředek kódu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, že prostředek k dispozici a správně odkazované.
