# <a name="option-ltoptiongt-cannot-be-followed-by--or---ignored"></a>možnost "&lt;možnost&gt;' nemůže následovat '+' nebo '-'. ignoruje
Byla zadána možnost kompilátoru s hodnotou `+` nebo `-`; tyto hodnoty nejsou podporovány.  
  
 Další informace o možnostech kompilátoru a hodnoty, které podporují, najdete v části [jazyka Visual Basic možnosti kompilátoru uvedené abecedně](../../visual-basic/reference/command-line-compiler/compiler-options-listed-alphabetically.md).  
  
 **ID chyby:** BC2028  
  
## <a name="to-correct-this-error-and-use-the-option"></a>Chcete-li opravit tuto chybu a použijte možnost  
  
-   Zadejte možnost kompilátoru s žádnou hodnotu.  
  
## <a name="to-correct-this-error-and-not-use-the-option"></a>Chcete-li opravit tuto chybu a ne pomocí možnosti  
  
-   Remove – možnost kompilátoru.
