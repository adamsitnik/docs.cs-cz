# <a name="the-keyword-keyword-is-used-to-overload-inherited-members-do-not-use-the-keyword-keyword-when-overloading-sub-new"></a>'\<– Klíčové slovo >' – klíčové slovo se používá k přetěžování zděděných členů, nepoužívejte'\<– klíčové slovo > 'při přetěžování Sub New – klíčové slovo
Konstruktor je deklarován s `Overloads` klíčovým slovem.  
  
 Visual Basic nepodporuje dědění konstruktorů nebo konstruktorů přetížení.  
  
 **ID chyby:** BC32040  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- `Overloads` Odeberte klíčové slovo ze všech deklarací konstruktoru.
