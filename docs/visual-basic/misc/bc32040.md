# <a name="the-ltkeywordgt-keyword-is-used-to-overload-inherited-members-do-not-use-the-ltkeywordgt-keyword-when-overloading-sub-new"></a>"&lt;– Klíčové slovo&gt;" klíčové slovo se používá k přetěžování zděděných členů, nepoužívejte "&lt;– klíčové slovo&gt;" při přetěžování Sub New – klíčové slovo
Konstruktor je deklarována s `Overloads` – klíčové slovo.  
  
 Visual Basic nepodporuje dědění nebo přetížení konstruktory.  
  
 **ID chyby:** BC32040  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Overloads` – klíčové slovo z všechny deklarace konstruktoru.
