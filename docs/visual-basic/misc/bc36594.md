# <a name="definition-of-method-ltmethodgt-is-not-accessible-in-this-context"></a>Definice metody &lt;metoda&gt; není v tomto kontextu k dispozici
Agregační funkce, která neexistuje, byla součástí `Into` operátor.  
  
 **ID chyby:** BC36594  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda název agregační funkce a provést potřebné změny.  
  
-   Přidejte metodu jako vlastní agregační funkci (metody rozšíření). Další informace o vytváření vlastní agregační funkce, najdete v části [Aggregate – klauzule](../../visual-basic/language-reference/queries/aggregate-clause.md).
