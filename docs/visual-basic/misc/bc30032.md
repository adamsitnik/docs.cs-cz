# <a name="events-cannot-have-a-return-type"></a>Události nemohou mít typ vrácené hodnoty
Události může přijímat argumenty, ale nelze přímo návratové hodnoty.  
  
 **ID chyby:** BC30032  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat typ návratové hodnoty z: `Event` příkazu.
