# <a name="labels-are-not-valid-in-the-immediate-window"></a>Návěstí nejsou platná v příkazovém
Popisků čáry povolit větvení a nejsou povoleny v rámci ladění.  
  
 **ID chyby:** BC30134  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nezadávejte popisku řádku v **okamžité** okna.
