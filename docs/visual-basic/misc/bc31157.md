# <a name="xml-attribute-attribute1-must-appear-before-xml-attribute-attribute2"></a>Atribut XML "atribut1" musí být před atributem XML "atribut2"
Neplatný popořadě jsou zadané atributy v dokumentu XML literál. Pořadí platný atribut pro literál dokumentu XML je podle specifikace XML 1.0. Například `encoding` atribut literál dokumentu XML musí bezprostředně následuje po `version` atribut.  
  
 **ID chyby:** BC31157  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Aktualizujte pořadí atributů pro literál dokumentu XML pomocí pokynů uvedených v specifikace XML 1.0.
