# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-accepts-this-number-of-arguments"></a>Rozlišení přetěžování se nezdařilo, protože žádnou dostupnou "&lt;metoda&gt;' Tento počet argumentů
Provedení volání přetížené metody, ale kompilátor nemůže najít metodu s odpovídající seznam argumentů.  
  
 **ID chyby:** BC30516  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte seznam argumentů, které jste zadali.
