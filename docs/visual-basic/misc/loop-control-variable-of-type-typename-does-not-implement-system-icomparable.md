# <a name="loop-control-variable-of-type-typename-does-not-implement-the-systemicomparable-interface"></a>Řídicí proměnná smyčky typu '\<typename >' neimplementuje rozhraní 'System.IComparable.
Proměnná je typu, který neimplementuje `System.IComparable` rozhraní. `System.IComparable` musí být implementované typem hodnoty nebo třídy.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že je proměnná typu hodnoty.
