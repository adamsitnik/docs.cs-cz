# <a name="loop-control-variable-of-type-lttypenamegt-does-not-implement-the-systemicomparable-interface"></a>Řídicí proměnná smyčky typu "&lt;typename&gt;' neimplementuje rozhraní 'System.IComparable.
Proměnná je typu, který neimplementuje `System.IComparable` rozhraní. `System.IComparable` musí být implementované typem hodnoty nebo třídy.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že je proměnná typu hodnoty.
