# <a name="ltstatementnamegt-statement-requires-an-array"></a>"&lt;statementname&gt;' příkaz vyžaduje pole
`Erase` a `ReDim` příkazy vyžadují pole datové typy. Zadaný typ není pole.  
  
 **ID chyby:** BC30049  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změna datového typu na typ pole.
