# <a name="instance-of-restricted-type-typename-cannot-be-used-in-a-lambda-expression"></a>Instanci omezeného typu '\<typename >' nelze použít ve výrazu lambda
Instance s omezením pomocí specifikátoru typy, jako například <xref:System.ArgIterator>, <xref:System.RuntimeArgumentHandle>, a <xref:System.TypedReference>, nelze použít ve výrazu lambda nebo výrazu dotazu.  
  
 **ID chyby:** BC36640
