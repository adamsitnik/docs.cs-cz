# <a name="unused-local-constant-constant"></a>Nepoužitá lokální konstanta: "\<konstantních >"
Lokální konstanta je deklarovaná, který není použit kdekoli ve vašem kódu.  
  
 **ID chyby:** BC42099  
  
### <a name="to-remove-this-warning"></a>Chcete-li odebrat toto upozornění  
  
1.  Tato chyba je upozornění a můžete ignorovat. Toto upozornění odebrat, odeberte z kódu nepoužívané deklarace konstanty.
