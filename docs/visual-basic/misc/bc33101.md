# <a name="type-lttypenamegt-must-be-a-value-type-or-a-type-argument-constrained-to-structure-in-order-to-be-used-with-nullable-or-nullable-modifier-"></a>Typ '&lt;typeName&gt;' musí být typ hodnoty nebo argument typu omezen Structure, "aby bylo možné použít s modifikátorem 'S povolenou hodnotou NULL' nebo s povolenou hodnotou Null"?"
Pouze typy hodnot, včetně struktur, mohou být deklarovány s možnou hodnotou Null.  
  
```vb  
' Valid.  
Dim n? As Integer  
Dim m As Integer?  
  
' Not valid.  
' Dim p? As Object  
' Dim q As Nullable(Of Object)  
```  
  
 **ID chyby:** BC33101  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte "?" nebo `Nullable`.  
  
-   Použijte datový typ hodnoty.
