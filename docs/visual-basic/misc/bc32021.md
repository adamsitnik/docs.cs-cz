# <a name="parameter-ltparameternamegt-in-ltmethodnamegt-already-has-a-matching-omitted-argument"></a>Parametr '&lt;parametername&gt;"in"&lt;methodname&gt;' již existuje odpovídající vynechaný argument
Postup volání je zadán argument podle názvu po vynechání stejné argumentů podle pozice; Příklad:  
  
```  
Public Sub ABC(ByVal X As Byte, Optional ByVal Y As Byte = 0, _  
                                Optional ByVal Z As Byte = 0)  
' ...  
Call ABC(6, , Y:=3)   ' Argument Y omitted by position, supplied by name.  
```  
  
 **ID chyby:** BC32021  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadání argumentů podle pozice nebo odstraňte čárku, který se vynechá.
