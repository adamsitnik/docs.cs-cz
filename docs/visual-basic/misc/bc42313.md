# <a name="xml-comment-tag-returns-is-not-permitted-on-a-writeonly-property"></a>Značka komentáře XML "vrátí" není povolená na vlastnost "jen pro zápis.
Značka komentáře XML "vrátí" není povolená na vlastnost 'Jen pro zápis'. Komentář XML bude ignorován.  
  
 Komentář XML pro deklaraci vlastnosti jen pro zápis nemůže obsahovat \<vrátí > značky.  
  
 **ID chyby:** BC42313  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat nepodporovanou \<vrátí > značky.
