# <a name="typename-is-a-namespace-and-cannot-be-used-as-an-expression"></a>"\<typename >' je obor názvů a nelze použít jako výraz
Název typu vyvolá, ve kterém jsou vyžadována výrazu. Výraz se musí skládat z proměnné, konstanty, literály, vlastnosti, a `Function` volání procedur.  
  
 **ID chyby:** BC30112  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte název typu a vytvořit výraz pomocí platné prvky.
