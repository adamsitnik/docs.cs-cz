# <a name="cast-from-type-lttypename1gt-to-type-lttypename2gt-is-not-valid"></a>Přetypování z typu '&lt;NázevTypu1&gt;' na typ'&lt;NázevTypu2&gt;' je neplatný
Pokus o převod proměnné z jednoho typu na druhý není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte zapojených do převodu typů a ujistěte se, že se pokoušíte převést na typ smysluplné.
