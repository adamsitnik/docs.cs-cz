# <a name="cast-from-type-typename1-to-type-typename2-is-not-valid"></a>Přetypování z typu '\<NázevTypu1 >' na typ '\<NázevTypu2 >' není platný
Pokus o převod proměnné z jednoho typu na druhý není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte zapojených do převodu typů a ujistěte se, že se pokoušíte převést na typ smysluplné.
