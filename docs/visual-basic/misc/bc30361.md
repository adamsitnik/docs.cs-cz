# <a name="ltproperty1gt-and-ltproperty2gt-cannot-overload-each-other-because-only-one-is-declared-default"></a>"&lt;vlastnost1&gt;"a"&lt;vlastnost2&gt;se nemohou vzájemně přetěžovat, protože pouze jeden je deklarována jako"Výchozí"
Pokud vlastnost určuje `Default`, musíte zadat také přetížit ve nejmenuje všechny vlastnosti `Default`.  
  
 **ID chyby:** BC30361  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že všechny přetížené vlastnosti jsou deklarovány `Default`.
