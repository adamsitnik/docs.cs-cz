# <a name="property1-and-property2-cannot-overload-each-other-because-only-one-is-declared-default"></a>'\<vlastnost1 >' a '\<vlastnost2 > se nemohou vzájemně přetěžovat, protože pouze jeden je deklarována jako'Výchozí'
Pokud vlastnost určuje `Default`, musíte zadat také přetížit ve nejmenuje všechny vlastnosti `Default`.  
  
 **ID chyby:** BC30361  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že všechny přetížené vlastnosti jsou deklarovány `Default`.
