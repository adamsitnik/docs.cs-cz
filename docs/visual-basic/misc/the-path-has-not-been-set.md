# <a name="the-path-has-not-been-set"></a>Cesta nebyla nastavena.
Operace se soubory, jako je například přesunutí nebo kopírování souboru, byla provedena bez poskytnutí nezbytné cestu. To může být následek nesprávně analýze cesty.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda cesta k souboru byl uveden a je správná.
