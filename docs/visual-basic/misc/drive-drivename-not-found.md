# <a name="drive-drivename-not-found"></a>Jednotka '\<drivename >' nebyl nalezen
Na jednotce se nenašel. Bylo ztraceno připojení k síti, nebo jednotku, kterou se pokoušíte získat přístup v režimu offline nebo neexistuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte energie k jednotce.  
  
2.  Ujistěte se, že jsou správně připojeny žádné kabely připojení počítače k síti.  
  
3.  Pro přístup k jednotce přes síť, zkontrolujte, zda existuje logické propojení mezi vaším počítačem a jednotku.  
  
4.  Připojení k síti a zkuste to znovu.
