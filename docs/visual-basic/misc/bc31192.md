# <a name="error-embedding-win32-manifest-manifest-file-file-cannot-be-found"></a>Chyba při vkládání manifestu Win32: Soubor manifestu \<soubor > nebyl nalezen
Zadaný pro soubor `/win32manifest` – možnost kompilátoru nebyl nalezen.  
  
 **ID chyby:** BC31192  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte platný název souboru a cestu pro `/win32manifest` – možnost kompilátoru.
