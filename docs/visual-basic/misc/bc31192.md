# <a name="error-embedding-win32-manifest-manifest-file-file-cannot-be-found"></a>Chyba při vkládání manifestu Win32: soubor manifestu \<> soubor nejde najít.
Nelze nalézt soubor zadaný pro možnost kompilátoru `/win32manifest`.  
  
 **ID chyby:** BC31192  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zadejte platný název souboru a cestu pro možnost kompilátoru `/win32manifest`.
