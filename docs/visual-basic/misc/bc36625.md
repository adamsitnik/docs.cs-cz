# <a name="lambda-expression-cannot-be-converted-to-lttypenamegt-because-lttypenamegt-is-not-a-delegate-type"></a><span data-ttu-id="a665f-102">Výraz lambda nejde převést na "&lt;typename&gt;'protože&lt;typename&gt;' není typ delegáta</span><span class="sxs-lookup"><span data-stu-id="a665f-102">Lambda expression cannot be converted to '&lt;typename&gt;' because '&lt;typename&gt;' is not a delegate type</span></span>
<span data-ttu-id="a665f-103">Výrazy lambda se dá, kde Delegáti jsou platné.</span><span class="sxs-lookup"><span data-stu-id="a665f-103">Lambda expressions can be used where delegates are valid.</span></span> <span data-ttu-id="a665f-104">Je lze převést na typy delegát kompatibilní, ale ne k žádným jiným typům.</span><span class="sxs-lookup"><span data-stu-id="a665f-104">They can be converted to compatible delegate types, but not to any other type.</span></span> <span data-ttu-id="a665f-105">Například můžete definovat typ delegáta a přiřaďte výraz lambda nebo odeslat výraz lambda jako argument <xref:System.Func%601> parametru.</span><span class="sxs-lookup"><span data-stu-id="a665f-105">For example, you can define a delegate type and assign a lambda expression to it, or send a lambda expression as the argument to a <xref:System.Func%601> parameter.</span></span> <span data-ttu-id="a665f-106">Tyto příklady jsou uvedeny v následujícím kódu.</span><span class="sxs-lookup"><span data-stu-id="a665f-106">These examples are shown in the following code.</span></span>  
  
```vb  
Module Module1  
  
    Delegate Function FunDel(ByVal m As Integer) As Boolean  
  
    Sub Main()  
  
        ' Assign a lambda expression to a function delegate.  
        Dim negative As FunDel = Function(n As Integer) n < 0  
        Console.WriteLine(negative(-3))  
  
        ' Send a lambda as the argument to a delegate parameter.  
        Dim numbers() As Integer = {3, 4, 2, 8, 1, 0, 9, 13, 42}  
        Dim evens = numbers.Where(Function(n) n Mod 2 = 0)  
  
        For Each even In evens  
            Console.WriteLine(even)  
        Next  
  
    End Sub  
  
End Module  
```
