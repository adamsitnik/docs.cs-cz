# <a name="lambda-expression-cannot-be-converted-to-lttypenamegt-because-lttypenamegt-is-not-a-delegate-type"></a>Výraz lambda nejde převést na "&lt;typename&gt;'protože&lt;typename&gt;' není typ delegáta
Výrazy lambda se dá, kde Delegáti jsou platné. Je lze převést na typy delegát kompatibilní, ale ne k žádným jiným typům. Například můžete definovat typ delegáta a přiřaďte výraz lambda nebo odeslat výraz lambda jako argument <xref:System.Func%601> parametru. Tyto příklady jsou uvedeny v následujícím kódu.  
  
```vb  
Module Module1  
  
    Delegate Function FunDel(ByVal m As Integer) As Boolean  
  
    Sub Main()  
  
        ' Assign a lambda expression to a function delegate.  
        Dim negative As FunDel = Function(n As Integer) n < 0  
        Console.WriteLine(negative(-3))  
  
        ' Send a lambda as the argument to a delegate parameter.  
        Dim numbers() As Integer = {3, 4, 2, 8, 1, 0, 9, 13, 42}  
        Dim evens = numbers.Where(Function(n) n Mod 2 = 0)  
  
        For Each even In evens  
            Console.WriteLine(even)  
        Next  
  
    End Sub  
  
End Module  
```  
  
 **ID chyby:** BC36625
