# <a name="sub-new-cannot-be-declared-ltmodifiergt"></a>Nelze použít deklaraci 'Sub New' '&lt;modifikátor&gt;.
Pokusili jste se deklarovat `Sub New` modifikátorem, jako například `Overrides`, která je není platný.  
  
 **ID chyby:** BC30364  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte modifikátor.
