# <a name="interface-ltinterfacenamegt-cannot-be-indexed-because-it-has-no-default-property"></a>Rozhraní '&lt;interfacename&gt;"nelze indexovat, protože nemá žádnou výchozí vlastnost
Výrazy indexu musí mít za následek hodnota, jejíž typ je pole, hodnota, jejíž typ má sadu přetížených výchozí vlastnosti nebo sadu přetížených vlastností. Rozhraní může mít pouze jeden výchozí metody nebo vlastnosti. To znamená, že buď zdědit rozhraní obsahující výchozí metodu nebo vlastnost, nebo jeho definice bloku může obsahovat jednu metodu nebo vlastnost, která je označena jako výchozí.  
  
 **ID chyby:** BC30547  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte výchozí vlastnosti v rozhraní.
