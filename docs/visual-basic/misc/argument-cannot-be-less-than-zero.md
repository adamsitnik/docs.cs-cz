# <a name="argument-cannot-be-less-than-zero"></a>Argument nemůže být menší než nula.
Byla zadána hodnota, která je menší než nula, ale argument nemůže být menší než nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, jak se hodnota v dotazu počítá, abyste se ujistili, že je správná.
