# <a name="argument-cannot-be-less-than-zero"></a>Argument nemůže být menší než nula
Hodnota, která je menší než nula není zadaný, ale argument nemůže být menší než nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, jak se počítá dotyčný hodnotu, abyste měli jistotu, že je správný.
