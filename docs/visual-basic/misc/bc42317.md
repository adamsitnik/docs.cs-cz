# <a name="xml-comment-type-parameter-parameter-does-not-match-a-type-parameter-on-the-corresponding-keyword-statement"></a>Parametr typu komentáře XML \<parametr > neodpovídá parametru typu u odpovídajícího \<– klíčové slovo > – příkaz
Parametr komentáře XML \<parametr > neodpovídá parametru typu u odpovídajícího \<– klíčové slovo > příkaz. Komentář XML bude ignorován.  
  
 Značka komentáře XML byl zadán pro některé, ale ne všechny parametry typu v metodě.  
  
 **ID chyby:** BC42317  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte značky pro komentáře XML pro všechny parametry typu v metodě.
