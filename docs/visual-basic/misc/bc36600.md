# <a name="range-variable-ltvariablegt-is-already-declared"></a>Proměnná rozsahu &lt;proměnnou&gt; již byla deklarována
Název proměnné rozsahu podle `Select` klauzuli, nebo `Into` součástí `Aggregate`, `Group By`, nebo `Group Join` klauzule duplikuje název proměnné rozsahu, který je už zadaná v klauzuli dotazu.  
  
 **ID chyby:** BC36600  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že všechny proměnné rozsahu v dotazu klauzuli mít jedinečné názvy.
