<span data-ttu-id="bb925-103">Výraz přistupuje k programovací element definovaný v třída, struktura, modul nebo rozhraní, které obsahuje více než jednoho člena se stejným názvem.</span><span class="sxs-lookup"><span data-stu-id="bb925-103">An expression accesses a programming element defined in a class, structure, module, or interface that contains more than one member with the same name.</span></span>
