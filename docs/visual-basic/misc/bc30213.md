# <a name="comma-or--expected-procedure-parameter-list"></a>Čárka nebo ")" očekáván (postup seznamu parametrů)
Seznam parametrů obsahuje neočekávaný element. Použít čárky (`,`) pro oddělení parametrů do tohoto seznamu a použít pravou závorku (`)`) na konec seznamu.  
  
 **ID chyby:** BC30213  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte čárku a pokračovat v seznamu parametrů typu nebo uzavírací závorkou znak ukončení seznamu.
