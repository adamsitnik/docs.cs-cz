# <a name="end-enum-must-be-preceded-by-a-matching-enum"></a>'End Enum' musí předcházet odpovídající "výčet"
`End Enum` Vyvolá příkaz bez odpovídající `Enum` příkazu. `End Enum` musí předcházet odpovídající `Enum` příkazu.  
  
 **ID chyby:** BC30184  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že předchozí `Enum` příkaz není překlepy nebo jinak neplatných.  
  
2.  Ujistěte se, že `Enum` členy jsou správně naformátované.
