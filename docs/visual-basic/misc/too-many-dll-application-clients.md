# <a name="too-many-dll-application-clients"></a>Příliš mnoho aplikačních klientů knihovny DLL
Dynamická knihovna (DLL) pro jazyk Visual Basic mohou pojmout přístup omezený počet hostování aplikací. Vaše aplikace a další aplikace, které jsou hostiteli jazyka Visual Basic (z nichž některé můžou být dostupné aplikace) jsou všechny pokusu o přístup ke knihovně DLL jazyka Visual Basic ve stejnou dobu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Snižte počet spuštěných aplikací, přístupu k jazyka Visual Basic.
