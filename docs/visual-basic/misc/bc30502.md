# <a name="shared-cannot-be-combined-with-specifier-on-a-property-declaration"></a>'Sdílené' nelze kombinovat s '\<specifikátor >' v deklaraci vlastnosti
Pokusili jste se zkombinovat `Shared` modifikátor specifikátorem, který není platný v kombinaci, v deklaraci vlastnosti.  
  
 **ID chyby:** BC30502  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
