# <a name="property-parameters-cannot-have-the-name-value"></a>Parametry vlastností nemohou mít název 'Value'
Parametr v `Property` příkaz jmenuje `Value`. Tento název je rezervovaný v rámci definice vlastnosti pro nová hodnota je přiřazeno k vlastnosti v `Set` postup.  
  
 **ID chyby:** BC32201  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte název parametru v `Property` příkazu.
