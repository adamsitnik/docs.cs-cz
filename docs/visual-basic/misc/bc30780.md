# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>'Příkaz continue' příkazy nejsou platné v příkazovém
'Příkaz continue' příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30780  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Příkaz 'Continue' odebrání ladění kódu.
