# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>Příkazy Continue nejsou platné v příkazovém podokně.
Příkazy Continue lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30780  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte příkaz Continue z ladicího kódu.
