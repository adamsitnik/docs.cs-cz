# <a name="managed-classes-derived-from-a-com-class-cannot-be-called-late-bound"></a>Spravované třídy odvozené od třídy COM nelze volat s pozdní vazbou.
Pokusili jste se provést volání s pozdní vazbou pro spravované třídy odvozené od třídy COM; taková volání nejsou podporovány.  
  
### <a name="to-correct-the-problem"></a>Chcete-li opravit problém  
  
-   Zkontrolujte volání časná vazba.
