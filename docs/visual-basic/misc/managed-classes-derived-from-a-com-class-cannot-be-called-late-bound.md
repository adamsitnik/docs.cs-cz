# <a name="managed-classes-derived-from-a-com-class-cannot-be-called-late-bound"></a>Spravované třídy odvozené od třídy COM nelze volat s pozdní vazbou.

Pokusili jste se provést volání s pozdní vazbou na spravovanou třídu odvozenou od třídy COM; taková volání nejsou podporovaná.

## <a name="to-correct-the-problem"></a>Postup odstranění problému

Zajistěte, aby hovor byl včasnou vazbou.
