# <a name="file-already-exists"></a>Soubor již existuje.
K této chybě dojde v důsledku pojmenování nový soubor s názvem existujícího souboru nebo pomocí **uložit jako** příkazu k uložení právě načtený žádný spustitelný projekt s názvem, který již existuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte název a buď ji změnit, který není v konfliktu, nebo odstranit existující soubor nebo projekt, pokud už je nepotřebujete.
