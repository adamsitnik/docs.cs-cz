# <a name="could-not-find-library-libraryname"></a>Nepovedlo se najít knihovnu '\<NázevKnihovny > "
Zadaná knihovna se nenašla.  
  
 Tomuto problému může dojít při kompilaci z příkazového řádku a pomocí odpovědí (.rsp) souboru, který odkazuje na mnoho sestavení  
  
 **ID chyby:** BC2017  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte název knihovny, abyste měli jistotu, že je správný.
