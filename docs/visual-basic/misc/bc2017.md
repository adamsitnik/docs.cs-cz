# <a name="could-not-find-library-libraryname"></a>Nepovedlo se najít knihovnu\<>.
Zadaná knihovna nebyla nalezena.  
  
 K tomuto problému může dojít při kompilování z příkazového řádku a použití souboru Response (. rsp), který odkazuje na mnoho sestavení.  
  
 **ID chyby:** BC2017  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte název knihovny, abyste se ujistili, že je správný.
