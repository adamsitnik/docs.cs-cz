# <a name="member-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>"\<člena >' je v konfliktu s vyhrazeným členem podle tohoto názvu, který je implicitně deklarován ve všech výčtech
Název Typ člena je v konfliktu s názvu členem implicitně deklarovaným ve všech výčtech.  
  
 Kompilátor jazyka Visual Basic vytvoří odpovídající některé programovací prvky, které deklarujete implicitní členy. Výčty implicitně deklarovat člen `value__member`.  
  
 **ID chyby:** BC31420  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Upravte název člena.
