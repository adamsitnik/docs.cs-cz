# <a name="ltmethod1gt-and-ltmethod2gt-cannot-overload-each-other-because-they-differ-only-by-parameters-declared-byref-or-byval"></a>"&lt;– metoda1&gt;"a"&lt;method2&gt;se nemohou vzájemně přetěžovat, protože se liší pouze parametry deklarovanými jako 'ByRef' nebo 'ByVal.
Pokusili jste se přetížení metody s jinou metodu, která se liší od prvního pouze parametr deklarovaný jako `ByRef` nebo `ByVal`.  
  
 **ID chyby:** BC30345  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že metody jsou rozlišené pomocí více než název `ByRef` nebo `ByVal` parametru.
