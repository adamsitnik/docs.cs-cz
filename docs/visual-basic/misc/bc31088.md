# <a name="notoverridable-cannot-be-specified-on-methods-that-do-not-override-another-method"></a>'NotOverridable' nelze zadat pro metody, které nepřepisují jinou metodu
Vlastnosti a metody jsou `NotOverridable` ve výchozím nastavení. `NotOverridable` Modifikátor jde použít jenom pro metody, které přepisují jiné vlastnosti nebo metody.  
  
 **ID chyby:** BC31088  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `NotOverridable` modifikátor z vlastností a metod, které nepřepisují jinou člena.
