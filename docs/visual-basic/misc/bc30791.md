# <a name="declare-statements-in-a-structure-cannot-be-declared-specifier"></a>'Deklarace' příkazy ve struktuře nelze deklarovat jako '\<specifikátor > '
A `Declare` má specifikátor, který není platný v rámci deklarace `Structure` deklarace.  
  
 **ID chyby:** BC30791  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
