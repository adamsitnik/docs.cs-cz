# <a name="declare-statements-in-a-structure-cannot-be-declared-ltspecifiergt"></a>"Deklarace" příkazy ve struktuře nelze deklarovat jako '&lt;specifikátor&gt;.
A `Declare` má specifikátor, který není platný v rámci deklarace `Structure` deklarace.  
  
 **ID chyby:** BC30791  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
