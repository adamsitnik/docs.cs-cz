# <a name="local-variable-ltvariablenamegt-is-already-declared-in-the-current-block"></a>Lokální proměnnou '&lt;NázevProměnné&gt;"je již v aktuálním bloku deklarována
Dvě proměnné se stejným názvem, které jsou deklarovány ve stejném bloku (která může být například procedury nebo řídicí blok `If...Then`).  
  
 **ID chyby:** BC30288  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Jak přesunout některou z deklarace do různých bloku nebo změňte název jedné proměnné.
