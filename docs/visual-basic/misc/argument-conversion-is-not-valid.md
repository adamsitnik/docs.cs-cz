# <a name="argument-conversion-is-not-valid"></a>Argument 'Conversion' je neplatný
Hodnotu `VbStrConv` výčet není platný, pravděpodobně protože je menší než nula. Hodnoty výčtu musí být větší než nebo rovna hodnotě nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že hodnota je větší než nebo rovna hodnotě nula.
