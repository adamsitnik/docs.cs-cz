# <a name="end-removehandler-must-be-the-first-statement-on-a-line"></a>'End RemoveHandler' musí být prvním příkazem na řádku
`End RemoveHandler` Příkaz následuje dvojtečka (:) oddělovač. `End RemoveHandler` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC31119  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na samostatné řádky.
