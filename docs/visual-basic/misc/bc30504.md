# <a name="property-propertyname-cannot-be-declared-default-because-it-overrides-a-property-on-the-base-type-typename-that-is-not-default"></a>Vlastnost '\<propertyname >' nelze použít deklaraci 'Default', protože přepisuje vlastnost v základní třídě \<typu >'\<typename > ', který je ne výchozí
Pokusili jste se deklarovat vlastnost `Default`, což není platná deklarace protože přepisuje vlastnost, která nebyla deklarována `Default` v základní třídě.  
  
 **ID chyby:** BC30504  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Default` deklarace.
