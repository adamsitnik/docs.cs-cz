# <a name="property-ltpropertynamegt-cannot-be-declared-default-because-it-overrides-a-property-on-the-base-lttypegt-lttypenamegt-that-is-not-default"></a>Vlastnost "&lt;propertyname&gt;' nelze použít deklaraci 'Default', protože přepisuje vlastnost v základní třídě &lt;typ&gt; "&lt;typename&gt;' to znamená ne výchozí
Pokusili jste se deklarovat vlastnost `Default`, což není platná deklarace protože přepisuje vlastnost, která nebyla deklarována `Default` v základní třídě.  
  
 **ID chyby:** BC30504  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Default` deklarace.
