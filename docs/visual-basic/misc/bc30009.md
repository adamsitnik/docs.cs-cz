# <a name="reference-required-to-assembly-assemblyname-containing-the-implemented-interface-interfacename"></a>Odkaz vyžadoval sestavení '\<assemblyname >' obsahující implementované rozhraní "\<interfacename >"
Odkaz vyžadoval sestavení '\<assemblyname >' obsahující implementované rozhraní "\<interfacename >'. Přidejte jej do projektu.  
  
 Rozhraní je definováno v dynamická knihovna (DLL) nebo sestavení, které není přímo odkazován ve vašem projektu. Kompilátor jazyka Visual Basic vyžaduje přidání odkazu na-li předejít nejednoznačnosti v případě, že rozhraní je definováno ve více než jedna knihovna DLL nebo sestavení.  
  
 **ID chyby:** BC30009  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zahrnout název neodkazovaná sestavení nebo knihovny DLL odkazy projektu.
