# <a name="redim-cannot-change-the-number-of-dimensions-of-an-array"></a>'ReDim' nelze změnit počet rozměrů pole
Pokusili jste se použít `ReDim` Chcete-li změnit pořadí (počet rozměrů) pole. `ReDim` Příkaz můžete použít ke změně velikosti jednu nebo více dimenzí pole, které již byl dříve deklarován, ale nedá se změnit počet rozměrů pole.  
  
 **ID chyby:** BC30415  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že řád namísto velikosti pole a pokud je to možné, používejte `Dim` deklarovat nové pole s požadovaný počet rozměrů.
