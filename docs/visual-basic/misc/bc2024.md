# <a name="the-default-assembly-reference-ltreferencegt-has-already-been-automatically-added-ignored"></a>Výchozí odkaz na sestavení '&lt;odkaz&gt;"má již byly automaticky přidány ignoruje.
Byl proveden pokus o přidání odkazu na výchozí sestavení, ale sestavení již byla přidána, takže tento pokus se ignoruje.  
  
 **ID chyby:** BC2024  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nadbytečné kód.
