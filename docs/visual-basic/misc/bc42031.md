# <a name="catch-block-never-reached-exception-handled-above-in-the-same-try-statement"></a>Blok nedospělo; 'catch' \<výjimky > již zpracováno výše ve stejném příkazu 'zkuste to!
A `Catch` blok v kódu není dostupný, protože je zpracována v předchozích `Try` bloku.  
  
Ve výchozím nastavení tato zpráva je upozornění. Další informace o zobrazení nebo skrytí upozornění zpracování upozornění jako chyby, najdete v části [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).
  
 **ID chyby:** BC42031  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte redundantní příkaz.
