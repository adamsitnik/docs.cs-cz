# <a name="inherits-statement-must-precede-all-declarations-in-a-class"></a>Příkaz 'Inherits' musí předcházet před všemi deklaracemi v třídě.
`Inherits` Příkazy musí být na začátku definice tříd.  
  
 **ID chyby:** BC30683  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesunout `Inherits` příkaz do horní části třídy.
