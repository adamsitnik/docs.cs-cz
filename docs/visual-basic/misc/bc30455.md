# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-ltmethodnamegt"></a>Nebyl zadán argument pro parametr '&lt;parametername&gt;"z"&lt;methodname&gt;.
Argument pro požadovaný parametr nebyla zadána.  
  
 **ID chyby:** BC30455  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte argument pro zadaný parametr.
