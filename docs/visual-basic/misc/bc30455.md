# <a name="argument-not-specified-for-parameter-parametername-of-methodname"></a>Nebyl zadán argument pro parametr '\<parametername > 'z'\<methodname > '
Argument pro požadovaný parametr nebyla zadána.  
  
 **ID chyby:** BC30455  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte argument pro zadaný parametr.
