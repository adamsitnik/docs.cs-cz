# <a name="event-lteventnamegt-cannot-be-found"></a>Událost '&lt;eventname&gt;' nebyl nalezen
Pokoušíte se o zpracování události, které kompilátor nemůže najít.  
  
 **ID chyby:** BC30590  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že je správně zadán název události.
