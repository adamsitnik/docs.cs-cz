# <a name="event-eventname-cannot-be-found"></a>\<Událost EventName > nebyla nalezena.
Pokusili jste se zpracovat událost, kterou kompilátor nemůže najít.  
  
 **ID chyby:** BC30590  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že je správně zadán název události.
