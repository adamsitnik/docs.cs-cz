# <a name="removehandler-is-already-declared"></a>'RemoveHandler' je už deklarovaný.
Více než jeden `RemoveHandler` deklarace zobrazovaly v deklaraci vlastní událost. `RemoveHandler` Deklarace deklaruje postup použitý k odebrání obslužné rutiny události.  
  
 **ID chyby:** BC31128  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat redundantní `RemoveHandler` příkazu.
