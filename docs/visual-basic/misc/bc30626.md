# <a name="namespace-statement-must-end-with-a-matching-end-namespace"></a>Příkaz "Namespace" musí končit odpovídajícího 'End Namespace
`Namespace` bloky musí končit `End Namespace` příkazu.  
  
 **ID chyby:** BC30626  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `End Namespace` příkaz na konec objektu `Namespace` bloku.
