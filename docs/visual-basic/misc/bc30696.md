# <a name="ltmembername1gt-and-ltmembername2gt-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>"&lt;membername1&gt;"a"&lt;membername2&gt;se nemohou vzájemně přetěžovat, protože se liší pouze typy nepovinných parametrů
Přetěžované členy musí mít různé datové typy pro jeden nebo více požadovaných parametrů.  
  
 **ID chyby:** BC30696  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Jako jiný datový typ definujte aspoň jeden povinný parametr.
