# <a name="integer-constant-expected"></a>Byla očekávána celočíselná konstanta
`#ExternalSource` Směrnice dochází, ve kterém není druhý argument celočíselného literálu. Pouze celočíselného literálu je v tomto kontextu platné. Pojmenované konstanty nebo výčtu člen není platný.  
  
 **ID chyby:** BC30204  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Použijte pojmenované konstanty nebo výčtu člen místo literál.  
  
2. Zadejte celé číslo literálu jako druhý argument `#ExternalSource` směrnice.
