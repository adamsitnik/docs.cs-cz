# <a name="nullable-modifier-cannot-be-specified-on-both-a-variable-and-its-type"></a>Modifikátor Nullable nelze zadat pro proměnnou a její typ
Modifikátor typu s možnou hodnotou Null (?) byl součástí pro deklarované proměnné a jejím deklarovaným typem.  
  
 **ID chyby:** BC33100  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte z proměnné nebo jejím deklarovaným typem modifikátor typu s možnou hodnotou Null (?).
