# <a name="lttype1gt-lttypenamegt-and-lttype2gt-lttypenamegt-conflict-in-ltnamespacegt-ltnamespacenamegt"></a>&lt;Type1&gt; "&lt;typename&gt;" a &lt;type2&gt; "&lt;typename&gt;" jsou v konfliktu v &lt;obor názvů&gt; "&lt;namespacename &gt;'
Dvě deklarované elementy, jako je například třídy a modul, mají stejný název ve stejném oboru názvů.  
  
 **ID chyby:** BC30179  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Prohlášení o jeden z elementů přesuňte do jiného oboru.  
  
-   Změňte název jedné z prvků.
