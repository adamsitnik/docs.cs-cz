# <a name="type1-typename-and-type2-typename-conflict-in-namespace-namespacename"></a>\<Type1 > '\<typename > "a \<type2 > '\<typename >' jsou v konfliktu v \<oboru názvů > '\<namespacename >"
Dvě deklarované elementy, jako je například třídy a modul, mají stejný název ve stejném oboru názvů.  
  
 **ID chyby:** BC30179  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Prohlášení o jeden z elementů přesuňte do jiného oboru.  
  
- Změňte název jedné z prvků.
