# <a name="conversion-from-double-to-date-requires-calling-the-datefromoadate"></a>Převod z 'Double' na 'Data' vyžaduje volání "Date.FromOADate.
Jste se pokusili přetypovat `Date` hodnota, která se `Double` hodnotu, která se nedá provést bez použití <xref:System.DateTime.FromOADate%2A?displayProperty=nameWithType> metody.  
  
 **ID chyby:** BC30533  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití <xref:System.DateTime.FromOADate%2A> způsobů, jak převést hodnotu.
