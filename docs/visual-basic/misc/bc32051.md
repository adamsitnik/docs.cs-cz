 <span data-ttu-id="2ad87-105">Obecný postup se volá bez zadání argumentů typu a kompilátor nemůže odvodit typ, který se předává jednomu z parametrů.</span><span class="sxs-lookup"><span data-stu-id="2ad87-105">A generic procedure is called without supplying any type arguments, and the compiler cannot infer the type to pass to one of the parameters.</span></span>

  
 <span data-ttu-id="2ad87-110">V předchozím příkladu kompilátor odvodí typ `Integer` pro `t` na základě hodnoty 42 předané do `arg2`.</span><span class="sxs-lookup"><span data-stu-id="2ad87-110">In the preceding example, the compiler infers type `Integer` for `t` based on the value of 42 passed to `arg2`.</span></span> <span data-ttu-id="2ad87-111">Nicméně, že odvození by vyžadovalo `arg1` typu `Integer()`, to znamená, že pole `Integer` a hodnota 6 předaná do `arg1` neodpovídá tomuto typu.</span><span class="sxs-lookup"><span data-stu-id="2ad87-111">However, that inference would require `arg1` to be of type `Integer()`, that is, an array of `Integer`, and the value 6 passed to `arg1` does not match that type.</span></span>
