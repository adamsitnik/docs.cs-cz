# <a name="type-argument-inference-failed-for-type-parameter-typeparametername-of-genericproceduresignature"></a><span data-ttu-id="4e1a0-102">Nepovedlo se pro parametr typu odvození argumentu typu "\<typeparametername >" z "\<genericproceduresignature >"</span><span class="sxs-lookup"><span data-stu-id="4e1a0-102">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'</span></span>

<span data-ttu-id="4e1a0-103">**ID chyby:** BC32051</span><span class="sxs-lookup"><span data-stu-id="4e1a0-103">**Error ID:** BC32051</span></span>

<span data-ttu-id="4e1a0-104">Nepovedlo se pro parametr typu odvození argumentu typu "\<typeparametername >" z "\<genericproceduresignature >'.</span><span class="sxs-lookup"><span data-stu-id="4e1a0-104">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="4e1a0-105">Argument typu nelze odvodit z argument předaný do parametru '\<parametername >'.</span><span class="sxs-lookup"><span data-stu-id="4e1a0-105">Type argument could not be inferred from the argument passed to parameter '\<parametername>'.</span></span>

 <span data-ttu-id="4e1a0-106">Obecný postup se volá bez zadání argumentů typu a kompilátor nemůže odvodit typ, který se předává jednomu z parametrů.</span><span class="sxs-lookup"><span data-stu-id="4e1a0-106">A generic procedure is called without supplying any type arguments, and the compiler cannot infer the type to pass to one of the parameters.</span></span>

 <span data-ttu-id="4e1a0-107">Obvykle při volání obecné procedury zadejte argument typu pro každý parametr typu, který definuje obecný postup.</span><span class="sxs-lookup"><span data-stu-id="4e1a0-107">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="4e1a0-108">Pokud nezadáte žádné argumenty typu, kompilátor se pokusí odvodit typy, které mají být předány do parametrů typu.</span><span class="sxs-lookup"><span data-stu-id="4e1a0-108">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="4e1a0-109">Pokud kontext volání poskytuje konfliktní informace o datových typech pro parametr typu, pak odvození typu se nezdařilo.</span><span class="sxs-lookup"><span data-stu-id="4e1a0-109">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>

 <span data-ttu-id="4e1a0-110">Následující kód může vygenerovat tuto chybu:</span><span class="sxs-lookup"><span data-stu-id="4e1a0-110">The following code can generate this error:</span></span>

```vb
Public Sub DoSomething(Of T)(arg1 As T(), arg2 As T)
End Sub

Public Sub Main()
    DoSomething(6, 42)
End Sub
```  
  
 <span data-ttu-id="4e1a0-111">V předchozím příkladu kompilátor odvodí typ `Integer` pro `t` na základě hodnoty 42 předané do `arg2`.</span><span class="sxs-lookup"><span data-stu-id="4e1a0-111">In the preceding example, the compiler infers type `Integer` for `t` based on the value of 42 passed to `arg2`.</span></span> <span data-ttu-id="4e1a0-112">Nicméně, že odvození by vyžadovalo `arg1` typu `Integer()`, to znamená, že pole `Integer` a hodnota 6 předaná do `arg1` neodpovídá tomuto typu.</span><span class="sxs-lookup"><span data-stu-id="4e1a0-112">However, that inference would require `arg1` to be of type `Integer()`, that is, an array of `Integer`, and the value 6 passed to `arg1` does not match that type.</span></span>
