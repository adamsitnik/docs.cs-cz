# <a name="inherits-not-valid-in-modules"></a>'Inherits' nejsou platné v modulech
`Inherits` Příkaz nastane uvnitř modulu. Moduly nelze dědit ze tříd.  
  
 **ID chyby:** BC30230  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Inherits` příkaz nebo modul jako třída zadejte znovu.
