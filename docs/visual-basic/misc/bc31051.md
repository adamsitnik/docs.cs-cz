# <a name="namespace-or-type-ltnamegt-has-already-been-imported"></a>Namespace nebo typ "&lt;název&gt;' již byl importován
Byl proveden pokus o import oboru názvů více než jednou.  
  
 **ID chyby:** BC31051  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat redundantní `Imports` příkazy.
