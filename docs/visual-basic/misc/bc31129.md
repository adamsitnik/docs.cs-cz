# <a name="raiseevent-is-already-declared"></a>Je už deklarovaná 'RaiseEvent.
Více než jeden `RaiseEvent` deklarace zobrazovaly v deklaraci vlastní událost. A `RaiseEvent` deklarace deklaruje proceduru slouží pro vyvolání události.  
  
 **ID chyby:** BC31129  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat redundantní `RaiseEvent` příkazu.
