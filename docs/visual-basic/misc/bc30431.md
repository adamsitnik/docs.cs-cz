# <a name="end-property-must-be-preceded-by-a-matching-property"></a>'End Property' musí předcházet odpovídající 'Property'
`End Property` Příkazu se zobrazí v kódu s žádná odpovídající `Property` před deklarací.  
  
 **ID chyby:** BC30431  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `End Property` příkazu, pokud je redundantní.  
  
- Zadat chybějící `Property` postup, pokud chybí.  
  
- Přesunout `End Property` na příslušné místo v kódu.
