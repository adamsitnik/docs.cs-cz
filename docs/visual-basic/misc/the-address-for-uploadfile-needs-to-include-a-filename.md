# <a name="the-address-for-uploadfile-needs-to-include-a-filename"></a>Adresa pro UploadFile musí zahrnovat název souboru
Při nahrávání souborů se `UploadFile` metoda, je nutné zadat název souboru v cílové cestě.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte název souboru.
