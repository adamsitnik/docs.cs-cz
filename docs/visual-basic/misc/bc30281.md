# <a name="structure-structurename-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Struktura '\<%{structurename/ >' musí obsahovat alespoň jednu členskou proměnnou instance nebo deklaraci události
Vyvolá se v funkčně prázdnou strukturou. Musí být deklarován nejméně jeden člen proměnné mezi `Structure` příkazu a `End Structure` příkazu.  
  
 **ID chyby:** BC30281  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidat `Dim` příkazu nebo `Event` příkaz do struktury.
