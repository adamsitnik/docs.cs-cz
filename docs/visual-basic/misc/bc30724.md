# <a name="error-statements-are-not-valid-in-the-immediate-window"></a>Příkazy "Chyba" nejsou platné v příkazovém
`Error` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30724  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `Error` příkazy z ladění kódu.
