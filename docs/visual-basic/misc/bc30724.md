# <a name="error-statements-are-not-valid-in-the-immediate-window"></a>Příkazy Error nejsou platné v příkazovém podokně.
příkazy `Error` lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30724  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte příkazy `Error` z ladění kódu.
