# <a name="typename-must-be-declared-mustinherit-because-it-contains-methods-declared-mustoverride"></a>"\<typename >' musí být deklarován 'MustInherit', protože obsahuje metody s deklarací 'MustOverride.
Typy, které obsahují `MustOverride` členy musí být deklarován `MustInherit`.  
  
 **ID chyby:** BC31411  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidat `MustInherit` modifikátor typu.
