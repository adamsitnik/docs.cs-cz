# <a name="with-contexts-and-statements-are-not-valid-in-debug-windows"></a>'With' příkazy a kontexty nejsou platné v ladicích oknech
`With` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30726  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `With` příkazy z ladění kódu.
