# <a name="enum-ltenumnamegt-must-contain-at-least-one-member"></a>Výčet "&lt;enumname&gt;' musí obsahovat alespoň jeden člen.
Vyvolá se prázdný výčet. Nejméně jeden člen musí být deklarován mezi `Enum` příkazu a `End Enum` příkazu.  
  
 **ID chyby:** BC30280  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidáte deklaraci člena výčtu.
