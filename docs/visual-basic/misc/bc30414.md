# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>Hodnotu typu '\<type1 >' nelze převést na "\<type2 >' protože typy polí mají různý počet rozměrů
Pokusili jste se převod jednoho pole na jiný s různým počtem dimenzí.  
  
 **ID chyby:** BC30414  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změnit jedno nebo obě pole tak, aby odpovídá počet rozměrů.
