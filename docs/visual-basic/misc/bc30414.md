# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-the-array-types-have-different-numbers-of-dimensions"></a>Hodnotu typu '&lt;type1&gt;"nelze převést na"&lt;type2&gt;' protože typy polí mají různý počet rozměrů
Pokusili jste se převod jednoho pole na jiný s různým počtem dimenzí.  
  
 **ID chyby:** BC30414  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změnit jedno nebo obě pole tak, aby odpovídá počet rozměrů.
