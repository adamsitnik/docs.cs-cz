# <a name="object-doesnt-support-current-locale-setting-visual-basic"></a>Objekt nepodporuje aktuální nastavení národního prostředí (Visual Basic)
Pokusili jste se přístup k objektu, který nepodporuje nastavení národního prostředí pro aktuální projekt.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte nastavení národního prostředí, které podporuje objektu.
