# <a name="unable-to-include-xml-fragment-elementpath-of-file-commentfile"></a>Nejde zahrnout XML fragment '\<objektu elementpath >' souboru '\<commentfile > "
`<include>` Značky v komentáři XML obsahuje výraz XPath pro element, který neexistuje, nebo soubor obsahuje chybně vytvořený kód XML.  
  
 **ID chyby:** BC42320  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, že je platný kód XML v souboru.  
  
2. Ověřte, zda existuje XML fragment v souboru.
