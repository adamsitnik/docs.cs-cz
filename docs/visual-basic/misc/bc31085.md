# <a name="date-constant-is-not-valid"></a>Datová konstanta není platná
Datová konstanta je deklarována s neplatné datum.  
  
 **ID chyby:** BC31085  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Opravte datum použité v konstantě.
