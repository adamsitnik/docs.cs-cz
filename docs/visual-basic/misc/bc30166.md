# <a name="cannot-create-an-instance-of-module-modulename"></a>Nelze vytvořit instanci modulu '\<modulename > "
Modul existuje pouze jako jediné sdílené instance a další instance nelze vytvořit.  
  
 **ID chyby:** BC30166  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změnit modul na třídu nebo nahradit ho `New` klauzule s názvem třídy.
