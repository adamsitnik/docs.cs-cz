# <a name="goto-ltlabelnamegt-is-not-valid-because-ltlabelnamegt-is-inside-a-with-statement-that-does-not-contain-this-statement"></a>' GoTo &lt;labelname&gt;' je neplatný protože "&lt;labelname&gt;" je uvnitř příkazu 'With', který tento příkaz neobsahuje
`GoTo` příkazy jsou omezeny na přejde v rámci aktuálního bloku kódu.  
  
 **ID chyby:** BC30756  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změny struktury kódu tak, aby `GoTo` příkaz i popisek jsou uvnitř `With` bloku.
