# <a name="white-space-may-not-appear-here"></a>Prázdné znaky se tady nemůžou vyskytovat
Mezera nebo mezery jsou obsažen v názvu elementu XML literál.  
  
 **ID chyby:** BC31177  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte z názvu literál XML elementu místa nebo mezery.
