# <a name="using-statements-are-not-valid-in-the-immediate-window"></a>Příkazy 'Pomocí' nejsou platné v příkazovém
`Using` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30186  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Using` příkaz z **okamžité** okna.
