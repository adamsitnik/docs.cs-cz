# <a name="sub-new-cannot-implement-interface-members"></a>'Sub New' nemůže implementovat členy rozhraní
`Sub New` je konstruktor a nelze ho implementovat členy.  
  
 **ID chyby:** BC31042  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `Implements` příkazy z `Sub New` postupy.
