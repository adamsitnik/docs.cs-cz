# <a name="enum-in-an-interface-cannot-be-declared-ltspecifiergt"></a>Výčet v rozhraní nejde použít deklaraci '&lt;specifikátor&gt;.
Byl zadán neplatný modifikátor výčet deklarované v rozhraní.  
  
 **ID chyby:** BC31069  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebere modifikátory například `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, nebo `Overridable`.
