# <a name="source-folder-and-target-folder-are-the-same"></a>Zdrojová složka a cílová složka jsou stejné
Má byl proveden pokus o kopírování nebo přesunutí souboru, ale zadaný zdrojový adresář a cílový adresář jsou stejné. To může být důsledek nesprávně analýzy cestu k souboru.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, jestli že jste zadali správné zdrojové a cílové složky.
