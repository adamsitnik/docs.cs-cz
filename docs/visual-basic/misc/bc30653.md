# <a name="reference-required-to-module-ltmodulenamegt-containing-the-type-ltmembernamegt"></a>Vyžaduje se odkaz na modul '&lt;modulename&gt;'obsahující typ'&lt;membername&gt;.
Vyžaduje se odkaz na modul '\<modulename >' obsahující typ "\<membername >". Přidejte jej do projektu.  
  
 Projekt obsahuje typ, který závisí na jiném typu ze sestavení, která není zahrnutá v odkazech projektu.  
  
 **ID chyby:** BC30653  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte odkaz na projekt u závislých typů v projektu.
