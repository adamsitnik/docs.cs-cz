# <a name="you-must-specify-a-name"></a>Musíte zadat název
Při provádění operace se soubory jako je například přejmenování souboru, je nutné zadat název.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte název.
