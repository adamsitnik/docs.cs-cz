# <a name="partial-method-methodname-cannot-use-the-implements-keyword"></a>Částečné metody '\<methodname > "nelze použít klíčové slovo 'Implements'
Definici dílčí metody nemohou implementovat rozhraní nebo člen rozhraní.  
  
 **ID chyby:** BC31439  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte `Implements` příkaz v definici dílčí metody.
