# <a name="extension-method-ltmethodnamegt-defined-in-ltmodulenamegt-is-not-generic-or-has-no-free-type-parameters-and-so-cannot-have-type-arguments"></a>Metody rozšíření '&lt;methodname&gt;"definované v"&lt;modulename&gt;' není obecná (nebo nemá žádné volné parametry typu) a nemůže proto mít argumenty typu
Argument typu byla zadána ve volání metody rozšíření, která buď nemá žádné obecné parametry nebo nemá žádné obecné parametry, pro které není typu již zadán. Například následující kód způsobí, že k této chybě.  
  
```vb  
' The extension method is not generic.  
<Extension()> _  
Sub Example(ByVal str As String)  
    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```  
  
 **ID chyby:** BC36907  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidání parametru typu k definici metody rozšíření.  
  
-   Argument typu navíc odebrání volání procedury.
