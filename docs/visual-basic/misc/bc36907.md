<span data-ttu-id="d8419-103">Argument typu byla zadána ve volání metody rozšíření, která buď nemá žádné obecné parametry nebo nemá žádné obecné parametry, pro které není typu již zadán.</span><span class="sxs-lookup"><span data-stu-id="d8419-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="d8419-104">Například následující kód způsobí, že k této chybě.</span><span class="sxs-lookup"><span data-stu-id="d8419-104">For example, the following code causes this error.</span></span>
