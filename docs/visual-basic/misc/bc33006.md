# <a name="end-operator-must-be-the-first-statement-on-a-line"></a>'End Operator' musí být prvním příkazem na řádku
`End Operator` Příkaz následuje dvojtečka (`:`) oddělovač. `End Operator` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC33006  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Oddělte více příkazů na samostatné řádky.
