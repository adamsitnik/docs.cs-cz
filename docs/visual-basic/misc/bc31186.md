# <a name="-is-not-allowed"></a>":" není povoleno
":" není povolený. XML kvalifikovaný v tomto kontextu nelze použít názvy.  
  
 Literál XML byla zadána s kvalifikovaným názvem v neplatném umístění, například cíl v literál instrukce pro zpracování.  
  
 **ID chyby:** BC31186  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte literálu se kvalifikovaný název XML nebo přesunout na platné umístění.  
  
     Informace o tom, které jsou platné v názvech XML znaky najdete v tématu [značek XML (Extensible Language) 1.0](https://www.w3.org/TR/xml).
