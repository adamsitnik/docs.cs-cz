# <a name="expected-beginning--for-an-xml-tag"></a>Byl očekáván úvodní '\<"značky XML
Značka XML pro literál XML chybí požadované začátku ' <' znaků.  
  
 **ID chyby:** BC31165  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidat "<" znak na úvodní značka XML pro literál XML.
