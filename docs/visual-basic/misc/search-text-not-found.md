# <a name="search-text-not-found"></a>Hledaný text nebyl nalezen
Nebyl nalezen text, který jste zadali.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte, jestli zadaný text je napsaný správně.
