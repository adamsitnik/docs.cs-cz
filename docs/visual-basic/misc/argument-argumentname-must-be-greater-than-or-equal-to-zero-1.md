# <a name="argument-argumentname-must-be-greater-than-or-equal-to-zero"></a>Argument \<název argumentu argumentname > musí být větší než nebo rovna hodnotě nula.
Hodnota argumentu menší než nula. Hodnoty argumentů musí být větší než nebo rovna hodnotě nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že hodnota argumentu je větší než nebo rovna hodnotě nula.  
  
2. Pokud je argument zadaný ve výpočtu, zkontrolujte hodnoty jsou zpracovávány výpočtu.
