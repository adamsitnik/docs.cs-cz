# <a name="option-statements-must-precede-any-declarations-or-imports-statements"></a>Příkazy 'Možnosti' musí předcházet před všemi deklaracemi a příkazy "importy.
`Option` příkazy by měl být první příkazy bez komentářů ve zdrojovém kódu.  
  
 **ID chyby:** BC30627  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesunout `Option` příkazy zdrojového kódu, bezprostředně před `Imports` a `Namespace` příkazy.
