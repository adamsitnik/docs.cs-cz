# <a name="declare-statements-in-a-module-cannot-be-declared-specifier"></a>Deklarovat příkazy v modulu nelze deklarovat jako '\<specifikátor > "
A `Declare` má specifikátor, který není platný v rámci deklarace `Module` deklarace. Moduly možné nikdy instancovat, nepodporují dědičnosti a nemůžou implementovat rozhraní.  
  
 **ID chyby:** BC30786  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
