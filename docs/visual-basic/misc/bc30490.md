# <a name="default-cannot-be-combined-with-specifier"></a>'Výchozí' nelze kombinovat s '\<specifikátor >'
Pokusili jste se kombinovat `Default` – klíčové slovo se specifikátorem, jako například `Shared` nebo `Private`, která je v tomto kontextu není platné.  
  
 **ID chyby:** BC30490  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `Default`.  
  
     -nebo-  
  
- Odeberte konfliktní specifikátor.
