# <a name="baselogname-cannot-be-nothing-or-an-empty-string"></a>BaseLogName nemůže být Nothing nebo prázdný řetězec
Hodnota <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> vlastnost nemůže být `Nothing` nebo prázdný řetězec.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> Vlastnost určuje základní název pro soubory protokolů.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nastavte <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> nastavte na řetězec, který obsahuje alespoň jeden znak.
