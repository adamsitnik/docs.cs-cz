# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-optional"></a>Nelze zadat výchozí hodnoty pro parametry, které nejsou deklarovány "Volitelné"
Při volání procedury, a jejich nepřebírají výchozí hodnoty v deklaraci je nutné zadat povinnými argumenty.  
  
 **ID chyby:** BC32024  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Deklarovat argument představoval `Optional` nebo odeberte výchozí hodnotu.
