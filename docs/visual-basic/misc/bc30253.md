# <a name="interface-must-end-with-a-matching-end-interface"></a>"Rozhraní" musí končit odpovídajícího 'End Interface
`Interface` Vyvolá příkaz bez odpovídající `End Interface` příkazu. Je nutné použít `End Interface` příkaz pro ukončení definici rozhraní.  
  
 **ID chyby:** BC30253  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, že `Interface` členy jsou správně naformátované.  
  
2.  Přidat `End Interface` příkaz položky celkem konec definici rozhraní.
