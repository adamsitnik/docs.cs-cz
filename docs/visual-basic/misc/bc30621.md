# <a name="end-structure-must-be-preceded-by-a-matching-structure"></a>'End Structure' musí předcházet odpovídající 'strukturu.
`End Structure` Příkaz lze použít pouze na konci bloku struktura, která začíná `Structure` příkazu.  
  
 **ID chyby:** BC30621  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že aktuální struktura blok začíná `Structure` příkazu.
