# <a name="module-statement-must-end-with-a-matching-end-module"></a>Příkaz 'Module' musí končit odpovídajícího 'End Module
Bloky modul musí končit `End Module` příkazu.  
  
 **ID chyby:** BC30625  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `End Module` příkaz na konec objektu `Module` bloku.
