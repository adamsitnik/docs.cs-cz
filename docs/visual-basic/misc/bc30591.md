# <a name="withevents-variable-does-not-raise-any-events"></a>Proměnnou "WithEvents" nevyvolává žádné události
Objekt proměnná deklarovaná pomocí `WithEvents` – klíčové slovo neodkazuje na instanci třídy, která může vyvolat události.  
  
 **ID chyby:** BC30591  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že proměnná objektu odkazuje na instanci třídy, která může vyvolat události.
