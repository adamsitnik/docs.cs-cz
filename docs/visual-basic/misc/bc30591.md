# <a name="withevents-variable-does-not-raise-any-events"></a>Proměnná WithEvents nevyvolává žádné události.
Objektová proměnná deklarovaná s `WithEvents` klíčovým slovem neodkazuje na instanci třídy, která může vyvolat události.  
  
 **ID chyby:** BC30591  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zajistěte, aby proměnná objektu odkazovala na instanci třídy, která může vyvolat události.
