# <a name="custom-modifier-can-only-be-used-immediately-before-an-event-declaration"></a>Modifikátor 'Custom' lze použít pouze bezprostředně před deklaraci 'události.
`Custom` – Klíčové slovo musí předcházet pouze `Event` – klíčové slovo v deklaraci vlastní událost.  
  
 **ID chyby:** BC31140  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesunout `Custom` – klíčové slovo předcházet `Event` – klíčové slovo.  
  
     —nebo—  
  
- Odeberte `Custom` – klíčové slovo.
