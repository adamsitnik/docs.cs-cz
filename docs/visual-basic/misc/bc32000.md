# <a name="local-variable-variablename-cannot-be-referred-to-before-it-is-declared"></a>Lokální proměnnou '\<NázevProměnné >' nelze odkazovat dříve, než je deklarována
Proměnné v postup se používá před vlastní deklarací.  
  
 **ID chyby:** BC32000  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunutí deklarace atributy stála před první odkaz na proměnnou.
