# <a name="alias-ltnamespacenamegt-is-already-declared"></a>Alias'&lt;namespacename&gt;' již byla deklarována
Pokusili jste se použít `Imports` Import oboru názvů, který je už deklarovaný.  
  
 **ID chyby:** BC30572  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zaškrtněte, obor názvů, který se pokoušíte importovat.
