# <a name="add-failed-duplicate-key-value-supplied"></a>Nepovedlo se přidáte. Zadán duplicitní hodnota klíče
`Add` Operace selhala, protože hodnota klíče je stejná jako jiné hodnoty klíče. Hodnoty klíče musí být jedinečný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že hodnota klíče je jedinečná.
