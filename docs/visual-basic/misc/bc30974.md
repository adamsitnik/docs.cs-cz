# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>Příkazy komentářů nelze vyhodnotit (ne v obsahu)
Příkaz komentář byl zadán v příkazovém během relace ladění. Příkazy komentářů do okna Příkazy se nevyhodnocují.  
  
 **ID chyby:** BC30974
