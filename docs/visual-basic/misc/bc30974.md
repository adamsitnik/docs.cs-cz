# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>Příkazy komentářů nejde vyhodnotit (není v obsahu).
Příkaz komentáře byl v příkazovém okně během relace ladění zadán. Příkazy komentářů v příkazovém okně nejsou vyhodnocovány.  
  
 **ID chyby:** BC30974
