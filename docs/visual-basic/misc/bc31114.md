# <a name="custom-event-must-end-with-a-matching-end-event"></a>'Custom Event' musí končit odpovídající "ukončit událost
A `Custom Event` deklarace musí končit `End Event` příkazu.  
  
 **ID chyby:** BC31114  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, že `Custom Event` deklarace končí `End Event` příkazu.
