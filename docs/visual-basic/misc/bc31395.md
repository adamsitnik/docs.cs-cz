# <a name="type-characters-are-not-allowed-in-label-identifiers"></a>Znaky typů nejsou povolené v identifikátorech popisků
Příkaz popisek obsahuje alespoň jeden znak typu identifikátoru.  
  
 Popisek příkazu musí být platný název jazyka Visual Basic. Povolené znaky nesmí obsahovat žádný z identifikátor – znaky typu (`%`, `&`, `@`, `!`, `#`, a `$`). Zobrazit [deklarované názvy elementů](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **ID chyby:** BC31395  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte popisek příkazu – znak typu identifikátoru nebo znaky.
