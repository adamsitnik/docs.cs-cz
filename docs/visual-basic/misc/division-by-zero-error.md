# <a name="division-by-zero-visual-basic-error"></a>Dělení nulou (Visual Basic)
Hodnota výrazu jako dělitel je nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda proměnné ve výrazu. Chybně proměnné implicitně vytvářet číselné proměnné inicializovány na nulu.  
  
2.  Zkontrolujte předchozí operace na proměnnou ve výrazu, zejména těch, které jsou předány do procedury jako argumenty z jiných postupů.
