# <a name="the-file-is-already-open"></a>Soubor je již otevřen
Byl proveden pokus o otevření souboru, který je již otevřen.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zavřete soubor před jeho otevřením.  
  
     -nebo-  
  
- Odebrat redundantní `FileOpen` příkaz v kódu.
