# <a name="keyword-is-not-valid-as-an-identifier"></a>Klíčové slovo není platné jako identifikátor
Klíčové slovo dochází, ve kterém jsou vyžadována název elementu. Kontext volání pro daný element jako je například proměnná nebo proceduru.  
  
 **ID chyby:** BC30183  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Klíčové slovo nahraďte platný název elementu.
