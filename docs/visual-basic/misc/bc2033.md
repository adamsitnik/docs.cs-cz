# <a name="error-embedding-win32-manifest-option-win32manifest-conflicts-with-nowin32manifest"></a>Chyba při vkládání manifestu Win32: možnost/win32manifest je v konfliktu s/nowin32manifest.
Kompilátor Visual Basic předal jak možnost kompilátoru `/win32manifest`, tak i `/nowin32manifest` možnost kompilátoru. Kompilátoru Visual Basic může být předána pouze jedna z těchto možností.  
  
 **ID chyby:** BC2033  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte buď možnost kompilátoru `/win32manifest`, nebo možnost kompilátoru `/nowin32manifest`.
