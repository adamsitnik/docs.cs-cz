# <a name="error-embedding-win32-manifest-option-win32manifest-conflicts-with-nowin32manifest"></a>Chyba při vkládání manifestu Win32: Možnost/win32manifest je v konfliktu s možností/nowin32manifest
Kompilátor jazyka Visual Basic se předal dvoubajtový i `/win32manifest` – možnost kompilátoru a `/nowin32manifest` – možnost kompilátoru. Kompilátor jazyka Visual Basic lze předat pouze jeden z těchto možností.  
  
 **ID chyby:** BC2033  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Buď odeberte `/win32manifest` – možnost kompilátoru nebo `/nowin32manifest` – možnost kompilátoru.
