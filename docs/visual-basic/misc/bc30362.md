# <a name="ltpropertyname1gt-cannot-override-ltpropertyname2gt-because-they-differ-by-readonly-or-writeonly"></a>"&lt;propertyname1&gt;nemůže přepsat&lt;Název_vlastnosti2&gt;' protože se liší příznakem 'ReadOnly' nebo 'jen pro zápis.
Pokusili jste se přepsat vlastnost s druhou vlastnost, která se liší od prvního pouze `ReadOnly` nebo `WriteOnly`.  
  
 **ID chyby:** BC30362  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že vlastnosti jsou rozlišené pomocí více než `ReadOnly` a `WriteOnly`.
