# <a name="propertyname1-cannot-override-propertyname2-because-they-differ-by-readonly-or-writeonly"></a>'\<propertyname1 >' nelze přepsat '\<Název_vlastnosti2 > 'protože se liší příznakem 'ReadOnly' nebo 'jen pro zápis.
Došlo k pokusu o přepsání vlastnosti druhou vlastností, která se liší od prvního pouze pomocí `ReadOnly` nebo. `WriteOnly`  
  
 **ID chyby:** BC30362  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že jsou vlastnosti odlišené o více `ReadOnly` než `WriteOnly`a.
