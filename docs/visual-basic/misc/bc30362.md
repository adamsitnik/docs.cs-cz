# <a name="propertyname1-cannot-override-propertyname2-because-they-differ-by-readonly-or-writeonly"></a>"\<propertyname1 >" nelze přepsat '\<Název_vlastnosti2 > "protože se liší příznakem 'ReadOnly' nebo 'jen pro zápis.
Pokusili jste se přepsat vlastnost s druhou vlastnost, která se liší od prvního pouze `ReadOnly` nebo `WriteOnly`.  
  
 **ID chyby:** BC30362  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že vlastnosti jsou rozlišené pomocí více než `ReadOnly` a `WriteOnly`.
