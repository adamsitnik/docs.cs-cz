# <a name="modifier-is-not-valid-on-a-local-constant-declaration"></a>"\<modifikátor >' není platný v deklaraci lokální konstanty
A `Const` výroku uvnitř procedury obsahuje neplatné klíčové slovo jako například `Overloads`.  
  
 **ID chyby:** BC30246  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte neplatné klíčové slovo z `Const` příkazu.
