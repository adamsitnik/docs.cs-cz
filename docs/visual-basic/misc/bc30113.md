# <a name="namespace-is-not-a-valid-name-and-cannot-be-used-as-the-root-namespace-name"></a>'\<oboru názvů >' není platný název a nelze použít jako název kořenového oboru názvů
Kvalifikovaný název oboru názvů není platný název elementu. Název může obsahovat jenom abecední znaky, desítková číslice a podtržítka a musí začínat znak abecedy nebo podtržítkem (`_`).  
  
 **ID chyby:** BC30113  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pokud obor názvů je součástí sady vnořené obory názvů, zkontrolujte, pokud některý z vnějšího názvy obsahují neplatné znaky.
