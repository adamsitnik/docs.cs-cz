# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<název >' nemůže vystavovat typ \<type1 > v \<specifikátor > \<type2 >'\<typename > 'prostřednictvím \<name >
Pokusili jste se vystavovat privátní typ mimo veřejnou třídu `MyClass`.  
  
 **ID chyby:** BC30508  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Deklaruje typ `Public`.
