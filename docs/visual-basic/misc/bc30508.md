# <a name="ltnamegt-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-lttypenamegt-through-ltnamegt"></a>"&lt;název&gt;' nemůže vystavovat typ &lt;type1&gt; v &lt;specifikátor&gt; &lt;type2&gt; "&lt;typename&gt;"prostřednictvím &lt;název&gt;
Pokusili jste se vystavovat privátní typ mimo veřejnou třídu `MyClass`.  
  
 **ID chyby:** BC30508  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Deklaruje typ `Public`.
