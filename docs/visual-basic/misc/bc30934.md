 <span data-ttu-id="91b84-104">Atribut poskytuje metadata pro element, na který je použit, a kompilátor musí být schopný sestavit všechna metadata v době kompilace.</span><span class="sxs-lookup"><span data-stu-id="91b84-104">An attribute provides metadata for the element it is applied to, and the compiler must be able to construct all the metadata at compile time.</span></span> <span data-ttu-id="91b84-105">Z tohoto důvodu musí každý atribut používat hodnoty, které jsou konstanty v době kompilace, a proto každý argument atributu musí být vyhodnocen na konstantní hodnotu v době kompilace.</span><span class="sxs-lookup"><span data-stu-id="91b84-105">For this reason, every attribute must use values that are constant at compile time, and therefore every attribute argument must evaluate to a compile-time constant value.</span></span>
