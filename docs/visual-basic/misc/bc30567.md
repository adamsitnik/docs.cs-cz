# <a name="array-initializer-is-missing-number-elements"></a>V inicializátoru pole \<chybí číslo > elementy.
Nezadali jste dostatek prvků v inicializátoru pole.  
  
 **ID chyby:** BC30567  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte chybějící argument nebo argumenty.  
  
- Slouží `ReDim` ke změně velikosti pole.
