# <a name="array-initializer-is-missing-ltnumbergt-elements"></a>Chybí inicializátor pole &lt;číslo&gt; elementy
Neposkytli jste dostatečný počet elementů v inicializátoru pro vaše pole.  
  
 **ID chyby:** BC30567  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte chybí argument nebo argumenty.  
  
-   Použití `ReDim` ke změně velikosti pole.
