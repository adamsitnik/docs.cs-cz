# <a name="you-must-specify-a-path-that-is-under-the-current-folder-or-one-of-its-sub-folders"></a>Musíte zadat cestu, která je v aktuální složce nebo jeden z jejích podsložek
Zadaná cesta se nenachází v aktuálním adresáři. To může být důsledek nesprávně analýzy cesty k souborům při zadávání relativní cestu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že zadáváte správnou cestu.
