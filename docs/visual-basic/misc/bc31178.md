# <a name="expected-closing--for-xml-entity"></a>Bylo očekáváno ukončení ";" entity XML
XML odkazu na entitu (například `>`) chybí uzavírací středníkem (;).  
  
 **ID chyby:** BC31178  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte středník na konci odkaz na entity XML.
