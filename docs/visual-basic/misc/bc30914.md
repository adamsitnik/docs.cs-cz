# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-to-specifier1-type1-through-specifier2-type2"></a>"\<název >' nemůže vystavovat nadřazený typ delegáta '\<delegatetype >" události, kterou implementuje do \<specifier1 > '\<type1 >' prostřednictvím \<specifier2 > '\<type2 > "
Kód poskytuje typ soukromého mimo veřejnou třídu.  
  
 **ID chyby:** BC30914  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Deklaruje typ `Public` nebo pomocí jiného typu.
