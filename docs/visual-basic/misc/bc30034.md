# <a name="bracketed-identifier-is-missing-closing-"></a>Identifikátor v hranatých závorkách nebyla nalezena pravá ']'
Hranaté závorce v názvu uvozený uvozovacím znakem musí proběhnout v odpovídajících dvojicích.  
  
 **ID chyby:** BC30034  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Místo se pravá závorka na konci názvu uvozený uvozovacím znakem.
