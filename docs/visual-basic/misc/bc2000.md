# <a name="initialization-error"></a>Chyba při inicializaci
Došlo k chybě při inicializaci program. To může být výsledkem zrušení instalace, i když je stále probíhá.  
  
 **ID chyby:** BC2000  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokuste se znovu spustit instalaci.
