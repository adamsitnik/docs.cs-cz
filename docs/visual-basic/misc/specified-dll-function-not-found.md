# <a name="specified-dll-function-not-found"></a>Zadaná funkce knihovny DLL nebyla nalezena
Dynamická knihovna (DLL) v referenční dokumentace knihoven uživatele byl nalezen, ale funkce knihovny DLL určené nebyl nalezen v rámci knihovny DLL.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte platné pořadí v deklaraci funkce.  
  
2.  Ujistěte se, že název knihovny DLL a alias jsou správné.
