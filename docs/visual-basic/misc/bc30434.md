# <a name="events-in-a-module-cannot-be-declared-specifier"></a>Události v modulu nejde použít deklaraci '\<specifikátor > "
Použili jste specifikátor, který není platný pro události v rámci `Module` příkazu. Moduly možné nikdy instancovat, nepodporují dědičnosti a nemůžou implementovat rozhraní.  
  
 **ID chyby:** BC30434  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
