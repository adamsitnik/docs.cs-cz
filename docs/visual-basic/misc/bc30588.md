# <a name="omitted-argument-cannot-match-a-paramarray-parameter"></a>Vynechaný argument nemůže odpovídat parametru ParamArray
Vynecháte argument vyžadovaný pro shodu `ParamArray` parametru. `ParamArray`, která umožňuje předat libovolný počet argumentů proceduře, lze použít pouze v posledním argumentu seznamu argumentů.  
  
 **ID chyby:** BC30588  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadáte vynechaný argument.
