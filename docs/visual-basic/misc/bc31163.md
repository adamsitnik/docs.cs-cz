# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>Byl očekáván párový koncový apostrof hodnoty atributu XML
Hodnota atributu, který je oddělen složenými jednoduchých uvozovek chybí ukončovací znak uvozovky.  
  
 **ID chyby:** BC31163  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidáte pro vymezení hodnotu atributu koncový znak jednoduché uvozovky (').
