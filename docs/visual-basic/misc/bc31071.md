# <a name="structure-in-an-interface-cannot-be-declared-ltspecifiergt"></a>Pro strukturu v rozhraní nejde použít deklaraci '&lt;specifikátor&gt;.
Byl zadán neplatný modifikátor struktura deklarována v rozhraní.  
  
 **ID chyby:** BC31071  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebere modifikátory například `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, nebo `Overridable`.
