# <a name="comma--or-a-valid-expression-continuation-expected"></a>Čárka, ")", nebo platné pokračování výrazu byl očekáván
Seznam, jako je například seznam argumentů v definici procedury nebo volání procedury, není správně oddělení nebo ukončit jeho členů. Jednou z možných příčin je dva členy seznamu se oddělené pouze mezery nebo mezery.  
  
 **ID chyby:** BC32017  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ukončit seznamu pomocí čárku (,) do samostatných seznam členů a uzavírací závorky ()).
