# <a name="methodname-cannot-be-implemented-more-than-once"></a>"\<methodname >" nejde implementovat víc než jednou.
Pokusili jste se implementovat metodu, která již byla implementována.  
  
 **ID chyby:** BC30583  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte název metody, abyste měli jistotu, že mají zadány správně.  
  
2. Pokud metoda je redundantní, odeberte nadbytečné deklarace.
