# <a name="xml-dtds-are-not-supported"></a>Soubory DTD kódu XML nejsou podporovány.
Definice typu dokumentu (DTD) je určený jako literál XML.  
  
 **ID chyby:** BC31175  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte DTD. Použijte definici schématu XML (XSD) k určení schéma pro dokument XML.
