# <a name="lteventnamegt-is-not-an-event-of-ltcontainernamegt"></a>"&lt;eventname&gt;"není událost"&lt;containername&gt;.
Události definované není deklarován pro objekt.  
  
 **ID chyby:** BC30676  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda název události.  
  
2.  Zkontrolujte, že při přístupu k správný objekt. Použít plně kvalifikované odkazy na objekt nebo `Imports` smlouvu pro import oboru názvů vhodné v případě potřeby.
