# <a name="comma-or--expected-type-parameter-list"></a>Čárka nebo ")' byl očekáván (seznam parametrů typu)
Seznam parametrů typu obsahuje neočekávaný element. Použít čárky (`,`) pro oddělení parametrů typu v tento seznam a použít pravou závorku (`)`) na konec seznamu.  
  
 **ID chyby:** BC32099  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte čárku a pokračovat v seznamu parametrů typu nebo uzavírací závorkou znak ukončení seznamu.
