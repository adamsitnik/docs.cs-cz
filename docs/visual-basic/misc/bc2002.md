# <a name="source-file-ltfilenamegt-specified-multiple-times"></a>Zdrojový soubor &lt;filename&gt; zadaný několikrát.
Souboru byla zadána více než jednou.  
  
 **ID chyby:** BC2002  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte redundantní souboru specifikátorů.
