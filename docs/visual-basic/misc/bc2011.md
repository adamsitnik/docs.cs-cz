# <a name="unable-to-open-response-file-ltfilenamegt"></a>Nelze otevřít soubor odpovědí '&lt;filename&gt;.
Soubor, který se pokoušíte získat přístup není k dispozici.  
  
 **ID chyby:** BC2011  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že je stále aktivní připojení k síti.  
  
-   Zkontrolujte vaše oprávnění k souboru.
