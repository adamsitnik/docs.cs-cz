# <a name="optional-and-paramarray-cannot-be-combined"></a>'Volitelné' a 'ParamArray' nelze kombinovat
`Optional` a `ParamArray` modifikátory se vzájemně vylučují.  
  
 **ID chyby:** BC30642  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte buď `ParamArray` nebo `Optional`, ale ne obojí.
