# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-"></a><span data-ttu-id="55887-102">Inicializátor pole nejde zadat pro nekonstantní dimenzi; Použijte prázdný inicializátor "{}.</span><span class="sxs-lookup"><span data-stu-id="55887-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer '{}'</span></span>
<span data-ttu-id="55887-103">Pole inicializuje dimenzi, která není v době kompilace znám.</span><span class="sxs-lookup"><span data-stu-id="55887-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="55887-104">Následující kód vygeneruje tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="55887-104">The following code generates this error.</span></span>  
  
```  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="55887-105">Následující kód se vyhnete chybu.</span><span class="sxs-lookup"><span data-stu-id="55887-105">The following code avoids the error.</span></span>  
  
```  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="55887-106">**ID chyby:** BC30949</span><span class="sxs-lookup"><span data-stu-id="55887-106">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="55887-107">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="55887-107">To correct this error</span></span>  
  
-   <span data-ttu-id="55887-108">Pokud je to možné Určete konstantní dimenze v deklaraci pole.</span><span class="sxs-lookup"><span data-stu-id="55887-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
-   <span data-ttu-id="55887-109">Pokud konstantní dimenze nelze zadat, je třeba inicializovat pole pomocí smyčky, když se stanou známými nekonstantním dimenze.</span><span class="sxs-lookup"><span data-stu-id="55887-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
