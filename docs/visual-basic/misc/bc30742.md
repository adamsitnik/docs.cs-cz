# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>Hodnota '\<valuename1 >' nelze převést na '\<valuename2 >'
Došlo k pokusu o převod typu mezi dvěma nekompatibilní typy.  
  
 **ID chyby:** BC30742  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte typy, které jsou přetypování.
