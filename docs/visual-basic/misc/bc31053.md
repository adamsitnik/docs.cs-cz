# <a name="implements-statement-must-follow-any-inherits-statement-and-precede-all-declarations-in-a-class"></a>Příkaz Implements musí následovat po příkazu Inherits a musí předcházet všechny deklarace ve třídě.
V neplatném umístění byl zjištěn příkaz `Implements`.  
  
 **ID chyby:** BC31053  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Místo příkazů `Inherits`, ale před všemi deklaracemi, umístěte příkazy `Implements` hned za ně. Příklad:  
  
    ```vb  
    Class Derived  
       Inherits Base  
       Implements One  
       Sub SubOne() Implements One.Method1  
          ' Add code to implement the method.  
       End Sub  
    End Class  
    ```
