# <a name="enums-must-be-declared-as-an-integral-type"></a>Výčty musí být použita deklarace integrálního typu
Jediné platné typy můžete použít v výčty jsou `SByte`, `Byte`, `Short`, `UShort`, `Integer`, `UInteger`, `Long`, a `ULong`. Můžete použít jiné datové typy.  
  
 **ID chyby:** BC30650  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte datový typ `SByte`, `Byte`, `Short`, `UShort`, `Integer`, `UInteger`, `Long`, nebo `ULong`.
