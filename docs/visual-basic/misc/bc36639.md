# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="3c4a9-102">'ByRef' parametru '\<parametername >' nelze použít ve výrazu lambda</span><span class="sxs-lookup"><span data-stu-id="3c4a9-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="3c4a9-103">Výraz lambda deklarován v rámci `Sub` nebo funkce nemůže používat žádné `ByRef` parametry, které `Sub` nebo funkce.</span><span class="sxs-lookup"><span data-stu-id="3c4a9-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="3c4a9-104">Následující kód například způsobí tuto chybu, protože `ByRef` parametr `n` se používá v lambda výrazu.</span><span class="sxs-lookup"><span data-stu-id="3c4a9-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="3c4a9-105">**ID chyby:** BC36639</span><span class="sxs-lookup"><span data-stu-id="3c4a9-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3c4a9-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="3c4a9-106">To correct this error</span></span>  
  
-   <span data-ttu-id="3c4a9-107">Přiřazení `ByRef` parametr pro místní proměnné a použijte místní proměnnou ve výrazu lambda, jak je znázorněno v následujícím kódu:</span><span class="sxs-lookup"><span data-stu-id="3c4a9-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
