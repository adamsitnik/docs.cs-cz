# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="15857-102">Ve výrazu lambda nejde použít parametr ByRef \<parametername >.</span><span class="sxs-lookup"><span data-stu-id="15857-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="15857-103">Výraz lambda deklarovaný v rámci `Sub` nebo funkce nemůže použít žádné parametry `ByRef` této `Sub` nebo Function.</span><span class="sxs-lookup"><span data-stu-id="15857-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="15857-104">Například následující kód způsobí tuto chybu, protože parametr `ByRef` `n` je použit ve výrazu lambda.</span><span class="sxs-lookup"><span data-stu-id="15857-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb

'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```

## <a name="to-correct-this-error"></a><span data-ttu-id="15857-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="15857-106">To correct this error</span></span>  
  
- <span data-ttu-id="15857-107">Přiřaďte parametr `ByRef` místní proměnné a použijte místní proměnnou ve výrazu lambda, jak je znázorněno v následujícím kódu:</span><span class="sxs-lookup"><span data-stu-id="15857-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>

    ```vb  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
