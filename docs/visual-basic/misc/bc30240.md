# <a name="exit-must-be-followed-by-sub-function-property-do-for-while-select-or-try"></a>"Konec" musí být následován znakem 'Sub', 'Funkce', 'Property' "Do", 'Pro', 'While', 'Select' nebo 'akci.
`Exit` Příkaz obsahuje neplatné klíčové slovo. `Exit` musíte zadat blok, ve kterém ovládací prvek přesunou do příkazu za příkazem bloku; například `End While`.  
  
 **ID chyby:** BC30240  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte následující platný – klíčové slovo `Exit`, nebo odeberte `Exit` příkazu.
