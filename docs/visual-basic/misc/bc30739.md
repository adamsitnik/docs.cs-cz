# <a name="raiseevent-statements-are-not-valid-in-the-immediate-window"></a>Příkazy 'RaiseEvent' nejsou platné v příkazovém
`RaiseEvent` nelze použít v **okamžité** okna.  
  
 **ID chyby:** BC30739  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Test `RaiseEvent` příkazy z kódu v **Editor kódu**.
