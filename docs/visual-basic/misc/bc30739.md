# <a name="raiseevent-statements-are-not-valid-in-the-immediate-window"></a>Příkazy RaiseEvent nejsou platné v příkazovém podokně.
příkazy `RaiseEvent` nelze použít v **příkazovém** okně.  
  
 **ID chyby:** BC30739  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Otestujte příkazy `RaiseEvent` z kódu v **editoru kódu**.
