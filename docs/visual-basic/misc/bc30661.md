# <a name="field-or-property-membername-is-not-found"></a>Pole nebo vlastnost '\<membername >' nebyl nalezen
Pro atribut odkazuje na nedefinované pole nebo vlastnost.  
  
 **ID chyby:** BC30661  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda pole nebo vlastnost.  
  
2.  Ověřte, že váš projekt můžete přístup ke třídě správné atribut a přidat do odkazů projektu v případě potřeby.
