# <a name="statements-and-labels-are-not-valid-between-select-case-and-first-case"></a>Příkazy a Návěstí nejsou platné mezi 'Select Case' a prvním 'Case'
Příkaz, který není komentář se zobrazí mezi otevírání `Select` nebo `Select Case` příkazu a jeho první `Case` příkazu.  
  
 **ID chyby:** BC30058  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokud je použité příkaz komentář, před oddělovačem komentáře (`'` nebo `REM`). V opačném případě přesuňte nebo odstraňte příkaz.
