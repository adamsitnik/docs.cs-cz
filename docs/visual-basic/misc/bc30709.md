# <a name="variable-declaration-statements-are-not-valid-in-the-immediate-window"></a>Příkazy deklarace proměnné nejsou platné v příkazovém
Deklarace proměnných nelze použít v **okamžité** okna.  
  
 **ID chyby:** BC30709  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nedeklarujte proměnné v **okamžité** okna.
