# <a name="variable-declaration-statements-are-not-valid-in-the-immediate-window"></a>Příkazy deklarací proměnných nejsou platné v příkazovém podokně.
Deklarace proměnných nelze použít v **příkazovém** okně.  
  
 **ID chyby:** BC30709  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nedeklarujte proměnné v **příkazovém** okně.
