# <a name="range-variable-name-cannot-match-the-name-of-a-member-of-the-object-class"></a>Název proměnné rozsahu nesmí shodovat s názvem členu třídy 'Object'
Proměnnou rozsahu v dotazu LINQ odpovídá názvu členem <xref:System.Object> třídy. To způsobí konflikt s objekty vytvořené pomocí jazyka Visual Basic pro dotaz LINQ.  
  
 **ID chyby:** BC36606  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zvolte nový název pro proměnnou rozsahu, který neodpovídá názvu členem <xref:System.Object> třídy.
