# <a name="get-statement-must-end-with-a-matching-end-get"></a>Příkaz 'Get' musí končit odpovídajícího 'End Get
`Get` procedury vlastností musí končit `End Get` příkazu.  
  
 **ID chyby:** BC30631  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že `Get` procedura property končí `End Get` vytvořit.
