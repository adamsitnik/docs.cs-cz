# <a name="too-few-type-arguments-to-genericmethodname"></a>Příliš málo argumentů typu "\<genericMethodName >"
Obecná metoda byla vyvolána s méně argumentů, než je počet parametrů typu.  
  
 Při vyvolání obecné metody, je nutné zadat jeden argument typu pro jednotlivé parametry typu určené metody.  
  
 **ID chyby:** BC32042  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidání argumentů na seznam argumentů typu tak, aby se jeden pro každý parametr typu na obecnou metodu, kterou vyvoláváte.
