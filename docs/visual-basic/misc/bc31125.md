# <a name="end-removehandler-must-be-preceded-by-a-matching-removehandler-declaration"></a>'End RemoveHandler' musí předcházet párová deklarace "RemoveHandler.
`End RemoveHandler` Příkazu došlo k chybě bez odpovídající `RemoveHandler` příkazu. `End RemoveHandler` musí předcházet odpovídající `RemoveHandler` příkazu.  
  
 **ID chyby:** BC31125  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, že předchozí `RemoveHandler` příkazu je platný a správně zadané.
