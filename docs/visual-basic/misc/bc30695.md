# <a name="typename-membername-must-be-declared-shadows-because-another-member-with-this-name-is-declared-shadows"></a>\<Název typu > '\<membername >' musí být deklarován 'Shadows', protože jiný člen s tímto názvem je deklarována jako 'Shadows'
Tato vlastnost nebo procedura má stejný název jako jiná vlastnost nebo procedura označené `Shadows` modifikátor.  
  
 **ID chyby:** BC30695  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použití `Shadows` modifikátor.
