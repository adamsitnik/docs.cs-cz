# <a name="evaluation-of-expression-or-statement-stopped"></a>Vyhodnocení výrazu nebo příkazu bylo zastaveno.
Byla vybrána položka nabídky **zastavit vyhodnocení** , která způsobila zastavení vyhodnocení.  
  
 **ID chyby:** BC30721  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Povolí, aby provádění pokračovalo bez výběru **zastavit vyhodnocení**.
