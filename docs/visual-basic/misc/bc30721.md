# <a name="evaluation-of-expression-or-statement-stopped"></a>Vyhodnocení výrazu nebo příkazu Zastavit
**Zastavit vyhodnocování** byla vybrána položka nabídky, způsobí vyhodnocení zastavit.  
  
 **ID chyby:** BC30721  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Povolit spuštění pro pokračování bez výběru **zastavit vyhodnocování**.
