# <a name="cannot-calculate-number-of-periods-using-the-arguments-provided"></a>Nelze vypočítat počet období, na základě zadaných argumentů
Volání `NPer` funkce neobsahuje všechny požadované argumenty.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, `Rate`, `Prnt` a `PV` hodnoty jsou zahrnuty ve volání funkce.
