# <a name="methods-or-events-that-implement-interface-members-cannot-be-declared-shared"></a>Pro metody nebo události, které implementují členy rozhraní nejde použít deklaraci 'Shared'
Pokusili jste se deklarovat jako `Shared` metody nebo události, která implementuje člena rozhraní. Metody a události je implementována ve třídě nelze označit `Shared` nebo `Private`, s výjimkou-dědičné třídy.  
  
 **ID chyby:** BC30505  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Shared` – klíčové slovo.
