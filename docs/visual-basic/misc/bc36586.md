# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>Nebyl zadán argument pro parametr '&lt;parametername&gt;"z metody rozšíření'&lt;methodname&gt;"definované v"&lt;typename&gt;.
Nebyl zadán argument pro požadovaný parametr v metodě rozšíření.  
  
 **ID chyby:** BC36586  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte argument pro zadaný parametr.
