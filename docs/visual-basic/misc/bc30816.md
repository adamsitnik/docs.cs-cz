# <a name="ltmembernamegt-is-not-declared"></a>"&lt;membername&gt;' není deklarována
"`<membername>`' není deklarována. `Debug` objekt funkce je dostupná v `System.Diagnostics.Debug` v `System` sestavení.  
  
 Název zadaného člena se nenašel.  
  
 **ID chyby:** BC30816  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte správnost člena.  
  
2.  Použití `Imports` příkaz nebo určete jednoznačně členy definované v jiných oborech názvů. Například `WriteLine` funkce je deklarována v `System.Diagnostics.Debug` oboru názvů.
