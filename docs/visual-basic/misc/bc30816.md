# <a name="membername-is-not-declared"></a>\<member > není deklarovaný.
'`<membername>`' není deklarován. funkce `Debug` objektů je k dispozici v `System.Diagnostics.Debug` v sestavení `System`.  
  
 Zadaný název člena nebyl nalezen.  
  
 **ID chyby:** BC30816  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte pravopis člena.  
  
2. Použijte příkaz `Imports` nebo plně zařaďte členy definované v jiných oborech názvů. Například funkce `WriteLine` je deklarována v oboru názvů `System.Diagnostics.Debug`.
