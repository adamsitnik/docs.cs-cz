# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-lttype3gt-is-not-a-reference-type"></a>Hodnotu typu '&lt;type1&gt;"nelze převést na"&lt;type2&gt;'protože&lt;type3&gt;' není typ odkazu
Příkaz se pokusí převést typ pole na jiný typ pole, kde datový typ prvků cílového pole není typ odkazu.  
  
 **ID chyby:** BC30333  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte datové typy prvky obou polí k určení příčiny chyby.
