# <a name="codepage-name-is-invalid-or-not-installed"></a>Znaková stránka '\<název >' je neplatný nebo není nainstalovaná.
Zadaná znaková stránka není platný nebo se nenašel.  
  
 **ID chyby:** BC2016  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkoumání kódu a určit, která je vhodná pro váš konkrétní případ řešení
