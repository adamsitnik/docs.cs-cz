# <a name="next-statement-names-more-variables-than-there-are-matching-for-statements"></a>Příkaz ' Next ' pojmenovává více proměnných, než jsou k dispozici párové příkazy ' for '
Vnořené smyčky jsou ukončeny jediným příkazem `Next`, který určuje více proměnných smyčky, než jsou smyčky v vnořování, jako v následujícím příkladu:  
  
```vb  
For I = 1 To 10  
   For J = 1 To 5  
      ' ...  
Next J, I, K   ' Next J, I is valid, but there is no loop on K.  
```  
  
 **ID chyby:** BC32037  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zajistěte, aby příkaz `Next` určoval všechny proměnné vnořené smyčky v obráceném pořadí iniciace smyčky.  
  
2. Odeberte z příkazu `Next` všechny nadbytečné proměnné.
