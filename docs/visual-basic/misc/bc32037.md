<span data-ttu-id="b80ec-103">Vnořené smyčky jsou ukončeny jediným příkazem `Next`, který určuje více proměnných smyčky, než jsou smyčky v vnořování, jako v následujícím příkladu:</span><span class="sxs-lookup"><span data-stu-id="b80ec-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>

```vb  
   For J = 1 To 5  
      ' ...

```
