# <a name="addressof-expressions-are-not-valid-in-debug-windows"></a>Výrazy 'AddressOf' nejsou platné v ladicích oknech
`AddressOf` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30731  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odebrat `AddressOf` příkazy z ladění kódu.
