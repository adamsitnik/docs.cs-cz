# <a name="attribute-constant-ltconstantnamegt-cannot-be-the-target-of-an-assignment"></a>Konstanta atributu '&lt;constantname&gt;' nemůže být cílem přiřazení.
Došlo pokusu o přiřazení hodnoty ke konstantě deklarované v atributu.  
  
 **ID chyby:** BC31510  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte chybné přiřazení.  
  
-   Pokud používáte vlastní atributy, které jste vytvořili, znovu definujte člen v atributu jako pole.
