# <a name="method1-cannot-override-method2-because-they-differ-by-the-default-values-of-optional-parameters"></a>"\<– metoda1 >" nelze přepsat '\<method2 > "protože se liší výchozími hodnotami nepovinných parametrů
Pokusili jste se přepsat metodu s jinou metodu, která se liší od prvního jako výchozí hodnoty jeho volitelné parametry, což znamená, že se liší jejich podpisy. Typ může přepsat zděděnou přepisovatelnou metodu deklarace metody se stejným názvem a signaturou, a označením deklarace se `Overrides` modifikátor.  
  
 **ID chyby:** BC30307  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že tyto dvě metody mají stejnou signaturu.
