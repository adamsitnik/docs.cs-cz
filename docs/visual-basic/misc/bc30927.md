<span data-ttu-id="7e3b1-103">Procedura nebo vlastnost je deklarován jako `MustOverride` v rámci třídy, která je definována více deklarací částečné, ale jedna Částečná definice určuje `NotInheritable` pro třídu.</span><span class="sxs-lookup"><span data-stu-id="7e3b1-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="7e3b1-104">Po dělení čísla definice třídy mezi několika částečné deklarace, kompilátor považuje za sjednocení všech částečné deklarace třídy.</span><span class="sxs-lookup"><span data-stu-id="7e3b1-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="7e3b1-105">To platí pouze pro členy, ale také k implementaci, dědičnosti a úroveň přístupu.</span><span class="sxs-lookup"><span data-stu-id="7e3b1-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>
