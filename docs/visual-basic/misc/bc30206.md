# <a name="option-must-be-followed-by-compare-explicit-infer-or-strict"></a>"Možnost" musí být následován znakem "Porovnání", "Explicitní", "Infer" nebo "Strict.
Pouze možnosti, které lze nastavit pomocí `Option` jsou `Compare`, `Explicit`, `Infer`, a `Strict`.  
  
 **ID chyby:** BC30206  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda option – klíčové slovo (`Compare`, `Explicit`, `Infer`, nebo `Strict`) je zadáno chybně.  
  
-   Zkontrolujte, zda `Option` příkaz je neúplná.
