# <a name="implements-not-valid-in-modules"></a>'Implements' není platné v modulech
`Implements` Příkazu nebo klauzuli nastane uvnitř modulu. Moduly nemohou implementovat rozhraní.  
  
 **ID chyby:** BC30231  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Implements` příkaz nebo klauzuli nebo zadejte znovu modulu jako třídu.
