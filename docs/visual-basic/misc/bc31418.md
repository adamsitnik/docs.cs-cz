# <a name="handles-in-modules-must-specify-a-withevents-variable-qualified-with-a-single-identifier"></a>"Handles v modulech musí určit"WithEvents"proměnnou kvalifikovaného pomocí jednoho identifikátoru
Chcete-li určit obslužné rutiny události `Handles` příkazy musíte zadat proměnné objektu, který byl deklarován s `WithEvents` – klíčové slovo.  
  
 **ID chyby:** BC31418  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `WithEvents` Modifikátor pro deklarování proměnných, které se použije `Handles` příkazu.
