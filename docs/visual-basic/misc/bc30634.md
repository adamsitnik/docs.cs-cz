# <a name="statement-cannot-appear-within-a-property-body"></a>Příkaz nemůže být použit v těle vlastnosti
Příkaz nemůže být použit v těle vlastnosti. Konec předpokládá, že vlastnost.  
  
 Uvnitř vlastnosti byl zjištěn neočekávaný příkaz.  
  
 **ID chyby:** BC30634  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, že vaše vlastnost používá správnou syntaxi a končí `End Property` vytvořit.
