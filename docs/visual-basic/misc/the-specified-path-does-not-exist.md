# <a name="the-specified-path-does-not-exist"></a>Zadaná cesta neexistuje.
Zadaná cesta neexistuje, pravděpodobně z důvodu chyby kombinovaných řetězců.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že cesta je správná.
