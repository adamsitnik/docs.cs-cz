# <a name="addhandler-definition-missing-for-event-eventname"></a>Definici 'AddHandler' nebyla nalezena událost "\<eventname >"
Pokud je událost deklarovanou jako `Custom`, je nutné zadat postup pro přidání obslužné rutiny události.  
  
 **ID chyby:** BC31130  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Patří `AddHandler` deklarace mezi `Custom Event` příkazu a `End Event` příkazu.  
  
2.  Další postupy v rámci konec deklarace události ověření správně.
