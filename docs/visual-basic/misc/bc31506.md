# <a name="typename-cannot-be-used-as-an-attribute-because-it-is-declared-mustinherit"></a>'\<typename >' nelze použít jako atribut, protože se deklaruje'MustInherit.
Třídy vlastního atributu nelze deklarovat jako `MustInherit`.  
  
 **ID chyby:** BC31506  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- `MustInherit` Odeberte modifikátor z vlastních atributů.
