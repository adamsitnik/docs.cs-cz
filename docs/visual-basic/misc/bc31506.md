# <a name="lttypenamegt-cannot-be-used-as-an-attribute-because-it-is-declared-mustinherit"></a>"&lt;typename&gt;' nelze použít jako atribut, protože se deklaruje"MustInherit.
Vlastní třídy atributu nemůže být deklarována jako `MustInherit`.  
  
 **ID chyby:** BC31506  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `MustInherit` modifikátor z vlastních atributů.
