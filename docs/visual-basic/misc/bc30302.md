# <a name="type-character-character-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Zadejte znak '\<znak >' nelze použít v deklaraci explicitního typu
Typ znaku, jste už použili, jako `$`, když explicitního určení typu.  
  
 **ID chyby:** BC30302  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odstranit znak typu nebo odstranit `As <Type>` klauzule od deklarace.
