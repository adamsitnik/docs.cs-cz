# <a name="class-classname-could-not-be-created-error"></a>Třídy\<classname > "nebylo možné vytvořit: \<chyby >
Třída inicializace se nezdařila. Jednou z možných příčin je poškozený [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] instalace.  
  
 **ID chyby:** BC32400  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte v uvozovkách chybovou zprávu a proveďte příslušné akce.  
  
2.  Znovu nainstalujte [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] a znovu kompilovat program.  
  
3.  Pokud se chyba objeví znovu, přeinstalujte kompilátor jazyka Visual Basic.  
  
4.  Pokud chyba přetrvává po přeinstalaci, shromážděte informace o okolnostech a upozornit Microsoft Product Support Services.
