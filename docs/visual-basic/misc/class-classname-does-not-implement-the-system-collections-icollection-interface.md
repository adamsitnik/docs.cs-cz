# <a name="class-ltclassnamegt-does-not-implement-the-systemcollectionsicollection-interface"></a>Třídy&lt;classname&gt;' neimplementuje rozhraní System.Collections.ICollection
Třída neimplementuje `System.Collections.ICollection` rozhraní, které definuje metody, velikost, výčty a synchronizaci pro všechny kolekce. Může být nutné přeinstalovat sadu Visual Studio.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přeinstalujte Visual Studio.
