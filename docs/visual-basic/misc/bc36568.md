# <a name="elementname-is-not-a-method-parameter"></a>'\<elementname>' není parametr metody.
Volání procedury poskytuje další argument, který není definován pomocí metody.  
  
 **ID chyby:** BC36568  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nadbytečný argument odeberte ze seznamu argumentů.
