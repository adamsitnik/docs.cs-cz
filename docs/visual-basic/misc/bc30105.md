# <a name="number-of-indices-is-less-than-the-number-of-dimensions-of-the-indexed-array"></a>Počet indexů je menší než počet rozměrů indexovaného pole.
Počet indexů pro přístup k elementu pole musí být přesně stejné jako řád objektu array, to znamená, počet rozměrů deklarované pro něj.  
  
 **ID chyby:** BC30105  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Dokud celkový počet dolních indexů rovná řád objektu array, přidejte odkaz na pole dolních indexů.
