# <a name="xml-comment-tag-returns-is-not-permitted-on-a-declare-sub-language-element"></a>Značka komentáře XML "vrátí" není povolená u elementu jazyka 'deklarovat sub.
Značka komentáře XML "vrátí" není povolená u elementu jazyka 'deklarovat sub'. Komentář XML bude ignorován.  
  
 Komentáře XML pro `Declare Sub` nemůže obsahovat prohlášení `<`vrátí`>` značky.  
  
 **ID chyby:** BC42315  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat nepodporovanou `<`vrátí`>` značky.
