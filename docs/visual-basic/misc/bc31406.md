# <a name="methodname-cannot-override-a-method-that-has-been-shadowed"></a>'\<methodname >' nelze přepsat metodu, která byla stínovaný
Metoda pokusil přepsat metodu, která zastiňuje jiného člena.  
  
 **ID chyby:** BC31406  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použití `Shadows` modifikátor přepsat stínovaný členy.
