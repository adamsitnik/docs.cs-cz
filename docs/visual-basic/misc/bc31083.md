# <a name="methods-in-a-module-cannot-implement-interface-members"></a>Metody v modulu nemohou implementovat členy rozhraní
Pomocí třídy nebo struktury musí implementovat členy rozhraní.  
  
 **ID chyby:** BC31083  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Implementujte členy rozhraní třídy nebo struktury.
