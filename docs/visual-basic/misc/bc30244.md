# <a name="modifier-is-not-valid-on-a-declare"></a>"\<modifikátor >' není platný v příkazu Declare
A `Declare` příkazu obsahuje neplatné klíčové slovo jako například `ReadOnly`.  
  
 **ID chyby:** BC30244  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte neplatné klíčové slovo z `Declare` příkazu.
