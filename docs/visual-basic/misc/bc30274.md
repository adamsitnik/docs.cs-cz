# <a name="parameter-parametername-of-methodname-already-has-a-matching-argument"></a>Parametr '\<parametername > "z"\<methodname >' již existuje odpovídající argument
Argument je předán podle názvu více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC30274  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zaškrtněte, pokud chcete zobrazit, pokud jeden z argumentů názvů je překlep.  
  
-   Duplicitní argument odeberte ze seznamu argumentů.
