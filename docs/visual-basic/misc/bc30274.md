# <a name="parameter-ltparameternamegt-of-ltmethodnamegt-already-has-a-matching-argument"></a>Parametr '&lt;parametername&gt;"z"&lt;methodname&gt;' již existuje odpovídající argument
Argument je předán podle názvu více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC30274  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zaškrtněte, pokud chcete zobrazit, pokud jeden z argumentů názvů je překlep.  
  
-   Duplicitní argument odeberte ze seznamu argumentů.
