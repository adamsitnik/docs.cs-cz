# <a name="end-statements-are-not-valid-in-the-immediate-window"></a>Příkazy 'End' nejsou platné v příkazovém
`Stop` a `End` příkazy pozastavení provádění a nejsou povoleny v rámci ladění.  
  
 **ID chyby:** BC30123  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Bez vyvolání `End` nebo `Stop` výroky **okamžité** okna.
