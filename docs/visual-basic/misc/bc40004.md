# <a name="type1-membername-conflicts-with-type2-membername-on-the-base-class-type3-classname-and-should-be-declared-shadows"></a>\<Type1 > '\<membername >' je v konfliktu s \<type2 > '\<membername >' v základní třídě \<type3 >'\<classname >' by měla obsahovat klíčové slovo 'Shadows'
Programovací element je deklarována se stejným názvem jako element definovaný v základní třídě. V takovém případě by měl stínové elementu v této třídě prvek základní třídy.  
  
 Tato zpráva je upozornění. `Shadows` předpokládá se ve výchozím nastavení. Další informace o zobrazení nebo skrytí upozornění zpracování upozornění jako chyby, najdete v tématu [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID chyby:** BC40004  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidat `Shadows` – klíčové slovo na deklarace, nebo změňte název elementu, který byl deklarován.
