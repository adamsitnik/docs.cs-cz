# <a name="expected-one-of-dim-const-public-private-protected-friend-shadows-readonly-or-shared"></a>Byl očekáván jeden 'Dimenze', 'Const', "Public", "Privátní", "Protected", 'Typu Friend', 'Shadows', 'ReadOnly' nebo 'Shared'
Příkazu deklarace. chybí klíčové slovo deklarace. Jednou z možných příčin je, že deklarace atributu volá metodu.  
  
 **ID chyby:** BC30195  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zaškrtněte, pokud chcete zobrazit, pokud je metoda deklarované uvnitř deklarace atributu.  
  
2. Spusťte příkaz s klíčovým slovem odpovídající deklarace.
