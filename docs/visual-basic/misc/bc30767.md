# <a name="get-statements-are-no-longer-supported-visual-basic"></a>Příkazy 'Get' již nejsou podporovány (Visual Basic)
`Get` příkazy jsou již nejsou podporovány. Funkce pro vstupně-výstupních operací souborů jsou běžně k dispozici v `Microsoft.VisualBasic` obor názvů, ale cílové verzi rozhraní .NET Compact Framework je nepodporuje.  
  
 **ID chyby:** BC30767  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Provedení operace se soubory pomocí funkce definované v <xref:System.IO> oboru názvů.
