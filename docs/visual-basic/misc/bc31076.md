# <a name="modulename-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>'\<modulename >' je modul a nelze na něj odkazovat jako na sestavení
Byl proveden pokus o použití modulu jako sestavení.  
  
 **ID chyby:** BC31076  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte odkaz na sestavení.
