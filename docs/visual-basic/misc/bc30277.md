# <a name="type-character-charactername-does-not-match-declared-data-type-type"></a>Zadejte znak "\<název_postavy >' neodpovídá deklarovanému datovému typu"\<typu > "
Proměnná je deklarovaná s jedním datovým typem, ale uvedené se znakem typu představující nekompatibilní datový typ, třeba `K$ = 10` při `K` je deklarován `Integer`.  
  
 **ID chyby:** BC30277  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změna deklarovanému datovému typu proměnné, nebo změňte nebo odeberte znak typu v referenci.
