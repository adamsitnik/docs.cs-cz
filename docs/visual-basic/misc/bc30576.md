# <a name="statement-does-not-declare-a-get-or-set-method"></a>Příkaz nedeklaruje metodu 'Get' nebo Set.
Váš výpis z nepodaří zadat buď `Get` nebo `Set` příkazu deklarace kolem `Property` postup. Vlastnost je definován jako blok kódu uzavřený do složených závorek `Property` a `End Property` příkazy. V tomto bloku každý `Property` postupu se zobrazí jako z vnitřního bloku uzavřeny v příkazu deklarace a příkazem end.  
  
 **ID chyby:** BC30576  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadat `Get` nebo `Set` příkazu deklarace.
