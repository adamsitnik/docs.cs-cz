# <a name="specified-registry-path-does-not-start-with-a-valid-hive-name"></a>Cesta v registru nezačíná řetězcem název platný hive
Cesta registru nezačíná hive platný název. Hive platné názvy jsou následující:  
  
- `HKEY_CLASSES_ROOT`  
  
- `HKEY_CURRENT_CONFIG`  
  
- `HKEY_CURRENT_USER`  
  
- `HKEY_DYNDATA`  
  
- `HKEY_LOCAL_MACHINE`  
  
- `HKEY_PERFORMANCE_DATA`  
  
- `HKEY_USERS`  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, že zadáváte název platný hive.
