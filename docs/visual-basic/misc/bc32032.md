# <a name="end-function-must-be-the-first-statement-on-a-line"></a>'End Function' musí být prvním příkazem na řádku
`End Function` Příkaz následuje dvojtečka (:) oddělovač. `End Function` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC32032  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na samostatné řádky.
