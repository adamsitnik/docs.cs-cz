# <a name="xml-namespace-prefix-prefix-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Předpona oboru názvů XML 'předpona"je vyhrazené pro XML a není možné změnit identifikátor URI oboru názvů
A vyhrazena Předpona oboru názvů XML, jako například `xmlns`, se použil literál XML.  
  
 **ID chyby:** BC31152  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte na hodnotu, která není vyhrazena Předpona oboru názvů XML.
