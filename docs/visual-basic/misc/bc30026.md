# <a name="end-sub-expected"></a>Byl očekáván příkaz 'End Sub'
`End` Příkaz musí odpovídat bloku jeho ukončení. A `Sub` definici procedury nebyl správně zavřen.  
  
 **ID chyby:** BC30026  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `End Sub` příkaz na konec procedury.
