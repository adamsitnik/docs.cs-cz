# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>Literály XML a vlastnosti OS XML nejsou k dispozici během tuto relaci ladění, protože nejsou v tomto sestavení použit
Literál XML nebo vlastnosti osy XML aplikace bylo odkazováno v **Watch** nebo **okamžité** okno během ladicí relace v XML, které v jazyce Visual Basic funkce nejsou dostupné. To platí pro sestavení, který nepoužívá XML v jazyce Visual Basic funkce nebo je sestavení pro vydání.  
  
 **ID chyby:** BC31196  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Spuštění nové ladicí relace použitím ladění sestavení ze sestavení.
