# <a name="case-else-can-only-appear-inside-a-select-case-statement"></a>'Klíčové slovo case Else' může být použito pouze uvnitř příkazu 'Select Case'
A `Case Else` příkaz proběhne mimo `Select` bloku. A `Case Else` příkaz lze použít pouze mezi `Select` nebo `Select Case` příkazu a odpovídající `End Select` příkazu.  
  
 **ID chyby:** BC30071  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Case Else` příkazu nebo přesunout v rámci `Select` bloku.
