# <a name="modifier-is-not-valid-on-a-delegate-declaration"></a>"\<modifikátor >' není platný v deklaraci delegáta
Pokusili jste se použít modifikátor, jako například `Shared`, v deklaraci delegáta. Delegáti jsou objekty používané pro volání metod jiné objekty, definovat konstruktor, který se předá specifikace metodu objektu. Není platný pro zadání modifikátor v deklaraci delegáta.  
  
 **ID chyby:** BC30385  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte modifikátor.
