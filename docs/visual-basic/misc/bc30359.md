# <a name="default-can-be-applied-to-only-one-property-name-in-a-lttypegt"></a>"Výchozí" lze použít pouze jeden název vlastnosti v &lt;typu&gt;
Pokusili jste se použít `Default` více než jeden název vlastnosti uvnitř třídy, rozhraní nebo struktura.  
  
 **ID chyby:** BC30359  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte všechny kromě jednoho z `Default` deklarace.
