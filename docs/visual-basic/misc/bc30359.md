# <a name="default-can-be-applied-to-only-one-property-name-in-a-type"></a>'Výchozí' lze použít pouze jeden název vlastnosti v \<typ >
Pokusili jste se použít `Default` více než jeden název vlastnosti uvnitř třídy, rozhraní nebo struktura.  
  
 **ID chyby:** BC30359  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte všechny kromě jednoho z `Default` deklarace.
