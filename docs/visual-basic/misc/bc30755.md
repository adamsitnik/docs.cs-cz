Nelze větvit do `SyncLock` bloku.  
  
 **ID chyby:** BC30755  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změny struktury kódu tak, aby předchází popisek `SyncLock` bloku.
