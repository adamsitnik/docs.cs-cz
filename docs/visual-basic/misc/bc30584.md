# <a name="type-cannot-be-inherited-more-than-once"></a>'\<typu >' nelze dědit více než jednou.
Duplikovali jste `Inherits` příkazu.  
  
 **ID chyby:** BC30584  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat redundantní `Inherits` příkazu.
