# <a name="specifiers-valid-only-at-the-beginning-of-a-declaration"></a>Specifikátory jsou platné pouze na začátku deklarace
Klíčová slova usnadnění přístupu a jiné označení musí předcházet deklarace – klíčové slovo, jako například `Dim` nebo `Function`.  
  
 **ID chyby:** BC30181  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Všechny specifikátory přesuňte mezi seznam atributů (pokud existuje) a deklarace – klíčové slovo.
