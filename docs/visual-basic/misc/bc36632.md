# <a name="nullable-parameters-must-specify-a-type"></a>Parametry s možnou hodnotou Null musí určovat typ.
Parametr obsahuje modifikátor typu s možnou hodnotou Null (?), ale neurčuje typu.  
  
 **ID chyby:** BC36632  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte typ při deklaraci parametru s možnou hodnotou Null.
