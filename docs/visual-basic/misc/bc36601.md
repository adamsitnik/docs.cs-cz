# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>Znaky typu nelze použít v deklaracích proměnných rozsahu
Typ znak byla zahrnuty na konci názvu deklaraci proměnné rozsahu v dotazu LINQ.  
  
 **ID chyby:** BC36601  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte znak typu v deklaraci proměnné rozsahu.
