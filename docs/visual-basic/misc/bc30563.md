# <a name="ltnamegt-is-ambiguous-in-the-application-objects-ltlistgt"></a>"&lt;název&gt;je nejednoznačný v objektech aplikace&lt;seznamu&gt;.
Zadali jste název, který je nejednoznačný a proto je v konfliktu s jiným názvem. Kompilátor jazyka Visual Basic nemá žádná pravidla řešení konfliktů; rozlišení názvů musí být sami. V seznamu k dispozici v chybové zprávě objekty aplikací, kde název je nejednoznačný.  
  
 **ID chyby:** BC30563  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Plně kvalifikujte název.
