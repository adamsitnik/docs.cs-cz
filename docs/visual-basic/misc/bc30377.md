# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>Metoda "\<methodname >" je již deklarována v rozhraní "\<interfacename >"
Můžete mít deklaraci metody rozhraní podruhé, jejich duplikování.  
  
 **ID chyby:** BC30377  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte duplicitní deklarace.
