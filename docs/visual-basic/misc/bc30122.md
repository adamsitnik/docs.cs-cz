# <a name="stop-statements-are-not-valid-in-the-immediate-window"></a>Příkazy 'Stop' nejsou platné v příkazovém
`Stop` a `End` příkazy pozastavení provádění a nejsou povoleny v rámci ladění.  
  
 **ID chyby:** BC30122  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Bez vyvolání `Stop` nebo `End` výroky **okamžité** okna.
