<span data-ttu-id="b443c-103">Odvozené třídy nedeklaruje konstruktor a Visual Basic nelze vygenerovala, protože neexistuje žádný konstruktor základní třídy, které můžete volat.</span><span class="sxs-lookup"><span data-stu-id="b443c-103">A derived class does not declare a constructor, and Visual Basic cannot generate one because there is no base class constructor it can call.</span></span>  
  
 <span data-ttu-id="b443c-104">Pokud odvozené třídy nedeklaruje konstruktor, Visual Basic se pokusí vygenerovat implicitní konstruktor bez parametrů, která volá `MyBase.New()`.</span><span class="sxs-lookup"><span data-stu-id="b443c-104">When a derived class does not declare a constructor, Visual Basic attempts to generate an implicit parameterless constructor that calls `MyBase.New()`.</span></span> <span data-ttu-id="b443c-105">Pokud neexistuje žádný dostupný konstruktor v základní třídě, kterou lze volat bez argumentů, nebo pokud existuje více než jeden, Visual Basic nelze generovat implicitní konstruktor.</span><span class="sxs-lookup"><span data-stu-id="b443c-105">If there is no accessible constructor in the base class that can be called without arguments, or if there is more than one, Visual Basic cannot generate an implicit constructor.</span></span>
