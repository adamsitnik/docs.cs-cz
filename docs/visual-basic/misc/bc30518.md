# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>Rozlišení přetěžování se nezdařilo, protože žádnou dostupnou '\<metoda >' lze volat s těmito argumenty:\<chyby >
Provedení volání přetížené metody, ale kompilátor nemůže najít metodu s odpovídající seznam argumentů.  
  
 **ID chyby:** BC30518  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte seznam argumentů, které jste zadali.
