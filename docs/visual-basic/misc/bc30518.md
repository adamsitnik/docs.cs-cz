# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-can-be-called-with-these-argumentslterrorgt"></a>Rozlišení přetěžování se nezdařilo, protože žádnou dostupnou "&lt;metoda&gt;' lze volat s těmito argumenty:&lt;chyba&gt;
Provedení volání přetížené metody, ale kompilátor nemůže najít metodu s odpovídající seznam argumentů.  
  
 **ID chyby:** BC30518  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte seznam argumentů, které jste zadali.
