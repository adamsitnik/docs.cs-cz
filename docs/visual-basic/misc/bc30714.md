# <a name="resume-statements-are-not-valid-in-the-immediate-window"></a>Příkazy Resume nejsou platné v příkazovém podokně.
Příkazy Resume lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30714  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Resume` příkazy z ladicího kódu.
