# <a name="resume-statements-are-not-valid-in-the-immediate-window"></a>Příkazy 'Pokračovat' nejsou platné v příkazovém
Příkazy Resume jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30714  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Resume` příkazy z ladění kódu.
