# <a name="set-statement-must-end-with-a-matching-end-set"></a>Příkaz 'Nastavení' musí končit odpovídajícího 'End Set
`Set` procedury vlastností musí končit `End Set` příkazu.  
  
 **ID chyby:** BC30633  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že `Set` procedura property končí `End Set` vytvořit.
