# <a name="end-event-must-be-preceded-by-a-matching-custom-event"></a>'End Event' musí předcházet odpovídající 'vlastní událost
`End Event` Vyvolá příkaz bez odpovídající `Custom Event` příkazu. `End Event` musí předcházet odpovídající `Custom Event` příkazu.  
  
 **ID chyby:** BC31123  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že předchozí `Custom Event` příkazu je platný a správně zadané.
