# <a name="members-in-a-module-cannot-be-declared-ltspecifiergt"></a>Členy v modulu nelze použít deklaraci '&lt;specifikátor&gt;.
Použili jste specifikátor, který není platný pro člena v rámci `Module` příkazu. Moduly možné nikdy instancovat, nepodporují dědičnosti a nemůžou implementovat rozhraní.  
  
 **ID chyby:** BC30436  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
