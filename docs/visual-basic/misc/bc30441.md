# <a name="catch-must-end-with-a-matching-end-try"></a>'Příkaz catch' musí končit odpovídajícího 'End Try
A `Catch` příkazu se zobrazí ve vašem kódu bez odpovídajícího `End Try` příkazu. `Catch` příkazy musí být uzavřena s `End Try` příkazu.  
  
 **ID chyby:** BC30441  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Catch` příkazu.  
  
2.  Přidat `End Try` příkaz uzavírat blok.
