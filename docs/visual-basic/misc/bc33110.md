<span data-ttu-id="34376-103">Nejde odvodit společný typ pro první a druhý operand binárního operátoru if.</span><span class="sxs-lookup"><span data-stu-id="34376-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="34376-104">Jeden musí mít rozšiřující převod na druhý.</span><span class="sxs-lookup"><span data-stu-id="34376-104">One must have a widening conversion to the other.</span></span>

<span data-ttu-id="34376-105">Binární `If` operátor vyžaduje, aby existoval rozšiřující převod mezi jedním z argumentů a druhým argumentem.</span><span class="sxs-lookup"><span data-stu-id="34376-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="34376-106">Například vzhledem k tomu, že není rozšiřující převod v obou směrech mezi `Integer` a `String`, následující kód způsobuje tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="34376-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb

'' Not valid.
' Console.WriteLine(If(first, second))
```

## <a name="to-correct-this-error"></a><span data-ttu-id="34376-108">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="34376-108">To correct this error</span></span>

  ```vb
  End If
  ```
