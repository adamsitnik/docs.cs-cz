<span data-ttu-id="057bb-103">Nepovedlo se pro parametr typu odvození argumentu typu '\<typeparametername1 >' z '\<genericproceduresignature >'.</span><span class="sxs-lookup"><span data-stu-id="057bb-103">Type argument inference failed for type parameter '\<typeparametername1>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="057bb-104">Argument typu odvozený od argument předaný do parametru '\<názevparametru1 >' je v konfliktu s argumentem typu odvozen z argument předaný do parametru '\<parametername2 >'.</span><span class="sxs-lookup"><span data-stu-id="057bb-104">Type argument inferred from the argument passed to parameter '\<parametername1>' conflicts with the type argument inferred from the argument passed to parameter '\<parametername2>'.</span></span>  
  
 <span data-ttu-id="057bb-105">Obecný postup je volat bez argumentů typu a odvození typu pokus o vytvořil typu konfliktu dat mezi parametry typu.</span><span class="sxs-lookup"><span data-stu-id="057bb-105">A generic procedure is called without any type arguments, and the attempted type inference has produced a data type conflict among the type parameters.</span></span>  
  
 <span data-ttu-id="057bb-106">Za normálních okolností při volání obecný postup je zadat argument typu pro každý z parametrů typu, který definuje obecný postup.</span><span class="sxs-lookup"><span data-stu-id="057bb-106">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="057bb-107">Pokud nezadáte žádné argumenty typu, poté kompilátor se pokusí odvodit typy, které se mají předat parametry typu.</span><span class="sxs-lookup"><span data-stu-id="057bb-107">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="057bb-108">Pokud konfliktní poskytuje kontext volání datový typ informace pro parametr typu odvození typu selže.</span><span class="sxs-lookup"><span data-stu-id="057bb-108">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>  
  
 <span data-ttu-id="057bb-109">Následující kód lze vygenerovat tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="057bb-109">The following code can generate this error.</span></span>  
  
```  
Public Sub takeTwoValues(Of t)(ByVal x As t, ByVal y As t)  
End Sub  
Call takeTwoValues(4, 2.5)  
```  
  
 <span data-ttu-id="057bb-110">Protože prvním argumentem způsobí, že kompilátor k odvození `Integer` pro parametr typu `t`, zatímco druhý argument způsobí, že k odvození `Double` pro stejný parametr typu, dojde ke konfliktu týkající se typů dat, které se mají předat `t`.</span><span class="sxs-lookup"><span data-stu-id="057bb-110">Because the first argument causes the compiler to infer `Integer` for type parameter `t`, while the second argument causes it to infer `Double` for the same type parameter, there is a conflict regarding which data type to pass to `t`.</span></span>  
  
 <span data-ttu-id="057bb-111">**ID chyby:** BC32116</span><span class="sxs-lookup"><span data-stu-id="057bb-111">**Error ID:** BC32116</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="057bb-112">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="057bb-112">To correct this error</span></span>  
  
- <span data-ttu-id="057bb-113">Zadejte argumenty typu pro obecný typ, tak, aby kompilátor k odvození je.</span><span class="sxs-lookup"><span data-stu-id="057bb-113">Supply type arguments to the generic type, so that the compiler does not have to infer them.</span></span>  
  
    ```  
    Call takeTwoValues(Of Double)(4, 2.5)  
    ```  
  
     <span data-ttu-id="057bb-114">Všimněte si, že v tomto případě tam, kde jsou dva normální argumenty různých typů dat, volající kód musíte předat argument typu, který zvládne obě tyto datové typy.</span><span class="sxs-lookup"><span data-stu-id="057bb-114">Notice that in this case, where the two normal arguments are of different data types, the calling code must pass a type argument that can accommodate both of those data types.</span></span> <span data-ttu-id="057bb-115">V takovém případě `Integer` rozšiřuje na `Double`.</span><span class="sxs-lookup"><span data-stu-id="057bb-115">In this case, `Integer` widens to `Double`.</span></span>  
  
     <span data-ttu-id="057bb-116">-nebo-</span><span class="sxs-lookup"><span data-stu-id="057bb-116">-or-</span></span>  
  
- <span data-ttu-id="057bb-117">Předefinování obecný postup zadejte jiný typ parametry pro normální parametry tak, aby nedocházelo k žádnému konfliktu odvození typů.</span><span class="sxs-lookup"><span data-stu-id="057bb-117">Redefine the generic procedure to specify different type parameters for the normal parameters, so that there is no conflict inferring the types.</span></span>  
  
    ```  
    Public Sub takeTwoValues(Of t1, t2)(ByVal x As t1, ByVal y As t2)  
    ```
