<span data-ttu-id="e2bc9-103">Nepovedlo se pro parametr typu odvození argumentu typu '\<typeparametername1 >' z '\<genericproceduresignature >'.</span><span class="sxs-lookup"><span data-stu-id="e2bc9-103">Type argument inference failed for type parameter '\<typeparametername1>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="e2bc9-104">Argument typu odvozený od argument předaný do parametru '\<názevparametru1 >' je v konfliktu s argumentem typu odvozen z argument předaný do parametru '\<parametername2 >'.</span><span class="sxs-lookup"><span data-stu-id="e2bc9-104">Type argument inferred from the argument passed to parameter '\<parametername1>' conflicts with the type argument inferred from the argument passed to parameter '\<parametername2>'.</span></span>  
  
 <span data-ttu-id="e2bc9-105">Obecný postup se volá bez argumentů typu a pokus o odvození typu vytvořil konflikt datového typu mezi parametry typu.</span><span class="sxs-lookup"><span data-stu-id="e2bc9-105">A generic procedure is called without any type arguments, and the attempted type inference has produced a data type conflict among the type parameters.</span></span>  
  
 <span data-ttu-id="e2bc9-106">Obvykle při volání obecné procedury zadejte argument typu pro každý parametr typu, který definuje obecný postup.</span><span class="sxs-lookup"><span data-stu-id="e2bc9-106">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="e2bc9-107">Pokud nezadáte žádné argumenty typu, kompilátor se pokusí odvodit typy, které mají být předány do parametrů typu.</span><span class="sxs-lookup"><span data-stu-id="e2bc9-107">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="e2bc9-108">Pokud kontext volání poskytuje konfliktní informace o datových typech pro parametr typu, pak odvození typu se nezdařilo.</span><span class="sxs-lookup"><span data-stu-id="e2bc9-108">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>  
  
 <span data-ttu-id="e2bc9-109">Následující kód může vygenerovat tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="e2bc9-109">The following code can generate this error.</span></span>  
  
```vb

End Sub  
Call takeTwoValues(4, 2.5)  
```  
  
 <span data-ttu-id="e2bc9-110">Vzhledem k tomu, že první argument způsobí, že kompilátor odvodí `Integer` pro parametr typu `t`, zatímco druhý argument způsobí, že je odvozen `Double` pro stejný parametr typu, dojde ke konfliktu, který datový typ předáte do `t`.</span><span class="sxs-lookup"><span data-stu-id="e2bc9-110">Because the first argument causes the compiler to infer `Integer` for type parameter `t`, while the second argument causes it to infer `Double` for the same type parameter, there is a conflict regarding which data type to pass to `t`.</span></span>  
  
 <span data-ttu-id="e2bc9-111">**ID chyby:** BC32116</span><span class="sxs-lookup"><span data-stu-id="e2bc9-111">**Error ID:** BC32116</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e2bc9-112">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="e2bc9-112">To correct this error</span></span>

    ```  
  
     <span data-ttu-id="e2bc9-114">Všimněte si, že v tomto případě, kde jsou dva běžné argumenty různých datových typů, volající kód musí předat argument typu, který může obsahovat oba tyto datové typy.</span><span class="sxs-lookup"><span data-stu-id="e2bc9-114">Notice that in this case, where the two normal arguments are of different data types, the calling code must pass a type argument that can accommodate both of those data types.</span></span> <span data-ttu-id="e2bc9-115">V tomto případě se `Integer` rozšíří na `Double`.</span><span class="sxs-lookup"><span data-stu-id="e2bc9-115">In this case, `Integer` widens to `Double`.</span></span>  
  
     <span data-ttu-id="e2bc9-116">-nebo-</span><span class="sxs-lookup"><span data-stu-id="e2bc9-116">-or-</span></span>  
  
- <span data-ttu-id="e2bc9-117">Předefinujte obecný postup pro určení různých parametrů typu pro normální parametry, aby nedošlo ke konfliktu při odvozování typů.</span><span class="sxs-lookup"><span data-stu-id="e2bc9-117">Redefine the generic procedure to specify different type parameters for the normal parameters, so that there is no conflict inferring the types.</span></span>

    Public Sub takeTwoValues(Of t1, t2)(ByVal x As t1, ByVal y As t2)  
    ```
