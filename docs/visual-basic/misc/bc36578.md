# <a name="too-few-type-arguments"></a>Nedostatečný počet argumentů
Obecná metoda byla vyvolána s méně argumentů, než je počet parametrů typu.  
  
 Při vyvolání obecné metody, je nutné zadat jeden argument typu pro jednotlivé parametry typu určené metody.  
  
 **ID chyby:** BC36578  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidání argumentů typu na seznam argumentů typu tak, aby se jeden pro každý parametr typu Obecné metody, kterou vyvoláváte.
