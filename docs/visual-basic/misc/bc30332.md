# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-type3-is-not-derived-from-type4"></a>Hodnotu typu "\<type1 >" nelze převést na "\<type2 >' protože"\<type3 >' není odvozen od "\<type4 >"
Příkaz se pokusí převést typ pole na jiný typ pole v situaci, kdy datové typy prvků pole nejsou oba typy odkazů, nebo pokud převod, rozšíření ani zúžení při, není možné mezi typy prvků ze dvou pole.  
  
 **ID chyby:** BC30332  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte datové typy prvky obou polí k určení příčiny chyby.
