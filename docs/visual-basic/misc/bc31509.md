# <a name="unable-to-open-resource-file-filename-error"></a>Nelze otevřít soubor prostředků '\<název souboru >': \<chyby >
Došlo k chybě při otevírání určeného souboru prostředků.  
  
 **ID chyby:** BC31509  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, že zadaný název a umístění jsou správné.  
  
2.  Ověřte, že soubor není poškozený.
