# <a name="specifier1-and-specifier2-cannot-be-combined"></a>"\<specifier1 >' a '\<specifier2 >' nelze kombinovat
Dva nekompatibilní specifikátory, jako například `Shadows` a `Overrides` se používají společně.  
  
 **ID chyby:** BC31408  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte jedno nebo jiných specifikátor.
