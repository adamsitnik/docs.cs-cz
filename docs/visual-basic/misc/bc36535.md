# <a name="instance-members-and-me-cannot-be-used-in-a-query-expression"></a><span data-ttu-id="231b8-102">Členy instance a "Me" nelze použít ve výrazu dotazu</span><span class="sxs-lookup"><span data-stu-id="231b8-102">Instance members and 'Me' cannot be used in a query expression</span></span>
<span data-ttu-id="231b8-103">V dotazu LINQ `Structure` obsahuje odkaz na `Me` nebo na instanci členu struktury.</span><span class="sxs-lookup"><span data-stu-id="231b8-103">A LINQ query in a `Structure` includes a reference to `Me` or to an instance member of the structure.</span></span> <span data-ttu-id="231b8-104">Odkazy na `Me` nebo členy instance nejsou povoleny ve výrazech dotazů v rámci `Structure`.</span><span class="sxs-lookup"><span data-stu-id="231b8-104">References to `Me` or instance members are not allowed in query expressions within a `Structure`.</span></span>  
  
 <span data-ttu-id="231b8-105">**ID chyby:** BC36535</span><span class="sxs-lookup"><span data-stu-id="231b8-105">**Error ID:** BC36535</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="231b8-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="231b8-106">To correct this error</span></span>  
  
1. <span data-ttu-id="231b8-107">Vytvořit kopii tohoto člena instance nebo hodnota vrácená odkazem na `Me` a jak je znázorněno v následujícím příkladu použijte kopii ve výrazu dotazu.</span><span class="sxs-lookup"><span data-stu-id="231b8-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb  
Structure SampleStructure  
    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv  
    End Sub  
End Structure  
```
