# <a name="file-io-of-a-structure-with-field-fieldname-of-type-typename-is-not-valid"></a>Soubor vstupně-výstupní operace se strukturou s polem '\<fieldname >' typu '\<typename >' není platný
Struktura, kterou se pokoušíte k zápisu do souboru obsahuje nepodporovaného typu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte strukturu určit, kde dochází k nepodporovaný typ a změňte ji na platného typu.
