# <a name="automation-object-does-not-have-a-default-value"></a>Automatizační objekt nemá výchozí hodnotu
Visual Basic nelze určit výchozí člen zadaný objekt.  
  
### <a name="to-correct-the-error"></a>Chcete-li opravit chyby  
  
- Dokumentaci k objektu a poskytnout explicitní specifikace vlastnosti nebo metody.
