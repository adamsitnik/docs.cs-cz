# <a name="as-comma-or--expected"></a>"Jako", čárkou nebo ')' očekávání
Chybí parametr typu, který byl uveden `As`, čárky (`,`), nebo pravou závorku ( `)` ).  
  
 **ID chyby:** BC32100  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadat chybějící `As`, čárkou nebo uzavírací závorka.
