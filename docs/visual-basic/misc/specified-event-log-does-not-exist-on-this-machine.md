# <a name="specified-event-log-does-not-exist-on-this-machine"></a>Zadaný protokol událostí na tomto počítači neexistuje.
Byl proveden pokus o přístup k protokolu událostí, který na tomto počítači neexistuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, že zadáváte správný název souboru a cestu.
