# <a name="anonymous-type-member-property-ltpropertynamegt-cannot-be-used-to-infer-the-type-of-another-member-property-because-the-type-of-ltpropertynamegt-is-not-yet-established"></a>Vlastnost člena anonymního typu "&lt;propertyname&gt;' nelze použít k odvození typu jiné vlastnosti členu, protože typ '&lt;propertyname&gt;" dosud není vytvořen
Dokud pokládáme stav, typ anonymního typu vlastnosti, nelze použít k vytvoření typu jiné vlastnosti. Například v následující deklaraci `.IDName = .LastName` je neplatný protože `.LastName` ještě nebyl inicializován.  
  
```  
' Not valid.   
' Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}   
```  
  
 **ID chyby:** BC36559  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Typ vlastnosti vytvořte než začnete používat k inicializaci jiné vlastnosti.  
  
    ```  
    Dim anon2 = New With {Key .LastName = "Jones", Key .IDName = .LastName}  
    ```
