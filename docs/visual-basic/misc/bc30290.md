# <a name="local-variable-cannot-have-the-same-name-as-the-function-containing-it"></a>Místní proměnná nemůže mít stejný název jako funkce, který ji obsahuje
A `Dim` příkaz Určuje proměnnou se stejným názvem jako `Function` postup, který ji obsahuje.  
  
 **ID chyby:** BC30290  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte deklarace proměnné nebo změňte název proměnné.
