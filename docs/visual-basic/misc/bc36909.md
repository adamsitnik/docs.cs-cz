# <a name="cannot-infer-a-data-type-for-ltvariablenamegt-because-the-array-dimensions-do-not-match"></a>Nedá se odvodit typ dat pro '&lt;NázevProměnné&gt;' protože si neodpovídají rozměry polí
Pokud dimenze použitý k inicializaci pole neodpovídají rozměry v deklaraci, kompilátor nemůže odvodit typ dat pro toto pole. Například následující kód způsobí, že k této chybě.  
  
```vb  
' Valid. exampleArray1 is a one-dimensional array of integers.  
Dim exampleArray1() = New Integer() {1, 2, 3}  
' Not valid.  
'Dim exampleArray2(,) = New Integer() {1, 2, 3}  
'Dim exampleArray3(,) = New Integer() {}  
```  
  
 **ID chyby:** BC36909
