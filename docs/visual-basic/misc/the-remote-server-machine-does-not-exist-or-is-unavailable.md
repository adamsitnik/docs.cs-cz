# <a name="the-remote-server-machine-does-not-exist-or-is-unavailable-visual-basic"></a>Vzdálený server neexistuje nebo je k dispozici (Visual Basic)
Pokoušíte se připojit ke vzdálenému serveru, který je k dispozici nebo neexistuje. Tato chyba může být způsobeno síťové připojení ke ztrátě.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Připojení k síti a zkuste to znovu.
