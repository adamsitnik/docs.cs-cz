# <a name="label-labelname-is-already-defined-in-the-current-methodmultiline-lambda"></a>Popisek "\<labelname >' již je definována v aktuálním metody nebo víceřádkového výrazu lambda
Název popisku je duplicitní v rámci procesu. Všechny názvy v rámci procedury musí být jedinečné.  
  
 **ID chyby:** BC30094  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přejmenujte duplicitní popisek.
