# <a name="cannot-rename-with-different-drive"></a>Při přejmenování nelze zadat jinou jednotku
`Rename` Příkaz nelze zadat písmeno jednotky v cestě k souboru.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte písmeno diskové jednotky ze cesty k souboru.
