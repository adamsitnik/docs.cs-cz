# <a name="variable-variablename-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>Proměnná '\<NázevProměnné >' se použije dřív, než jí byla přiřazena hodnota (Chyba jazyka Visual Basic)
Proměnná se používá dříve, než jí byla přiřazena hodnota, což <xref:System.NullReferenceException>.  
  
 **ID chyby:** BC42109  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že proměnná má přiřazenou hodnotu.
