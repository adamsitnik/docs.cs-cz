Proměnná se používá dříve, než jí byla přiřazena hodnota, což <xref:System.NullReferenceException>.  
  
 **ID chyby:** BC42109  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že proměnná má přiřazenou hodnotu.
