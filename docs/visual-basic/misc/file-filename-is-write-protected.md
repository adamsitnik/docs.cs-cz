# <a name="file-ltfilenamegt-is-write-protected"></a>Soubor '&lt;filename&gt;"je chráněna
Soubor, který se pokoušíte zapisovat do je chráněn proti zápisu a proto nejde změnit.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změna oprávnění u souboru pro čtení a zápis nebo zápis do jiného souboru.
