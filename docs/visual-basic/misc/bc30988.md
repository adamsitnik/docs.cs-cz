# <a name="type-or-with-expected"></a>Očekával se typ nebo with.
Při deklaraci instance třídy `New` musí být klíčové slovo následováno názvem typu `With`nebo. Například následující příkazy jsou deklarovány `client` jako instance `Customer` třídy. `Customer` Následuje`New`název typu.  
  
```  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 Počínaje Visual Basic 2008 můžete deklarovat objekt, který bude instancí anonymního typu, v takovém případě neurčíte datový typ. V deklaracích anonymního typu následuje `With` `New`klíčové slovo.  
  
```  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```  
  
 **ID chyby:** BC30988  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte deklaraci tak, `With` že následuje `New`název typu.
