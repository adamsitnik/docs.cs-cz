# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>Příkazy LOOP nejsou platné v příkazovém
`Loop` nelze použít v **okamžité** okna.  
  
 **ID chyby:** BC30708  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Loop` příkazy z ladění kódu.
