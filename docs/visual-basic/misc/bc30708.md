# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>Příkazy Loop nejsou platné v příkazovém podokně.
příkazy `Loop` nelze použít v **příkazovém** okně.  
  
 **ID chyby:** BC30708  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Z ladicího kódu odeberte příkazy `Loop`.
