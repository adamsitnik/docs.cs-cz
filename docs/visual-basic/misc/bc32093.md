# <a name="of-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>"Z" vyžaduje při zadávání argumentů typu pro obecný typ nebo metodu
Příkaz se pokusí vytvořit typ z obecného typu nebo volání obecné metody, bez použití [z](../../visual-basic/language-reference/statements/of-clause.md) klauzuli.  
  
 Syntaxe jazyka Visual Basic pro obecné typy volání pro parametry typu a argumentů a být zavedené prostřednictvím `Of` – klíčové slovo. Kromě toho typ seznamu parametrů nebo seznam argumentů typu musí být uzavřen v závorkách.  
  
 **ID chyby:** BC32093  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zahrnout `Of` – klíčové slovo na začátku argument typu seznam a uzavřít celý seznam v závorkách.
