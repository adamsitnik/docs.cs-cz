# <a name="reference-required-to-module-ltmodulenamegt-containing-the-definition-for-event-lteventnamegt"></a>Vyžaduje se odkaz na modul '&lt;modulename&gt;"obsahující definici události"&lt;eventname&gt;"
Vyžaduje se odkaz na modul ' <`modulename`>' obsahující definici události ' <`eventname`>'. Přidejte jej do projektu.  
  
 Událost je definována v modulu, který není přímo odkazován ve vašem projektu. Kompilátor jazyka Visual Basic vyžaduje přidání odkazu na-li předejít nejednoznačnosti v případě, událost je definována ve více než jeden modul.  
  
 **ID chyby:** BC30006  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Název modulu neodkazovaná zahrnout do odkazů projektu.
