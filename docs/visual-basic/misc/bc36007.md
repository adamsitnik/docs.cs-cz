# <a name="end-using-must-be-preceded-by-a-matching-using"></a>'End Using' musí předcházet odpovídající 'Using'
`End Using` Příkazu se zobrazí s žádná odpovídající `Using` před deklarací.  
  
 **ID chyby:** BC36007  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `End Using` příkazu, pokud je redundantní.  
  
-   Zadat chybějící [příkazu Using](../../visual-basic/language-reference/statements/using-statement.md) Pokud chybí.  
  
-   Přesunout `End Using` příkaz na příslušné místo v kódu.
