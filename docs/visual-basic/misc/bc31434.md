# <a name="method-methodname1-cannot-implement-partial-method-methodname2-because-methodname3-already-implements-it"></a>Metoda "\<methodname1 >' nemůže implementovat částečnou metodu"\<methodname2 > "vzhledem k tomu,"\<methodname3 > "ji už implementuje
Metoda "\<methodname1 >' nemůže implementovat částečnou metodu"\<methodname2 > "vzhledem k tomu,"\<methodname3 > "ji už implementuje. Částečnou metodu smí implementovat jenom jedna metoda.  
  
 Nemůžete mít dvě částečné metody, které implementují stejnou deklaraci částečné metody. Následující kód způsobí, že k této chybě.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub ValueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' First implementation of the partial method.  
    Private Sub ValueChanged()  
        MsgBox(Value was changed to " & Me.Quantity)  
    End Sub  
  
    ' Second implementation of the partial method causes this error.  
    'Private Sub ValueChanged()  
    '    Console.WriteLine("Quantity was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 **ID chyby:** BC31434
