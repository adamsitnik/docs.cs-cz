# <a name="cannot-implement-ltinterfacename1gtltmembernamegt-because-its-implementation-could-conflict-with-the-implementation-for-ltinterfacename2gtltmembernamegt-for-some-type-arguments"></a><span data-ttu-id="0124b-102">Nemůže implementovat '&lt;interfacename1&gt;.&lt; MemberName&gt;'protože jeho implementace by mohla konfliktu s implementací pro"&lt;interfacename2&gt;.&lt; MemberName&gt;' u některých argumentů typů</span><span class="sxs-lookup"><span data-stu-id="0124b-102">Cannot implement '&lt;interfacename1&gt;.&lt;membername&gt;' because its implementation could conflict with the implementation for '&lt;interfacename2&gt;.&lt;membername&gt;' for some type arguments</span></span>
<span data-ttu-id="0124b-103">Třída implementuje více než jeden obecné rozhraní, z nichž jeden dědí z jiné, a dvě implementace člena rozhraní může u určitých konfliktní hodnoty argumentů typu.</span><span class="sxs-lookup"><span data-stu-id="0124b-103">A class implements more than one generic interface, one of which inherits from another, and two implementations of an interface member could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="0124b-104">Tato chyba může generovat následující příkazy.</span><span class="sxs-lookup"><span data-stu-id="0124b-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1(Of t)  
    Sub testSub()  
End Interface  
Public Interface iFace2(Of u)  
    Inherits iFace1(Of u)  
End Interface  
Public Class testClass(Of y, z)  
    Implements iFace1(Of y), iFace2(Of z)  
    Public Sub testSuby() Implements iFace1(Of y).testSub  
    End Sub  
    Public Sub testSubz() Implements iFace1(Of z).testSub  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="0124b-105">Protože `iFace2` dědí z `iFace1` pomocí vlastní typ parametru (`u`), `testClass` by implementovat dvě verze `iFace1.testSub` s identickými podpisy, pokud bylo předáno stejný argument typu `y` a `z`.</span><span class="sxs-lookup"><span data-stu-id="0124b-105">Because `iFace2` inherits from `iFace1` using its own type parameter (`u`), `testClass` would implement two versions of `iFace1.testSub` with identical signatures if the same type argument were passed to `y` and `z`.</span></span> <span data-ttu-id="0124b-106">To vytvoří nejednoznačnost o přístup k verzi.</span><span class="sxs-lookup"><span data-stu-id="0124b-106">This would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="0124b-107">**ID chyby:** BC32125</span><span class="sxs-lookup"><span data-stu-id="0124b-107">**Error ID:** BC32125</span></span>
