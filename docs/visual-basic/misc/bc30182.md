# <a name="type-expected"></a>Očekával se typ.
Dojde k znaky zdrojového kódu, které nejsou rozpoznat jako název typu. Kontext volá základní datový typ nebo element deklarovaný název představující typu.  
  
 **ID chyby:** BC30182  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, že typ je definována.  
  
- Zkontrolujte, jestli název typu je napsaný správně.
