# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>Byly očekávány párové koncové uvozovky hodnoty atributu XML
Hodnota atributu, který je oddělen složenými dvojitých uvozovek chybí ukončovací znak uvozovky.  
  
 **ID chyby:** BC31164  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte koncový znak dvojitých uvozovek pro vymezení hodnotu atributu.
