# <a name="case-cannot-follow-a-case-else-in-the-same-select-statement"></a>Nemůže následovat po 'Case' 'Case Else"ve stejném příkazu 'Select'
A `Case Else` příkaz zavádí příkazů ke spuštění, pokud není nalezena žádná shoda pro počáteční `Case`. A `Case` příkazu bylo zjištěno, po `Case Else` ve stejném `Select` bloku.  
  
 **ID chyby:** BC30321  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesunout `Case Else` na příslušné umístění po `Case` příkazu.
