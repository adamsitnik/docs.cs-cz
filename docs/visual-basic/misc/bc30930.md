# <a name="name-necessary-for-compiling-this-construct-cannot-be-found"></a>"\<název >", je nezbytné pro kompilaci tento konstruktor nelze najít
Nebyl nalezen element potřebné pro kompilaci.  
  
 **ID chyby:** BC30930  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte nezbytné pomocný element.  
  
2.  Ujistěte se, že kód platným spravovaným kódem.
