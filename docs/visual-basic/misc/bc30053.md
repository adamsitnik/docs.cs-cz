# <a name="arrays-cannot-be-declared-with-new"></a><span data-ttu-id="968ef-102">Pole se nedá deklarovat pomocí 'New'</span><span class="sxs-lookup"><span data-stu-id="968ef-102">Arrays cannot be declared with 'New'</span></span>
<span data-ttu-id="968ef-103">`New` – Klíčové slovo se mohou objevit pouze v části inicializace deklarace pole.</span><span class="sxs-lookup"><span data-stu-id="968ef-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="968ef-104">To znamená, že `New` musí být na pravé straně znaménka rovnosti (`=`), může vytvořit nový typ pole má být přiřazena k proměnné pole.</span><span class="sxs-lookup"><span data-stu-id="968ef-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>  
  
 <span data-ttu-id="968ef-105">Zástupce pro inicializaci třídy není k dispozici pro pole.</span><span class="sxs-lookup"><span data-stu-id="968ef-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="968ef-106">Následující řádky kódu dva jsou oba platné a jsou ekvivalentní, protože inicializují objekt ze třídy.</span><span class="sxs-lookup"><span data-stu-id="968ef-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>  
  
```  
Dim formA as Form = New Form  
Dim formA as New Form  
```  
  
 <span data-ttu-id="968ef-107">Inicializace pole však nelze používat stejnou klávesovou zkratku jako třída inicializace.</span><span class="sxs-lookup"><span data-stu-id="968ef-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>  
  
 <span data-ttu-id="968ef-108">Všimněte si, že `New` klauzule pro pole musí obsahovat i závorky `()`a složené závorky, `{}`.</span><span class="sxs-lookup"><span data-stu-id="968ef-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="968ef-109">Závorky určit, že je nový typ pole a složené závorky zadejte hodnoty inicializace.</span><span class="sxs-lookup"><span data-stu-id="968ef-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="968ef-110">Kompilátor, vyžaduje složené závorky i v případě, že jsou prázdné, to znamená, i když nejsou inicializace jakoukoliv hodnotu pole.</span><span class="sxs-lookup"><span data-stu-id="968ef-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>  
  
 <span data-ttu-id="968ef-111">**ID chyby:** BC30053</span><span class="sxs-lookup"><span data-stu-id="968ef-111">**Error ID:** BC30053</span></span>
