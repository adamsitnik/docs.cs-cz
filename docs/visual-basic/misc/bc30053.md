<span data-ttu-id="fd8ca-105">Zástupce pro inicializaci třídy není pro pole k dispozici.</span><span class="sxs-lookup"><span data-stu-id="fd8ca-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="fd8ca-106">Následující dva řádky kódu jsou platné a jsou ekvivalentní, protože inicializují objekt z třídy.</span><span class="sxs-lookup"><span data-stu-id="fd8ca-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>

