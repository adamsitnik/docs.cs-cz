# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-the-lambda-expression-and-delegate-delegatename"></a><span data-ttu-id="f68a2-102">Direktiva Option Strict On nepovoluje zúžení v implicitních převodech typů mezi výrazem lambda a delegátem '\<vlastnost delegatename > "</span><span class="sxs-lookup"><span data-stu-id="f68a2-102">Option Strict On does not allow narrowing in implicit type conversions between the lambda expression and delegate '\<delegatename>'</span></span>
<span data-ttu-id="f68a2-103">S `Option Strict` , nemůže mít zužující převod mezi datový typ parametru delegáta a odpovídajícího parametru výrazu lambda, který je přiřazen proměnné typu delegáta.</span><span class="sxs-lookup"><span data-stu-id="f68a2-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a lambda expression assigned to a variable of that delegate type.</span></span> <span data-ttu-id="f68a2-104">Například v následujícím kódu, delegátu `Del` má jeden parametr typu `Integer`.</span><span class="sxs-lookup"><span data-stu-id="f68a2-104">For example, in the following code, delegate `Del` has one parameter of type `Integer`.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
```  
  
 <span data-ttu-id="f68a2-105">Proto odpovídající parametr libovolný výraz lambda, který je přiřazen proměnné typu `Del` může být `Integer` nebo libovolného datového typu, pro kterou je rozšiřující převod z `Integer`.</span><span class="sxs-lookup"><span data-stu-id="f68a2-105">Therefore, the corresponding parameter of any lambda expression assigned to a variable of type `Del` can be an `Integer` or any data type for which there is a widening conversion from `Integer`.</span></span>  
  
```vb  
' Valid.  
Dim example1 As Del = Function(n As Integer) "Valid"  
Dim example2 As Del = Function(n As Long) "Valid"  
  
' Not valid.  
Dim example3 As Del = Function(n As Short) "Not Valid"  
```  
  
 <span data-ttu-id="f68a2-106">**ID chyby:** BC36662</span><span class="sxs-lookup"><span data-stu-id="f68a2-106">**Error ID:** BC36662</span></span>

- <span data-ttu-id="f68a2-108">Změňte datový typ parametru delegátu nebo lambda výraz tak, aby požadované rozšiřující relace existuje.</span><span class="sxs-lookup"><span data-stu-id="f68a2-108">Change the data type of the parameter in the delegate or the lambda expression so that the required widening relationship exists.</span></span>
