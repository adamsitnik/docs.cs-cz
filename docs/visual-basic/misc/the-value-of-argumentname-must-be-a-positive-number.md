# <a name="the-value-of-argumentname-must-be-a-positive-number"></a>Hodnota \<název argumentu argumentname > musí být kladné číslo.
Hodnota <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> vlastnost musí být větší než nula.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> Vlastnost určuje množství volné místo na disku v bajtech, nezbytné předtím, než mohou být zprávy zapisovány do souboru protokolu  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nastavte <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> vlastnost kladné číslo.
