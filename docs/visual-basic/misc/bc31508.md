# <a name="cannot-find-net-framework-directory-error"></a>Nelze najít adresář rozhraní .NET Framework: \<chyby >
Adresář ve kterém [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] sestavení jsou uložena nebyla nalezena.  
  
 **ID chyby:** BC31508  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Kompilovat program znovu, abyste viděli, pokud se chyba objeví znovu.  
  
2.  Pokud chyba přetrvává, uložte si práci a restartujte aplikaci Visual Studio.  
  
3.  Pokud se chyba objeví znovu, přeinstalujte Visual Basic.  
  
4.  Pokud chyba přetrvává po přeinstalaci, upozorní Microsoft Product Support Services.
