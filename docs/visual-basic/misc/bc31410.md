# <a name="overloading-methods-declared-in-multiple-base-interfaces-is-not-valid"></a>Přetěžování metod deklarovaných ve více základních rozhraních není platné
Více zděděných rozhraních implicitně přetížení stejným způsobem.  
  
 **ID chyby:** BC31410  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `Shadows` modifikátor místo `Overloads` modifikátor.
