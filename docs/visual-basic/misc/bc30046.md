# <a name="method-cannot-have-both-a-paramarray-and-optional-parameters"></a>Metoda nemůže obsahovat oba ParamArray a volitelné parametry
A `ParamArray` argument je nepovinný automaticky a musí být pouze nepovinný argument v deklaraci procedury. Všechny předchozí argumenty musí být povinné.  
  
 **ID chyby:** BC30046  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte volitelné argumenty v deklaraci.
