# <a name="evaluation-of-lambda-expressions-is-not-valid-in-the-debugger"></a>Vyhodnocení výrazů lambda není v ladicím programu platné.
V okně **kukátko** nebo v **příkazovém podokně** během relace ladění byl zadán výraz lambda. Výrazy lambda nejsou k dispozici v ladicích oknech.  
  
 **ID chyby:** BC36627  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Otestujte výrazy lambda v editoru kódu.
