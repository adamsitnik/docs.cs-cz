# <a name="generic-methods-cannot-be-exposed-to-com"></a>Obecné metody nelze vystavit rozhraní COM
Součásti rozhraní .NET Framework, který obsahuje jeden nebo více obecné procedury se exportují do komponenty modelu COM.  
  
 Modelu COM (Component Object) nepodporuje obecné typy a nemůže spolupracovat s nimi.  
  
 **ID chyby:** BC30943  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Obecný postup nebo postupy odebrání součásti rozhraní .NET Framework, nebo nepoužívejte ho pro spolupráci s COM.
