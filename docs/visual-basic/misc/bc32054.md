# <a name="ltmembernamegt-has-the-same-name-as-a-type-parameter"></a>"&lt;membername&gt; má stejný název jako parametr typu
Člen obecnou třídu nebo strukturu má stejný název jako parametr typu této třídě nebo struktuře.  
  
 Každou proměnnou, – konstanta, výčtu, proceduru, vlastnost, událostí a delegáta musí mít název liší od každý parametr typu.  
  
 **ID chyby:** BC32054  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte název člena třídy nebo struktury tak, aby se liší od každý název parametru typu.
