# <a name="specifiers-and-attributes-are-not-valid-on-namespace-statements"></a>Specifikátory a atributy nejsou platné v příkazech 'Namespace.
Obory názvů jsou používány pro hierarchické uspořádání program bloků a programovací prvky. Jejich nepřebírají žádné modifikátory ani atributy.  
  
 **ID chyby:** BC32026  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte všechna klíčová slova a atribut bloky ze `Namespace` příkazu.
