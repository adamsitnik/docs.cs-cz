# <a name="response-file-filename-included-multiple-times"></a>Soubor odpovědí \<název souboru > zahrnut několikrát
Soubor odpovědí byla zadána více než jednou. Soubory odpovědí obsahovat – možnosti kompilátoru a soubory zdrojového kódu pro kompilaci.  
  
 **ID chyby:** BC2003  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte redundantní odpovědi specifikátory souboru.
