# <a name="registry-key-keyname-could-not-be-created"></a>Klíč registru '\<Název_klíče >' nelze vytvořit
Zadaný klíč registru se nezdařilo, pravděpodobně z důvodu nedostatečná oprávnění zabezpečení.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že máte dostatečná oprávnění k přístupu ke klíči.
