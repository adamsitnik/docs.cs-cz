## <a name="example"></a><span data-ttu-id="800d9-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="800d9-109">Example</span></span>  
 <span data-ttu-id="800d9-110">Následující třída obsahuje výchozí vlastnost `defaultProp`, který vyžaduje celočíselný argument.</span><span class="sxs-lookup"><span data-stu-id="800d9-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```

