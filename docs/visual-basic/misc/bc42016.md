# <a name="implicit-conversion-from-lttypename1gt-to-lttypename2gt"></a>Implicitní převod z '&lt;NázevTypu1&gt;"do"&lt;NázevTypu2&gt;.
Výrazu nebo příkazu přiřazení přijímá hodnotu z jednoho typu dat a převede na jiného typu. Vzhledem k tomu, že žádný převod klíčové slovo se používá, je převod *implicitní*.  
  
 Ve výchozím nastavení tato zpráva je upozornění. Informace o zobrazení nebo skrytí upozornění zpracování upozornění jako chyby, najdete v části [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID chyby:** BC42016  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokud je to možné použijte hodnoty stejného datového typu, takže není nutné provádět jakýkoli převod jazyka Visual Basic.  
  
-   Použití `CType` nebo jeden z jiných klíčová slova převodu tak, aby převod *explicitní*.
