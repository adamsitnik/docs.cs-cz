# <a name="typename1-cannot-be-converted-to-typename2"></a>'\<NázevTypu1 >' nelze převést na'\<NázevTypu2 > '
Došlo k pokusu o převod typu mezi dvěma nekompatibilní typy.  
  
 **ID chyby:** BC30741  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte typy, které jsou přetypování.
