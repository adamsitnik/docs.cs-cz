# <a name="try-statements-are-not-valid-in-the-immediate-window"></a>Příkazy try nejsou platné v příkazovém podokně.
příkazy `Try` lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30717  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte příkazy `Try` z ladění kódu.
