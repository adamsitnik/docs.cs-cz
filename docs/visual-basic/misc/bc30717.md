# <a name="try-statements-are-not-valid-in-the-immediate-window"></a>Příkazy 'Zkuste' nejsou platné v příkazovém
`Try` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30717  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `Try` příkazy z ladění kódu.
