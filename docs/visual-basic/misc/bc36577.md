# <a name="anonymous-type-initializers-are-not-valid-in-debug-windows"></a>Inicializátory anonymního typu nejsou platné v ladicích oknech.
V okně **kukátko** nebo v **příkazovém podokně** během relace ladění byl zadán inicializátor anonymního typu. V ladicích oknech nejsou k dispozici Inicializátory anonymního typu.  
  
 **ID chyby:** BC36577  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Otestujte Inicializátory anonymního typu v editoru kódu.
