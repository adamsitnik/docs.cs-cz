# <a name="anonymous-type-initializers-are-not-valid-in-debug-windows"></a>Inicializátory anonymních typů nejsou platné v ladicích oknech
Inicializátor anonymního typu byl zadán v **Watch** nebo **okamžité** okno během relace ladění. Inicializátory anonymních typů nejsou k dispozici při ladění systému windows.  
  
 **ID chyby:** BC36577  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Otestujte inicializátory anonymních typů v editoru kódu.
