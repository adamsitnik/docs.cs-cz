# <a name="types-are-not-available-in-this-context"></a>Typy nejsou v tomto kontextu k dispozici.
Došlo k pokusu o odkaz na typ v `#If` rámci direktivy.  
  
 **ID chyby:** BC30458  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte odkaz.
