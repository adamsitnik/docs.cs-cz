# <a name="currency-is-no-longer-a-supported-type-use-the-decimal-type-instead"></a>"Měna" již není podporovaným typem; Místo toho použití typu "Decimal"
`Currency` Datový typ nahradila ji `Decimal` datového typu.  
  
 **ID chyby:** BC30803  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použití `Decimal` datový typ místo `Currency`.
