# <a name="argument-argumentname-cannot-be-a-multidimensional-array"></a>Argumentem\<' argumentu > ' nemůže být vícerozměrné pole.
Pořadí argumentu pole je větší než 1, ale může to být jenom jednorozměrné pole.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte pořadí argumentů na hodnotu 1.
