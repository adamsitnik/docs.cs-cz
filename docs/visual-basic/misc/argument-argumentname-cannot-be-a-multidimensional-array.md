# <a name="argument-ltargumentnamegt-cannot-be-a-multidimensional-array"></a>Argument '&lt;název argumentu argumentname&gt;' nemůže být vícerozměrné pole
Řád objektu argument pole je větší než 1, ale může být jenom jednorozměrné pole.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte pořadí argument na 1.
