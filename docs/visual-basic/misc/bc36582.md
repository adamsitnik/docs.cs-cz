# <a name="too-many-arguments-to-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>Příliš mnoho argumentů metody rozšíření '&lt;methodname&gt;"definované v"&lt;typename&gt;.
Počet argumentů zadána ve volání metody rozšíření přesahuje počet argumentů, které jsou definovány pro metodu.  
  
 **ID chyby:** BC36582  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte definici metody rozšíření a odebrání dalších argumentů volání.
