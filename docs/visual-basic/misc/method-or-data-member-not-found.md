# <a name="method-or-data-member-not-found"></a>Metoda nebo datový člen nebyl nalezen
Metoda nebo datový člen, který jste se pokusili získat přístup, nebyla nalezena.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že není názvu metody nebo datový člen mít překlep.
