# <a name="removehandler-declaration-must-end-with-a-matching-end-removehandler"></a>Deklarace 'RemoveHandler' musí končit odpovídajícího 'End RemoveHandler
A `RemoveHandler` deklarace musí končit `End RemoveHandler` příkazu.  
  
 **ID chyby:** BC31116  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, že `RemoveHandler` deklarace končí `End RemoveHandler` příkazu.
