# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>Je v konfliktu s '\<name1 >', která se implicitně vytvoří pro'\<name2 >' v \<typu >'\<name3 > '
Název člena typu je v konfliktu s jiným implicitně vytvořeným členem.  
  
 **ID chyby:** BC31058  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přejmenujte explicitně deklarovaný člen na odebrání konfliktu pojmenování.
