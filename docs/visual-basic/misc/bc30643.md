# <a name="property-propertyname-is-of-an-unsupported-type"></a>Vlastnost '\<propertyname >' je nepodporovaného typu
Vlastnost v odkazovaném sestavení obsahuje typ, který nejde reprezentovat.  
  
 **ID chyby:** BC30643  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte typ, který lze použít s vaší aplikací.
