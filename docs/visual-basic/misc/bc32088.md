# <a name="type-arguments-unexpected"></a><span data-ttu-id="75c98-102">Očekávaly se argumenty typu</span><span class="sxs-lookup"><span data-stu-id="75c98-102">Type arguments unexpected</span></span>
<span data-ttu-id="75c98-103">`Implements` Klauzule poskytuje argumentů typu pro tohoto člena rozhraní implementuje.</span><span class="sxs-lookup"><span data-stu-id="75c98-103">An `Implements` clause supplies type arguments for the interface member it is implementing.</span></span>  
  
 <span data-ttu-id="75c98-104">`Implements` Klauzule měli určit pouze rozhraní a člen implementuje.</span><span class="sxs-lookup"><span data-stu-id="75c98-104">The `Implements` clause should only identify the interface and member it is implementing.</span></span> <span data-ttu-id="75c98-105">To znamená, že pokud deklarujete obecný postup je `Of` klauzule a argumenty typu by se zobrazit v hlavní části prohlášení, stejně jako by nebyly implementaci procedury rozhraní.</span><span class="sxs-lookup"><span data-stu-id="75c98-105">This means that if you are declaring a generic procedure, the `Of` clause and the type arguments should appear in the main part of the declaration, just as they would if you were not implementing an interface procedure.</span></span>  
  
 <span data-ttu-id="75c98-106">Následující kód lze vygenerovat tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="75c98-106">The following code can generate this error.</span></span>  
  
```  
Public Interface testInterface  
    Sub testSub(Of t)()  
End Interface  
Public Class testClass  
    Implements testInterface  
    Public Sub testSub() Implements testInterface.testSub(Of t)()  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="75c98-107">Předchozí deklarace `Implements` klauzule by měl vypadat jako definice rozhraní, s výjimkou možné přidání modifikátory přístupu nebo proceduru.</span><span class="sxs-lookup"><span data-stu-id="75c98-107">The declaration preceding the `Implements` clause should look like the interface definition, except for the possible addition of access or procedure modifiers.</span></span> <span data-ttu-id="75c98-108">Následující kód se vyhnete chybu.</span><span class="sxs-lookup"><span data-stu-id="75c98-108">The following code avoids the error.</span></span>  
  
```  
Public Sub testSub(Of t)() Implements testInterface.testSub  
```  
  
 <span data-ttu-id="75c98-109">**ID chyby:** BC32088</span><span class="sxs-lookup"><span data-stu-id="75c98-109">**Error ID:** BC32088</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="75c98-110">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="75c98-110">To correct this error</span></span>  
  
-   <span data-ttu-id="75c98-111">Odebrat ze seznamu argumentů typu `Implements` klauzuli.</span><span class="sxs-lookup"><span data-stu-id="75c98-111">Remove the type argument list from the `Implements` clause.</span></span>  
  
-   <span data-ttu-id="75c98-112">Pokud implementujete obecný člen rozhraní a pak vložte seznam argumentů typu v hlavní části prohlášení, předcházející `Implements` – klíčové slovo.</span><span class="sxs-lookup"><span data-stu-id="75c98-112">If you are implementing a generic member of the interface, then put the type argument list in the main part of the declaration, preceding the `Implements` keyword.</span></span>
