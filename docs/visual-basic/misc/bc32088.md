# <a name="type-arguments-unexpected"></a><span data-ttu-id="04b41-102">Neočekávané argumenty typu</span><span class="sxs-lookup"><span data-stu-id="04b41-102">Type arguments unexpected</span></span>
<span data-ttu-id="04b41-103">Klauzule `Implements` poskytuje argumenty typu pro člena rozhraní, které implementuje.</span><span class="sxs-lookup"><span data-stu-id="04b41-103">An `Implements` clause supplies type arguments for the interface member it is implementing.</span></span>  
  
 <span data-ttu-id="04b41-104">Klauzule `Implements` by měla identifikovat pouze rozhraní a člen, který implementuje.</span><span class="sxs-lookup"><span data-stu-id="04b41-104">The `Implements` clause should only identify the interface and member it is implementing.</span></span> <span data-ttu-id="04b41-105">To znamená, že pokud deklarujete obecný postup, klauzule `Of` a argumenty typu by se měly zobrazit v hlavní části deklarace stejně, jako by nedošlo k implementaci procedury rozhraní.</span><span class="sxs-lookup"><span data-stu-id="04b41-105">This means that if you are declaring a generic procedure, the `Of` clause and the type arguments should appear in the main part of the declaration, just as they would if you were not implementing an interface procedure.</span></span>  
  
 <span data-ttu-id="04b41-106">Následující kód může vygenerovat tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="04b41-106">The following code can generate this error.</span></span>  
  
```vb  
Public Interface testInterface  
    Sub testSub(Of t)()  
End Interface  
Public Class testClass  
    Implements testInterface  
    Public Sub testSub() Implements testInterface.testSub(Of t)()  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="04b41-107">Deklarace před klauzulí `Implements` by měla vypadat jako definice rozhraní, s výjimkou možných přidání přístupu nebo modifikátorů procedury.</span><span class="sxs-lookup"><span data-stu-id="04b41-107">The declaration preceding the `Implements` clause should look like the interface definition, except for the possible addition of access or procedure modifiers.</span></span> <span data-ttu-id="04b41-108">Následující kód vylučuje chybu.</span><span class="sxs-lookup"><span data-stu-id="04b41-108">The following code avoids the error.</span></span>  
  
```vb  
Public Sub testSub(Of t)() Implements testInterface.testSub  
```  
  
 <span data-ttu-id="04b41-109">**ID chyby:** BC32088</span><span class="sxs-lookup"><span data-stu-id="04b41-109">**Error ID:** BC32088</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="04b41-110">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="04b41-110">To correct this error</span></span>  
  
- <span data-ttu-id="04b41-111">Odeberte seznam argumentů typu z klauzule `Implements`.</span><span class="sxs-lookup"><span data-stu-id="04b41-111">Remove the type argument list from the `Implements` clause.</span></span>  
  
- <span data-ttu-id="04b41-112">Pokud implementujete obecného člena rozhraní, pak vložte seznam argumentů typu v hlavní části deklarace, před klíčovým slovem `Implements`.</span><span class="sxs-lookup"><span data-stu-id="04b41-112">If you are implementing a generic member of the interface, then put the type argument list in the main part of the declaration, preceding the `Implements` keyword.</span></span>
