# <a name="too-many-arguments-to-procedurename"></a>Příliš moc argumentů '\<název_procedury >'
Počet argumentů zadána ve volání procedury přesahuje počet argumentů, které jsou definovány pro proceduru.  
  
 **ID chyby:** BC30057  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte definici procedury a odebrání dalších argumentů volání.
