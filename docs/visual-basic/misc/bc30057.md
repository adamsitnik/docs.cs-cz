# <a name="too-many-arguments-to-ltprocedurenamegt"></a>Příliš moc argumentů "&lt;název_procedury&gt;.

 **ID chyby:** BC30057  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte definici procedury a odebrání dalších argumentů volání.
