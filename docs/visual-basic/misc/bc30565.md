# <a name="array-initializer-has-too-few-dimensions"></a>Inicializátor pole nemá dostatečný počet rozměrů
Neposkytli jste dost dimenzí v inicializátoru pro vaše pole.  
  
 **ID chyby:** BC30565  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte vaše inicializátor pole. Chcete-li zjistit, kolik rozměry jsou nezbytné.  
  
- Zadejte chybí argument nebo argumenty.
