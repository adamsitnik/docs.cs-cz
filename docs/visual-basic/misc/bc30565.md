# <a name="array-initializer-has-too-few-dimensions"></a>Inicializátor pole má příliš málo dimenzí.
V inicializátoru vašeho pole jste nezadali dostatek dimenzí.  
  
 **ID chyby:** BC30565  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte inicializátor pole, abyste zjistili, kolik dimenzí je potřeba.  
  
- Zadejte chybějící argument nebo argumenty.
