# <a name="parameter-ltparameternamegt-in-extension-method-ltmethodnamegt-defined-in-lttypenamegt-already-has-a-matching-omitted-argument"></a>Parametr '&lt;parametername&gt;"v metodě rozšíření"&lt;methodname&gt;"definované v"&lt;typename&gt;' již existuje odpovídající vynechaný argument
Volání procedury s metodou rozšíření vynechá argument podle pozice a pak poskytuje argument podle názvu. Například následující volání metody rozšíření `ABC` první argument pro parametr vynechá `Y`a poté podle názvu.  
  
```  
<Extension()> _  
Public Sub ABC(ByVal X As Integer, Optional ByVal Y As Byte = 0, _  
               Optional ByVal Z As Byte = 0)  
End Sub  
' . . .  
' Calling extension method ABC.  
Dim number As Integer  
' Not valid.  
' number.ABC(, 4, Y:=5)  
```  
  
 **ID chyby:** BC36583  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadání argumentů podle pozice nebo odstraňte čárku, který se vynechá.
