# <a name="registry-key-ltkeynamegt-could-not-be-found"></a>Klíč registru '&lt;keyname&gt;' nebyl nalezen

## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda název klíče.  
  
2.  Ujistěte se, že máte dostatečná oprávnění k přístupu ke klíči.
