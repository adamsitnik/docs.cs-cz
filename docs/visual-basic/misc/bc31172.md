# <a name="an-embedded-expression-may-not-be-used-here"></a>Zde nelze použít vložený výraz
Vložený výraz byl součástí nepodporované umístění v literálu XML. Všimněte si, že vložené výrazy nelze použít k určení hodnoty oboru názvů XML.  
  
 **ID chyby:** BC31172  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte z nepodporované umístění vložený výraz.
