# <a name="finally-statements-are-not-valid-in-the-immediate-window"></a>"Finally" příkazy nejsou platné v příkazovém
`Finally` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30716  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Finally` příkazy z ladění kódu.
