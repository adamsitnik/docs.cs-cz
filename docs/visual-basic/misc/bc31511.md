# <a name="attribute-member-ltmembernamegt-cannot-be-the-target-of-an-assignment-because-it-is-not-declared-public"></a>Člen atributu '&lt;membername&gt;' nemůže být cílem přiřazení, protože není deklararovaný jako 'Veřejné'
Chcete-li přiřadit hodnotu členu soukromé atributu byl proveden pokus o.  
  
 **ID chyby:** BC31511  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte přiřazení.  
  
2.  Pokud používáte vlastní atributy, které jste vytvořili, změňte atribut členský modifikátor přístupu k `Public`.
