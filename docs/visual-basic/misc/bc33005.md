# <a name="end-operator-expected"></a>Byl očekáván příkaz 'end operátor.
`End` Příkaz musí odpovídat bloku jeho ukončení. `Operator` Definici procedury nebyl správně zavřen.  
  
 **ID chyby:** BC33005  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidat `End Operator` příkaz na konec procedury.
