# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="01b2e-102">Literál XML nemůže objevit sem, pokud není uzavřen v závorkách</span><span class="sxs-lookup"><span data-stu-id="01b2e-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>
<span data-ttu-id="01b2e-103">Deklarace literálů XML je použít ve výrazu do umístění, které je nejednoznačné pro kompilátor jazyka Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="01b2e-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="01b2e-104">To znamená, kompilátor jazyka Visual Basic nelze určit, zda je menší – než znak (<) má sloužit jako srovnávací operátor nebo spuštění literál XML.</span><span class="sxs-lookup"><span data-stu-id="01b2e-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="01b2e-105">Následující kód představuje příklad:</span><span class="sxs-lookup"><span data-stu-id="01b2e-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _

                  Select element  
```  
  
 <span data-ttu-id="01b2e-106">**ID chyby:** BC31198</span><span class="sxs-lookup"><span data-stu-id="01b2e-106">**Error ID:** BC31198</span></span>
