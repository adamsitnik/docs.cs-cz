# <a name="could-not-complete-operation-since-target-directory-is-under-source-directory"></a>Operaci nelze provést, protože cílový adresář se nachází ve zdrojovém adresáři
Cyklické operace se nezdařila. Cyklické operace cyklu a proto nelze dokončit. Například může pokusit objekt A dědí z objektu B, který zase dědí z objektu A.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Při dědění, ujistěte se, že neexistují žádné cyklické odkazy.
