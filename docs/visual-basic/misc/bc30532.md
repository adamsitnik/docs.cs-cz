# <a name="conversion-from-date-to-double-requires-calling-the-datetooadate-method"></a>Převod z 'Data' na 'Double' vyžaduje volání metody 'Date.ToOADate.
Jste se pokusili přetypovat `Date` hodnota, která se `Double` hodnotu, která se nedá provést bez použití <xref:System.DateTime.ToOADate%2A?displayProperty=nameWithType> metody.  
  
 **ID chyby:** BC30532  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití <xref:System.DateTime.ToOADate%2A?displayProperty=nameWithType> způsobů, jak převést hodnotu.
