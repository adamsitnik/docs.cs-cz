# <a name="project-ltprojectnamegt-cannot-generate-a-reference-to-file-ltfilenamegt"></a>Projekt '&lt;projectname&gt;"nelze vygenerovat odkaz na soubor"&lt;filename&gt;"
Projekt ' <`projectname`>' nelze vygenerovat odkaz na soubor ' <`filename`>'. Budete muset odstranit a znovu přidat tento odkaz a pak provést úplné sestavení řešení.  
  
 Došlo k chybě při vytváření odkaz na soubor.  
  
 **ID chyby:** BC31514  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte odkaz kliknutím pravým tlačítkem myši položku z **odkazy na projekt** v **Průzkumníka řešení** a kliknete na **odebrat**.  
  
2.  Obnovit odkaz tak, že kliknete pravým tlačítkem **odkazy** v **Průzkumníka řešení** a pak levým na **přidat odkaz**.
