# <a name="type-lttypenamegt-cannot-implement-interface-ltinterfacenamegt-because-it-declares-lteventsignaturegt-which-has-a-return-type"></a>Typ '&lt;typename&gt;'nemůže implementovat rozhraní'&lt;interfacename&gt;'protože deklaruje"&lt;eventsignature&gt;' která má návratový typ.
Třídy nebo struktury se pokouší implementovat rozhraní, který deklaruje událost, která vrací hodnotu.  
  
 Visual Basic nepodporuje aktuálně deklarující události, které vracejí hodnoty.  
  
 **ID chyby:** BC30945  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Implements` příkaz z třídy nebo definice struktury nebo jiné rozhraní implementovat.
