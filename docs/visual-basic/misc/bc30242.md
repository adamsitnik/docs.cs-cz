# <a name="modifier-is-not-valid-on-a-method-declaration"></a>"\<modifikátor >' není platný v deklaraci metody
A `Function` nebo `Sub` příkaz obsahuje neplatné klíčové slovo.  
  
 **ID chyby:** BC30242  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte neplatné klíčové slovo z `Function` nebo `Sub` příkazu.
