# <a name="option-ltspecifiergt-statement-can-only-appear-once-per-file"></a>"Možnost &lt;specifikátor&gt;' příkaz může být pouze jednou na soubor
`Option Compare`, `Option Explicit`, A `Option Strict` příkazy jde použít jenom jeden čas ve stejném zdrojovém souboru.  
  
 **ID chyby:** BC30225  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat duplicitní `Option` příkazu.
