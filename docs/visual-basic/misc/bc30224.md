# <a name="is-expected"></a>Byl očekáván 'Is'
A `TypeOf` operátor probíhá, aniž by `Is` klauzuli.  
  
 **ID chyby:** BC30224  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Is` následující klauzule `TypeOf` operátor; například `TypeOf MyControl Is ComboBox`.
