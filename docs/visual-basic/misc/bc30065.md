# <a name="exit-sub-is-not-valid-in-a-function-or-property"></a>'Klíčové slovo exit Sub' není platné v proceduře Function ani Property
`Exit Sub` Zobrazí se v `Function` procedury nebo `Property` postup. `Exit` Příkaz musí odpovídat bloku ve kterém se vyskytuje.  
  
 **ID chyby:** BC30065  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahradit `Exit Sub` s `Exit Function` nebo `Exit Property` příkaz podle potřeby.
