# <a name="set-not-supported-read-only-property"></a>Metoda Set není podporována (vlastnost jen pro čtení)
Pokusili jste se nastavit nebo změnit vlastnost, která je jen pro čtení.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte odkaz na vlastnost z vašeho kódu.  
  
2.  Změňte odkaz na pouze vrátí hodnotu vlastnosti v době běhu.
