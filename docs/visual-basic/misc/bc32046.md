<span data-ttu-id="eeb5f-103">Příkaz deklarace používá [New – operátor](../../visual-basic/language-reference/operators/new-operator.md) klauzule určující typ parametru jako typ, který má být vytvořena a parametr typu je deklarovaná bez vlastnosti `New` omezení.</span><span class="sxs-lookup"><span data-stu-id="eeb5f-103">A declaration statement uses a [New Operator](../../visual-basic/language-reference/operators/new-operator.md) clause specifying a type parameter as the type to be created, and the type parameter is declared without a `New` constraint.</span></span>  
  
 <span data-ttu-id="eeb5f-104">A *omezení* na typ parametru ukládá požadavek na libovolný typ předaný argument pro parametr typu při vytvoření obecného typu.</span><span class="sxs-lookup"><span data-stu-id="eeb5f-104">A *constraint* on a type parameter imposes a requirement on any type argument passed to that type parameter when the generic type is created.</span></span> <span data-ttu-id="eeb5f-105">`New` Omezení určuje, že argument typu musí vystavit konstruktor bez parametrů, s přístupem k vytváření kódu.</span><span class="sxs-lookup"><span data-stu-id="eeb5f-105">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="eeb5f-106">Toto je co umožňuje `New` klauzule v příkazu deklarace pro vytvoření instance daného typu.</span><span class="sxs-lookup"><span data-stu-id="eeb5f-106">This is what allows a `New` clause in a declaration statement to create an instance of that type.</span></span>
