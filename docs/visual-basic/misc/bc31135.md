# <a name="specifiers-are-not-valid-on-addhandler-removehandler-and-raiseevent-methods"></a>Specifikátory nejsou platné pro metody AddHandler, RemoveHandler a RaiseEvent.
`ReadOnly` `Public` `RemoveHandler`Deklarace metod `AddHandler`, a`RaiseEvent` nelze upravovat pomocí klíčových slov, jako je například nebo. Taková klíčová slova můžou obsahovat jenom upravitelné deklarace.  
  
 **ID chyby:** BC31135  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte klíčové slovo z deklarace metody.
