# <a name="case-statements-are-not-valid-in-the-immediate-window"></a>Příkazy 'Case' nejsou platné v příkazovém
`Case` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30719  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `Case` příkazy z **okamžité** okna.
