# <a name="case-statements-are-not-valid-in-the-immediate-window"></a>Příkazy Case nejsou platné v příkazovém podokně.
příkazy `Case` lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30719  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte příkazy `Case` z příkazového **podokna** .
