# <a name="xml-comment-parameter-parameter-does-not-match-a-parameter-on-the-corresponding-keyword-statement"></a>Parametr komentáře XML '\<parametr >' neodpovídá parametru u odpovídajícího \<– klíčové slovo > – příkaz
Parametr komentáře XML \<parametr > neodpovídá parametru u odpovídajícího \<– klíčové slovo > příkaz. Komentář XML bude ignorován.  
  
 Značka komentáře XML byl zadán pro některé, ale ne všechny parametry v metodě.  
  
 **ID chyby:** BC42307  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Značka komentáře XML zadejte pro všechny parametry v metodě.
