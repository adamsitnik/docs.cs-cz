# <a name="end-addhandler-must-be-the-first-statement-on-a-line"></a>'End AddHandler' musí být prvním příkazem na řádku
`End AddHandler` Příkaz následuje dvojtečka (:) oddělovač. `End AddHandler` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC31118  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na samostatné řádky.
