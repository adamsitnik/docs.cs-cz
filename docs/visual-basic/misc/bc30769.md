# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-end-statement"></a>Cílová verze rozhraní .NET Compact Framework nepodporuje příkaz 'End'
Verze rozhraní .NET Compact Framework práci se nepodporuje `End` příkazu.  
  
 **ID chyby:** BC30769  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte příkaz.
