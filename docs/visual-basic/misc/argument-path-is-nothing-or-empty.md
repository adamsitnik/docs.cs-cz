# <a name="argument-path-is-nothing-or-empty"></a>Argument 'Path' má hodnotu Nothing nebo je prázdný
Název cesty není platný, protože je inicializován na `Nothing` nebo je prázdný. Dráha musí mít určitý datový typ. a platný název.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte platný název cesty.
