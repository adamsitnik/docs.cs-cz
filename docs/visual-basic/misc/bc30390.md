# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>"\<qualifiedmethod >' není v tomto kontextu dostupná, protože je '\<modifikátor >"
Pokusili jste se přístup k metodě, která není přístupná v tomto kontextu, protože modifikátor, který byl použit k němu.  
  
 **ID chyby:** BC30390  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte modifikátor.
