# <a name="ltqualifiedmethodgt-is-not-accessible-in-this-context-because-it-is-ltmodifiergt"></a>"&lt;qualifiedmethod&gt;"není v tomto kontextu dostupná vzhledem k tomu, že je"&lt;modifikátor&gt;.
Pokusili jste se přístup k metodě, která není přístupná v tomto kontextu, protože modifikátor, který byl použit k němu.  
  
 **ID chyby:** BC30390  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte modifikátor.
