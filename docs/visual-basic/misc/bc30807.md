# <a name="let-and-set-assignment-statements-are-no-longer-supported"></a>"Let" a "Nastavení" přiřazovací příkazy již nejsou podporovány.
`Let` a `Set` nelze použít v přiřazeních.  
  
 **ID chyby:** BC30807  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Vynechat `Let` a `Set` z přiřazovací příkazy; například:  
  
     `X = 2`
