# <a name="ltspecifiergt-is-not-valid-on-an-enum-declaration"></a>"&lt;specifikátor&gt;' není platný v deklaraci výčtu
Jste už použili, který není platný na specifikátoru `Enum` deklarace.  
  
 **ID chyby:** BC30396  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte specifikátor.
