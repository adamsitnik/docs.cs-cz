# <a name="paramarray-parameter-must-be-a-one-dimensional-array"></a>Parametrem ParamArray musí být jednorozměrné pole
A `ParamArray` argument musí být deklarován jako typ jednorozměrné pole. Určuje více než jednou dimenzí pro deklarace `ParamArray` argument.  
  
 **ID chyby:** BC30051  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebere všechny mezery uvnitř závorek `ParamArray` název argumentu.
