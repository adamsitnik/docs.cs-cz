# <a name="end-set-must-be-the-first-statement-on-a-line"></a>'End Set' musí být prvním příkazem na řádku
`End Set` Příkaz následuje dvojtečka (:) oddělovač. `End Set` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC32034  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na samostatné řádky.
