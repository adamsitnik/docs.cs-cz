# <a name="-requires-its-left-operand-to-have-a-type-parameter-class-or-interface-type-but-this-operand-has-the-type-type"></a>'!' vyžaduje, aby levý operand byl parametr typu, typ třídy nebo rozhraní, ale tento operand má typ'\<typu > '
Přístup ke členu slovník, který používá vykřičník (`!`) namísto tečku (`.`), je platný pouze pro třídu nebo rozhraní.  
  
 **ID chyby:** BC30103  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte výraz na levé straně vykřičník s jedním, který je vyhodnocen s definicí třídy nebo typu rozhraní.
