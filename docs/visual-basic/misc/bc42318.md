# <a name="xml-comment-type-parameter-must-have-a-name-attribute"></a>Parametr typu komentáře XML musí mít atribut name.
Parametr typu komentáře XML musí mít atribut 'name'. Komentář XML bude ignorován.  
  
 Značka komentáře XML vyžaduje `name` atribut, který chybí nebo není úplná.  
  
 **ID chyby:** BC42318  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat nebo dokončete požadované `name` atribut ke značce.
