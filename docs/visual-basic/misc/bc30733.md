# <a name="ltmembernamegt-is-already-declared-by-ltprocedurenamegt-which-was-generated-for-this-ltcontainernamegt"></a>"&lt;membername&gt;"je již deklarován"&lt;název_procedury&gt;", který se vygeneroval pro tuto &lt;containername&gt;
Člen se deklaroval, který má stejný název jako člen, který je deklarován comom modul runtime jazyka.  
  
 **ID chyby:** BC30733  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte název členu, který chcete odebrat tento konflikt.
