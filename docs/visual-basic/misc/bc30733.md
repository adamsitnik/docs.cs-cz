# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>"\<membername >' již byla deklarována ve"\<název_procedury > ", který se vygeneroval pro tuto \<containername >

Člen byl deklarován, který má stejný název jako člen, který je deklarován modulem common language runtime.

**ID chyby:** BC30733

## <a name="to-correct-this-error"></a>Oprava této chyby

- Změňte název členu, který chcete odebrat tento konflikt.
