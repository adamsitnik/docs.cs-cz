# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<member > ' je již deklarován '\<procedury > ', která byla vygenerována pro tento \<ContainerName >

Byl deklarován člen, který má stejný název jako člen deklarovaný modulem CLR (Common Language Runtime).

**ID chyby:** BC30733

## <a name="to-correct-this-error"></a>Oprava této chyby

- Chcete-li odstranit konflikt, změňte název člena.
