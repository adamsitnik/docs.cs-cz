<span data-ttu-id="8bf8c-103">Obecné rozhraní dědí ze dvou nebo více obecných rozhraní a dvě dědičnost by mohly kolidovat pro určité hodnoty argumentů typu.</span><span class="sxs-lookup"><span data-stu-id="8bf8c-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>

 <span data-ttu-id="8bf8c-104">Následující příkazy mohou vygenerovat tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="8bf8c-104">The following statements can generate this error.</span></span>

```vb

    Inherits interfaceA(Of v)
End Interface
Public Interface interfaceY(Of w)
    Inherits interfaceA(Of w)

Public Interface derivedInterface(Of t1, t2)
    Inherits interfaceX(Of t1), interfaceY(Of t2)
End Interface
```

<span data-ttu-id="8bf8c-105">Pokud `derivedInterface` je vytvořena nebo implementováno poskytnutí stejného typu pro obojí `t1` a `t2`, musí dědit dvě verze `interfaceA` s identickými argumenty typu.</span><span class="sxs-lookup"><span data-stu-id="8bf8c-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="8bf8c-106">Tím by vznikla nejednoznačnost, o které verzi přistupuje.</span><span class="sxs-lookup"><span data-stu-id="8bf8c-106">Doing so would produce an ambiguity about which version to access.</span></span>
