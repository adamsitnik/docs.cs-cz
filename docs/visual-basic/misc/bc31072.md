# <a name="warning-treated-as-error-error"></a>Upozornění bylo zpracováno jako chyba: \<chyby >
Při spuštění sestavení zkompilovaná došlo k upozornění **zpracovávat upozornění jako chyby** možnost nastavená na on.  
  
 **ID chyby:** BC31072  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Opravte upozornění.  
  
2.  Kompilace sestavení **zpracovávat upozornění jako chyby** možnost vypnuta.
