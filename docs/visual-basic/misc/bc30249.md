# <a name="-expected-declaration"></a>očekává se znak = (deklarace).
`Mid` Příkaz nebodirektivakompilace`#Const` nezahrnuje přiřazení hodnoty.  
  
 **ID chyby:** BC30249  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte znak rovná se (`=`) a hodnotu, která má být přiřazena řetězci nebo konstantě kompilace.
