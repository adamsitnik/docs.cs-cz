# <a name="-expected-declaration"></a>"=" očekáván (deklarace)
A `Mid` příkazu nebo `#Const` direktivy kompilace nezahrnuje přiřazení hodnoty.  
  
 **ID chyby:** BC30249  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte znaménko rovná se (`=`) a hodnota má být přiřazena k řetězci nebo konstantu kompilace.
