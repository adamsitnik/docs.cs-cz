# <a name="unable-to-parse-xml-lterrorgt"></a>Nepovedlo se parsovat kód XML: &lt;chyba&gt;
V souboru XML, který je používán dojde k chybě.  
  
 **ID chyby:** BC31023  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ověřte, že kód XML je správně vytvořený, platný a že není poškozený soubor.
