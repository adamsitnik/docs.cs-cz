# <a name="unable-to-parse-xml-error"></a>Nepovedlo se parsovat kód XML: \<chyby >
V souboru XML, který je používán dojde k chybě.  
  
 **ID chyby:** BC31023  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, že kód XML je správně vytvořený, platný a že není poškozený soubor.
