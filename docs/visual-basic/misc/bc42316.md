# <a name="xml-documentation-parse-error-start-tag-tag-doesnt-have-a-matching-end-tag"></a>Chyba analýzy dokumentace XML: Počáteční značka '\<značka >' nemá odpovídající koncová značka
Chyba analýzy dokumentace XML: Počáteční značka \<značka > nemá koncová značka. Komentář XML bude ignorován.  
  
 Komentář XML obsahuje počáteční značku, ale neobsahuje odpovídající koncová značka.  
  
 **ID chyby:** BC42316  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte se koncová značka odpovídající počáteční značce.  
  
     – nebo –  
  
- Pokud značka neobsahuje žádný vnitřní text, například [ \<seealso >](../../visual-basic/language-reference/xmldoc/seealso.md), zadejte lomítkem před ukončovací lomená závorka.
