# <a name="option-optionname-can-be-followed-only-by--or--"></a>možnost \<název_možnosti > může následovat pouze '+' nebo '-'
Text, který následuje možnost není znaménko plus (+) nebo mínus (-).  
  
 **ID chyby:** BC2009  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte text, který následuje možnost zapnout možnost se znaménkem plus (+).  
  
     – nebo –  
  
- Nahraďte text, který následuje možnost znaménko minus (-) Chcete-li možnost vypnout.
