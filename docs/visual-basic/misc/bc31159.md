# <a name="expected-closing-gt-for-embedded-expression"></a>Bylo očekáváno ukončení. %&gt;"pro vložený výraz
Vložený výraz v literálu XML chybí požadovaný ukončovací `%>` znaků.  
  
 **ID chyby:** BC31159  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte požadované uzavírací `%>` znaky na konec vložený výraz.
