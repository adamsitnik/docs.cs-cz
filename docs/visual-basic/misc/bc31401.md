# <a name="static-local-variable-ltvariablenamegt-is-already-declared"></a>Statická lokální proměnná '&lt;NázevProměnné&gt;' již byla deklarována
Další statické místní proměnné se stejným názvem již byla deklarována.  
  
 **ID chyby:** BC31401  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte redundantní statické místní deklarace.  
  
2.  Zadejte jedinečný název každé statické místní proměnné.
