# <a name="suspend-request-is-pending"></a>Požadavek na pozastavení čeká na vyřízení.
V ladicím programu sady Visual Studio určuje výraz volání procedury, ale existuje žádost o pozastavení vlákna.  
  
 Ladicí program se nepokouší volat proceduru v neaktivním vlákně.  
  
 **ID chyby:** BC30947  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pokud je to možné, určete zdroj žádosti o pozastavení vlákna, takže se můžete vyhnout tomu, aby se znovu stalo.  
  
- Ukončí a znovu spustí ladění.
