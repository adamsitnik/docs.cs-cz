# <a name="sub-or-function-expected"></a>Byl očekáván 'Sub' nebo 'Function'
A `Declare` příkaz neurčuje buď `Sub` nebo `Function`. `Declare` identifikuje externí procedura a určit, jaký druh procedury.  
  
 **ID chyby:** BC30215  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat buď `Sub` nebo `Function` klíčové slovo `Declare` příkazu.
