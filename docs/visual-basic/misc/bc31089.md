# <a name="types-declared-private-must-be-inside-another-type"></a>Typy s deklarací 'Private' musí být uvnitř jiného typu
`Private` Byl použit modifikátor typu není uvnitř jiného typu.  
  
 **ID chyby:** BC31089  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Použít méně omezující modifikátor přístupu, například `Friend`.  
  
2. Deklarujte typ v rámci jiného typu.
