# <a name="argument-per-is-not-valid"></a>Argument "Na" je neplatný
Argument zadaný pro `Per`, která musí být `Double` zadání platebních období v rozsahu 1 až `NPer`, není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že argument spadá v platném rozsahu.
