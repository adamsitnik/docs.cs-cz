# <a name="byval-and-byref-cannot-be-combined"></a>'ByVal' a 'ByRef' nelze kombinovat
`ByVal` a `ByRef` klíčová slova se vzájemně vylučují.  
  
 **ID chyby:** BC30641  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte buď `ByVal` nebo `ByRef`, ale ne obojí.
