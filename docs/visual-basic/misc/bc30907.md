# <a name="this-inheritance-causes-circular-dependencies-between-lttype1gt-lttypename1gt-and-its-nested-lttype2gt-lttypename2gt"></a>Toto dědění způsobuje cyklické závislosti mezi &lt;type1&gt; "&lt;NázevTypu1&gt;" a vnořeným &lt;type2&gt; "&lt;NázevTypu2&gt;.
Struktury dědičnosti za následek cyklickou závislost mezi vnořené třídy, tedy dvě třídy, které dědí od sebe navzájem.  
  
 Následující kód lze vygenerovat tuto chybovou zprávu.  
  
```  
Public Class c1  
    Inherits c3.c4  
    Public Class c2  
    End Class  
End Class  
Public Class c3  
    Inherits c1.c2  
    Public Class c4  
    End Class  
End Class  
```  
  
 V předchozím kódu třídy `c1` dědí z třídy `c4`, ale `c4` je vnořená do `c3`, který dědí z `c2`vnořené uvnitř `c1`.  
  
 **ID chyby:** BC30907  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změna struktury dědičnosti tak, aby se žádné cyklické závislosti.
