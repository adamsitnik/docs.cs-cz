# <a name="readonly-attribute-property-propertyfield-cannot-be-the-target-of-an-assignment"></a>Vlastnost ' propertyfield > ' atributu\<' ReadOnly ' nemůže být cílem přiřazení.
Byl proveden pokus o přiřazení hodnoty k `ReadOnly` vlastnosti v atributu.  
  
 **ID chyby:** BC31501  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte příkaz pro přiřazení vlastnosti.  
  
2. Pokud používáte vlastnosti, které jste vyvinuli `ReadOnly` , `Shared` odeberte modifikátory nebo z vlastnosti atributu.
