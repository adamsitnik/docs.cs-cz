# <a name="readonly-attribute-property-ltpropertyfieldgt-cannot-be-the-target-of-an-assignment"></a>Atribut 'ReadOnly' vlastnosti '&lt;propertyfield&gt;' nemůže být cílem přiřazení.
Byl proveden pokus o přiřazení hodnoty k `ReadOnly` vlastnost v atributu.  
  
 **ID chyby:** BC31501  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte příkaz přiřazení vlastnosti.  
  
2.  Pokud pomocí vlastnosti, které jste vytvořili, odeberte `ReadOnly` nebo `Shared` modifikátory z vlastnost atributu.
