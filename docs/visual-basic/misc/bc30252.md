# <a name="end-interface-must-be-preceded-by-a-matching-interface"></a>'End Interface' musí předcházet odpovídající 'rozhraní'
`End Interface` Vyvolá příkaz bez odpovídající `Interface` příkazu. `End Interface` musí předcházet odpovídající `Interface` příkazu.  
  
 **ID chyby:** BC30252  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, jestli předchozí `Interface` příkaz je chybně napsaná nebo jinak neplatných.  
  
2.  Ujistěte se, že `Interface` členy jsou správně naformátované.
