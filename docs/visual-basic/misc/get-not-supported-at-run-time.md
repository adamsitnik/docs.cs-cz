# <a name="get-not-supported-at-run-time"></a>Metoda Get není podporována v době běhu
Pokusili jste se číst vlastnost v době běhu, který je přístupný jenom v době návrhu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte vlastnosti a určit, za jakých podmínek je možné nastavit.  
  
2. Odeberte odkaz na vlastnost.
