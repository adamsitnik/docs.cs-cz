# <a name="ltspecifiergt-is-not-valid-on-a-constant-declaration"></a>"&lt;specifikátor&gt;' není platný v deklaraci konstanty
A `Const` příkaz obsahuje neplatné klíčové slovo. A `Const` výraz může obsahovat pouze `Friend`, `Private`, `Protected`, `Public`, nebo `Shadows` klíčová slova.  
  
 **ID chyby:** BC30233  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že klíčové slovo není napsaný špatně.  
  
2.  Odeberte neplatné klíčové slovo z `Const` příkazu.
