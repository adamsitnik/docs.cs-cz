<span data-ttu-id="42339-103">Datové typy parametrů typu v metodě rozšíření "\<methodname >" definované v "\<typename >' nelze odvodit z těchto argumentů, protože je přípustný více než jeden typ.</span><span class="sxs-lookup"><span data-stu-id="42339-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="42339-104">Zadání dat explicitně podaří k této chybě.</span><span class="sxs-lookup"><span data-stu-id="42339-104">Specifying the data type(s) explicitly might correct this error.</span></span>  
  
 <span data-ttu-id="42339-105">Byl proveden pokus o použití odvození typu k určení typu (nebo typy), parametr typu (nebo parametry) ve volání metody obecných rozšíření.</span><span class="sxs-lookup"><span data-stu-id="42339-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="42339-106">Kompilátor najde více než jeden typ. možná data pro jeden nebo více parametrů typu a ohlásí tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="42339-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

    Sub Main()

        '' Not valid.
