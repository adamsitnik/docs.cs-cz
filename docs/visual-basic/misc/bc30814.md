# <a name="gosub-statements-are-no-longer-supported"></a>Příkazy "GoSub" již nejsou podporovány.
`GoSub` nelze použít k volání procedury.  
  
 **ID chyby:** BC30814  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Volání procedur přímo bez použití `GoSub`; například:  
  
    ```  
    CalculateInterest(Amount, Rate, Time)  
    ```
