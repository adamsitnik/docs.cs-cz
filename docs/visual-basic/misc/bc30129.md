# <a name="assembly-attribute-ltattributenamegt-is-not-valid-lterrorgt"></a>Atribut sestavení "&lt;attributename&gt;' není platný: &lt;chyba&gt;
Modul common language runtime nepřijímá atribut v sestavení.

**ID chyby:** BC30129

## <a name="to-correct-this-error"></a>Oprava této chyby

1. Atribut z deklarace ho odeberte nebo nahraďte atributem, který je platný pro daný kontext.

2. Zkontrolujte v uvozovkách chybovou zprávu a proveďte příslušnou akci.
