# <a name="expected-closing--for-xml-processor-instruction"></a>Bylo očekáváno ukončení '? >' instrukce procesoru XML
XML literál chybí požadovaný ukončovací instrukce ke zpracování `?>` znaků.  
  
 **ID chyby:** BC31160  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte požadované uzavírací `?>` znaky na konec instrukce zpracování XML literál.
