# <a name="type-parameter-lttypeparameternamegt-can-only-have-one-constraint-that-is-a-class"></a>Parametr typu '&lt;typeparametername&gt;"může mít jenom jedno omezení, které je třídou
Seznam omezení obsahuje více než jednu třídu.  
  
 Seznam omezení pro parametr typu může přijmout libovolný počet rozhraní, ale maximálně jednu třídu. Argument typu zadaný pro, že parametr typu musí dědit z třídy a Visual Basic nepodporuje vícenásobnou dědičnost.  
  
 **ID chyby:** BC32047  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Vyberte jednu třídu a obsahovat pouze třídy v seznamu omezení.  
  
-   Je možné definovat další typy parametrů za účelem přizpůsobení třídy nebo třídy, které nelze zahrnout do tohoto seznamu omezení.
