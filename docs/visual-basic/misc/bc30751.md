# <a name="unable-to-get-type-information-for-typename"></a>Nepovedlo se získat informace o typu '\<typename >'
Typ odkazovalo, který není načtený modul common language runtime.  
  
 **ID chyby:** BC30751  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změnit strukturu kódu tak, že typ je definován v aktuálním oboru.
