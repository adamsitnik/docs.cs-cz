# <a name="unable-to-get-type-information-for-typename"></a>Nepovedlo se získat informace o typu pro\<TypeName >.
Odkazovalo se na typ, který nebyl načten modulem CLR (Common Language Runtime).  
  
 **ID chyby:** BC30751  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Restrukturalizujte kód tak, aby byl definován typ v rámci aktuálního oboru.
