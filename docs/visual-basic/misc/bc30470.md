# <a name="myclass-cannot-be-used-outside-of-a-class"></a>"MyClass" nelze použít vně třídy
Použili jste `MyClass` – klíčové slovo mimo strukturu třídy.  
  
 **ID chyby:** BC30470  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunout `MyClass` deklarace do struktury vaší třídy.
