# <a name="myclass-cannot-be-used-outside-of-a-class"></a>Klíčové slovo MyClass nejde použít vně třídy.
Použili `MyClass` jste klíčové slovo vně struktury třídy.  
  
 **ID chyby:** BC30470  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- `MyClass` Přesuňte deklaraci do struktury vaší třídy.
