# <a name="event-lteventnamegt-event-specified-by-the-defaultevent-attribute-is-not-a-publicly-accessible-event-for-this-class"></a>Událost '&lt;eventname&gt;"událost určenému atributem"DefaultEvent"není veřejně přístupná událost pro tuto třídu
<xref:System.ComponentModel.DefaultEventAttribute> Atribut musíte zadat název veřejně přístupná událost ve třídě, na který se atribut používá.  
  
 **ID chyby:** BC30770  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že třída definuje veřejně přístupná událost.  
  
2.  Ujistěte se, že název události ve třídě odpovídá název určený <xref:System.ComponentModel.DefaultEventAttribute> atribut.
