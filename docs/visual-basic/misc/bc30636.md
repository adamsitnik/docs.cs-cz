# <a name="gt-expected"></a>"&gt;' byl očekáván
Syntaxe pro zadaný kód není správná. Tato chyba často znamená chybí ukončovací lomená závorka pro atribut nebo literál XML elementu.  
  
 **ID chyby:** BC30636  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte syntaxi pro kód používá.
