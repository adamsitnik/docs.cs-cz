# <a name="shared-cannot-be-combined-with-ltspecifiergt-on-a-method-declaration"></a>"Sdílené" nelze kombinovat s "&lt;specifikátor&gt;' v deklaraci metody
Pokusili jste se zkombinovat `Shared` modifikátor se specifikátorem, například `Overridable`, `NotOverridable`, nebo `MustOverride`, která je v kombinaci, v deklaraci metody není platný.  
  
 **ID chyby:** BC30501  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
