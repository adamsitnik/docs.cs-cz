# <a name="attribute-value-is-not-valid-expecting-value"></a>Hodnota atributu není platná. byl očekáván 'value'
Byla zadána neplatná hodnota pro atribut v dokumentu XML literál. Literál dokumentu XML platný atribut hodnoty jsou uvedeny v specifikace XML 1.0.  
  
 **ID chyby:** BC31181  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Aktualizujte hodnotu atributu na platnou hodnotu, jak je uvedeno v specifikace XML 1.0.
