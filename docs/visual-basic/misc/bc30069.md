# <a name="for-loop-control-variable-variablename-already-in-use-by-an-enclosing-for-loop"></a>Řídicí proměnná smyčky for '\<NázevProměnné >' již v nadřazeném cyklu for použití
Vnořené `For` smyčky všechny musí používat proměnné jedinečný ovládacího prvku.  
  
 **ID chyby:** BC30069  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte jednu z proměnné názvy ovládacích prvků.
