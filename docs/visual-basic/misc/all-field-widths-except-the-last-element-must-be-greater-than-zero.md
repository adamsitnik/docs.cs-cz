# <a name="all-field-widths-except-the-last-element-must-be-greater-than-zero"></a>Šířky všech polí, s výjimkou posledního prvku, musí být větší než nula.
Šířky všech polí, s výjimkou posledního prvku, musí být větší než nula. Šířka pole, menší nebo rovna hodnotě nula. v posledním prvkem označuje, že poslední pole má proměnlivou délku.  
  
 Operace se nezdařila, protože šířku pole kromě posledního prvku je nastavená na nulovou nebo méně. Šířka pole menší než nebo rovna hodnotě nula. je možné jako poslední prvek, označuje, že poslední pole proměnné délky, ale nelze použít jako libovolný element.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nastavte šířku pole na správnou délku.
