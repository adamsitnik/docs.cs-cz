# <a name="class-in-an-interface-cannot-be-declared-specifier"></a>Třídy v rozhraní nejde použít deklaraci '\<specifikátor > "
Byl zadán neplatný Modifikátor třídy deklarované v rozhraní.  
  
 **ID chyby:** BC31070  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebere modifikátory například `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, nebo `Overridable`.
