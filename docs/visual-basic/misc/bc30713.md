# <a name="exit-statements-are-not-valid-in-the-immediate-window"></a>Příkazy 'Exit' nejsou platné v příkazovém
`Exit` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30713  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `Exit` příkazy z ladění kódu.
