# <a name="exit-statements-are-not-valid-in-the-immediate-window"></a>Příkazy Exit nejsou platné v příkazovém podokně.
příkazy `Exit` lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30713  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Z ladicího kódu odeberte příkazy `Exit`.
