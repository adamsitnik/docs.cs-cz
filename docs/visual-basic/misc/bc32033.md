# <a name="end-get-must-be-the-first-statement-on-a-line"></a>'End Get' musí být prvním příkazem na řádku
`End Get` Příkaz následuje dvojtečka (:) oddělovač. `End Get` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC32033  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Oddělte více příkazů na samostatné řádky.
