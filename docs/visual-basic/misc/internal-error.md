# <a name="internal-error-visual-basic"></a>Vnitřní chyba (Visual Basic)
V jazyce Visual Basic došlo k interní selhání.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že k této chybě negenerovaly `Error` příkazu nebo `Raise` metody. Pokud byla, kontaktujte Microsoft Product Support Services hlášení podmínky, za kterých zprávy zobrazovaly.
