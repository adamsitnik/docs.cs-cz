# <a name="class-ltclassnamegt-cannot-be-indexed-because-it-has-no-default-property"></a>Třídy&lt;classname&gt;"nelze indexovat, protože nemá žádnou výchozí vlastnost
Se nepodařilo zadat výchozí vlastnost pro třídu.  
  
 **ID chyby:** BC30367  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte vlastnost jako výchozí vlastnost spuštěním deklaruje se `Default` – klíčové slovo.
