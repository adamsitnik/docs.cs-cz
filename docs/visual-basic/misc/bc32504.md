# <a name="microsoftvisualbasiccomclassattribute-cannot-be-applied-to-ltclassname1gt-because-its-container-ltclassname2gt-is-not-declared-public"></a>'Microsoft.VisualBasic.ComClassAttribute' nejde použít u "&lt;classname1&gt;" protože její kontejner "&lt;classname2&gt;' není deklarována jako 'Veřejné'
Třídy pomocí `COMClassAttribute` bloku atributu je deklarované uvnitř třídy, která není `Public`. Pokud třída je vystavena jako objekt modelu COM, jeho hierarchie celého obsahu musí být deklarován s `Public` přístup.  
  
 **ID chyby:** BC32504  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Deklarovat všechny třídy obsahující `Public`, nebo odeberte `COMClassAttribute` bloku atributu.
