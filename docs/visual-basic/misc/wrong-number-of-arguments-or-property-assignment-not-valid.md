# <a name="wrong-number-of-arguments-or-property-assignment-not-valid"></a>Nesprávný počet argumentů nebo přiřazení vlastnosti není platná
Byla provedena přiřazení, který není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte, že počet argumentů, které jste zadali, které odpovídá počtu požadovaného cíle.  
  
2. Zkontrolujte přiřazení vlastnosti.
