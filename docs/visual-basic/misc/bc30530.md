# <a name="parameter-cannot-have-the-same-name-as-its-defining-function"></a>Parametr nemůže mít stejný název jako jeho funkce, která definuje
Použili jste se stejným názvem v parametru i jeho funkce, která definuje.  
  
 **ID chyby:** BC30530  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte názvy parametrů v rámci této funkce a změnit název konfliktní parametru.
