# <a name="constants-must-have-a-value"></a>Konstanty musí mít hodnotu.
Deklaraci konstanty bez jeho přiřazení hodnoty.  
  
 **ID chyby:** BC30438  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přiřadíte hodnotu konstanty.
