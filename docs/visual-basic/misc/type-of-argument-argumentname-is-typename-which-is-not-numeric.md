# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>Typ argumentu '\<název argumentu argumentname >' je "\<typename >", což není numerický

## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že je číselný argument.
