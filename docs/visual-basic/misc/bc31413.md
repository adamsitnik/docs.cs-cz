# <a name="member-implicitly-defined-for-eventname-cannot-shadow-a-mustoverride-method-in-the-base-class-classname"></a>'\<člena >', implicitně definovaný pro'\<eventname >', nemůže zastiňovat metodu 'MustOverride' v základní třídě \<třídy > '\<classname >'
Zadanou událost implicitně deklaruje člen se stejným názvem jako deklarovat metodu `MustOverride` modifikátor.  
  
 **ID chyby:** BC31413  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `MustOverride` modifikátor na metodu v základní třídě nebo poskytněte vlastnost nebo metoda jedinečný název.
