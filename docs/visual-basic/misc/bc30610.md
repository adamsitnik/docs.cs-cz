# <a name="class-ltclassnamegt-must-either-be-declared-mustinherit-or-override-the-following-inherited-mustoverride-members-ltmembernamesgt"></a>Třídy&lt;classname&gt;se musí buď mít deklaraci MustInherit "–" nebo přepsat následující zděděné členy 'MustOverride': &lt;membername(s)&gt;
Třídy odvozené od základní třídy, které obsahují `MustOverride` členy musí přepsat tyto členy nebo použít `MustInherit` modifikátor.  
  
 **ID chyby:** BC30610  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `MustInherit` modifikátoru v definici třídy.  
  
-   Deklarovat pomocí přepsání `Overrides` – klíčové slovo.
