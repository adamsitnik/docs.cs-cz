# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-latebound-overload-resolution"></a>Cílová verze rozhraní .NET Compact Framework nepodporuje rozpoznání přetížené
Cílová verze rozhraní .NET Compact Framework nepodporuje řešení přetížení s pozdní vazbou.  
  
 **ID chyby:** BC30764  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Parametry typu objektu na příslušný typ přetypování.
