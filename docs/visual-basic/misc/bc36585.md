# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>'\<elementname >' není parametr metody rozšíření '\<methodname >' definované v '\<typename >'
Volání procedury poskytuje další argument, který není definován metodou rozšíření.  
  
 **ID chyby:** BC36585  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nadbytečný argument odeberte ze seznamu argumentů.
