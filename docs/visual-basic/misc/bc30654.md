# <a name="return-statement-in-a-function-get-or-operator-must-return-a-value"></a>Příkaz "Return" ve funkci, Get nebo Operator musí vracet hodnotu
`Return` příkazy musí lze použít k vrácení hodnoty volání procedury. Nemůžete použít `Return` příkazy samy ovládací prvek programu toku.  
  
 **ID chyby:** BC30654  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zadejte hodnotu, která vrací funkce nebo procedury.  
  
2. Použití `End` příkaz způsobí program ukončit aktuální proceduře.
