# <a name="import-of-type-typename-from-assembly-or-module-name-failed"></a>Import typu '\<typename >' ze sestavení nebo modul '\<název >' se nezdařilo
Ujistěte se, že jsou všechny odkazované knihovny navzájem kompatibilní.  
  
 **ID chyby:** BC31091  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, že nejsou poškozené soubory obsahující odkazovaného typu.
