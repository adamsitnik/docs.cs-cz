# <a name="too-few-type-arguments-to-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>Nedostatečný počet argumentů metody rozšíření '&lt;methodname&gt;"definované v"&lt;typename&gt;.
Obecný rozšiřující metoda byla vyvolána s menším počtem argumentů, než je počet parametrů typu v definici metody.  
  
 Při vyvolání obecné metody, je nutné zadat jeden argument typu pro jednotlivé parametry typu určené metody.  
  
 **ID chyby:** BC36590  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidání argumentů typu na seznam argumentů typu tak, aby se jeden pro každý parametr typu metody rozšíření, která jsou volání.
