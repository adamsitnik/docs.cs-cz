# <a name="propertyname-implicitly-defines-membername-which-conflicts-with-a-member-of-the-same-name-in-type-typename"></a>"\<propertyname >" implicitně definuje "\<členské jméno >", který je v konfliktu se členem stejného názvu v \<typu > "\<typename >"
Název Typ člena je v konfliktu s názvem členu, který se implicitně vytvářejí pro vlastnost. Vlastnosti implicitně vytvořit několik implicitní proměnné, které může dojít ke konfliktu s členy definované.  
  
 **ID chyby:** BC31060  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přejmenujte explicitně deklarované členské odebrat konflikt názvů.
