# <a name="procedurename-has-no-parameters-and-its-return-type-cannot-be-indexed"></a>"\<název_procedury >' nemá žádné parametry a její typ vrácené hodnoty nelze indexovat.
Volání `Function` nebo `Sub` postup poskytuje jeden nebo více argumentů, ale postup nepřijímá žádné argumenty a její typ vrácené hodnoty (Pokud se jedná `Function`) není typem pole.  
  
 **ID chyby:** BC32016  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Argument nebo argumenty odeberte z volání procedury.
