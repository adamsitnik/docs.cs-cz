# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>Metoda "\<methodname >" nelze deklarovat "Partial, protože pouze jednu metodu '\<methodname >" může být označený "Části"
Nemůžete mít dvě deklarace částečné metody, které mají stejný název a signaturu.  
  
 **ID chyby:** BC31433  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změňte název nebo podpis deklarace částečné metody.
