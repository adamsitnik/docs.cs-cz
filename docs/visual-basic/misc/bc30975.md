# <a name="for-loop-control-variable-already-in-use-by-an-enclosing-for-loop"></a>Řídicí proměnná smyčky for již v nadřazeném smyčce pro použití pro
Vnořené `For` všechny smyčky musí používat proměnné jedinečný ovládacího prvku.  
  
 **ID chyby:** BC30975  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte jednu z proměnné názvy ovládacích prvků.
