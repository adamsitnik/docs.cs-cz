# <a name="the-addressof-expression-has-no-effect-in-this-context-because-the-method-argument-to-addressof-requires-a-relaxed-conversion-to-the-delegate-type-of-the-event"></a>Výraz 'AddressOf' nemá žádný účinek v tomto kontextu, protože argument metody pro výraz 'AddressOf' vyžaduje volný převod na typ delegáta události
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Výrazu 'AddressOf' přiřadit proměnné a proměnnou použijte k přidání nebo odebrání metody jako obslužnou rutinu.
