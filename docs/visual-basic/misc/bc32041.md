# <a name="type-character-cannot-be-used-in-a-type-parameter-declaration"></a>Znak typu nejde použít v deklaraci parametru typu
Deklarace parametru typu obsahuje alespoň jeden znak typu identifikátoru.  
  
 Typ parametru obecného typu musí být platný název jazyka Visual Basic. Povolené znaky nesmí obsahovat žádný z identifikátor – znaky typu (`%`, `&`, `@`, `!`, `#`, a `$`). Zobrazit [deklarované názvy elementů](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **ID chyby:** BC32041  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte – znak typu identifikátoru nebo znaky z deklarace parametru typu.
