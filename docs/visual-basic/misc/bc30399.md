# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>'MyBase' nelze použít s metodou '\<methodname >' protože se deklaruje 'MustOverride.
Pokusili jste se použít `MyBase` metodou, která byla deklarována `MustOverride`.  
  
 **ID chyby:** BC30399  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `MustOverride` deklarace.
