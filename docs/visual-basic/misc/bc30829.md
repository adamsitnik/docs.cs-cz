# <a name="get-statements-are-no-longer-supported"></a>Příkazy 'Get' již nejsou podporovány.
`Get` příkazy jsou již nejsou podporovány. Vstupně-výstupní funkce jsou dostupné v `Microsoft.VisualBasic` oboru názvů.  
  
 `Get` není podporována pro operace se soubory a jde použít jenom v syntaxi procedury vlastností.  
  
 **ID chyby:** BC30829  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Provedení operace se soubory pomocí členy `System.IO`, `FileSystemObject`a funkce za běhu jazyka Visual Basic.
