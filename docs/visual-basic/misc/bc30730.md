# <a name="methods-declared-overrides-cannot-be-declared-overridable-because-they-are-implicitly-overridable"></a>Metody s deklarací 'Přepsání' nelze použít deklaraci 'Overridable' protože již jsou přepsatelné
Na rozdíl od většiny metody, metody označené `Overrides` modifikátor jsou přepisovatelným ve výchozím nastavení.  
  
 **ID chyby:** BC30730  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Vynechat, nechte `Overridable` označené modifikační z metod `Overrides` modifikátor.
