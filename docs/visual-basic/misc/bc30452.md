# <a name="operator-ltoperatornamegt-is-not-defined-for-types-lttype1gt-and-lttype2gt"></a>Operátor '&lt;operatorname&gt;"není definován pro typy"&lt;type1&gt;"a"&lt;type2&gt;.
Pokusili jste se pomocí binárního operátoru provádět operace, která není platná pro dvě hodnoty vrácením prvky kódu potřebný.  
  
 **ID chyby:** BC30452  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte dva prvky a ujistěte se, že jsou kompatibilní.
