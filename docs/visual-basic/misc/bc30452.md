# <a name="operator-operatorname-is-not-defined-for-types-type1-and-type2"></a>Operátor '\<operatorname >' není definován pro typy\<type1 > "a"\<type2 > "
Pokusili jste se pomocí binárního operátoru provádět operace, která není platná pro dvě hodnoty vrácením prvky kódu potřebný.  
  
 **ID chyby:** BC30452  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte dva prvky a ujistěte se, že jsou kompatibilní.
