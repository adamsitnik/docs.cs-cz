# <a name="method-ltmethodnamegt-cannot-handle-event-lteventnamegt-because-they-do-not-have-the-same-signature"></a>Metoda "&lt;methodname&gt;"nemůže zpracovávat událost"&lt;eventname&gt;' vzhledem k tomu, že nemají stejnou signaturu
Parametry nebo typy parametrů pro obslužné rutiny zadané události se neshodují s těmi definovanými zadané události.  
  
 **ID chyby:** BC31029  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Znovu definujte událost nebo obslužná rutina události tak, aby parametr obsahuje shodu.
