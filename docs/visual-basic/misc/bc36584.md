# <a name="parameter-parametername-in-extension-method-methodname-defined-in-typename-already-has-a-matching-argument"></a>Parametr '\<parametername >' v metodě rozšíření "\<methodname >" definované v "\<typename >' již existuje odpovídající argument
Argument je předán podle názvu více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC36584  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda některý název argumentu je zadáno chybně.  
  
-   Duplicitní argument odeberte ze seznamu argumentů.
