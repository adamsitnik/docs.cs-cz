# <a name="parameter-ltparameternamegt-in-extension-method-ltmethodnamegt-defined-in-lttypenamegt-already-has-a-matching-argument"></a>Parametr '&lt;parametername&gt;"v metodě rozšíření"&lt;methodname&gt;"definované v"&lt;typename&gt;' již existuje odpovídající argument
Argument je předán podle názvu více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC36584  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda některý název argumentu je zadáno chybně.  
  
-   Duplicitní argument odeberte ze seznamu argumentů.
