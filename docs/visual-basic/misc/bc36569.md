# <a name="argument-not-specified-for-parameter-ltparameternamegt"></a>Nebyl zadán argument pro parametr '&lt;parametername&gt;.
Argument pro požadovaný parametr nebyla zadána.  
  
 **ID chyby:** BC36569  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte argument pro zadaný parametr.
