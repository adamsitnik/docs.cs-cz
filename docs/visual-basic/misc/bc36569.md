# <a name="argument-not-specified-for-parameter-parametername"></a>Nebyl zadán argument pro parametr '\<parametername > "
Argument pro požadovaný parametr nebyla zadána.  
  
 **ID chyby:** BC36569  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte argument pro zadaný parametr.
