# <a name="the-value-ltvaluegt-is-invalid-for-option-ltoptionnamegt"></a>Hodnota '&lt;hodnotu&gt;'je neplatný pro možnost'&lt;název_možnosti&gt;.
Byla zadána hodnota pro možnost příkazového řádku, která nebyla platná.  
  
 **ID chyby:** BC2014  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte hodnoty a vyberte ten, který není platný.
