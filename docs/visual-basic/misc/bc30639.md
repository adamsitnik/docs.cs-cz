# <a name="properties-cannot-be-declared-modifiername"></a>Vlastnosti nelze použít deklaraci '\<modifiername > "
Dojde k chybě v syntaxi používané `Property` deklarace. Tato chyba obvykle znamená, že zadané klíčové slovo jako například `WithEvents`, nelze použít v `Property` deklarace.  
  
 **ID chyby:** BC30639  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte syntaxi `Property` deklarace.
