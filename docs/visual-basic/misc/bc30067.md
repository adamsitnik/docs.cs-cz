# <a name="exit-function-is-not-valid-in-a-sub-or-property"></a>'Slovo exit Function' není platné v proceduře Sub ani Property
`Exit Function` Zobrazí se v `Sub` procedury nebo `Property` postup. `Exit` Příkaz musí odpovídat bloku ve kterém se vyskytuje.  
  
 **ID chyby:** BC30067  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahradit `Exit Function` s `Exit Sub` nebo `Exit Property` příkaz podle potřeby.
