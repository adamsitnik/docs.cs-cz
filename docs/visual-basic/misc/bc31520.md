# <a name="unable-to-apply-security-attributes-to-name-error"></a>Nelze použít atributy zabezpečení-'\<název >': \<chyby >
Došlo k chybě při provádění atributy zabezpečení. To může být výsledkem nesprávného použití atributy zabezpečení nebo mohlo by to znamenat problém s kompilátor.  
  
 **ID chyby:** BC31520  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, že syntaxe a argumenty pro atribut.  
  
2.  Kompilovat program znovu, abyste viděli, pokud chyba bude opakovat.  
  
3.  Pokud chyba přetrvává, uložte si práci a restartujte aplikaci Visual Studio.  
  
4.  Pokud chyba přetrvává, přeinstalujte Visual Basic.  
  
5.  Pokud chyba přetrvává po přeinstalaci, upozorní Microsoft Product Support Services.
