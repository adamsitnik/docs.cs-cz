# <a name="this-system-does-not-contain-support-for-the-japanese-locale"></a>Tento systém neobsahuje podporu pro japonské národní prostředí
Pokoušíte se použít `VbStrConv.Japanese` člen výčtu, ale váš systém nepodporuje japonském národním prostředí.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Upgrade počítače pro podporu japonské národní prostředí nebo spusťte aplikaci na počítač, který podporuje národní prostředí.
