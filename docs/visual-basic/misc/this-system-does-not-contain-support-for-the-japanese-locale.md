# <a name="this-system-does-not-contain-support-for-the-japanese-locale"></a>Tento systém neobsahuje podporu pro japonské národní prostředí.
Pokoušíte se použít `VbStrConv.Japanese` člen výčtu, ale systém nepodporuje japonské národní prostředí.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Upgradujte počítač pro podporu japonského národního prostředí nebo spusťte aplikaci na počítači, který podporuje národní prostředí.
