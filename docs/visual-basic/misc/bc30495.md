# <a name="exponent-is-not-valid"></a>Exponent není platný
Pokusili jste se použít exponent, který není platný s literál s plovoucí desetinnou čárkou. K této chybě obvykle dochází, když literál s plovoucí desetinnou čárkou typu je v rozporu s velikost potřebná k uchování literál s plovoucí desetinnou čárkou.  
  
 **ID chyby:** BC30495  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Kontrola typu literál s plovoucí desetinnou čárkou a změňte ji na platného typu.
