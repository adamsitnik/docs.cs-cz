# <a name="ltprocedurenamegt-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>"&lt;název_procedury&gt;" návratový typ, který není podporován nebo typy parametrů, které nejsou podporovány
Postup je deklarována s parametry nebo návratový typ, který nemůže být reprezentována sady Visual Studio.  
  
 **ID chyby:** BC30657  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Definujte návratové typy nebo parametry jako podporovaného typu.  
  
-   Vytvořte nový typ, který popisuje data používá tento postup.
