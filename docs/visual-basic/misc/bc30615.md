# <a name="end-statement-cannot-be-used-in-class-library-projects"></a>Příkaz 'End' nelze použít v projektech knihoven tříd
Projekty knihovny tříd pro vytvoření DLL knihoven neumožňují použití `End` – klíčové slovo na svém vyvolání zastaví provádění kódu v postupu.  
  
 **ID chyby:** BC30615  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití řídicích struktur, jako `While` a `For` k řízení toku programu.
