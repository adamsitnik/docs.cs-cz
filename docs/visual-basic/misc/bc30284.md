# <a name="lttype1gt-lttypenamegt-cannot-be-declared-overrides-because-it-does-not-override-a-lttype1gt-in-a-base-lttype2gt"></a>&lt;Type1&gt; "&lt;typename&gt;' nelze použít deklaraci 'Přepsání', protože nepřepisuje &lt;type1&gt; v základní třídě &lt;typ2&gt;
A `Sub`, `Function`, nebo `Property` příkaz určuje `Overrides` Pokud neexistuje žádný typ se stejným názvem v základní třídě.  
  
 **ID chyby:** BC30284  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, jestli název typu je napsaný správně.  
  
2.  Odebrat nadbytečný `Overrides` – klíčové slovo.
