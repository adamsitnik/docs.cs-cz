# <a name="argument-argumentname-must-be-greater-than-or-equal-to-zero"></a>Argument '\<název argumentu argumentname >' musí být větší než nebo rovna hodnotě nula.
Argument není platný, protože je menší než nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte, zda argumenty. Chybně napsaná název proměnné implicitně vytvářet číselné proměnné, která je inicializována na nulovou hodnotu.  
  
2. Zkontrolujte předchozí operací pro proměnné ve výrazu, zejména těch, které jsou předány do procedury jako argumenty z jiných postupů.
