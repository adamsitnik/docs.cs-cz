# <a name="property-propertyname-with-no-parameters-cannot-be-found"></a>Vlastnost '\<propertyname >' bez parametrů nebyla nalezena
Atribut používá má vlastnost, která vyžaduje jeden nebo více parametrů.  
  
 **ID chyby:** BC30658  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zadejte parametry pro vlastnost.
