# <a name="property-ltpropertynamegt-with-no-parameters-cannot-be-found"></a>Vlastnost "&lt;propertyname&gt;" bez parametrů nebyla nalezena
Atribut používá má vlastnost, která vyžaduje jeden nebo více parametrů.  
  
 **ID chyby:** BC30658  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte parametry pro vlastnost.
