# <a name="methods-cannot-be-declared-static"></a>Metody nelze použít deklaraci 'Static'
`Static` Modifikátor nejde použít u celé procedury.  
  
 **ID chyby:** BC30810  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `Static` Modifikátor pro deklarování jednotlivých položek v rámci procedury jako statické.
