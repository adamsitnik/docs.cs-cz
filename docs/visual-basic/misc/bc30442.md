# <a name="finally-must-end-with-a-matching-end-try"></a>"Finally" musí končit odpovídajícího 'End Try
A `Finally` příkazu se zobrazí ve vašem kódu bez odpovídajícího `End Try` příkazu. `Finally` příkazy musí být uzavřena s `End Try` příkazu.  
  
 **ID chyby:** BC30442  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Finally` příkazu.  
  
2.  Přidat `End Try` příkaz uzavírat blok.
