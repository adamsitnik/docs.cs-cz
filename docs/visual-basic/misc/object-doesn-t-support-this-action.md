# <a name="object-doesnt-support-this-action-visual-basic"></a>Objekt nepodporuje tuto akci (Visual Basic)
Můžete odkazovat, metodu nebo vlastnost, která není podporována tímto objektem.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přečtěte si dokumentaci objektu pro další informace o objektu.  
  
2.  Ujistěte se, že jsou správně zadány názvy vlastností a metod.
