# <a name="expected-closing----for-xml-comment"></a>Bylo očekáváno ukončení '-->' pro komentář XML
Literál komentáře XML chybí požadovaný ukončovací '-->' znaků.  
  
 **ID chyby:** BC31161  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidáte požadované uzavírací '-->' znaky na konec literál komentáře XML.
