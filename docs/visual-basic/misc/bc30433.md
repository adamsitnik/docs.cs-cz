# <a name="methods-in-a-module-cannot-be-declared-ltspecifiergt"></a>Metody v modulu nelze použít deklaraci '&lt;specifikátor&gt;.
Jste už použili, který není platný pro metodu ve specifikátoru `Module` příkazu. Moduly možné nikdy instancovat, nepodporují dědičnosti a nemůžou implementovat rozhraní.  
  
 **ID chyby:** BC30433  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
