# <a name="operator-operator-is-not-defined-for-types-type1-and-type2"></a>Operátor 'operator' není definován pro typy 'type1' a 'type2'
Operátor 'operator' není definován pro typy 'type1' a 'type2'. Vlastnost 'Value' slouží k získání řetězcové hodnoty prvního prvku řetězce "\<parentElement >'.  
  
 Literál XML se použil binární soubor nebo `Boolean` porovnání.  
  
 **ID chyby:** BC31195  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `Value` vlastnost literálu pro binární soubor XML nebo `Boolean` porovnání.
