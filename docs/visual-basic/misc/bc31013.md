# <a name="unable-to-load-dll-filename-error"></a>Nelze načíst knihovnu DLL '\<název souboru >': \<chyby >
Nelze načíst knihovnu DLL odkazuje tento projekt.  
  
 **ID chyby:** BC31013  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, že soubor je v zadaném umístění a není uzamčený jinou aplikací.
