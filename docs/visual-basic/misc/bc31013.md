# <a name="unable-to-load-dll-ltfilenamegt-lterrorgt"></a>Nelze načíst knihovnu DLL '&lt;filename&gt;': &lt;chyba&gt;
Nelze načíst knihovnu DLL odkazuje tento projekt.  
  
 **ID chyby:** BC31013  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, že soubor je v zadaném umístění a není uzamčený jinou aplikací.
