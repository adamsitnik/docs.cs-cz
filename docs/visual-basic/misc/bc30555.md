# <a name="default-member-ltmembernamegt-is-not-a-property"></a>Výchozí člen "&lt;membername&gt;" se nenachází ve vlastnosti
Pokusili jste se deklarovat člen `Default`. Ale `Default` – klíčové slovo se používá k identifikaci výchozí vlastnosti.  
  
 **ID chyby:** BC30555  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, ujistěte se, že identifikujete vlastnost s `Default`, nebo odeberte ostatní `Default` – klíčové slovo.
