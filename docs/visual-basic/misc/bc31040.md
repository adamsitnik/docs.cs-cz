# <a name="ltmembernamegt-exists-in-multiple-base-interfaces"></a>"&lt;membername&gt;' existuje ve více základních rozhraních
"\<členské jméno >' existuje ve více základních rozhraních. Použijte název rozhraní, které deklaruje "\<membername >' v 'Implements' klauzuli místo názvu odvozeného rozhraní.  
  
 Toto rozhraní dědí z více rozhraní nejasnosti členy se stejným názvem.  
  
 **ID chyby:** BC31040  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití názvu definující rozhraní v `Implements` klauzule místo názvu odvozeného rozhraní.
