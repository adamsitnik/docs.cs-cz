# <a name="variable-cannot-be-initialized-with-non-array-type-ltelementnamegt"></a>Proměnnou nejde inicializovat s použitím mimo pole typu "&lt;elementname&gt;.
Proměnná, která je deklarována jako pole musí být inicializován s hodnotou pole.  
  
```  
' Not valid.  
' The following line causes this error when executed with Option Strict off.  
' Dim arrayVar1() = 10  
```  
  
 **ID chyby:** BC36536  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Inicializace proměnné pole s hodnotou pole:  
  
    ```
    ' With Option Strict off.  
    Dim arrayVar2() = {1, 2, 3}  
    ' With Option Strict on.  
    Dim arrayVar3() As Integer = {1, 2, 3}  
    ```
