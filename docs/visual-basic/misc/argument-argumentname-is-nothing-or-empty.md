# <a name="argument-ltargumentnamegt-is-nothing-or-empty"></a>Argument '&lt;název argumentu argumentname&gt;' má hodnotu Nothing nebo je prázdný
Název argumentu není platný, protože je inicializován na `Nothing` nebo je prázdný. Argument musí mít určitý datový typ. a platný název.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte název platný argument.
