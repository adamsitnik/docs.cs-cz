# <a name="argument-argumentname-is-nothing-or-empty"></a>Argument '\<název argumentu argumentname > "má hodnotu Nothing nebo je prázdný
Název argumentu není platný, protože je inicializován na `Nothing` nebo je prázdný. Argument musí mít určitý datový typ. a platný název.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte název platný argument.
