# <a name="xml-declaration-does-not-allow-attribute-attributename"></a>Deklaraci XML nelze použít atribut %{attributename/
Byl zadán neplatný atribut pro literál dokumentu XML. Literál dokumentu XML platný atribut hodnoty jsou podle specifikace XML 1.0.  
  
 **ID chyby:** BC31154  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Neplatný atribut odebrání literál dokumentu XML.
