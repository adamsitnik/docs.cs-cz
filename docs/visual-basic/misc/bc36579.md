# <a name="too-many-type-arguments"></a>Příliš mnoho argumentů typu
Obecná metoda byla vyvolána s více argumentů, než je počet parametrů typu.  
  
 Při vyvolání obecné metody, je nutné zadat jeden argument typu pro jednotlivé parametry typu určené metody.  
  
 **ID chyby:** BC36579  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Argumenty typu odeberte ze seznamu argumentů typu, tak, aby se jeden pro každý parametr typu Obecné metody, kterou vyvoláváte.
