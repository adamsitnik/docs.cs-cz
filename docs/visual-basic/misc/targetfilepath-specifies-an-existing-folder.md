# <a name="targetfilepath-specifies-an-existing-folder"></a>Určuje TargetFilePath existující složku
Parametr `TargetFilePath` určuje existující adresář, ale musíte zadat název nového adresáře.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změna `TargetFilePath` určit nový, spíše než existující adresář.
