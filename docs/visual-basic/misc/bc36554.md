## <a name="to-correct-this-error"></a><span data-ttu-id="28fd6-111">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="28fd6-111">To correct this error</span></span>

- <span data-ttu-id="28fd6-112">Pokud pole parametrů neurčuje datový typ, který chcete zvětšit, přidejte nový první parametr, který určuje tento typ.</span><span class="sxs-lookup"><span data-stu-id="28fd6-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>

  ```vb

  Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)
  ' Concatenate the strings in addOns to str.
  End Sub
  ```

- <span data-ttu-id="28fd6-113">Pokud pole parametrů Určuje datový typ, který chcete zvětšit, zvažte jejich změnu na regulární pole, a to vyžadováním argumentu namísto pole parametrů.</span><span class="sxs-lookup"><span data-stu-id="28fd6-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="28fd6-114">Běžná pole lze rozšířit.</span><span class="sxs-lookup"><span data-stu-id="28fd6-114">Regular arrays can be extended.</span></span>

  ```vb

  Public Function Sum(ByVal ints() As Integer) As Integer
      Dim total As Integer = 0
      For Each i As Integer In ints
          total = total + i
      Next i
      Return total
  End Function
  ```
