# <a name="insufficient-security-permissions-to-set-the-system-date"></a>Nedostatečná oprávnění zabezpečení k nastavení systémového data
Oprávnění zabezpečení jsou dostatečná k nastavení systémového data.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte oprávnění zabezpečení, aby nastavení systémového data.
