# <a name="method1-cannot-override-method2-because-they-differ-by-their-return-types"></a>"\<– metoda1 >" nelze přepsat '\<method2 > "protože se liší typy vrácených hodnot
Pokusili jste se přepsat metodu s jinou metodu, která se liší podle jejího návratového typu. Typ může přepsat zděděnou přepisovatelnou metodu deklarace metody se stejným názvem a signaturou, a označením deklarace se `Overrides` modifikátor. Podpisy dvě metody se musí shodovat.  
  
 **ID chyby:** BC30437  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte návratové typy dvě metody a změnit podle potřeby tak, aby odpovídaly.
