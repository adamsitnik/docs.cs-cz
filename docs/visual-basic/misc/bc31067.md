# <a name="method-in-a-structure-cannot-be-declared-protected-or-protected-friend"></a>Metodu ve struktuře nelze deklarovat, "Protected" nebo Protected Friend
Metodu ve struktuře se deklaroval jako `Protected` nebo `Protected Friend`.  
  
 **ID chyby:** BC31067  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Deklarovat metody uvnitř struktury, jako jsou `Public` nebo `Private`.
