# <a name="lttypenamegt-is-an-unsupported-type"></a>"&lt;typename&gt;' jedná se nepodporovaný typ.
Zadaný typ nemůže být reprezentována sady Visual Studio.  
  
 **ID chyby:** BC30649  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pomocí podporovaného typu.  
  
-   Vyvíjejte nového typu.
