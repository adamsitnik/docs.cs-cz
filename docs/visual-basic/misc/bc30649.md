# <a name="typename-is-an-unsupported-type"></a>'\<typename >' není podporován
Zadaný typ nemůže být reprezentována sady Visual Studio.  
  
 **ID chyby:** BC30649  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pomocí podporovaného typu.  
  
- Vyvíjejte nového typu.
