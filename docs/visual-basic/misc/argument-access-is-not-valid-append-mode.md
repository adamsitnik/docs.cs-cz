# <a name="argument-access-is-not-valid-valid-values-for-append-mode-are-openaccesswrite-and-openaccessdefault"></a>Argument "Přístup" není platný. Platnými hodnotami režimu Append jsou "OpenAccess.Write" a "OpenAccess.Default.
Hodnotu přístupu není platný v režimu přidávání.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změňte hodnotu na buď `OpenAccess.Write` nebo `OpenAccess.Default`.
