# <a name="project-level-conditional-compilation-constant-is-not-valid-error"></a>Konstanta podmíněné kompilace na úrovni projektu není platná: \<chyba >
Pro projekt je zadána neplatná možnost podmíněné kompilace.  
  
 **ID chyby:** BC31031  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte konstanty podmíněné kompilace používané v tomto projektu.
