# <a name="project-level-conditional-compilation-constant-is-not-valid-lterrorgt"></a>Konstanta podmíněné kompilace na úrovni projektu není platná: &lt;chyba&gt;
Možnost neplatná Podmíněná kompilace je vybrané pro tento projekt.  
  
 **ID chyby:** BC31031  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte konstanty pro podmíněnou kompilaci použita s tímto projektem.
