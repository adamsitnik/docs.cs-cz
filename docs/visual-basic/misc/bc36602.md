# <a name="readonly-variable-cannot-be-the-target-of-an-assignment-in-a-lambda-expression-inside-a-constructor"></a><span data-ttu-id="bcb91-102">Proměnná 'ReadOnly' nemůže být cílem přiřazení. ve výrazu lambda obsaženém v konstruktoru</span><span class="sxs-lookup"><span data-stu-id="bcb91-102">'ReadOnly' variable cannot be the target of an assignment in a lambda expression inside a constructor</span></span>
<span data-ttu-id="bcb91-103">Se odkazuje `ReadOnly` proměnné z v rámci výrazu lambda, což není povoleno.</span><span class="sxs-lookup"><span data-stu-id="bcb91-103">You have referred to a `ReadOnly` variable from within a lambda expression, which is not permitted.</span></span> <span data-ttu-id="bcb91-104">Následující kód způsobí, že k této chybě odesláním `ReadOnly` proměnnou `m` jako argument `ByRef` parametru.</span><span class="sxs-lookup"><span data-stu-id="bcb91-104">The following code causes this error by sending the `ReadOnly` variable `m` as the argument to a `ByRef` parameter.</span></span>  
  
```vb  
Class Class1  
  
    ReadOnly m As Integer  
  
    Sub New()  
        ' The use of m triggers the error.  
        Dim f = Function() Test(m)  
    End Sub  
  
    Function Test(ByRef n As Integer) As String  
    End Function  
  
End Class  
```  
  
 <span data-ttu-id="bcb91-105">**ID chyby:** BC36602</span><span class="sxs-lookup"><span data-stu-id="bcb91-105">**Error ID:** BC36602</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="bcb91-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="bcb91-106">To correct this error</span></span>  
  
- <span data-ttu-id="bcb91-107">Pokud je to možné, změňte parametr `n` ve funkci `Test` k `ByVal` parametru, jak je znázorněno v následujícím kódu.</span><span class="sxs-lookup"><span data-stu-id="bcb91-107">If possible, change parameter `n` in function `Test` to a `ByVal` parameter, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix1  
  
        ReadOnly m As Integer  
  
        Sub New()  
            Dim f = Function() Test(m)  
        End Sub  
  
        Function Test(ByVal n As Integer) As String  
        End Function  
  
    End Class  
    ```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="bcb91-108">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="bcb91-108">To correct this error</span></span>  
  
- <span data-ttu-id="bcb91-109">Přiřazení `ReadOnly` proměnnou `m` místní proměnné v konstruktoru a použijte místní proměnnou místo `m`, jak je znázorněno v následujícím kódu.</span><span class="sxs-lookup"><span data-stu-id="bcb91-109">Assign the `ReadOnly` variable `m` to a local variable in the constructor, and use the local variable in place of `m`, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix2  
         ReadOnly m As Integer  
  
        Sub New()  
            Dim temp = m  
            Dim f = Function() Test(temp)  
        End Sub  
  
        Function Test(ByRef n As Integer) As String  
        End Function  
  
    End Class  
    ```
