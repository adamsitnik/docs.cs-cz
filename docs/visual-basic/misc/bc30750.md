# <a name="information-for-the-type-of-lttypenamegt-has-not-been-loaded-into-the-runtime"></a>Informace o typu "&lt;typename&gt;' nebylo načteno do modulu runtime
Typ odkazovalo, který není načtený modulem runtime.  
  
 **ID chyby:** BC30750  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změnit strukturu kódu tak, že typ je definován v aktuálním oboru.  
  
2.  Ověřte, že člen je definovaný a jestli mají název člena napsaný správně.  
  
3.  Zkuste jednu z členy deklarované v modulu. V některých případech nelze ladicího prostředí najít členy, protože nejsou načtené moduly, ve kterém jsou deklarovány.
