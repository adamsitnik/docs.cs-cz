# <a name="new-is-not-valid-in-this-context"></a>"Nové" není v tomto kontextu platné.
`New` – Klíčové slovo nastane, pokud je požadován název typu. Kontext volání pro hodnotový typ, který se nedá kombinovat s `New`.  
  
 **ID chyby:** BC30200  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `New` – klíčové slovo.
