# <a name="vbstrconvwide-and-vbstrconvnarrow-cannot-be-combined"></a>Argumenty VbStrConv.Wide a VbStrConv.Narrow nelze kombinovat
Vaše aplikace se pokouší zkombinovat `VbStrConv` členy výčtu `Wide` a `Narrow`, které se vzájemně vylučují.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Buď odeberte `VbStrConv.Wide` nebo `VbStrConv.Narrow`.
