# <a name="vbstrconvwide-and-vbstrconvnarrow-cannot-be-combined"></a>Argumenty VbStrConv. roztažitelné a VbStrConv. Narrow nelze kombinovat.
Vaše aplikace se pokouší kombinovat členy výčtu `VbStrConv` `Wide` a `Narrow`, které se vzájemně vylučují.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte buď `VbStrConv.Wide`, nebo `VbStrConv.Narrow`.
