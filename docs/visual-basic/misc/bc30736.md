# <a name="expression-cannot-be-evaluated-at-this-time"></a>V tuto chvíli nelze vyhodnotit výraz
Zadaný výraz nelze vyhodnotit v aktuálním kontextu.  
  
 **ID chyby:** BC30736  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změníte syntaxe výrazu.
