# <a name="the-file-is-currently-closed"></a>Soubor je aktuálně zavřen.
Byl proveden pokus o provedení operace, které lze provést pouze na otevřený soubor, ale soubor je aktuálně zavřen.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Otevřete soubor a opakujte operaci znovu.
