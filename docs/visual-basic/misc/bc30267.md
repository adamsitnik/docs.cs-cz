# <a name="ltdeclaration1gt-cannot-override-ltdeclaration2gt-because-it-is-declared-notoverridable"></a>"&lt;declaration1&gt;nemůže přepsat&lt;declaration2&gt;' protože se deklaruje"NotOverridable.
Deklaraci procedury nebo vlastnost pokusí přepsat zděděný element se stejným názvem, ale zděděné element je zadán jako `NotOverridable`.  
  
 **ID chyby:** BC30267  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `NotOverridable` – klíčové slovo z deklarace zděděné elementu nebo odebrat přepsání deklarace.
