# <a name="declaration1-cannot-override-declaration2-because-it-is-declared-notoverridable"></a>'\<declaration1 >' nelze přepsat '\<declaration2 > 'protože se deklaruje'NotOverridable.
Procedura nebo deklarace vlastnosti se pokusí přepsat zděděný prvek se stejným názvem, ale zděděný element je zadán jako `NotOverridable`.  
  
 **ID chyby:** BC30267  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- `NotOverridable` Odeberte klíčové slovo z deklarace zděděného prvku nebo odeberte překrytou deklaraci.
