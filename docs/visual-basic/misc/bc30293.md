# <a name="error-structurename1-contains-structurename2"></a>\<Chyba >: "\<structurename1 >' obsahuje '\<structurename2 >"
Definice cyklické struktury vyvolá se v případě struktury samotné označí jako jeden z jeho vlastní členů.  
  
 **ID chyby:** BC30293  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte název členu struktury.
