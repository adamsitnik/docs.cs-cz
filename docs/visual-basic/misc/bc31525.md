# <a name="type-lttypenamegt-is-imported-from-different-versions-of-assembly-ltassemblynamegt"></a>Typ '&lt;typename&gt;"je importován z různých verzí sestavení"&lt;assemblyname&gt;.
Typ '\<typename >' je importován z různých verzí sestavení "\<assemblyname >'. Různé verze téhož sestavení nelze použít ve stejném projektu.  
  
 Typ byl importován z různých verzí stejného sestavení, vytvoření nejednoznačnost.  
  
 **ID chyby:** BC31525  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte odkazy na více verzí stejného sestavení.
