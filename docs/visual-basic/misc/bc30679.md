# <a name="array-initializers-are-valid-only-for-arrays-but-the-type-of-ltvariablenamegt-is-lttypenamegt"></a>Inicializátory polí jsou platné pouze pro pole, ale typu "&lt;NázevProměnné&gt;'is'&lt;typename&gt;.
Došlo k pokusu o inicializaci proměnné bez pole se seznamem hodnot.  
  
 **ID chyby:** BC30679  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Deklarujete a inicializujete proměnnou jako pole. Příklad:  
  
     `Dim intarray As Integer() = {1, 5, 9}`  
  
-   Inicializace proměnné jako jedinou hodnotu; Příklad:  
  
     `Dim intvalue As Integer = 1`
