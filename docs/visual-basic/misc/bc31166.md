# <a name="expected-closing--for-xml-tag"></a>Bylo očekáváno ukončení '>' pro – značka XML
Chybí vyžaduje ukončovací značka XML pro literál XML '>' znak.  
  
 **ID chyby:** BC31166  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidání ukončovací '>' znak na konci značky XML pro literál XML.
