# <a name="string-constant-expected"></a>Byla očekávána řetězcová konstanta
Znak kromě dvojité uvozovky (") nastane, pokud kontext nevyžaduje řetězcová konstanta. Jednou z možných příčin je, že se z řetězce chybí uvozovky.  
  
 **ID chyby:** BC30217  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že je řetězec správně uzavřen do dvojitých uvozovek.  
  
2. Zadejte řetězcová konstanta.
