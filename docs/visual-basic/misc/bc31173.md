# <a name="missing-required-white-space"></a>Chybí vyžadované mezerové znaky
Literál XML byl zadán bez požadovaný oddělovací mezery. Příklad místo požadované oddělovač je mezera mezi následující atributy v elementu XML literál.  
  
 **ID chyby:** BC31173  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidáte oddělovač požadovaného místa.
