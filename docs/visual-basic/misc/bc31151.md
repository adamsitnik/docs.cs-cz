# <a name="element-is-missing-an-end-tag"></a>V elementu chybí koncová značka
Literál XML element chybí koncová značka XML nebo identifikátor prázdnou značkou "/" (například `<tagName />`).  
  
 **ID chyby:** BC31151  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidat značku koncové značky XML nebo přidat prázdnou značkou identifikátor ' /' jazyka XML označit.
