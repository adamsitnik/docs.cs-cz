# <a name="array-exceeds-the-limit-of-32-dimensions"></a>Pole překračuje omezení 32 rozměrů
Pole nemůže mít více než 32 rozměry.  
  
 **ID chyby:** BC30052  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Snížit počet dimenzí v deklaraci pole.
