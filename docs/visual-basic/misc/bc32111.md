 <span data-ttu-id="a1a10-105">Nepřímým omezením `Structure` a přímé omezení `Class` způsobit konflikt pro parametr typu `t1`, protože `Structure` vyžaduje omezení, že odpovídající argument typu být typ hodnoty, zatímco `Class` vyžaduje, aby se typ odkazu.</span><span class="sxs-lookup"><span data-stu-id="a1a10-105">The indirect constraint `Structure` and the direct constraint `Class` cause a conflict for type parameter `t1`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
