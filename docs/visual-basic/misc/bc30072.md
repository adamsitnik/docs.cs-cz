# <a name="case-can-only-appear-inside-a-select-case-statement"></a>'Case' může být použito pouze uvnitř příkazu 'Select Case'
A `Case` příkaz proběhne mimo `Select` bloku. A `Case` příkaz lze použít pouze mezi `Select` nebo `Select Case` příkazu a odpovídající `End Select` příkazu.  
  
 **ID chyby:** BC30072  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Case` příkazu nebo přesunout v rámci `Select` bloku.
