# <a name="referenced-array-element-has-a-value-of-nothing"></a>Odkazovaný prvek pole s hodnotou 'Nothing'
Není nastaveno prvku pole, na kterou se odkazuje, nebo pokud byla dříve nastavena na `Nothing`.  
  
 **ID chyby:** BC30744  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ověřte, že jsou hodnoty v poli nastavena správně.
