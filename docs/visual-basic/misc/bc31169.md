# <a name="character-character-0xhexadecimal-is-not-allowed-at-the-beginning-of-an-xml-name"></a>Znak 'znak' (0 x\<šestnáctkové >) na začátku názvu XML není povoleno
Nepodporovaný znak je zahrnuty na začátku názvu XML literál nebo vlastnosti osy XML.  
  
 **ID chyby:** BC31169  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte nepodporované znaky ze začátku názvu XML literál nebo vlastnosti osy XML.
