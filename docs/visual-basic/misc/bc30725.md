# <a name="throw-statements-are-not-valid-in-the-immediate-window"></a>Příkazy throw nejsou platné v příkazovém podokně.
příkazy `Throw` lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30725  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte příkazy `Throw` z ladění kódu.
