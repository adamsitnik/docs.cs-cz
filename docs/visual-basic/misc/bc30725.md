# <a name="throw-statements-are-not-valid-in-the-immediate-window"></a>Příkazy 'Throw' nejsou platné v příkazovém
`Throw` příkazy jde použít jenom ve zdrojovém kódu.  
  
 **ID chyby:** BC30725  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `Throw` příkazy z ladění kódu.
