# <a name="ltmodifiergt-is-not-valid-on-a-local-variable-declaration"></a>"&lt;modifikátor&gt;' není platný v deklaraci lokální proměnné
A `Dim` výroku uvnitř procedury obsahuje neplatné klíčové slovo jako například `Overloads`.  
  
 **ID chyby:** BC30247  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte neplatné klíčové slovo z `Dim` příkazu.
