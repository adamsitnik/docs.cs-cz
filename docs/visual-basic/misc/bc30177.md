# <a name="only-one-of-notoverridable-mustoverride-or-overridable-can-be-specified"></a>Lze zadat pouze jeden z těchto elementů: ' NotOverridable ', ' MustOverride ' nebo ' overridable '.
Konfliktní přepisující specifikace se vyskytují v deklaraci elementu. Přepisující klíčová slova se vzájemně vylučují.  
  
 **ID chyby:** BC30177  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vyberte jeden překrytý specifikátor a odeberte další klíčová slova z deklarace.
