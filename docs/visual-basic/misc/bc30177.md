# <a name="only-one-of-notoverridable-mustoverride-or-overridable-can-be-specified"></a>Lze zadat pouze jedno "NotOverridable", "MustOverride" nebo "Overridable.
Konfliktní přepsání specifikace objevit v deklarace elementu. Přepsání klíčová slova se vzájemně vylučují.  
  
 **ID chyby:** BC30177  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zvolte jeden specifikátor přepsání a odebrání dalších klíčových slovech deklarace.
