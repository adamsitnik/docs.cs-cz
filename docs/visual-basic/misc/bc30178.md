# <a name="specifier-is-duplicated"></a>Specifikátor je duplicitní.
Klíčové slovo uveden více než jednou ve stejné deklaraci.  
  
 **ID chyby:** BC30178  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte duplicitní výskyt klíčové slovo z deklarace.
