# <a name="syntax-error-visual-basic-error"></a>Chyba syntaxe (Chyba jazyka Visual Basic)
Řádky zdrojového kódu generuje chybu Obecná syntaxe.  
  
 **ID chyby:** BC30035  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Prozkoumejte dokumentaci pro každou klíčová slova používaná při řádky zdrojového kódu. Porovnejte kód příkladu v dokumentaci pro zdrojový řádek generování této chyby.  
  
2. Pokud nelze určit příčinu chyby, shromážděte informace o okolnostech a upozornit Microsoft Product Support Services.
