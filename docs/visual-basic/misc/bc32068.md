# <a name="local-variables-within-generic-methods-cannot-be-declared-static"></a>Lokální proměnné v rámci obecných metod nelze deklarovat jako 'Static'
Určuje deklaraci lokální proměnné v rámci obecné procedury `Static`.  
  
 Visual Basic a rozhraní .NET Framework nepodporují aktuálně libovolnou kombinaci statických proměnných a obecné procedury.  
  
 **ID chyby:** BC32068  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Static` – klíčové slovo z deklarace proměnné.
