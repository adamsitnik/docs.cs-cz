# <a name="xml-namespace-prefix-prefix-is-not-defined"></a>Předpona oboru názvů XML 'předponu' není definován.
Pro literál XML nebo vlastnosti osy XML byla zadána Nedefinovaná Předpona oboru názvů XML.  
  
 **ID chyby:** BC31148  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Definovat předponu oboru názvů XML pomocí `Imports` příkazu.  
  
 -nebo-  
  
-   Upravte předpony oboru názvů XML pro literál XML nebo vlastnosti osy XML k dříve definovanou předponu oboru názvů XML.
