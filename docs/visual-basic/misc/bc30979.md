# <a name="option-infer-can-be-followed-only-by-on-or-off"></a>"Možnost Infer" může následovat pouze 'Na' nebo 'Off'
`Option Infer` příkazy musí určovat buď `On` nebo `Off`. Jiné hodnoty nejsou platné.  
  
 **ID chyby:** BC30979  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte buď `Option Infer On` nebo `Option Infer Off`.
