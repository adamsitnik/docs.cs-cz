# <a name="character-constant-must-contain-exactly-one-character"></a>Znaková konstanta musí obsahovat přesně jeden znak
Znaková konstanta je neplatný, protože buď obsahuje více než jeden znak nebo znak null (nula všechny bity jsou).  
  
 **ID chyby:** BC30004  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změnit Znaková konstanta jeden znak nebo znak null k platnému znaku.
