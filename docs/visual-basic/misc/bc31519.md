# <a name="ltfilenamegt-cannot-be-referenced-because-it-is-not-an-assembly"></a>"&lt;filename&gt;" nelze odkazovat, protože se nejedná o sestavení
Se odkazuje na soubor, který není sestavením .NET.  
  
 **ID chyby:** BC31519  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte název a umístění sestavení.  
  
2.  Ověřte, že sestavení není poškozený.
