# <a name="attributes-cannot-be-applied-to-local-variables"></a>Atributy nelze použít pro lokální proměnné.
Atributy nelze použít pro lokální proměnné.  
  
 **ID chyby:** BC30660  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Deklarujte tuto proměnnou na další vyšší úrovni oboru, například na úrovni třídy nebo modulu.
