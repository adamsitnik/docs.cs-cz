# <a name="attributes-cannot-be-applied-to-local-variables"></a>Atributy nelze použít pro lokální proměnné
Atributy nejde použít u místních proměnných.  
  
 **ID chyby:** BC30660  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Deklarování této proměnné na další vyšší úroveň oboru, například třídy nebo na úrovni modulu.
