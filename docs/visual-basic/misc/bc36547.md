# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="bff41-102">Člen anonymního typu nebo vlastnost '\<propertyname >' již byla deklarována</span><span class="sxs-lookup"><span data-stu-id="bff41-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="bff41-103">Název vlastnosti lze vytvořit pouze jednou v deklaraci anonymního typu.</span><span class="sxs-lookup"><span data-stu-id="bff41-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="bff41-104">Například následující deklarace nejsou platné:</span><span class="sxs-lookup"><span data-stu-id="bff41-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="bff41-105">**ID chyby:** BC36547</span><span class="sxs-lookup"><span data-stu-id="bff41-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="bff41-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="bff41-106">To correct this error</span></span>

```vb  
' Valid.

```  
  
-   <span data-ttu-id="bff41-108">Zadejte nové názvy pro proměnné nebo názvy vlastností, z nichž jsou odvození názvů a hodnot.</span><span class="sxs-lookup"><span data-stu-id="bff41-108">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```
