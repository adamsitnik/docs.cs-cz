# <a name="anonymous-type-member-or-property-ltpropertynamegt-is-already-declared"></a><span data-ttu-id="3ab94-102">Člen anonymního typu nebo vlastnost '&lt;propertyname&gt;' již byla deklarována</span><span class="sxs-lookup"><span data-stu-id="3ab94-102">Anonymous type member or property '&lt;propertyname&gt;' is already declared</span></span>
<span data-ttu-id="3ab94-103">Název vlastnosti lze vytvořit pouze jednou v deklaraci anonymního typu.</span><span class="sxs-lookup"><span data-stu-id="3ab94-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="3ab94-104">Například následující deklarace nejsou platné:</span><span class="sxs-lookup"><span data-stu-id="3ab94-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="3ab94-105">**ID chyby:** BC36547</span><span class="sxs-lookup"><span data-stu-id="3ab94-105">**Error ID:** BC36547</span></span>
