# <a name="catch-cannot-catch-type-lttypenamegt-because-it-is-not-systemexception-or-a-class-that-inherits-from-systemexception"></a>'Příkaz catch' nemůže zachytit typ "&lt;typename&gt;' protože se nejedná 'System.Exception' ani o třídu odvozenou od třídy 'System.Exception'
`Catch` můžete pouze zachytávat výjimky a pokusili jste se něco, co není odvozen od výjimku zachytit.  
  
 **ID chyby:** BC30392  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Catch` příkazu, nebo změňte cíl `Catch` na skutečné výjimku.
