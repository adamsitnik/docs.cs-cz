# <a name="module-attribute-attributename-is-not-valid-error"></a>Atribut Module '\<attributename >' není platný: \<chyby >
Modul common language runtime nepřijímá atribut v modulu.

**ID chyby:** BC30130

## <a name="to-correct-this-error"></a>Oprava této chyby

1. Atribut z deklarace ho odeberte nebo nahraďte atributem, který je platný pro daný kontext.

2. Zkontrolujte v uvozovkách chybovou zprávu a proveďte příslušné akce.
