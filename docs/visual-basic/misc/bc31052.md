# <a name="modules-cannot-be-declared-specifier"></a>Moduly nelze použít deklaraci '\<specifikátor > "
Modul byl deklarován s neplatným modifikátorem přístupu.  
  
 **ID chyby:** BC31052  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Modifikátor přístupu nahraďte modifikátor platný přístup, jako například `Public` nebo `Friend`.
