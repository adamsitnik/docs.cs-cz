# <a name="type-arguments-cannot-be-applied-to-the-expression-expression"></a>Argumenty typu nelze použít pro výraz '\<výraz >.
Alias importu je definována s [z](../../visual-basic/language-reference/statements/of-clause.md) klauzule, předá typ argumentů aliasu importu.  
  
 Argumenty typu se používají pro obecné typy a pouze třídy, struktury, rozhraní a postupy a delegáti může být obecný. Obory názvů ani import aliasů můžou být obecné.  
  
 **ID chyby:** BC32058  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Of` klauzule a jeho argumentů typu z aliasu importu.
