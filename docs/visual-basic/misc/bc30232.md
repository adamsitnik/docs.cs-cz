# <a name="implemented-type-must-be-an-interface"></a>Implementovaným typem musí být rozhraní
`Implements` Příkaz neurčuje rozhraní. Třída může implementovat pouze rozhraní.  
  
 **ID chyby:** BC30232  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že není správně napsaný název rozhraní.  
  
2.  Pokud příkaz určuje třídu, zvažte použití `Inherits` příkazu.
