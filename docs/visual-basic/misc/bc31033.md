# <a name="interface-interfacename-can-be-implemented-only-once-by-this-type"></a>Rozhraní '\<interfacename >' můžete pomocí tohoto typu implementovat jenom jednou
Zadané rozhraní je již implementováno prostřednictvím tohoto typu a nejde znova implementovat.  
  
 **ID chyby:** BC31033  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Implementace tohoto rozhraní pomocí tohoto typu pouze jednou.
