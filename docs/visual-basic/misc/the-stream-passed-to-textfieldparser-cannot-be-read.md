# <a name="the-stream-passed-to-textfieldparser-cannot-be-read"></a>Datový proud předaný parametru TextFieldParser nelze číst.
[Objektu TextFieldParser](../../visual-basic/language-reference/objects/textfieldparser-object.md) nelze číst datový proud, který byl předán do něj. To může být výsledkem pokusu o čtení souboru, který není textový soubor.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, že soubor je textový soubor.
