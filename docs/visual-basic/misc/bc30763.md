# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>Cílová verze rozhraní .NET Compact Framework nepodporuje použití modifikátor Ansi, Auto ani Unicode
Verze rozhraní .NET Compact Framework práci se nepodporuje `Ansi`, `Auto`, a `Unicode` modifikátory.  
  
 **ID chyby:** BC30763  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte modifikátor.
