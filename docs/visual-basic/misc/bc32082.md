<span data-ttu-id="e551e-103">Obecný typ se vyvolala s `MustInherit` třídu jako typ argumentu, zatímco odpovídajícího parametru typu je deklarována s `New` omezení.</span><span class="sxs-lookup"><span data-stu-id="e551e-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>
