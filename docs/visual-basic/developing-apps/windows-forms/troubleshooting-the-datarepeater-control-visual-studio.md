 <span data-ttu-id="23979-110">Výjimkou je, když <xref:System.Windows.Forms.Control.Padding%2A> je nastavena na velkou hodnotu dost vystavit okraji <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> ovládacího prvku.</span><span class="sxs-lookup"><span data-stu-id="23979-110">The exception to this is when the <xref:System.Windows.Forms.Control.Padding%2A> property is set to a large enough value to expose the edges of the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control.</span></span> <span data-ttu-id="23979-111">V takovém případě kliknutí na okraji vystavené vyvolá událost myši.</span><span class="sxs-lookup"><span data-stu-id="23979-111">In this case, clicking in the exposed margin will raise mouse events.</span></span>
