# <a name="getting-information-about-the-computer-visual-basic"></a>Získání informací o počítači (Visual Basic)
`My.Computer.Info` Objekt poskytuje vlastnosti pro získání informací o paměť, načtená sestavení, název a operační systém počítače.  
  
## <a name="remarks"></a>Poznámky  
 V této tabulce jsou uvedeny úlohy, které běžně provést prostřednictvím `My.Computer.Info` objektu a odkazy na témata představením toho, jak provést jednotlivé.  
  
|Chcete-li|Další informace naleznete v tématu|  
|---|---|   
|Určení, kolik virtuální adresní prostor je k dispozici pro počítač, na kterém je nainstalovaná aplikace|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.TotalVirtualMemory%2A>|  
|Určete typ platformy počítače, na kterém aplikace běží|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.OSPlatform%2A>|  
|Zjistit operační systém počítače, na kterém aplikace běží|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.OSFullName%2A>|  
|Určit, jaké aktualizace service Pack byly nainstalovány v počítači, na kterém aplikace běží|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.OSVersion%2A>|  
|Určení nainstalované `UICulture` na počítači, na kterém je aplikace spuštěna.|<xref:Microsoft.VisualBasic.Devices.ComputerInfo.InstalledUICulture%2A>|
