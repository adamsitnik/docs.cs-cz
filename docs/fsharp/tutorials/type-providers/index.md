<span data-ttu-id="21225-114">Tvoří poskytovatelů typů vytvořit typy, které lze zapsat jako typy .NET do sestavení, ve kterém se vytváří.</span><span class="sxs-lookup"><span data-stu-id="21225-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="21225-115">To umožňuje jejich využít z kódu v jiných sestaveních.</span><span class="sxs-lookup"><span data-stu-id="21225-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="21225-116">To znamená, že zadaný reprezentaci zdroje dat obecně musí být ten, který je možné použít k reprezentaci s typy .NET.</span><span class="sxs-lookup"><span data-stu-id="21225-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>
