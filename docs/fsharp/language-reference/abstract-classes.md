# <a name="abstract-classes"></a><span data-ttu-id="3ca2b-103">Abstraktní třídy</span><span class="sxs-lookup"><span data-stu-id="3ca2b-103">Abstract Classes</span></span>

<span data-ttu-id="3ca2b-104">*Abstraktní třídy* jsou třídy, které ponechte některé nebo všechny členy neimplementované, takže může být poskytnuty implementace z odvozených tříd.</span><span class="sxs-lookup"><span data-stu-id="3ca2b-104">*Abstract classes* are classes that leave some or all members unimplemented, so that implementations can be provided by derived classes.</span></span>

## <a name="syntax"></a><span data-ttu-id="3ca2b-105">Syntaxe</span><span class="sxs-lookup"><span data-stu-id="3ca2b-105">Syntax</span></span>

```fsharp
// Abstract class syntax.
[<AbstractClass>]
type [ accessibility-modifier ] abstract-class-name =
[ inherit base-class-or-interface-name ]
[ abstract-member-declarations-and-member-definitions ]

// Abstract member syntax.
abstract member member-name : type-signature
```
